


Een waard vrouw // doen ik veel vreugde bedrijven
Al hebben zeggen rouw // somtijds dat kan nz schaen
Op eer en trouw // min ic die fleur der wijven
Als ickse aanschouw // zo moet ik zingen gaan
Want zeggen ik nauw // bevrijd voor al vaar
Op dat zeggen mogen ghedencken mijn
Sal ik mijn hertcken schenk // fijn
Mijn lief eerbaar // tot haren nieuwen Iaer


Al was rijzen // tot indië dood vol vrees
Mijn jonstich hart ja indië grond versmoord
Sy kant genezen // als zeggen dokter wil wees
Van al smart met een wellustig wart
Dus wil ic voort verlaten droefheid zwaar
Wil zeggen slechts liefelijk winken // neer
En trouw ghewillich schenk // weer
My voor een aat tot mijn nieuwen jaar


Venus kind // wild daar enten hard steken
Dat zeggen beminnen een weinig t ’ mijn baat
op dat verwinnen mijn veel ootmoedig smeken
dat nu noch enten // alderquaetst blaken staan
Ick roep om raad // al die Goden te gaan
Dat zeggen ik helpen minnen // toch
Op dat ik mogen gewinnen // noch
Een blij maar voor tgeheele nieuw uaer


Groot geneugte // met dansen en springen
Dal ic volbrengen // om mijn Princesse god
Of zij deur vreugde of deur wellustig steeg
Eens willen geheng dat ic de vruchten zoet
Pluckte tot boet // van ons liefde eenpaar
En ickse mogen verwarmen // lang
En rusten in haar armen // blank
Dicht nevens haar // van een jaar enten aar
