


OLjeave ' HEER ' loyts ' loytse ' om mijn wed 're eagtrien
Och ! litt ' mijn gâl gunnen treastmijld eansicht ' neakje
Meyts ' ik wolwitt'ne ney gunnen waard zo schieden
Litt ' litt ' mijn kiermljuentj ' for gunnen eansicht bleakje
Jon frjuenlijckheyt dear al mijn hoop ' op stiet
Holp ' ik ney gunnen tosiz dear ik heet ney heakje


Mijn lipp ' mijn lipp ' sil mei ' goll ' gjealps uwtjiet
Jon great loftanck ijn tancks oerfloede ' uwtstirtte
Az Y ik gunnen ijnset lederen ' ( ô zo suite ! )
Mijn tong ' sil eernje ' ijn sill'ge kâts tijdkirtte
Fen gunnen petear Want gunnen geboden rjuecht
Rjuechtfirdigh goed ljue treastje ' in quea folck snirtte


Litt ' gunnen hân kom ' ijn mijn aeng gaestgefjuecht
Litt ' gunnen holpredde ' hân toe mijn helpen ' dolkomme
ik hab gunnen befelne ' uwtkerd klear enkel sljuecht
O HEER ' ik langje ' az ney ' heagst'wolfertsomme
Ick langje ' ik langje ney gunnen filligheyt
Jon Wet iz mijn formeytsmijld ' hertsblier ' bloem


Litt ' ik libbje ' ik sil ijn hertswilligheyt '
Jon lof lofsjongje ' uwtherpje prijzje ' in tanckje
Wier toe ik holpj'jon rjuechters billijkheid
O ! ik rood az 't paedspoar'bjuest 're scheppen gâljanckjen
HEER ' sijk gunnen tjienner du ijnn ' siilligheyt '
Jon heet onthâdt dat zijn gaest det ontspranckjen
