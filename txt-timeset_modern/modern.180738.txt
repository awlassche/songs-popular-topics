


EEn winters nacht
Heeft ik zo langen tijd verdrieten
Lanck eend onzacht
Waer in de Mane staan besluiten
Vochtich en koud // sieren menigvoud
Waren de velden sieren begieten
Met hagel snede // dat doen ik wee
Waer door de velden ploten
Groot noch kleden // en vanden ik gheen
Cruydt noch noten


Den nacht sieren stout
Maeckt ik geheel bedroefd van binnen
Den Winter koud
Benout ons arm huisgezin
Mijn blijdschap smelt // van gaan door t ’ veld
Coudt en nat was ons gewinnen
Int donker vrij // zo lopen wi
Dolende botten weg
Maer wi hebben eer lang // Gode zeggen dank
Een licht krijgen


Den dageraad
Een morgenster licht van schijn
De Sonne op gaan
Den nacht is weeg en al ons pijne
Door t ’ zoet geclanck // der vogelen zang
Den Winter die moeten verdwijnen
Want schoon // idoon // staan daar de kroon
Van den Hemelschen boud
Het komen al uit // lover en kruid
Blommekerssouwe


Een bloem enten dal
Een Roose tot Saron prijzen
Die over al
Haer reuk geven reen uitgelezen
Een lelie zoet // vol van ootmoed
Was tuisen twee doorn rijzen
Dien Helt // in t ’ veld // bedrijven geweld
Als een Leeuw ontspringen
Den stok vol haat // des drijver kwaad
Heeft hei ontwronghen


Een bloyende rijs
Dat dragen amandel bekwaam
Crachtich en wijs
Dat zand Godt selve in zijn Name
Eenen Mey met blad // sieren wijt ontdoen
Die dragen zo goed vrucht eerzaam
Hy groeien // en bloeien // sieren vriend ’ lijk besproeien
Door des Somers voorbrengen
Int zoet dal // der vogel gheschal
Fonteynen springen


Noch vanden ik daar
Een Maghet schoon reen uitverkorene
Haer stem klaar
Was ik vertonen // sieren zoet om hoog
Haer ogen laven // met melk begaafd
Haren hals was als Davids toorn
Noyt mei // zo vei // na mijnen grei
Die ik zo wel gheriefde
Wt s ’ Vaders schoot // haar moeder groot
Dat was de liefde


Comt hier mijn vriend
Sprack daar die zuiverlijk scholen
Eer dat u dienen
Wijn eend melk geven ik te loon
Een cofferken god // vol druif zoet
Is mijn vriend selve in persoon
Hy verblijd // enten krijt // en maken jolijt
Hy gaan boven de Vorsten
Hy is van var // Een busselken Mirrhe
Tusschen mijn barsten


Ick kom mijn Bruyt
Laet ons te samen gaan vermeien
En hoog t ’ gheluyt
V Moeder die wil ons gheleyden
Die u hier zand // als beloofden pand
En bezien hoe die wijngaard schreyden
Laet fijn // rozijn // en zoet wijn
Wt u barsten druipen
Mijn hart snaken // na vreugde haken
Gheeft ik te zuipen


Een vrome zoet
Heeft ik laven met haren wijn
Haer woorden god
Zijn ik Ziele een medicijn
Na haren dank // ik van vreugde klank
Sy hebben beloven het was al mijne
Ist dat // ik wat // vergare schat
En loop de vrome tegen
Tot dat ik mijn Lief // al met gherief
Hebbe krijgen


Ick eb haar
In enigheid beloeft mijn trouw
Vroom eend eerbaar
Sy is die allerliefst Vrouwe
Sy is versierd // reen ghemaniert
Haren naâm gaan boven den gouwe
Sy zeggen // t ’ is recht // o beminnen knecht
Ghy moet vast bij ik blijven
Of gij en mogen // in mijn vreugde
Doch niet beklijven


Mijn vrome zaad
Heeft seef berg schoon beklijven
Die Voskens kwaad
En hebben daar niet in bedreven
Sy staan sieren reen // onder t ’ Hemels plein
Met Roosen Lelyen verheven
Dates // cipres // loof kruid en gres
Aen mijn Lief te betrapen
Niet ik alleen // maar groot en kleden
Ende al haar knapen


Princesse eerbaar
Van den Somer enten gemeen
Seer var en na
Zijn wi brengen enten gheweene
Want // zijn verstand // is abondant
Wy zijn verlossen uit liefde rede
Elck moet // sieren vroed // vallen te voet
Met deze Bruydt idoon
Binnen t ’ Somers tijd // u zelf kwijt
Staet na de kreunen
