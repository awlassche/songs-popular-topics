


SPoeyt u voetje treden wat aan
Want Apol zijn Paerden in gaan slaan
En Auroor haar vertonen // zoetjes voor
Dus mijn vee // gaan wat rasjens dwaelt niet na de Zee :
Zoeckt Klaverwey // schuwen de Hey // op dat gij
Niet geniet // 't gheen u baren mogen verdriet :
Pluckt gewas // knabbelen gras // voor 't gebas
Van 't gedierte // ik u hoed schoon hoe sieren het tieren


Stroomt neer Beeckjes spoed u ras
Segh mijn Lief dat ik hem hier in 't gras
Heb vertoeven // al te lang ik ben bedroefd
Om dat hei // dus gaan wachten en ik laat in lij
Ey bortelen uit // met geluid // dat het stuit
Dicht omtrent // daar mijn Lief zek is gewend
Met zijn Vee // 't houden stede // laat hem ree
Hier doch zijn // ach ! ik bidden doen zo veel voor mijn


Daer rijst zoetjes 't Morgenlicht
Die het vier hebben in mijn borst gesticht
Wiens gelaat / m'heeft bekoren // en zijn praat
Mijn vervoer // doen hei eeuwig ik zijn trouwheid zweren
'K acht Konincks Kroon // en zijn Throon // niet zo schoon
Noch zo waard // al de rijkdom van de aard
Als die geen // die met reen // lang geleen
M'heeft bekoren // tot zijn Min dies hei ik behoren


Kom dan Cloris kust uw ' Vrouw
En geniet het geen gij eertijdts wouw
Want de Trouw // van ons twee Man en Vrouw
Heeft gemaakt // dies de lusjes na 't genieten haken
Kust Cloris kust // laat de lust // zijn blussen
En voldoen // d'langhgewenschte vreugde zoet
Beesjes Fluyt // maken geluid // ik ben de Bruydt
Weest verheugen // want de tijd roepen nu om geneugte
