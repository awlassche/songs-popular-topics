


OP 't rantje // aan 't kantje van een Fonteyn
Met vochtig // lochtighe Mos bedekken
Daer elk Adertje in het plein
Quam krommelingh dwalen ' afgheleckt
Een Harder zat // en in het nat
Hem spiegh'lende zang op zijn Fluyt
Laet een Dagh af // o scholen ! eer straf
De wreed Doodt mijn ogen sluiten


Kan 't wees // dat deze groot pijn
Die ik u om u Minne lij
Indien de Goon rechtvaardig zijn
Niet komen tot een zoet gevrij ?
Kan hartens gunst // en Minnekunst
Ja Liefd ' die 't al te boven gaan
U heus gemoed // niet tot het zoet
Medooghen dwingen toeverlaat ?


Die pronckend ' // en lonckende ooghjes zoet
Liefglurich // doch durich vol veynsery
Dwongen mijn dikwijls tot goeden moed
Willen zij 't nu verzweren aan ik ?
Haer hart in 't einde // zijn strafheydt wenden
Hebbense ik zo vaak zeggen :
Sou zij deze spraak // een vals spaeck
Wel lijden van haar schoonheid ?


Hoe ! zullen die ogen // ghedooghen dat zij
Verradertjes waren ? lockend de ziel
Mindeling door haar bedrieghery ?
BosNimphjes indien het zo gheviel
Al dragen gij dan // 't blanketsel van
Gheveynstheydt op u voorhoofd niet
't Schuylt in u hart // daar gheleert waard :
Paeyt haar met woorden en anders niet


In 't eend // wild wenden u strafheydt / o scholen
Hard'rinne // Vriendinne 't is meer als tijd
En geloven zo de schoonheid niet meer wonen
Houdt jeugdig vermaak en zoetigheid lijd
Is ghelijck 't gezicht dat ( laas ) zijn licht
Moet missen ja de jeugd hoe fris
Sonder lief aard // is als t ' onwaard '
De Lichaemen daar de Ziel uit is
