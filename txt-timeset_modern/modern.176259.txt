


SChoon Nymphelijn // ach mindje mijn // wat soud ik al versieren
Om na mijn wens // deze ledetjens // zo wel maken te sieren ?
Met blinken goud // of parel sold // gij voelen ras belasten
Vw halfgen zoet // dat kraal als bloed // daarom niet beter passen


Ick soud u kleden // met keursgens reen // van licht verf blijtjens
Die zullen staan // schilderen aan // u breedachtighe zijdtjens
Vw voet met // haar schoentjens wit // daar bij gheval i n't bok
Het inkarnaat // zo wel bij staan // van hoosgens net trekken


Vw armpje  mede // zo wit als snede // soud ik koraal om schikken
Dees vlechgens blond // op nieuw vinden // soud ik u leder strik
Met snoertgens veel // nu groen en geel // bij lodderlijk beurtgens ;
Voor wat jent haar // of cruyvendt daar // en duizend zoet leur


'Tperruyckgen sou // ik trek nauw // wat laechjens dan wat hoochjens
En als het klaar // ghefutselt waar // ik spiegelen in u oochjens
Dan werpen licht // nu mijn gezicht // op't een nu op't ander :
Dan nemen raam // hoe't altezamen // sou voeg bij elckander


Indien dat gij // uw oochjens blij // en liefelijk zeeden
Soo vreuchderijck // zo vriendelijk // zo vol bevallijkheid
En u aanschijn // vergenoegd tot mijn // dan met een lachen wenden :
Soo soud ik hiel // in u mijn ziel // gaan met wonen zenden


En als ik wat // beloncket hebben // den brand van al knechgens
Vw zedetjens // uw ledetjens // uw fraai tooien vlechgens ;
Soo soud ik streng // met armen eng // uw jent lijfje prangen
Tot ik daar uit // krijgen buien om buien // u zieltje weer vangen


De vaar van // mijn lippen an // uw wangsjens soud ' ik plek
Door soentgens zacht // en met haar kracht // u ziel te mond uit trek
Ick wed ik weet // gij dan beleid // dat niemand van u susjens
Daer gij zo schijnen // nu smaak in vijndt // geven ooit zo zoet kus
