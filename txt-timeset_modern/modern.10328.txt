


Als Coridon zien aan de rei // om de Mey
Elck Harder met een Harderin // na zijn zijn
Gescakelt zo spelen hei een Liet // op zijn Riet
Daer 't Meysjen op zang // en ee Knaepjen op springen
Al onder de Lindeboom // schoon om de Croon


Doch Erickje dat gij aan de rei // om de mei
Gaet spiepele zondaar ik // aan u zeggen
ik Was immers zo niet zeggen // licht meid
Maer Erickje is wel // Ick zwaar u ik zullen
Siet dat is nu evenveel // en ik speel :


Mooy zijn ja zondaar ghelijck en ook rijk
Welgunstigh aan dieje beminnen maar gij vint
Soo deugdelijk niet alsje schijnen // Ick bevijnt
Ghy spelen met mijn Hart // Ghy spelen met mijn smart
Maerr denken ik hier bij deze u val // wees zullen


Als Coridon 't Lietge hebben uit // zijn geluid
Begaf hem een droef zucht // in de Lucht
Deed steen en wede de Ieught // om zijn deugd
Doch Erickje acht // belofte noch klacht
Soo licht is de liefde verdraaien met de Meyt
