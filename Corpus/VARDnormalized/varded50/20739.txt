
    
      
        DE groene Meye // brengt ons den Feest
        Van Sint Philip' we'erom;
        wiens eer verbreien // wil onze geest,
        Door het gantse Christendom.
        Hij is // het is wis // geweest een vromen Held,
        Die Satan met geweld
        Gedreven heeft uit het veld;
        Hij is het die in onze druk
        Ons verkrijgen zal geluk,
        Hier toe is Sint Ph'lip gesteld.
      
      
        Des laat ons prijzen // des groten man,
        Des groot' Apostel Gods,
        Die zal verdrijven // ons zwaar gestan
        In de laatste uyr des doods;
        Die sall' // ons all' // beschermen op het lest,
        Die voor ons doet zijn best;
        En vrijdt van Satans nest;
        Die is oorsaeck van onze vreugd,
        Die ons voordert in de deugd,
        Die ons brengt tot des Hemels vest.
      
      
        Hij is verkoren // geweest van God
        Tot ene Kapitein,
        Eer dat geboren // (ziet wat een lot)
        Hij was op des werelds plein:
        Hij zocht // Hij brocht // tot Christvm zijnen Heer,
        De zielen meer en meer
        Door zijne trouwe leer'.
        Hij geweest is enen Tolk
        Bij den Heer, voor het Heidens volk,
        Wat is dit een grote eer!
      
      
        In swarigheden // te geven raad
        Was Ph'lippus wijs en vroed:
        Zijn goede reden // de'en menig baat,
        Zij vrochten menig goed.
        In nood // van brood // vraagt Christus Hem bescheid
        Waar me'e des zwarigheid
        Sou worden afgeleid?
        Van waar zullen kopen wij,
        het Welken mogen eten zij?
        Heeft den Heer tot Hem gezeid.
      
      
        Maar veel geleden // en uytgestaen
        Heeft dezen groten Sant,
        En vroom gestreden // op des werelds baan
        Voor des Hemels Vaderland.
        Hij lijdt // Hij strijdt // wel twintig jaren lang
        Voor Christo met verlang;
        Voor het sterven geenszins bang.
        Hij heeft Asia bekeert,
        Hij heeft Schytia geleerd
        Den oprechten Christen-gang.
      
      
        Toen is gevangen // van het vuil gespuis
        Te Aleppo in de Stad,
        En opgehangen // aan het houte kruis
        Dit Hemel-waerde Vat:
        Om-hoogh // zijn oog // Hij opsloegh na de lucht,
        Hij bad, Hij de'e noch vrucht,
        Hij was voor het Volk beducht.
        Maar eilaas! Hij wordt gesteent,
        Dewijl voor het volk Hij heeft gewend,
        All' dit was Hem groot genucht.
      
      
        De aarde scheurden // met groot gedruis,
        Aleppo was in nood;
        De beulen treurden // na Plutoos huis
        Zij daelden voor haar dood.
        Dit gaf // tot straf // die het all' geschapen heeft,
        Bij Wien nu Ph'lippus leeft,
        Voor Wien den Duivel beeft.
        O hoe wonder is den Heer,
        In het voorstaan van de eer,
        Van die lof aan Hem hier geeft!
      
      
        O Sint Philipp! // wij groeten U,
        Met grote ned'righeyd:
        Uwe mond, en lippen // doet open nu
        Tot onzer zaligheid.
        Verwerft // dat erft // onze ziel het eeuwig wel;
        Bevrijd ons van de Hell',
        En Satans boos gekwel:
        Ach! uw ogen tot ons wendt,
        Staat ons bij tot des levens end
        Aldertrouste Gods Gesel.
