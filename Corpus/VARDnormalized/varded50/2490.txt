
    
      
        Wat last is liefd' // ja recht een zware last
        Voor mijn ‘tondiefd’ // dit spreekwoord gaat vast,
        Dat voel ik nu wel in // mijn zin
        Want ik moetse nu dragen:
        Dus durf ik het klagen // vrij wagen
        Van des vervloekte min.
      
      
        O Razernij // het onrecht men u liefd' noemt,
        U dwaeserny // met schonen schijn verbloemt
        Nu heel het jonge hert // dat werd
        Door uwe glans bedrogen
        En door dijn vermogen // getogen
        In des siel-sieckt zo vert.
      
      
        Dat schijn bedriegt // dat werd ik nu wel waar,
        U faam die vliecht // van u denkt hier en daar
        Ik gh'loofden u gelaat // en praat;
        Maar foei mijn licht vertrouwen
        Nu mag ik het onthouwen // dat vrouwen
        Zijn te licht van beraad.
      
      
        het Scheen of u hert // aan het mijne waar gehecht,
        Maar wat een pert // hebt gij nu aengherecht,
        Gij verlaat mijn arm bloed // om het goed
        Moet ik mijn Lief verliezen:
        Dus moet ik van biezen // nu kiezen
        Een dor gevlochten hoed.
      
      
        U Mins ghedaant // heb ik te haast gelooft
        Vergeefs gewaand // en voor niet al geslooft
        Geen groter ongeluk // of stuk
        En mag mijn overkomen:
        Want het is mij door schromen // ontnomen
        En ik blijf in de druk.
      
      
        Maar noch Goddin // kan ik u haten // niet
        Door ware Min // al ben ik verlaten / ziet;
        Maar weest hier op niet trots // noch spots
        Ik denck het Kan verkeren,
        Dat u den tijd zal leren // begeren
        Die gij nu weigert schots.
      
      
        Prinses ik roem // van u grote kracht,
        Een Jongebloem // hebt gij te ondergebracht,
        Mijn voedsel is verkort // verdord
        Alleenlijk door u te derven,
        Ik wens menig werven // om het sterven,
        Of dat het beter wordt.
