
    
      
        AVrora den dage brengt
        Als Luna gaat te rust
        Metten dan het loof besprengt
        Tot der groen kruiden lust
        Phoebus fijn // helder schijn
        Met zijn wagen guldig // straelt
        Ceres trou // doort ghebou
        Vruchten menichfuldich // haalt
        Die vermakelijk / ende smakelijk
        Brengen schatten rijk.
      
      
        Hier door me ontluiken gaat
        De el Pomona schoon
        Deur Apol in hogen graat
        En ook Flora ydoon
        Die in fleur // zoet van geur
        Tvelt veelverwich schoon versierd
        T'Nimphen plein // net en rein
        Vreugdig vol vant zoet ghediert
        Hierin rapē // vreugd // en betrapen // g'neucht
        D'eerbaer zuiver jeugd.
      
      
        Tellus verheven Goddin
        Der vruchten principaal
        Tgeen dor was / daar sentse in
        Den wasdom teenemael
        Cleetse koen // rein en groen
        Tot des maechdes minnen // lof
        Dies verblijd // zijn altijd
        D'Nimphes in het Goddinnen // hof
        Om de Ieuchden zoet // diet veel vreugden doet
        In het klaver groen met spoed.
      
      
        Dies met Hijmen vol ghenucht
        Het Maagden bloemken bloeyt
        Laat dan varen zucht en ducht
        Dewijl het loofken groeyt
        Na t'betooch // schoon in het ooch
        Om bekoren tsoet gezicht
        In Slieffs tuin // d'ooghen bruin
        Dat gheliefgens wel verlicht
        In het betrouwē // maar // en aanschouwen // klaar
        Het zoet ghewinck van haar.
      
      
        Princelijcke Muses el
        Vyt levendige jonst
        Chiert ons aardse schoon tonneel
        Met Redenrijcke gonst
        Zo verhief // ieders Lief
        Trou een slaver die zo slaeft
        Dat hij wacht // dag en nacht
        Om van t'lieff te zijn begaeft
        Met een schone // eer // en trou loone teer
        Deur hertlijck begeer
