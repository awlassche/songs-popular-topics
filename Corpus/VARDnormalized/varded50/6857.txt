
    
      
        Glorieuse violette // Koninginn’ van Hemelrijk,
        Zuiver vat vrij zonder smette // In den geur aromatijck,
        Zoetigheid zonder gelijk,
        Zulk een vreugd gij sijt // die vermengt altijd
        Smert en leed te scheuren,
        Met vreugd, jolijt // verheugt, verblijd
        Ons weemoedig trueren.
      
      
        Precieus Paleis van trooste // blinckend’ als den dageraad,
        Sterre lichtend’ uit den ooste, als de Zonne klaar opgaat,
        Uwen lof elk metter daad
        Hondertsout verklaart // als een goe bewaard
        Iun der Hemels throone:
        Benaut, vervaard // verstand, bezwaard
        Is nu elk persoone.
      
      
        Alderheylichste Princesse // Maria Gods Moeder rein,
        Ons aller Advocatersse // Ziet ons in dit droevig plein,
        Tot u roepen wij ghemeyn:
        Voor u dienaars doet // der genaeden vloed
        Vwen Soon gedenken,
        Dat hij ons ‘tgoet // zijns rijke zoet
        Will’ hier naemaels schenken.
