
    
      
        Ik ben deurwont // Heel onghesont
        Tot in den gront // Met Venus stralen
        Ik maeckts v cont // In corter stond,
        Haar roder mond // doet mij dit verhalen
        Mag ik geen troost van v ontfaen
        Zo is mijn vreugden alghedaen
        Ik roep, o dood, wilt mij deur keruen,
        Met zwaar verdriet ben ik bevaen,
        Wilt gij mijn niet ter noot bijstaan
        dat ghi eens ziet op mijn vermaan, zeer zaan
        Door liefden plaan, Zo moet ik steruen.
      
      
        Wat zijt gij ghebaet//schone sieraad
        Dat ik den Prelaat // zal moeten blijuen
        Mijn doch ontfaet // eer tis te laat
        Wilt niet niet obstinaat verstijuen
        Ik hadts nooit op v bertrout
        Dat gij mijn dus verlaten zout
        Ik meende vreugd te verweruen,
        Nu compt mijn tlijden menichfout
        Dat mijn tlijden den Geest verrout
        Ik moet vermijden voor jonk en oud, verstond
        Al dat mijn rond // dat ik moet steruen.
      
      
        Princesse fier // al ben ik hier
        Van v dus schier // doorsteken
        Met zwaar dangier // en druk bestier
        Rein balsonier // zegt v ghebreecken
        Waarom gij mij dat lijden aendoet
        Ik min v bouen goud ofte goed
        Mocht ik eens vreugd van v eruen
        Al is het mijn zwaar dit tegenspoed,
        Berooft dat waar mijn lijdens spoed
        schoon lief erbaer verheugen moet, mijn groet
        Met woorden zoet // eer ik moet steruen.
