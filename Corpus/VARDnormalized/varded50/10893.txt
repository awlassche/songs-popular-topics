
    
      
        Schoon bloemken jent // gij staat in mijne zinnen
        Want excellent // zijd gij versierd met denkt,
        Waar deur present // in thert verheugt van binnen
        Als ik ontrent // u ben, dan heb ik vreugd,
        Gheen meerder denkt // en weet te verr’ oft naar
        Dan dat ik u verwerven // mocht
        Eer dat mijn ziell’ om sterven // docht
        En mij tot uwer erven // brocht
        Met dezen nieuwen jaar.
      
      
        Al mijn gemoet // heeft naar u een verlangen
        Door liefde zoet // die mij tot u verwekt,
        Mijn vlees en bloed // is met liefde bevangen
        Hierom Lief doet enig teken perfeckt
        Het welke strekt // tot rein liefde eerbaar:
        Doet als een goey vriendinne // plag
        Op dat ickt zo verwinne // mag
        Twelck mij ooit hert van binne // lach
        Met dezen nieuwen jaar.
      
      
        Rein Maget fier // hier in deze Landbouwen
        Weest goedertier // want ik nacht ende dag,
        pijn en dangier // heb om u bloen der vrouwen
        Zo dat ik schier // niet en ben die ik plag
        V oog-opslag // verwekt mijn lijden zwaar
        Daarom wordt u gebeden // fijn
        Dat gij eens laat de reden // dijn
        Mij wezen tot een medecijn
        Met dezen nieuwen jaar.
      
      
        Al mijn gepeys // strekt tot uwen persoone
        Menigher eis // komt gij in mijnen zin:
        Der liefden eis // doet u spannen de Krone
        Nochtans mijn vleys // ligt daarom niet te min
        Hierom vriendin // zendt mij een blijde maar:
        Met een woord dat vertroosten // moet
        En beleefde propoosten // zoet
        Eer mij den wind van Oosten // doet
        Weg varen dit Nieu-jaer.
      
      
        Princeslijck zaad // verstaat // wel mijn begeren
        Doch niet en laat // verstaat // mij met confoort,
        Der liefden graat // aenvaet // zonder cesseren
        met zulke daad // en praat // die druk verstoort
        Hebt geen discoord // met mij uwen dienaar,
        Wilt mij liever ontfermen // och
        In u blancke schoon armen // doch
        Verhoord mijn bitter karmen // noch
        Met dezen Nieuwen jaar.
