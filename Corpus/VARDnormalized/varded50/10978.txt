
    
      
        Cupidoos schicht // mz recht bedicht
        Die kracht der minnen,
        In het hard brand sticht // en mij aan een verplicht,
        Wiens lieffelijck gezicht // mij als der Zonne licht
        Doorschijnt van binnen,
        In gaven rijk, ik nooit en vandt
        Een haars gelijk // in kloek verstand,
        Maar laas ik vind in mij gheplant
        Twee die partijen,
        Om de overhand // beyd’ even vaillant,
        Dats hope en vreze, want // zij steken vrij den brand
        In het harde om stryen.
      
      
        De hope zoet // wil in t’ghemoet
        Een starckte bouwen,
        Maar vreze ontdoet // en werpts’ onder de voet,
        De een tuyght kwaad dander goed, geluk oft tegenspoed
        Wordt mij ghebrouwen,
        Och of tgheluck // mij gunstig waar,
        En vreugd na druk // wou schenken klaar
        Gheen pijn zo groot die mij te zwaar
        Zoudt zijn om dragen,
        Mocht ik met haar // mijn liefste Lief eerbaar
        In echt verlost van vaar // als een getrou Minnaar,
        Einden mijn dagen.
      
      
        Wat vreugde kan doen // dit te vermoen
        Alleen inwendig?
        Ik wil mij spoen // bij haar, om mijn behoen
        En maken mij zo koen // dat ik haar deed bevroen
        Mijn druk ellendig:
        Maar schaemtens macht // verbiedet mij
        En liefkens kracht // zeid, doet het vrij,
        Op hoop van troost wil ik mij bij
        De Liefste voegen,
        Weg schaamte gij // en vreze aan de een zij
        gij dos slechs dz ik lij // o venus maakt dz wt
        Ons beide ghenoeghten.
      
      
        Prinses mijn noot // en lijden groot
        Wilt doch ontfermen,
        Levende dood // verwacht schoon Lief minjoot
        Wt u mondeken root // een troostelijk woord alsoot
        Vereyscht mijn kermen:
        Mijn dienst oprecht // ik presenteer
        V inder Echt // op denkt end’ eer,
        Gij zijt alleen die ik begeer,
        Mijns hartsen leven,
        En werpt met veer // hem jonstich te zien weer,
        wiens pijne is veel meer // dan hij kan klagen zeer,
        Oft kan aangeven.
