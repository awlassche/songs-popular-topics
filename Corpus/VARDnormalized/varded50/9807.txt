
    
      
        O Mense // gij
        Die nu Gods heilig woord // hoort,
        Ten wensche // vrij
        Brengt vruchten in akkoord // voort
        Verduldig // lijdt gehuldig // strijdt,
        Verblijd in 'sHeeren // kruike vol eren:
        Als God zijn kind, bemindt, sorchvuldigh smijt,
        Onschuldig kwijt, hij het door 'tverseeren,
        Van het Hels zwaar // misbaar maar den zondaar
        Die moet hier naar // lijden eenpaar
        In het vier, tis klaar // dus wee die haar,
        Niet en bekeren.
      
      
        Onlanghe duert,
        Der beproevinge vier // hier,
        Die ganghe truert,
        Een lachen goedertier // schier,
        Met vreugden // bloed // in jeugden // zoet
        Eeuwig na dezen // salt al genezen,
        Bidt en waakt // en maakt // verheuchdenmoet
        In deuchden // goed, looft God geprezen,
        In het foreest, en vreest, eest, geen tempeest,
        Gods woord ghekeest // ter herten meest
        Alzo wel heeft // laat in den geest
        Vruchtbarig wezen.
      
      
        Den sterken // schild
        Des geloofs neemt nu an // dan,
        Aanmerken // wilt
        Christus uwen Hoofdman // van,
        Zijnen wegen // bloot, door tegen // stoot,
        En wilt niet wijken // noch ook gelijken
        Loths Wijf // wiens lijf // heeft kregen // noot,
        Versleghen // dood, door het ommekijcken
        Werd zij een, zoutsteen, gheen teken kleen,
        hetwelk zij gemeen // wel mogen reen,
        Trekken te been // om in het gheween
        Niet te bezwijken.
      
      
        In het Aarden // vat,
        Zo den Prinse behaagt // draagt
        Den waerden // schat,
        Na vrede onvertsaeght // jaagt,
        In smeken // zijn, wilt weken dijn
        Bedd' en slaapstede // in den gebede:
        Als Paulus // doet dus, in het rekken, pijn
        Na teken // fijn, met neerstichede
        Dag en nacht, heb acht, wacht, na u macht
        Die met heyrkracht // komt onverdacht,
        Op dat gij lacht // bij hem gebracht,
        Eeuwig in vrede.
