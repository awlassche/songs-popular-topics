
    
      
        O Soetvloeyende beke //
        Rein speciale // dracht:
        Iont (troostbaar apote-ke) //
        Medecinale // kracht:
        En maakt mijn quale // zacht //
        Want hier ter wereld gene //
        En heeft dan gij allene //
        De principale // macht.
      
      
        Schoon lief, in wien ik stille
        Heusche maniere // vant:
        Ontfermt (is het uwen wille)
        T'hert, dwelck in viere // brand:
        O schoon Lauriere // plant,
        Al doet gij mij des smerte,
        Noch schinck ik v mijn herte,
        Is het niet een diere // pand?
      
      
        Dit pand, als cleen van werde,
        Gij luttel gade // slaat:
        Ik moet hemel // en eerde,
        Klagen, des kwade // daad:
        Lief, eens te rade // gaat,
        Of gij geen troost zijt schuldig,
        V dienaar, die gehuldig
        Naar v genade // staat?
      
      
        Al is het, dat ik ontsinne,
        Ik heb om klagen // noot:
        Want onvergolden minne
        Moet ik nu dragen // bloot,
        Liever geslagen // dood,
        Dan te minnen met pijne,
        En niet gemind te zijne,
        T'is een mishagen // groot.
      
      
        Wat baat dat aardse weelde
        Is om verwerven // goed:
        Als gij (o schoonste beelde)
        Mijn herten sterven // doet,
        Cruyt, noch conserven // zoet,
        Konnen geen droufheyt scheuren,
        Al s'weerelts vreugd is treuren,
        Als ik u deruen // moet.
      
      
        Prinselijk zal ik blijuen
        V dienaar enen // fijn:
        Al wildy mij ontlijuen,
        En voor troost, genen // brijn,
        Coemt met verheuen // wijn,
        En laeft mij, o Meestresse:
        Want gij zult mijn Princesse
        Nu, en naar t'leuen // zijn.
