
    
      
        In het Hemelrijk // daar Christus die zal wijzen
        Die hem gelijk // hebben gediend voorwaar,
        Met daad en blijk // in kleden, drinken, spijzen
        Zonder bezwijk // in allen noden zwaar,
        Nu hier en daar // aan die klein Christen schaer
        Is vreugd, zo daar geschreven staat,
        Die hij den zijnen geven gaat,
        Hoort allegaar // van dezer vreugden klaar.
      
      
        Af nu geleid // den dood, en van het sterven
        Verzekerdheid // nu van den tweeden dood
        Verlost, bevrijd // het leven te beerven
        Eeuwig voorzeid // van den zwaren aanstoot
        Wt aller noot // verlost uit liefden bloot,
        Want die dood is verslonden snoo,
        Als zij dit wel deurgronden zo,
        Verheugt devoot // zo wordt haar blijschap groot.
      
      
        Komen dan voort // uit alle p’rijckels noden
        S’duyvels verstoort // en van der hellen pijn,
        Om met akkoord // God te loven der goden,
        Die naar zijn woord // haar heeft v’lost devijn
        Dit sl den schijn // van haarder vreugden zijn,
        Verlost uit s’vyandts tanden fel
        Door Christi sterke handen snel,
        Des medicijn // verhoogd haar t’herte fijn.
      
      
        Ook zullen zij // vrolijk zijn en v’hoogen,
        Van herten blij // om dat ze binnen gaan
        Ter plaatsen vrij // daar zij g’lucksalich mogen
        Eeuwelijk bij // die uitverkoren staan,
        God wel gedaan // eeuwelijk schouwen aan,
        Och wat vreugde komt daar ontrent
        Boven in het Hemels klaar convent
        Naar Schrifts vermaan // die zij zullen ontfaen.
      
      
        Blijdlic v’jeucht // zal haar hert zijn ontsteken
        Heel wel v’neucht // om dat ze Christum schoon
        Vol aller denkt // zullen zien onbezweken,
        T’vermeerdert haar vreugd // boven in s’Hemels troon,
        Die haar ydoon // op zet des levens Kroon,
        Palmtacxkens vyerich groeyende
        In haar hand tierig bloeyende,
        Dit geeft tot loon // Christus Jezus Gods Soon.
      
      
        Treuren vergat // die blijschap zal v’meeren
        Als zij t’ghelaet // der Engelen getal
        Hebben te baat // bij den Here der heren
        Vroeg ende laat // tot gezelschap voor al:
        Vrienden hoe zal // daar vreuchd’ zijn met geschal,
        Als zij bij alle vromen hoort,
        En heiligen daar komen voort
        Van het jammerdal // bevrijd van ongeval.
      
      
        Op dat ze niet // naakt en bloot en v’toonen
        God haar aanziet // het tweevoudighe cleyt
        Des lichaams ziet // eerlijk bij hem te wonen
        Daar over giet // hij nu die zaligheid,
        Maakt ook bereid // t’verciersel uitgebreid
        Der zielen // d’welck is hovelijck,
        Gerechtigheid zeer lovelijck,
        All’ dit voorzeid // bringt blijschappe planteit.
      
      
        Nooit smaak zo zoet // noch lieflijcker aanschouwen
        Dan ons ghemoet // met onzen God certein
        Verenigd vroed // zal zijn met vast betrouwen
        God met ons goed // die heiligen gemeen,
        Alle groot en cleen // worden te samen een
        ‘Met ons, en wij met haar, en wel
        Zal men daar op het snaren-spel
        Des herten rein // spelen in s’Hemels plein.
      
      
        In het hemels land // zeer vast met God v’bonden
        Der liefden band // ons zo met hem verbint,
        Aan elken kant // zal des liefd’ te allen stonden
        Zeer abondant // wassen als zij begint,
        Seere bemint // staat dit niet in den wind,
        Om gaan tot deze rusten in
        Keert daar toe uwen lust en zin,
        Wordt zo gezind // t’is t’beste dat men vint.
      
      
        Sluytlic hier me // vrienden wilt doch gedincken
        In s’hemels ste // te komen, op den pad,
        Maakt u doch ree // dinct hoemen daar zal blinken
        Witter dan snee // ja als der Zonnen rat,
        Dinct vrienden wat // groter vreugden is dat
        Wie machse doch uitspreken hier,
        T’verstandt moet hem ontbreken schier,
        Zoekt dezen schat // boven in s’hemels stad.
