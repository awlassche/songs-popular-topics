
    
      
        MElancolijck,
        Smelt in verzijk,
        Mijn ionckhert afgestooten in den gront,
        O mangnifijck,
        Fleur van aardrijk,
        Gij zijt hier, die mijn lijden stelpen cont:
        Hoe fel Cupido mij deurwont,
        Noch even seere,
        Gij troostbaar ieught,
        Krachtig vermeught,
        Dat ik met vreugd,
        Victorieus floreere:
        Dies moet ik u mijn leven // geven // ere.
      
      
        Eeuwig gij zijt,
        Gebenedijt,
        Eerweerdich, als een wtgesondert // kind:
        Of iemand spijt,
        Mijn hert altijd,
        V alleen bouen duizend honderd // mint:
        Den geest hem gans verwonderd // vint,
        Als ik aenschouwe,
        Hoe gij u draagt,
        Als eerbaar maagd:
        Mij en behaagt
        (Gesproken op mijn trouwe)
        Nergens geen zo idoonen // schonen // vrouwe
      
      
        Zo menigmaal,
        Maak ik verhaal,
        Van u (schoon lief) als mijn jonk hert verzucht:
        Gij maagd loyaal,
        Zijt principaal,
        In sulcken noot, mijn schild, borght, en toevlucht
        Voor geen dangier mijn ziele ducht,
        Als gij goddinne
        Geheel mijn wreed,
        Hertgrondich leed,
        Zo overkleed,
        Dat ik secours gewinne:
        Och gij draagt boven laste // vaste // minne!
      
      
        Van u absent
        T'zijn een moment,
        Mij dit leven zeer doloreus bezwaard:
        Zo partinent,
        En excelent,
        Is u minzaam bijwezen rein geaard:
        Daar u schoonheid haar niet verklaart,
        De vreugd is kleine:
        Gelijk een hert,
        Naer-geiaeght werd,
        En dorstig smert,
        Naar een zoete fonteine,
        S'gelijcx ik naar u touve // souve- // reine.
      
      
        Sukade // gij
        Voor schade // mij
        Bevrijd, als mij de boze weder // staan:
        Is t'quade // bij?
        Te rade // dij
        Heb ik, wie zal mij konnen neder // slaan?
        Tot u moet ik (O cedar) // gaan,
        Om t'zijn genezen:
        V puere // hand
        De cuere // vant,
        Des ure // spant
        Gij de kroon wtgelesen,
        Hoe kan ergens zo reene // ene // wezen?
      
      
        Princesse wijs,
        Gij zijt den prijs,
        Die mij elk dus lange vertouven // sach:
        Gij dan (propijs,
        Bloyende rijs)
        Weet al dat tot mijn smert behouven // mag:
        Geeft (voor zo menig drouven // dag)
        Een cord verblijden:
        T'zy wat verdriet,
        Mij dan geschiet,
        Ken achtet niet,
        Dan een vast toevertijden:
        Al moet ik noch langdeurich // treurig, lijden.
