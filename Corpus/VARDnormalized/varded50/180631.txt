
    
      
        Rein van manieren // is den Echten staat,
        Die hem hantieren // in den rechten graat,
        T’is slechten raad // die hem verbieden, want
        Beter is trouwen // dan te lijden brand:
        Want die hoereren // mal,
        God condemneren // zal,
        Des wij lauderen // al,
        T’huwelijcx band.
      
      
        T’is waar dat vrienden // in malcanders noot,
        Dikwijls wel dienden // als bystanders bloot,
        Maar anders groot // met een affecty vrij,
        Staan man en wijf ter noot // malkander bij:
        Wie een goe Vrouwe // heeft,
        Die God getrouwe // geeft
        Bevrijd van rouwe // leeft
        Te langer hij.
      
      
        Een wijf die deugdig, is van wezen vroed
        s’Mans herte vreugdig // zij genezen doet,
        Geprezen moet zij zijn om haar bedrijf,
        Een edel gave Gods is zulk een Wijf
        Tot troost van haren // Man
        Die wel bewaren // kan,
        Haar tong en sparen // van
        Weder gekijf.
      
      
        Zo lief bekwame // op der eerden niet
        Als een eersame // Wijf vol werden ziet,
        Met zweerden vliet // een Rover in der nacht,
        En brengt den roof haar die boven geslacht
        Hij voor vriendinne // koos
        Veel zijn door minne // boos
        Geworden sinne-//loos
        En omgebracht.
      
      
        Een vroom Wijf schone // is als brande daar
        De lamp ydoone // op’t heilig Kandelaar,
        Zijnde bestande haar // eerbaar ghemoet
        Een gulden Pilaar op een zilveren voet,
        Is zij gheleken // wel
        Om niet te breken // snel
        Oft te ontsteken // fel
        Door bozen gloed.
      
      
        Die een Wijf zedig // voor zijn proven hout
        Huylich, cuysch, zedig // machse loven stout,
        Al boven goud // en boven edel steyn,
        Haar sieraad is wat neerstich is en rein
        Al moets’ hier lijden // smert
        Haar vreugd in tijden // werd
        Met een zo blijden // hert
        In s’Hemels plein.
      
      
        t’Wijf zal door baren // ende kinder dracht,
        Ten Hemel varen // dat ze ginder lacht
        Zo in der kracht // ghleoove, liefde jent,
        Heiligheid, eerbaarheid, haar blijft ontrent
        Voor des versijcke // wat
        Werd zij in Gods rijke // stad
        de engels gelijke // dat
        Werd excelent.
      
      
        Prinse de oprechte // zo zij dede neer,
        Zal men in Echte // niet besteden meer,
        Te vreden zeer // wel eeuwig zonder pijn
        Zullen zij in s’Lams bruiloft feeste zijn
        Zo vrolijk maken // schier
        Elk mag wel haken // fier
        En neerstich waken // hier
        Om daar te zijn.
