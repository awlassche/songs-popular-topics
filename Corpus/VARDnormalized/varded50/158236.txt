
    
      
        Gij Christen schaer // eerbaar
        neemt uwen roep klaar // wel waar
        Om verblijden hier naar // aldaar
        Geen scheiden en werd meer
        Als rechte Israelijten,
        Volgende Christus leer
        Laat ons beloften kwijten
        Dat wij vaten ter eer,
        Oorbaar tot elken keer.,
        Wezen vanden Huis Heer
        Dat door goed gerucht // bewucht
        De blinde met vreugd // tot denkt
        En op dat gij meucht // vol jeugd
        wassen in Christo zeer.
      
      
        Wast nu ter tijd // met vlijt
        Als ranken wijt // in het krijt
        Op dat gij sijt // bevrijd
        Van Gods toren vertsaagd
        Die hem zal openbaren
        Zo Paulus gewaagd,
        Op alle boze scharen
        Die na God niet en vraagt
        Tsal noch worden beklaeght
        Wie in hem geen vrucht en dracht
        Zal hij na schrifs bedien // af snien
        Dus wilt wel in tien // toe zien
        En de boosheid vlien // verbien
        Zal hij, die God behaagt,
      
      
        De boosheid kwaad // verstaat,
        Niet metten praat // maar daad
        Met goeden raad // na gaat
        Christum den Herder fijn
        Die ons heeft genezen.
        met Oly, en met wijn
        En veel liefde bewezen
        Als onzen Medecijn
        Hij droeg al ons gepijn
        Deur zijn woorden devijn,
        Genas na schrifs vertel // zeer snel
        Israel // ook wel
        Van der Slangen fel // tgequel
        Was dodelijk fenijn
      
      
        Tfenijn zeer bloot // was groot
        Dat de Slangen hot // uit schoot
        Maar deur Christus dood // minjoot
        Is dat tot niet gebracht,
        Sathan die heeft verloren
        Zijn geweldige macht
        Daarom is hij vol toren
        Zoekende dag en nacht
        Laat ons houden goe wacht
        Het uitverkoren geslacht
        Gaat hij met behent // ontrent,
        Als bekent // die schent
        Op dat hij verblent // afwent
        Van God hoge geacht
      
      
        Gaat ongestoord // doch voort
        Door de enge Poort // zot behoort
        En maakt geen discoord // in het woord
        Als eerlijke leen,
        En wilt niet tegen strijden
        Gelijk de Phariseen
        Die den Here benijden,
        En verstieren zijn reen
        Dat was meest dat zij deen
        Want zij waren van benen,
        En hebben den Heiland // vayljant
        Aan een kruis gespant // met schandt
        En dien hij sant // in het land
        Die hebben zij ook bestreen
      
      
        Sathan bevecht // Gods knecht
        Die nu tonrecht // afleght
        Zo ons Job zegt // die slecht,
        God vreesde en zijn gebod
        Zijn wijf die hem wel kende
        Die heeft hem ook bespot
        En sprak in zijn ellende
        Blijft gij noch vast bij God
        Job die antwoorde tot
        Haar, gij spreekt als wijven zot
        Want hebben wij dat goed // zeer vroed
        Met overvloed // men moet
        Ook tegenspoed // onzoet
        Ontvangen voor ons Lot.
