
    
      
        SChoon da- geraet //
        Die als Aurora klaar opgaat //
        En alle duisterheid verclaer- de //
        V amoureus seedbaer gelaat //
        Passeert zeer wijt der maag den staat //
        O alderschoonste Rosegaer- de! //
        Ten leeft van aarde //
        Zo wreden man op aerderijck,
        Die u zoet by-zijn ooit beswaerde //
        Dies ik u in mijn hert bewaarde //
        Als enen bruyt- schat weerdelick.
      
      
        V weerdicheyt?
        Drijft mij tot meerder veerdicheyt,
        Als t'quaet gespuis tempeestich dondert:
        Den loon van mijn volherdicheyt,
        Schat ik in zulker eerdicheyt,
        Al had' ik Iouis hof geplondert:
        Gans wtgesondert,
        Vermaard, boven wtlanders // wijn,
        Zydy schoon lief: wt duizend honderd,
        Dat het de goden zelfs verwonderd:
        Hoe zoudt ook konnen anders // zijn.
      
      
        Reden is het dan,
        Dat ik u (als vroom edelman)
        Bouen mijn eigen herte minne:
        Deur u, ickt al verwinnen kan,
        Gelijk als Iason t'vlies gewan
        Bij raad van zijn liefste vriendinne:
        Als ik versinne,
        Wat gij mij doet, naar wensen // meest:
        Zo dunkt mij dat een Cherubinne,
        Mij voetstaans daalt ter herten inne,
        De vreugd begreep nooit mensen // geest.
      
      
        Het is zonder noot,
        Dat ik in vreze van der dood,
        Als Schaepken zonder herder dwale:
        T'vertrouwen van dijn hulpe groot,
        Veriaeght den angst van tegenstoot
        Der helscher geesten t'eenemale:
        Schoonste Omphale,
        Geen tonge dij vollouen // zal:
        Als ik der maagden lof verhale,
        En komen moet ten Principale,
        Zo triumpheert gij boven // al.
      
      
        En of ik stout
        Als Midas konde wensen goud,
        Al wat ik mij ter handen vouge:
        Ja dat elk Cederboom in het wout
        Peerlen, Robijnen, menichfout,
        T'mijnen beliefste altijd drouge:
        En dat ik slouge
        (Deur een tempeest) u van der // hand
        Waart niet al ijdel wat ik jonge,
        Naar dien ik mij altijd vernouge
        Met u, voor enig ander // pand?
      
      
        Princesse lief,
        Troostbaerste minder herten grief,
        Hier mede sluit ik mijn missijue:
        Hoe weerdig ik v ooit verhief,
        Getuigt den lasten minnebrief,
        Waar bij, ik noch gestadig blijue:
        Hoe veel ik schrijue,
        Ten is noch maar een naakte // denkt,
        Maar als ik kome met den lijue,
        Gelijk een man tot zijnen wijne,
        Zo werdet een volmaakte // vreugd.
