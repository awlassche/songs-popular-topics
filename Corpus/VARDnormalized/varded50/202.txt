
    
      
        VVt liefden ziet // Lijdt ik verdriet,
        Ik en macht ook niet, wtspreken,
        Hoe dat zij hiet // Die t'harte doorschiet
        Twelck Venus heeft ontsteken
        Nooit smert zo groot // In dezen noot
        Wie zoude den cloot geraken
        Rein roosken root // Troost mij tis noot
        Eer ik den dood moet smakken.
      
      
        Ik ben deurwont // Sijt dies oorcont
        In t'shartzen gront met lijden,
        Nochtans haar mond // Haar borstkens rond
        Kan mijn terstond verblijden,
        In t'swerelts kroon // Door haren toon,
        Spant zij den kroon van Vrouwen
        In dezen hoon // Haar dienaars loon
        Wie sou die schoon betrouwen.
      
      
        Liefs woorden zoet, Voor geld voor goed
        Want lief bij lief moet blijuen
        Noch vintmen goed // Dat troosten doet,
        Twelck liefde doet verdrijuen
        Maar des Kerssou // Daar ik op bon,
        Is bij gheen Vrou te gelijken
        Verdrijf ik den don // Ik sterue van rou,
        Schoon Liefken trou moet blijken.
      
      
        Laat hope mijn trooster zijn,
        Schoon Roosemareyn geprezen,
        Geeft Medecijn // Eer ik verdwijn,
        Zo is mijn pijn genezen
        Maar weigert zijt // Gheen meerder spijt,
        Mijn Hart dat split door deruen
        Vermaledijt // Zij ure oft tijd,
        Ik roep ik krijt om steruen
      
      
        Is het haar behaagt, Eer gij lof dracht
        Haar zij bevraecht met smeken,
        Ik hebt gewaagd // Tsij God geclaecht
        Dus hoort die Maagd eerst spreken,
        Loopt niet verblindt // Al tegen die wind
        Ook niet en mint met ogen,
        Maar wel versint // Eer dat gij mint
        Eer gij v vint bedrogen.
      
      
        Met oorlof rein, Ziet in het ghemeyn
        Ik blijff alleyn in smerte,
        Ik prijs dat grein // In swerelts plein,
        Zij blijft alleyn in het Harde
        Zij is eerbaar, haar oogskens klaar
        Al zonder maar gheschepen,
        Een ure van haar // Is duizend Jaar
        t'Block valt te zwaar te slepen.
