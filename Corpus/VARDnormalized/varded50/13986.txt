
    
      
        EEn roede groen // uit Jesse Stam gesproten,
        des Winters saysoen // wij het vruchtbaar hout genoten,
        Wiens vrucht is spijs // om de ziel te verkwikken,
        Heilig en wijs // met noch meer ander stikken,
        Die door zijn kracht // levendig overvloeien,
        In ons gebracht // als rancxkens op te groeien.
      
      
        Dit vruchtbaar hout // staat aan een schoon Reviere,
        het Brengt twaalf fout // vruchten van waerden diere:
        En op een Jaar // alle maanden zijn oogsten
        Voor al die daar // van te rapen begosten
        Zijn geresolveerd // voor alle vremde qualen
        Ongesommeert // wie kan zijn deugd betalen.
      
      
        Wie hier de smaak // van zijn vrucht heeft gekregen,
        het Geeft hem oorsaeck // om te meer zijn genegen:
        het Is in den mond // boven spijze der sterken,
        Het maakt gezond // in den Lust-hof te werken:
        Men wordt daar door // voorzichtig en kloekmoedig,
        En om wel voor // het kwade te zijn verhoedigh.
      
      
        Prins uwen lust // zij in Gods Wet gestadig
        Vierigh gerust // in Gode zeer genadig,
        Gebruikt dit hout // tot een staf in u handen,
        Het maakt u stout // te gaan door de waranden
        Van het Jammer-dal // daar men veel verdriet merkte,
        Het leit u al // van de een tot de ander sterkte.
