
    
      
        Waarom wroeten doch de mensen // waarom jagen zij
        Na de schijn vreugd van de wereld // die zo haast voor bij
        Vlieght gelijk de vluggen Arent // of een schip in zee
        Welckers spoor gheen mens kan vinden // noch van het schip een stee.
      
      
        Idel mens de vreugd der Aarden // het zij hoe schoon die schijnt,
        In een ogenblik daar henen // als een rook verdwijnt.
        Alle vreugd, het zij hoe bevallig // metter tijd verslijt.
        Gheen vermaak, dat niet ten lasten // ons in d'ooghen bijt.
      
      
        Als een boom is 'svleesch vermakken // die wel lief'lijck staat,
        Maar haar wortel tot beneden // in den afgrond gaat.
        Na het oog zij schone vruchten // als ook bloeysem draagt;
        Doch de na-smaeck is Gal-bitter // en gheen ziel behaagt.
      
      
        Deze schone Meereminne // geeft een korte vreugd:
        Geeft ook die haar vriend'lijck strelen // lange ongheneught;
        Ongheneucht in hert en zinnen // en in ons gewis;
        Want zij geeft een bitt're voorsmaak van verdoemenis.
      
      
        het Honigh-Byken geeft wel honig // doch met een ook steekt.
        Al te veel van het zoet genomen // onze ghesontheydt breekt.
        Matigheid in alle dingen // is gants prijselijk.
        Overdaad (van God verboden) is verdoemelijk.
      
      
        Schijn-vreucht kan gheen ware vreugden // of goe blijschap zijn.
        Niet kan ons te recht verblijden // welckers eynd' is pijn.
        Niemand ware vreugd geniet // welckers hert getuigt
        Dat hij sigh niet gants oprecht'lijck // na Gods wegen buigt.
      
      
        Overmits zijns herten inspraak, (is het niet afgebrant)
        Hem ver-oordeelt en kendt schuldig // aan Gods straffe hand.
        Kan een zondig mens verblijd zijn // die Gods straf verwacht?
        Die verblijd is in zijn zonden // is niet wel bedacht.
      
      
        God en Wereld gaan niet tezamen // wie zijn vreugde zoekt
        In de Wereld buiten Gode // werd van God vervloekt.
        Niemand kan twee Heren dienen // die tweedrachtig zijn;
        Of hij zal den een van herten // dienst doen, d'aer in schijn.
      
      
        Wilt gij vaste vreugd, o! mensen // en recht herts-vermaeck:
        Soecktse niet in het ijdel wezen // dat's te slechten sak.
        Daar in stelt uws herten blijschap // dat gij God bemint,
        En van God zijt aen-genomen // voor zijn lieve kind.
      
      
        Denkt geeft vreugd; doch zonde treuren // Denkt met God vereend.
        Zonden heeft ook zelfs in het lachghen // oorsaeck dat men weent.
        Denkt verseeckert van Gods vriendschap // en Gods vriendschap, dat
        Wij als vrienden van ons Heiland // komen in zijn stat.
      
      
        Denkt een vaste wijk der vromen // ieder mens verblijd
        Hier en naemaels, ja op aarden // in een zwaren strijdt:
        Deze kan gheen mens ontroven // het is een vast Cement,
        Die volhart, schoon al de Wereld // ons veel kwaad toesent
      
      
        Zij gegrond is in den Here // en zijn trouwigheid.
        Hij vertroost de vrome zielen // in haar zwarigheid.
        Daarom schoon de vromen treuren // somtijds na het vleysch;
        In haar ziel van God zij krijgen // vreugd en troost na eis.
      
      
        Staat dan mens na deuchden wilt gij // vrolijk zijn van Geest
        Weest voor deuchts gedaante nimmer // in u hert bevreesd.
        Is haar weg bestroyt met doornen // schrik niet, ga kloek heen,
        Want haar einde is zo kwaad niet // als het beginsel scheen.
      
      
        Beter is het hier wat te lijden // om het eeuwig heil;
        Dan om Aardse vreugd te dragen // lijf en zielen veil:
        het Komt niet kwalijk voor de tonge // het bitter eerst, dan het zoet:
        Die om Gods vreugd laat de zonden // heeft een wijs ghemoet.
