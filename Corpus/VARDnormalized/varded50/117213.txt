
    
      
        JAn Kooptmen een Kermis,
        Mooi Meisje ik heb ‘er geen geld,
        Ik sou u wel een Kermis kopen,
        Maar het geld is door mijn broek gedropen,
        Ian koopt men een Kermis,
        Ach Meysjen ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Mooi Meisje ik heb ‘er geen geld,
        Ik sou uw Cupidootje geven,
        Maar de krachten zijn verdreven,
        Ian kooptmen een Kermis,
        Ach Meisje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Mooi Meisje ik heb ‘er geen geld,
        Ik sou u wel een Roosje plukken,
        Maar het Bloempje mocht verongelukken,
        Ian kooptmen een Kermis,
        Ach Meisje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Mooi Meisje ik heb ‘er geen geld,
        Ik wou uw liever Carizeeren,
        En met uw Roosje Kwinkeleren,
        Ian kooptmen een Kermis,
        Ach Meisje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Mooi Meisje ik heb ‘er geen geld,
        Mijn beste Pandje is verzet,
        Ach Liefje op mijn woorden let,
        Ian kooptmen een Kermis,
        Ach Meisje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Mooi Meisje ik heb ‘er geen geld,
        De Waaffel Kramer wil niet borgen,
        De Tapper zeid komt overmorgen:
        Ian kooptmen een Kermis,
        Ach Meisje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Mooi Meisje ik heb ‘er geen geld,
        Ik sou met u wel eens vrolik zijn,
        Maar de Schyven leiden pijn,
        Ian kooptmen een Kermis,
        Ach Meisje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Mooi Meisje ik heb ‘er geen geld,
        Ik sou u wel een Fluytje kopen,
        Maar het geld is gisteren al verzopen:
        Ian kooptmen een Kermis,
        Ag Meisje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Mooi Meisje ik heb ‘er geen geld,
        Ik sou u wel een Popje schenken,
        Maar het mochte uw Leden krenken:
        Ian kooptmen een Kermis,
        Ach Meisje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Mooi Meisje ik heb ‘er geen geld,
        Want ik moet haast na Indiēn varen,
        Al door Nephtunis woeste Baren:
        Ian kooptmen een Kermis,
        Ach Meisje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Mooi Meisje ik heb ‘er geen geld,
        Ik moet een Zieltje op gaan zoeken,
        Ian Kalis zit in al mijn broeken:
        Ian kooptmen een Kermis,
        Ach Meisje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Mooi Meisje ik heb ‘er geen geld,
        Ian Credit en wil niet borgen,
        Mooi Meisje laat mijn niet meer zorgen:
        Ian kooptmen een Kermis,
        Ach Meisje ik heb ‘er geen geld.
