
    
      
        VVAck’re Herders vrolijk // luchtig ander heide
        Laat den grijzen molik // het Vee allenig weide,
        Want tis tijd // dat ghyder zijt
        Daar men S wijngaards lote snyt.
      
      
        Ziet hoe dicht de heuvelen // vol syn van Naejaden
        Laat u Nimf niet sneuvelen // t’mocht haar ergens schaden,
        Want het stof // te zwaar en grof
        Plukken zij t’ghereetste lof.
      
      
        Tyter laat u Schaepjes  // onder d’holle eiken
        Voegt u bij de knaepies // die vast opwaarts reiken,
        Na de kuin // de pople bruin
        Gelijk d’Hard’rinnetjes in het Duin.
      
      
        Vant gewas der velden // gaan ons schuren zwanger
        Dus dat wij uit stelden // onze Jaarfeest langer
        Waar een sant // komt hand aan hand
        Moet den Oost-May zijn gheplant.
      
      
        Coridon is wakker // ziet hem in de kuipen
        Parsschen op de akker // dat de beesje druipen,
        Van het nat // t’gheen Jaerlijckx wat
        Bachus ons geschonken had.
