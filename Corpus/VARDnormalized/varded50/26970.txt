
    
      
        Wiewoll ik arm vnnd ellendt bin /
        Zo hab ik doch einenn stättigenn Syn /
        Hoffn ng tuit mik ernerenn /
        Das mir vonn gott bescherett iß /
        kan mir kein kleffer nitt werhen.
      
      
        Der falschenn z ngenn seintt zo fill /
        Jch hof eß soll jnn helffenn nitt
        Gott is noch voller g ette /
        Dem ik mik gants unnd gar ergebenn hab /
        Der wirtt mik woll behuten.
      
      
        Mein hertz is het mir bek mmertt sehr
        gants  will aldinck z m bestenn kerenn /
        Jch far darhin mitt schmertzenn /
        ik sehenn das ichs nitt kan abelan /
        gott trost all bedrubte herzenn.
      
      
        Ferst  darher vnnd ließ mik
        was lest  mir zur letzenn hier /
        das ik meins leidtz vergesse /
        Stett lieb / trew vnnd stattigkeitt /
        Las ik dir z  der letzenn heir.
      
      
        Der  vnß dies newe leidtlein sanck /
        ein freier Schreibentt was ehr gnantt /
        ehr hats gar frei ges ngenn /
        ehr hatt ein feins mägdlein jm hertzenn lieb
        z  ihr kundt  ehr woll komen.
