
    
      
        O Venus jent // een zwaar torment
        Een groot ellent lyd' ik bij dezen,
        Gij staat gheprent // in het hert bekent,
        Zeer excelent // lieffelijck van wezen,
        Wilt gij mij niet // verlaten ziet
        Alle mijn verdriet // wat mij gheschiet
        Zal ik achten niet schoon lief geprezen.
      
      
        Mocht ik van haar // schoon aanschijn klaar
        Al zonder vaar // eens troost verwerven,
        Geen Goud zo zwaar // koos ik voor haar
        Een lieffelijck paar // zouden wij sterven,
        Alle mijn jolijt, in dezen tijd,
        Mijn hert verblijd // in het s'werelts krijt
        Het is meer dan tijd // zal ik troost erven.
      
      
        O Maechdelijck zaad // komt mij te baat
        Eert valt te laat, wilt druk verjagen,
        Zo zoet gelaat // zeer delicaat,
        Doet Venus raad // schout nijders kwaad,
        Geduld tot mij, die in lijden zij
        Zo waar ik blij // dat zeg ik vrij
        Niemand dan gij // en derf ik klagen
      
      
        Princesse rein // gij zijt alleyn,
        Lief dien ik mein // schoon bloem verheven
        Dat weertste grein // zijt gij certein,
        In het swerelts plein // wilt mij niet begeven
        Geeft mij confoort, een troostelijk woord
        Lief rechtevoort // weest niet gestoord
        Mijn beden aenhoort // en laat mij leven.
