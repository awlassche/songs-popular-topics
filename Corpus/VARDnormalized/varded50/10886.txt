        Al heeft zij rou // somtijds dat kan nz schaen
        Op eer en trou // min ik die fleur der wijven
        Als ik ze aenschou // zo moet ik zingen gaan
        Want zij mij non // bevrijd voor alle vaar
        Op dat zij mag gedenken mijn
        Zal ik mijn hertcken schenken // fijn
        Mijn lief eerbaar // tot haren nieuwen Iaer.
      
      
        Al was geresen // tot in den dood vol vrezen
        Mijn jonstich hart, ja in den gront versmoord
        Zij kant genezen // als zij doctor wil wezen
        Van alle smart, met een wellustig woord
        Dus wil ik voort, verlaten droefheid zwaar
        Wil zij slechts lieffelijck winken // neer
        En trou gewillig zinken // weer
        Mij voor een aat, tot mijnen nieuwen iaer.
