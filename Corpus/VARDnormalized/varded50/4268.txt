
    
      
        PRompt, en volmaakt //
        Is mijn wtvercoren //
        Zo wel geraakt //
        En was nooit Held geboren //
        Alvoren // kan
        Hij weg, en regel // schrijven //
        Op t'herte zal dan //
        Des jongen man //
        Voortan // als zegel // blijven.
      
      
        In een loyal
        Hert, heb ik besloten:
        Nu dat den straal
        Der minnen is geschoten,
        Genoten // zijn
        Moet met hem vreugd' // en // wenen,
        Ken achte gepijn,
        Ik zal t'sanguijn
        Aanschijn // in deugden // nemen.
      
      
        Eenpaarlijk staat
        T'concept mijner zinnen,
        Dat ik (naar raad)
        Hem niet en zal beginnen
        Te minnen // small,
        Maar als mijn here // kronen:
        In spijte t'getal
        Der tongen al
        Ik zal // hem ere // tonen.
      
      
        Troostbaerder // geen
        Inder Goden troone,
        Al waarder // een
        Adonis in persoone,
        Zijn schone // glans
        Moet moet noch menig // buigen:
        Want, hij dracht den krans
        Der jonger mans,
        Dits gans // mijn enig // tuigen.
      
      
        Een Paragoen
        Is hij, vroom gezegend:
        Van dien Baroen,
        Mij alle troost beiegent,
        Genegent- // heyt
        Theimwaerts, kan genen // smerte,
        Maar laat met bescheid,
        V zijn gezeid,
        Hij leit, mij neuen // t'herte.
      
      
        Rustig, en fray,
        Moet mijn Prinse varen:
        Den Rijnschen bay
        Wens ik hem zonder sparen:
        Geen zwaren // moet
        Wil ik ten fijne // maken:
        Dit gracelick bloed,
        Mij altijd doet,
        Meer zoet // dan pijne // smaken.
