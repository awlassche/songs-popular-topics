
    
      
        WIe zal ons geraden // vroed,
        Welken vogel snel
        Des Landen veel weldaden // doet?
        Hij en heeft een veder // niet,
        Doch al heen en weder // ziet
        Men hem vliegen wel.
      
      
        En is dit gheen wonder // werk?
        Mochtmen zeggen dan,
        Dat zonder pluim, en zonder // vlerk,
        Enig vogel wezen // mag,
        Ter wereld op dezen // dag,
        Die ghevlieghen kan.
      
      
        Hij heeft vlees noch benen // werd,
        Is te raden dit,
        En dan en heeft hij genen // steert,
        Maar hij heeft twee bekken // stijf,
        Lank is hij van nek en // lijf,
        'tInghewant is wit.
      
      
        Somtijds graan, oft ander // verf,
        Naar het wijlen past,
        Men kander huis, men kander // erf,
        Denkt en ere winnen // met,
        Die daar aan zijn zinnen // zet,
        Hem te jagen vast.
      
      
        Door een smal bosschage // dicht,
        Die naar hem toenijpt,
        Daar neemt hij zijn passage // licht,
        Aan elk endt om vangen // gaan,
        Vijf altijdts haar gangen // zaan,
        En elk naar hem grijpt.
      
      
        'tSchijnt een kromme droge // schuit,
        Men wind van den buik
        Den darrem hem door d'ooghe // uit,
        En dat van den darrem // coemt,
        Die men rijk oft arrem // noemt.
        Dienet tot ghebruyck.
      
      
        het is weerdig verkoren // wis,
        Tot des mensen nood,
        Van als men kind geboren // is,
        En mogen wy't derven // niet,
        Iae tot dat wij sterven // ziet,
        En noch na den dood.
      
      
        'tLeert ons ook den rechten // pad,
        Tot dat eeuwig Land:
        Dus en is het genen slechten // schat,
        Die hier is te krijgen // bij,
        Van dit Raadsel zwijgen // wij
        Niet langer het verstand.
      
      
        'tVlasbloemken ter eren // zoet,
        De Schietspoele rein,
        Die meen ik, des ons leren // moet,
        Hoe voorbij ons dagen // smal
        Gaan, zonder vertraghen // al,
        Haastig hier in het plein.
      
      
        Den lijdsamen vromen // Iob
        Van haar snelheid leert,
        Hiskia zonder schromen // op
        Den Wever gedachtig // werd,
        Doe hij leed onzachtig // smert,
        Na de wand gekeerd.
      
      
        Ons dagen vervliegen // doch
        Als de Spoele hier,
        Maar vele haar bedriegen // noch,
        Met te menen leven // lank,
        Die haast dodig sneven // krank,
        In het zondig bestier.
      
      
        Tot eeuwigen mooien // loon,
        Doe hier elk een vlijt,
        Den enen zal in het lopen // Kroon,
        De ander 'tSweerdt ghenaecken // maar,
        Hierom elk met waken // waar
        Neme zijnen tijd.
      
      
        Pallas-gheesten konstigh // laat
        V niet vallen zwaar,
        Als iemand u afjonstigh // smaad,
        Als d'Eyptsche knapen // koen,
        Die voor t'vromer schapen // hoen
        Grouwelden eenpaar.
      
      
        Oorlof, u te samen // draagt,
        Vroom en wel gemoed,
        Den Volghel zonder schamen // jaagt
        Door het woud, met lust en // hand,
        Laat hem weinig rusten // want
        Lust den arbeid voedt.
