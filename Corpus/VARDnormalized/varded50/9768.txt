
    
      
        MOyses Gods Knecht geprezen,
        Leide leven en Dood
        Voren gehoorzaam wezen
        Den Heer, is'tleven klaar // maar
        De dood lont den zondaar // zwaar,
        Dit onderscheid is groot.
      
      
        Twee wegen staan te kiezen,
        Elk moet voor hem zien, want
        Zijn ziele te verliezen
        Namaals in het eeuwig vier // schier,
        Wij behoren te dier // hier
        Toe achten dezen pand.
      
      
        Twee poorten, en twee wegen,
        Wie den wijden onrein
        In duisterheid wil plegen
        Te gaan, komen ten val // zal
        Den anderen, die al // smal
        Is, leit in 'sHemels plein.
      
      
        Den Geest na d'enghe poorte
        Ons wijst, en te gaan in het
        Bad der wedergeboorte,
        Want vruchtbaar rechte boet // goed,
        Voor al gedaan zijn zoet // moet,
        Ootmoedig als een kind.
      
      
        Die hier niet gaan in erven
        Gods Rijk als kinders kleen,
        Die sullent namaals derven,
        Gods kinders onderdaan // zaan,
        Met kruis de smalle baan // gaan
        Ten leven in alleen.
      
      
        Christus is tot den Vader,
        De deur, om ingaan vrij,
        En waar hij gheen ontlader,
        In al ons ongemak // zwak,
        Gevallen onder 'tpack // vlak,
        Te gronde laghen wij.
      
      
        Weinig zoeken en vinden,
        Den smallen weg oprecht,
        Door het vleeschelijck verblinden
        En der lusten opstel // fel,
        Wordt menig mense wel // snel
        Der sond' en 'swerelts knecht.
      
      
        Ter wijder poorten vele
        Lopen al over hoop,
        De Hell' ontdoet haar kele,
        Heer en gemenen Man // dan,
        Gaan in, 'tschijnt niemand kan // van
        Daar keren zijnen loop.
      
      
        Maar wilt doch niet aflaten,
        Gods Kinders al gelijk,
        Den Geest komt u te vaten,
        Wat lijden u gheschiet // ziet,
        God laat u in het verdriet // niet,
        Hij helpt u in zijn Rijk.
      
      
        Laat ons in tijden // mijden
        Den weg der duisternis,
        En liever strijden // lijden,
        Op 'slevens engen pad // wat,
        Te komen in de stad // dat
        Enig ons nodig is.
