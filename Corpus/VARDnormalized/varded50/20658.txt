
    
      
        KOmt hier; bij mij vergaard;
        Komt, die ik heb gebaard
        In den Heer // wonder teer // door mijn lering,
        Die Gh' in u hart bewaard:
        Vervordert hier uwe gang,
        Aenhoort des zoeten zang,
        Neemt die aan // van De Zwaan // wiens regering
        U zoet viel jaren lang.
      
      
        Gij zijt die ik bemin
        Naast God meest van het begin,
        Tot het end // zo bekent // is u alle,
        Gij zijt mijn hart, mijn zin.
        Ik draegh U op dit werk,
        U, Le'en van Godes Kerk,
        Die mij sint // die mij mint // zonder galle,
        G'lijck dag'lijcx ik aenmerck.
      
      
        Met Venus vuil fenijn
        Vervult veel Boecxkens zijn,
        Die aan kant // uit de hand // gij moet leggen,
        Of gij smaakt de helsche pijn.
        Neemt aan dit Boek van mij,
        het Welk is van vuilheid vrij;
        het Minste Lied // dat gij ziet // zal u zeggen
        Dat ik de vuilheid vlij.
      
      
        Niet als van eerbaarheid,
        Niet als van suyverheyd,
        Handelt hier // dit papier // (het is waeractich)
        En des levens heiligheid.
        Tot deughd' ik u vermaan,
        Tot God doe ik u gaan;
        Te allen tijd // 'k bidd' U, zijt // toch gedachtig
        Des liefelijcke Zwaan.
      
      
        Vaart wel, de deugd bemindt,
        Aan God u vast verbindt;
        Vliet het kwaad, en verlaat // vuyle Liedtjes,
        Tot suyv're zijt gezind.
        Begint den Hemel hier,
        Zingt Liedtjes met playsier;
        Ei! zijt all' // op dit dal // zoet als Bietjes;
        Volgt des hemelrijcx manier.
