
    
      
        ACh ! wat een smerte // braeckt het minnen uit,
        Wanner een herte // zijnen brand besluit,
        Zonder te klagen // daar het niet kan verdragen
        Het vinnig prangen // van het groot verlangen ?
      
      
        Het stoute minnen // zegt mij stadig aan,
        Dat ik mijn zinnen // soud' verklaren gaan,
        Maar bleeuwe schaamte // en vrees' van de blaamte
        Doet mij vertraghen // mijn Lief naar te jagen.
      
      
        En noch doet hij mij // d'alder-meeste pijn,
        Om dat hij bij mij // mijn Zuster komt vry'n:
        Ach ! hij is stadig // fel en zeer ongenadig
        Ontrent mijn deure // het welk mij meer doet treuren.
      
      
        Dus ben ik even // Tantalus die haeckt
        Naar het vocht dat neven // aan zijn lippen raakt,
        En nooit kan proeven // dan een staegh bedroeven,
        Om dat zijn lippen // nooit aan het water stippen.
      
      
        Wat heeft hij aardig // doch in haar besint,
        Dat zij is waardig // boven mij bemindt ?
        Nooit wirdt haar wezen // boven het mijn geprezen,
        Noch haar zoet spreken // tegen het mijn gheleken.
      
      
        Ik ben niet spijtig // zo hij van haar klaeght,
        Maar ziet ik vlijtig // kom' hem naer-ghejaeght;
        Wilt hij mijn bange // ziel' maar eens ontvangen
        In zijn genade // eer het wordt te spade.
      
      
        Ik zal verzaken // mijn Zuster ter nood,
        Mits ik niet haken // mag haar blijde dood:
        Maar wil eer sterven // en ook metter spoed
        Mijn borst rood verven // met mijn heete bloed.
