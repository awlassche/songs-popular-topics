
    
      
        DE swoele Lent die het al ontfonckt
        Wt haar Moeders schoot,
        'tRood' Angier staat op-gepronkt
        Met het Roosjen Root.
        d'Lely Wit,, ontspruyt snel //
        Aenschout dit // Pijn-Appel,, komt uit
        En des Groene Laurier-spruit;
        Ziet den Coorenaer,, staat nu weer vruchtbaar,
        Door den Geest begloost // en des Opper-Oogen-troost.
      
      
        Wij danken den Heer die het al geeft,
        En ook voor ons strijd;
        Zo dat Orangjen nu verd' streeft,
        Tot des Spanjaerts spijt;
        Vromen Held,, zijt gegroet //
        Die daar stelt // lijf en bloed // te pand,
        Voor ons lieve Vaderland:
        V manhaftig toont // gij bent waard gekroond
        Van u jonkheid of // met een Laurier vol Lof.
      
      
        Die Staten die ons Land ghebien,
        Groeten wij nu mee:
        Den Tromp die hem niet gaan ontzien,
        Den Vijand op Zee,
        Altijd tracht // om te slaan //
        Dag en nacht // den Maraan // tot niet,
        Zo onder Duins is gheschiet,
        Daar hij hem manlijk queet // zo ieder wel weet,
        Met zijn macht zeer groot // en verwon de Spaensche vloot.
      
      
        Den Edelen Heer van Warmont
        Groeten mede wij,
        En wensen dat hij lang ghesont
        Bedien des Oud' Abdij:
        Den Bailjou,, zeer bekwaam //
        Groeten non // wij eerzaam // voorwaar,
        Wel-geboren allegaar,
        Al die het Dorp bedient // ik wens, God verlient
        V al het zam wijsheid // dat gij goed en kwaad verscheid.
      
      
        Gegroet o gij Parnassche Maagd
        Van Konst-Redenrijck,
        Die voor Zoylus niet is vertsaagd,
        Zoo het nu klaar geeft blijk,
        Onze lust // dat wou staech //
        het Had gheen rust // alle dag // noch macht
        Om te zien u Woord vol Kracht:
        Nu komen wij aan // met Trommel en Vaan;
        Voor de Maagd een jeugd // van wegen des Oestgeest-vreught.
