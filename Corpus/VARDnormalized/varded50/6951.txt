
    
      
        AStrea zoete Meid
        Die uwe schaepjes weit
        Door t'over-breede Klaver-gras,
        En daar de beeckjes klaar als glas
        Door vloeien // besproeien // het vruchtbaar land
        Van d'Amstels groene kant,
        Hoe jeugdig // hoe vreugdig
        Wandelen wij nu tsamen hand aan hand.
      
      
        k'Weet gheen Goddin die leeft
        Die zo veel schoonheyts heeft
        Als mijne Zon Astrea doet:
        O! Harderinne van mijn ghemoet
        Het grafje // vervarsje // door t'luchtich gaan,
        De zoete Klaver-blaen.
        Begroeten // u voeten
        En gaan van vreugde recht-op staan.
      
      
        Ziet hoe het Pluym-ghediert
        Rondom ons henen swiert,
        En van ons twen een Lietgen zingt:
        Ay: ziet hoe t'weelighe gaytgen springt,
        Het schaepje // zijn knaepje // zijt wel ghemoet
        Zeid: Paren zijt gegroet,
        Mijn Harder // waar varder?
        Niet als een teiken gij mijn daar van doet.
      
      
        Op dit heuveltge neer
        In t'aenghename weer,
        Astrea lief, u leden rust,
        En laat u treurig oog na lust
        Aan d'heyden // gaan weiden // door t'soete Vee
        Van Trijn Wouters dree,
        Gaat strelen // en delen
        Malkander haar zoete kusjens mee.
      
      
        De Zon zijn aessem scheid,
        Gheen blaetgen hem verreyt,
        Het vliegend wild is moe en loom,
        En t'Aemsteltge leit onder stroom,
        t'Wort duister // stil luister // daar is gewag:
        Dus Sela-dongh ghenach
        Zeid Astree // en hier mee
        Liet zij mij daar alleen vol droef gheklach.
