
    
      
        T'Valt zwaar // zonder stadigen hoep
        Van troost te winnen // iemand te minnen
        Voorwaar, in dezen baan ik loop,
        T'vlas moet ik spinnen, t'cost mij mijn zinnen
        Want die mijn hart begeert, Met liefd' bevangen
        Mijn bidden en smeken werd
        Hoer oogskens van mij keert,
        En zoekt t'verlanghen.
      
      
        Eertijdt // scheen mij Cupidos gonst
        Na mijn behagen // zelf na te iaghen,
        Verblijd // doch ik door slechte const,
        En raad van magen // haar liefd' te dragen.
        Maar t'radt is zeer verdraeyt, Nu blijft die ogen,
        Doort schijn ben ik verleid
        Die liefste van mij scheid
        Ik ben bedrogen.
      
      
        Mijn hart // zal ik van haar oprecht
        Al eer niet wenden // t'leven zal enden,
        Met smart, mag mij haar trouwe knecht
        En veel ellenden, Venus toezenden,
        Ik blijf mijn lief getrou, t'hart zal niet sneven
        Al treur ik nu in rou //
        ik zal des werde vrou
        Noch niet begeven.
      
      
        Zij is // mijn vreugd, Mijn troost, mijn hoop
        Naast God mijn Here, die ik begeere,
        Gewis // om des Prinses ik loop,
        Ende t'bloet vertere, t'neemt eens een kere
        Zo sou ik mij t'geluck // geheel toeschrijven
        Verwerpen t'sware juk,
        Bevrijd van pijn en druk,
        Ghenuecht bedrijven.
