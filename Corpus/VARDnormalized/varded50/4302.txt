
    
      
        Minzaam zoete nature //
        Die mij contrariert,
        Hoe zeer dijn willecuere //
        Ootmoedig werd begeert,
        Toogt heden // maar //
        Een reden // klaar //
        Waarom gij rebelleert?
      
      
        Een wtgelesen // vrouwe
        Scat ik naar werde // dij,
        Om wien ik dezen // rouwe,
        Lijdzaam aenveerde // vrij:
        Geen lieuer ieught,
        Die mij verheugt,
        Op geheel de aarde // zij:
      
      
        Zal mij maken mismoedig,
        Der wonden een, of twee,
        O neen! ik strijde bloedig
        Tot in de laatste wee:
        Al briesschen som
        Vijandig om
        Mij, als een wilde zee.
      
      
        Verhard vileyne // tongen
        Verbergt u wreed aanschijn,
        Vipers onreine // jongen,
        Serpentich boos fenijn:
        V paden // gaat,
        Geen kwaden // raad,
        Zal hier van werde zijn.
      
      
        Edel, ia magnifijcke
        Beelde, pointich gewracht:
        Theseus ik mij gelijke,
        Doch Ariadne slacht:
        Zo werd ik uit
        Den Dool-hoff ruit,
        Bij uwer hand gebracht.
      
      
        Roup ik aldus om bate,
        Princesse, altijd meer:
        Sijt mij een honighrate,
        In al mijn cranckheyt teer,
        Daar voren zal,
        Ik over al,
        V schoonheid loven zeer.
