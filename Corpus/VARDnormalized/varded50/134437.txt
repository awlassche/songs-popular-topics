
    
      
        d'VYtmutend Herderin,
        In schoonte, prachtige Hemels Engelin,
        Eerwaarde Vrou // 'tpronckje van Trou,
        Die ik door rechte liefde aenschou,
        Wiens bloosje // en roosjen // uit spruit,
        In het Zuyt // verr' boven het Kruid,
        'tCierlijckst' sieraad // Onachtzaam staat,
        Ha! Nimphje vol jeugdig gelaat.
      
      
        k'Ben met u Zoeterd vrij,
        Te velde, voegt u doch aan mijn groene zij,
        Ik heb geen rust // voor d'liefd' u kust,
        Dat het jonk herte, natuurlijk lust,
        Hier enig // allenig // te zijn,
        Aanschijn // geneest mijn pijn,
        Cupido ziet // zijn schichten ziet,
        En stoockt een heten brand, om niet.
      
      
        Waar zijn u Schaepjes veel,
        Godinne, noch in de Stal, mijn zoet gestreel,
        Mijn dert'le vreugd, die het hertje verheugt,
        Werelds cieraetje, in het best van u jeugd,
        Min viertjen // pleysiertjen // ter wei,
        het Is Mei // ei sinckter een Rei,
        Drift het Vee en boet // al u gemoet,
        Gedenkt bij u dat liefd' veel doet.
      
      
        In het groen begraesde gras,
        Gewentelt, Ik ben noch die ik eertijds was,
        In het kort gezeid // u dienst bereid,
        Nevens mijn Zijtje soetst' enigheid,
        Ei queestert // geneester // mijn smert,
        In het hert // lief gans verwert,
        Door d'heete Min // verkeert de zin,
        Doch k'huysvest wel een veld-godin.
      
      
        Prinses mijn Granida,
        Gans geestig, denkt Vrou een leeft op u gena,
        Van lijf, en leen // 'talbastert steen,
        Ja het witst' yvoor, verdooft in het gemeen,
        V glansje // 'tgroen kransje // (o! Goon,)
        Ten toon // liefd' aengeboon,
        Nimf Amariel // Ei staat wat stil,
        k'Voel door mijn leven een wond're gril.
