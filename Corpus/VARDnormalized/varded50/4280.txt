
    
      
        CEderboom jeugdig groen // plantsoen //
        Deurluchtich paragoen // u doen //
        Is zo betrout //
        Waar ik ter dood benout //
        Gij zout // zijn mijn rantsoen.
      
      
        Abel, en magnifijck // gelijk
        Pallas, die gans aardrijk // publijck
        Sticht in het gemeen,
        Zijt gij beelde alleen,
        Want geen // zo autentijck.
      
      
        Troostbaar in tegenspoed // onzoet
        Sydy, als t'quaet gebroed // mij doet
        Zuchten vervaard:
        Bij dien vreugd openbaart
        T'beswaert // treurig gemoet.
      
      
        Hierom ben ik perfect // gewect,
        Een dienaar t'sijn subiect // gestrekt,
        Voor u: zo mag
        Einden (naar lang verdrag)
        T'geclach // secreet bedekt.
      
      
        Als gij (Liefste) dit wrocht // mij docht
        Gij waart zo wtgesocht // ten mocht
        Schoonder Robijn
        Onder der zonnen schijn,
        Niet zijn // zo dier gekocht.
      
      
        Rond om d'aerdtsche tonneel // geheel
        En is liever Juweel // tot deel
        Gewenst voorwaar,
        Dan u, die zijt een zwaar
        Dierbaar // gulden parceel.
      
      
        In mijn hert nooit gegrond // en stond
        Zulk lief, wiens roden mond // mij iont
        Troostig geniet,
        In alle mijn verdriet,
        Aanziet // gij mij gewont.
      
      
        Nu gij mij diligent // bekent,
        In dienst, geen tijd absent // gewend,
        Schinct mij (voor loon)
        Wt uwen hogen troon
        Een schoon // coronnement.
      
      
        Aangenaam edel kind // gezind,
        Naar dien gij u dus vint // bemint,
        Jont uwen knecht
        Bijstand, dat hij met recht,
        T'gevecht // eens overwint.
