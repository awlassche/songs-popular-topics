
    
      
        BRoeders eersame // als gij present
        In des Heren name // vergadert bent,
        En daar anhoren, zijns woords vermaan
        Zo komt weer voren, het scheiden gaan
        Ziet dat gij het woord // dan niet versmoord,
        Maar zo het behoort // vierig zijt onderdaan.
      
      
        Want die het woord horen // en zijn voor God
        Niet uitverkoren // maar die het gebod
        Hoort en bewaren // is zaligheid,
        Na Schrifts verklaren // dan toegezeid:
        Dus ieder een // vrienden gemeen,
        Zijn herte reen // daar toe make bereid.
      
      
        Want in een stenen // herte of land,
        Wil Gods woord reene // niet zijn geplant;
        Het woord des Heren // als ploeg bekwaam,
        Moet het eerst omkeren // dat aangenaam
        Het stenig hert // vermurwet werd,
        En het woord anvert // vruchtbarig na het betaam.
      
      
        Die het woord ang'name // ontvangt en plant
        In zulk bekwame // een hert of land,
        Dien zal het dragen // in overvloed
        Naar Gods behagen // des Geests vruchten zoet,
        En naar des tien // zal dien gezien
        Eeuwig verblien // aan hert, zin, en gemoet.
      
      
        Broeders in het scheiden // het woord vol vree
        In soo'n bereiden // hert, draagt doch mee,
        het Zal u vrucht bringen // die te boven gaat
        All' aardse dingen // hoe delicaat,
        Ia niet daar bij // te gelijken zij;
        Broeders tot dij // zo ook ons hope staat.
      
      
        Anders waar het horen // des woords vermaan
        Niet dan verloren // kosten gedaan,
        Ia om bespotten // van ieder een,
        En het doen eens sotte // was men gemeen,
        Die los op het sant // gants onbestand
        Timmerde, van // zijn werk dat viel daar heen.
      
      
        Van diergelijcken // Ezechiël
        Ons ook doet blijken, hoe Israël
        Hem daar te voren // des Heren woord
        Zouden anhooren // maar nochtan voort
        Doen na haar raad // en opzet kwaad;
        Maar op des daad // was de Here getroost.
      
      
        Schoon wij zo spreken // O broeders dij,
        Hier bij geleken // gants niet en zij,
        te Uwaerts betrouwen // al beter wij,
        Dan te een waerschouwen // is dus van mij
        Gedaan alleen // of wij naar het scheen
        Eeuwig bij een // mochten vergaren blij.
      
      
        Nu ik hier mede // bevel u God,
        Dan noch mijn bede // tot u in te slot,
        Is dat vermeere // u liefd' en deugd,
        Dat te allen kere, gij zuiver meugt
        Wezen ter noot // zonder anstoot,
        Voor klein en groot, zo suldy zijn verheugt.
