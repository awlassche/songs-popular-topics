
    
      
        VVT liefden ziet, Lyd' ik verdriet
        Ik en macht ook niet uitspreken
        Hoe dat zij hiet diet herte doorschiet
        Twelck Venus heeft ontsteken,
        Nooit smert zo groot // in dezen noot,
        Wie sou den cloot, geraken
        Rein roosken root, troost mij het is noot
        Eer ik den dood, moet smakken,
      
      
        Ik ben doorwont, zijt dies oorcont
        In s'herten gront met lijden,
        Nochtans haar mond, haar borsten rond
        Kan mijn terstond verblijden,
        In het s'werelts troon, door haren toon
        Spant zij die kroon van vrouwen
        In dezen hoon // haar dienaars loon
        Wie sou die schoon betrouwen.
      
      
        Liefs woorden zoet, voor geld voor goed
        Want lief bij lief moet blijven
        Noch vintmen goed, dat troosten doet,
        hetwelk liefde doet verdrijven,
        Maar des Kerssou // daar ik op bon
        Is bij gheen vrou te gelijken
        Verdrijf ik den don // ik sterve van rou
        Schoon liefken trou moet blijken.
      
      
        Laat hope mijn, trooster zijn
        Schoon Rosmareyn geprezen,
        Geeft medecijn // eer ik verdwijn
        Zoo is mijn pijn genezen,
        Maar weigert zijt, geen meerder spijt
        Mijn hert dat split doort derven,
        Vermaledijt // zij ure of tijd
        Ik roep ik krijt om sterven.
      
      
        Is het haar behaagt, Eer gij lof dracht,
        Haar zij gevraagd // met smeken
        Ik hebt gewaagd, hetzij God gheclaecht
        Dus hoort de maagd // eerst spreken,
        Loopt niet verblindt // al tegen de wind,
        Ook niet en mint, met Ogen
        Maar wel versint // eer dat gij mint
        Eer gij u vint, bedrogen.
      
      
        Met oorlof rein, ziet in het gemeyn
        Ik blijve alleyn, in smerte
        Ik prijs dat grein // is s'werelts plein
        Zij blijft alleyn // in het herte
        Zij is eerbaar // haar oogskens klaar
        Al zonder maar // geschepen
        Een ure van haar // is duizend jaar
        het Blok valt zwaar // te slepen.
