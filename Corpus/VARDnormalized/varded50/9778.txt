
    
      
        NEemt waar, o Heer, ws Kerken // bede
        Laat Sathan haar niet ziften // kwaad,
        Die op u Woord en Schriften // staat
        Door uwen Geest wilt werken // mede
        Tot sterken // vrede.
      
      
        Hoewel ik met de nydelijcke
        Slang oud, en wil paeys maken // niet,
        Zij toont, om mij t'ontschaken // ziet,
        Een vluchtig mooi arts tydelijcke
        Doch ijdel // rijke.
      
      
        En laat mij niet om vallen // glijden
        Noch breden pad verkiesen // daar
        Ik laas u sou verliezen // maar
        Tis beter op den smallen // lijden,
        Hier te allen // tijden.
      
      
        Acht Babel niet, o gij // getrouwen
        Haar zij u Koninginne // roemt,
        En mij u vyandinne // noemt,
        Maar nemmermeer zal mij // berouwen
        Op dij / te bouwen.
      
      
        Want ziet om niet bedrogen // varen
        Mijn kracht zijn het Euangely // zal,
        Gemeen gij niet met Belial
        En zijt, bij u gheen loghenaren
        En mogen // paren.
      
      
        O Heer, al ben ik nu verdreven
        Gekweld oft moet nu zeer zijn // spot,
        Ik hop', ik werd', o Heer // mijn God
        Eens in het volkomen zuiver leven
        Van u // verheven.
      
      
        'tQuaet eerst zult gij den kwaden // lonen
        Die kwalijk doen oft spreken // hier,
        V gramschap werd ontsteken // vier,
        Maar wilt mij met ghenaden // kronen
        Weldaden // tonen.
      
      
        Och laat doch bij t'waerachtich // schrijven
        Vwer Euangelisten // mij,
        Volherdich al van twisten // vrij,
        O Heer, door uws Geests krachtig // drijven,
        Eendrachtig // blijven.
