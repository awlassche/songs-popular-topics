
    
      
        O Ionckheydt groene, Bloeyende Ieught,
        In dit saysoene, Schickt u tot deugd,
        Wilt u begeven, tot zedigheid,
        Alle u leven // daar toe bereid,
        Met zout doch menght Wat gij voortbrenght,
        Spreekt ofte zingt // alzo ons Paulus zeid.
      
      
        Laat u doch raden, o Jongheid fier
        Teder van bladen zoet van manier,
        Wilt u doch keren, van des werelds vreugd,
        Vreest doch den Here // terwijl gij meugt,
        Want ziet die dood, spart klein noch groot,
        Hoe jong, hoe oud, het wordt al tot hem geveught.
      
      
        En wilt niet treden // om wat nieus te zien,
        Als Dyna deden // ten ouden tien
        Als zij aenschoude, des Landes Heer,
        Kwam zij in rouwe, en zwaar verzeer,
        Haar schoonheid jent // heeft hij geschend,
        V hier van went // gij jonge dochters teer.
      
      
        Maar wilt aanschouwen, Het vroom ghemoet
        Van de getrouwe // dochter Iephta zoet,
        Die ging versmaden // al 'swereldts eer,
        En liet verbraden, haar lichaam teer
        Op den Outaer // van haren Vaar
        Spieghelt u klaar // en houdt dit voor een leer.
      
      
        Dit differenti, wilt doch aanzien,
        En maakt doch menti // al van des twieÃ«n
        Hoe de een bedorven // ende beschaemt,
        De ander gestorven // heel gehoorzaamt,
        Dus wilt doch fijn // de ouders dijn
        Gehoorzaam zijn // zedig en bekwaam.
      
      
        Op dat gij princelijcke Dochters fijn,
        Mocht mede van de wijze Maagden zijn,
        Die metten Here, werd ingheleyt,
        Want tot de oneere // zal worden gezeid,
        Dit droevig Lied // ik ken u niet,
        Gaat in het verdriet // day voor u is bereid.
      
      
        Daar zal geschieden // zo groot dangier,
        Na Schrifts bedieden // in het Helsche vier,
        Wringen der handen // druk ende pijn,
        Knerssen der tanden // t'eeuwighen tijn,
        Hierom o jeugd // schickt u tot deugd,
        Op dat gij meugt // hier van bevrijdet zijn,
      
      
        En dan meugt horen, het zoet geluid,
        Mijn uytvercooren // mijn lieve Bruid,
        Wilt tot mij treden, ontvangt de kroon,
        En gaat in vreden // in des Hemels troon,
        Blinken hoort mijn // als der Zonnen schijn,
        Dat zal dan zijn // der uitverkoren loon.
      
      
        Dus wilt versieren, u Lampen subijt,
        Met goe manieren // waar dat gij zijt,
        Laat u licht rijzen, wilt liefde schoon
        Altijd bewijzen, zo zal Gods Soon,
        V blijde ontfaen // en voor u staan,
        En dienen zaan // al in zijn Vaders Troon.
      
      
        Oorlof hier mede, o Ieught eerbaar,
        Gods eeuwige vrede wens ik u klaar,
        Dat bij ons blijve // tot het ende goed
        En ons verstijve // in een vroom ghemoet,
        En na dezen tijd, te zijn verblijd,
        Wens ik met vlijt // de Ieught tot enen groet.
