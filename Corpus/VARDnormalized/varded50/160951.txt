
    
      
        OM dat daar gheen mense zalig // wordt
        Deur eigen werk tallen stonden,
        O Heer op ons liberalich // stort,
        U genade niet om deur gronden
        Wie zeid hij zonder zonden // zij
        Leugen die gaat vermonden // hij
        Daar en is niemand gevonden.
      
      
        Ia noch hoeft men groot en kleine // die,
        Prijs die God hoorde tontfangen
        Zo Paulus zeid in Romeyne // drie
        Zo lagen wij gevangen,
        In zonden na Paulus bediet // reen
        Niemand gerechtig ja ook niet // een
        Al verslonden in doods verstrangen.
      
      
        Neemt ons aan Heer den genade // groot
        stelt al onze schulden bezijden
        die ons hebt van de kwade // dood
        Verlost deur u dood en lijden
        Deur uwe liefde grondeloos,
        Genaest gij al mijn zonden boos,
        Dat wij met u zouden verblijden.
      
      
        Kennelijk is het dat ons allegaar
        door Adams overtreden
        Zijn wij gekomen ten valle // zwaar
        Maar deur u Here in vreden
        In de vrede ons bewaren // wilt
        Op dat ons geen bezwaren // schild
        Daarom Here heb ik gebeden.
      
      
        Lam en kreupel, O Here // mijn
        Laat mij niet gaan hier beneven
        Dat ik mag zijn // deur u lere // fijn
        Ghesont in het gelove verheven,
        Op dat ik hier in mijn Aarden // Vat
        Altijd beware dien waerden // schat
        Die gij mij o Heer hebt gegeven
      
      
        Berecht Mij Here na u goedheid
        bewaard mijn herte en zinnen
        dat strijd // met vlijt // onder den voetscheyt
        Gij kond o Here druk verdinnen
        Sterk mij met u genadig woord
        En help mij altijd gestadig // voort
        Dat ik komme ter stede binnen
      
      
        Van Deze stad die vol goeden // leit,
        Vinden wij klaarlijk geschreven
        Van Esdras die zijn bemoeden // zeid
        En van Gods gest gedreven
        Al die hier overwinnen // zaan
        Die zullen daar mogen binnen // gaan
        Om bij u Heer eeuwig te leven
      
      
        Den Inganc tot deze reine // stad
        Is vol prikkel en dangiere
        een mensen trede is de kleine // pad
        Bezet met water en met vierre
        Hoe zal ik o Heer noch komen // daar
        Door dezen weg, vol verschromen // zwaar
        Helpt mij o Heer goedertijere
      
      
        Dalen en lechten ghedincken /// wilt
        Dat sijt ook mogen beseffen
        U goedheid wilt haar zinken // milt
        Dat zij u waarheid treffen,
        Ootmoedig met een verslagen // geest
        Al die u na u behagen // vreest
        Die zult gij hier naar verheffen
      
      
        Doch-ter wijlen o Vader // rijk
        wij donder u niet en vermogen
        Wij zijn doch allegader // slijk
        En stof die nergens toe en dogen
        O Prinselijk God van machten // groot
        Voor u zijn mijn gedachten // bloot
        Ziet ons aan met u genadige ogen.
