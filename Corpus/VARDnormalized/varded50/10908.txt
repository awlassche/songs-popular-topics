
    
      
        O Mei playsant // coutheyts vijand
        die weelde plant // zeer triumphant,
        Aan elken kant // u komen loven,
        want u aessem zacht // zeer Lieflijk lacht,
        Door u dons kracht // werd voortgebracht
        Dat lach versmacht // in coud’ bedoven,
        V oude fleur komt weer boven,
        Omhelzen doet // gij Lucht en Aarde,
        T’welck overvloed // voort brengen doet,
        Door vruchten zoet // zo dat men spoed,
        Nuttigheid goed // voorcomt in waarde.
      
      
        T’gheboomt neemt aan // cleedingh van blaen,
        Die groenlijck staan // tot lust verlaen,
        Gaat ook ontfaen // zachte winds bewegen,
        Die velden groen // die wat leeft voen,
        Tot ruigte spoen // in dit saysoen,
        Door het toedoen // van uwen zegen:
        T’ghebloemt heeft me verkregen
        Schoonheid, reuk, en sierlijk bloeien,
        De Tuinen gemeyn // veelverwich rein
        Staan groot en klein // in onze plein,
        Door u certein // lieflijk en groeien.
      
      
        Het pluymgediert // zeer schoon versierd,
        Zijn zang bestiert // en vreuchlijck tiert,
        Zot gemanierd // is van naturen:
        Onrustig noch zwaar // zonder misbaar
        Is t’water klaar // en lijdt voorwaar
        T’ghebruyck van haar // tot allen neren,
        Gheen onghemacx bezuren
        Wilt gij lustighe Mei lijen,
        Voorteling bespoort // gij aan allen oord
        Met zoet akkoord, nu rechte voort,
        Ende oorboort // lieflijk verblijen.
      
      
        O vrolijke tijd // diet’ al verblijd,
        Prinslijcke vlijt // gij waardig zijt,
        Ghebenedijt // gaat men u achten
        Want Auroras gsicht // Flora u Nicht
        Phebus zijn licht // Venus gericht,
        Cupido schicht // brengen haar krachten,
        Nuttelijckst zij altoos wrachten
        Wanneer gij Iaerlijcx zijt verschenen,
        Met snaar gheclanck // lieflijk gesanck,
        Deuchtlijcke dank // zijt hier in zwank
        Den Lauren Rank // wij nemen.
