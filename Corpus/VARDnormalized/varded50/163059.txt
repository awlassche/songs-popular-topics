
    
      
        O Verenigd Land, Gezegend
        Zydy boven al vermaard
        Door de vlugge Faam befaemt // hoog vernaemt
        En als de gulden Sonn' verschoont:
        Want met' Hemels Dauw, beregent
        Wordt, bevochtight uwe Aerd'
        Popilier, Palm, en Laurier // Is het vercier
        Daar u Hoofd med' is gekroond
        Of het strijden // het benijden // het ontglijden
        Bij tijden // brengt het lijden
        het Verblijden geeft u God,
        Dit heeft ooit als ook nu gebleken
        Dat u viel het beste Lot,
        Want het bommen // het geklommen
        Al het trommen // al het krommen // met het schrommen
        Is gekommen al tot spot,
        En zo aerdighlijck uit gestreken
        Dus verloren Stad en Slot.
      
      
        Hoe geblixemt, en gedondert
        Hoe geplondert en vernield,
        Door het ysselijck erg getergh // van den Berg
        En den Koeckoecks droef gefluit
        Elk versneld, gesteld verwonderd
        Uytgesondert veel ontzielt,
        Elk verschrikt, beraest, verbaasd // als verdwaest
        Zo heeft daar den Hoorn getuit,
        Door het geruchte // elk pluchte,
        Elk duchte // elk vluchte // elk suchte
        Dan vloed elk overliep,
        Goede Hoop was om leegh gezonken,
        Als versmacht ten gronde diep,
        het Gedeckte // het bevleckte // het bepleckte
        Hem strekte // het ontweckte
        Dat hier bedrieglijk sliep,
        het Schijn gebluste ging weer ontvonken
        Als verblijd, verwinning riep.
      
      
        Maar doen het scheen het gemeen verloren
        In het verstoren heel gebracht
        Al het gekweel // en het straf gespeel // O te veel,
        O te veel ons toe gesent,
        Onverwacht en vreemdt om horen,
        Van te voren nooit bedacht
        Kwam een glans en schonen Krans // met gedans
        Door een Weselken bekent
        Door het spieden // het Uitwieden // het gebieden
        Van Dieden // geschieden
        Saghmen des vrome daad
        Dat in het Bos werd den reuk geroocken,
        Ja verdorret Plant en Zaad,
        het Beschikken // het recht mikken // het verklikken
        Bracht het schrikken // in strikken
        het Verkwikken te onzer baat
        Draken steert, Arends-bek gebroken,
        En gevestigd onzen Staat.
      
      
        Nu het verloren is gevonden
        En het vervallen op-gerecht,
        En den Yssel-Zoom en stroom // Bos in toom,
        Mits het beduchte is bewaard,
        Nu het benoude is ontbonden,
        Onze gronden vast gelegd
        Onze Hope vast gepast // Onzen last
        Al vrij wat verlicht, ontswaert
        het Verwezen // het mispresen // het verknezen
        Door dezen // genezen
        Geresen het laegh vermoeid
        En gekroockt nu den Staf van Spangien
        de Arends-veren af-gesnoeyt,
        Zijn Jongen // ontwrongen // ontdrongen
        Ontsprongen // met Tongen
        Gezongen onvermoeid
        Tot Gods Lof en de schoon Orangien
        Moet dan zijn daar Liefde groeit.
