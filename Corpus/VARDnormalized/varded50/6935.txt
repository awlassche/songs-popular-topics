
    
      
        Elk claeght dat hij // niet onbekommerd heeft,
        Vrolijk en blij // den gulden tijd beleeft,
        Dies op het pas // al zijn wij vol gewens,
        Is als zij was // maar t'volck is niet alleens.
      
      
        Tot nooddruft cleen // heeft dat Volk haar gevoecht,
        Daar nu elk een // aan overvloed misnoecht,
        Geeft dan de schuld // den tijd niet maar t'gemoet,
        Dat niet vervult // en werd in overvloed.
      
      
        Was in de mens // begeerlijkheid noch niet,
        Hij had zijn wens // als in de gulde tijd,
        Nu leven wij // in vol op noch bedroeft,
        Maar t'comt daar bij dat elk te veel behoeft.
      
      
        Die dan gerust // wil leven zonder haat,
        Zoeke zijn lust // in een gemene staat,
        In het heersen niet // van anderen t'ghebien,
        Dat men doch ziet // gehaat altijd gezien.
      
      
        Doch die van Stof // is, dat hij geern heerst
        Dwingh niemand of // bedwinght zich zelven eerst
        Want t'sal gewis // niet eender beter gaan,
        Voor dat elk is // zijn eigen onderdaan.
