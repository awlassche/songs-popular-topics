
    
      
        O Christus lief minnare zoet,
        Die mij schenckt u dierbare bloed,
        Na u verlangt mij te allen uren,
        Zo den Hert zijnd' in vare //doet,
        Na de Fonteine klare // vloed,
        In stortich pijnelijck bezuren,
        Ik leve in zwaar dolueren,
        Hoe lange, Lief, zal dit geduren?
        Wanneer zal ik uit rouwen // zijn?
        Wanneer zal ik aanschouwen // dijn
        Aanschijn, schoon boven naturen.
      
      
        O Lief in 'sWerelts wikke // niet
        Van mij in gheen versijcke // vlied,
        Maar helpt mij, dat mijn teere leden
        Niet en buigen gelijke // riet,
        Maar van u zoete Rijke // ziet
        Mij hier in dit dal beneden:
        Want ik vinde mij bestreden
        Van den Prinse der duysterheden,
        De Wereld in het verstrangen // kwaad,
        Alle dat nijdig Slangen // zaad
        En laat, mij niet met vreden.
      
      
        Ionstigh voor mij genegen // vecht
        Leidt mij in uwe wegen // recht,
        Door uwen Geest, o Lief verheven,
        Want ziet, ik ben bedeghen // slecht,
        Een teken daar elk tegen // zegt,
        Voor rasernije acht men mijn leven:
        En ik nochtans bekleven
        Wt uwe zijde zonder sneven,
        V Lief, u Bruid, u Kercke // reen,
        Ik doe door u o sterke // steen
        De Helsche poorten beven.
      
      
        O Prinse der Princieren // jent,
        Gij die herten en nieren // kent,
        Aenhoort mijn bitter klagen,
        Vwen Geest goedertieren // zendt,
        Dat hij door zijn regieren // wendt
        Mijn ledekens na u behagen,
        Want 'tzijn nu kwade dagen,
        Hierom Lief moet ik u vragen,
        Zal ik van al dit lijden // vrij,
        Met u niet haast verblijden // mij,
        En 'sLevens Krone dragen.
