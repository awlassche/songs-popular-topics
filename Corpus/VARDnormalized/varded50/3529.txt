
    
      
        Gij Amoureuse zinnen
        Die een schoon dochter vrijt
        Stelt niet zo vast u Minnen
        Op dat gij genen strijd // lijt
        Voor een zulk klein playsier // fier
        De loose Min vermijt.
      
      
        Een heb ik uitverkoren
        Zeer vast door Lieffd' eerbaar
        Doch weinig wiltse horen
        Na mij heur trou Minnaar // maar
        Dat menig droeven traan // zaan
        Mij kost met lijden zwaar.
      
      
        Bij Dagen ende nachten
        Heb ik zo droeven ganck
        Gegaan / en tot verachten
        Zo wordt nu mijn ontfanck // krank
        En voor dit trouwe leed / weet
        Zijt mij noch luttel dank.
      
      
        Dies heb ik niet ontvangen
        Voor al des moeyten ziet
        Dan droefelijk verlangen
        Van troost mij gants gheschiet // niet
        Want hare vrienden zijn // mijn
        Heel bitter met verdriet.
      
      
        T'hert heeft zij mij ontogen
        Deur haar bruin ooch opslag
        Mijn verstand inghesoghen
        Want zij een helen dag // lach
        In mijn herte gepast // vast
        Dit wercket groot geklag.
      
      
        Lief wat ghingt gij al spreken
        Tgheen nooit en was gemeend
        Als dikmaals heeft ghebleken
        Thert dat nooit was versteend // weent
        Om dat gij deur ontrou // ou
        Een kwade Faam verleend.
      
      
        Hoewelt mij zwaar zal vallen
        Te stellen wt den zin
        V Liefste Lief met allen
        Want tbrengt mij klein // gewin // in
        Daarom zeg ik dat gij // mij
        Ghetoocht hebt valse Min.
      
      
        Gij Iongmans vroom in eren
        Zulk dochters doch versmaad
        Zij doen den druk vermeren
        En werken vroeg oft laat // haat
        Waar deur een jong ghesel // wel
        Moet leven desperaat.
      
      
        Princessen hoog van moede
        Al zijdy rijk in pracht
        Verneert u als de vroede
        Want trotsheid hoog geacht // bracht
        Hemzelven totten val // al
        Ter sant vant heel geslacht.
