
    
      
        PErsse vol drucx //
        Die ik met drouven geeste tre- de //
        Nu moet ik fluks //
        Drinken den kelk des ongelucx //
        Tot afganck van mijn le- vens vrede,
        Ik die nooit mense iet misde- de //
        Ben van een maagd t'leven ontseyt //
        O woord van onbermhertichede //
        Moordadich zweert zeer scherp van snede //
        Laat deure van mij zijn beschreyt //
        Eer ik moet nemen mijn af scheid.
      
      
        Ik arm verdwaald
        Ballinck, dole in veel gewesten:
        Wat dinck mij faelt,
        Is haar zo beeldig afgemaelt,
        Daar kan niet el dan t'wesen resten:
        Des waters druppen, breekt ten lesten
        Een steen: maar aan dit kind // verdrayt,
        En kan God moet, noch reden vesten,
        Al is mijn dienst tot haren besten,
        Nimmermeer zij haar vindt // gepaeyt,
        T'zijn pluimen in den wind // gezaaid.
      
      
        Een sulcken wee
        Heeft mijn jong herte gans deurtloven:
        Gelijk de zee
        Passeert der drupkens een of twee,
        Gaat ook mijn smert, elks smert te boven:
        Eilaas! deur Venus vals beloven,
        Ik voor een lam, des Tigre koos:
        Mij ziende nu deerlijk verschoven,
        Inwendig branden als een Oven,
        Mishandelt als quaetdoender boos,
        Laat zij mij sterven troosteloos.
      
      
        Tot klein verblien
        Is t'sien, t'gehoor, t'gevoel, volhandig:
        Wat kant bedien
        Dat ik de liefste mag aanzien?
        Of horen haar woorden verstandig:
        Den middel smal, wt liefde brandig,
        T'omreyken, met behendigheid,
        Sulcx maakt mijn zinnen meer wtlandich,
        Verstaande haren haat vijandig,
        T'is een rechte ellendigheid,
        Blijft ongetroost d'inwendicheyt.
      
      
        Eerloos gespuis,
        Die vrouwen herten cont bekeuren,
        En laat confuys,
        Als gij haar ziet opnemen t'cruys,
        T'sy! t'is daad boven nateuren,
        Mocht mij van t'lief, zulks eens gebeuren,
        Met blijschap waar ik zaan // bekleed:
        Maar neen! veel wensen werkt doleuren,
        Ik arm gevangen moet nu treuren
        Met angst, en kommer, staan // besweet,
        Den rechter is om slaan // bereet.
      
      
        Rust, mijn gemoet,
        Niet meer suldy bange // sneven?
        V naakt voorspoed,
        Want gij zult achten t'sterven zoet,
        Ten aanzien van dit strange // leven
        Waar sijdy dood, dus lange // bleuen,
        Coemt, dat ik mag omhelzen dij:
        Mijn Cus zal aan dijn wange // kleven,
        Niet goeds kan mij een slange // genen,
        Die mij wil schenden, zulks dat gij
        Mij aengenamer zijt, dan zij.
