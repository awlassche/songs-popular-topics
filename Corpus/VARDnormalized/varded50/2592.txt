
    
      
        O Droeve tijd // die ik verslyt
        Met vruchteloos beklagen
        O aardse Hel // dood mijn gekwel,
        Door t'snoeyen van mijn dagen.
      
      
        Ik ben alleen // verd van de geen
        En tegen dank gebannen,
        Helaas! van die // ik het liefste zie
        Van alle jonge Mannen.
      
      
        Ach ongeluk // ach leide druk
        Zult nimmer gij ophouwe,
        Dit hart geweld // dat nu dus knelt,
        Des teer benaude Vrouwe?
      
      
        O schraele dood // komt en door stoot
        Des vrouwelijcke Borsten,
        Ach! streng gemoet // dat zo verwoed,
        Na u Kinds bloed gaat dorsten.
      
      
        Mijn Moeder wreed // doet mij dit leed
        En gij versufte Grijzer:
        Een Keizers kind // om dat het mint,
        Besluytmen straf in het Ijzer.
      
      
        U fel gegrim // u wreedheid slim
        U dreigen, noch u zweren,
        Geen schrik noch noot // noch duivel dood
        En kan mijn Liefde deren.
      
      
        Maar als ik zal // in het tranen dal
        Door ongheneught verdwijnen,
        Mijn schim bevreesd // of bleecke geest,
        Zal na mijn dood verschijnen.
      
      
        En kwellen zwaar // mijn Ouders maar
        Mijn Echte Man geprezen,
        Die zal ik dan // waar in ik kan,
        Noch even dienstig wezen.
      
      
        Mijn Prins vertoeft // weest niet bedroeft
        Bedaard, u wilt verlangen,
        Mijn hart blijft u // al is het lijf nu,
        In een Kerker gevangen.
      
      
        De trou en eed // dien ik u deed
        Die zullen geen sinds liegen
        Hoopt, en vaart wel // de dingen snel,
        Op aarden licht vervliegen.
