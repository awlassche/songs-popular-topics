
    
      
        Joden men hiet // die nasi ziet,
        Van Iacob af gekomen,
        Maar ten baat niet // namen noch iet
        Zonder boete met schromen,
        Men moet besneen // aan s’herten leen,
        Inwendig zijn een Jode,
        Christum (voorheen // zijnde ) naetreen,
        Als navolgers van Gode.
      
      
        Ontstaan is vrij // t’Antiochy,
        Den naam der Christen vaten
        Gij maakt al bij // dat ik ook zij
        Een Christen door u praten,
        Sprak Agrippa // Paulus daar na
        Sprak, ik zoude wel wensen
        Vrij zonder sa // dat zo waar ja
        Met u en alle Mensen.
      
      
        Christen oprecht // te zijn zo slecht
        Gheen sak en is beseven,
        Doch menig knecht // der zonden lecht
        Hem toe in zijn boos leven,
        Den name van // een Christen Man,
        Maar ziet lieve beminde,
        Christen en kan // niemand zijn, dan
        De geestelijke gezinde.
      
      
        Het eigen woord // Christen brengt voort
        Een gezalfde, mits dezen
        Wedergheboort // hem toebehoort
        Wt den Geest, die wil wezen
        Een Christen vroed // wezen hij moet
        s’Gheests zalvinge deelachtig,
        Met wandel goed // den roke zoet
        Daar van uitstorten krachtig.
      
      
        Ioannes zeid // gij hebt planteit
        De zalvinge des Heren,
        Die met bescheid // ter zaligheid
        V alle dinck kan leren,
        Niemants verstand // behoeft gij, want
        Verleyders Antichristen
        Zijn voor de hand // aan elken kant
        Verwekkende veel twisten.
      
      
        Mattheus zaan // wijst ons ook aan
        Dat valse Christen lieden,
        Zullen opstaan // met zoet vermaan
        Het volk wonder bedieden,
        Door haar praktijk // waart mogelijk
        Zij zouden d’uytvercoren
        Buiten Gods rijk // al door afwijck,
        Ook doen dwalen verloren.
      
      
        Antechrists geest // in dit foreest
        Met achterklap en smaden,
        Maakt veel tempeest // maar die God vreest
        En mag dit vier niet schaden,
        Hoe menig hoop // buiten den knoop,
        Der liefden stelt te werke
        Avondmaal, Doop // haren aanloop,
        Is tegen Christi kercke.
      
      
        En gaat niet uit // na t’vreemt geluid
        Vrienden, al doen zij horen
        Hier is de bruyt // in ons besluit
        Is Christus, want te voren
        Gij breet en wijt // ghewaerschout zijt,
        Elk wacht hem van verdolen,
        T’is nu den tijd // dat elk met vlijt
        Goed toezicht is bevolen.
      
      
        Recht zonder mis // te kennen is
        Een Christen bij onchristen,
        Klaar licht gewis // bij duisternis
        Vol nevelen en misten
        Men wel bekent // ook t’different,
        Van geest ende vlees vruchten
        T’recht fundament // der waarheid jent
        En zal voor niemand vluchten.
      
      
        Ten baat geen rom // op Christendom,
        Al waar s’gheests vruchten falen
        Liefde hooftsom // ik horen nom,
        Als van den principalen
        Vruchten ook me // vreugd ende vre,
        Gheloovich goedertierig,
        Lanckmoedighe // en cuyssche ze,
        Een gants wezen manierig.
      
      
        Zulke zeer schoon // vruchten als kroon
        Hebben gerust ooch zelve,
        Op den Gods soon // die spruit ydoon
        In Esaiam elve,
        Want op dit Lam // den Geest afquam,
        In ghedaent’ ener Duiven
        Des wijngaards stam // zijn ranken tam
        Dragen ook zoete druiven.
      
      
        Onchristen zijn // al die den Wijn
        stok Christi niet aankleven,
        Die elk termijn // niet worden zijn,
        Van den geest Gods gedreven
        Mank gaande slap // der deuchden stap,
        Oft ook levend’ in zonde
        Van gemeenschap // met God, t’gheclap
        Is al logen bevonden.
      
      
        Onze ghemeyn- // saemhede plein
        Bestaat in wandelinge,
        Gelijk hij rein // in het licht certein
        Is voorghegaen geringe,
        Maar zwak, wij zien, in ons gezien
        Gebreken met onvreuchden,
        Dan doch indien // wij recht belien,
        God reinigt van ondeuchden.
      
      
        Neemt dit in dank // elk Christen rank
        Laat den geest niet verkouden,
        Neffens den sanck // is meest belanck,
        Bidden zonder ophouden,
        Dat ons betaam // den Christen naam,
        In zwakheid die wij romen,
        God maak ons t’saem // voor hem bekwaam
        Te zijn oprecht volkomen.
