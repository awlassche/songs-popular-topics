
    
      
        Naar dien dat ik moet //
        Subiect, en dienstbaar staan,
        En mijn vrijheid zoet //
        Geheel te buiten gaan,
        Meer onderdaan //
        Dan mij rebel // vertoond //
        Heb ik, naar recht //
        Alzulk een knecht //
        Werd namaals wel // geloont.
      
      
        Edel, dier // en // werd,
        Is zij, die t'hert aanziet:
        Al dreyght vier, en // zweert,
        Ik sta tot haar gebiet,
        Deed'- ik zulks niet,
        Zo waar ik fel // gehoond:
        Wie twijffelt dies,
        Zulk tijd verlies,
        Werd namaals wel // geloont.
      
      
        Een alsulcken ieught
        Blijf ik eigen voortan,
        Werd het einde vreugd,
        Zo zit ik edel man
        Als Coninck van
        Gans Israhel // getroont,
        T'sy hoe dat gaat,
        De minste daad
        Werd naermaels wel // geloont.
      
      
        Liever smert // en leed,
        T'aenveerden, angstig zwaar,
        En haar t'zijn gereed
        Een knecht, drie seuen iaer,
        Zo werd' ik naar,
        T'gewonnen spel // gekroond:
        Dat dunkt mij zienst,
        Getrouwen dienst,
        Werd namaals wel // geloont.
      
      
        Keert zij mijn labeur
        Te zijden, als iet klens:
        Mij arm serviteur
        Toelaet luttel gemeens:
        T'sal werden eens
        Dat mijn gekwel // verschoont,
        Want die altijd
        Verdraagt, en lijdt,
        Werd namaals wel // geloont.
      
      
        Es ook meerder last,
        Vervaarlijk elks gepeys:
        Dat een dorper gast,
        Geniet der minnen cheys,
        Venus paleis
        Zonder bevel // bewoond:
        Och! t'is te klein,
        Dat zulk vilein,
        Werd namaals wel // geloont.
      
      
        Niet versagen // zal
        Mijn Prinselijk gemoet:
        Liever dragen // al,
        Des werelds tegenspoed
        Want diet mij doet,
        Is zo ydoonen maagd,
        Voor klein geween,
        Gij namaals geen
        Zo wel belonen // saeght.
