
    
      
        CVpido god der minnen // stout //
        Gij // geeft mij //
        In het hert menig wonde,
        En in bedwanck mijn zinnen // hout //
        Dat // ik mat //
        Ben versmacht te gronde,
        Nooit en konde // mij ontmoeten //
        Troost voor een verslegen // moet //
        Ander ionde // gij t'versoeten //
        Mij, veel drucx en tegen // spoed.
      
      
        Als gij te mywaerts wrimpich // ziet,
        Lijdt // en strijdt
        Mijn ieught overstreden:
        Gij zijt die mij zo schimpig // liet
        Van // elk man,
        Als een worm vertreden:
        Bij wat reden // hebdy voren
        Mij te doden zonder schuld:
        Laar verbeden // zijn dijn toren,
        Naar u goddelijk geduld?
      
      
        Tuwaerts moest te allen dagen // ooit
        Mijn // aanschijn
        Hem subiect verschonen:
        Nochtans gij mij veel lagen // strooyt
        Ach // ik mag
        Zuchten voor t'beloonen:
        Om zo schonen // kind der eren
        Lijd ik veel, die mij kan doen
        Deur haar Kronen // triumpheeren,
        Als deurstreden Kampioen.
      
      
        Heb ik Apollo dan // geslacht,
        Met // opzet,
        V geschut verschoven:
        O neen ik niet! wie kan // de // kracht
        Gaan // weerstaan,
        Diet al gaat te boven?
        Prinsen hoven // u geboden
        Nemen met groot wonder // aan:
        Dies u loven // alle goden,
        En u macht zijn onder // dan.
      
      
        Al lijd ik dan een smerte // wreed,
        Lof // hier of
        Moet u zijn gezongen:
        Venus (t'moederlick herte // heet)
        Heeft // beleeft,
        Vwen schicht deurdrongen:
        Hoe bedwongen // gij elk slave,
        In het prisoen van zuchten leidt:
        Als ontsprongen // wt den grave,
        Geeft mij voetstaans liberteit.
      
      
        Reden? beveelt met billig // recht
        V // alls nu
        Vroom te adoreren:
        Ik dan, als een goedwillig // knecht,
        Naar // d'Altaer
        Wil ik t'aenschijn keren:
        Supplieeren // daar zoetmondig,
        Om gehoor, tot mijn propoost,
        En begeren // zeer hertgrondich,
        Eens in liefden t'sijn getroost.
      
      
        Is het dat u gramschap neder // daalt,
        Is // gewis
        T'eynden mijn vernooien:
        Mijn aanschijn bleek // dan weder // haalt
        Een // zeer cleen
        Schoon rosierich bloyen:
        Weeldig groyen // is voorhanden,
        Al was ik zeer straf // gevaen:
        Kerkers, boyen // ende banden,
        Zijn subyt mij af // gedaan.
      
      
        Naar dat gij (als machtig // princier)
        Slaat // en laat
        Wederom genezen:
        Regiert deur u krachtig // bestier,
        Haar // eerbaar
        Princesselick wezen,
        Die geprezen // ieught bepeerelt
        Heb ik wtvercoren // ras,
        Wtgelesen // nooit ter wereld
        Soeter kind geboren // was.
