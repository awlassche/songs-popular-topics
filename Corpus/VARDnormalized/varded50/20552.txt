
    
      
        OCh ! hoe vast staat hij, o Here, Die op u betrouwt
        Op u, Rosteen, t'uwer ere // Heb ik vast gebouwd;
        Hierom zal ik niet bezwijken // Want in allen noot
        Zal mijn vaste bouwsel blijken // Tegen al aanstoot.
      
      
        Gij beschaemt al mijn Vijanden // En haar wreed' besluit
        Maakt gij in het geheel te schanden // En redt mij daar uit:
        Doen mij meende uit te roden // Mijne snoo partij,
        Ging den Heer mijn Vijand doden // En ik raakte vrij.
      
      
        Die u liefde mag genieten // O ! wat vreed', en vreugd,
        Gaat gij stadig op hem gieten // Die hem zo verheugt,
        Dat hij moet met aandacht zingen // Vrolijk uwen lof,
        Hier toe voel ik mij ook dringen // Hier toe heb ik stof.
      
      
        Komt mijn tong de kracht t'ontbreken // het Herte blijft vervult,
        Stadig wil ik van u spreken // Want ik weet gij zult
        Altijd zijn in mijne wegen // In vreugd, angst en nood:
        Ik vrees' niet, al heb ik tegen // Wereld, Duivel, Dood.
      
      
        Treet tot u, al mijn gedachten // Maakt mij, Heer, bereid,
        Ik wil stadig tot u trachten // Met eerbiedigheid:
        En wil vrolijk in u wegen // Tot u ere gaan,
        Och ! maakt mij bereid te degen // Dan zal ik vast staan.
      
      
        Kond' de vreuchd' in het herte blijven // (Door u goedigheid)
        Die soud' uit mijn herte drijven   des Werelds ijdelheid.
        het Herte kan het al niet vaten // het Wil niet zijn gemoeyt
        Om qua lusten in te laten // Want het overvloeyt.
      
      
        Soud' men, Heer, u niet beminnen ? Die u zoetheid smaakt
        Komt de grootste Rijkdom winnen // Die recht is volmaakt
        Want gij kont het hert vernoegen, Maar nooit staat, of eer.
        het Beste gaat gij mij toe-voegen // Lof zij u, mijn Heer.
