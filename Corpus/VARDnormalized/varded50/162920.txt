
    
      
        DIe vred' Gods die te boven // gaat
        Diens name hoog te loven // staat,
        Wens ik u al te samen Waar dat gij
        woont, met namen,  In het groene land // aan
        elken kant, Wandelt na u betamen,
        De Heer biedt u zijn hand.
      
      
        Al schijnt gij nu verlaten // schier,
        De Heer kom u te baten // hier, Och wilt dat
        wel bewaren, Dat hij u ginck verklaren,
        Zijn heilig woord // dat gij gehoord
        Hebt, van zijnen dienaren,
        Brengt doch goed' vruchten voort.
      
      
        Mijn liefste uitverkoren // al,
        Wel hem die des stem horen // zal, Komt
        hier mijn uitverkoren, Mijn Rijk staat u te
        voren, Ontvangt het loon // in 'sHemels
        troon, Gij zult vreugden orbooren,
        Ontvangen 'slevens Kroon.
      
      
        Ziet, niemand zal verblijden // zaan,
        Dan die nu deur het lijden // gaan, En die
        als rechte leden, Christi voetstappen treden,
        Den smallen ganck // haar leven lank
        Met Christelijcken zeden,
        Gelijk een vruchtbaar rank.
      
      
        Treurt nu niet om den tegen // spoed
        Den welken gij verdragen // moet,
        Is het zo Gods wel-behaghen, Zo wilt hem
        willig dragen, Lijdtsamich zijt // in uwen
        tijd, Het duyrt hier weinig dagen
        Gij zult noch zijn verblijd.
      
      
        En wilt doch des vergeten // niet
        Dat u de Heer noch weten // liet,
        Zijn heilig woord geprezen Wilt spreken
        ende lezen Te samen hout // elkander bout,
        Elk zij ghetrou in dezen,
        Dat hem hier is betrout.
      
      
        Recht houdt doch u becoepen // vrij,
        Gij Ouden, dit ontknopen // wij, Wilt
        een goed voorbeeld geven, Den Jongen met
        u leven, 'Tmoet zijn gewaagd, weest niet
        vertsaagd Gij zult worden verheven,
        Zo gij u dapper draagt.
      
      
        Ziet toe gij jonge Ieuchde // bloot,
        En hebt doch gheen gheneuchde // groot
        In s'weerelts lust voor ogen Dat gij niet
        wordt bedrogen, Van 'svyandts strijd // die
        u benijt, Zijn doen is al gelogen
        Wacht u daar voor met vlijt.
      
      
        Den Weduwen tot reender // eer,
        Geeft Paulus schoon tot eender // leer
        Dat zij haar hope maken, Op God voor
        alle zaken, Want hij beleeft // d'uytcomen geeft
        Wel hem die dit kan smaken,
        Gods Geest niet tegen streeft.
      
      
        Iaeght doch nu na den vreden // zoet,
        En zout in al u reden // doet, Elk wil zijn
        tongh' bewaren, Om niemand te bezwaren,
        Ook leugens kwaad // de Here haat,
        Hier in als Gods dienaren,
        Bruyckt doch de rechte maet.
      
      
        Tezamen wilt doch nu dragen // 'tlast,
        Elkanders na't gewagen // vast,
        Het is Christus Wet waarachtig, Weest des
        altijd gedachtig, Al dat gij zaan // geern
        had gedaan, Doe na dat gij zijt machtig,
        Zo zult gij wel bestaan.
      
      
        Vrienden gij staat in het herte // mijn,
        Het zoude mij groot smerte // zijn,
        Dat ik van u moest horen, Anders als hier
        te voren, Doe gij ghesont // bekendet rond
        Dat gij als uytvercooren,
        Woud houden Gods verbond.
      
      
        Gezonden is dit cleen // gedicht
        Tot u, dat gij wordt reen // gesticht,
        En dat gij ook meucht vaten, Dat gij niet
        zijt verlaten, Maar dag en nacht // in ons gedacht,
        De Heer com' u te baten,
        Bewaar u deur zijn kracht.
