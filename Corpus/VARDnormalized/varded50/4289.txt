
    
      
        E En excelent // en jent //
        Bloemken playsant //
        Heeft mij eilaas gewont
        Schoonder en kent // noch sent //
        Venus niemand //
        Noch is op swe- relts gront,
        Al doet zij mij veel lijdens on- gezond //
        Noch blijft in mij haar minne //
        Want zij van den beginne //
        Ooit in het herte stond.
      
      
        T'comt nu zo bij // dat zij
        Luttel verschoont,
        Mijn doloreus vermaan:
        Maar als zij mij // een blij
        Gezichte toont,
        Is al mijn druk gedaan.
        Daar heb ik zulk een vast betrouwen aan,
        Al moet ickt wat bezuren,
        Zij zal (tot allen uren)
        Vast in het herte staan.
      
      
        Certein, zo waar // als daar
        Den hemel schijnt,
        Zij heeft mij gans deurtast:
        Dies t'herte zwaar // in haar
        Liefde verdwijnt,
        Dits cause vanden last,
        Den brand (die in mij noch veel meerder wast)
        Zal mij (ducht ik) verteren,
        Nochtans blijft zij met eren,
        In mijn herte vast.
      
      
        Deur niemandts daad // noch raad,
        Des liefde fier,
        Nimmermeer te onder drijft,
        Hoe wel den haat // die kwaad
        Is in het bestier,
        Veel schanden van mij schrijft,
        Nochtans werd' ik duur haar weder gestijft,
        Zij is de weerdste vrouwe,
        Deur haar liefde getrouwe,
        Zij in het herte blijft.
      
      
        Princesse vroed // zijn moet
        (En niemand el)
        Mijn liefken elegant:
        Die mij den gloed // onzoet
        Der minnen fel
        Vierich in het herte sant.
        Mag haar (dit mijn gedich) komen ter hand,
        Zo ben ik vrij van smerte,
        En blijve in haar herte
        Eewich vast geplant.
