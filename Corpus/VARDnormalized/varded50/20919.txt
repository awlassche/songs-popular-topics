
    
      
        Staat op, en wordt verlicht
        O Mensen! staat op, en gaat na het eeuwig Licht,
        Verlaat uwe sond' // staat op terstond,
        En wordt na uwe ziel gezond.
        Gekomen // vernomen // is God, o lot!
        In het beesten schot;
        Wien zoeken gaan // (ziet het werk eens aan)
        Des Koningen met Goud bela'en.
      
      
        Uit Oosten komen Zij
        (O wonder!) door vorst, en hagel, even blij.
        Zij volgen na // de Sterre dra,
        Zij vinden haarder ziels gena.
        Zij alle // ne'ervallen // te voet // (hoe zoet!)
        Het opper-goed:
        het Welk in een stal // op het aardse dal
        Ne'er was gekomen voor ons all'.
      
      
        Ziet aan wat grote eer
        Des Prinsen, bewijzen aan dien groten Heer.
        Haar zoet gezucht // vlieght tot de lucht,
        Haar schreien is aan God ghenucht,
        Zij kussen // Zij sussen // het Kind // het welk sind,
        En Haar bemint.
        Zij geven Goud ,, aan die verkoudt
        Op het hooy leit ne'er, die d'Hemel boud.
      
      
        Het Wierook wordt gebrocht
        En Mirre, wie had zulks ooit van Haar gedocht.
        Met groot' ootmoed // Zij all' dit goed
        God dragen op, ja ook haar bloed.
        'k Wil mede // gaan treden // bij Haar // en naar
        Des tijd voorwaar
        Aan dezen Heer // uit liefde teer
        Mijn hart op dragen het sijnder eer.
      
      
        Zij groeten ook de Vrouw
        MARIAM, die ons verlost heeft uit den rouw.
        het Is aanziens waard // het moet zijn bewaard
        In het hart, hoe dat des totter aerd'
        Haar buigen, en tuigen // met le'en // en re'en,
        Met blij geween,
        Dat Zij de kroon // en de Opper-loon
        Verdiend van Christo haren Soon.
      
      
        Laat ons nu desgelijk
        Ook knielen, die anders-niet en zijn als slijk.
        Laat zeggen uit // ons overluid,
        MARIA! Gij zijt Godes Bruid.
        G'ontfermt ons // G'erbermt ons // in strijd // Gij zijt
        Die ons verblijd.
        Ei ! zeg ons aan // dat Gh' ons ontfaen
        Zult naemaels binnen des Hemels baan.
