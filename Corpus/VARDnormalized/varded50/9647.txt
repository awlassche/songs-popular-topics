
    
      
        DOchter Syon eersame // maagd,
        Gij die des Heren name // draagt,
        Volgt mij in swerelts plein,
        V Bruidegom alleyn:
        Na mijnen prijs bekwame // jacht,
        Dat elk wie na u fame // vraagt,
        Getuigenisse rein
        Van u hoor' in het ghemeyn,
        Want kleinen tijd zal toeven
        V verlossinge naar
        En wilt u niet bedroeven
        In al u lijden zwaar,
        Met goed geduld // Wilt in onschuld
        Wassen af uwe handen // fijn,
        Eerlanck gij zult // Met vreugd vervult,
        Verlost uit banden // zijn.
      
      
        Onder 'tverkeert geslachte // kwaad,
        O Sion op de wachtte // staat
        Met goede wapen // gij
        Om te verknapen // mij,
        En op Sathan wel achte // slaat,
        Die met een loos gedachte // gaat
        Om te betrapen // dij,
        Houdt u van slapen // vrij,
        Dat hij u niet en scheyde
        Van uwer simpelheid,
        Gelijk als hij verleidde
        Eva, zo de Schrift zeid,
        Rein ongestoord // Blijft in mijn woord,
        Alwaar gij zijt geboren // uit
        Met goed akkoord // Zo blijft gij voort
        Mijn uitverkoren // Bruid.
      
      
        Mammon en al afgoden / vlied,
        Altijdts op mijn geboden // ziet,
        En onderscheid zo wel,
        Van al 'sMenschen instel,
        Gedenkt hoe ik de Joden // liet
        Zij en zijn mij ontvloden // niet,
        Maar ik straftese fel
        Die mij waren rebel,
        Dat volk mij kwalijk luste,
        Ik zwoer met grammen zin:
        Dat zij tot mijner ruste
        Niet en zouden gaan in,
        Vriendinne dan // Zo veel u kan
        Mijn rein liefde bewegen // doet,
        En wijkt niet van // Mij uwen Man,
        Om druk noch tegen // spoed.
      
      
        Princesse, die in het lijden // quelt,
        Al u treuren bezijden // stelt,
        Want ziet mijn komste naakt,
        Dus mij niet en versaeckt:
        Alleen t'wettelijck strijden // geld,
        Hierom zonder vermijden // snelt,
        In mijn liefde doorblaeckt:
        Mijn vriendelijkheid smaakt,
        Mijn schatten, mijn tresooren,
        Mijn vreugd, mijn heerschappie
        Zal u al toebehoren,
        Nooit soeter melodie,
        Van harpen klank // En nieuwen sanck.
        Van lustighe Muzieke // goed,
        Zonder verganck // Werd u ontfanck
        Mijn eeuwig Rijke // zoet.
