
    
      
        EEr, Lof, Prijs, in het Hollands plein
        Zij u o Liefste schoon
        Door de zedigheid ghemeyn
        En aardigheid ydoon
        Gratieus // zoet en heus
        Zijt gij maechdelijcke zaad
        Waarom trou // El jongvrou
        Ik u Min in Lieffdes graat
        O gij reine // Maagd
        Die alleyne // plaecht
        Hem / die Lieffd' naar jacht.
      
      
        Menich lieven zoeten praat
        Wij hebben tsaem ghepleecht
        Lieffd' waarbij de wereld staat
        V nooit en heeft beweegt
        Mijn begeer // was op eer
        Door een onghevalste Min
        Maar gij vroed // Liefgen zoet
        Van mij keerden uwen zin
        Ik betrouden // mijn
        Op u goude // schijn
        Maar die bracht des pijn.
      
      
        Als Aurora in klaarheid
        V rein Versiersel blinckt
        Waarin hert en zinnen leit
        Die tsamen gij bedwinckt
        Als de Zon,, Helicon
        Overstraelet wel van vart
        V gezicht // alzo licht
        Ook inwendig me het hart
        Dat mits dezen // dan
        Het niet wezen // kan
        V te wijken van.
      
      
        Met des wijsheyts deuchden kroon
        Ik chieren zal u hoofd
        En dan stellen opten troon
        Van Pallas, dit geloofd
        Dein gheplant // kloek verstand
        Sulcx in alles Meriteert
        Dan het schijnt // diet zo pint
        Gij in Lieffden niet begeert
        Mijn verheven // dier
        V trots leven // fier
        Kan verandren schier.
      
      
        Mijn Prinses vol hoocheyts aard
        Ik wens u meerder vreugd
        Dan gij voor u Dienaar spart
        Die u niet jont dan denkt
        Die gij lont // onverschoont
        Doch ik u niet haten // zal
        Dies voort lest // op zijn best
        Moetmen ook wat praten // mal
        Hout u waren // gij
        En laat varen // mij
        Adieu zeg ik dij.
