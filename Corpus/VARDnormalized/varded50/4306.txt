
    
      
        NObel divijn //
        Wezen sanguijn //
        Prijs naar ge- dienstig pogen,
        Deurluchtich fijn //
        Als Christalijn //
        Rayen u Klaar bruin ogen,
        Lichten als twee //
        Sterren ter zee //
        Die mij mijn wee // vertogen //
        Ondanks ik moeds gedogen.
      
      
        Edel gezet
        Staat lijf // en ledt,
        T'mach zulk een herte vromen:
        Was Venus net,
        Gij meugt u met
        Veel meerder recht beromen:
        Nooit en sach man
        Volmaecter an,
        Al watmen kan genomen,
        Gij draagt in al t'volcomen.
      
      
        Enig gewin,
        Die van t'begin,
        Waart cause van versijcke,
        Maar niet te min,
        Herte en zin,
        Prent u in mijn Chronijcke,
        Tot veel gepijns,
        Geen medecijns
        En weet ik dijns // gelijke,
        Op geheel aarden-rijke.
      
      
        Leven, en dood,
        Vreugd, wederstoot,
        Staat in dijn welberaden:
        Gelijk als brood
        Bluscht hongers noot,
        Condy mijns drucx verzaden,
        Dus welgemint,
        Soethertich kind,
        Weest mij doch in het // beschaden,
        Een voorbeeld der genaden.
      
      
        Kies' ik niet best
        In dit arest,
        Aflaat van mijn doleren,
        Hier aengevest,
        Volgt een rekwest,
        Om u te remonstreren:
        Den suppliant,
        Verzoekt bijstand,
        Wilt metter hand // schriberen
        Fiat, naar t'supplieren.
      
      
        Een sterk bewijs
        Van lof en prijs,
        Zal u bij dien gewerden:
        Schoon ieughdich rijs,
        Gestaan propijs,
        Ik moet u naarder terden,
        Waar mij ontseyt
        Dijn mogendheid,
        En gaaft afscheid ten derden,
        Noch soud' ik t'stuck volherden.
      
      
        Niet el schoon lief,
        Heyscht dezen brief,
        Bij minder hand geschreven:
        Doen ik besief,
        Dat al mijn grief,
        Bij u mocht zijn verdreven,
        Heb ik mijn leed,
        Voor u ontcleet,
        Dus wilt gereed // doch geven,
        T'behouden van mijn leven.
