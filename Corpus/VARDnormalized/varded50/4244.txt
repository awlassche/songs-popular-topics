
    
      
        CLouck ingenieus //
        volmaectste aller flueren //
        Vroom, en gratieus //
        in gauen der naturen //
        Enig boeten //
        en verzoeten //
        daart al bij verheugt //
        u t'ontmoeten //
        ende groeten //
        is een zoete vreugd.
      
      
        Al wat zonder // u
        Geschiet, en kan niet treffen:
        Ligh ik onder // nu,
        Gij cont mij haast verheffen,
        Bij gedogen,
        Wilt mij togen
        In mijn wederspoed,
        V bruin ogen,
        Of verdrogen
        Moet mijn vlees, en bloed.
      
      
        Te  waar wel reden // dat
        Ik u van minnen / sprake:
        Als gestreden // zat
        Ik brandig binnen // blake //
        Wtgelesen,
        Hoog geprezen,
        Beeldeken geeert,
        V bij wezen
        Kan genezen,
        T'swaerste dat mij deert.
      
      
        Hebt in handen // gij
        De macht, dat (naar t'volherden)
        Al des banden // mij
        Een vreugd konnen gewerden,
        Wijt vermaarde,
        Jonk geiaerde,
        Zijn mij (alst u lust)
        Een vermaarde
        Sauegaerde,
        Daar mijn herte rust.
      
      
        Als ik schade // loos
        Aan u vinde ontzetten,
        Alle kwade // boos,
        En konnen geenszins letten:
        Wat verscheiden
        Tongen zeiden,
        Acht voor schimp, en spijt,
        Wilt verbeiden
        V bereiden
        Dienaar, taller tijd.
      
      
        Ramp, en schade // dan
        Acht ik voor windig blazen,
        V genade // kan
        Mijn droufheyt versolacen:
        T'soet beluchten
        Kan doen vluchten,
        Des bruinen zwaar
        Dat mijn duchten,
        Treuren, zuchten,
        Moet verdwijnen // klaar.
      
      
        In het vertrouwen // vast,
        Van u deught-saem belonen,
        Geen berouwen // wast,
        Van mijn gedienstig tonen,
        Hadde konnen
        Zijn gewonnen,
        Van u, iet gesienst,
        Onder zonnen
        Waar begonnen
        Nooit geen meerder dienst.
      
      
        Neemt (O werde // maagd)
        Dit dankbaar in het gehele,
        Al wat d'eerde // draagt,
        Schinck ik tot uwen dele
        Daar beneven
        Lijf, en leven,
        Is u, metter daad,
        Toegeschreven,
        
          Overbleuen
        
        Niet, dat mij aangaat.
      
      
        Adieu vreden // borght,
        T'scheyden heeft nu oorzake,
        Voor u heden // zorgt
        Den geest, altijd in wake,
        Dit bedrijven!
        Laat nu blijven
        V contentement,
        Voor te schrijven
        Veel meessijven,
        Wens ik tzijn present.
