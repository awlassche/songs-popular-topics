
    
      
        TOt maechdelijcke spruiten
        Mijn dicht // ik richt // eerzaam,
        Die binnen ende buiten,
        Aan geeft // ook leest // lichaam,
        Heilig rein en bekwaam,
        Ook meer // den Heer // te achten,
        Dan terecht // van d'echt // betracht,
        Dit leert Paulus te zam.
      
      
        Ik wens u voor een groete,
        Veel denkt // o jeugd // verlicht,
        Ghenade vrede boete,
        Een waar // een klaar // gezicht,
        Een wandel wel gesticht,
        Bedacht // dag nacht // met smeken,
        Voort val // der smal // ghebreecken,
        Dat doch t'ghemoet niet zwicht.
      
      
        Ziet deze salutacy,
        Tot goed // met spoed // om meest,
        Zij u van Godes gracy,
        Ghewracht // met kracht // des geest,
        Door Christum onbevreest,
        Op dat // ons schat // geladen,
        Verdriet // gheschiet // noch schaden,
        Die Schrift doorzoekt en leest.
      
      
        Gelijk gij zijt natuurlijk,
        Elk maghet // waghet // zacht,
        Met Gods Heiligen puurlijk,
        Dat gij // en wij // veracht,
        Maagden zijn inder kracht,
        Die t'Laem // vreedzaam // navolgen,
        Noch list // noch twist // verbolgen,
        Moet zijn van ons bedacht.
      
      
        Eerstelingen verkoren,
        Geen goed // maar bloed // ghecost,
        Kinders weder geboren,
        Die elk // met melk // en most
        Ghelaeft zijn en verlost,
        Des prijs // als wijs // van moede,
        Des naam // bekwaam // en goede,
        V neme brood' noch post.
      
      
        Roem van naam of waerdye
        Niet vaat // is daad // absent,
        S'gheestes of vleysch hoverye,
        Slaat dood // hoe groot / present
        God bewaar ongeschend
        Die recht // en slecht // hier leven,
        Met vreugd // tot denkt // haar geven
        In blyschap en torment.
      
      
        Rein heilig van manieren,
        Zulk maagden // haechden // t'lam,
        Die haar met denkt vergieren,
        Niet stout / met goud // aan kwam,
        Noch onrechtveerdich mam,
        T'gheslacht // hier zacht // van zinnen,
        Dit stil // Gods wil // beminnen,
        Zijn vanden rechten stam.
      
      
        In Exody // wij // leren,
        Tien paar // wel klaar // subijt
        Vader en moeder t'eeren,
        Ephesen // lezen // wy't
        Sierach drie dreycht met vlijt,
        Die kinder // hinder // t'senden
        Die d'ouden // zouden // schenden,
        Als nu in dezer tijd.
      
      
        T'wort in Matheo vonden
        Ik mien // vijftien // die vaat,
        In Marco ook des gronden
        In het zeven // schreven // staat,
        Collossen drie ook gaat,
        Die meer // begeer // om weten
        Bezie ook die Profeten
        Wat hier is Godes raad.
      
      
        Spieghelt hier in u werken,
        T'is wel // t'bevel // van God,
        Maar weinig die het bemerken,
        Met hopen // lopen // zot,
        Die jeugd acht Gods gebod
        Voor niet // besiet // haar gangen,
        Zij dichte // lichte // sanghen,
        Dit is des werelds rodt.
      
      
        Sara in der stad mede,
        Zie zeid // bescheid // aenhoort,
        Met die ijdel van zeden,
        En meen // ik gheen // akkoord,
        En die Psalmist opboort,
        Wel dien // die vliet // den kwaden,
        Altijd // met vlijt // beladen,
        In Godes heilig woord.
      
      
        Ook heeft Salomons leren
        Een schone // krone // toe zeid,
        Die d'ouders obedieren,
        Der looser // bozer // zeid,
        Geenszins en zijn bereid,
        Door woeden // spoeden // waarlijk,
        Tot kwaden // schaden // vaarlijk,
        T'verderven haar verbeid.
      
      
        Och waar gaar ik mij keren,
        Wat zake // maak // ik hier,
        Met vermanen en leren,
        Diens vaar // om zwaar // dangier
        Te weren, heeft regier,
        Zijn huis // t'abuys // te letten,
        Gods geest // dits meest // ginck zetten,
        Hem over zijn bestier.
      
      
        Neemt mij doch dit ten besten,
        Want ik // verschrick // aan een,
        Nu ik bedenk ten lesten,
        Tot wien // gezien // mijn reen,
        Want mijn wijsheid is cleen,
        Gheen const // maar gonst // en minne,
        Wt richt // en dicht // den sinne,
        Denkt mij in u gebeen.
      
      
        Ik was sorchd ik verbonden,
        V kloek // verzoek // zulks de,
        Door u laatste vermonden,
        Het geschenk // gedenk // ik mee,
        Voort liefde trou en vree,
        Die drijven // t'schrijven // tezamen,
        Maar ziet // t'gheschiet // met schamen,
        Hoewel met danckbaerhee.
      
      
        Die dochters die ik meene,
        Haar vaar // wordt klaar // genaamd,
        Eerste letters bij ene,
        Elks vers // zo dwers // versaemt,
        Tys Gherritsen het raemt,
        Waar door // ter coor // bedacht // ziet,
        Met haast // het naast // die klacht // ziet,
        Memory wordt gheblaemt.
