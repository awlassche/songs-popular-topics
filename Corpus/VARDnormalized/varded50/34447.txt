
    
      
        WEe mij! wee mij! eilaas wat zal dit zijn?
        Zie ik u Pyrame, oft uwen schijn?
        Wee mij! ik sterve // Als ik u derve;
        Moest gij mijn leven // Mij aldus begeven?
      
      
        O wreede Goden waarom laetje toe
        Dat mijnen Pyramus (‘k en weet niet hoe)
        Zo ligt door-steken? In purp’re beken
        Zijns bloeds ghedolven // Tot spijs’ der wolven?
      
      
        Ach had’ ik immers hier geweest ontrent
        Eer hij zijn leven had aldus vol-endt!
        ‘k Hadd’ beyd’ sijn’ ogen // Door zoet med’-dooghen
        Flux toe-geloken // Kwalijk noch gebroken.
      
      
        Den laatsten adem hadd’ ik ook ontsaen,
        Eer hij naar Charons schuytjen waar gegaan:
        Maar nu och-ermen! Alleen door kermen
        Kan ik doen merken // Wat de minn’ kan werken.
      
      
        Gij doet, o Thisbe, het is noch niet laat,
        Wilt gij hem volgen, ‘k Weet u goeden raad:
        Den bloed’ghen degen // Die hij heeft kregen
        Om zich te doden // Schijnt u ook te noden.
      
      
        Gij sweirt die hebt de een helft van mij door-boort
        Dood de ander ook, en mij geheel vermoort:
        ‘k Wil liever sterven // Als het leven erven,
        Daar hij is bleven // Die mijn’ ziel’ ded’ leven.
      
      
        Wel-aen mijn’ ziel’, en weest dan niet vervaard,
        Toont u manhaftig (schoon ook tegen aard)
        Doet vrees’ verdwijnen // Wilt gij verschijnen
        Flux daar uwe Lief, is // het Sterven uwe gerief, is.
      
      
        Ei toeft wat Lief, ei toeft, ik kom u bij,
        Gij zijt gestorven om den will’ van mij:
        Ik zal ook tonen // Dat ik kan lonen
        Als uwe vrindinne // Minn’ met weder-minne.
      
      
        Ziet daar mijn’ Liefste, neemt dat gaat voor goed,
        Ziet daar al-reede spruit het purper bloed;
        Mijn hert’ is zuchtig // Mijn zieltjen vluchtig;
        ‘k Ben heel onmachtig // God weest mijns gedachtig.
