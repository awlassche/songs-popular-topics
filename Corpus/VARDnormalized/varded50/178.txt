
    
      
        O Venus ient, een zwaar torment,
        Een groot allent // lijdt ik bij dezen
        Gij staat geprent, in het hart bekent
        Zeer excelent // lieffelick van wezen
        Wilt gij mij niet, verlaten ziedt,
        Alle mijn verdriet // wat mij gheschiet
        Zal ik achten niet // schoon Lief geprezen.
      
      
        Mocht ik van haar, schoon anschijn klaar
        Al zonder vaar, eens troost verweruen,
        Geen goud zo zwaar, koos ik voor haar,
        Een lieflijk paar // zouden wij steruen
        Alle mijn jolijt // in dezen tijd
        Mijn Hart verblijd in het swerlts krijt,
        Tis meer dan tijd, zal ik troost eruen.
      
      
        O Maechdelick saet, compt mij te baat
        Eert valt te laat, helpt wilt druk veriagen
        Zo zoet gelaat // zeer delicaat
        Doet Venus raad // schout nyders kwaad
        Geduld tot mij, die in lijden zij
        Zo waar ik blij, dat zeg ik vrij,
        Niemand dan gij // en derf ik klagen.
      
      
        Princesse rein, gij zijt alleyn,
        Lief dien ik mein // schoon bloem verheuen
        Dat weerste grein // gij zijt certein,
        In het swerlts plein // wilt mij niet begeuen
        Geeft mij confoor // een troostelic woord
        Lief rechte voort, weest niet verstoort
        Mijn beden anhoort // en laat mij leun.
