
    
      
        PAscha, dat hier // voor bij te lijden,
        'tSondigh bestier // moeten wij mijden,
        'tDangier // van dien
        Als fenynighe Slange vlien.
      
      
        De Christen schaer // heeft hier beneden
        Geen blijf-stadt, maar // voorbij moet treden
        Al naar // de stad
        Daar haar // hert is, en haren schat.
      
      
        Geacht zijn zij // als vreemdelingen
        Of wij voorbij // door een stad gingen,
        Daar wij // schier niet
        Ons aantrekken wat er geschied.
      
      
        Of men hier hoort veel vreugd verwekken
        Wij moeten voort // recht henen trekken
        Na het woord // van hem
        Die ons roept na Ierusalem.
      
      
        Tis groot abuis // hier lanterfanten,
        Om vlieden het kruis // en blijven planten,
        Zijn huis // opt zand,
        Buiten het eeuwige Vaderland.
      
      
        Des Heren dag zal // als strik bevangen
        Die op dit dal // wonen en hangen
        Met al // den zin,
        Op de dingen die hier zijn in.
      
      
        Want vyerigh moet // vergaan des stede,
        Iae al haar goed // en borghers mede,
        Dus doet // hij wel
        Die uit midden van haar wijkt snel.
      
      
        Want die hier geen woning en maken,
        Dan op de been // om voort te raken,
        Alleen // door vlijt,
        Die blijven in 'sHeeren bevrijd.
      
      
        'tGhemoedt geschort // op, onversteenight
        Het herte wordt, met God verenigd,
        Ghegort // in band
        Der liefde die vyerigh brand.
      
      
        'sLams goedheid meest // de zulke smaken
        Ter Passen-feest // daar zij vermaken
        Den geest // met vreugd,
        Die de ziele van binnen verheugt.
      
      
        Door d'ongheree // en onghebaende
        Woestijn oft Zee // recht henen gaande,
        Gheen wee // noch leer
        En leide ons af van den Heer.
      
      
        Zij zullen met // hem tehuis verblijden
        Die na zijn Wet // verwinning strijden,
        Ook het // hier zal
        Wel haast wezen volstreden al.
      
      
        Dan zal men hoen // en cranssen delen,
        Des levens groen // en den gehelen
        Roof koen // Sioen
        Zal onsterffelijckheydt aandoen.
