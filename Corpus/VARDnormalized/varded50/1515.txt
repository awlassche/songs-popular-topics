
    
      
        Gij Ionghmans fray
        Die vreugd zoekt te hanteren
        Gaat plant de Mei
        Met u liefste vol eren,
        Op dat u liefd' macht groeien // daar
        En met haar in liefd' bloeien // klaar
        Wilt snoeien // haar, wilt snoeien // haar
        Die nijdigheid useren.
      
      
        Als gij dan koen
        Met u Lief zijt gezeten
        Buiten in het groen
        Laat droefheid zijn vergeten,
        En brengt Harpen en Luiten // bij
        Ook Rethrozijnsche kluyten // blij
        Wilt uiten // vrij, wilt uiten // vrij
        Om haar als lief te heten.
      
      
        Loer-ooght en winckt
        Minlijck aan beide zijden
        Kuskens bedinckt
        Onder u lieflijk vrijen,
        Op dat ze u mag kronen // wis
        En liefd' met liefd' te lonen // is
        Wilt tonen // fris, wilt tonen // fris
        Dat gij garen zout bedijen.
      
      
        Zo gij dan weet
        Dat zij u gaat toe dragen
        Haar liefd secreet
        Zo wilt u lust bejagen
        Om Venus te verfraaien // hoort
        Haar akker te bezaaien // sport
        En maaien // voort, en maaien // voort
        Vruchten na u behagen.
      
      
        Maar nimmermeer
        Wilt brengen in benouwen
        Noch om haar eer
        V fleur van alle Vrouwen,
        Want zij is haast daar van // berooft
        maar op ere haar dan // belooft
        Als man // en hoofd, als man // en hoofd
        Om voor u lief te houwen.
      
      
        Princesse jent
        Ontvangt op deugd // verheven
        Des May bekent
        Die ons veel vreugd // gaat geven,
        Want door des Mays virtuyten // zoet
        Ons vele bloemkens spruiten // goed
        U buiten // spoed, u buiten // spoed
        Daar wordt gheneught // bedreven.
