
    
      
        Schoon lief wat gady brouwen,
        Dat gij mijn dus verlaat,
        Eilaas het zal mij wel rouwen //
        Dat ik o maechdelijck zaad //
        Ginck stellen al mijn betrouwen
        Op v woordekens delicaat //
        Bij Dalida die schone //
        Mag ik v gelijken wel
        Die Samson t'haer idone //
        Afsneet door vals opstel //
        Alzo krijg ik nv loone //
        van v zeer kwaad en fel //
        Ik loofde samblans der ogen,
        Die gij mij lief ginck togen
        Een Amoreus gebaar,
        Het dunkt mij een paar //
        V woorden zijn gelogen //
        Waar door ik ben bedrogen //
        Dies faute vind ik waar //
        Dus zeg ik voor en naar,
        Adieu schoon lief eerbaar.
      
      
        Mijn liefde zal wel vercouwen //
        Al is zij nv wat heet //
        Daar zijn meer schone vrouwen //
        Wijs eerbaar en discreet //
        Ik en zal v niet meer benouwen //
        Wie dat het zij lief of leed //
        Wat leyter ook aan mij bedreuen
        Daar sijnder veel schoonder certein,
        Mannen als Bomen verheuen
        Die noch leun in het aardse plein //
        Dat gij mij hebt begeuen //
        Dat deed ik was te klein //
        Dat gady lief verspreyen //
        Alzo ik hoorde greien //
        Van een iegelijk in het openbaar //
        dat dunkt mijn een kleine maar
        Al gady dit verbreien //
        Ik wilder niet om scheyen //
        Ik acht het niet een haar //
        Verkiest vrij een aar,
        Adieu schoon lief eerbaar.
      
      
        Adieu o Roosken geprezen //
        Adieu daar mijn troost op stond //
        Adieu ik waar genezen //
        Met een cusken aan v rooder mond
        Adieu het moet nu wezen //
        Adieu en blijft ghesont //
        Adieu ik neem mijn kere,
        Na v ere heb ik gestaan //
        Adieu ik en kom niet weere //
        Ik heb mij mede beraen //
        Ik denck daar zijn der meere //
        diet pat hebben mede gegaan //
        Meyndy dat ik zal treuren
        En t'hart van binnen schuren,
        Van druk ancxt en vaar //
        Om v liefken eenpaar
        Neent zal zo niet geburen //
        Na een ander zal ik speuren //
        Ik denck om v niet een haar,
        dus peynst gij om een aar,
        Adieu schoon Lief eerbaar.
      
      
        Adieu ik maak een einde //
        Genoeg muechdy verstaan //
        Van t'ghene ik v hier zende //
        O princelijcke grein //
        Te voren ik om v stende //
        Dit is nu al gedaan.
        Dit kwam al door v wezen //
        Dat gij een ieder toont //
        Zeer lieflijk geprezen
        Ik wasser mede gehoond
        Zij zullen noch bij dezen //
        Als ik worden gheloont
        Met loghens ginck zij mij payen,
        en ook twee Orens an nayen //
        Ten viel mij niet zwaar //
        Ik wil den dag ende Jaar
        Van v wel zijn verscheyen //
        Ik sou om v niet schreien //
        Noch maken groot misbaar
        Van v neem ik mijn verhaer
        Adieu schoon Lief eerbaar.
