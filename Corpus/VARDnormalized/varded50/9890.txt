
    
      
        Gods wijsheid slechtich // voor swerelts oog,
        Gedaald oprechtig // van boven hoog
        Hoog disputeren // niet en behoeft,
        Zulk argueren // den geest bedroeft,
        Terecht simpel verstand,
        In waarheid geplant // Wel berypen kant
        T'woort door den geest gheproeft.
      
      
        Al wat niet geestig // en smaakt gezond
        Dat moet tempeestigh // als ongegrond,
        Opt zand verdwijnen, neven den steen,
        Donker bruwijnen // en mogen gheen,
        De waarheid perfeckt // Houden bedekt,
        Oft schoon vertrekt,
        Zij verwint in het gemeen.
      
      
        'sHeeren Gemeente // met hert en lust,
        In goe vereente // staat vast gerust
        Op de rein waarheid // als een pilaar
        Van zuiver klaarheid // die hier noch daar
        Niet henen en draeyt // wat wind dat waeyt,
        In haar ghezeayt
        Gods woord, verstercks' eenpaar.
      
      
        Broeders eendrachtig // elk in zijn graat
        Laat ons aandachtig, na Petrus raad,
        Onder den Heiden // ghelijckt betaamt,
        Goe wandel leiden // dat elk hem schaamt
        Tot beterschap // die met gesnap,
        Door achterklap,
        Ons hebben zal gheblaemt.
      
      
        'tRecht vromer leven // van Abel aan,
        Staat al voorschreven // ons tot vermaan,
        Schoeriem oft pleytken // van enig draad
        'tBleetende geytken // daar ook al staat:
        Niet schuldig en zijt // Dan liefden vlijt,
        Die het gener tijd
        Den naasten en doet kwaad.
      
      
        In des qua tijden // voorhanden bloot,
        Laat ons vermeyden // in kleen oft groot,
        Dat met bezinnen // zelfs niemand wordt
        Buiten oft binnen // van ons verkort,
        Dat zoete lucht // Van goed gherucht,
        Door goede vrucht,
        Van ons zij uitgestort.
      
      
        hetzij vremd' oft vrienden // wie dat ons borght,
        Zo wel wij dienden // te zijn verzorgd:
        Want, ziet, den name // den welckens wij
        Aanroepen tsame // dezen wordt vrij
        Voor borge betrout // En wie hier op bout
        Hoe zal doch stout
        Dezen te kort doen hij.
      
      
        Wie en zal pogen // om laten niet
        Enen zo hogen // borg in het verdriet
        Of zo in schaden // dat men gestoord,
        Daar door versmaden // zijnen Naam hoort,
        En het volk vermaard // Van zijner aard,
        Iae recht als waart
        Bedrieghelijck in zijn woord.
      
      
        Die dus laat steken // zijn borg in last,
        Godloze treken // doet hij, tis vast:
        Ook staat geschreven // hoe dan in al
        T'slecht overbleven // Israels getal,
        Gheen leughenen doch // Iae tonge noch
        Doende bedrog,
        Gevonden zijn en zal.
      
      
        Dus moet den Tempel // Gods staan gesticht,
        Christi exempel // is klaar en licht,
        Te genen stonden // was ooit onvroed
        Bedrog gevonden // in zijn mond zoet:
        Wie valsheid bedrijft // Oft leugen verstijft,
        Niet bij hem blijft,
        Maar uit zijn huis hij moet
      
      
        Niemand bedriegen // gij niet en zult,
        Sprak die van liegen // nooit hadde schuld:
        Ziet, wilt aanschouwen // een Israelijt
        Oprecht vol trouwen // daar gener tijd,
        T'zy meer oft min // Bedrog en is in,
        Des Heren zin
        Die blijckt hier breed en wijt.
      
      
        Och gaat niet kwalijk // als vriend omgaet
        Met vriend verralijck // is het ook niet kwaad,
        Dat Broeders nouwe // en zouden zeer
        Mogen betrouwe // den Broeders meer:
        Twelck haastig gheviel,
        Soo't straffen ophiel // 'Twaer tot verniel
        Van 'sHeeren volk en leer.
      
      
        'tWaer een oncieren // der leer Christi,
        Zo qua manieren // te lijden vrij.
        Daar zoudt uit groeien // een kwaad geklank,
        Elk soud' verfoeien // zulk volk eer lank
        Onstichtig het waar // Ia duisterheid zwaar
        Voor licht, ook maar
        Voor reuke vuilen stank.
      
      
        Als wij wel schaffen // op waerheyts plicht,
        Zulk kwaad te straffen // is meerder licht,
        Na 'svromer wensen // en 'sHeeren Wet
        Voor God en mensen // t'antwoorden bet,
        Dan dat men becroont // Oft iet verschoont
        Want t'hem vertoond
        Weerstrijdig Gods opzet.
      
      
        De liefd, om sluiten // nu dit propoost,
        Wat daar is buiten // deugd niet altoost:
        Zij denkt gheen schade // te doen iemand,
        Zij slaat wel gade // eens anders pand,
        Gheen vreugd zij en legt // In enig onrecht,
        De waarheid slecht
        Verblijd al haar verstand.
      
      
        Broeders met enen // wilt doch verstaan
        Men leest in genen // Brief van te zaan
        Gestraft te wezen // wat God mishaeght,
        Maar wel wij lezen // van te vertraagt,
        En hoe dat, dat // Verkoren vat
        Paulus, vrij wat
        Daar over heeft gheklaeght.
      
      
        Prinsen bevroedigh // elk in het gemeen,
        Door een ootmoedig // hert, acht elk een
        Hoger een ander // dan hij zelfs is,
        Onder malkander // gij zult gewis
        In vredig gespan // Ghegordt zijn dan,
        Met den band van
        Rechter volkomen is.
