
    
      
        Eilaas hoe zaan // gaan // hier voorbij
        s’Menschen dagen gelijk in het veld daar wij
        Zien schoon ontdaan // staan // die lely
        Die t’sander daags verdorven zij
        Alzo gheschiet // het, met ons ziet,
        Haast wordt hier iet // niet // door s’doodts kruis,
        Menselijk riet // vliet // boos abuis
        Hier is u stad niet, gij hoort elders t’huys.
      
      
        Merkt hoe den tijd // lijt // hier present
        En alle dingen komen tot den ent
        Hierom met vlijt // zijt // diligent
        Om God te kennen, eerst u zelven kent
        En weest niet prat // op goed noch schat
        Ellendig vat // wat // gij begint
        Een ander stad // dat // zoekt en vint
        Of het is al verlies wat gij hier wind.
      
      
        Wat batet al // dal // berg oft land
        Iae die wereld met haar lusten playsant
        Zo in den val // zal // lijden brand
        De ziele verloren t’costelijck pand
        O Mens vermaard // dwinght uwen aard,
        Want onghebaert // waart // beter zijn
        Dan dat men vaart // daart // doods gepijn
        En schreien zal duren eeuwig termijn.
      
      
        Tegen den dood // noot // wendig raad
        Gebruikt, dat gij niet eeuwig en vergat
        Wt Gods woord bloot // t’groot // weerdig zaad
        De rechte forme des geloofs ontfaet,
        Baard liefden kracht // tot Gods geslacht
        Voor die tweedracht // zacht // loos aenricht
        V wijs bedacht // wacht // met opzicht,
        Blijft op den gront der waarheid vast gesticht.
      
      
        Prinse gemaakt // naakt // hier van slijk,
        Een schepsel den Schepper te zijn gelijk,
        In deuchden waakt // haect // na Gods rijk
        Om bij hem t’huys te zijn doet geen afwijc,
        Houdt in t’ghemoet // als honig vloed
        s’Heeren woord zoet // vroed // en vreest God
        Latet kwaad, doet // goed // na t’ghebodt,
        Gij krijgt uit genaden t’Een nodig lot.
