
    
      
        BLijdschap van mij vlied //
        laat ik mij bereyen,
        Om mijns ziels verdriet //
        droevig te beschreien:
        Want ik in mijn jeugd //
        heb mijn uit-verkoren,
        Al mijn hoogste vreugd //
        en vermaak verloren:
        Berst uit bracke dauw //
        ach! ik smelt in rouw,
        Als ik over-denck met smert,
        Dat de schone Vrouw //
        die den hemel wouw
        Mij te plaatsen in mijn hert,
        Daar ik op had geslagen //
        het opperst van mijn behagen;
        Die ik hiel voor al mijn lust,
        Is in haar jonge dagen //
        mij van de dood ontdragen
        Naar d’on-eyndelijcke rust.
      
      
        Ach! als ik haar deugd //
        haar roem-werde zeden,
        Eerbare gheneught //
        wel-gestelde leden,
        Vriendelijk onthaal //
        lieffelijcke woorden,
        Daar met menigmaal //
        zij mijn ziel bekoorden,
        Bij mij zelfs betracht //
        en op alles acht
        Dat mij is van haar geschied,
        Klaegh’ ik klacht op klacht //
        ja zwem dag en nacht
        In een borne van verdriet,
        Ach! dat de dood zo spoede //
        en eer wij het beyd’ vermoeden,
        V van mij heeft af-gerukt!
        Waarom, o wreede roede //
        hebt gij niet in het woeden
        Mij met een in het graf gedrukt?
      
      
        Wat ik doe oft laat //
        ‘k voed’ noch steeds mijn klachten,
        Mijn beminde gaat //
        nooit uit mijn gedachten,
        Zit ik aan den dis //
        (naar mijn oude wijze)
        Dad’lijck ik haar mis //
        weg dan met de spijze,
        Wil ik daar van dan //
        naar mijn kamer gaan,
        Spreken mijn gedachten weer:
        Hier zag ik haar aan //
        daar plag zij te staan,
        Ginder zat zij bij mij neer,
        En mijn Lief, mijn waarde //
        mijn deugdelijk vermaarde
        Troostertje, leidt nu, helaas!
        In de zwarte aarde //
        bij die haar eerst baarde,
        En is daar den wormen aas.
      
      
        Dan is het oft mijn hert //
        berst in duizend stukken,
        Dan vernieuwt mijn smert //
        met mijn ongelucken;
        Ga ik tot mijn rust //
        dan op het bedde leggen,
        Mijn verlopen lust //
        port mij straks te zeggen:
        Hier genood ik vaak //
        eerbaar mijn vermaak,
        Zoete-lam, in de echt, met u,
        En (o droeve sak!)
        Lief, mijns levens baak,
        Waar berust uwe lichaam nu?
        Ging de echt ons niet bereiden //
        doen het ons te samen leiden,
        Een geluk, en een gevaar?
        Waarom, in het verscheiden //
        hadden wij niet beiden
        Insgelijks ook ene baar?
      
      
        Doch, bedacht ik recht //
        ik en moest niet klagen,
        d’Aerd’ was doch te slecht //
        om dat beeldt te dragen,
        Zij was al te vroom //
        deugdig en Godvruchtig,
        Loffelijk van roem //
        heus, beleeft en tuchtig,
        Dies heeft haar den Heer //
        op-getogen t’eer
        In zijn hemelse gebouw,
        Daar zij haar veel meer //
        zal vermaken weer,
        Als zij hier op de aarde sou,
        Zij leeft in genuchten //
        daar ik hier in duchten,
        Wachtend’ tezamen naar de uur,
        Die niet is t’ontvluchten //
        het einde van mijn zuchten,
        Tot dat God ons tezamen vuur.
