
    
      
        O klagen een paar // En zuchten zwaar
        Met groter vaar // Mijn harde doorboren
        Mijn Lief eerbaar, Dien ik voorwaar,
        Bouen goud zeer klaar Had wtvercoren,
        Valt mijn rebel, O Venus fel //
        V wreed op stel, doet mijn gekwel
        Door dit bevel // blijf ik verloren.
      
      
        Der minnen kracht, Met groter macht
        Compt mijn onzacht // Zeer fel bestrijen
        Al mijn gedacht // Dag ende Nacht,
        Heeft zij voorpacht // Tot allen tien,
        Zij is cortuees // Lustig en prues,
        Zeer Amoreus // En gratieus,
        Haar wezen heus // mijn Hart verblijen.
      
      
        Haar jonge ieucht // Nooit meer ver vreugd
        Sietmen in denkt // Lieflijk floreren
        t'Welck mijn verhuecht, met zoet geneucht
        En tHarte veucht // Om haar te eren,
        In swerlts troon // Spant zij die kroon
        Wel duizend don // Krijg ik te loon
        Zal ik die schoon // moeten ontberen.
      
      
        Zal ik met sant // Des diamant,
        Zeer triumphant // Nu moeten laten
        Venus playsant // Goddin vailgiant
        Der minnen brand // mag ik wel haten.
        Zij is certein // die Liefste grein,
        Dit mijn alleyn // Verheugt ghemeyn
        O vrouwelic grein // Coemt mi te baten.
      
      
        Princesse fiet // Zoet van manier,
        Al mijn dangier // Doet gij weg stake
        V zoet bestier // Vrouwelijk rozier
        Der minnen vier, In het hart doet blaken
        Met v eenpaar // Schoon Lief eerbaar
        Hoop ik voorwaar // Zonder bezwaar,
        Mijn leun klaar, Teynde te maken.
