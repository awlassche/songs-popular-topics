
    
      
        T'Drij enig wezen // het eeuwig ende heilig woord
        Kan genezen // daar de ziel leidt in versmoord
        Wie de deuchden // leiden uit dit tranig dal
        Naar de vreugden // daar de ziel rust hebben zal.
        Die de zonden blusten // waar med' de ziel besproeyt
        Brengt met lusten // in veel rusten
        Waar Gods genade eeuwig vloeyt.
      
      
        Die wilt smakken // het hemels licht van onzen al
        En gheraecken // daar men noot eynt vinden zal
        Noot u zinnen // zetten wilt op t'swerels pracht
        Want van binnen // allenskens dan u ziel versmacht
        En wilt dan noot liegen // en u van d'onsuyver zond
        In slaap wiegen // en bedriegen
        Want zij trekt de ziel tot grond.
      
      
        Wilt u dan keren // tot God den Heer die t'al beooght
        En vermeren // zijnen naam ende verhoogd
        Die geschapen // heeft den mens tot zijne vreugd,
        Wilt die dan rapen // uit de waar en zuiver denkt
        Verlaat s'werels quaelen // Gods glori dan hebben zult
        Die neer kwam dalen // om te betalen
        Met de dood ons grove schuld.
