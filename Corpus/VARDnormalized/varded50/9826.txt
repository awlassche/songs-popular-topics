
    
      
        SChoon duyvinne // die rein van sinne
        Mij meer bemint dan uws zelfs leven
        Want gij inne // het vier mijner minne
        Verslonden zijt, van hier gedreven,
        Door mijnen geest // uit 'tAerdtsch foreest,
        Tot bij mij in het Hemelse wezen,
        In mijn slaapkamer uytghelesen,
        Rust u hier dan // tot dat voorbij
        De hitte van // den middag zij.
      
      
        Zonder scheiden // wilt u vermeyden
        Met mij, en vast verenigd blijven,
        Gheen verleiden // hoe zij arbeiden,
        V deren mag, gij zult beklijven,
        En vreest niet want // elken vijand
        Zal ik met krachte neder vellen,
        En leggen tot ons voerschabellen,
        Dan werd altoost // ghewisselt dijn
        Droefheid in troost, haar vreugd in pijn.
      
      
        Openbaren // met mijn Heirscharen
        Zal ik eer lank, tot u verblijden,
        Den zondaren // tot een vervaren,
        Die mijn aanschijn zullen vermijden,
        Vluchten beducht // zonder ontvlucht
        Werden benout van herten sprekende',
        Och ziet, hoe lief en werd gerekend
        Is zij, die wij // hielden voor dwaas,
        Zij is nu blij // wij droef eilaas.
      
      
        O Princiere // blijft goedertiere
        Lijdtsamigh tot behoudt der zielen,
        Met den viere // uwe bloedtgiere
        Vervolgers rijk zal ik vernielen,
        Den nieuwen Wijn // in het rijke mijn
        Zal u herte vreugdig ontwaken,
        En uwen mond vol lakens maken,
        Bij mij u steed' // eeuwig zijn zal,
        Troost u hier meed, in het aardse dal.
