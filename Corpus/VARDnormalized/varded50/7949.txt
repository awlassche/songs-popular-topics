
    
      
        Last ghinck ik in de Mei spaceren /
        Wandelen met lusten // fijn
        Waar door mijn hert moest confirmeren
        Ja geheel in rusten // zijn:
        Vrou Venus karen excellent /
        Sach ik bij paren / daar ontrent /
        Hand aan hand // zeer playsant /
        Danssen en springen /
        Wentelen in het groene gras /
        Zout een kwant // syn verstand
        Konnen bedwingen /
        Die het gewend te doene was.
      
      
        Terstond kwamen zij voort gedrongen /
        Met Herp'n / Luiten / Velen goed /
        Zij dansten / spelden ende zongen /
        Sulcx 'tGhehoor ontsluiten doet.
        Musica schoon was daar bekent /
        Elk hiel den toon op het Instrument /
        Delicaat / op zijn maet
        Seere manierig /
        Pas a pas voort danssende /
        Nijders kwaad // wordt versmaad
        Maar goedertierig
        Met Blomkens beschanssende.
      
      
        Des Maechdekens lustig van leven
        Stelden mij de Rozen // bij /
        Waarom soud'ic gheen vreugd aankleven
        Die tot zulks verkoosen // zij:
        Veel Blomkens ook zuiver en klaar /
        Lieflijk van Reuk brochten zij daar /
        Sulcken vreugd // mij verheugt
        Zeer boven maten
        En't Jonk gezelschap schoon / eerbaar /
        tSa gij Jeugd // maakt gheneucht
        Met zoet praten /
        tIs couragie voor en naar.
      
      
        Terstond kwamen zij zeer geladen
        Om den last te boeten // meer
        Met tarten / Vladen en Gebraden /
        Wijn om druk verzoeten // zeer
        Ik dacht ik haak / na sulcken geur /
        Doen werd den Smaak noch Governeur /
        Heus en blij // willen wij
        Nu Banketteren /
        Sprak een van des Maagden fier /
        Hout v vrij // wie sou dij
        Nu mogen deren /
        Spijt al die het benijden hier.
      
      
        Als ik sach al des vreugd voor ogen /
        des Hertsen bloed verwermden // zij
        Terstond kwamen zij toeghevlooghen
        Helsden / kusten n'armden // mij
        Ik hiel mij koel op dit termijn
        Maar het Gevoel dede mij pijn
        Die nu mocht // als hij docht
        Naar zijnen wille
        In Venus hof graven ziet
        Sulcken tocht // mocht gewrocht
        Syn al stille /
        Maar elk heeft de gracy niet.
