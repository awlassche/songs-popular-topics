
    
      
        O Machtig, eeuwig God!
        Die door u groot gebod
        De Engelen geschapen hebt,
        En die den mens noch heden schept.
        'k Wil schuilen // 'k wil kuilen // mij onder de aerd',
        Want ik en ben niet waard
        Mijn ogen // nahoogen
        Te lichten op tot U, die mij bewaard.
      
      
        Ik weet dat Gij zijt Een,
        Een Godheid, anders geen,
        Personen Drie altijd geweest,
        God Vader, Soon, God Heyl'gen Geest:
        Wien loven // daar boven // met grote praal,
        De Eng'len altemaal,
        Wien wensen // de mensen
        Te komen bij, in des Hemels hoge zaal.
      
      
        God Vader! ik beken,
        Dat ik niet waardig ben
        Mijn hoofd te lichten tot U op,
        Die woont in des Hemels hoogsten top.
        Mijn zonden // mijn wonden // doen mij dit aan,
        Dat ik het niet derf bestaan:
        Eylacy! // uwe gracy
        Heb ik verzuimt, ik ben met sond' bela'en.
      
      
        Dan evenwel ik moet,
        En wil, voldoen de boet
        Die ik verdiend heb door mijn kwaad;
        Dan ik bidd' U, bermhertich slaat.
        God Vader // weldader // vergeeft de schuld,
        Waar me'e ik ben vervult:
        'k Zal dragen // de slagen
        Die Gij mij geeft, met ootmoed, en verduld.
      
      
        God Soon! tot U ik kom,
        Wien ik mijn Jesum noom,
        Die mij verlost hebt door u bloed,
        Die mij wilt schenken het eeuwig goed.
        Ik jachje // ik trachje // die grote eer
        Te geven, Jesu! we'er,
        Gods Zone // 'k wil loone
        Aan U mijn God, dat Gij mij mindt zo teer.
      
      
        Ik weet dat Gij, Gods Soon,
        Begeert geen and're loon,
        Dan dat ik mij van zonden wacht;
        Dit is den loon daar Gij na tracht.
        Ontbinden // verslinden // wil ik de sond',
        Van nu op dezer stond.
        Vertreden // (het is reden)
        Wil ik het kwaad, het welk heeft mijn ziel gewondt.
      
      
        Ik bidd' U, Heyl'gen Geest!
        Mijn zondig ziel geneest;
        U licht ontsteekt in mijn verstand,
        In het hart uwe liefde stort, en plant.
        Laat dalen // uwe stralen // in mijne ziel,
        Och of die nooit verviel!
        Ontfermen // erbermen
        Wilt over mij, die omga als een wiel.
      
      
        O des levens schoon Fontein!
        Maakt mij in deughden rein:
        Neemt weg de zonden uit mijn hart,
        God Trooster! neemt weg all' mijn smart.
        Genadig // gestadig // wilt zijn aan mij,
        Komt Heyl'gen Geest mij bij.
        Gods minne // van binnen
        Volmaakt mijn hart, en maakt dat eeuwig blij.
      
      
        O hoogh' Drievuldigheid!
        Mijn hartje bitter schreyt,
        Om dat'et U vergramt heeft zeer,
        Die met U Drien zijt enen Heer.
        'k Wil wezen // na dezen // uwe trouwen vrind,
        Aan U mij vast verbindt,
        Komt lonen // komt wonen,
        Drievuldigheid! in het hart dat U bemindt.
