
    
      
        SChoon en playsant // aan elken kant
        Sietmen des Meis kruiden groeien // Bomen bloeien,
        Lief triumphant // wilt met verstand
        U nu tot liefde spoeyen // twist verfoeien,
        Want men ziet merkelijk certein
        Dat menig zoet vogelken klein
        Met liefde bij een vergaren,
        Laat ons och te samen paren, mijt bezwaren.
      
      
        Aanziet het kruid // dat nu komt uit
        Op dat wij zouden sporen // vreugd t'orboren
        Maar het zoet geluid // hetwelk uit u spruit
        Dat gaat mij meer bekoren // wilt doch horen
        Na uwen dienaar te aller tijd,
        Die om u zijnen tijd verslijt,
        Op dat hij u sou behagen,
        Wilt hem rein liefde dragen // te allen dagen.
      
      
        Rein onbevlekt // zijt gij perfect
        Als Aurora zeer koen // komt in het groen,
        V zal correct // worden ontdekt
        Dat ik heb troost van doen // in het saysoen,
        Staat gij van Flore delicaat,
        Hero gij veer te boven gaat,
        In schoonheid en practijcken
        Wilt mij geenszins nu bezwijken // trou laat blijken
      
      
        Aenschout met vreugd // Mijn jonge jeugd
        Die met hert en zinnen // gaat beminnen
        Op eer en deugd // maakt mij verheugt,
        Zo zal ik vreugd // gewinnen // druk bezinnen
        Want Cupido heeft mij doorwont,
        Zo dat ik nu ben onghesont,
        En altijd zal moeten wezen
        Tot dat gij mij lief geprezen // komt genezen.
      
      
        Prins dochter prent // zeer excellent
        In u dit slecht ontvouwen // laat ik schouwen,
        Dat liefd' zeer jent // is het fondament,
        Daar ik vast mag op bouwen // en betrouwen,
        Op u woordekens zoet en fray
        Die ik verwacht nu in des May
        Van u om te verschonen,
        Wilt doch liefd' met liefde lonen // mijn liefde kronen.
