
    
      
        Komt ter feesten // zuiver gheesten
        Gods dienaren groot,
        Die daar boven // in Gods hoven
        Zijnen lof verbreyt // tot alder tijd.
      
      
        Helpt ons zingen // grote dingen
        Van een zuiver Maagd,
        Die ons’ leven // heeft gegeven
        Brengende hier voort // Gods eeuwig woord.
      
      
        Hoog geprezen // uytghelesen
        Maagd en moeder rein,
        Naar u wenscyhen // alle mensen,
        V zoekt groot en klein // in d’aerdsche plein.
      
      
        Laat doch dalen // uwe stralen
        Der bermherticheyt,
        Wilt u ogen // naar ons bogen
        Keert van ons het kwaad // o toeverlaat.
      
      
        Heel deurwonden // deur de zonden,
        Maakt ons al ghesont
        En u daden // vol ghenaeden
        Toont ons al ghemeyn // als medicijn.
      
      
        Al ons krachten // en gedachten
        Zijn tot uwen lof,
        Elk een prijst // en bewijst
        V allerhoogste eer // rein maghet teer.
      
      
        In de zure // laatste ure
        Als men sterven moet
        Bij ons staat // metter daad
        En neemt in u behoed // Maria zoet.
