
    
      
        De liefde blent // gaat mij bekent
        Met groot torment // Nu zwaarlijk plagen,
        waar ik mij went // Zijnde absent
        van u lief jent // Zo moet ik klagen,
        In verdriet // Door liefde hiet men mij ziet,
        gaan te niet // En gans verjagen
        dus zuiver grein // Acht u zo klein
        t’aensien certein // Mijn liefde rein
        wilt mijn gheweyn // Doch helpen dragen.
      
      
        Want u licht klaar // wezen eerbaar
        Doet u dienaar // in liefden derven‘
        V sanck eenpaar, uytspraeck voorwaar
        Brengt mij bezwaar // en doet mij sneven
        V bestier // brengt mij hier, veel dangier,
        Dat mij schier // rust gaat bewegen.
        Dies mij doch jont // Lief u verbond
        Zo zal terstond // mijn hert doorwondt
        Wezen ghesont // in vreugden leven.
      
      
        Goddin u keert // en u verneert,
        Hem die u eert // laat troost verwerven
        Liefde useert // qua raad afkeert.
        Na u begeert // suldy vreugd erven
        Des nijders macht // nu veracht
        Want haar kracht // alleen tracht’V te bederven
        dies nu verlaat // Door mijnen raad
        des valsen praat // T’gheschiet door haat
        en mij bystaet, V druk zal sterven.
      
      
        Prinses zoet // hart lijf en goed
        In zuur in zoet // zal u standig eren,
        Zo gij wel doet // Lief met ootmoed
        Dat gij u spoed // mij te salveren
        Reine vrucht // en geneucht, zal u jeugd
        Door mijne denkt // altijd useren
        Hartzeer en pijn // nijders fenijn
        Met valsen schijn // zal van u zijn
        Gij lief mijn // troost gaat vermeren.
