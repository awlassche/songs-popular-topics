
    
      
        GOds wijsheid slechtich // voor s'wee-relts oog,
        Gedaald oprechtig // van bouen hoog,
        Hoog disputeren niet en behoeft:
        Zulk argueren den geest bedroeft,
        Terecht simpel verstand
        In de waarheid gheplant,
        Wel begrijpen kant,
        T'wordt door den Geest gheproeft.
      
      
        Al wat niet geestig // en smaakt gezond,
        Dat moet tempeestich // als ongegrond
        Op het zand verdwijnen // neuen den steen,
        Donker bruwijnen  // en mogen gheen,
        De waarheid perfeckt
        Houden bedekt,
        Oft schoon vertrekt,
        Zij verwint in het gemeen.
      
      
        S'Heeren gemeente // met hert en lust,
        In goe vereente // staat vast gerust
        Op de rein waarheid // als een pilaar,
        Van suyuer klaarheid // die hier noch daar
        Niet henen en draeyt
        Wat wind dat waeyt,
        In haar ghesaeyt
        Gods woord, versterckts' eenpaar.
      
      
        Broeders eendrachtig // elk in zijn graat,
        Laat ons aandachtig // na Petrus raad,
        Onder den heiden // gelijk betaamt,
        Goe wandel leiden // dat elk hem schaamt
        Tot beterschap
        Die met gesnap
        Door achterklap
        Ons hebben zal gheblaemt.
      
      
        Terecht vromer leun // Van Abel aan,
        Staat al voorschreuen // tot ons vermaan
        Schoeriem, oft pleytken // van enig draad,
        T'bleetende geytken // daar ook al staat:
        Niet schuldig en zijt
        Dan liefden vlijt,
        Die t'gheener tijd
        Den Naasten en doet kwaad.
      
      
        In des que tijden // voorhanden bloot,
        Laat ons vermijden // in cleen oft groot,
        Dat met bezinnen // zelfs niemand wordt
        Buiten oft binnen // van ons verkort
        Dat zoete lucht
        Van goed gherucht,
        Door goede vrucht
        Van ons zij uitgestort.
      
      
        T'zy vremd' oft vrienden // wie dat ons borght,
        Zo wel wij dienden // te zijn bezorgt:
        Want ziet den name // den welken wij
        Aanroepen t'same // dezen wordt vrij
        Voor borge betrout,
        En wie hier op bout,
        Hoe zal doch stout
        Dezen te cord doen hij.
      
      
        Wie en zal pogen // om laten niet
        Enen zo hogen // borg in het verdriet,
        Oft zo in schanden // dat men gestoord
        Daar door versmaden, zijnen Naam hoort,
        En t'volck vermaard
        Van zijner aard,
        Iae recht als waart
        Bedrieglijk in zijn woord.
      
      
        Die dus laat steken // zijn borg in last,
        Godloze treken // doet hij, t'is vast:
        Ook staat gheschreuen // hoe dat in al
        T'slecht ouerbleuen // Israels getal,
        Gheen leughenen doch,
        Iae tonge noch,
        Doende bedrog,
        Gevonden zijn en zal.
      
      
        Dus moet den Tempel // Gods staan gesticht,
        Christi exempel, is klaar en licht,
        Te genen stonden // was nooit onvroed
        Bedrog gevonden // in zijn mond zoet:
        Wie valsheid bedrijft,
        Oft leughen verstijft,
        Niet bij hem blijft,
        Maar uit zijn huis hij moet.
      
      
        Niemand bedriegen // gij niet en zult,
        Sprak die van liegen // nooit hadde schuld:
        Ziet, wilt aanschouwen // een Israelijt
        Oprecht vol trouwen // daar genen tijd,
        T'zy meer oft min
        Bedrog en is in,
        Des Heren zin
        Die blijckt hier breed en wijt.
      
      
        Och gaat niet kwalijk // als vriend omgaet,
        Met vriend verralijck // is het ook niet kwaad:
        Dat Broeders nouwe // en zouden zeer
        Mogen betrouwe // den Broeders meer:
        T'welck haastig gheviel,
        Soo't straffen ophiel,
        T'waer tot verniel
        Van s'Heeren volken leer.
      
      
        T'waer een oncieren // der leer Christy,
        Zo qua manieren // te lijden vrij:
        Daar soud' uitgroeien // een kwaad geklank
        Elk soud' verfoeien // zulk volk eer lank
        Ontstichtich het waar,
        Iae duisterheid zwaar
        Voor licht, ook maar
        Voor reuke vuilen stank.
      
      
        Als wij wel schaffen, op waerheyts plicht,
        Zulk kwaad te straffen // is meerder licht.
        Na s'vromer wensen // en s'Heeren Wet,
        Voor God en mensen // t'antwoorden bet,
        Dan datment becroont,
        Oft iet verschoont,
        Want't hem vertoond
        Weerstrijdig Gods opzet.
      
      
        De Liefd' om sluiten // nu dit propoost,
        Wat daar is buiten // denkt niet altoost:
        Zij denkt gheen schade // te doen iemand,
        Zij slaat wel gade // eens anders pand:
        Gheen vreugd zij en lecht
        In enig onrecht,
        De waarheid slecht
        Verblijd al haar verstand.
      
      
        Broeders met enen // wilt noch verstaan,
        Men leest in genen // Brief van te zaan
        Gestraft te wezen // wat God mishaeght,
        Maar wel wij lezen // van te vertraagt,
        En hoe dat, dat
        Verkoren vat
        Paulus, vrij wat
        Daar ouer heeft gheclaeght.
      
      
        Prinsen bevroedich // elk in het gemeen
        Door een ootmoedig // hert, acht elk een
        Hoger een ander, // dan hij zelf is,
        Onder malkander // gij zult gewis
        In vredig gespan
        Ghegordt zijn dan,
        Met den band van
        Rechter volcomenis.
