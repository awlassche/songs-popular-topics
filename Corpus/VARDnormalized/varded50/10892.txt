
    
      
        Bloeyende jeugd, Edelen aard der vrouwen
        Wanneer gij meucht // zo denkt, toch eens om mij
        Maakt mij verheugt, deur u goedheid vol trouwen
        Door sulcken denkt // zal ik dan wezen blij
        Een zoete melody // ontvang ik dan eenpaar,
        Den troost eender jonkvrouwen zoet
        Mij langen tijd onthouwen // doet
        Deur hope ickt vertrouwen // moet
        Met dezen Nieuwen Iaer.
      
      
        Menige reis // Lief komt gij mij te voren
        Met zwaar ghepeys // ben ik dicwils gekweld
        Der liefden eis // doet mij dit al oorboren,
        Waar door mijn vleys // menigmaal is ontstelt
        Wat hier wordt vertelt // ghebeur alzo voorwaar
        Daar jonkvrouwen // met eren, zijn
        Machmen trouwen // verkeren // zijn
        Wilt onthouwen // dit leren // mijn
        Met dezen nieuwen Iaer
      
      
        Mocht ik verstaan // schoon lief wt uwen monde
        Zulk een vermaan // daar ik na heb gewacht
        Dan waar gedaan // in alzo korten stonde,
        mijn droefheid zaan // die ik heb dag en nacht,
        V liefde heeft gewracht, in mij een lijden zwaar
        schoon Bloemen mijns gedachtig // zijt
        Hoort hoe v dienaar klachtig // lijt
        Toont hem vriendschap warachtich // wijt
        Met deze nieuwen Iaer
      
      
        Maechdelic zaad // wel versierd met veel deuchden
        Komt mij te baat // en geeft mij doch confoort,
        denkt op den praat // die wi tsamen in vreugden
        Vroeg ende laat // hadden zonder discoord,
        Laat ons ongestoord // worden een vrientlick paar
        Zo u persoon // mij de hand // biet,
        En alst met schoon // verstand // gheschiet,
        Alsulcken loon // verplant // ik niet,
        Met dezen nieuwen Iaer.
      
      
        Neemt toch in dank // den sanc // van mijn ghedichte
        Schier met bedwanc // zo schanc // ik u dit liet:
        deur liefde frank // den ganc // viel mij nz lichte
        Met zinnen krank // d’aenvanc // is zo geschiet
        Laat gij mij tot verdriet // Edel Prinses eerbaar
        Zo lijd’ ik dan van rouwe // pijn
        Wilt gij mij niet getrouwe // zijn
        Gedenkt mij schoon jonkvrouwe // fijn
        Met dezen nieuwen Iaer.
