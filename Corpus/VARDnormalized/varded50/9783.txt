
    
      
        ONder veel Meyskens die men vindt,
        Is een haars Moeders liefste kind,
        Mijn duive bemint mijn wederpaar
        Excellent // Excellent
        In mijn herte gheprent,
        Is zij bekent // en ik ben haar
        Een fundament // Onwankelbaar,
        Tot 'swereldts endt.
      
      
        Ik ben haar tot een open deur,
        Een troost in lijden en getreur,
        Een vyerich meur // dag ende nacht,
        In den noot, In den noot
        Een beschutsel groot,
        De tweede dood // en heeft gheen macht,
        Met haar aanstoot // aan mijn geslacht
        Te doen exploot.
      
      
        Ik en laat haar gheen Weze vrij,
        Met mijnen Geest blijf ik haar bij,
        Tot troost heeft zij // mijn Woord ontfaen,
        En het verbond // En het verbond
        Van mijnen mond,
        Zal het eeuwigher stond // wel blijven staan,
        Maar bergen rond // zullen vergaan,
        Al in den grond.
      
      
        Ik en heb haar met geld noch goed
        Gekocht, maar met mijn dierbaar bloed
        Blijven zij moet // mij ingelijft,
        Wt mijn hand // Wt mijn hand
        En zal haar niemand
        Getrekken, want // Zo lange zij blijft
        In mijn verstand // Als Boom beclijft
        Zij, wel gheplant.
      
      
        Vergeet ik Ierusalem fijn,
        Mijn rechter hand moet vergeten zijn,
        Mijn tong aan mijn // mond kleve ziet,
        En al waart // En al waart
        Dat tegen den aard,
        Haar kind gebaard // Een Moeder verliet,
        Syon vermaard // En zal ik niet
        Laten bezwaard.
      
      
        Mijn Princesse, mijn lieve Bruid,
        Als taruw hier onder onkruid
        Zal zijn daar uit // vergadert, en
        Boven dien, boven dien
        In heerschappien,
        Zal zij mij zien // gelijk ik ben,
        En haar verblien // Eeuwig, Amen:
        Dat zal gezien.
