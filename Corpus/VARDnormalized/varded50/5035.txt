
    
      
        In ‘s Heren Naam moet ik beginnen,
        Een troostbaar Lied voor uwe zinnen,
        Waar bij // dat gij,
        Door dit beschenken // ons meucht gedenken.
      
      
        Al zijt gij wijt van ons gezeten,
        Nochtans en zijt gij niet vergeten,
        Veel min // noch in,
        Des Heren ogen // die ‘het al vermogen.
      
      
        Nu ‘het gheen dat gij hebt te aanschouwen,
        Dat is ‘het exempel van de Vrouwen,
        Die God // tot lot
        Hebben verkoren // zijn wil te horen.
      
      
        Neemt eerst aan Sara deze Lere,
        Die haren Man genoemd heeft Here,
        Haar wil // was stil
        Overgegeven // in haar Mans leven.
      
      
        Een Vrouw kan door ootmoedicheden,
        Met haren Man leven in vreden,
        Niet zij // maar hij,
        Komt toe ‘het regieren // en al ‘het bestieren.
      
      
        Konden de Vrouwen dit bevaten,
        Het soud’ haar dik veel onrust baten,
        En ‘het sou // de trouw,
        Met liefd’ bespoeyen // den Geest doen bloeien.
      
      
        Elkanas Wijf leert hoe aandachtig,
        Een Vrouw haar nood moet wezen klachtig,
        Aan hem // die stem,
        Noch woord behoevet // als ‘het hert bedroevet.
      
      
        Neerstich en in armoed’ geduldig,
        Bewees haar Hanna eenvuldich,
        Zij was // te pas,
        En wel te vreden // in kleynicheden,
      
      
        Koningen zijn het hert gewonnen,
        Van Vrouwen, als wij lezen konnen,
        Zo heeft // beleeft,
        Hester verkregen // haar Mans bewegen.
      
      
        Abigail was een wijze Vrouwe,
        Zij sweech in tijd van druk en rouwe,
        Met goed // doen vroed,
        Kon zij ‘het gheresen // onheil genezen.
      
      
        Rein was Maria in haar leven,
        Deur ghehoorsamich ondergeven,
        In vlucht // gezucht,
        Boog zij haar herte // tot pijn en smerte.
      
      
        Een voorbeeld van Godvruchticheden,
        Is Priscilla, wiens kloeke reden,
        Verwon // en kon,
        Leren manierig // Apollo vierich.
      
      
        Laat des exempels u behagen,
        En wilt u zelven daar na dragen,
        De Kroon // en ‘het loon,
        Van deze vromen // zult gij bekomen.
      
      
        Sijt God bevolen nu in vreden,
        Draagt ons ghemeynschap in gebeden,
        Dit deel // blijft heel,
        En ongescheiden // in eeuwicheyden.
