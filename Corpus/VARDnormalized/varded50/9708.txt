
    
      
        HErders eerbaar // alle die daar
        Met arbeid zwaar // nu nemen waar
        Die schaer // der Christen Schapen,
        Onder wolven in het veld // gesteld,
        Hoe nijdig dat ze gapen,
        En vreest niet haar geweld.
      
      
        En vreest niet, want // op geen drijf-zant
        En staat gheplant // Gods huis vailjant
        Zijn hand // krachtig van daden,
        Ook nemmermeer verkort // en wordt,
        Den vloed zijn der ghenaden
        Hij vriendelijk uytstort.
      
      
        Hij sort zijn bloed // in overvloed,
        Dus Herders vroed // hebt goeden moet
        Al boet // die oude slange
        Nu met discoordigh zaad // veel kwaad,
        En zijt daarom niet bange,
        Gods woord zij uwen raad.
      
      
        Met Gods woord raad // slaat daarop staat,
        Hoewel veel praat // nu ommegaet,
        Van smaad // achterklap laster,
        Iae roverye van faam // ende naam,
        En steunt doch maar te vaster
        Op 'sHeeren woord bekwaam.
      
      
        Bekwaam is 'twoort // des Heren hoort,
        Want het verstoort // alle discoord,
        Wat voort // verheven tegen
        De kennisse Gods komt // vermomt,
        En laat u niet bewegen
        Met redenen verblomt.
      
      
        Redenen kloek // buiten het boek,
        Werpt in den hoek // hoe schoon verzoek,
        Den vloek // menig exempel
        Leert, af noch toe te doen // zo koen,
        Tabernakel en Tempel
        Moet zijn na Gods fatsoen.
      
      
        God alleen wijs // zij eeuwig prijs,
        Al zijn advijs // is zeer propijs
        Hoe kis // de Mensen ramen,
        Al haar wijsheid voor God // is zot,
        Gods dwaasheid maakt te samen
        Al 'sMenschen wijsheid bot.
      
      
        des Mensen wijsheid // van God afleyt,
        Dus die bereid // nu 'tkudde weidt,
        Arbeid // in het woord des Heren:
        Houdt vast daar over, tis // ghwis,
        Zo zult gij heylsaem leren,
        En keren // ergernis.
      
      
        Ergernis zien, wij veel gezien,
        Maar wee voor wien // men moet hen-lien
        Afsnien // en mijen als Kanker,
        Haar woord, eermen versint // verslint,
        In deuchden wordt hij krancker,
        Die kwaad geklap bemint.
      
      
        Oorlof ydoon // Herders, de kroon
        Der eren schoon // werd uwen loon,
        Den soon // werd zoet om horen,
        Ay gij getrouwe knecht // oprecht,
        Mijn rijk is u te voren,
        Gij die wettelijk vecht.
