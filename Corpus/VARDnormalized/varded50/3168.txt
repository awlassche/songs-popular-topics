
    
      
        God verlaat zijn Dienaars niet,
        Schoon hij haar laat proeven veel verdriet:
        Na den druk // geeft hij haar weer geluk, Heil en lust // troost en
        Heil en lust // troost en zegen, En de Hemelrust.
        Op het Pinksterfeest,, bleeck'et meest // aan de Geest,
        Aan de taal // van'er juichen en verhaal;
        het Harde sprong // en de tong // vrolijk
        zong Godes lof // met een stijl en hoog achtbre stof.
      
      
        O! wat vrolijkheid en vreugd,
        Wat vermaak heeft haar de ziel verheugt,
        Toen de wind // over haar kwam gheswindt,
        Kwam gedaald // en 'er zielen heilig heeft bestraelt.
        het Volk zeer verbaasd //heeft geraest // en geblaest,
        En gezeid // het is de Wijn die haar verleid:
        Maar een deel // waren veel // in het geheel
        Zeer bevreesd // door het gehoor van'er grote Geest.
      
      
        Maar zij zeiden overluid,
        het Is gheen drank die onze lipp' ontsluyt;
        het Is geen most // die het verstand maakt verbost,
        het Is gheen sap // van de Druiven, maar het is wetenschap,
        Die Godes kracht // en zijn macht // heeft gebracht
        In ons hert // dat'et vol van tongen werd;
        God is zoet // God is goed // die dit doet
        Tot u baat // na zijn Wil en Al-wijzen raad.
      
      
        Daar vloeyt in uitheemse Taal
        Voor het volk een Goddelijk verhaal,
        Dat'er ziel // en 'er hart wel beviel,
        En de staan // dat ze vraagden, acht! wat dient gedaan?
        Zij zeiden: doet // in ootmoed // recht boet,
        Valt den Heer // voor zijn heyl'ghe Voeten neer,
        En dan voort // alst behoort // op zijn woord
        Laat dan toe // damen u't Heilig Doopsel doe.
      
      
        Ieder, als dit was gezeid,
        Was terstond tot dezen dienst bereid,
        En't getal // groeyde strak dat 'et al
        Kwam terstond // op drie duizend/ die in het Heil-verbond
        Waren ghetre'en // onder een // wel te vre'en,
        Wel gerust // met een ziel en harten-lust.
        Heilig God,, leidt ons tot // u Gebod
        Ook met vreugd // dat wij zo blijven in de Deugd.
