
    
      
        O Grote Catharyn!
        Lof moet U altijd zijn,
        Die op de aerd' // hebt bewaard // uwe leden;
        Eer zijt Gij al-tijd waard.
        Door des harten reinigheid
        Hiel des lichaams dartelheid,
        In uwe strijdt // Gij altijd // onder reden:
        Wat loon is U bereid!
      
      
        G' hebt zo, O reine Maagd!
        God uwen Heer behaagt,
        Dat den Heer ,, U die eer // heeft bewezen,
        Dat na uwe Trouw Hij vraagt.
        Gods Soon heeft U getrouwd
        Met enen ring van goud:
        Uwe hand // kreeg een band // die voor dezen
        Nooit oog had aengeschout.
      
      
        Hoe groot was uwe vreugd,
        Hoe wel waart Gij verneught
        Als dien ring // Gij ontving // Catharyne!
        Van God, die het hert verheughdt!
        Toen wiert G' onststeken meer
        In liefde tot uwe Heer,
        Die door druk // tot geluk, en door pijnen
        U broght tot Martyrs eer.
      
      
        'G hebt Martijrs-kroon ontfaen
        Als Gij waart heen gegaan
        Om zijn naam // en zijn faam // te verbreiden,
        Gij storten niet een traan.
        Gij kwaamt bij Maximyn,
        Een man vol sond's fenijn,
        Een Tyran // die den ban // U bereiden,
        En dreyghden grote pijn.
      
      
        Hij uit zijn Rijk ontbood
        In wijsheid Mannen groot
        Vijf-mael tien // (g'lijck belie'n // ons de Schrijvers)
        En hij bij haar U nood.
        Gij komt blijmoedig in;
        Van God neemt G' U begin,
        En Gij brocht // (g'lijck Gij zocht //) de Bedrijgers,
        Tot Jesu Christi min.
      
      
        Gij hebt haar all' geleerd,
        Gij hebt haar all' bekeert:
        Zo dat zij // wonder blij // het Christen-leven,
        En Christum hebben g' eert.
        Zij leden voor Hem schand',
        Zij levend' zijn verbrand;
        Maar bij God // O wat lot! // hoog verheven
        Zijnz' nu in des hemels land.
      
      
        Bij deze is verzelt,
        In strijdt, en des Hemels veld,
        des Keizers Vrouw // die met rouw // haarder zonden,
        Verdrough des doods geweld.
        Ja ook des Legers Hoofd,
        Porfier, in God gelooft;
        Hij aenkent // zijn ellend' // zijne wonden,
        Hij wordt van ziel berooft.
      
      
        Toen wierdt verbitterd zeer
        Den Keizer als een Beer,
        Hij verboodt // dat met brood // niemans spijzen
        U mocht na des niet meer.
        Maar te vergeefs; want ziet!
        Den Hemel U versiet;
        God U spijst // Gij Hem prijst // het goeder wijze,
        Hij helpt U uit het verdriet.
      
      
        Na dat G' had meer gele'en,
        Om dat G' had God bele'en,
        Wierdt een rad // boven dat // afgevaerdicht,
        Maar het brak door uwe gebe'en.
        Toen heeft het zwart u hoofd
        Van het lichaam afgeklooft.
        Deze eer // heeft den Heer // U gewaerdicht,
        Wien Gij nu eeuwig looft.
      
      
        Maar slaat uit des Hemels boog
        Op ons (Catryn) u oog:
        Maakt dat wij // eeuwig blij // God aanschouwen
        Hier namaals ook om-hoogh:
        Op dat wij van het geschreeuw,
        En van den helschen Leeuw
        Steeds bevrijd // zijn verblijd // en behouwen
        God loven te allen eeuw.
