
    
      
        Ter wereld is geen meer ellent // dan in armoed te wezen,
        Maar als men die ten besten went // zo wordt de ziel genezen,
        Wanneer men wil bemerken // hoe zij het hert kan sterken,
        Hoe zij ‘het gemoet // noch voordeel doet,
        Meer dan den overvloed.
      
      
        Een arme die Godvrugtig is // en zijns gemoets een weter,
        Die vint het licht in duisternis // en spijze bij den eter,
        Hij suycht Honig, uit stenen // en blijschap uit het wenen,
        Hebbende niet // hij volheid ziet,
        ‘het Welk deur Gods kracht gheschiet.
      
      
        Ten eersten heeft hij geenene noot // om ampten te bedienen,
        Burg’meester, ofte schepen groot // zal niemand op hem mienen
        Hij hoeft hem niet te moeien // met Peerden, of met Koeien,
        Met dijk of dam // maar als een lam //
        mag hij hem houden tam.
      
      
        Wanneer den rijken wordt belast // met tafelen vol eten,
        Zo blijft hij in zijn hutjen vast // met stilheid neer gezeten,
        Hij mag wel gaan spanceren // als ander grote Heren,
        Op die rechtbank // tegen haar dank //
        wel zitten uren lank.
      
      
        Komter een oorlog in het land // hij heeft niet te verliezen,
        Oft zo daar schepen zijn gestrant // hij staat zonder verkiesen,
        Om schattinge te geven // is het hem gelijke even,
        Die niet en heeft // ook niet en geeft //
        en evenwel hij leeft.
      
      
        Komt hij in grote steden gaan // men zal hem niet verdragen,
        De beurs mag hij wellaten staan // daar Kooplieden haar plagen,
        Wandelt hij op de velden // Hij is bekommert zelden
        Met land of zand // maar zijn verstand //
        dat zet hij na zijn hand.
      
      
        Van haat en nijt is hij bevrijd // in erf-goeden te delen,
        Zijn schonen tijd hij niet en slijt // met jaren lank krakelen,
        Hij mag zijn tijd besteden // in aandacht, en gebeden,
        Van grote moeyt // is hij besnoeyt //
        op dat den geest herbloeyt.
      
      
        ‘het Geld brengt hem niet dan droefheid in, Met dag en nacht in peizen,
        Hoe hij ‘het zal brengen tot gewin // zijn vreugd begint te deysen,
        Daar anderszins met zingen // en ander zoete dingen
        Hij op zijn werk // verheugt hem sterk,
        Gelijk als in een Kerk.
      
      
        Als d’rijcken een huis bouwen zal // daar is zo veel te stellen,
        Dat hij mag kijken boven al // daar gaat hij hem med’ kwellen
        Schier boven zijn vermogen // dient hij den lust der ogen,
        Den armen recht // wel overlegt //
        hoe hij ‘het zal maken slecht.
      
      
        Ziet hoe den rijken is bezweet // met varen ende vletten,
        Speel-jachten, en Karossen breet // hem niet dan moeite zetten,
        Veel onbetemde lusten // houden hem in onrusten,
        Sulck’ heerschappij // is slavernij //
        al scheen het noch zo vrij.
      
      
        Pytagoras heeft wel gezeid // in ‘het verhuizen der zielen,
        Dat de een en de ander majesteit // hem niet zo wel bevielen,
        Als ‘het vrolijk tierelieren // van klein ghe-achte dieren,
        Schoen-lappers stoel // na zijn gevoel,
        Ginck boven ‘s Konincks gh’woel.
      
      
        Denkt als den rijken hier verarmd // hoe zeer hij is verlegen,
        Hij is zo in het goed verwarmt // hij kan geen kleine wegen,
        Hij geeft hem dik tot liegen // uitstrijken, en bedriegen,
        Een ambachts man // die werken kan //
        en weet hier al niet van.
      
      
        t’Aensien en hoeft hij in het kleed // noch huizen niet te zoeken,
        Het is den rijken dikwijls leed // dat hij in slechte hoeken,
        Wt nood hem moet generen // en als zij veel verteren,
        Een groot gewin // moet komen in //
        den armen hoeft veel min.
      
      
        Wanneer ‘het gemoet den rijken wijst // religie aan te nemen,
        En als er dan een kruysjen rijst // daar is zo veel te temen,
        Zijn eer en hoge staten // die vreest hij te verlaten,
        Den arme, hij // is hier van vrij //
        en leeft gerust en blij.
      
      
        Wanneer den rijken hier misdoet, Niemand en derf hem straffen,
        Hier in heeft het den armen goed // want op hem gaetmen schaffen,
        Als hij niet wel en wandelt // of ergens in mishandelt:
        Dit is van God // een groot genot //
        want ‘het is der zielen lot.
      
      
        Wordt hier de’t rijken ziek of krank // de menicht’ van Doctoren,
        Helpen hem dikwijls van de plank // al is hij nood’ verloren,
        Den armen en geringen // gebruikt maar slechte dingen,
        En wordt ghesondt // met kleine vond,
        Wanneer het God hem jont.
      
      
        Maar noch is wel het aldermeest // als hier de Rijck’ moet scheiden
        Dat hij zo zeer het sterven vreest // of laat hij hem om leiden,
        Hij veynst hem op te geven // maar sou noch geren leven,
        Liet God hem hier // in zijn playsier //
        hij name wel vertier.
      
      
        De dood is hem een bitter pijn // zo Syrach gaat vermonden,
        [...]en armen is het een Medecijn // te werden hier ontbonden.
        [...]e rijke komen waarlijk // in ‘het Hemelrijcke zwaarlijk,
        Want haren zin // staat meerof min //
        gehecht op ‘het arts gewin.
      
      
        De arme hebben enen troost // aan Christi leer en leven,
        [...]ft men den rijken al wat g’looft // het kan geen zekerheid geven,
        Want met den rijken vrecken // zo moet hij hier vertrekken,
        Lazarus deel // moet hij geheel //
        aanzien als een tonneel.
      
      
        Als hier den armen lijden moet // gebrek aan kost, en kleren
        Hij heeft een groot gezelschap zoet // die met hem pilgremeren,
        Die eertijds gingen dolen // in Bergen en in holen,
        En dezen druk // was haar geluk //
        gedenkt dit wonderstuk
      
      
        Den rijken die zijn schuren houd’, Die moest van droefheid huilen,
        Apostelen en hadden goud // noch zilver in haar builen,
        Noch konden zij berlyden // in ‘het midden van haar lijden,
        Want al haar goed // was in het gemoet //
        die God den armen doet.
      
      
        God heeft de arme verkoren hier // die rijk zijn in ‘het gelove,
        Al schynen zij verloren schier // en alleman ten rove,
        Zij zijn zalig geheten // o wonderbaar secreten,
        Zij geven meer // met mijtjes teer //
        dan enen rijken Heer.
      
      
        Is dit niet wel een voordeel groot // die de arme mag geven,
        In deze tijd, en in de dood // bevrijd te zijn van treuren,
        Oft schoon natuur moet lijden // het gemoet kan hem verblijden,
        Wenst gheen rijkdom // het zijn wegen krom,
        Gij hoort nu den waarom.
