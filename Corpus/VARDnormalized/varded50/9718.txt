
    
      
        HEt is een schoon dierbaar zake,
        Dat men make // onzes Gods Naam
        Groot met lofsanck ende sprake,
        Zoet van smake // is hij bekwaam,
        Iae vriendelijk ende minzaam,
        Laat ons zijn faam // verbreiden wijt
        Looft hem te rechten // die zijn knechten // zijt
        Met groot jolijt,
        Looft den Schepper ghebenedijt,
        Die alle dingen heeft gemaakt subijt,
        Alle ere // tot hem kere // nu ende te aller tijd.
      
      
        Zijnen stoel is boven schreven,
        Hoog verheven // wijd ontdaan,
        Veel duizend Engelen beven,
        Die beneven // voor hem staan,
        Zijn gramschap zaan, doet bergen vergaan,
        De diepten zaan // verdrogen vrij,
        Voor zijn aanschouwen // mans en vrouwen // gij
        Looft hem waart zij,
        Meerder dan alle lof is hij,
        God is over al, verr' en bij,
        In hem leven // ende zweven // allegader wij.
      
      
        Laat ons God een nieu lied zingen,
        Grote dingen // doet hij bloot
        Voor hem beeft swerelts omringen,
        Ook springen // die steenrotsen groot
        In stukken, voor zijn toornig exploot,
        Maar in den nood // is hij zeer goedertier,
        Die betrouwen // op hem bouwen // hier,
        Hij helptse schier
        Wt lijden ende groot dangier,
        Hij gaat met haar door water ende vier,
        Hij kan merken // al ons werken, hij proeft hert' en nier.
      
      
        Hij weet al 'sMenschen gedachten,
        Al haar trachten // en verstand,
        Klaar als dag zijn hem de nachten,
        Groot van machten // is zijn hand,
        Hij hoort en ziet wel alle dingen, want
        D'oore geplant // hebbend', als wijs engien,
        Soud' al voren // ook niet horen // dien?
        Zoude, door wien
        D'ooge gemaakt is, ook niet konnen zien?
        Hij weet alle dingen eer zij gezien,
        Na Schrifts getuigen // voor hem buigen moet alle knieÃ«n.
      
      
        God is een Coninck geprezen,
        Ende dezen // woont in een licht,
        Daar niemand toekomt gheresen,
        Zijn wezen // van Mensen gezicht
        Is verborgen, hij heeft gesticht
        Al het gewicht // der Aarden zwaar,
        Zeer groot bevangen // en gehangen // daar
        Opt water klaar,
        Met zijnen woorde krachtig ende waar,
        Ende hij onderhoudet ook all'gaer,
        Zijn vermogen // ongelogen // blijckt wel openbaar.
      
      
        Al zijn werken groot bevonden,
        Die men gronden // niet en mag,
        Van zijn heerlijkheid oorkonden,
        Te allen stonden // enen dag
        Doet den anderen daar af gewag,
        Maar nooit en zag // iemand God vanden troon,
        Maar zijn verkoren // eengheboren // Soon
        Heeft ons zeer schoon
        Dat verkondigd, lof prijs zij hem te loon,
        Die op den stoel zit, weerdig alle kroon,
        Eeuwig geprezen // moet hij wezen // en dat Lam ydoon.
      
      
        Doe wij Gods vijanden waren,
        Iae zondaren // boos en fel,
        Ginck hij ons zijn liefde baren,
        Zonder sparen // gaf hij snel
        Voor ons allen, in doods gekwel,
        Zijnen zeer wel // beminden Zone zoet,
        Die ons in node // met, zijn rode // bloed
        Verloste vroed,
        Ziet, wat een liefd' ons God de Vader doet
        Dat hij ons als kinderen zijn goed
        Maakt deelachtig // hem eendrachtig // eeuwig lof zijn moet.
      
      
        Zijn genade onverdwenen,
        Is verschenen // over al,
        Tot profijte vanden genen,
        Die recht henen // in dit dal
        In Christo wandelt den weg smal,
        Maar het en zal // na Schrifts bediet,
        Den obstinaten // zondaar baten // niet,
        Hem en gheschiet
        Gheen genade, maar ongenade, ziet:
        Toorn, angst, druk, en zwaar verdriet,
        Wie te voren // zijnen toren // hier niet en ontvliet.
      
      
        Tot quijtschelt van het zondig sneven,
        Is gegeven // die geloofd,
        Jezus den name verheven,
        Tot ons leven // ende Hoofd,
        Des vijands macht heeft hij berooft,
        Blijft niet verdooft // wie oren heeft die hoort,
        Wilt ontvangen // met verlangen // t'woort,
        Rein ongestoord,
        Schickt u te gaan door d'enghe poort,
        Met liefd' en broederlijk akkoord,
        Brengt den Name // Gods bekwame, vrucht der lippen voort.
      
      
        Vreest God al gij Mensen teere,
        Geeft hem ere // want het naakt
        Nu den tijd zijns oordeels seere,
        Na zijn lere // altijd waakt,
        Aanbidt hem, en niet en versaeckt,
        Die heeft gemaakt // Hemel en 'tAertsche veld,
        Zee en Fonteinen // dezen reinen // Held
        Zij toeghestelt
        Eeuwig lof, ere, prijs ende geweld,
        Gij Conincklijck Priesterdom, hier vertelt
        Vrij met vreugden // al zijn deuchden // al wordt gij gekweld.
      
      
        In Gods lof wilt niet verkranken,
        Christen ranken // onversteend,
        Wilt in alle dingen danken,
        Spijs en drancken // hij verleend,
        Gheen Mus en wordt voor hem vergeten neent,
        Zorgt noch en weent // niet, om dat vlees aangaat
        Maar wel gade // Gods genade // slaat
        Daar in bestaat,
        Met vaster herten, blijft vroeg ende laat
        Bij den Here, volgt zijnen raad
        Om niet verdolen // hem bevolen // blijft Chrsitelijck zaad.
