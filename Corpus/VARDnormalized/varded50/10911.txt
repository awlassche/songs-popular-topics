
    
      
        Den mei komt ons lieffelijck aan
        de baan // wordt nu betreden, uiter steden,
        Van Minnaers die daar buiten gaan,
        Zeer fan // volgt haar lief meden // meest in vreden,
        Met harpen, luiten zoet gheclanck
        Violoneen, Fluiten ende sanck
        Over dz schoon groen heye,
        Met dezen koelen meye.
      
      
        Daar gaan geliefkens hand aan hand
        Opt land // langs de Rivieren // zo playsieren,
        Bevangen metter liefden brand,
        Playsandt // daar al de dieren // den Mei chieren,
        de Vischkens in het water zoet
        Spelen daar met een blij ghemoet
        In die zelve valleie,
        Met dezen koelen meye.
      
      
        Die Veldekens staan abondant
        Geplant // met Bloemkens reine // in het gemeine
        d’Amoureusen zeer triumphant
        Vayllant // na de Fonteine // in dat pleine
        Treden zij dan // paar ende paar,
        Menich Iong’man, met zijn Lief daar
        Zij plukken d’Acolye
        Daar in den koelen Meye.
      
      
        Menich Bloemken stater gebloeid
        Daar groeyt // in die landouwe // menig Karssouwe
        Zijnde met sHemels don besproeyt
        Ook vloeyt, dwater nouwe // schoon Jonkvrouwe
        Phebus geeft alle kruiden kracht
        Aurora vochticht dag en nacht
        Tgewas in menigherleye
        Met dezen koelen Meye.
      
      
        Gij Princelicke minnaers hoort
        Treet voort // condyse vinde // u beminde
        Maakt met u twen een akkoord.
        Oerboort // als wel gezinde // onder een Linde
        Zo Amoreusen schuldig // zijn
        Wilt uwen tijd geduldig // zijn
        Wie kan, dat hij verbeye,
        Tot in den koelen Meye.
