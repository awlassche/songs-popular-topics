
    
      
        EEn wintersche nacht
        Heeft mij zo langen tijd verdroten,
        Lank ende onzacht,
        Waar in de Mane staat besloten,
        Vochtig en cout // zeer menichfout
        Waren de velden zeer begoten,
        Met hagel snee // dat doet mij wee,
        Waar door de velden ploten,
        Groot nochte cleen // en vandt ik gheen
        Kruid nochte noten.
      
      
        Den nacht zeer stout,
        Maakt mij geheel bedroeft van binnen,
        Den Winter cout
        Benout ons arme huisgezinnen,
        Mijn blijschap smelt // van gaan door t’velt,
        Coudt en nat was ons gewinnen,
        In het donker vrij // zo liepen wij,
        Dolende buiten weghe,
        Maar wij hebben eer lank // Gode zij dank
        Een licht gekregen.
      
      
        Den dageraad,
        Een morghensterre licht van zijne,
        De Zonne op gaat,
        Den nacht is weg en al ons pijne,
        Door t’soet geklank // der vogelen sanck
        Den Winter die moeste verdwijnen,
        Want schoon // ydoon // staat daar de kroon
        Van den Hemelschen bouwe,
        Het komt al uit // loovers en cruyt,
        Blommekerssouwe.
      
      
        Een blomme in het dal,
        Een Roose tot Saron geprezen,
        Die over al
        Haar reuke geeft rein uytghelesen,
        Een lelye zoet // vol van ootmoed
        Was tussen twee doornen gheresen,
        Dien Held // in t’velt // bedrijft geweld,
        Als enen Leeuw ontsprongen,
        Den stok vol haat // des drijvers kwaad
        Heeft hij ontwronghen.
      
      
        Een bloyende rijs,
        Dat droeg amandelen bekwame,
        Krachtig en wijs,
        Dat zand God zelve in zijnen Name,
        Enen Mei met blaen // zeer wijt ontdaan
        Die droeg zo goede vrucht eersame,
        Hij groeyt // en bloeyt // zeer vriend’lijck besproeyt,
        Door des Zomers voortbringhen,
        In het zoete dal // der vogelen geschal,
        Fonteinen springen.
      
      
        Noch vandt ik daar
        Een Maghet schoon rein uytvercooren,
        Haar stemme klaar
        Was mij vertoond // zeer zoet om horen,
        Haar ogen laeft, // met melk begaeft,
        Haren hals was als Davids toren,
        Nooit mei // zo vei // na mijnen grei,
        Die mij zo wel gheriefde,
        Wt s’Vaders schoot // haar moeder groot
        Dat was de liefde.
      
      
        Komt hier mijn vriend,
        Sprak daar die suyverlijcke schone,
        Eer dat u dient,
        Wijn ende melk geef ik te loone,
        Een cofferken goed // vol druiven zoet
        Is mijn vriend zelve in persoone,
        Hij verblijd // in het krijt // en maakt jolijt,
        Hij gaat boven de Vorsten,
        Hij is van verre // Een busselken Mirre
        Tussen mijn borsten.
      
      
        Ik comme mijn Bruyt,
        Laat ons te samen gaan vermeyden,
        En horen t’gheluyt,
        V Moeder die wil ons geleiden,
        Die u hier zand // als beloofden pand,
        En bezien hoe die wijngaarden schreyden,
        Laat fijn // rozijn // en zoeten wijn
        Wt uwe borsten druipen,
        Mijn herte snaeckt // na vreugden haeckt,
        Geeft mij te zuipen.
      
      
        Een vrome zoet,
        Heeft mij ghelaeft met haren wijne,
        Haar woorden goed
        Zijn mijner Ziele een medecijne,
        Na haren dank // ik van vreugden klank,
        Zij had belooft het was al mijne,
        Is het dat // ik wat // vergare schat,
        En loope de vrome tegen,
        Tot dat ik mijn Lief // al met gerief
        Hebbe gekregen.
      
      
        Ik hebbe haar
        In enigheid beloeft mijn trouwe,
        Vroom ende eerbaar,
        Zij is die alderliefste Vrouwe,
        Zij is versierd // rein gemanierd,
        Haren name gaat boven den gouwe,
        Zij zegt // t’is recht // o beminden knecht,
        Gij moet vast bij mij blijven,
        Of gij en meucht // in minder vreugd
        Doch niet beklijven.
      
      
        Mijn vrome saet
        Heeft zeven bergen schoon bekleven,
        Die Voskens kwaad
        En hebben daar niet in bedreven,
        Zij staan zeer rein // onder t’Hemels plein
        Met Rozen, Lelyen verheven,
        Dates // cypres // loof, kruid en gres,
        Aan mijn Lief te betrapen,
        Niet ik alleen // maar groot en cleen,
        Ende al haar knapen.
      
      
        Princesse eerbaar,
        Van den Zomer in het gemene
        Zeer verre en naar
        Zijn wij ghebrocht in het gheweene,
        Want // zijn verstand // is abundant,
        Wij zijn verlost uit liefden reene,
        Elk moet // zeer vroed // vallen te voet,
        Met deze Bruid ydoone,
        Binnen t’Somers tijd // u zelven kwijt,
        Staat na de krone.
