
    
      
        Zeer minnelijcke groet // zijn moet
        Alle die doet // het goed
        Elk minnaar vroed // zeer zoet
        Des geests coragieus
        Die met een goed verlangen
        Dezen Mei precieus
        Vierich begeert tontfanghen
        Van zijn lief amoreus
        Dats Christus glorieus
        De Mei zeer wijt ontdaan
        Schoonder is nooit in het land // gheplant
        Zeer triumphant // playsant
        Met sGheests verstand // vailliant
        Wilt dezen Mei ontfaen.
      
      
        Dezen Mei delicaat // ontfaet
        En niet en laat // maar gaat
        Want ziet hij staat // ornaat
        Zeer lieffelijck ten toon
        Lustig ient bloemkens cierich
        En die vruchten ydoon
        Veel rooskens balsemierich
        Onder des Hemels troon
        En vintmen gheen zo schoon
        Als dezen Mei propijs
        Bouen Calmus, Caneel, geheel
        Is het tbeste deel // perceel
        Tlustich juweel // geheel
        Recht als een Paradijs.
      
      
        Zo gaat nu al ghemeyn // certein
        Tot dedel grein // in het plein
        Sleuens Fontein // alleyn
        Alle die zijn vermoeid
        Om te vermakken vruechdich
        Recht als de Palmboom bloeyt
        Ende olie duechdich
        Ziet hoe dezen Mei groeyt
        Met sGheestes dan besproeyt
        Zijn bloemkens Dauid zeid
        Lieffelijck in het aanschijn // deuijn
        Een Medecijn // zeer fijn
        Tallen termijn // zij zijn
        Der Heiden gezondheid.
      
      
        Princesse excellent // bekent
        En reuerent // v went
        Dezen Mei ient //  die sent
        Ons God ghebenedijt
        Wt liefden liberalich
        Ontvangt hem met jolijt
        Zijn vruchten triumphalich
        Geeft hij ter rechter tijd
        Geestig in hem verblijd
        Voor tslot met melody
        Dezen Mei exalteert // en eert
        Glorificeert // laudeert
        Onghecesseert // vermeert
        Altijd Gods heerschappij.
