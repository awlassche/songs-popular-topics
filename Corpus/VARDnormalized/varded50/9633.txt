
    
      
        BRoeders houdt vast aan Gods gebod
        Al wordt gij hier der mensen spot,
        Haar fabel gerel // haar lied, haar spel
        Mensen zaan // al vergaan,
        Maar indien men kan
        Volherdigh God aankleven // vroed,
        Men krijgt dat eeuwig leven // zoet,
        Wat noot is dan?
      
      
        De Christen Mensen zijn, of zij
        Ter wereld slechts reisden voorbij,
        Ghetroost in het kruis // om wezen tehuis
        Bij den Heer // Wat verzeer
        Wij in dit gespan
        Lijden, zo wij geraken // daar,
        Zo staan geheel ons zaken // klaar,
        Wat noot is het dan?
      
      
        T'schijnt wel een treurig dingen kwaad,
        Als 'sHeeren kastijdinge slaat,
        Doch het arts verdriet // gaat haast te niet,
        Maar den loon // wonder schoon,
        Is zeer groot hier van:
        Als naar zo weinig lijden // kort
        Ons deel het eeuwig verblijden // wordt,
        Wat noot is het dan?
      
      
        Mishagen wij de Mensen // fel,
        Dat zij ons niet en wensen // el
        Dan lijden, druk // en ongeluk,
        Als die werd // zijn het zweert,
        Iae dood oft ban,
        Oft zegt m'ons veel blamatie // naar
        Staan wij wel in Gods gratie // maar
        Wat noot is het dan?
      
      
        En hadde 'sHeeren Woord gedaan
        David waar in ellend vergaan,
        Want t'eenen troost // hem God altoost
        Ter nood gaf // dezen staf,
        Daar hij op lan,
        Dus als ons komt benouwen // last,
        Zo wij Gods Woord betrouwen // vast,
        Wat noot is het dan?
      
      
        De Here groot, zonder gelijk,
        Ginck door veel lijdens in zijn Rijk,
        Wie nu ter tijd // des lijdens strijdt
        Wel ghesint // ook verwint,
        Zo hij verwan,
        Die werd gekleed met witten // al
        Alle dinck hij bezitten // zal,
        Wat noot is het dan?
      
      
        Paulus hadde victory, want
        T'gheloof als enen werden pand
        Behouden rein // had hij in het plein,
        Ongeschend // tot den endt,
        Zo hij began,
        De Kroon was zijn verwachten // schier,
        Konnen wij hem geslachten // hier,
        Wat noot is het dan?
      
      
        Prinse wat nood is het, oft het vlees
        Niet al en gaat na zijnen hees?
        Als na Gods wens // den binnen mens
        T'alder stond // is ghesont,
        En zo kloek Man:
        Dat men tot 'sHeeren glory // recht
        In zijnen Naam victory // vecht,
        Wat noot is het dan?
