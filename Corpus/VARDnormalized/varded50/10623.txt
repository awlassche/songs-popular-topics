
    
      
        ACh God O Heer // wie zal ik dit toeschrijven
        Die mijn hart teer // met lijden dus turbeert
        Hoe zal ik wrijven // haar wt mijn zinnen drijven,
        Een fleur der Wijven, die t'hert altijd begeert
        En hem vermeert, haar dienaar dag en nacht
        Want zij daar in eren // staat,
        Waar deur t'hert tot haar keren // gaat,
        Liefde niet veracht
        Want reine Liefde baard kracht.
      
      
        Lieflijk is zij // ondeucht heeft haar begeven,
        Beleeft daar bij // barmhertig te aller tijd,
        Eerbaar van leven, reinigheid zonder sneven
        Is haar aankleven // klaarlijk in sweerelts krijt
        Haat nochte nijt // nimmermeer in haar gedacht
        Dus als ik om haar denke // zoet
        Mijn hert haar tranen schencke // moet
        Liefde wel acht.
        Want reine liefde baard kracht.
      
      
        Kloek van verstand, Is zij die ik beminne
        Der deuchden brand, heeft haar zeer hoog versierd,
        Wat ik beginne, t'is al tegens hart en sinne,
        Dus ik bevinne // dat zij mij gants regiert
        Wel gemanierd // is zij daar ik na wacht,
        Als ik haar aenschouwe // ziet,
        Van mijn druk benouwe // vliet,
        Liefde wel acht
        Want reine Liefde baard kracht.
      
      
        Zuiver en net // van aanschijn is zij schone
        Zeer fris gezet // van leden wel volmaakt,
        Zij is ydoone // om te spannen de krone
        In sweerelts troone, die mij in het herte blaect
        Want zij druk laect, en al mijn pijn versacht,
        Zij is die te allen uren // jent
        Mijn hart wt allen trueren // went,
        Liefde wel acht,
        Want reine Liefde baard kracht.
      
      
        Waerste Prinses wilt reine liefde baren,
        Weest mijn Maistres, troost mij in dezen noot
        Laat ons vergaren // in liefden tezamen paren
        Zo zullen verharen // des nijders te aller stoot
        Die klein en groot zoeken altijd tweedracht
        Maar lief wilt haar versmaden // gaan,
        En op der Liefden paden // staan,
        Liefde wel acht
        Want reine liefde baard kracht.
