
    
      
        Ik weet een Peerle ient,
        Gheprent // in Schoonheid rijke
        Nooit Man en heeft present,
        Bekent // haarder gelijke
      
      
        Diamant ofte Robijn
        Sangwijn versierd in gouwe
        En vintmen tot genen termijn,
        Zo fijn // In gheen Lantsdouwe,
      
      
        Gelijk die Zonne klaar //
        Eenpaar, In het tshemels throone
        Zo is mijn Lief voorwaar,
        Aldaar blinkende schone.
      
      
        Waar ik een Cooninck vayliant,
        Playsandt // Rijk van persoone
        Ik versette voor haar mijn Land
        En sant // En ook mijn krone
      
      
        Dat ik mijn gedacht.
        Dag en nacht, op haar zet als donwijse
        Zij is mij te hoge van macht
        Geacht // Ende ook te groot van prijse.
      
      
        Des peerle daar mij dat leed
        Zeer wreed // om moet geburen
        Tis mijn lief dat ik weet,
        Hoe zij heet // die fluer der flueren.
      
      
        Mocht ik met mijn shersen bloed,
        Onzoet // haar noch te winnen
        Zo zout ik tot een vloed
        Mijn bloed // wt laten rinnen.
      
      
        Sturffe ik dan van rouwe groot
        Den dood // om haar duur dezen
        Zo ware mijn wondekens root,
        Ter noot // Geheel genezen.
      
      
        Ik gene elk minnaar wel,
        Beul dat hij liefde drage
        Maar dat hij der liefden gekwel
        Zeer fel, Niet en beclaghe.
      
      
        Die hem spieghelt vrij,
        Aan mij // die spieghelt hem zachte
        Elk minnaar verre ofte bij
        Wie tsij, slaat hier wel op achte.
      
      
        Prinse mindy een edel grein,
        In het plein // Als ik bouen maten
        Zo en muechdy der liefden trein,
        Certein, met recht haten.
