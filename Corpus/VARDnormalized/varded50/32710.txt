
    
      
        GRoot wonder, nu onder den hemel gheschiet.
        Een maghet // die draget een kindeken ziet,
        Te voren // geboren: nooit mens sulckx sach,
        Dat een zuiver maghet van kinde ghelach.
      
      
        Hoort lieden // t'gheschieden, dat God heeft behaagt,
        Te geven // het leven door die zuiver maagd.
        Te voren // verloren die waren als dood,
        Die kwam hij verlossen door liefde zeer groot.
      
      
        Veel suerheyt // en stuurheid des winters men zag?
        Als jeugdig // en vreugdig des Maghet ghelach
        Zij baarden // op aarden den schepper van al
        In enen zeer kleinen doorluchtighen stal.
      
      
        Doen waren // daar scharen van herders gesteld,
        Die wachten // bij nachten de schaepkens opt veld.
        Zij hoorden // de woorden van d'Enghelkens zoet:
        Loopt Herders, loopt vlijtig na Bethlem met spoed.
      
      
        Zij liepen // en riepen malkanderen aan:
        En kwamen // te samen na Bethlem gegaan:
        Zij vonden // bewonden een kindeken cleen,
        Met luyers en tuyers bij Moeder alleen.
      
      
        Doen vielen, door knielen de herderkens neer:
        Ontladen // aen-baden dit kindeken teer,
        Zij lachten // en dachten: wat verwonderen nacht,
        Die ons een zo zoete kind voort heeft gebracht:
      
      
        Doen voeyden // en hoeyden den Engel haar Vee
        De schaepkens // al raeykens door sprongen de wee.
        De geiten // in het bleyten die schenen verheugt:
        De koeien // in het loeien die schenen vol vreugd.
      
      
        O mensen, die wensen naar dit droeve dal
        Te loven, hier boven den Stierder van al.
        Zoekt vlietich, en schietich dit hemelse kind,
        'Tgheeft deuchden // en vreugden aan die het bemindt.
