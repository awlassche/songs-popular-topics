
    
      
        Wijk, wijk, alle Werelds schat,
        Met al het gheen dat uwe praal omvat,
        Met u Goud // hoe versierd, en gebouwd,
        Hoe bepraelt // en hoe konstigh, en hoe schoon bemaelt;
        Hoe machtig rijk // het is als slijk // in het gelijk,
        Van de Jeugd // raar vereend, met witte deugd;
        Die u naam // heel bekwaam // op de Faam
        Henen voert // daar altijd yders oog na loert.
      
      
        Waarde pronk van het ruchtb're Spar,
        Aardig Beeldt, o die gedurig waar
        Bij het licht // bij de glans, van het gezicht,
        Van de gleur // die het alderdighste Hart boord deur;
        Die met 'er straal // en onthaal // menigmaal
        Pordt mij vaak // dat ik na u schoonheid haak;
        Dat ik sta'egh // alle dag // even graag
        In mij vin // grote trek steeds tot u Goddin.
      
      
        Want uwe schone lieflijckhe'en,
        En uwe stem, en pit vol zoete re'en,
        Houdt mijn Geest // stille staan, heel bevreesd,
        Dat ik dick' // dus mijn bede na den Hemel schik:
        O goede Goo'n // watten kroon // wat al schoon,
        Wat al zoet // heeft des Maagd in overvloed:
        het Schijnt u kracht // en u macht // heeft gewracht
        In dit Beeld // daar geheel uwe gunst in spelt.
      
      
        Wat al blos en aardigheid
        Dat er is op het heele lijf ghespreydt,
        Kan mijn tong // hoe volmaakt dat ze zong
        Niet ten deel // openbaren, en noch min geheel.
        Ay! zoete Bloem // dien ik roem // die ik noem,
        Dien ik toon // voor het allerschoonste schoon;
        Ziet een lonk // ziet een flonk // en een vonk
        In mijn Hart // dat ik smeul in de zoete smart.
