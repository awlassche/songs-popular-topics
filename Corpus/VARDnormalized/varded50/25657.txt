
    
      
        In onze roepen ende kermen
        Dat God hem onzer wil ontfermen,
        Zo laat // die maet // ons doch betrachten
        In ons gedachten.
      
      
        Aanmerken waar toe hij gaat keren,
        En zijn barmhertigheid vermeren,
        Dat wij // zo vrij // die met verlangen
        Mogen ontvangen.
      
      
        Krachtig de zonden hier beklagen,
        Want die nu leed en rouwe dragen,
        Haar is // gewis // de Here goedig,
        En zeer lankmoedig.
      
      
        Op Ninive was God verbolgen
        Maar doe haar betering deed’ volgen,
        Was God // doe tot // haar zeer beradig,
        Ende genadig.
      
      
        Bij den tegenspoed in het benouwen
        Staat God barmhertioch in het aanschouwen,
        Is groot // in noot // zijn zijn ghenaden
        Groter in daden.
      
      
        Groot is het aan Israel ghebleken,
        Deur t’roepen en t’ghebedt uitspreken,
        Doe zij // na bij // nu den dood waren
        Naar Schrifts verklaren.
      
      
        En die den Armen hier bedincken,
        In het kleden, spijzen ende drinken,
        Des zijn // nu fijn // werd te verbeiden
        Barmherticheyden.
      
      
        Naar Christi Iesu eigen reden,
        Die de barmhertigheid hier deden,
        Zij zal // voor al // haar ook geschieden
        Die’t zo berieden.
      
      
        Cortlijck zal hij barmhertig wezen
        Den gehoorzamen in Gods vrezen,
        Erbermt // ontfermt // hem haarder spoedig
        Nu overvloedig.
      
      
        Recht mag men dit aan Paulo merken,
        Dat God barmhertig in zijn werken
        Is ziet // t’gheschiet // doe hij sprak coene
        Wat zal ik doene.
      
      
        In Godes vreze te aenverden,
        Betrouwende, daar in volherden,
        Verwerft // beerft // ook vroeg en spade
        Hier Gods genade.
      
      
        In het Boek der Psalmen staat geschreven
        Dat God hem ontfermt hoog verheven,
        Als een // gemeen // vader niet minder
        Over zijn kinder.
      
      
        Nalatende nu die zondaren
        Die tegen ons doen, niet bezwaren,
        Maar kwijt // altijd // schelden van herten
        Al zonder smerten.
      
      
        En vergeven al zonder trueren
        Onzen Naasten tot elker uren,
        Wie vroed // dus doet // zal God verkoren
        Nu geern verhoren.
      
      
        Neemt dit van mij liefste in het goede,
        Met enen Christlijcken ghemoede,
        Ik groet // u zoet // vrij onverholen
        Weest God bevolen.
