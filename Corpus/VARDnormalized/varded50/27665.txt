
    
      
        Lest ginck ik in de mei spanceren
        Wandelen met Lusten fijn
        Waar door het hert most conforteren
        Ja geheel in Rusten syn
        Vrou venus kaeren excelent
        Sach ik bij paren daar ontrent
        Hand aan hand // zeer plaisant // dansen en springen
        Wentelen in het groene // gras
        Zoudt een quandt // syn verstand // konnen dwingen
        Die gewoon te doene // was.
      
      
        Terstond kwamen zij voorts gedrongen
        Herpen velen Luiten goed
        Zij dansten spelden ende zongen
        Sulckx tgehoor ontsluiten doet
        Musyka schoon was daar ontrent
        Elk hield syn toon opt instrument
        Delykaet op syn maet // zo manierig
        Pas a pas voorts dansende
        Nyders Raad // werd versmaad // puttertierich
        Met vreugd bloomkens schansende.
      
      
        Des machdekens Lustig van leun
        Stelden mij die Rozen bij
        Waarom zoude ik geen vreugd aencleuen
        Die tot sulckx verkosen zij
        Veel bloemkens ook suyuer ende Klaar
        Lieflijk van Reuk brochten zij daar
        Sulcken vreugd // mij verheugt // bouen matten
        Dacht ik hier end most horen daar
        Tsa gij Ieucht // maakt geneucht // met zoet praten
        Tis corage voor ende naar.
      
      
        Terstond kwamen zij voorts geladen
        Omden Lust te boeten meere
        Met vladen tarten ende gebraden
        Wijn om druckx verzoeten zeer
        Mijn docht ik haak naar sulcken geur
        Doen werd de smaak noch gouuerneur
        Heus en blij // willen wij // banketteren
        Sprak een van des maagden fier
        Hout v vrij // wie soudy // mogen deren
        Spijt al diet mishaechden hier.
      
      
        Dus ziende alle des vreugd voor ogen
        Therten bloed verwermden zij
        Terstond syn zij toe gevlogen
        Helsden custen narmden mij
        Ik hiel mij coele op dit termijn
        Maar het gevoele dede mij pijn
        Dye nu mocht als hij docht na syn wille
        Venus hof omgraeuen ziet
        Sulcken tocht moet gewrocht syn al stille
        Maar elk heeft de gaeuen niet
