
    
      
        OM in den Here // den Echt t’aengaen,
        Die na Schrifts lere // recht mag bestaan,
        Sulcx van twee vromen // geschieden moet
        T’heeft zo genomen // beginsel goed,
        In het Paradijs // door Gods advijs,
        Want groot aenwijs
        Dezen staat rein ons doet.
      
      
        Christi gemeente // is van zijn vlees,
        En zijn gebeente // dus na den es
        Zo t’hooft is heilig // moet zijn t’lichaem,
        Hij gaf hem veilig // aan t’cruycen raam,
        En suyverd’ haar // in zijn woord klaar,
        Op dat zij waar,
        Heerlijk voor hem bekwaam.
      
      
        T’moeten twee wezen // leden met t’lijf
        Aan t’hooft geprezen // houdende stijf,
        Heylsaem eendrachtig // in het recht verstand
        Met God almachtig // noch in den band,
        Der liefden recht // staend’ in den Echt,
        Wie’t zo aenlecht
        Die en bout niet opt zand.
      
      
        Daar dus twee leden // in liefden t’saem
        Den Echt in treden // in Christi naam,
        Daar wil den derden // met zijnen geest
        Christus wel werden // zo hij ter feest,
        Was ljvich daar // hij voor de schaer
        Een wonderbaar
        Teken deed’ alder eest.
      
      
        Tot Canan stede // in Galileen
        Met liefd’ en vrede // waren gebeen,
        Onzen Behoeder // en Jongers vroed,
        Ook Iesu Moeder // ter Bruiloft zoet
        Dus daar gegaan // zeer wel ontfaen,
        Ter tafel zaan
        Men t’volck aan zitten doet.
      
      
        De Bruid wel statig // is te bemoen,
        Versierd ook matig // zat daar als doen,
        Wat uytghelesen // gezelschap fijn
        Eerbaar van wezen // is daar ghezijn,
        Wat soeter praat // maar doch wat raad,
        Recht nu alst gaat
        Ten besten faelter wijn.
      
      
        Van het wijns ontbreken // den Heer Iesu
        Moeder ginck spreken // wat roert mij u,
        O Vrou mijn ure // is komen niet
        Sprak Christus pure // maar zij ghebiet
        Den dienaars dat // zij deden rat,
        Al t’ghene wat
        Hij haar seyd’ ofte hiet.
      
      
        Zes stenen vaten // stonden van welk
        Twee of drie maten // water hield elk,
        Daar sonderlinghe // sijnde gezet,
        Na die reyninghe // der Joden Wet
        Den Heer gebood // dat men ze goot,
        Vol waters bloot,
        De dienaars dedent net.
      
      
        Als zij vol waren // sprak Christus milt
        Tot den dienaren // schenckt uit en wilt
        Spijsmeester geven // gedaan dus is het,
        Hij heeft beseven // goed onghemist,
        Van smake fijn // den nieuwen Wijn,
        Die water zijn,
        Plag, welk hij niet en wist.
      
      
        Riep goedertier // den Bruidegom,
        T’is de manier // (sprak hij) al om
        Eerst geeft den besten // wijn alle man
        Maar als ten lesten // t’volck dronk is dan
        Slechten, maar gij // houdt tot nu bij,
        Den besten vrij,
        Zo ik wel smaken kan.
      
      
        Dus heeft des Heren // eerst wonderdaad
        Hier gaan vereren // den Echten staat,
        Om brengen dezen // vrolijk ten ent,
        Liefde moet wezen // het fundament,
        Want in het gemeen // zijn onder een,
        Vreugd en gheween,
        Gemengeld hier ontrent.
      
      
        Den man om voren // te staan zijn huis,
        Wel na behoren // heeft voor zijn kruis,
        Arbeids bezwaren // kommer in het hert,
        Ook t’wijf in het baren // verdriet en smert,
        Toezien men moet // dat watervloed,
        Van druck’ onzoet,
        In Wijn verandert werd.
      
      
        Oorlof met namen // al die den Echt
        Beleven tezamen // oft aangaan recht,
        Blijft zonder flouwen // in vroomheyts jeugd
        T’water van rouwen // door Christi denkt
        Verkeren zal // vrij van misval
        In Wijn, dats al
        In een eeuwige vreugd.
