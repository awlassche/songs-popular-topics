
    
      
        DEn Here triumphant // zij prijs en dank gestade,
        Die bijzonder // wonder//lijk zijn volk bevrijd,
        Want met zijn sterke hand //om Israel voor schade
        Te verlossen // rossen // wagen en subijt
        Wierp hij omme zonder sparen // zo dat de egyptenaren
        Dreven op de baren // aan den oever dood,
        Die Israelijten waren // dikmaals verlost van haren
        Vijanden, en bezwaren // angst ende noot.
      
      
        Christus Hoochpriester groot // heeft om eeuwig te blijven,
        Tegen zonden // vonden // een Verlosser goed,
        Door zijn eigen bloed root // tot een zalig beklijven
        Hier voor leven // welken // hem ontvangen zoet,
        Haar wordt de macht gegeven // kinderen Gods te leven,
        En God heeft verheven // zijn nieu verbond
        Met die zo zijn gedreven // om enigheid t’aencleven
        Na Gods Wet geschreven // in s’herten gront.
      
      
        Al wijken onghesont // veel argherlijcke leden,
        Die verdwenen // henen // gaan, en d’eenicheyt
        Des Geests verlaten rond // ja en den band des vreden,
        Dat haar laatste // t’quaetste // wordt // zo die Schrift zeid,
        God blijft noch zonder mijden // bij zijn volk te allen tijden,
        Die hem oprecht belijden // voor deze zal
        Hij schermen en strijden // zij en zullen niet glijden,
        Al viel ook bezijden // een groot getal.
      
      
        Al waart dat heel end’ al // die Wereld t’onder ginge,
        Al verdronken // zonken // bergen in Zee,
        Zo zal // vrij van den val // de stad, s’Heeren woninge
        Noch gerustich // lustig // blijven zonder wee,
        Het Wijze-mans ghestichte // Onbeweghelijck dichte,
        Op des steens gewichte // niet en vergaan
        Dat vredelijcke gezichte // Zijnde vol klaren lichte,
        Met oordeel, gerichte // eeuwig bestaan.
      
      
        Dat huis Gods delicaat // lieflijk met zoet behagen,
        Davids hert en zinnen // binnen // waren daar.
        Iae bad zijn toeverlaat // om daar zijn levens dagen
        Met verheugen // mengen // blijven eenpaar
        Om daar zonder verflouwen // den schonen Godsdienst t’aenschouwen,
        Met een vast betrouwen // in God vervroyt
        Maar die bezijden bouwen // haar aan t’Hooft niet en houwen,
        Zij komen in rouwen // zeer wijt verstroyt.
      
      
        Prinselijk Herder nooit // en verliet gij u Schapen,
        Die u woorden // hoorden // gehoorzaam zijn
        Met u te zijn ghekoyt // al haar ghenoechte rapen,
        Doort volherden // werden // eeuwig termijn
        Gheluckich en geprezen // van alle druk genezen,
        Een weyd’ uytghelesen // in s’Hemels plein
        Vinden zij boven dezen // vol rust en onverknesen,
        Haren drank zal wezen // s’Levens Fontein.
