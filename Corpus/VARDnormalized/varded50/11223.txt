
    
      
        EEn nieuwe Liet // vaat dit bediet
        Tantwerpen geschiet // al in den Jare
        Van tzestig ziet // en verwonderd v niet
        Een Broeder met vliet
        Met groter schare
        Van diefleyders hoort // gebracht al voort
        Van de Steenpoort
        In de Vierschare.
      
      
        Willem hiet hij // och hij was zo blij
        Een Kleermaker vrij // binnen Stede
        Als hij nv bij // de Heren tfij
        Kwam, ende zij
        Hem vraagden mede
        Oft hij was herdoopt // ons dit ontknoopt
        Waar gij op hoept
        De Schout dede de reden.
      
      
        Hi sprak tot haar // waarom vraechdy maar
        Na mijnen doop klaar // ende niet na mijn leun
        Op dat elk voorwaar // en gij allegaar
        Een ordeel daar
        Na mochte genen
        Want ik ben een roof // vraagt na mijn geloof
        Op dat des wereld doof
        Daar na doe zonder sneuen .
      
      
        De Schout zonder deer // die vraagde zeer
        Of hi herdoopt weer // als hi eens had beleden
        Wat is v begeer // gij begeert niet meer
        Dan een woord, O Heer
        Om mij te vertreden
        Hij sprak als een goed knecht // ordeelt doch recht
        En ouerlecht
        Aldus gingen zijn reden.
      
      
        De Schout in het gemeen // hert als een steen
        Was niet te vreen // vraagt wederom met lagen
        Willem sprak hoort reen // gij hebt voorheen
        Mij geuraecht alleen
        Mag ik ook nv niet vragen
        De Schout sprak ontbeyt // geeft eerst bescheid
        Al van v feit
        Als dan zal ickt gewagen .
      
      
        Ik wilde wel // sprak Willem snel
        Met goed opstel // dat gij eens dachte
        Op den dag zeer fel // die branden zel
        Ghelijk een Ouen rebel
        V doch daar voor wilt wachten
        Als de godloze snoo // verbranden zal als stro
        Al doet hit noo
        Dus wilt daar doch op achten.
      
      
        Willem sprak zonder vaar // in Marco klaar
        In zijn zestiende daar // muecht git lezen
        Al wie gelooft voorwaar // en wordt gedoopt eenpaar
        Die zal hier naar
        Zalig wezen
        Men vraagt, hoort mijn // sprak de Schout zeer fijn
        Op dat termijn
        V niet na dezen.
      
      
        Willem sprak voortaan // als hij had gedaan
        De kinder onbekwaam // en konnen niet geloouen
        Noch sprak hij zaan // ook onbelaen
        Wilt dit verstaan
        Nochtans voor de doouen
        Op mijn gelooue rein // verstaat dit plein
        Ben ik gedoopt alleyn
        Hierom ben ik verschonen.
      
      
        De Heren onuroet // al metter spoed
        Maakten hem op de voet // en gingen binnen vlieden
        Met goeden moet // sprak hij zeer zoet
        Gij Borghers doet boet
        De Schout ginct hem verbieden
        Willem sprak: Heer Schout // laat mij spreken onuerflout
        Niet lange (ik hout)
        Zal mogen spreken van v lieden.
      
      
        Terstond verstaat // de Heren kwaad
        Met haren raad // kwamen wilt horen
        Willem sprak, vaat // mijn Heren het doet v baat
        En wijst niet na tPlaccaet
        Wilt gij nit zijn verloren
        Denkt doch opten dag // die niemand voorbij mag
        Als gij zult roepen ach
        Oft gij moet zijn herboren.
      
      
        Zij hebben terstond // al gelijk in het rond
        Hem met enen mond // verwezen
        O Heer doet cont // haren kwaden vont
        Dat zij worden ghesont
        En de Schriftuur lezen
        Maakt dat haar hert // hart als een berd
        Doch soeter werd
        Dat bid ik v mits dezen.
      
      
        Smorgens vroeg tijd // op de merkt wijt
        Met klein jolijt // zij hem brachten
        Aan den staak zonder strijd // worgen met nijt
        Zonder langer respijt
        Gingen zij hem versmachten
        Och God, O Heer // ziet doch eens neer
        Na v leden teer
        Want zij haar zeer verachten.
      
      
        Prinse ient // mijn liet bekent
        Is nv ten ent // gezongen
        Ik wasser present // en hem ontrent
        Als hij was ghesent
        Voor den ouden ende jongen
        Daar hij belet // zijn geloof gereed
        Voor dOuerheet
        God moeten lonen alle tongen.
