
    
      
        ALs d'Harders haar Schaepjes bewaeckten, zo scheen
        De Stralen // te dalen // op de aarde benen,
        Den Engel vertelt // de Harders in het Veld
        Gods Soon is geboren, dus maakt u te been.
      
      
        Dus loofden en zongen de Engelen rein,
        Treedt vaardig // het is waardig // gij Harders gemeyn,
        Verhoord onze stem // te Bethlehem,
        Daar zult gij vinden dat Kindeken klein.
      
      
        De Hardertjes liepen, en lieten haar Vee,
        Met zingen // en springen // vol vreugde al-ree;
        Dus liepen zij al // en vonden de Stal;
        Daar brochten zij Jezus veel zoetigheid mee.
      
      
        Daar zagen zij Jezus bij Beesten geleid,
        Ghewonden // en vonden // als haar was gezeid:
        Doen gingen zij weer // en loofden den Heer;
        Zij hebben Gods grote ghenaden verbreydt.
      
      
        Non laten wij als deze Harders bij nacht,
        Ons leven // begeven // om houden goe wacht,
        En zoeken Gods Rijk // op dat wij gelijk,
        Door Jezus, Gods Kinderen worden geacht.
