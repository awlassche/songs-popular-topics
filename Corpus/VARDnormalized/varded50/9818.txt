
    
      
        Rein van manieren // is den Echtenstaet,
        Die hem hantieren // in den rechten // graad,
        Tis slechten raad, die hem verbieden, want
        Beter is trouwen, dan te lijden brand:
        Want die horeren // mal
        God condemneren // zal,
        Des wij lauderen // al
        T'Huwelijcx band.
      
      
        Tis waar dat vrienden // in malcanders noot
        Dikwijls wel dienden // al bystanders bloot
        Maar anders groot, met een affecty vrij
        Staan man en wijf ter noot // malkander bij
        Wie een goe Vrouwe // heeft
        Die God getrouwe // geeft,
        Bevrijd van rouwe // leeft
        Te langer hij.
      
      
        Een Wijf die deugdig // is van wezen vroed
        'sMans herte vreugdig // zij genezen doet
        Geprezen moet // zij zijn om haar bedrijf,
        Een edel gave Gods is zulk een Wijf,
        Tot troost van haren // Man,
        Die wel bewaren // kan,
        Haar tong, en sparen // van
        Weder gekijf.
      
      
        Zo lief bekwame // op der eerden niet
        Als een eersame // Wijf vol werden ziet,
        Met zweerden vliet // een rover in der nacht
        En brengt den roof // haar die boven geslacht
        Hij voor vriendinne // koos,
        Veel zijn door minne // boos,
        Geworden sinne // loos,
        En omgebracht.
      
      
        Een vroom wijf schone // is als brand daar
        De Lamp ydoone // opt heilig Kandelaar
        Zijnde bestande haar // eerbaar gemoed,
        Een gulden Pilaer op een zilver voet
        Is zij gheleken // wel
        Om niet te breken // snel.
        Oft te ontsteken // fel
        Door bozen gloed.
      
      
        Die een Wijf zedig // voor zijn proven houdt,
        Huysich, cuysch, zedig // maghse loven stout
        Al boven goud // en boven edel steyn,
        Haar sieraad is, wat neerstigh is en rein,
        Al moets' hier lijden // smert,
        Haar vreugd in tijden // werd
        Met een zo blijden // hert,
        In 'sHemels plein.
      
      
        'tWijf zal door baren // ende kinder dracht
        Ten Hemel varen // dat ze ginder lacht,
        Zo inder kracht // gelove, liefde jent
        Heylicheydt, eerbaarheid haar blijft ontrent
        Voor des versijcke // wat,
        Werd zij Gods rijke // stad,
        D'Enghels gelijke // dat
        Werd excellent.
      
      
        Prinse d'oprechte // zo zij deden eer,
        Zal men in Echte // niet besteden meer,
        Te vreden zeer // wel, eeuwig zonder pijn
        Zullen zij in 'sLams Bruylofts feeste zijn,
        Zo vrolijk maken // chier,
        Elk mag wel haken // fier,
        En neerstigh waken // hier,
        Om daar te zijn.
