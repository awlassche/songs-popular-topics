
    
      
        Lieftallig groen prieel
        Daar smaecklijck enen dis
        Was bereidet na gewoon
        Daar ik met de Liefste schoon
        Het hert vermaecten fris
        Bacchides wijn zeer el
        En Ceres smaecklijck brood
        Bracht ons in het bij een gevoeg
        Een zulk hertelijck genoeg
        Dat ons gheen tijd verdrot
        Phoebus was met haast gegaan
        Op zijns wagens Gouden ran
        Ginck beschijnen // wijt
        D'Medecijnen // blijt
        D'onbesinde // wicht
        Schoot een blinde // schicht
        Op dezen zelven tijd.
      
      
        Trou Lieffdes heten brand
        In mij werd straks gevoeld
        Die in alles zeer beleeft
        Mijns Lieffs Hert ontsteken heeft
        En twert niet haast ghecoelt
        Voor dat rein Liefdes band
        Gebonden werd zo vast
        Dat t'Lieff van de Liefste niet
        Wijken kan / oft doen verdriet
        Door Venus overlast
        O Cupido hoog vermaard
        Stort gheen Lieffd' zo zoet ghepaert
        Met opmerken // zaan
        Wilt versterken // gaan
        Onze jeugden // dan
        Komt meer vreugden // an
        Deur schichten die niet schaen.
      
      
        Mijn Flora schone Bloem
        Ons Lusthof is gheciert
        Met twee spruiten die gemeen
        Zullen zijn gent aan een
        Heel vrientlijck gemanierd
        Rein Liefde dracht gheen Roem
        Want tloose nijders zaad
        Woud' het werd zo verd' gebracht
        Dat het kwame in het veracht
        En eeuwig enen haat
        Doch dat selfst zal niet gezien
        Dat de een wil van de ander vlien
        Het trou Minnen // zal
        Beyd' ons zinnen // al
        Gouverneren // wel
        Gheen tempteren // zel
        Ons brengen ongeval.
      
      
        Nu Lief stelt u gerust
        Verbindet hert aent mijn
        Dan zo zullen zeer bekwaam
        Beyd' ons ziels in een lichaam
        Wel vast gesloten zijn
        Zo werd te recht gheblust
        Het lang ghewachte goed
        Tgheen de jeugd Couragieus
        Deur de Liefde Amoureus
        In vreugd ontspringen doet
        Nectars drank zal principaal
        Doen verdrijven d'oude quael
        Die veel jaren // wreed
        Ons tbeswaren // deed
        Met een zoete // vlaech
        Komt zo groeten // staech
        Ons met een langen vreed'.
      
      
        Wel loffelijcken held
        Ik prijs u Venus kind
        Om dat al tgheleen getreur
        Is verkeert in goed faveur
        Men seylt recht voor den wind
        Oft Momus hem nu quelt
        En wezen wil partij
        Dat zal komen veel te laat
        Niet voortspruiten kan zijn zaad
        Hoed ook begonnen zij
        Want rein Lieffd' die eerbaar blijckt
        Nemmermeer uit therte wijkt
        Maar in het groeien // hoort
        Wil die bloeien // voort
        Met ghenuchten // maar
        Zonder zuchten // zwaar
        Door tvriendelijcke woord.
      
      
        Prinslijcke Reine Maagd
        Vaart wel mijns herten vreugd
        Denken Liefste wilt op mijn
        Als ik hoop te doen op dijn
        Alleen in eer en denkt /
        Nu mij dan Liefde dracht /
        Die u ben toegheveucht
        En de wereld in het gemeen
        Nemmermeer en kan ghescheen
        Ten zij met ongheneucht
        Maar wat Liefde heeft voorzien
        Moet ten lesten zo gezien
        Spijt de haters // loos
        Die met snaters // boos
        Door den tongen // raad
        Onbedwongen // kwaad
        Zijn werkende altoos.
