
    
      
        Komt Aura lief //
        V zoetheid nu ontmoete // mij,
        In dezen brief //
        Val ik absent te voete // dij,
        Een groete // zij //
        V van herten daar neven //
        Geschreven //
        Laat geschieden //
        Mijn ontbieden //
        Die mij smerten // cont //
        Wilt af vlieden //
        Zulke lieden //
        Die u rieden //
        Mij te wieden //
        Wt der herten // gront.
      
      
        O f gij mij als
        Balling, van u gezichte // drijft,
        Op mijnen hals
        Dit jok (zwaar van gewichte) // blijft,
        In dichte // schrijft
        Mijn hand noch even seere,
        V ere,
        Wat kan schaden
        Der genaden
        Tijd, verwachten // lang
        Ben beraden,
        Met versmaden
        Dus geladen,
        T'gaen den kwaden
        Ongeachten // gang.
      
      
        Rayende licht
        Ontsprongen in den morgen // vrough,
        Mijn ziel verplicht
        Aan u, altijd met zorgen // iough,
        Verborgen // drough
        V overgroote minne:
        Goddinne,
        Laat der schanden,
        Niet verbranden,
        Mij, met groot // verwijt:
        Spijt vijanden:
        Kerckers banden,
        Deur u handen.
        Werd' ik vanden
        Swaersten noot // bevrijd.
      
      
        Naar redens recht
        V lief, Laurier, en krone // vought:
        Wee zij den knecht,
        Die iet op dijn persoone // wrought:
        Hoe schone // lought
        Gij, in ons eerst genaken,
        Wij spraken
        Zo besneden,
        Zoeten reden,
        Zonder schamen // iet:
        Waar is heden
        Dien voorleden
        Tijd getreden,
        Die met vreden,
        Ons verzamen // liet?
      
      
        Een uerken cord
        Dunkt mij gelijk veel iaren // zijn)
        Wanneer mij wordt
        Ontvreemt zo hemels klaren // schijn:
        Hoe zwaren // pijn,
        Ik draagt in uwen name:
        Eersame,
        Wat gij dochte,
        Ik volbroghte,
        Ooit al even // vroed:
        Wtgesochte,
        Welgewrochte,
        Diergecochte,
        Peerle, mochte
        Gij mij genen // spoed.
      
      
        Lef ik in het vier,
        Deur u, als den Salmander // doet,
        Cupido schier
        T'hert af, met zijnen brander // wroet,
        Leander // moet
        Ik volgen, niet gedincken
        T'verdrincken:
        Deur mijn pogen
        Zal ik bogen
        S'waters sterken // vloed:
        Wilt medoogen
        Vanden hogen
        Toren, togen,
        Dat mijn ogen,
        V licht merkten // zoet.
      
      
        Iason verslough
        Den Draak, die hem fel tegen // blies,
        Heeft in den plough
        De Stieren wreed geslegen // dies
        Verkregen // t'vlies:
        Cause was d'uytvercoren
        
          Alvoren
        
        Dat gij ionde
        Zulk gezonde,
        Zonder scheiden // van,
        Quets', noch wonde,
        Deren konde,
        D'aertsche ronde,
        Dienstbaar stonde
        Onzer beiden // dan.
      
      
        Als magnifijck
        Princesse, gij ten thoone // zit,
        Ja puur gelijk
        een duyfken zijt: idoone // wit,
        Mignoone // dit
        Ontvangt (van mij arm kranke)
        In danke:
        Zo zal dezen
        Brief, in het lezen,
        V behagen // meest:
        Dijn geprezen,
        Hoog geresen,
        Zoet bij wezen,
        Zal genezen,
        Mijn verslagen // geest.
