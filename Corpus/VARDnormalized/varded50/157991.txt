
    
      
        Hoog en duur // is het bestuur // van het vuur
        Levitici tien // t'sien
        Tot figuur // moestet doen gezien,
        Maar heel puur // is den zin van dien,
        Liefde is het ware wezen,
        Some Canticorum lezen,
        Nemet merk // zij is sterk // in haar werk,
        Gelijk dood en hel // fel
        Houdt haar perk // tegen veel gekwel,
        Leek of klerk // maakt haar niet rebel.
      
      
        Een geleerd // hoog gheeert // die temteert,
        Christum onder het schijn // fijn,
        Als begeert // onderwezen t'zijn,
        Maar verkeert // t'hert is vol fenijn,
        Hij en was niet verslagen,
        Ginck nochtans Christo vragen,
        Wat gebod // t'hoochste lot // heeft van God
        T'welck is in de Wet // zet,
        Deze zot, ongeregeld tret,
        Christus tot // hem antwoorde net.
      
      
        Niet allien // zes en tien // maar ik mien
        Ook in het dertichst plaan // t'staen
        T'vijfde boek // deed' Christus vermaan,
        Zijn verzoek // die het recht kan verstaan,
        Lief te hebben God u Here,
        Hert en ziele daar toe kere,
        Ook 'tghemoet // hij is goed // die het zo doet,
        Dit is het grootst bevel // wel,
        T'ander zoet // dit gelijken zel,
        D'naesten moet // gij ook beminnen snel.
      
      
        Deze twee // gants gheree // daar ter stee,
        Van Christo belast // vast,
        Wel ter snee // worter aenghepast,
        Wet ende mee // der Profeten wast,
        Ditte gaat ons onderwijzen
        Dat die liefd' is hoog te prijzen
        Wie misdoet // uit vleys oft bloed // om tijtlijc goed,
        Oft om anders wat // schat,
        Is onvroed // boven Gode dat,
        T'sijn onspoed // heeft hij lief gehad.
      
      
        Raad nu voort // met akkoord // soo't behoort
        Die des liefd' uit twist // mist,
        Die verloort // al daar hij op gist,
        Telcker oord // oft hij wel al wist,
        Tongen ende Prophecyen,
        Niet zal hyder mee bedijen,
        Gelove vaat // wonder daad // hoort oft laat
        Zonder liefd is het niet // ziet
        Elker maet // t'goet den armen biet,
        Oock't gewaad // t'lichaem branden liet.
      
      
        In haar zoet // is ootmoed // vriendschap goed
        Zij toont niemand spijt // nijt,
        Schalkheid doet // zij tot gender tijd,
        Noch verwoed // opgeblazen krijt,
        Zij bedriecht in genen hoeken,
        Zij en gaat t'haer niet zoeken,
        Niet verwacht // verdriets klacht // erg gedacht,
        Noch dats' om tweedracht // lacht,
        Maar met kracht // die waarheid betracht
        Dag en nacht // tdoet haar wee of zacht.
      
      
        Klaar bescheid // hier verbeid // als gezeid
        Wordt in Moyses Wet // net,
        Alle beit // Gods en s'naesten met
        Ook bereid // u zin hier toe zet,
        Siedy s'broeders vee gaan dolen
        Exo.23,4. V wordt neerstelijk bevolen,
        Niet ontreckt // u suspeckt // maar perfeckt
        Tot u broeders deer // keer,
        Isset sect // ofte valse leer
        Niet begect // maar helpet doch weer.
      
      
        Toont nu klaar // met gebaar // hier en daar,
        Dat in u God bemint // vint
        Is niet waar // s'werelts liefd' ontrint
        In u maar // zijn gebod versint,
        Die hier s'werelts vriend wil wezen
        Zal Gods vijand zijn in dezen,
        Wat getier // oft manier // dat men hier
        Stellen tot een toon // schoon
        Voor het vier // gheltet met een boon
        Hert en nier // doorzocht elk zijn loon.
      
      
        Is er nu // vrees en schu // dat gij u
        Voor het kruis versaagd // dracht,
        Al is het ru // en u seere plaecht,
        Zulks verdu // ende van u jacht:
        Iacob schrijft wilt verblijden
        Als gij veel bekoring lijden,
        Dit versint // wie hem vint // een Gods kind
        Zal voor gheen kastien // vlien,
        Troost ontrint // is er te allen tien,
        Wie verwint // zal veel goeds gezien.
      
      
        In den Heer // stelt nu teer // u begeer
        Om der liefden werk // sterk
        Langs zo meer // te brengen in het perk, Esa.58,8.
        Na zijn leer // tot stichting der kerk,
        Hij verbiet // overspelen,
        Doodslaan ziet // ende stelen,
        Getuige vals // geen van als // dies getals
        Nimmer doch begeer // meer
        Myr of te als // zijnde bitter zeer
        Met regals // neemt al hier zijn keer.
      
      
        Zonderling // in dit ding // zo ik sing,
        V naasten wel ghesint // mint
        Gants gering, oft gij het zelver bint,
        Zonder dwing // als die eerst begint:
        Die liefde doet niet ten quaesten
        Ymmermeer zijnen naasten;
        Maar ghehult // zij vervult, alle schuld
        Van der gantser Wet // net,
        Met geduld // u toch hier na zet
        En gij zult // dan zijn onbesmet.
      
      
        Zaligheid // is toeseyt // met bescheid
        Niet die alleen t'woort // hoort
        Maar bereid // tottet werk akkoord,
        Twellick leit // inder liefden oord,
        Liefde is die hooftsomme
        Der Geboden Gods alomme,
        Hoort ghemien // t'kan niet schien // dan allien,
        Alst een rein hert // werd,
        Ende vlien // dat consciÃ«ntie smert,
        Ook bij dien // in het geloof volhert.
      
      
        Och wat raad // menig kwaad // van des maet,
        Hebben veer ghefaelt // dwaalt
        Onnutte praat // bij haar toe ghehaelt,
        Zonder baat // sierlijk af ghemaelt,
        Als leeraars die wet vertellen,
        Verstaan niet wat ze zeggen oft stellen,
        De Wet is goed // tot behoed // van het ghemoet,
        Ghebruycktse iemand recht // slecht,
        Maar men moet // weten die oprecht
        Is zeer zoet // gheen Wet opghelecht.
      
      
        OP dat dit // elk lit // onbesmit,
        Doch in rechter maet // vaat,
        Neerstich bidt // den Heer om ghenaet,
        Hier in zit // ledig vroeg noch spaat,
        Laat u leven klaar zo blijken,
        Dat er elk op mag kijken,
        Metter kracht // met gedacht // houdt goe wacht
        Op dies' Heren geeft // vreest
        Dit geslacht // een goed voorbeeld weest,
        Inder nacht // gaan die mensen meest.
      
      
        Neemt in dank // dezen sanck // gaat niet mank,
        Al waar schoon die noot // groot,
        V verlanck // zij in elk aanstoot,
        Eer iet lank // na een ander boot,
        Komt nu gij ghebenedijde,
        Die bokken aan de linker zijde,
        Leert dit voort uit Gods woord // met akkoord,
        V zelfs onverschoont // toont,
        Ongestoord // wordt gij dan beloont,
        Wie u hoort // wordt met u ghecroont.
