
    
      
        Phoebus God vant zoet Muziek
        Maakt gij me ongheneucht
        Om u Daphne schoon en rijk
        Gij varen laat veel vreugd
        Des Goddin,, vliet de min
        En het spelen zoet der Lier
        Dies verkeert // transformeert
        Zij in enen Lauvverier
        Van gelijken // gaat
        Nu afwijken // kwaad
        Een door Venus raad.
      
      
        Ik nu wel beklagen mag
        Dat door lieffd' mijn vrijheid
        Die ik zelfs te bruiken plag
        Dus schandig heb geleid
        Op een schoon // maagd ydoon
        Die ghevaen mij heeft gebracht
        In het net // en percket
        Waarom dat Vrou Venus lacht
        Mijn lief waarde bruyt
        Door u zoet geluid
        Mij verlost hier uit.
      
      
        Wilt Lief overdenken staech
        Al mijn affectie groot
        Wat Lieffd' ik in het herte draak
        Om u zoet wezen bloot /
        Edel maagd // niet verjaagd
        Hem die u zo troulijck mint /
        Maar veel meer // krijgt de eer
        Dat gij mijn pijn overwint /
        Vytvercoren // rein
        Hoog geboren // grein
        Lavende Fontein.
      
      
        Waarom Lief acht gij zo slecht
        D'onlydelijcke pijn
        Die ik Minnaar trou oprecht
        Ben dragende devijn
        Laat gij mij // dencket vrij
        Wat met waarheid ik beken
        Waar ik heen // ben ghetreen
        Niemand wetet waar ik ben
        Want mijn traneleyt
        Als de swane schreyt
        Mits ik ben verleid.
      
      
        In zwaar lijden ende druk
        Moet ik dus leven ziet
        Met patientich ongeluk
        Tot dat mijn Lief eens biet
        het Zoete woord // van confoort
        Ende helpen goedertier
        Mij Minnaar uit 'tghevaer
        En dit overgroot dangier
        Ik beklagen // mag
        Al mijn dagen // ach
        Dat ik ooit haar sach
      
      
        Om mij peizen wilt schoon Lief
        Als ik ben uiter stee
        Denkt om hem die u verhief
        Wat u persoon hem de
        Gij Iongvrou // voor het trou
        Minnen hebdy hem gehoond
        En voor 'tsoet // bitter roet
        In de steed' van lieffd' gheloont
        Dus dan mijn Voogdes
        Onthout deze les
        Adieu schoon Prinses.
