
    
      
        IN mijnen geest // moet ik veriubileren
        Nooit soeter feest // dan de Sabboth des Heren
        Wie hem met lust //  viert van zijn kwade werken
        Van zonden rust // in God die hem kan sterken
        Die heeft zeer plein // den Sabboth recht gehouden.
        God heeft hem rein //  zijn zonden kwijt gheschouden.
      
      
        Ook zalig dan // is hij tot allen stonden
        Alsulcken man // en rekent God geen zonden
        In wiens geest fris // geen bedrog noch qua perten
        Beuonden is // ook niet dobbel van herten
        Maar als Gods knecht // de vrome Job geprezen
        Slecht ende recht // volherdich in Gods vrezen
      
      
        Ook die geen geld // op woeker heeft gegeuen
        Noch geen geweld // aan iemand heeft bedreuen
        Maar leent met vlijt // den armen telcker vre
        Zonder profijt // alleen wt liefden pure
        En geeft haar al // die hem niet konnen lonen
        Dien sulcken zal // in Godes Tempel wonen.
      
      
        Zo wie met vliet // wt lust hier in beclijuen
        Zij zullen niet //  ledig onvruchtbaar blijuen
        In kentenis // van Gods volmaakte lere
        Die heylsaem is // in Christum onzen Here
        God die vermeert // de gaaf zijns geests beseuen
        Wie hem verneert // die wordt van God verheuen.
      
      
        Tis wel geuiert // die zo den Sabboth rusten
        Zulk een versierd // Gods leringe zoet met lusten
        De weeskens klein // beschermt gelijk  een vader
        Voor tgewelt plein // en die als wijs berader
        Hem neerstich keert // dat weesken onbedrogen
        Geschiet, geleerd // in Gods woord opgetogen.
      
      
        Is een in noot // die hem dan troost van herten
        En breekt zijn brood // een ziel vol honger smerten
        Die niet en kan // rusten, eten noch drinken
        Als hij een man // nootdruftich kan bedincken
        Maar hem eerst spijst // wt liefden na en verre
        Zijn licht dat rijst // op als de morgensterre.
      
      
        Als hij bezorgt // de weeukens rein van zede
        Geeft, leent en borcht // elk na gelegenthede
        Die haar zo kan // voorstaan in alle zaken
        Recht als haar  man //  raad tot bidden en waken
        Geestig ghesint // zacht, stil tot allen tijden
        Dat si verwint // haar druk en tsviants strijden.
      
      
        Niemand en kan // tot dezen Sabboth komen
        Oft ten zij dan // dat hij tot zijn der vromen
        Gantschlic versaect //  de wereld met haar lusten
        En niet en  smaakt // dat arts vol van onrusten
        Mer zijn gemoet // gants dracht in God verheuen
        In zuur en zoet //  dancbaerlijcke te leun.
      
      
        Zeer onbesmit // is hi voor God eerweerdich
        Die niet en zit // bij de spotters lichtueerdich
        In haren raad // noch in haar kromme wegen
        Geenszins en gaat // maar contrari daar tegen
        Al zijnen ganck // heeft in het hemels wezen
        Met lof en dank // als een met God verrezen.
      
      
        Zoet en fijn // moet teynde aller geboden
        De Sabboth zijn // daar maar  een is van noden
        Dits teenich een // stil houden God gelaten
        Als Magdaleen // die koos tot haarder baten
        Tbest enig goed // dwelc haar bij is gebleuen
        Doprechte vroed // en is geen wet gegeuen.
      
      
        O Gods geest net // gij komt ons wel regeren
        Gij zijt de Wet // niemand heeft v te leren
        God heeft v voort // den heiligen ghegeuen
        En na zijn woord // in  haar herten geschreuen
        Gij zijt een licht // lanterne thaerder voeten
        Na Dauids dicht // twelc zij nauolgen moeten.
      
      
        Och liefden graat // gij band alder secreten
        In v bestaat // de Wet en de Profeten
        Gij zijt niet smal // het eynt en  de hooftsomme
        Der Wetten al // haar oorsaeck en waaromme
        Na Schrifts wtlech // groot bouen alle gauen
        De hoogste weg // de rustdag alder slauen.
      
      
        Nu Prinse  God // diet alles werkt in allen
        Gij zijt mijn lot // laat mij van v niet vallen
        Werkt nacht en dag // wilt al mijn kracht regieren
        Op dat ik mag // in v den Sabboth vieren
        V jok is zoet // v last is licht om dragen
        Wie leert ootmoed // zijn ziel rust tallen dagen.
