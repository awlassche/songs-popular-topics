
    
      
        O Heer wanneer ik met aandacht
        De grote kracht, der liefden mercke,
        Die gij aent vervallen geslacht
        Van Adam hebt gesteld te werke,
        En hoe gij noch daar onder // doet
        Grote genade wonder // goed,
        Ik uwen Naam loven bijzonder // moet
      
      
        Wat wonderlijk verkleenen // slecht,
        Hebt gij oprecht, ghekroonste // Koning,
        Bewezen dat gij enen // knecht
        Kwaamt zijn van uit u schoonste woning
        En naemt op u der zonden // last,
        Van mij arem ghewonden // gast,
        In lief' aan u ben ik verbonden // vast.
      
      
        Hoe laat gij slechte weten // bloot,
        Dat wijze duizenden verborgen
        Is, al die schoon secreten // groot
        Ws wils, hoe lust u zo te zorgen
        Voor kleen verworpen scharen // milt,
        Dat gij haar openbaren // wilt,
        Het welk gij voor den zeer ervaren // stilt.
      
      
        Hoe spraeckt gij al in het duister // maar
        De Wereld in gelijkenissen,
        En niet met enen luister // klaar,
        Moest iemand van u Rijke // missen
        De kennis? om zo hogen // schat,
        Behoortmen wel te pogen // dat,
        Men vint, al sou men daarom dogen // wat
      
      
        Zo wie begint, aan u bevel,
        En boete snel, doet goederhandigh,
        In u kennis en leering' wel,
        Zal hij worden allencx verstandig,
        Maar zonder dat, hij even // bot
        Blijft in het zondig leven // zot,
        Waar in men niet en kan aankleven // God.
      
      
        Een zeggen in het aanhoren // schrickt,
        Als of den Vader tegen // hetgene
        Hij heren heeft te voren // schickt,
        Dat boze gaan haar wegen // hene,
        Doch den zaaier van het kwade // zaad,
        Zijn zelven maar te rade // gaat,
        Geen bouman goed na eigen schade // staat.
      
      
        Den Huysvader voorwistet // ziet,
        Maar voorschickte nooit kwaad oft zonden,
        Dijn zaad willens gij quistet // niet,
        Al vallet weinig in goe gronden,
        Waarom? om dat tot dezen // zij
        Niet doen het gelove wezen // bij,
        Gelijk gij wilt, om haar genezen // vrij.
      
      
        Gij sijt een Medecijn, en elk
        Roept gij en biedt gezonde // gaven,
        Levende Water, Wyyn // en Melk,
        Maar der zonden gewonde // slaven
        Weigeren laas te kommen // meer,
        Haastende na het verdommen // zeer,
        In onschuld zij moeten verstommen // Heer.
      
      
        Ziet, mist men u rechtveerdig, kleed
        Opt voorbeschikken Gods te // maken,
        Gheen onschuld men dan weerdig // weet,
        Van onbereid zijn, oft niet te // waken,
        Als men handen en voeten // bindt,
        Dus nu men tijd ter boeten //vindt,
        Al willens wij niet dwalen moeten // blind
      
      
        Een plaatse van begin // bereid
        Is er, ja vant begin der // Wereld,
        Daar werden zij al in // geleid,
        Die met deuchden werden bepeerelt,
        Maar de ander werd ghebleken // hier,
        In het eeuwig fel ontsteken // vier,
        Och men schromt daar van te spreken // schier.
      
      
        Gods voorschicking', en willen // noch
        In eenderlei ten mag niet liegen,
        Maar waar in zoudt verschillen // doch?
        Hij wil altijd, zonder bedriegen,
        Dat alle Mensen zouden // zijn
        Inder waarheid behouden // fijn,
        Och of zij al daar toe doen wouden // pijn.
