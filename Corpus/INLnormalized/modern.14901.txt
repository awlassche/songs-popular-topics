


DEes Goddelijcke Nimph is zo gezind
Dat zij noden Liefde weerlief minnen
Want d ’ Harder stout // van zoet Wout
Die zij al schempende gaan hout
Kranste met Laure kruid // deze Bruyt
Knielende met bidden
Of Tyter klagen // om hulp haar vragen
D ’ afkerig Maecht // zeggen altijd neen


Hoe dikwijls Tyter hebben verlaten ’ t Vee
Om deze scholen Galete
Quelent van min // om deze Goddin
Vluchten haar na ten Boswaert in
Doch as zij hem komen zaak // t ’ gelach
Klonck door het Bosjen heen
Of Tyter klagen // om hulp haar vragen
Dafkeerighe Maecht zeggen altijd neen


Lest vont ik haar alleyn bij een Fonteyn
Wies haar Albaste Handen reen
Kuerschuylde ras // wat vreugde ik was
Als zij enten Waeter lobberent was :
So haast zij ik wier gewaer // laten daar
Myn doen alleen
Of Tyter klagen // om hulp haar vragen
Dafkeerighe Maecht zeggen altijd neen


Hadieu versteend Vrou // helaas ik schouw
Duer u stuersheden van u ! Trou
Dees pijn onzoet // die jij ik doen
Ghy zelf in ent lijden moet
Dit licht Lofje teer // hebben meer
Beweghen van u stuerheen
‘ K daal grafwaert in // vaart wel Goddin
In plaats van Min zeggen ja voor neen
