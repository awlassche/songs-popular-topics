


Met Godes hulp laat ons beginnen
En ons verzinnen // den rechten weeg te gaan
Die ons nu leed ten Hemel binnen
T ’ ghene beminnen // dat eeuwig zullen bestaan
Haest komen de dood en elk een moet daar aan
Het dijl beseffen // is hoog opheffen
Vruchten wild geven // na des Heeren v ’ maan


In Godes hand staan al ons bouwen
Corts afghehouwen // zo worden wu subiet
Waer den boom vallen t ’ zeggen enten benauwen
Oft vreuchd ’ t ’ aanschouwen // daar blijven hei alletijt :
Wie gheen vrucht en brengen en zo zijn tijd verslijt
Sal men beroven // die vreugde hier boven
Altijdt verschoven // zijn en vermaledijd


Schenen die niet wel wijs van raad
Die Gods genade // verzuimen heel onvroed
Eertijdts : men kan nu ook te spade
Afstaen van ’ t kwaad // en hem schikken tot god
Dinckt dat een bloemken sieren haast af vallen moet
Al sietment groeien // t ’ zullen niet eeuwig bloeien
Wilt u bemoeien // na het Hemelsche spoed


Reyn wild u schouderen bogen
Voor Godes ogen // nu onder zijn last
Soo licht dat wi hem dragen mooghen
T ’ dienen aenghetooghen // zijn jok sieren wel passen
Wilt dat aanvaarden // zo men na t ’ goud nu tast
Ghy woord ontladen // van d ’ eeuwig versmaden
Deur Gods genade // in zijn Bruyloft een gast


Richt u gemoed in Christi wees
Soo sullen na deze // als hei komen zullen
Ghy ook dan zijn van hem prijzen
En uitgelezen // voor zijn leven ghetal
Och zulk blijdschap verkiezen doch boven al
Van s ’ Weerelts vreugde // en al haar gheneuchde
Geeft u tot deuchde // hier in deze Aerdsche dal


Tot Godt wild roepen eend klage
Sonder versagen // nu om zijn bijstand
Segt : Lieve Heer na u behagen
Wilt ik nu dragen // ja leggen met der hand
En ik vergaren in u Hemelsche laden
Heere rechtvaardig // maken ik daar toe waardig
En nu volhardig // met kracht aan elk kant


Soo zullen ik in mijn hert ’ opspringhen
Als ik deze ding // alzo bevind ’ bij ik
En ik deze hoop zullen omringhen
Soo zullen ik zingen // u Lofsanghen blij :
O leven Heer mijn helper wezen doch gij
Ghy Godt almachtig // wezen ik ghedachtich
Mijn bidden klachtig // laat nu komen voor bij
