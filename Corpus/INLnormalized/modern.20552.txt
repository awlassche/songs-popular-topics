


OCh ! hoe vast staan hei o Heere Die op u betrouwen
Op u Rosteen t'uwer eer // Heb ik vast bouwen ;
Hierom zullen ik niet bezwijken // Want in al noot
Sal mijn vaste bouwsel blijk // Teghen al aanstoot


Gy beschaamd al mijn Vyanden // En haar wreed ' besluit
Maect gij in 't geheel te schande // En redden ik daar uit :
Doen ik menen uit te roden // Mijne snood partij
Gingh den Heer mijn Vyandt doden // En ik raken vrij


Die u liefde mogen genieten // O ! wat vrede ' en vreugde
Gaet gij stadig op hem gieten // Die hem zo verheugd
Dat hei moet met aandacht zingen // Vrolijck u lof
Hier toe voel ik ik ook dringen // Hier toe heb ik stof


Komt mijn tong de kracht t'ontbreken // 't Herte blijven vervuld
Stadigh wil ik van u spreken // Want ik weet gij sullen
Altijdt zijn in mijne wegen // In vreugde angst en node :
Ick vrees ' niet al heb ik tegen // Werelt Duyvel Doot


Treet tot u al mijn gedachte // Maeckt ik Heer bereid
Ick wil stadig tot u trachten // Met eerbiedigheid :
En wil vrolijk in u weeg // Tot u eer gaan
Och ! maken ik bereid te degen // Dan zullen ik vast staan


Kond ' de vreuchd ' in 't hart blijven // ( Door u goedigheid )
Die soud ' uit mijn hart drijven de Werelts ijdelheid
't Herte kan het al niet vaats // 't Wil niet zijn moeien
Om qua lusten in te laten // Want het overvloeien


Soud ' men Heer u niet beminnen ? Die u zoetheid smaken
Komt de groot Rijcdom winnen // Die recht is volmaakt
Want gei kont het hert vernoegen Maer noden staan of eer
't Beste gaan gij ik toevoegen // Lof zeggen u mijn Heer
