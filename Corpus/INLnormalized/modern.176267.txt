


VOor 't al te schittrich licht // dat Venus uit haar lampen
Gheblasen hebben stichten // in u hel aangezicht
Hemel en Aerde zwicht // en afgrond met haar dampen


De StroomGodinnen glad // die door den Aemstel zwier
O pracht van ons Stadt // hebben een nijd vatten
Op u aanschijn en dat // met nygend voorhoofd vieren


Zy kwel haar gemoed // met honderd duizend smarten
Om dat u schoonheid doen // onsteecken in haar vlieden
Met onleschbare gloed // der WaterGoden harten


Met moedeloos geween // deze Meereminnen kwelen
Dat uw volmaakt leen // van boven tot benenen
Met nieuw kunst ghesneen // hen al haar Vryers stelen


Een troostje hebben zeggen // dat de hoop van te beleven
Dat door uw strafheid gij // den tijd sullen gaan voorbij
Van zoet vrijerij // en hebben niet bedreven
