


Alle ik betrouwen // stel ik op Godt den heer
Alleen op v ik bouwen // verlaat ik nimmermeer
Ghy zijn ik troost ik toeverlaat
Daer al ik hoop op staan
Voor schande en oneer // behoed ik doch o heer


Want gij o Godt prijzen // helpen ik uit al noot
En gij kont ik genezen // door u genade groot
Dus bidden ik vroeg eend laat
My toch met u hulp bijstaan
Voor schande en oneer // behoed ik doch o Heer


Wilt ik doch regeren // v arm schaepken teer
Dat ik ik mogen vernederen // ootmoedig na u leer
So zullen ik verlaten zaad
Die wereld met haar boos vermaan
Voor schande en oneer // behoed ik doch o heer


O Godt almachtig // verhoren ik met daad
Want ik ben v sieren klachtig // ik doch niet en versmaad
Laet ik doch beschamen niet
Maer helpen ik doch uit deze verdriet
Voor schande en oneer // behoed ik doch o Heer


O Godt der prinschen // laat ik zijn onvertsaecht
Want al ik winschen // van v arm dienstmaagd
Syn u bekend o heer divyn //
Verlost ik doch van druk en pijn
Voor schande en oneer behoed ik doch o heer
