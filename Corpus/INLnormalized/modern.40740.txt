


ICk trouwje ' ik bouwje ' ik hab mijn dwaan gruwndeare
( It gong it run it druwck it klieu zo 't wol : )
Yn God ik stân fest op uwz ljeave HEERE :
Ho guwchjes ' toe mijn ziel ijn laeyts in schrol
OEf imme du op God stict aat koe deare :
Flueg sizzes ' suwggelfaey berg ' op berg'dol
Uwz sjietreauw ' sil dogg ' deadlijck du beseere


Op 't waard de died ' Want loytse ! du godleaze
Struwp't mouw op mei ' bleak'rende ' earmen bleat
Fet boage ' in pees ' hei spannetse ' elk mei freeze
De boute er op fornijnnigd az de dead
In dozoer schuwlle ' ontsjeande wede nogg ' weaze
Om sjietten zo de ' oprjuecht ' naettinckopnaed
Hy de rek ' hei leed hei de lijv'leaz wiermen eaze


Ja sijcker ! ja stiette ' om du fuwneminten
Fen rjuecht in war : Het het du goed bedreaun ?
Mar de ' HEERE ' iz ijn zijn sillig stede ' heilig Tinten
Sijn Throone ' iz heag ijnne ' Hijmmel onbekleaun
Sijn eag prieuw't minscheberne vals minten
Hy prieuw ' de ' oprjuechte ' ijn 't libbens buwck opschreaun '
Mar snoeien du godleaze ' uwt as ontijgge ' ent


Hy schil op du godleaze rein litte
Fjoer zwavel strijckjearn heylle ' ijn twierrewijn
Dit de 't eyndeel ' foor du godleaz God forjitte
God iz rjuechtfirddig dat 't har ' hels pijnn '
Hy het gerjuechtigheyt ljeaf môt elk witte
Hy laket de ' oprjuechte ' om mei blier oonschijn
Ja sill ' ijn him 't heag stede ' yvigwol bezitten
