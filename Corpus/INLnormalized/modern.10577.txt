


MYn tijd // verslijt // ik lief prijzen
Met vrees // gans en gaar
Die liggen // subiet // wild dienstig wees
Ghenesen // zijn gheuaer
Maeckt dat hei waard gewaar
V hulp tot hem genegen
Door praat // gelaat // zijn hart verslaan
Met raad // bijstaan // in deze nieuw Jaer


V hebben // beleefd // dienen jonkvrouw
Ghetrouwe // deze dienaar/diener
Hem geest // hei beven // blijdschap voor rouw
Voor trouw // vreugde eerbaar
Maeckt v en hem eenpaar
Soo zullen zijn druk cesseren
Met spoed // lief doen // al v begeren
Hem zoet // ontmoet // in deze nieuw Jaer


Dat hei onvrij // v troost moet deruen
Doet steruen // vreugde maar
Seer blij // hopen hei // v hart te eruen
T'verweruen // zullen voorwaar
Doen slijten t'lyden zwaar
Dat hem nu kwellen gestadich
Dus vrouw // zijn rouw // verdrijven genadich
Die iou // dienen trouw // In deze nieuw jaar
