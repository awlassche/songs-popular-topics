


MEt Broodt // met Bier // sieren fijn met Wijn
Pastey // Lamprey // met Haes en Conijn
Met Hoen // Capoen // Wiltbraet // verzaad
Elck is // gewis // op deze termijn


Ons hert is heel met vreugde // verheugd
Ons moet // sieren zoet // strekken tot gheneucht
Wy zingen // en springen // wi klinken en winken
Versterckt o Heer ons jeugd // en deugd


Men zieden hier staan // zieden braden
Die sparen // die Taert // die is beladen
Mersepeyn // met klein // beginnen het hint
Die haan // en zwa // kan niemand schaen


Wy danken u veel meer // o Heer
Dan wi // sieren blij // u schaepkens teer
met monden // verkonden // versterken en werk
In ons nu voort // u wart en leer
