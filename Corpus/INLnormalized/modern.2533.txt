


Lief wezen groeten // gij die mijn gemoed
Soo sachjes kunde strelen
En lek voet // met meer als Hemels god
En leven lonckjens zoet
Die u ooghjens ik bevel


Als gijse slaan // en liefelijk gaan
Daer lodderen me spel
In mijn ghelaat // zo voel ik inder daat
Een Vreuchde zondaar maat
In al mijn Lichaams deel


Want door u leen // zo vlo heen
Lieffelijcke straal
Soo onghemeen // mijn verstand is te kleden
Om levendig met reen
Mijn blijdschap af te malen


Mijn bloed dat wald // dat bruisen dat brallen
Met innich kryoelen
In dier gestalte // mijn hart van vreugde ontvallen
Maer 'trecht zek weder bald
Door 'thueghelijck ghevoelen


Dat gij hier bent // doch ik bekend
Voor u ik Vriendinne
Als gij afwent // u ooghjens en die zenden
Na iemand hier omtrent
Soo mis ik schier mijn zin


O daagheraat // wanneer gij staat
En ander horen spreken
Of met haar praat // 'tvermoeden dat is kwaad
Mijn dunken ik waard versmaat
Dies wil ik 'thartje breken


Als op u schoot // door vrijheid groot
De Jonglinghen vallen
En kusten bloot // u Roose Lipjens root
Ick zacht en zacht ik dood
Met bitterheydt der Zielen


Ghebenedyt // zij uur en tijd
Want mijn uitverkoren
Heeft mijn verblijd // en van Wanhoop bevrijd
Vermits zij ( wie het spijt )
Haer trouw ik hebben zweren
