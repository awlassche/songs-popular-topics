


EEn roede groen // uit Jesse Stam spruiten
de Winters seizoen // wi 't vruchtbaar hout genieten
Wiens vrucht is spijs // om de ziel te verkwikken
Heylig en wijs // met noch meer ander stik
Die door zijn kracht // levendig overvloeien
In ons brengen // als rank op te groeien


Dit vruchtbaar hout // staan aan een schoon Reviere
't Brengt twaalf fout // vruchten van waard dier :
En op een Jaer // al maand zijn oogst
Voor al die daar // van te rapen beginnen
Zijn geresolveerd // voor al vreemd kwaal
Ongesommeert // wie kan zijn deugd betalen


Wie hier de smaak // van zijn vrucht hebben krijgen
't Geeft hem oorzaak // om te meer zijn genegen :
't Is in den mond // boven spijs der sterk
Het maken gezond // in den Lusthof te werk :
Men wort daar door // voorzichtig en kloekmoedig
En om wel voor // 't kwaad te zijn verhoedig


Prins u lust // zeggen in Gods Wet gestadig
Vierigh gerust // in Gode sieren genadig
Gebruyckt deze hout // tot een staf in u handen
Het maken u stout // te gaan door de warande
Van 't Jammerdal // daar men veel verdriet merken
