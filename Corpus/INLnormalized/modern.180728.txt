


DEn Heere triomfant // zeggen prijs en dank ghestade
Die bijzonder // wonder//lijck zijn volk bevrijd
Want met zijn sterk hand //om Israel voor schade
Te verlossen // rossen // waag en subiet
Wierp hei om zondaar sparen // zo dat d ’ Egyptenaren
Dreven op de baren // aan den oever dood
Die Israelijten waren // dikmaals verlossen van haren
Vyanden en bezwaren // angst eend noot


Christus Hoochpriester groot // hebben om eeuwig te blijven
Tegen zenden // vinden // een Verlosser god
Door zijn eigen bloed root // tot een zalig beklijven
Hier voor leven // welk // hem ontvangen zoet
Haer woord de macht ghegheven // kinderen Gods te leven
En Godt hebben verheven // zijn nieuw verbinden
Met die zo zijn drijven // om enigheid t ’ aankleef
Nae Gods Wet schrijven // in s ’ hart grond


Al wiek ongezond // veel argherlijcke leden
Die verdwijnen // heen // gaan en d ’ enigheid
Des Gheests verlaten rond // ja en den band des vreden
Dat haar laatst // t ’ kwaad // woord // zo die Schrift zeggen
Godt blijven noch zondaar mijden // bij zijn volk t ’ al tijden
Die hem oprecht belijden // voor deze zullen
Hy schermen en strijden // zij en sullen niet glijden
Al viel ook bezijden // een groot ghetal


Al waar dat heel eend ’ al // die Weerelt t ’ onder ginghe
Al verdrinken // zinken // bergen in Zee
Soo zullen // vrij van den val // de stad s ’ Heeren woning
Noch gerustich // lustig // blijven zondaar wee
Het Wijsemans gesticht // Onbeweghelijck dicht
Op des steen ghewichte // niet en vergaan
Dat vredelijk gezicht // Zijnde vol klaar licht
Met oordeel gherichte // eeuwig bestaan


Dat huis Gods delicaat // liefelijk met zoet behagen
Davids hert en zin // binnen // waren daar
Iae bad zijn toeverlaat // om daar zijn leven dag
Met verheugen // mogen // blijven eenpaar
Om daar zondaar verflauwen // den schonen Godsdienst t ’ aanschouwen
Met een vast betrouwen // in God vervroyt
Maer die bezijden bouwen // haar aan t ’ Hooft niet en houwen
Sy komen in rouwen // sieren wijt verstrooid


Princelijck Herder noden // en verlaten gij u Schapen
Die u woorden // horde // gehoorzaam zijn
Met u te zijn kooien // al haar ghenoechte rapen
Doort volharden // waard // eeuwig termijn
Gheluckich en prijzen // van al druk genezen
Een weiden ’ uitgelezen // in s ’ Hemels plein
Vinden zij boven deze // vol rust en onverknesen
Haren drank zullen wees // s ’ Levens Fonteyn
