


In Jesus naeme // Broeders eerzaeme
Stelt u eenpaar // Nu te beginnen
Hem te beminnen // Al in deze zoete Nieuwjaer


Een hart rein // Ook anders geyne
Dit de al voorwaar // Minnen alleen
't Kindeken klein // Al in deze zoete Nieuwejaer


Hij hebben geleden // Hy is besnijden
Dit de openbaar // Wilt u besnijden
Zonder vermijden // Al in deze zoete Nieuwejaer


Zyn heylig sterven // Genoegte derven
Dits ' openbaar // Die doen verwerven
Dit hemels erven // Al in deze zoete Nieuwejaer


Och Maria scholen // Bid u Zoone
O Maget klaar // Den God hier boven
Wy eeuwig loven // Al in deze zoete Nieuwejaer


Vader almachtig // Zone eendrachtig
Heylig Geest klaar // Een God Dryvuldig
Verheft ons schuldig // Al in deze zoete Nieuwejaer
