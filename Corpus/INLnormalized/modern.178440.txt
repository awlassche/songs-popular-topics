


Comt te feesten // zuiver gheesten
Godts dienaar/diener groot
Die daar boven // in Godts hoven
Sijnen lof verbreiden // tot al tijd


Helpt ons zingen // grond ding
Van een zuiver Maeght
Die ons ’ leven // hebben ghegheven
Brenghende hier voort // Godts eeuwig wart


Hoogh prijzen // uitgelezen
Maeght en moeder reen
Naer u wenscyhen // al mens
V zoeken groot en klein // in d ’ aards plein


Laet doch dalen // u stralen
Der barmhartigheid
Wilt u ogen // na ons bogen
Keert van ons het kwaad // o toeverlaat


Heel doorwonden // deur de zonde
Maeckt ons al gezond
En u daad // vol ghenaeden
Toont ons al gemeen // als medicijn


Al ons kracht // en ghedachten
Zijn tot u lof
Elck een prijzen // en bewijzen
V alderhooghste eer // reen maghet teer


In de zuur // laatst uier
Als men sterven moet
By ons staan // met daad
En nemen in u behoed // Maria zoet
