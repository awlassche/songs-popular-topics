


DEn Heere triomfant // zeggen prijs en dank ghestade
Die bijzonder wonderlijk zijn volk bevrijd
Want met zijn sterk hand // om Israel voor schade
Te verlossen // rossen // wagenen subiet
Wierp hei om zondaar sparen // zo dat d'Egyptenaren
Dreven op de baren // aan den oever dood
Die Israeliten waren // dikmaals verlossen van haren
Vyanden eend ' bezwaren // angst eend noot


Christus Hoochpriester groot // hebben om eeuwig te blijven
Teghen zenden // vinden // een verlossen god
Door zijn eigen bloed root // tot een zalig beklijven
Hier voor elk // welk // hem ontvangen zoet
Haer wort de macht ghegeven // kinderen Gods te leven
En Godt hebben verheven // zijn nieuw verbinden
Met die zo zijn drijven // om enigheid t'aencleven
Nae Gods Wet schrijven // i n't hart grond


Al wiek ongezond // veel argerlijcke leden
Die verdwijnen // heen // gaan en d'eenicheyt
Des Geests verlaten rond // ja en den band des vreden
Dat haar laatst // t'quaetste // wort zo die Schrift zeggen
Godt blijven noch zondaar mijden // bij zijn volk t'allen tijden
Die hem oprecht belijden // voor deze zullen
Hy schermen en strijden // zij en sullen niet glijden
Al viel ook bezijden // een groot ghetal


Al waar dat heel eend ' al // die wereld t'onder ginghe
Al verdrinken // zinken // berg ind ' Zee
Soo zullen vrij van den val // de stad s'Heeren woning
Noch gherustich // lustig // blijven zondaar wee
Het Wijsemans gesticht // onbeweghelijck dicht
Op des steen ghewichte // niet en vergaan
Dat vredelijk gezicht // zijnde vol klaar licht
Met oordeel gherichte // eeuwig bestaan


Dat huis Gods delicaat // liefelijk met zoet behagen
Davids hert en zin // binnen // waren daar
Jae hebben zijn toeverlaat // om daar zijns leven dag
Met verheugen // mogen // blijven eenpaar
Om daar zondaar verflauwen // den schonen Godsdienst t'aenschouwen
Met een vast betrouwen // in Godt vervroyt
Maer die bezijden bouwen // haar ae n't hoofd niet en houwen
Sy komen in rouwen // sieren wijd verstrooid


Princelijck Herder noden // en verlaten gij u Schapen
Die u woorden // horde // ghehoorsaem fijn
Met u te zijn kooien // al haar genoechte rapen
Door't volharden // waard // eeuwig termijn
Gheluckigh en prijzen // van al druk genezen
Een weiden ' uitgelezen // in s'Hemels plein
Vinden zij boven deze // vol rusten onverknesen
Haren drank zullen wees // s'Levens fontein
