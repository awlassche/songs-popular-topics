


GOdt verlaat zijn Dienaers niet
Schoon hei haar laat proeven veel verdriet :
Nae den druk // geest hei haar weer geluk Heyl en lust // troost en
Heyl en lust // troost en zegen En de Hemelrust
Op't Pinxterfeest bleeck'et meest // aan de Geest
Aen de taal // van'er juichen en verhaal ;
't Harte springen // en de tong // vrolijk
zang Godes lof // met een stijl en hoog achtbaar stof


O ! wat vrolijkheid en vreugde
Wat vermaak hebben haar de ziel verheugen
Toen de wind // over haar komen gheswindt
Quam dalen // en er ziel heilig hebben bestralen
't Volck sieren verbaasd //heeft razen // en geblaest
En zeggen // 't is de Wijn die haar verleggen :
Maer een deel // waren veel // i n't geheel
Seer bevreesd // door 't gehoor van'er grond Geest


Maer zij zeggen overluid
't Is gheen drank die ons lipp ' ontsluiten ;
't Is geen most // die 't verstand maken verbost
't Is gheen sap // van de Druyven maar 't is wetenschap
Die Godes kracht // en zijn macht // hebben brengen
In ons hert // dat'et vol van tong worden ;
Godt is zoet // Godt is goed // die deze doen
Tot u baat // na zijn Wil en Alwijsen raad


Daer vloeien in uitheems Tael
Voor het volk een Goddelijck verhaal
Dat'er ziel // en er hart wel bevallen
En doen staan // dat vragen acht ! wat dienen doen ?
Sy zeggen : doen // in ootmoed // recht boet
Valt den Heer // voor zijn heyl'ghe Voeten neer
En dan voort // alst behoren // op zijn woord
Laet dan toe // damen u't Heyligh Doopsel doen


Yder als deze was zeggen
Was terstond tot deze dienst bereid
E n't ghetal // groeien strak dat het al
Quam terstond // op drie duysent/ die i n't Heylverbondt
Waren ghetre'en // onder een // wel te vre'en
Wel gerust // met een ziel en hartenlust
Heyligh Godt leed ons tot // u Ghebodt
Oock met vreugde // dat wi zo blijven in de Deught
