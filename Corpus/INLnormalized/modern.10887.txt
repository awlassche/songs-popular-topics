


Myn tijd // verslijt ik lief prijzen
Met vrees gans en gaar
die liggen // subiet wild dienstig wees
Genesen // zijn gevaar maken dat hei waard gewaer
V hulp tot hem genegen
Door praat // gelaat // zijn hart verslaan
Met raad // bijstaan // in deze nieuw jaar


V hebben // beleefd dienen Ionckvrouwe
Ghetrouwe // deze dienaar/diener
Hem geven // beleefd blijdschap voor rouw
Door trouw // vreugde eerbaar
Maect u nu eenpaar // zo zullen zijn druk cesseren
Met spoed // sieren zoet // al u begeren
Hem doen // ootmoed // in deze nieuw jaar


Dat hei onvrij // u troost moet derven
Doet sterven // vreugde maar
Seer blij // hopen hei u hart te erven
T ’ verwerven zullen voorwaar doen slijten t ’ lijden zwaar
Dat hem nu kwellen ghestadich
Dus vrouw // zijn rouw verdrijven ghenadich
Die rouw // dienen trouw // in deze nieuw jaar


V vlijen // sieren blij // Princes verheven
Beneven u dienaar/diener
Wien spijt // benijden // laat zijn verdrijven
Het sneveen en misbaar twelck hen kwellen voor als na
Wilt doch zijn pijn verzachten
Dat hei // lief bij // u mogen vernachten
Heel vrij // en blij // in deze nieuw jaar
