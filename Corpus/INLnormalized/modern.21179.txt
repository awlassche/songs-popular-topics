


Hoe boosaardig // zijn nu de Menschen kind
Van ons tijd ?
Die zo vaardig // eens anders eer verminderen
Door Haet en Nijdt :
Hoe verkeer hoe nijdig spreken de mond ?
Veel vals ding // ja beuselinghen
Om zo te brengen // haar Naesten in 't verdriet
Verdriet verdriet ja te storten in den grond


Loosen Klapper // Faemrover Deugtverberger
Elcks ongerief
Leugensnapper // weer vrij dat gij veel erg
Zijt als een Dief :
Want een Dief zoeken maar het tijd'lijck goed
Tot zijn behoeven // Maer zulk Boeven
Sal Godt bedroeven die haren vriend zijn eer
Sijn eer zijn eer nijdig treden onder voet


Doch de Vrome // die op Godes goedheid hopen
Sullen 't verdriet
Boven komen // Want Godts genaed ' staan open
Die tot hem vlieden :
Laet de Booz ' uitspuwen haar venijn
Die Godt betrouwen // en op hem bouwen
In haar benauwen // die sullen noch in 't einde
In 't einde in 't einde wel gemoed en vrolijk zijn
