


EEn waard vrouw // doen ik veel vreugde bedryuen
Al hebben zij rouw // somtijds dat kan niet schaen
Op eer en trouw // min ic die fleur der wyuen
Als ickse aanschouw zo moet ik zingen gaan
Want zij ik nauw // beuryt voor al vaar
Op dat zij mogen ghedencken // mijn //
Zal ik ik herteken schenk // fijn
Myn lief eerbaar // tot haren nieuwen jaar


Al was rijzen // tot indië dood vol vrees
Myn jonstich hart // ja indië grond versmoord
Sy kant genezen // als zij dokter wil wees
Van al smart // met een wellustig wart
Dus wil ik voort // verlaten droefheid zwaar
Wil zij slechts liefelijk winken // neer
En trouw ghewillich schenk // weer
My voor een aar // tot mijn nieuwen jaar


Venus kind // wild haar enten hard steken
Dat zij beminnen // een weinig tmijnder baat
Op dat verwinnen // ik veel ootmoedig smeken
Dat nu noch enten // alderquaetste blaetgen staan
Ick roep om raad // al die god te gaar
Dat zij ik helpen minnen // toch
Op dat ik mogen gewinnen // noch
Een blij maar // voor t ' geheel nieuw jaar


Groote geneugte // met dansen en springen
Zal ik volbrengen // om ik Princesse god
Of zij deur vreugde // of deur wellustig zingen
Eens willen ghehinghen // Dat ik die vruchten zoet
Plocke tot boet // van ons liefde eenpaar
En ickse mogen verwarmen // lang
En rusten in haar arm // blank
Dicht neuens haar // van een jaar enten aar
