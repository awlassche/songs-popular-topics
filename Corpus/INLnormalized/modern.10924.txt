


De liefde blind // gaan ik bekend
Met groot torment // Nu zwaarlijk plagen
waar ic ik wenden // Zijnde absent
van u lief jent // Soo moet ik klage
In verdriet // Door liefde heet men ik zieden
gaan te niet // En gans verjagen
dus zuiver grein // Acht u zo klein
t ’ aanzien certein // Mijn liefde reen
wild mijn gheweyn // Doch helpen dragen


Want u licht klaar // wees eerbaar
Doet u dienaar/diener // in liefde derven ‘
V zang eenpaar uitspraak voorwaar
Brengt ik bezwaar // en doen ik sneven
V bestier // brengen ik hier veel dangier
Dat ik schier // rust gaan beweeg
Dies ik doch gunnen // Lief u verbinden
So zullen terstond // mijn hert doorwonden
Wesen gezond // in vreugde leven


Goddin u keren // en u vernederd
Hem die u eren // laat troost verwerven
Liefde useren // kwaad raad afkeren
Nae u begeren // suldy vreugde erven
Des nijder macht // nu veracht
Want haar kracht // alleen trachten ’ V te bederven
dies nu verlaat // Door mijnen raad
deze vals praat // T ’ gheschiet door haat
en ik bijstaan V druk zullen sterven


Princes zoet // hart lijf en god
In zuur in zoet // zullen u standig eer
Soo gij wel doen // Lief met ootmoed
Dat gij u spoed // ik te salveren
Reyne vrucht // en geneugte zullen u jeugd
Door mijn deugd // altijd useren
Hartseer en pijn // nijder venijn
Met vals schijn // zullen van u zijn
Ghy lief ik // troost gaan vermeren
