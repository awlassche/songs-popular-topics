


BLoeyende Jeucht // gij mogen // wel zijn verheugd
Want dat u lang hebben ontbreken
Is nu beweucht // met vreugde // uit goed deugd
Dat in den mei sieren zoet van roken
Meybloemken liefelick ontluiken
Die winter lang // sieren wrang // met stranc // bedwang
Is vrij en vrank voor ons duiken


Schoon melodie // ook wie // zullen deze scene
En hem na deze mei niet pogen
Melancolien // moet vlieden // men zieden geschieden
Dat de Vyghboomen cnopen togen
En den wijnstok crijght nu ogen
Noyt oog opslag // en zaak // den dag // die mogen
t ' Winters geclach dus doen verdrogen


Noyt mei zo zoet // en stoet // gebloeid enten bloed
Compt hier mijn Bruyt mijn Coninghinne
Valt ik te voet // met spoed // storm regen vlieden
Is al vergaan leven Vriendinne
Aensiet hoe de Tortelduyvinne
Int Lusthof zingen // ontspringen // kom drinken //
Hier drank die blinken uit rein min


Vercooren graan // sieren zaad // wild snellic gaan
Tot de Fonteyn zondaar verdrieten
Den dorst beladen // verstaan // klaar als een tra
Sietmen den vlieden daar heen vlieten
Van alles mogen gij daar genieten
Rijeckende kruid en fruit den buien delen uit
Des Meys viertuyt doen al opschieten


Een Roosken root // mijnioot // staan daar sieren bloot
Onder de Doorens uitspruiten
Van 't hemels brood // te noot // uit s'Vaders schoot
Is over al het Velt begieten
t'Lam Syons hebben Boeck ontslooten
t'Hantschrift in twee gescheen tgeween niet kleden
Is al gemeen daar deur verschieten


Princesse fier // Rosier // lief goedertieren
Wilt voor den Winter niet meer schromen
Maer maken u schier // tot hier // u princier
Heeft zijn loon met hem genomen
Hy zullen haastelijk tot u komen
In s'Hemels Throon idoon de kroon sieren schoon
Wert u loon met al vromen
