


EEn Liet // wil ik gaan beginnen
't Bediet // sullen gij ook wel beminnen
Want al mijn hart en zin
Die winnen // nu niet dan vreugde
Dus schenckt en brengen
'tHert denken // en wenken // op u reen jeugd


Ick moet // deze nu schoontjens uit vegen
Het voet // de mens in al wegen
Dus moet // ik deze te degen
Gaen ledig // en klaren uit
Volght na // zeggen ja
Het gaan sieren dra al in mijn huid


Dit glas is nu ledig drinken
Seer ras zat 't worden schenken
En niet lang laten pronk
Maer lonk // hier aan mijn zeggen
Op dat // het nat
Int vat // ons mogen maken blij
