


UWz ljeav ' HEER ' wol ik toe dizz ' stuwn '
Lofsjongje ' uwt hert ja hertegruwn '
ik Wol kiel hert pin in tombe ijn 't herpjen
Om 't wond 're Goads klear t'uwttjen scherppje


Yn jong sil ik steez om blier in blij
Op donssje ' in gaestformeytssje ik :
Jôn Namme sil ik for âd in jong
O allerheagste ! lofpsalmsjonge


Om dat mijn fijnne ' iz efterbeck
Oertomle ' in fluechtjend ' romt it feck
Hy plof 't forsleyn ' hei mot fordearre
Fen gunnen eansichte ' holpredd ' ljeave ' HEERE


Want Y habbe ' HEER ' mijn lijk mijn rjuecht
Mijn rjuechtsake ' oefdien lijk ' in sljuecht '
Y sietene ' op gunnen Troon O Rjuechter !
O rjuechtvirddigheyts lijckbesljuechter !


Dy goadleaz ' Heyd'nen habbe ' Y schieden
Trog Jon gleon scheldjen gled fordien
Herr ' Namme habbe ' Y gruwnn'fordijlge
Ja yvig yvig tor forwijlge


O fijnne ' ! iz nieuw trog leaz beleid
't Forwoestjen eynge ' yn yvigheyt ?
Sint ' nieuw uwz Steden az opschobbe ?
In iz har ' huwgg'niss ' djiep bedobbe ?


Ne fijnne ô en : God du 't al wit
Al sjoecht al heart zalm ' yvig zit ;
In het zijn War toe 't rjuecht ontditsen
In het zijn Troons gordijnne ' ontlitsen


Hy schil psalm ' rjuechtje d'hele wrad
Nimmen ontsjeande ' az oerste Fâd
In uier Ljue zok oordeel lezze
Dat sil ruwn om rjuechtmiettig wezze


Den iz hei toflechte ' heagte ' in schuwll '
Jerm ' du fortreaun ' gâlje ' yn druwcks kuwll '
Ynn ' grijmme tijd yn grouwle stuwne
Iz hei de ' oerlest'ge ' yen honk befuwne


Dearom HEER ' du ion Namme kin
Festlijck fortrouw't mei hert in zijn
Om dat Y habbe nea forlitten
Wa jong sijk't yn oprjuecht meywitten


Psalmsjong ô sjong in sjong al weer
Dy t'Syon wennet de ' oerste HEER '
Forkundje 't folck zijn heage ' ynsond 're
Greatdiedig boppemiette wond 're


Hy sijk't hei tinckt 't fromm ' bloed forstirt
Wier oer hei heftig hietfâll'g worden
Hy sil forjitt'leaz wis forhearje
't AEng ' rop fen du zijn helpen ' bejearje


Wezz ' ik genadig HEER ' stea bij
Loytse ' het mijn hater dwaene oer ik :
O ! Y du ik 't fordear ontfnelje !
Ja uwt deads portte ' in graefstuwt helje


Op dat ik gunnen great lof fortolck '
Yn Syons portten om gunnen folck
Dear binnen ik op 't heaghlijckste ' hertfolnuwgge
Yn gunnen locksilligheyt forhuwgge


d'Onrjuecht ' worden homppe ' yn 't selle wiel
Uwtyerde for de ' oprjuechte ziel
Herr ' foet worden finzene ' yn du netten
Dy ja toe gluwp for oog ' uwtsetten


Doz iz d'HEER ' kinbar heag ontsjoen
Hy het rjuecht hoff'ne mei fetsoen
Dy godleaze ' iz fortijsd ( ô pleagen ! )
Yn eyn hann'wirck ( ô spijt ! ) bedreagen Higgajon Sela !


Dy godleaz ' schil doz yn 't fordear
Oerbeck neye ' hol toe nimme ' in keer
Dy Godforjitt'ne ' ( yn 't quea oerjuwne
't Nearsilligpaedspoarbjuest 're ) huwne


Mar de ' earm ' worden naad yn 't eynleaz ' soer
Forjitten hei de God waard in djoer
Eack iz d'illind'ge lockforwachtjen
Forlern naad mar yn Goads heage ' achtjen


Oereyne ' ! op op ! HEER ' lit du minsch '
Naet kreftigje ' yn zijn gruw'le tinsch '
Lit de Heyd'nen foor gunnen eansicht ' schoddje
In trog gunnen oordeel blea fordoddje


O HEER ! oonjeyse bang in blea
Deads kijll'kâdde ' eangst ' trog hert in ljea
Lits ' zo bedearje ' in rjuecht forsinne
Datse ' earme naet'ge Minsken binnen Sela !
