
    
      
        Min hartge, dat wil ic Gode opgheven
        Ende doen my alle menschen of,
        Ende soeken troest in alle minen leven,
        Daer icse ghestadich vinden mach.
        Ay laes, Ay laes, Ay laes,
        Heer Jhesus mijn,
        Meer druck, meer dan ghelucx,
        So lijdt dat herte mijn.
        Ay laes, Ay laes, Ay laes,
        Heer Jhesus mijn,
        Meer druck, meer dan ghelucx,
        So lijdt dat herte mijn.
      
      
        Och, harte ende sin wilic Hem gheven,
        Maer boven alledien ic weet:
        Dat ic dus spade heb begonnen,
        Dat is my van gueder herten leet.
        Ay laes, Ay laes, Ay laes,
        Heer Jhesus mijn,
        Meer druck, meer dan ghelucx,
        So lijdt dat herte mijn.
        Ay laes, Ay laes, Ay laes,
        Heer Jhesus mijn,
        Meer druck, meer dan ghelucx,
        So lijdt dat herte mijn.
      
      
        Conde ic dat weesen Gods ghebruken
        Alsoe vriendelijc inder sielen mijn,
        Soe soude ic alle dinck buten sluten,
        Dat my een hinder mochten sijn.
        Ay laes, Ay laes, Ay laes,
        Heer Jhesus mijn,
        Meer druck, meer dan ghelucx,
        So lijdt dat herte mijn.
        Ay laes, Ay laes, Ay laes,
        Heer Jhesus mijn,
        Meer druck, meer dan ghelucx,
        So lijdt dat herte mijn.
      
      
        Och, daer en kan ic niet toe gheraken,
        Hy en wort in my een stil afgront,
        Dat ic mach sien ende leeren smaken
        Hoe lief tot lief spreect sonder mont.
        Ay laes, Ay laes, Ay laes,
        Heer Jhesus mijn,
        Meer druck, meer dan ghelucx,
        So lijdt dat herte mijn.
        Ay laes, Ay laes, Ay laes,
        Heer Jhesus mijn,
        Meer druck, meer dan ghelucx,
        So lijdt dat herte mijn.
      
      
        Al en soude ic nymmermeer Sijn hulde vercrighen,
        Noch troest tot ghenre tijt ontfaen,
        Nochtans wilic Sijn eyghen bliven,
        Ghestadich vast al inder doet.
        Ay laes, Ay laes, Ay laes,
        Heer Jhesus mijn,
        Meer druck, meer dan ghelucx,
        So lijdt dat herte mijn.
        Ay laes, Ay laes, Ay laes,
        Heer Jhesus mijn,
        Meer druck, meer dan ghelucx,
        So lijdt dat herte mijn.
      
      
        Ic gheef my op, tis meer dan tijt,
        Ic legghe mijn hooft in Uwen scoet,
        Want Ghij temael mijn eyghen sijt.
        Nu helpt ons, God, uut alre noot.
        Ay laes, Ay laes, Ay laes,
        Heer Jhesus mijn,
        Meer druck, meer dan ghelucx,
        So lijdt dat herte mijn.
        Ay laes, Ay laes, Ay laes,
        Heer Jhesus mijn,
        Meer druck, meer dan ghelucx,
        So lijdt dat herte mijn.
