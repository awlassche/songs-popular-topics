
    
      
        Lest ginck ick inde mey spanceren
        Wandelen met Lusten fyn
        Waer door het hert most conforteren
        Jae geheel in Rusten syn
        Vrou venus kaeren excelent
        Sach ick by paeren daer ontrent
        Hant aen hant // seer plaisant // dansen en springen
        Wentelen int groene // gras
        Soudt een quandt // syn verstandt // connen dwingen
        Die gewoon te doene // was.
      
      
        Terstondt quaemen sy voorts gedrongen
        Herpen veelen Luyten goet
        Sy dansten speelden ende songen
        Sulckx tgehoor ontsluyten doet
        Musyka schoon was daer ontrent
        Elck hielt syn thoon opt instrument
        Delykaet op syn maet // soo manierich
        Pas a pas voorts dansende
        Nyders Raet // wert versmaet // puttertierich
        Met vreucht bloomkens schansende.
      
      
        Dees machdekens Lustich van leuen
        Stelden my die Rosen by
        Waerom soud'ick geen vreucht aencleuen
        Die tot sulckx verkosen sy
        Veel bloemkens oock suyuer ende Claer
        Liefflick van Reuck brochten sy daer
        Sulcken vreucht // my verheucht // bouen maetten
        Dacht ick hier end most hooren daer
        Tza ghy Ieucht // maeckt geneucht // met soet praeten
        Tis corage voor ende naer.
      
      
        Terstont quaemen sy voorts gelaeden
        Omden Lust te boeten meere
        Met vlaeden taerten ende gebraeden
        Wyn om druckx versoeten seer
        Myn docht ick haeck naer sulcken geur
        Doen wert de smaeck noch gouuerneur
        Heus en bly // willen wy // bancketteren
        Sprack een van dees maechden fier
        Hout v vry // wie soudy // moghen deeren
        Spyt al diet mishaechden hier.
      
      
        Dus siende alle dees vreucht voor ooghen
        Therten bloet verwermden sy
        Terstondt syn sy toe gevlogen
        Helsden custen narmden my
        Ick hiel my coele op dit termyn
        Maer het gevoele deede my pyn
        Dye nu mocht als hy docht nae syn wille
        Venus hoff omgraeuen siet
        Sulcken tocht moet gewrocht syn al stille
        Maer elck heeft de gaeuen niet
