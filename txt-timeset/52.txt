        Segh wanckelbare vrouw // hoe derft ghy om my schrijven?
        Waent ghy dat u ontrou // voor my versust sal blijven.
      
      
        Neen neen ick weet het wel // wat dat ghy hebt bedreven,
        Oock mee tot wat gesel // dat ghy u hebt begeven.
      
      
        Gedenck die swaren eet // die u en my verbonden
        Te laeten 't geen ghy weet // ick aen u heb bevonden.
      
      
        Was Frederick niet genoech // u weeldich lijf te stillen,
        't Scheen neen Want 't geen ghy droech // had noch een meerder willen.
      
      
        't Was Lucidor wiens brand // u redens kas soo sengde,
        Dat ghy ons waerde bant // als glants vergeeten plengde.
      
      
        Is hy 't dan die u ziel // soo machtich heeft beseeten,
        Dat ghy hem voor my hiel // laet niet my te vergeten.
      
      
        En vly, u met u brunt // ia t' dickste der Bosschagie,
        Daer eensaemheydt u gunt // een middel ter boelagie.
      
      
        Of wel u lust betracht // in duyst're naer speloncke,
        Of in de swarte Nacht // als yeder lecht te roncke.
      
      
        Ga schuyl u vry voor 't ooch // van redelijcke zielen,
        Doch 't geen ghy noyt bedrooch // treet u steets op de hielen.
      
      
        Een knagende geweet // sal u gedeurich quellen,
        En 't swaerste harten leet // sal u gestaech versellen.
      
      
        Waer heen dan wroegent hert // tot decking uwre daden,
        Stick in u eygen smart // om dree te zijn ontladen.
      
      
        Doch echter sal u ziel // voor onse oogen waren,
        Eer ghy gedoemt de kiel // van Charon sult bevaren.
