
    
      
        Verblijt // met vlijt // in dese Meye tijt
        En spoet u nu int groen, en spoet u nu int groen,
        Vermijt // en slijt // de vileyny en spijt,
        Reyn liefde wilt bevroen, reyn liefde wilt bevroen
        Haest u int velt seer coen // om de Mey eer te doen,
        In vriendelijckheyt vergaert // doch met u Liefken paert,
        Bedaert // eer ghy beswaert,
        Verclaert // u aert // en liefd’ by u bewaert
        In dese Mey vermaert, in dese Mey vermaert
      
      
        Int velt // u stelt // met droefheyt u niet quelt,
        Maer vreuchde doch aenvaert, maer vreuchde doch aenvaert,
        Druck smelt, met ghewelt, weest vrolijck als een Helt,
        Toont liefde onbeswaert // toont liefde onbeswaert
        V liefste Lief openbaert // soo werdt u liefd’ verclaert,
        Dan suldy troost ontfaen // als ghy doet sulck vermaen
        Tot die u hert heeft bevaen,
        Dees paen // wilt gaen // met vreucht wilt nemen aen
        Dees Mey seer soet ontdaen, dees Mey seer soet ontdaenmmdees
      
      
        In deucht // verheucht // met u Lief als ghy meucht,
        Int wout seer schoon geplant, int wout seer schoon geplant
        En veucht // geneucht // lieflijck als jonghe jeucht,
        Neemt vrolijckheyt ter hant, neemt vrolijckheyt ter hant
        Spoet u vaerdich int lant // daer de voghelkens triumphant
        Singen met stemmen veel // en prijsen met luyder keel
        Ghelijck die boven al is eel,
        Quereel // t’ghescheel // doet van onse Taneel,
        Soo valter geen crackeel, soo valter geen crackeel,
      
      
        Prins tracht // met macht // na haer die u verwacht,
        In liefd’ bloeyende soet, in liefd’ bloeyende soet
        u clacht // versacht // haestich rijt op de jacht
        Tot dat ghy haer ontmoet, tot dat ghy haer ontmoet,
        Verthoonr u haer met der spoet // als een recht minnaer doet
        Die tot haer lieffde draecht // en haer geerne behaecht,
        Niet vertsaecht // te comen by die maecht
        maer waecht // en vraecht, weest wech eer dat het daecht,
        dat ghy u niet beclaecht, dat ghy u niet beclaecht.
