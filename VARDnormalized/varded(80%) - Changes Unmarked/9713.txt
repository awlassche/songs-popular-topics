
    
      
        HOe magh, o God, het volk zo rasen, koen,
        En Mensen dus uws Woordts vruchtbaerheydt // myden,
        Te zwak eylaes zij doch als dwazen // doen
        Die tegen dy en dyne waarheid // stryden,
        V heylich kint en can geen swaerheyt // lyden
        Meer door vervolg, dan aan zijn leden // teer,
        Zo heden doen, die blint u claerheyt // myden,
        Als Pilaet, ook Herodes deden // eer.
      
      
        Gelijk als daar, in zo versteenden // stad
        Ierusalem, Mensen gheneghen // waren
        Leed doen u kint en volk, ja meenden // dat
        Te roepen uit, met een fel tegen // varen,
        Gaat het noch toe, dan gij door zegen //baren
        Wt eeuwig zaad u noch veel kinders // laat
        Does laat gij noch om wel bewegen // naren
        Heer, uwen Geest, die in veel hinders // baet
      
      
        Doe Farao ook meend' uit te roeien // fel
        Israel wies door u vermoghen // krachtig
        'sGelijcx u volc heeft men zien groeyen // wel
        Daar doch al veel benynders loghenachtigh
        V maert' eylaes zeer loos bedrogen // sachtigh,
        Dat zij het zweerd van u ghegheven // uit,
        Om doden toog, en houdt ghetoghen // machtig,
        Op u huysvrou u zeer verheven // Bruydt.
      
      
        Maar Syons Vorst zal in de wolcken // klaer
        Zijn Bruydt tot troost zeer haast verschijnen // horen
        Zal men geschrei, want alle volken // daar
        Zeer zullen zijn bevreesd voor zijnen // tooren,
        Vergolden werd al dobbel 'tpijnen // voren
        V volc gedaan, met brand en enen // rook,
        Die quelligh zal zonder verdwijnen // smooren,
        Ia eeuwig werd daar druck en wenen // ook.
      
      
        De Bruydt zal hoog uit d'Ardtsche banen // gaan,
        Want bij zijn hoofd moet zijn t'lichame // boven
        'tVerwinnigh Lam zal haar van traenen dwaen,
        Aan slevens born' in die bequaeme // hoven,
        Gherust zij zal eeuwig den Name // loven,
        Met blijdschap groot, van haren weerden // Man,
        Maar vlucht, en sucht, in onveylsame // cloven
        Den rootsten, gij Tyrants op eerden // dan.
