
    
      
        Als Coridon zag aan de rey // om de Mey,
        Elk Harder met een Harderin // naar zijn sin,
        Gescakelt zo speelt hij een Liet // op zijn Riet,
        Daar 't Meysjen op zong // en ee Knaepjen op sprong,
        Al onder de Linde-boom // schoon om de Croon.
      
      
        Doch Erickje dat gij aan de rey // om de mey,
        Gaat spiepele zonder mij // aan u zij,
        'k Was immers zo niet gezeid // licht meyt,
        Maar Erickje is wel // Ik zweer u ik sell,
        Ziet dat is nu evenveel // en ik speel:
      
      
        Mooy benje ja zonder ghelijck, en ook rijk,
        Welgunstigh aan dieje bemint . maar je vint,
        Zo deughdelijck niet alsje schijnt // Ik bevijnt,
        Gij speelt met mijn Hart // Gij speelt met mijn smart,
        Maerr denckt mij hier bij dit u val // wezen zal.
      
      
        Als Coridon 't Lietge had uit // zijn geluid,
        Begaf hem een droeve sucht // in de Lucht
        Deed stenen en wenen de Ieught // om zijn deugd,
        Doch Erickje achte // belofte noch klachte,
        Zo licht is de liefde verdreyt, met de Meyt.
