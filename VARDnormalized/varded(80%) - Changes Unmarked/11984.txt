
    
      
        Waarom wroeten doch de mensen // waarom jaghen zij
        Na de schijn vreugd van de weerelt // die zo haast voor bij
        Vlieght gelijk de vlugghen Arent // of een schip in zee
        Welckers spoor gheen mensch kan vinden // noch van 't schip een stee.
      
      
        Idel mensch de vreugd der Aarden // 't zij hoe schoon die schijnt,
        In een ooghenblick daar henen // als een rook verdwijnt.
        Alle vreugd, 't zij hoe bevalligh // metter tijd verslijt.
        Gheen vermaeck, dat niet ten laesten // ons in d'ooghen bijt.
      
      
        Als een boom is 'svleesch vermaecken // die wel lief'lijck staat,
        Maar haar wortel tot beneden // in den afgront gaat.
        Na het oogh zij schone vruchten // als ook bloeysem draagt;
        Doch de na-smaeck is Gal-bitter // en gheen ziel behaecht.
      
      
        Deze schone Meereminne // geeft een korte vreugd:
        Gheeft ook die haar vriend'lijck streelen // langhe ongheneught;
        Ongheneucht in hert en sinnen // en in ons ghewis;
        Want zij gheeft een bitt're voorsmaeck van verdoemenis.
      
      
        't Honigh-Byken gheeft wel honig // doch met een ook steekt.
        Al te veel van 't zoet genomen // onze ghesontheydt breeckt.
        Matigheydt in alle dingen // is gants prijselijck.
        Overdaet (van God verboden) is verdoemelijck.
      
      
        Schijn-vreucht kan gheen ware vreuchden // of go blijschap zijn.
        Niet kan ons te recht verblijden // welckers eynd' is pijn.
        Niemand ware vreugd gheniet // welckers hert ghetuyght
        Dat hij sigh niet gants oprecht'lijck // na Gods weghen buigt.
      
      
        Overmits zijns herten inspraeck, (ist niet afgebrant)
        Hem ver-oordeelt en kendt schuldig // aan Gods straffe hand.
        Kan een sondigh mensch verblijdt zijn // die Gods straff verwacht?
        Die verblijd is in zijn zonden // is niet wel bedacht.
      
      
        God en Weerelt gaan niet t'zamen // wie zijn vreuchde zoekt
        In de Weerelt buiten Gode // werd van God vervloekt.
        Niemand kan twee Heren dienen // die tweedrachtich zijn;
        Of hij zal den een van herten // dienst doen, d'aer in schijn.
      
      
        Wilt gij vaste vreugd, o! mensen // en recht herts-vermaeck:
        Soecktse niet in 't ydel wezen // dat's te slechten saeck.
        Daar in stelt uws herten blijschap // dat gij God bemint,
        En van God zijt aen-genomen // voor zijn lieve kint.
      
      
        Deucht geeft vreugd; doch zonde treuren // Deucht met God vereent.
        Zonden heeft ook zelfs in 't lachghen // oorsaeck dat men weent.
        Deucht verseeckert van Gods vriendschap // en Gods vriendschap, dat
        Wij als vrienden van ons Heylant // komen in zijn stat.
      
      
        Deucht een vaste wijck der vromen // yder mensch verblijd
        Hier en naemaels, ja op aarden // in een zwaren strijdt:
        Deze kan gheen mensch ontrooven // 't is een vast Cement,
        Die volhart, schoon al de Weerelt // ons veel kwaad toesent
      
      
        Zij ghegront is in den Here // en zijn trouwigheyt.
        Hij vertroost de vroome zielen // in haar swarigheyt.
        Daarom schoon de vroomen treuren // somtijts na het vleysch;
        In haar ziel van God zij krijgen // vreugd en troost na eysch.
      
      
        Staat dan mensch na deuchden wilt gij // vroolijck zijn van Gheest
        Weest voor deuchts ghedaente nimmer // in u hert bevreesd.
        Is haar weg bestroyt met doornen // schrik niet, ga kloeck heen,
        Want haar einde is zo kwaad niet // als 't beginsel scheen.
      
      
        Beter ist hier wat te lijden // om het eeuwig heil;
        Dan om Aertsche vreugd te draghen // lijff en zielen veyl:
        't Komt niet qualijck voor de tonghe // 't bitter eerst, dan 't zoet:
        Die om Gods vreugd laat de zonden // heeft een wijs ghemoet.
