
    
      
        Myn sinnen // ziet // die minnen // hiet,
        U overschoone Maacht,
        Met liefde vast // als een trouw gast,
        Die reine liefde draacht.
        U klaer aenschijn // verheuchtet harte mijn
        Als u klaarheyt // inder waarheid
        Bij mij slech mach zijn.
        Droefheede // vlucht, de wreede // sucht,
        Met al 't verdriet en pijn
        Tot niet ist haast gebracht
        Myn rouwe // snoot,
        o vrouwe // groot,
        Als gij mij maar toelacht.
      
      
        Ik b'nijd // en haat, met spijd // en smaat,
        Dat gij zo bent verkuyst
        Met die gesel // gij kent hem wel,
        Die t'uwent dikwijls ruyst
        Als ik bij u wil gaan // zie ik hem bij u staan,
        Ik ga weder // op en neder,
        Swaermoedich belaen,
        Mijn yver // klaacht, die styver // jaacht,
        En port het hart vast aan
        Dat willend' niet en mach,
        Door teghenheyt,
        
          gheleghentheyt
        
        Verwacht ik nacht en dach.
      
      
        Doch selden // tijd // oft quelt // en spijt,
        Mij staach in myn ghemoet
        Dat gij hier lijdt // of werd ghevrijdt,
        Van sulcken slechten bloed,
        Wiens roem is syn gheslacht // en is zelfs niet geacht,
        Van ellick // een // het wellick // gheen,
        Kennis geeft de macht.
        De luyden // nu // zien huyden // nu
        Al na de weytse dracht,
        Doch is zij best bewaart
        Die merken // set //
        op werken // let,
        En op go geest en aart.
      
      
        Verkoren // schat // te voren // had
        Ghedocht en dan de daat,
        Wel voor u ziet // alst is gheschiet //
        Berouwt het meest te laat,
        Al raed' ik u hier toe // gij meucht wel zeggen hoe,
        Dus beladen // met myn schaden,
        Raackt u wat ik doe?
        Dan waarde // maacht // die de aarde draacht,
        Duyt myn go sorch in te go,
        U welvaert ik besin,
        Kend' u letten doch,
        Annetta, och
        Dat ik u dus bemin?
      
      
        U waerdigheyt en aardigheyt
        Heb ik nooit eer gedaan,
        Als 't wel betaamt, dies ik beschaamt
        Voor u altijd zal staan.
        Ik heb u, lief, gelaeckt, en ben zelfs niet volmaakt,
        Ik soud' wel laten, had zijn praten
        Mij eerst niet gheraeckt,
        O braven dier, 't begraven vier
        Na u begheeren haackt
        En ziet niet na de konst,
        Lief, ik ten dele,
        mij beveele
        In goede jonst.
