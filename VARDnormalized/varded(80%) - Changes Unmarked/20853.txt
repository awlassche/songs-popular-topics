
    
      
        O Grote Catharyn!
        Lof moet U altijd zijn,
        Die op de aerd' // hebt bewaard // uwe le-den;
        Eer zijt Gij al-tijd waerd.
        Door 's harten reynigheyd
        Hiel 's lichaams dartelheyd,
        In uwe strijdt // Gij altijd // onder re-den:
        Wat loon is U bereid!
      
      
        G' hebt zo, O reine Maagd!
        God uwen Heer behaagt,
        Dat den Heer ,, U die eer // heeft bewezen,
        Dat na uwe Trouw Hij vraagt.
        Goods Soon heeft U getrouwd
        Met enen ring van goud:
        Uwe hand // kreeg een band // die voor dezen
        Nooit oogh had aengeschout.
      
      
        Hoe groot was uwe vreughd,
        Hoe wel waert Gij verneught
        Als dien ring // Gij ontving // Catharyne!
        Van God, die 't hert verheughdt!
        Toen wiert G' onststeken meer
        In liefde tot uwe Heer,
        Die door druck // tot geluk, en door pijnen
        U brought tot Martyrs eer.
      
      
        'G hebt Martijrs-kroon ontfaen
        Als Gij waert heen gegaan
        Om zijn naam // en zijn faem // te verbreyden,
        Gij storten niet een traen.
        Gij kwaamt bij Maximyn,
        Een man vol sond's fenijn,
        Een Tyrant // die den ban // U bereiden,
        En dreyghden grote pijn.
      
      
        Hij uit zijn Rijk ontbood
        In wijsheyd Mannen groot
        Vijf-mael thien // (g'lijck belie'n // ons de Schrijvers)
        En hij bij haar U nood.
        Gij komt blymoedigh in;
        Van God neemt G' U begin,
        En Gij brocht // (g'lijck Gij zocht //) de Bedrijgers,
        Tot Jesu Christi min.
      
      
        Gij hebt haar all' geleerd,
        Gij hebt haar all' bekeert:
        Zo dat zij // wonder bly // 't Christen-leven,
        En Christum hebben g' eert.
        Zij leden voor Hem schand',
        Zij levend' zijn verbrand;
        Maar bij God // O wat lot! // hoog verheven
        Zijnz' nu in 's hemels land.
      
      
        Bij deze is verzelt,
        In strijdt, en 's Hemels veld,
        's Keysers Vrouw // die met rouw // haarder zonden,
        Verdrough des doods geweld.
        Ja ook des Legers Hoofd,
        Porphier, in God gelooft;
        Hij aenkent // zijn ellend' // zijne wonden,
        Hij wordt van ziel berooft.
      
      
        Toen wierdt verbitterd zeer
        Den Keyser als een Beer,
        Hij verboodt // dat met brood // niemans spijzen
        U mocht na does niet meer.
        Maar te vergeefs; want ziet!
        Den Hemel U versiet;
        God U spijst // Gij Hem prijst // 't goeder wijze,
        Hij helpt U uit 't verdriet.
      
      
        Na dat G' had meer gele'en,
        Om dat G' had God bele'en,
        Wierdt een rad // boven dat // afgevaerdicht,
        Maar 't brak door uwe gebe'en.
        Toen heeft het swaert u hoofd
        Van 't lichaam afgeklooft.
        Deze eer // heeft den Heer // U gewaerdicht,
        Wien Gij nu eeuwig looft.
      
      
        Maar slaat uit 's Hemels boog
        Op ons (Catryn) u oogh:
        Maakt dat wij // eeuwig bly // God aenschouwen
        Hier namaels ook om-hoogh:
        Op dat wij van 't geschreeuw,
        En van den helschen Leeuw
        Steeds bevrijdt // zijn verblijdt // en behouwen
        God loven te allen eeuw.
