
    
      
        Myn sinlijckheyd // heb ik geleyd // op een die ik moet derven,
        Die mij onthoudt // hetwelk myn zo rout // dat ik vaak wensch om sterven:
        Niet dat ik haar // mijn Lief eerbaar // iet kwaads souw willen gunnen,
        Ja woud' ik al // (dat ik niet zal) 'k en souw voorwaar niet kunnen.
      
      
        Want d'oogjens bly // die zij tot mij // eertijds zo lieflijck keerde,
        Die heeft zij noch // waar door ik, och! haar hertelijck begeerde.
        En haar wel-vaert // is lief en waert // in mijn getrouwe sinnen,
        Souw 't hart van mijn // wel konnen zijn // zonder haar te beminnen?
      
      
        Neen't, zeker neent // het sucht en weent // als het mijn Lief ziet lijen,
        En in haar vreugd // schep ik geneucht // en over groot verblijen,
        Want zij besint // die haar bemint // soo't onberoemt mach blijcken,
        Geen last of noot // zal voor de dood // mij van mijn Lief doen wijken.
      
      
        Hoe soud' ik // ziet, beminne niet // die zo men mach bemercken;
        Is wijs van Raad // arrem van praat // en rijk van goede werken:
        Hier door 't oogh viel // op d'ed'le Ziel // van mijn Lief uytgelesen,
        In wien ik, ach! docht dat ik sach // het alder-schoonste wezen.
      
      
        Klein is mijn roem // of ik schoon noem // haar wel-geschickte zeden,
        Want d'Ouders die // gaan zo ik zie // haar aan Rijcker besteden.
        En die haar eerst // scheen wel begeerst // gaat vluchtigh zij verlaten,
        Maar die aen-neemt // voor eigen, vreemd // laat zich door 't geld bepraten.
      
      
        Het duyvels Geld // heeft ook geweld // zie ik in 's mensen ziele,
        Want die quaat doet // kan metter spoet // de Liefde licht verniele.
        Geen Liefd maar schijn // want ziet an mijn // of het mij Kan verkeren,
        Daar men Jonst koopt // hoe dat afloopt // zal ons de Tijd wel leren.
