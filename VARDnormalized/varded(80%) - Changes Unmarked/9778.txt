
    
      
        NEemt waar, o Heer, ws Kercken // bede
        Laat Sathan haar niet siften // kwaad,
        Die op u Woord en Schriften // staat
        Door uwen Gheest wilt werken // mede
        Tot sterken // vrede.
      
      
        Hoewel ik met de nydelijcke
        Slang oud, en wil paeys maken // niet,
        Zij toont, om mij t'ontschaken // ziet,
        Een vluchtigh moy aertsch tydelijcke
        Doch ydel // rijke.
      
      
        En laat mij niet om vallen // glyden
        Noch breeden pad verkiesen // daar
        Ik laes u sou verliesen // maar
        Tis beter op den smallen // lyden,
        Hier te allen // tyden.
      
      
        Acht Babel niet, o gij // ghetrouwen
        Haar zij u Koninginne // roemt,
        En mij u vyandinne // noemt,
        Maar nemmermeer zal mij // berouwen
        Op dy / te bouwen.
      
      
        Want ziet om niet bedroghen // varen
        Mijn kracht zijn 't Euangely // zal,
        Ghemeen gij niet met Belial
        En zijt, bij u gheen loghenaren
        En moghen // paren.
      
      
        O Heer, al ben ik nu verdreven
        Ghequelt oft moet nu zeer zijn // spot,
        Ik hop', ik werd', o Heer // mijn God
        Eens int volkomen suyver leven
        Van u // verheven.
      
      
        'tQuaet eerst zult gij den kwaden // loonen
        Die qualijck doen oft spreken // hier,
        V gramschap werd ontsteken // vier,
        Maar wilt mij met ghenaden // croonen
        Weldaden // tonen.
      
      
        Och laat doch bij t'waerachtich // schryven
        Vwer Euangelisten // mij,
        Volherdich al van twisten // vrij,
        O Heer, door uws Gheests krachtich // dryven,
        Eendrachtich // blyven.
