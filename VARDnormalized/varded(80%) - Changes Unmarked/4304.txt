
    
      
        SVchten, treuren, is nu mijn lot //
        Als menich edel ieught verfrooyt,
        Een ionge maagd hout mij versot //
        S'gelijcx en stond getytelt nooyt,
        Bij al die oyt // veel wonder // schreven //
        Gelijk de snee op t'velt gedooyt //
        Ligh ik nu onder- // bleven //
        Gans zonder // leven.
      
      
        V ermaledijt zij dach // en // stond,
        Die mij brocht dit verstrangen // an:
        Want eer ik wel deursach // den // gront,
        Was ik een arm gevangen // man:
        T'verlangen // van // een troostbaer reden,
        Die ik geenszins ontvangen // can,
        Heeft mij t'herte deursneden,
        Ten duizend steden.
      
      
        Rechtveerdich is mijn ingement,
        Dat ik om haar zulks lijden moet:
        Ik en in alles patient,
        Wat smert, en leed zij mij aendoet:
        Om smaken t'soet, Geniet ik t'suere:
        Want naar veel drucx en wederspoet,
        Zal vreugd staan voor duere,
        Bij avontuere.
      
      
        Tooght zij mij wederspannicheyt,
        Die niet dan deugd haar voren // legt:
        En pleegt wreede tyrannicheyt,
        Aan mij, gelijk gezworen // recht:
        Geboren // zegt // mij elk te wezen
        Een scu, een arm verloren // knecht,
        Zo mij falgeert in dezen
        Troostbaer genesen.
      
      
        Eylaes Venus mij doch ontbint,
        T'benouwen dezer banden // smal:
        Ik minne een dat soetste kint,
        Dat mij mijns lijfs ontpanden // zal,
        Wilt vanden // val // mij preserveren:
        Gij hebt het doch in handen // al,
        Slaat dood, of wilt fineren,
        Mijn lamenteren.
      
      
        Peynst wat verdriet, en smerten // wreet,
        Mij int begin, was opgeleid:
        En met hoedanich herten- // leed,
        Ik nu moet nemen mijn afscheyt,
        Is mij ontseyt // troost te verwerven,
        Zo werd de dood lijdsaem verbeyt:
        Beter ist, eens bederuen,
        Dan duystmael steruen.
