
    
      
        ACh ! wat een smerte // braeckt het minnen uit,
        Wanner een herte // zijnen brandt besluyt,
        Zonder te klaghen // daar 't niet kan verdraghen
        Het vinnigh pranghen // van het groot verlanghen ?
      
      
        Het stout minnen // zegt mij stadigh aan,
        Dat ik mijn sinnen // soud' verklaren gaan,
        Maar bleeuwe schaamte // en vrees' van de blaemte
        Doet mij vertraghen // mijn Lief naar te jaghen.
      
      
        En noch doet hij mij // d'alder-meeste pijn,
        Om dat hij bij mij // mijn Zuster komt vry'n:
        Ach ! hij is stadigh // fel en zeer onghenadigh
        Ontrent mijn deure // 't welk mij meer doet treuren.
      
      
        Dus ben ik even // Tantalus die haeckt
        Naar 't vocht dat neven // aan zijn lippen raeckt,
        En nooit kan proeven // dan een staegh bedroeven,
        Om dat zijn lippen // nooit aan 't water stippen.
      
      
        Wat heeft hij aerdigh // doch in haar besint,
        Dat zij is waerdigh // boven mij bemindt ?
        Nooit wirdt haar wezen // boven 't mijn ghepresen,
        Noch haar zoet spreken // tegen 't mijn gheleken.
      
      
        Ik ben niet spijtigh // zo hij van haar klaeght,
        Maar ziet ik vlijtigh // kom' hem naer-ghejaeght;
        Wilt hij mijn bange // ziel' maar eens ontvangen
        In zijn ghenade // eer het wordt te spade.
      
      
        Ik zal versaken // mijn Zuster ter nood,
        Midts ik niet haken // magh haar blijde dood:
        Maar wil eer sterven // en ook metter spoet
        Mijn borst roodt verven // met mijn heete bloed.
