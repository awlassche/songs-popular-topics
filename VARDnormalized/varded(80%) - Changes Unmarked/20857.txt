
    
      
        O Hemels groot Cieraed,
        Xaveri! Met deugd verciert in de hoogste graed:
        Ik wil uwe deughd // en uwe vreughd
        Verhaelen hier met 's harts geneught:
        'k Wil schrijven // 'k wil stijven // uwe eer // uwe leer,
        Staeghs meer en meer;
        Want ik mijn min // en al mijn sin
        Op U gesteld heb van 't begin.
      
      
        Van uwe gehoorsaemheyd,
        Xaveri! kan worden nooit genoeg gezeid:
        Met ene I // ging vlijtigh Gij
        De wereld rondom even bly:
        Gij eerden // Gij leerden // te saem // de faem,
        En Jesus Naam,
        Met grote vlijt // tot 's viands spijt,
        Gheen tegenspreken Gij en lijdt.
      
      
        Gij had zo suyv'ren hart,
        Xavieri! dat vuyl gedacht U was een smart:
        Want als bij nacht // een vuyl gedacht
        Den Duyvel U had voor-gebracht,
        Van minne // van sinne // Gij bloed // hoe zoet
        Was deze vloed!
        Het speet U dat // uw 's lichaams vat
        Ook in de slaap zijn wet vergat.
      
      
        Uwe yver was zo groot,
        Xaveri! dat Gij niet hebt ontsien de dood:
        U drouf gesteen // u drouf geween
        Vermurwden harten hart als steen.
        Gij smeeckten // Gij preeckten // staegs aan: // Goods baen
        Deed' Gij 't volk gaan:
        Het speet U zeer, dat God uwe Heer
        Vergeten worden, en zijn eer.
      
      
        Gij waert Mariaes vrind
        Xaveri! Zij achten U te zijn haar kind;
        In all' uwe rouw // was deze Vrouw
        Een troost aan U, ja u Behouw:
        In 't lijden, in 't strijden stond Zij // U bij,
        Gij raakten vrij.
        Des helschs geweld // heeft S' ne'er geveldt;
        Gij wiert door Haar zo vroomen Held.
      
      
        Gij sijt geweest gesteld,
        Xaveri! een Prins in 't Indiaensche veld,
        Dat land geleerd // dat land bekeert
        Hebt Gij, en met Goods Wet vereert.
        Wat loont // wat krone // wat lot // geeft God
        Van Sabaoch,
        Aan U zijn vrind // die Hem zo mindt,
        Dat Gij aan Hem die Landen wind.
      
      
        God heeft U zo verheugt,
        Xaveri! en ingestort zo grote vreughd,
        Dat Gij riept, Heer! // gheeft mij niet meer,
        Van waar komt mij toch deze eer?
        Est Satis, est Satis ,, ay mij! // ik ly
        Goods slaverny!
        Ik ben niet waerd // dat mij ervaert,
        Ik bidd' U, Heer! uwe vreugde spaert.
      
      
        Is 't zake dat Gij zijt,
        Xaveri! geweest zo op does aerd' verblijdt:
        Wat zal dat zijn // een zoete pijn
        Te aenschouwen eeuwig Goods aenschijn!
        Vernomen // bekomen // die kans // Goods glans,
        Hebt Gij nu gants:
        Gij slaat om-hoogh // op God U oogh,
        Wien ik altijd te dienen poh.
      
      
        Maar aangezien ik ben,
        Xaveri! een sondigh mensch, g'lijck ik beken,
        In ziel gewondt // met sware sond',
        Zo roept uwe hulp mijn droeve mond.
        Gestadich // genadig // U toont // mij loont,
        Die boven woont:
        Dat is Xaveri! ,, bidt dat ik hier
        Wel-levende ontga het vier.
