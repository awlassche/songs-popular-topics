
    
      
        Al die met Noe in de Arcke // zijt
        Getreden zonder mincken
        En soect geen vreemde parcken // wijt
        Op Gods tooren wilt gedincken
        Want die ten zelven stonden // ras
        Buiten die Arcke bevonden // was
        Die moesten tsamen verdrincken
      
      
        Laat u niet troosten met woorden // maar
        Gedenct dat gruwelijck sneven
        Dat alle mensen versmoorden // daar
        Die buiten de Arcke bleven
        De welk ons als een figuere // schijnt
        Alzo men wel in Schrituere // vint
        Niet te vergeefs en ist geschreven.
      
      
        In Noachs Arke wilt blyven // vroet
        Maar buiten u niet begeven
        de welc is Gods huis na beschrijven // goed
        Van die Apostelen verheven
        men mach daar niet uytwijcken // gaan
        Als Dina zoude om kijcken // gaan
        Zij werd geschend in haar leven.
      
      
        Rahab woonachtich in Jericho,
        Zij werd bewaard van zorgen
        Met al hares vaders huis, gelic, zo
        Haar die bespieders verborgen
        Zenden om dat weldadich // werk
        Zo gaven zij een genadich // merck
        Om haar zo niet te verworgen.
      
      
        Op dat zij in doots noode // heel
        Niet weg sou worden genomen
        Spraken zij bindt dat rode // zeel
        Aan u veynster zonder schromen
        En blyft daar in de dagen // fijn
        Of wij willen ontslagen // zijn
        Des eets, van die daar uytcomen.
      
      
        Rahab die hield in gedachten // snel
        Dat zeel aan veynster te knopen
        Zij wisten haar al te wachten // wel
        Wt dat huis niet te loopen
        Want Wijf kinders en mannen // ziet
        In Jericho al verbannen // hiet
        Zij mostent met de dood becoopen.
      
      
        In huis te blyven geraden // goed
        Om niet te comen in blamen
        Als die van Ay met kwaden // moet
        Liepen ter stad uit al te samen
        En werden zo wij weten // bloot
        Van haar vijanden gesmeten // dood
        Dat er geen weder en kwamen.
      
      
        Benjamin hadde in voorleden // tijd
        Op Israel kwaad benoegen
        Zij namen ooc zonder reden // strijd
        Wel veertich duizend versloegen
        Dat kwam door den verzoeke // boos
        Van die van Gibia roecke // loos
        En al die rest zij verjoeghen
      
      
        De Israliten toerusten // haar
        Voor Gibie met droefhede
        Die Benjamiten met lusten // naar
        Liepen meest al uit de Stede
        maar gene weder in stad // ghekeert
        Hier door zijn wij al wat // geleerd
        Wt loopen is een kwade zede.
      
      
        Semi ontfing van Salomoen
        Een bevel, hij zoude maken
        een huis, dat ginck hij te male doen
        Om zijn vloeckende spraken
        Binnen Jerusalem // gepast,
        niet uit te gaan werd hem // belast
        Of die dood zoude hij smaken
      
      
        Een goede meeningh sey Semi,
        Heeft den Coninc gesproken
        een huis getimmert, dit dede hij
        En heeft hem daar in beloken
        Hij wonder zonder bezwaren // stranck,
        Wel den tijd van drie Jaren // lanck
        maar zelve heeft hyt noch verbroken
      
      
        Daar ontliep uytten huize hem
        Twee knechten zonder falen,
        Semi ginck uit Jerusalem
        Na Gath om haar daar te halen
        Niet eens bedenckend anden ent,
        Semi nae's Conincx Mandement
        Most dat met de dood betalen.
      
      
        Gehasy den Knecht Elisa
        Liep uit den huize met list,
        Achter Naman, maar niet bij ra
        Vanden Man Gods diet wel wiste
        Zijn hert in die geschincken // zat
        Zo ginc hij leugens dincken // wat
        Om zo te vullen zijn Kiste.
      
      
        Als hij na zijn begeere // dat
        Goed nu al hadde versteken
        Zo vraegd' hem zynen Here // plat
        Van waar hij kwam, hij ginc spreken
        Nergens en heeft geweest u // knecht
        Daar op hem den Man Gods nu // zeght
        En openbaart hem zijn treken.
      
      
        Nu gij tot uwen solase // die,
        Geschincken aan hebt gaan vaten
        Zo zal u Namans laserie
        Aenhangen zonder verlaten,
        U en u zaad hij ditte // sey
        en werd melaets ghelijck witte // sney
        Wat mochte hem uit loopen baten.
      
      
        Wtloop gheschiedt zeer menich // fout
        Dat er veel moeten om treuren
        als mensen worden een weynich // stout
        Zij licht afdeelen en scheuren
        Want na Joannes luid // vermaen
        zegt hij, zij zijn van ons uit // gegaan
        Zij hebbent ook wel zien gebeuren.
      
      
        Op zee Petrus handlen // lag
        Als vissers zonder verdolen
        maar als hij Christum wandlen // zag
        Op Zee bij 't Schip onverholen
        Zo heeft hij dat begrepen // niet
        Te treden buiten Schepen // iet
        Eer dat hem worde bevolen.
      
      
        Ghedenct wat Paulus uit Room // schreef
        Den Timotheum vailiandich
        Dat er zo weinige vroome // bleef
        In Asien, en volstandich,
        Zij keerden af bij vele, dus
        Met Hermogeni en Philetus
        Op enen onwegh zeer schandich.
      
      
        Prinse laat ons altijd blyven // vast
        In 's Heren Tempel met vreden
        Dat 't Lichaam tot zijn verstyven // wast
        Gelijk die van Roomen deden
        Om geen druc noch ellende // scheid
        Denckt dat de Kroon int ende // leit
        Als den strijd wordt gestreden.
