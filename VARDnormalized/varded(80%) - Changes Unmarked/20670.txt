
    
      
        Onze lieven Heer wil dat den mensch
        In den Hemel komt,
        (Dit is geweest al-tijd zijn wensch)
        En niet werd' verdomt.
        Zo veel als // is in Hem,
        't Is niet vals // 't is Zijn stem // waarom
        Godes Soon ons zoekt rond-om.
        Op dit jammerdal // in het arme stal
        Is gekomen ne'er,
        Dezen lieffelijcken Heer.
      
      
        Door d'Engel heeft den Heer bij nacht
        Aangezeid de free
        Aan d'Herders, die daar hielen wacht
        Over hare vee:
        Ende-Hy // heeft gezeid
        Zijt doch bly // ende vleydt // den Heer
        In het stal gebooren teer:
        Dezen is aan u // voortgebrocht nu, nu,
        Door een reine Maagd,
        Die ten hooghst' heeft God behaagt.
      
      
        De Herders zijn hier op gegaan
        Na Bethl'hem met spoed:
        Zij vonden 't Kind met druck belae'n,
        Yder dee zijn groet;
        De ene kust // 't kleine Kind,
        De ander sust // dien hij mindt // zijn God,
        Voortgebrocht in 't Beesten-schot.
        O met wat een vreughd // zijn Zij doen verheughdt
        Als Zij sagen aan
        Die onze zonden heeft gelae'n.
      
      
        Zo mede zijn voorwaar verblijdt
        En verheughdt geweest
        De Koningen op dezen tijd
        Als Zij onbevreest
        Wonder varr' // het is waar)
        Door een starr' // wonder klaer // gebrocht
        Zijn tot Jesum, die Haar zocht.
        Zo dree als Zij die // starre met haar drie
        Sagen in de lucht,
        Zijn gereisd Zij metter vlucht.
      
      
        De kisten, koffers, zijn gereed
        Staende-voet gemaakt,
        En niemand zijne schat vergeet,
        Op den weg men raeckt.
        Wind, noch weer // Haar verdroot
        Om den Heer // noch de dood // maar Zij
        Zijn gereisd van herten bly.
        O wat zoete praet // onder Haar omgaet!
        Niet en viel Haar hart,
        D'yver weg nam all' haer' smart.
      
      
        Zij kwamen te Jerusalem,
        Zij vernamen daar
        Met hart, met sin, met ziel, met stem,
        Of daar Christus waar.
        Men ziet in // de Schriftuyr
        Van 't begin // tot die uyr // daar staat,
        Na het stalken henen gaat.
        Daar wordt voortgebracht // uit het grootst' geslacht,
        Jesus, die regeert
        Israël, die Haar bekeert.
      
      
        Als dit Herodus hadd' verstaen
        Uit den Wijzen mond,
        Zo heeft dien loosen Vos doen gaan
        Deze Drie terstond,
        En hij seyd' // gaat vrij heen,
        God u leyd' // 'k ben te vre'en // maar keert
        We'er, als Gij Hem hebt geëerd;
        Dan zal ik met spoet // doen aan Hem mijn groet,
        Als ik weten zal
        Waar den Heer is van ons all'
      
      
        Zij zijn gegaan terstond uit Stad
        Op Herodes woord,
        Die Haar verlof gegeven had:
        En Zij gingen voort
        Met jolijt // ende vreughd,
        Zeer verblijdt // en verheughdt // want ziet!
        Haar de Starre niet verliet;
        Maar heeft Haar geleid // waar Haar wierdt gezeid,
        Dat den Hoogsten Heer
        Bij de beesten lag ter ne'er.
      
      
        De starre stil stond boven 't stal
        Waar geboren lag
        Den groten Schepper van ons all';
        (O gewensten dag!)
        O wie zal // zeggen hoe
        Voor het stal // zijn te moe // geweest
        Deze Drie, hoe onbevreest!
        Ja van blyschap sprong // haar gemoed en tong,
        Die met vreugde sey,
        Och of hier Messias ley!
      
      
        De Dromedaris' stonden stil,
        De Kameelen mee,
        Balthassar op zijn knyeen ne'er vil
        Met de and're twee:
        Kist, en kast // van het goed
        Wordt ontlast // ziet hoe zoet // ei! ziet
        Hoe de Maagd haar welkom hiet.
        Welkom Balthassar ,, die komt van zo varr':
        Gaspar komt tot mij,
        En Gij Melchior komt bij.
      
      
        Toen zijn Zij met eerbiedigheyd
        Tot de Maagd getre'en;
        En S' hebben met haar Drien gezeid,
        Deze zoete re'en,
        Neemt Wierook ,, neemt dit Goud,
        Myrrhen ook // ei! dit houdt // voor 't Kind,
        't Welken ons zo zeer bemint,
        Dat Hij komt hier ne'er // gants'lijck zonder eer,
        Want Hij leit op strooy
        Voor ons, die vergaen als hooy.
      
      
        O Hogen God! w' aenbidden U,
        Die waerachtich zijt
        Een kind geworden voor ons nu
        In does kouden tijd;
        Hogen God // lovens waerd,
        Die als zot // op does aerd' // hier legt,
        Wat Gij wilt, gebiedt, en zegt;
        Wij het kwaad altoos // en de zonden boos
        Nu van stonden aan
        Worpen weg: Adieu, wij gaan.
      
      
        Nu hadden hare groet gedaan
        De Heidenen aan 't Kind:
        Verlicht zijn Zij na huis gegaan
        Die eerst waren blind.
        't Heydendom // is bekeert,
        't Christendom // is geleerd // God loff,
        Die regeert het Hemelsch Hoff.
        Maar een ander pad // 't Heydendom in trad,
        Als door Godes hand
        't Weg geleid wierdt na haar Land.
      
      
        Leert Christe-menschen al te saem
        Soecken hier uwe God;
        Gij zult Hem zijn zeer aangenaam,
        Leeft na zijn gebod.
        Ei! keert we'er // van u kwaad
        Tot den Heer // en verlaat // de sond',
        Die u totter dood toe wondt.
        Draagt Hem op u goed // draagt Hem op u bloed,
        Van nu af begint
        God te dienen, die u mindt.
      
      
        O Jesu ! die staechs hebt getracht
        Na mijn saligheyd,
        Mijn hartje na U groot'lijcx jacht
        En het bitter schreyt;
        Het beweent // zijn misdaed,
        En het steent // over 't kwaad // voortaan
        Zal het U we'er dienen gaan:
        Het zal nimmermeer // 't kwaad aannemen we'er,
        Maar het wil de deughd
        Oeff'nen met een grote vreughd.
