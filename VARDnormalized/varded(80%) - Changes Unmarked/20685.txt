
    
      
        DOrathee ,, ach! uwe twee Zusters, God begeven.
        Zij bezoeken // ('t wil niet loecken)
        Of Gij, g'lijck als zij, wilt leven.
        Zij bezoeken.
      
      
        Maar gehecht // en gevlecht
        Zijn met liefdes-banden
        All' uwe sinnen // all' uwe minne,
        Aan den Heer van alle landen.
      
      
        G' hebt geleerd // G' hebt bekeert
        Uwe Zusters deftich;
        En Zij leden // en Zij streden
        Voor de leeringh Christi treftich.
      
      
        Grote nood // ja de dood
        Hebben Zij verdragen:
        Hoge gaven // 's Hemels haven
        Kregen Zij na deze vlagen.
      
      
        Doen zijt Gij // in de ly
        We'er daarom gekomen:
        In die dagen // hebt Gij slagen
        Lustich op uwe le'en vernomen.
      
      
        Ei! Gij hebt // en Gij schept
        Vrolijckheyd in 't lijden.
        In het blaecken // in het kraecken
        Uwer le'en, Gij U verblijden.
      
      
        Gij ginght heen // Gij ginght tre'en
        Na 't schavot blymoedich.
        U verbeyden // 's Hemels weiden
        En Gij kwaamt daar in gansch bloedich.
      
      
        Uit dat land // heeft een mand'
        God met schone Roosen
        Ne'er gezonden // op dier stonden,
        Die Theophilus zag bloosen.
      
      
        Deze man // nam die an,
        En Hij zich bekeerden;
        Ja zijn leven // heeft gegeven
        Voor den Heer does Rechtsgeleerden.
      
      
        's Hemels lot // schonck hem God
        Voor does minne-voncken;
        Om zijn deughden // is van vreugden
        Hij, en Gij, gestadich dronken.
      
      
        Och of Gij // ook op mij,
        Dorothee, woud dencken!
        Ei! bidt krachtich // God Almachtich,
        Dat Hij mij zijn Hof wilt schencken.
