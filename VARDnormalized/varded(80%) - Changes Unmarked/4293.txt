
    
      
        Naar dien dat ik moet //
        Subject, en dienstbaar staan,
        En mijn vrijheid zoet //
        Geheel te buiten gaan,
        Meer onderdaen //
        Dan mij rebel // verthoont //
        Heb ic, naar recht //
        Alsulck een knecht //
        Werd namaels wel // geloont.
      
      
        Edel, dier // en // weerdt,
        Is zij, die t'hert aanziet:
        Al dreyght vier, en // zweert,
        Ik sta tot haar gebiet,
        Deed'- ik zulks niet,
        Zo waar ik fel // gehoond:
        Wie twijffelt dies,
        Zulk tijd verlies,
        Werd namaels wel // geloont.
      
      
        Een alsulcken ieught
        Blijf ik eigen voortan,
        Werd het einde vreugd,
        Zo zit ik edel man
        Als Coninck van
        Gans Israhel // getroont,
        T'sy hoe dat gaat,
        De minste daad
        Werd naermaels wel // geloont.
      
      
        Liever smert // en leed,
        T'aenveerden, angstich zwaar,
        En haar t'zijn gereed
        Een knecht, drie seven iaer,
        Zo werd' ik naar,
        T'gewonnen spel // gecroont:
        Dat dunct mij sienst,
        Getrouwen dienst,
        Werd namaels wel // geloont.
      
      
        Keert zij mijn labeur
        Te zijden, als iet cleens:
        Mij arm serviteur
        Toelaet luttel gemeens:
        T'sal werden eens
        Dat mijn gequel // verschoont,
        Want die altijd
        Verdraagt, en lijdt,
        Werd namaels wel // geloont.
      
      
        Es ook meerder last,
        Vervaerlick elcx gepeys:
        Dat een dorper gast,
        Geniet der minnen cheys,
        Venus paleys
        Zonder bevel // bewoond:
        Och! t'is te cleyn,
        Dat zulk vileyn,
        Werd namaels wel // geloont.
      
      
        Niet versagen // zal
        Mijn Princelick gemoet:
        Liever dragen // al,
        Des weerelts tegenspoet
        Want diet mij doet,
        Is zo ydoonen maagd,
        Voor cleyn geween,
        Gij namaels geen
        Zo wel beloonen // saeght.
