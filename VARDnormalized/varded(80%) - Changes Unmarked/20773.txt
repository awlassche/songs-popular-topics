
    
      
        't IS den Feest, den Feest van groter waerde
        Van Sint Jans Geboorten-dagh;
        Die hier door Goods beleyt
        Staeghs heeft geleid
        Zo heiligen leven als oyt iemand zag.
        Daar is nooit, persoon op dezer aarde
        Heiliger geweest als Hij,
        Zijn ziel was nooit gewondt
        Met een een'ge sond',
        Ter wereld Hij kwam van alle zonden vrij.
        Des laat ons zingen // op does blijde Feest,
        Laat ons opspringen // vrolijck in den geest,
        't Is tijd // ei! zijt // nu mensen all'
        Verheugt // wel geneught,
        Die zijt op 's werelts dal.
      
      
        Door dit Kind zal worden aangewezen,
        Die den mensch verlossen zal.
        Messias, Godes Soon,
        Die uit zijn troon
        Hier komen zal ne'er in't Bethlemijtsche stal.
        't Is dien geen, (zo als wij konnen lezen)
        Die geboodschapt heeft den Heer,
        Does is't die voor Hem loopt,
        Does heeft gedoopt
        Messiam, die Hem gehouden heeft in eer.
        Men kan niet horen // dat'er enig kind
        Van Vrouw gebooren // is, dat God meer mindt
        Als Jan // die an // Goods een'gen Soon
        Verdiendt // als zijn vriend,
        Den alderhoochste kroon.
      
      
        Godes Maagd gaat Hem alleen te boven,
        Aan wien niemand is gelijk,
        Noch boven noch bene'en;
        En dat met re'en,
        Want niemand is oyt in deughd geweest zo rijk.
        Als does Maagd, Elisabeth ging loven
        Over het geberght' met spoet,
        En wenschten haar geluk,
        Om dat haar druck
        Was over, g'lijck haar den Engel had gegroet:
        Zo is verresen // 't Kindtjen uit zijn stee,
        Want 't wiert genesen // van des sondens wee:
        Het sprong // het zong // door 's Moeders mond,
        De eer // van zijn Heer,
        Die Hem van sond' ontbondt.
      
      
        Dit is 't Kind, door wien de tongh ontbonden
        Van Zach'rias is geweest,
        Die God gebenedijdt
        Heeft op dien tijd,
        Die met zijne Vrouw was vol van Godes Geest.
        Dit is 't Kind van God den Heer gezonden
        Om dat 't ons getuygen sou,
        Dat was gekomen aan,
        Op 's werelts baen,
        Messias, die ons sou lossen uit den rouw.
        Laat ons eendrachtigh // 't Kindtje geven lof,
        't Welk ons gedachtig // is in 's Hemels hof:
        Het is // 't is wis // geseten hoog,
        Het ziet // ons verdriet,
        Het slaat op ons zijn oogh.
