
    
      
        SOonjen loopt niet zo henen //  met enen // dullen //
        kop om u te vullen // vol van mensen bloed;
        Maar wilt gij smullen // in het hoogste goed,
        Reyckt naar d'olyven,
        En wilt hier blyven,
        Laat al den storm des leghers dryven,
        Schyven // die de legher-wyven
        Slicken in hun hert', staan mij al te dier;
        Laat dan den vrede-zeghen // u beweghen // hier.
      
      
        Moertjen, laat die gebeden // want heden // willen,
        wij ons lyf verspillen // voor 't ghemeyne best,
        Al valt ons spill // in het asch op het lest,
        Roemigh is 't sterven,
        Dat ons doet erven,
        (Daar alle goederen bederven)
        Erven // gheene aarde scherven,
        Maar den altydt bloeyenden Lauwerier:
        Laat dan den oorloghs zegen // u beweghen // hier.
      
      
        Den Lauwer geeft maar blad'ren // doch nad'ren //
        wij den // frissen, ende blyden // oli-tack, wij zien
        Vlughs van hem scheiden vettigheyt, en wien
        Zal dit niet vermaken?
        Zal dit niet doen staken
        't Bersten, en slinden van d'oorloghs zaken?
        't Kraken // van zo vele staken,
        Die tot de sterren hun verhieven schier:
        Laat dan den vrede-zeghen // u beweghen hier.
      
      
        Al dat gij wilt betrachten // is 't achten // minder //
        dan den minsten hinder die mij dryft van hier,
        'k Ben hier een vinder // rechts van niet een sier;
        Ginder zal ik leven,
        Ginder zal ik zweven,
        Ginder zal ik, door deugd ghedreven,
        Geven // kennis hoe dit leven
        Is het roemighste van d'aerde schier,
        Laat dan den oorloghs zegen // u beweghen // hier.
      
      
        Wel laat gij dan u leven // beneven // 't goedtjen //
        dat zo menigh moedjen // heeft gekost aan mij?
        Ja 'k treck myn voetjen // lieve Mo'er van dy,
        Maar om te koemen
        Tot 't edel roemen,
        Dat mij zal enen Heldt doen noemen,
        Bloemen // die gheen mensch kan doemen:
        Lieve laat dan 't wenen en ghetier,
        En laat dan oorloghs zegen // u beweghen // hier.
