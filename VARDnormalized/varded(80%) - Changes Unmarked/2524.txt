
    
      
        Vraaght gij, waarom sucht gij zo zeer,
        Dat doet mij wel nu:
        En ben ik u Lief, ben ik u niet veer?
        Ja Vriendinne // mijn sinnen // die minnen
        Alleenigh u.
      
      
        U by-zijn mishaeght mij niet, ach!
        Nooit liever stond;
        Maar dat ik niet bij u blyven mach
        Doet mijn zuchten // ia vruchten // en duchten
        O rooder mond!
      
      
        Ik wensch dat gij genieten meugt
        't Geen gij mij geeft,
        Dat is een over-soete vreugd,
        Die begeerlijck // en heerlijck // doch eerlijck
        Om 't herte sweeft.
      
      
        De vreze van 't verand'ren haast
        Doet mij zuchten zo,
        Zo dat mijn blijdschap suft verbaasd,
        U uytghelesen // ghepresen // eel wezen
        Derf ik zeer noo.
      
      
        U lodderlijck wezen braef en fris
        Verheugt mij 't hart,
        Maar ik weet wat mij nakende is,
        Door u derven // te erven // een sterven
        Daaglijcks met smart.
      
      
        Is dan dit zuchten zo zonder reen,
        Zegt liefste dra?
        Ik acht wel gij zult zeggen neen:
        Op mijn jaghen // en klaghen // of vraghen,
        Zegt daar op ja.
