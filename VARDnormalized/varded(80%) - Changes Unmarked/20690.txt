        All' den haat // die daar is, die daar staat
        Tussen mij // en mijn God, want ik ben in de ly.
        Ik zal niet meer // ach! mijn Heer // uwe eer
        Tegen staan // noch m' in enig kwaad ontgaen;

        Van berouw, en hoe ik mij daar in benouw.
      
      
        Ach! ik treck een diep gesucht,
        Sint Guilljam! om dat ik ben gevlucht
        Van mijn God // met Wien ik hiel de spot;
        Die nochtans // zocht aan mij te geven 's Hemels krans.
        Waar was mijn sin // van 't begin // dat mijn min
        Na de leer // ik niet stelden van mijn Heer!
        't Valt mij hart // 't is een smart // aan mijn hart,
        Dat voor does // ik aan God geen meer eer bewees.
      
      
        Gij der Zondaars Spiegel zijt,
        Sint Guilljam! Gij zijt die haar verblijdt.
        Gij den val // door 't gebed, van haar all'
        Schutten kent // Gij haar by-staet op haar 's levens end.
        Waarom ook ik // stof en slick // niet meer schrik
        Voor de hell' // al bestrijdtse mij hier fel,
        Want Gij tracht // en Gij jacht // dach en nacht
        Te helpen mij // door U raeck ik van Satan vrij.
      
      
        Ik nu Dyn bedancken wil,
        Sint Guilljam! Ik wil niet swijgen stil,
        All' het goed // all' de hulp, all' het zoet,
        All' de deughd // waar mee Gij mij dickmaels hebt verheugt.
        G' hebt mij verblijdt // en bevrijdt, menigh tijd
        Uit mijn nood, dickmaels ook uit ziele-dood.
        Wilt voortaan // voor mij gaan // in does baen;
        Wilt de kroon // mij verkrijgen in 's Hemels Troon.
