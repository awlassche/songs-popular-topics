
    
      
        IN mijnen geest // moet ic veriubileren
        Nooit soeter feest // dan de Sabboth des Heren
        Wie hem met lust //  viert van zijn kwade werken
        Van zonden rust // in God die hem can sterken
        Die heeft zeer pleyn // den Sabboth recht ghehouden.
        God heeft hem rein //  zijn zonden kwijt gheschouden.
      
      
        Ook zalig dan // is hij tot allen stonden
        Alsulcken man // en rekent God geen zonden
        In wiens geest fris // geen bedroch noch qua perten
        Beuonden is // ook niet dobbel van herten
        Maar als Gods knecht // de vrome Job ghepresen
        Slecht ende recht // volherdich in Gods vrezen
      
      
        Ooc die geen geld // op woeker heeft gegeuen
        Noch geen geweld // aan iemand heeft bedreuen
        Maar leent met vlijt // den armen telcker vre
        Zonder profijt // alleen wt liefden pure
        En geeft haar al // die hem niet konnen loonen
        Dien sulcken zal // in Godes Tempel wonen.
      
      
        Zo wie met vliet // wt lust hier in beclijuen
        Zij zullen niet //  ledich onvruchtbaar blijuen
        In kentenis // van Gods volmaakte leer
        Die heylsaem is // in Christum onzen Here
        God die vermeert // de gaef zijns geests beseuen
        Wie hem verneert // die wordt van God verheuen.
      
      
        Tis wel geuiert // die zo den Sabboth rusten
        Sulc een verciert // Gods leringe zoet met lusten
        De weeskens cleyn // beschermt gelijc  een vader
        Voor tgewelt pleyn // en die als wijs berader
        Hem neerstich keert // dat weesken onbedrogen
        Geschiet, geleerd // in Gods woord opgetogen.
      
      
        Is een in noot // die hem dan troost van herten
        En breect zijn brood // een siel vol honger smerten
        Die niet en can // rusten, eten noch drinken
        Als hij een man // nootdruftich can bedincken
        Maar hem eerst spijst // wt liefden na en verre
        Zijn licht dat rijst // op als de morgensterre.
      
      
        Als hij besorcht // de weeukens rein van zede
        Geeft, leent en borcht // elk na gelegenthede
        Die haar zo can // voorstaen in alle zaken
        Recht als haar  man //  raad tot bidden en waken
        Gheestich ghesint // saecht, stil tot allen tijden
        Dat si verwint // haar druc en tsviants strijden.
      
      
        Niemand en can // tot dezen Sabboth comen
        Oft ten zij dan // dat hij tot zijn der vromen
        Gantschlic versaect //  de wereld met haar lusten
        En niet en  smaect // dat aertsch vol van onrusten
        Mer zijn gemoet // gants draecht in God verheuen
        In suer en zoet //  dancbaerlijcke te leven.
      
      
        Zeer onbesmit // is hi voor God eerweerdich
        Die niet en zit // bij de spotters lichtueerdich
        In haren raad // noch in haar cromme wegen
        Geenszins en gaat // maar contrari daar tegen
        Al zijnen ganck // heeft in het hemelsch wezen
        Met lof en dance // als een met God verresen.
      
      
        Zoet en fijn // moet teynde aller geboden
        De Sabboth zijn // daar maar  een is van nooden
        Dits teenich een // stil houden God ghelaten
        Als Magdaleen // die coos tot haarder baten
        Tbest eenich goed // dwelc haar bij is gebleuen
        Doprechte vroet // en is geen wet gegeuen.
      
      
        O Gods geest net // gij komt ons wel regeren
        Gij zijt de Wet // niemand heeft v te leren
        God heeft v voort // den heylighen ghegeuen
        En na zijn woord // in  haar herten geschreuen
        Gij zijt een licht // lantern thaerder voeten
        Na Dauids dicht // twelc zij nauolgen moeten.
      
      
        Och liefden graet // gij bant alder secreten
        In v bestaat // de Wet en de Profeten
        Gij zijt niet smal // het eynt en  de hooftsomme
        Der Wetten al // haar oorsaeck en waaromme
        Na Schrifts wtlech // groot bouen alle gauen
        De hoogste weg // de rustdach alder slauen.
      
      
        Nu Prinse  God // diet alles werct in allen
        Gij zijt mijn lot // laat mij van v niet vallen
        Werct nacht en dach // wilt al mijn kracht regieren
        Op dat ic mach // in v den Sabboth vieren
        V iock is zoet // v last is licht om draghen
        Wie leert ootmoet // zijn siel rust tallen dagen.
