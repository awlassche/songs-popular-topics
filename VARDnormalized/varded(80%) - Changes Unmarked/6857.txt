
    
      
        Glorieuse violette // Koninginn’ van Hemelrijck,
        Suyver vat vrij zonder smette // In den geur aromatijck,
        Soeticheyt zonder ghelijck,
        Zulk een vreugd gij sijt // die vermeucht altijd
        Smert en leed te scheuren,
        Met vreugd, jolijt // verheucht, verblijd
        Ons weemoedich trueren.
      
      
        Precieus Paleys van trooste // blinckend’ als den dagheraet,
        Sterre lichtend’ uit den ooste, als de Zonne claer opgaat,
        Uwen lof elk metter daad
        Hondertsout verclaert // als een go bewaard
        Iun der Hemels throone:
        Benaut, vervaert // verstaut, bezwaard
        Is nu elk persoone.
      
      
        Alderheylichste Princess // Maria Gods Moeder rein,
        Ons aller Advocatersse // Ziet ons in dit droevich pleyn,
        Tot u roepen wij ghemeyn:
        Voor u dienaars doet // der genaeden vloet
        Vwen Soon ghedencken,
        Dat hij ons ‘tgoet // zijns rijke zoet
        Will’ hier naemaels schencken.
