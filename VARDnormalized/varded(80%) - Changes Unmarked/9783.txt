
    
      
        ONder veel Meyskens die men vindt,
        Is een haars Moeders liefste kind,
        Mijn duive bemint mijn wederpaer
        Excellent // Excellent
        In mijn herte gheprent,
        Is zij bekent // en ik ben haar
        Een fundament // Onwanckelbaer,
        Tot 'swereldts endt.
      
      
        Ik ben haar tot een open deur,
        Een troost in lyden en ghetreur,
        Een vyerich meur // dag ende nacht,
        In den noot, In den noot
        Een beschutsel groot,
        De tweede dood // en heeft gheen macht,
        Met haar aenstoot // aan mijn gheslacht
        Te doen exploot.
      
      
        Ik en laat haar gheen Weze vrij,
        Met mijnen Gheest blijf ik haar bij,
        Tot troost heeft zij // mijn Woord ontfaen,
        En 't verbond // En 't verbond
        Van mijnen mond,
        Zal 't eeuwigher stond // wel blyven staan,
        Maar berghen rondt // zullen vergaen,
        Al in den grondt.
      
      
        Ik en heb haar met ghelt noch goed
        Gekocht, maar met mijn dierbaer bloed
        Blyven zij moet // mij ingelijft,
        Wt mijn hand // Wt mijn hand
        En zal haar niemand
        Ghetrecken, want // Zo langhe zij blijft
        In mijn verstand // Als Boom beclijft
        Zij, wel gheplant.
      
      
        Vergheet ik Ierusalem fijn,
        Mijn rechter hand moet vergheten zijn,
        Mijn tong aan mijn // mond kleve ziet,
        En al waert // En al waert
        Dat tegen den aard,
        Haar kint gebaard // Een Moeder verliet,
        Syon vermaert // En zal ik niet
        Laten bezwaard.
      
      
        Mijn Princess, mijn lieve Bruydt,
        Als taruw hier onder onkruydt
        Zal zijn daar uit // vergadert, en
        Boven dien, boven dien
        In heerschappien,
        Zal zij mij zien // ghelijck ik ben,
        En haar verblien // Eeuwig, Amen:
        Dat zal gheschien.
