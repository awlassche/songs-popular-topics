
    
      
        Wel op myn luth / met droef acoort wylt claghen
        Den noot en tlast / dat ik eylaes moet draghen
        Synght myn fortuyn / claer op den hoechsten troon
        Verhaelt de vluecht / van de wise clergesse
        Betruert tafzyn / van zyn lieste meestresse
        Synght ouerluydt / myn onverdyenden loon
      
      
        Schreyt ogen been / over myn zwaar verdryeten
        Och tranen heet / wylt uwen vloet wt schyeten
        Dat uwen loop / Immermeer zij vercort
        Voor dat de dood / mij tleuen eens zal einden
        En de fataels / zusters mij zullen schenden
        of dat myn lyefd / ook eens verandert wordt
      
      
        Myn hope sterft / wat wyl ik meer verwachten
        Troost en confoort / dat mij zoude versaechten
        Vlyet nv van mij / vruecht In voorleden tyt
        Ik bem eylaes / als de vertrede wormen
        Of als een schep / zonder mast In den stormen
        Dusent droefheyt / doet mij nv dezen stryt
      
      
        Maar op wat Ionst / zoude ik myn hope strecken
        Myn schone son / syetmen heel ver vertrekken
        Met haar schoon lycht / dat mij eer heeft verclaert
        In duysterheyt / moet ik eylachen suechten
        Sorghe mij quelt / tys al vrezen en duechten
        Myn herten lust / Is met rauwe bezwaard
      
      
        Haar fyer ghelaet / en haar wreet eerbaer wezen
        Deerste blom vruecht / van haar Ionckheyt geprezen
        Haars herten lycht / met den vermeyllen mond
        Haar zoeten sangh / bouen den nachter ghale
        Myns leuens gheest / dies is noch prynsepale
        Werd mij ontstolen / dies leef ik nv doorwondt
      
      
        O lyefde blynt / wel blent In al myn lyden
        Waarom moecht ik / de kennys niet vermyden
        Van haar schoonheid / die ik doch deruen moet
        Ist om dat ik / deruende zulckx ghesyechte
        Leuende dood / zou blyuen Int tghewychte
        Dat is gheen lyefde maar tyeranie verwoet
      
      
        Doen ik met doogen / de nymph mocht ghenaken
        Pine was vruecht / hoe wel dat mij doorstaken
        Haar pylen scherp / van onderscheen vyertuyt
        Laat Iupiter / met al zyn bleyxem commen
        Hoe groot ghewelt / ik zalder niet voor schroomen
        Can hij meer last / op mij nv ghyeten wt
      
      
        Den drouven dach / dat zij mij is ontvloden
        Ik most dood zyn / door die craecht van de goden
        Den gheest was weg / tverstant al om beraest
        Want hadde ik dien / schecht doen syen konnen vlyegen
        Het Is alzoo / wat wyl ik veele lyeghen
        Ik ware ghestoruen seven maal metter haast
      
      
        Nu ik maar leef / In duister naechten doodych
        O ogen twee / waartoe zyt gij mij noodych
        Tys al verdryet / Ik en bem nerghens vrij
        Ghehoor eylaes / zult gij mij ook ghebreken
        Neen . tsoet gheluyt / haars monds hoor ik niet spreken
        Daarom vertrekt / gij hebdt oorloef van mij
      
      
        Gheluckych is / voor / damourues verstanden
        Den voghel schoon / van de Indysche landen
        Hij sterft en ziet der zonne claerheyt groot
        Waarom ist dat / ik tleuen zoo niet mess
        Ghewillych Int ansyen van myn prynsesse
        Zoude ik eylaes / mij gheuen In de dood
