
    
      
        SPoeyt u voetjes, treedt wat aan,
        Want Apol zijn Paerden in gaat slaan,
        En Auroor,, haar vertoond // soetjes voor,
        Dus mijn vee // gaat wat rasjens, dwaalt niet na de Zee:
        Zoeckt Klaverwey // schuwt de Hey // op dat gij,
        Niet geniet // 't gheen u baren mocht verdriet:
        Plucked ghewas // knabbelt gras // voor 't gebas,
        Van 't gediert // ik u hoed, schoon hoe zeer het tiert.
      
      
        Stroomt neer Beeckjes spoet u ras,
        Segh mijn Lief dat ik hem hier in 't gras
        Heb vertoeft // al te langh, 'k ben bedroeft,
        Om dat hij // dus gaat wachten, en mij laat in ly,
        Ei bortelt uit // met geluid // dat het stuyt,
        Dicht ontrent // daar mijn Lief zich is gewend,
        Met zijn Vee // 't houden stee // laat hem ree,
        Hier doch zijn // ach! ik bid, doet zo veel voor mijn.
      
      
        Daar rijst soetjes 't Morgen-licht,
        Die het vier heeft in mijn borst gesticht,
        Wiens gelaat / m'heeft bekoort // en zijn praet,
        Mijn vervoer // doen hij eeuwig mij zijn trouheyt zwoer,
        'K acht Konincks Kroon // en zijn Troon // niet zo schoon,
        Noch zo waerd // al de rijkdom van de aerd,
        Als die geen // die met reen // langh geleen,
        M'heeft bekoort // tot zijn Min, dies hij mij behoort.
      
      
        Kom dan Cloris kust uwe Vrouw,
        En geniet het geen gij eertijdts wouw,
        Want de Trouw // van ons twee, Man en Vrouw,
        Heeft gemaakt // dies de lusjes na 't genieten haeckt,
        Kust Cloris kust // laat de lust // zijn geblust,
        En voldoet // d'langh-gewenschte vreugde zoet,
        Beesjes Fluyt // maakt geluid // 'k ben de Bruydt.
        Weest verheugt // want de tijd, roept nu om geneugt.
