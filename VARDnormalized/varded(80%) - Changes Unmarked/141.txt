
    
      
        Ik weet een Peerle ient,
        Gheprent // in Schoonheid rijke
        Nooit Man en heeft present,
        Bekent // haarder ghelijcke
      
      
        Diamant ofte Robijn
        Sangwijn verciert in gouwe
        En vintmen tot ghenen termijn,
        Zo fijn // In gheen Lantsdouwe,
      
      
        Ghelijck die Zonne claer //
        Eenpaer, Int tshemels throone
        Zo is mijn Lieff voorwaar,
        Aldaar blinkende schone.
      
      
        Waar ik een Cooninck vayliant,
        Playsandt // Rijk van persoone
        Ik versette voor haar mijn Land
        En sant // En ook mijn crone
      
      
        Dat ik mijn ghedacht.
        Dach en nacht, op haar set als donwijse
        Zij is mij te hooghe van macht
        Gheacht // Ende ook te groot van prijse.
      
      
        Does peerle daar mij dat leed
        Zeer wreet // om moet ghebueren
        Tis mijn lief dat ik weet,
        Hoe zij heet // die fluer der flueren.
      
      
        Mocht ik met mijn shersen bloed,
        Onsoet // haar noch te winnen
        Zo sout ik tot een vloet
        Mijn bloed // wt laten rinnen.
      
      
        Sturffe ik dan van rouwe groot
        Den dood // om haar duer dezen
        Zo ware mijn wondekens root,
        Ter noot // Geheel ghenezen.
      
      
        Ik gheue elk minnaer wel,
        Beuel dat hij liefde draghe
        Maar dat hij der liefden ghequel
        Zeer fel, Niet en beclaghe.
      
      
        Die hem spieghelt vrij,
        Aan mij // die spieghelt hem zachte
        Elk minnaer verre ofte bij
        Wie tzy, slaat hier wel op achte.
      
      
        Prinse mindy een edel greyn,
        Int pleyn // Als ik bouen maten
        Zo en muechdy der liefden treyn,
        Certain, met recht haten.
