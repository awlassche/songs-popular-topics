
    
      
        LAnghe dagen // wordt ghedraghen
        Godes Arck, door 't woeste veld:
        Menighwerven // most zij swerven,
        Nimmer wordt zij vast ghestelt,
      
      
        Tot zeer schone // Davids Zone,
        Haar in de Heyl'ghe Tempel trekt,
        Daar twee Eng'len // tezamen streng'len,
        't Welk haar tot een deksel strekt.
      
      
        Ziet hier vinden // Gods beminden
        Enen heerlijck teghen-beeldt:
        Want zij sweven // ook in 't leven,
        Rust wordt haar hier mis-ghedeelt.
      
      
        Maar de Here // die wij ere,
        Davids Heer, en ook zijn Zoon,
        Die zit mede // nu in vrede
        Op des Hemels gulde Troon:
      
      
        Daar hij stadigh // zeer ghenadigh,
        Van des Wereldts woeste kust
        Vol van Doornen // zijn verkoor'nen,
        Bij hem stelt in sal'ghe rust.
