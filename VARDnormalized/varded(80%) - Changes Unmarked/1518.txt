
    
      
        EEn Liet // wil ik gaan beginnen,
        't Bediet // zult gij ook wel beminnen,
        Want al mijn hart en sinnen
        Die winnen // nu niet dan vreugd,
        Dus schenckt en brengt
        'tHert denckt // en wenckt // op u rein jeugd.
      
      
        Ik moet // dit nu schoontjens uit vegen,
        Het voet // de mensch in alle wegen,
        Dus moet // ik dit te degen
        Gaan legen // en klaren uit,
        Volgt na // zegt ja,
        Het ga zeer dra al in mijn huyt.
      
      
        Dit glas is nu ledig ghedroncken,
        Zeer ras zat 't worden gheschoncken,
        En niet langh laten proncken,
        Maar loncken // hier aan mijn zij,
        Op dat // het nat,
        Int vat // ons mach maken bly.
