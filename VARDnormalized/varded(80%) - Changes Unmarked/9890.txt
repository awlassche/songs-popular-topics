
    
      
        Gods wijsheid slechtich // voor swerelts oogh,
        Gedaald oprechtigh // van boven hoog
        Hoog disputeren // niet en behoeft,
        Zulk argueren // den gheest bedroeft,
        T'recht simpel verstand,
        In waarheid geplant // Wel berypen cant
        T'woort door den gheest gheproeft.
      
      
        Al wat niet gheestigh // en smaakt gesont
        Dat moet tempeestigh // als onghegront,
        Opt zand verdwijnen, neven den steen,
        Donker bruwijnen // en moghen gheen,
        De waarheid perfeckt // Houden bedekt,
        Oft schoon vertrekt,
        Zij verwint int ghemeen.
      
      
        'sHeeren Ghemeente // met hert en lust,
        In go vereente // staat vast gherust
        Op de rein waarheid // als een pilaer
        Van suyver klaarheid // die hier noch daar
        Niet henen en draeyt // wat wind dat waeyt,
        In haar ghezeayt
        Gods woord, verstercks' eenpaer.
      
      
        Broeders eendrachtich // elk in zijn graet
        Laat ons aendachtigh, na Petrus raad,
        Onder den Heyden // ghelijckt betaamt,
        Go wandel leiden // dat elk hem schaemt
        Tot beterschap // die met ghesnap,
        Door achterklap,
        Ons hebben zal gheblaemt.
      
      
        'tRecht vromer leven // van Abel aan,
        Staat al voorschreven // ons tot vermaen,
        Schoeriem oft pleytken // van enig draad
        'tBleetende geytken // daar ook al staat:
        Niet schuldig en zijt // Dan liefden vlijt,
        Die't gheener tijd
        Den naasten en doet kwaad.
      
      
        In does qua tijden // voorhanden bloot,
        Laat ons vermeyden // in kleen oft groot,
        Dat met besinnen // zelfs niemand wordt
        Buiten oft binnen // van ons verkort,
        Dat zoete lucht // Van goed gherucht,
        Door goede vrucht,
        Van ons zij uytghestort.
      
      
        hetzij vremd' oft vrienden // wie dat ons borght,
        Zo wel wij dienden // te zijn versorght:
        Want, ziet, den name // den welckens wij
        Aenroepen tsame // dezen wordt vrij
        Voor borge betrout // En wie hier op bout
        Hoe zal doch stout
        Dezen te kort doen hij.
      
      
        Wie en zal pooghen // om laten niet
        Enen zo hooghen // borgh int verdriet
        Of zo in schaden // dat men ghestoort,
        Daar door versmaden // synen Naam hoort,
        En 'tvolck vermaert // Van syner aard,
        Iae recht als waert
        Bedrieghelijck in zijn woord.
      
      
        Die dus laat steken // zijn borgh in last,
        Godloze treken // doet hij, tis vast:
        Ook staat gheschreven // hoe dan in al
        T'slecht overbleven // Israels ghetal,
        Gheen leughenen doch // Iae tonghe noch
        Doende bedrog,
        Gevonden zijn en zal.
      
      
        Dus moet den Tempel // Gods staan gesticht,
        Christi exempel // is klaer en licht,
        Te gheenen stonden // was oyt onvroet
        Bedrog gevonden // in zijn mond zoet:
        Wie valsheid bedrijft // Oft leugen verstijft,
        Niet bij hem blijft,
        Maar uit zijn huis hij moet
      
      
        Niemand bedrieghen // gij niet en zult,
        Sprak die van liegen // nooit hadde schuld:
        Ziet, wilt aenschouwen // een Israelijt
        Oprecht vol trouwen // daar gheener tijd,
        T'zy meer oft min // Bedrog en is in,
        Des Heren sin
        Die blijckt hier breedt en wijt.
      
      
        Och gaat niet qualijc // als vriend omgaet
        Met vriend verralijck // ist ook niet kwaad,
        Dat Broeders nouwe // en zouden zeer
        Moghen betrouwe // den Broeders meer:
        Twelck haestich gheviel,
        Soo't straffen ophiel // 'Twaer tot verniel
        Van 'sHeeren volk en leer.
      
      
        'tWaer een oncieren // der leer Christi,
        Zo qua manieren // te lyden vrij.
        Daar soud uit groeyen // een kwaad geclanc,
        Elk soud' verfoeyen // zulk volk eer lanck
        Onstichtich het waar // Ia duysterheyt zwaar
        Voor licht, ook maar
        Voor reucke vuylen stank.
      
      
        Als wij wel schaffen // op waerheyts plicht,
        Zulk kwaad te straffen // is meerder licht,
        Na 'svromer wenschen // en 'sHeeren Wet
        Voor God en mensen // t'antwoorden bet,
        Dan dat men becroont // Oft iet verschoont
        Want t'hem vertoond
        Weerstrijdich Gods opset.
      
      
        De liefd, om sluyten // nu dit propoost,
        Wat daar is buiten // deugd niet altoost:
        Zij denckt gheen schade // te doen iemand,
        Zij slaat wel gade // eens anders pant,
        Gheen vreugd zij en legt // In enig onrecht,
        De waarheid slecht
        Verblijdt al haar verstand.
      
      
        Broeders met enen // wilt doch verstaen
        Men leest in gheenen // Brief van te saen
        Ghestraft te wezen // wat God mishaeght,
        Maar wel wij lezen // van te vertraagt,
        En hoe dat, dat // Vercoren vat
        Paulus, vrij wat
        Daar over heeft gheklaeght.
      
      
        Prinsen bevroedigh // elk int ghemeen,
        Door een ootmoedigh // hert, acht elk een
        Hoogher een ander // dan hij zelfs is,
        Onder malkander // gij zult ghewis
        In vredigh ghespan // Ghegordt zijn dan,
        Met den band van
        Rechter volkomen is.
