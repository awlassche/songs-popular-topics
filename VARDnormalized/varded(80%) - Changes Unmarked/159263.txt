
    
      
        Komt jonghe Ieughden // u verlusticht hier,
        En schept eens vreugden // in den Hoff van Lier,
        Maar wilt niet vluchten // tot een yders spijt
        D'onrijpe vruchten // door haat ende nijd.

        Die niet en weten // de gheleghentheyt,
        Hoe't dor werd groeyende // ziet dezen Hoff
        Zeer aerdigh bloeyende // beschrijft den loff.
      
      
        Ziet nu Diana // met vreughdt komt hier bij
        Ops en Rosana // met veel melody
        Zij plucken geurich // en rieckende cruyt,
        Ialemum treurig // sluyten zij daar uit.
      
      
        Hoort nu Auterpe ,, zeer zoet singhen daar,
        Op de Luyd' Herpe // van Apollo klaer,
        Bij Nethe stromen // waar de Saters al
        Geyt-voetigh comen // in het Liersche dal.
      
      
        Daar Deycken groeyen // sietmen lustig de
        Naiaden spoeyen // met t'soetbletigh vee,
        De Nymphen springhen // uit haar malsche lis
        Armvlechtigh singhen // hoort wat vreugd daar is.
      
      
        Daar Blommen brochten // maakten enen dans,
        Ende zij vlochten // enen groenen Crans,
        Die nu Rhethorica ,, te saemen bint
        Door-wroght van Laura ,, met een groene lint.
      
      
        De Eendrachs vrienden // verdoemen den haat
        Die zocht t'ontbinden // veel twist ende kwaad,
        Comen besetten // dit groen reine land
        Met eel Iennetten ,, en Eeckels beplant.
      
      
        Prinsen verheven // die wilt voeden wel
        V hong-righ leven // met zoet snaeren spel,
        Wilt nijd versaecken // en in't Liersche Hoff,
        Komt u vermaecken // tot Gods meerder loff.
