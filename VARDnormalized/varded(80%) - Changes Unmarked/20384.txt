
    
      
        OM het sondigh overtreden // zijn tot heden
        Ons veel straffen opgheleydt:
        Vol van nare schrik en pijne // elk de zijne
        Best ghevoelt, met onderscheydt.
      
      
        Boven alle straffen swaerlijck // en vervaerlijck,
        Is de Dood vol vrees en schrik:
        Die den mensch komt overvallen // onder allen,
        Dickmaels in een ooghenblick.
      
      
        Daarom soecken haar met zuchten // veel t'ontvluchten,
        Doch vergeefs, waar dat zij gaan:
        Want de Dood volcht op de hiellen // te vernielen
        Al wat leven heeft ontfaen.
      
      
        Als de schielijck' ongelucken // yemandt drucken,
        Treft den mensch Doots ongheval:
        Staat hij 'smorghens op, met vreden, wie weet heden
        Wat hem noch ghebeuren zal.
      
      
        Maar dit straffe monsters pijlen // onderwijlen
        Tot een zegen zijn ghedijt:
        't Welk ons Christus, door zijn sterven // kwam verwerven
        En maeckt' ons Doods prickel kwijt.
      
      
        Want hij heeft de Dood verslonden // en ghebonden
        Haar ghewelt, door zijne macht,
        Dat haar, die in hem ghelooven // hoe verschoven,
        Niet kan schaden Doodes kracht.
      
      
        Hierom vrezen zij gheen sterven // want zij erven
        Na de Dood des Levens Kroon:
        't Moet haar als een deur-gangh strecken // in't voltrecken,
        Tot de vreugd van 's Hemels Troon.
      
      
        Paulus wenscht' te zijn ontbonden // uit de zonden
        Door de Dood, die 't Lichaam scheid,
        En de Ziel verlost van lijden // om verblijden,
        Bij den Heer in eeuwigheydt.
