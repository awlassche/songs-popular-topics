
    
      
        PRonck je van does Maechden
        Heil moet u gemoeten,
        Dus het flickrent daeghen
        Met ghesanghe groeten,
        Zo sulje boeten
        Hier int bedoude gras,
        De schulde // die ik dulde
        Om reden // lest leden
        Doen ik bij u was.
      
      
        Coridon ay stille
        Wat zal dit nu wezen,
        Waar toe deze grille
        Laat myn kruytjens lezen,
        Nimphjen ‘kwaer ghenesen
        Gij mij een kus toe liet
        Gaat vanme // wilt anme
        Rood’ kaecken // niet raecken
        Want ten lust mij niet.
      
      
        Sydy graech om kussen
        Gaat zoekt u beminde,
        Die zal t’vier wel blussen
        Gins aan gheene Linde
        Daar zult gij haar vinde
        Pluckende Lauwerlof,
        Pryst die // bewyst die
        De smart // uws harte
        Tredet van mij of.
      
      
        Straffe Galatea
        Kout als Noorder Klippen
        Om dat ik niet ree sta
        V te bien myn Lippen
        Coridon myn slippen
        Laat los schaemt gij u niet
        V smeecken // schoon spreecken
        Noch loncken // ontfoncken
        Mijn boezem niet.
      
      
        Waar ik eens Amintas
        K’sou eer kusjens kryghen,
        O die zo bemint was
        Wilt vrij daar van swyghen
        K’sie een bloosje styghen
        Int hooghe voorhoofd blank,
        Harder // loopt varder
        Dit lellen // en quellen
        Weet ik u gheen danck.
      
      
        Gij behoort u te schaemen
        Sulckx een Nymph te verghen,
        Met verdichte naemen
        Schaemt root mij te terghen
        Galatea  verberghen
        Wilt u ontstelde sin
        T’lichte // t’ghesichte
        Vws ogen // vermoghen
        Veel Aertsche Goddin.
      
      
        Sou mijn ooghens luyster
        V het hart ontfoncken,
        K’wensje liever duister
        Datse nimmer bloncken,
        Coridon beschoncken
        Is van een vreemde lust,
        Laat mij //en gaat gij
        V zoete // te moete
        En gunt mij eens rust.
