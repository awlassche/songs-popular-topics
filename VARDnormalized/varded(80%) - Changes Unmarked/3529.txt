
    
      
        Gij Amoureuse sinnen
        Die een schoon dochter vrijt
        Stelt niet zo vast u Minnen
        Op dat gij gheenen strijd // lijt
        Voor een zulk cleyn playsier // fier
        De loose Min vermijt.
      
      
        Een heb ik uytvercoren
        Zeer vast door Lieffd' eerbaer
        Doch weynich wiltse horen
        Na mij heur trou Minnaer // maar
        Dat menich droeven traen // saen
        Mij cost met lijden zwaer.
      
      
        Bij Dagen ende nachten
        Heb ik zo droeven ganck
        Gegaan / en tot verachten
        Zo wordt nu mijn ontfanck // cranck
        En voor dit trouwe leed / weet
        Zijt mij noch luttel danck.
      
      
        Dies heb ik niet ontvangen
        Voor al does moeyten ziet
        Dan droeffelijck verlanghen
        Van troost mij gants gheschiet // niet
        Want hare vrienden zijn // mijn
        Heel bitter met verdriet.
      
      
        T'hert heeft zij mij ontoghen
        Deur haar bruin ooch opslach
        Mijn verstand inghesoghen
        Want zij een helen dach // lach
        In mijn herte ghepast // vast
        Dit wercket groot geclach.
      
      
        Lieff wat ghingt gij al spreken
        Tgheen nooit en was ghemeent
        Als dickmaels heeft ghebleken
        Thert dat nooit was versteent // weent
        Om dat gij deur ontrou // ou
        Een kwade Faem verleend.
      
      
        Hoewelt mij zwaer zal vallen
        Te stellen wt den sin
        V Liefste Lieff met allen
        Want tbrengt mij cleyn // ghewin // in
        Daarom seg ik dat gij // mij
        Ghetoocht hebt valse Min.
      
      
        Gij Iongmans vroom in eeren
        Zulk dochters doch versmaet
        Zij doen den druck vermeeren
        En werken vroeg oft laat // haat
        Waar deur een jong ghesel // wel
        Moet leven desperaet.
      
      
        Princessen hoog van moede
        Al zijdy rijk in pracht
        Verneert u als de vroede
        Want trotsheyt hoog gheacht // bracht
        Hemzelven totten val // al
        Ter schant vant heel gheslacht.
