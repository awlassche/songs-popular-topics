
    
      
        OM in den Here // den Echt t’aengaen,
        Die na Schrifts leer // recht mach bestaan,
        Sulcx van twee vromen // geschieden moet
        T’heeft zo genomen // beginsel goed,
        Int Paradijs // door Gods advijs,
        Want groot aenwijs
        Dezen staat rein ons doet.
      
      
        Christi ghemeente // is van zijn vlees,
        En zijn ghebeente // dus na den eesch
        Zo t’hooft is heylich // moet zijn t’lichaem,
        Hij gaf hem veylich // aan t’cruycen raem,
        En suyverd’ haar // in zijn woord claer,
        Op dat zij waar,
        Heerlijck voor hem bequaem.
      
      
        T’moeten twee wezen // leden met t’lijf
        Aan t’hooft ghepresen // houdende stijf,
        Heylsaem eendrachtich // int recht verstand
        Met God almachtich // noch in den bant,
        Der liefden recht // staend’ in den Echt,
        Wie’t zo aenlecht
        Die en bout niet opt zand.
      
      
        Daar dus twee leden // in liefden t’saem
        Den Echt in treden // in Christi naam,
        Daar wil den derden // met zijnen gheest
        Christus wel werden // zo hij ter feest,
        Was ljvich daar // hij voor de schaer
        Een wonderbaer
        Teken deed’ alder eest.
      
      
        Tot Canan stede // in Galileen
        Met liefd’ en vrede // waren ghebeen,
        Onzen Behoeder // en Ionghers vroet,
        Ook Iesu Moeder // ter Bruyloft zoet
        Dus daar gegaan // zeer wel ontfaen,
        Ter tafel saen
        Men t’volck aan sitten doet.
      
      
        De Bruydt wel statich // is te bemoen,
        Verciert ook matich // zat daar als doen,
        Wat uytghelesen // gheselschap fijn
        Eerbaer van wezen // is daar ghezijn,
        Wat soeter praet // maar doch wat raad,
        Recht nu alst gaat
        Ten besten faelter wijn.
      
      
        Van’t wijns ontbreken // den Heer Iesu
        Moeder ginck spreken // wat roert mij u,
        O Vrou mijn ure // is comen niet
        Sprak Christus pure // maar zij ghebiet
        Den dienaars dat // zij deden rat,
        Al t’ghene wat
        Hij haar seyd’ ofte hiet.
      
      
        Zes stenen vaten // stonden van welk
        Twee of drie maten // water hield elk,
        Daar sonderlinghe // sijnde gheset,
        Na die reyninghe // der Joden Wet
        Den Heer gheboot // datmense goot,
        Vol waters bloot,
        De dienaars dedent net.
      
      
        Als zij vol waren // sprak Christus milt
        Tot den dienaeren // schenckt uit en wilt
        Spijsmeester geven // gedaan dus ist,
        Hij heeft beseven // goed onghemist,
        Van smake fijn // den nieuwen Wijn,
        Die water zijn,
        Plach, welk hij niet en wist.
      
      
        Riep goedertier // den Bruydegom,
        T’is de manier // (sprak hij) al om
        Eerst gheeft den besten // wijn alle man
        Maar als ten lesten // t’volck dronk is dan
        Slechten, maar gij // houdt tot nu bij,
        Den besten vrij,
        Zo ik wel smaken can.
      
      
        Dus heeft des Heren // eerst wonderdaet
        Hier gaan vereren // den Echten staat,
        Om brenghen dezen // vrolijck ten ent,
        Liefde moet wezen // het fundament,
        Want int ghemeen // zijn onder een,
        Vreugd en gheween,
        Ghemengelt hier ontrent.
      
      
        Den man om voren // te staan zijn huis,
        Wel na behooren // heeft voor zijn cruys,
        Arbeids beswaren // commer int hert,
        Ook t’wijf int baren // verdriet en smert,
        Toesien men moet // dat watervloed,
        Van druck’ onsoet,
        In Wijn verandert werd.
      
      
        Oorlof met namen // al die den Echt
        Beleven tezamen // oft aengaen recht,
        Blijft zonder flouwen // in vroomheyts jeucht
        T’water van rouwen // door Christi deucht
        Verkeren zal // vrij van misval
        In Wijn, dats al
        In een eeuwighe vreugd.
