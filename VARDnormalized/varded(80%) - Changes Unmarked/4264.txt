
    
      
        Nooit int gemoet // mocht drouver //
        man // versuchten:
        En diet mij doet, heeft touver //
        van // genuchten:
        Tot mijn verdriet, ver-
        wect zij schade // boos //
        ben ik dan niet //
        Een minnaer rade- // loos ?
      
      
        Eertijds ik docht
        Vrueght wil ik mainteneren:
        Meende ik mocht
        Op liefs samblant fonderen:
        Maar zulk confoort
        Schat ik vervlogen // wind,
        Als die haar woord
        Nu gans gelogen // vint.
      
      
        Eigen ben ik
        Haars dienst, bij goed gedoogen:
        T'sy in den strik,
        Cause zijn haar bruin ogen:
        Mijn herte thoont
        Haar eerbaerheyt, naar schuld:
        Ist ongeloont,
        Ik neme mijn gedult.
      
      
        Laestmael wij twee,
        De zoetste vrueght bedreven
        Droufheyt, en wee,
        Hadden mij doen begeven,
        Nu gaetse // bij
        Een ander treden // ziet,
        Och! haetse // mij,
        Ken weet de reden // niet.
      
      
        Keert zij mij t'hooft,
        De schoonste der iongvrouwen,
        Als niet gelooft
        Haar handekens te douwen,
        Staat mijn dangier
        Meer, en meer, voort // te // gaan,
        Al had' ik schier,
        Den wreetsten moord // gedaan.
      
      
        Eylaes, eylaes,
        Dit zijn der minnen // treken,
        Die mij (arm dwaes)
        Mijn crancke sinnen // breken:
        T'hert leit mij zwaar
        D'welck plocht verhueght // te // zijn
        Absent van haar
        Is alle vrueght // fenijn.
      
      
        Nu mach ik wel
        Mijn eigen trooster wezen,
        En mijn gequel
        Met goed verdrach genesen,
        Niemand mij geen
        Bekwamer less // schrijft,
        Nu dat zij een
        S'anders Princess // blijft.
