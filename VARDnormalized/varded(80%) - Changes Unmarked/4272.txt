
    
      
        PLach mijn jonck hert
        Dus te versuchten // zwaar,
        Gelijk het nu begint:
        Zo dat mij smert
        Volgt, met veel duchten // naar,
        Om een maagd wel bemint:
        Geen zo gesint
        Te voren // oyt,
        Geen schoonder kint // en is geboren // nooit.
      
      
        Een sulcken last
        Doet mij der minnen // god,
        Dat ik vertwijffelt sta:
        Of zij mij vast
        Zal doen gewinnen // t'lodt
        Daar ik om dolen ga,
        Ik overslae
        Mijn smerten vrij,
        T'blijft even na // geen troost en is voor mij.
      
      
        Troostbaerder was
        Zij ten beginne // dan
        Zij mij nu dunct te zijn:
        Hoe can zo ras
        Sulcken vriendinne // van
        Mij keren haar aenschijn?
        Meerder gepijn
        Doolyck bereid,
        En heeft nooit mijn // jonck hert neder geleid.
      
      
        Roup ik om troost
        Wie zal verhoren // mij,
        T'is al om niet geweent:
        Voor mijn propoost
        Verstopt haar oren // zij,
        Rebel, en hert versteent:
        Ziet mij vercleent
        Arm serviteur,
        Ander verleend // zij troost, en mij rigeur.
      
      
        Van Antheros,
        Is zij deurschoten // sterk,
        Wiens schickt vercoelen biedt,
        Dies Atropos.
        Met een gesloten // zerck
        Fineert al mijn verdriet:
        Misdeed' ik iet,
        Dat zij mij haat,
        Och neen ik niet // dan een rein hert toestaet.
      
      
        Scheyd' ik dan met
        Een deerlick schromen // ach?
        Wt dezen weder-spoet:
        Van zulk op set
        Zij haar beromen // mach,
        Als een maagd straf verwoet,
        Mijn rein gemoet
        Niet falen // zal,
        Dwelck met zijn bloed // t'rantsoen betalen // zal.
