
    
      
        Phoebus God vant zoet Musijck
        Maakt gij me ongheneucht
        Om u Daphne schoon en rijk
        Gij varen laat veel vreugd
        Does Goddin,, vliet de min
        En het speelen zoet der Lier
        Dies verkeert // transformeert
        Zij in enen Lauvverier
        Van ghelijcken // gaat
        Nu afwijcken // kwaad
        Een door Venus raad.
      
      
        Ik nu wel beclaghen mach
        Dat door lieffd' mijn vrijheid
        Die ik zelfs te bruycken plach
        Dus schandich heb gheleyt
        Op een schoon // maagd ydoon
        Die ghevaen mij heeft gebracht
        In het net // en percket
        Waarom dat Vrou Venus lacht
        Mijn lieff waerde bruyt
        Door u zoet gheluyt
        Mij verlost hier uit.
      
      
        Wilt Lieff overdencken staech
        Al mijn affectie groot
        Wat Lieffd' ik int herte draech
        Om u zoet wezen bloot /
        Edel maagd // niet verjaecht
        Hem die u zo troulijck mint /
        Maar veel meer // crijcht de eer
        Dat gij mijn pijn overwint /
        Vytvercoren // rein
        Hoog geboren // greyn
        Lavende Fonteyn.
      
      
        Waarom Lieff acht gij zo slecht
        D'onlydelijcke pijn
        Die ik Minnaer trou oprecht
        Ben dragende devijn
        Laat gij mij // dencket vrij
        Wat met waarheid ik beken
        Waar ik heen // ben ghetreen
        Niemand wetet waar ik ben
        Want mijn traneleyt
        Als de swane schreyt
        Midts ik ben verleid.
      
      
        In zwaar lyden ende druck
        Moet ik dus leven ziet
        Met patientich ongheluck
        Tot dat mijn Lieff eens biet
        't Zoete woord // van confoort
        Ende helpen goedertier
        Mij Minnaer uit 'tghevaer
        En dit overgroot dangier
        Ik beclaghen // mach
        Al mijn dagen // ach
        Dat ik oyt haar sach
      
      
        Om mij peysen wilt schoon Lieff
        Als ik ben uyter stee
        Denckt om hem die u verhieff
        Wat u persoon hem dee
        Gij Iongvrou // voor het trou
        Minnen hebdy hem ghehoont
        En voor 'tsoet // bitter roet
        In de steed' van lieffd' gheloont
        Dus dan mijn Voochdes
        Onthout deze les
        Adieu schoon Princes.
