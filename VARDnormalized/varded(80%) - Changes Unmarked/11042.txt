
    
      
        'T godlijck wezen goed // heeft des mensen ghemoet
        Met vlees en bloed // becleet in dit foreest
        Die de mensch onvroet // twist in zijn leden voet
        Tot zijn onspoet // van vleysch tegen den Gheest,
        Zo dat in onrust minst en meest
        Dier leeft bedeest // door aartsche lusten swaar,
        Maar God den sulcken van quaalen gheneest
        Die zijn woorden vreest // in dezen nieuwe Iaar.
      
      
        Die d'oude mensch trekt uit, met lust die daar wt spruyt
        Wordt dan Christi bruyt // en trect den nieuwen aan
        Zo t'Sarpent beduyt, verschut hij zijnen huyt
        Soect het eenich cruyt // dat hem verlost van waan,
        Die zal van dezen Pellicaan
        Ghenaad' ontfaan // als een ghetrou dienaar
        Dat hij het vlees den Gheest maakt onderdaan
        En leest onbelaan // in dezen nieuwe Iaar.
      
      
        Maar eyghen wil verblindt // die iet geschapens mint
        Daar zelven bevint // in benaude commer groot
        Want de schrift ontbint // wie vleyschelijc is gesint
        Dat die hier verwint // int ent tot loon de dood
        Die bracht den eersten mensch in noot
        Doen lust hem bood // D'appel als een verraar,
        Hebt hier op acht gij mensen snoot
        D eyghen wil verstoot // In dezen nieuwen Iaar.
      
      
        Christus ons Eglentier,, In liefd' bloeyende fier
        Heeft u Drinskens hier // verlost uit dezen strijd,
        Schuylt onder den banier // van die u cocht zo dier,
        Zo wordy schier // van eyghen wil bevrijt,
        Ziet dat gij ook quaa lusten mijt,
        U hart besnijt // en zoekt does Middelaar,
        Elk Spieghelt hem tot zijn eyghen profijt
        Zo leeft hij verblijd // in dezen nieuwen Iaar.
