
    
      
        PAscha, dat hier // voor bij te lyden,
        'tSondigh bestier // moeten wij myden,
        'tDangier // van dien
        Als fenynighe Slange vlien.
      
      
        De Christen schaer // heeft hier beneden
        Geen blijf-stadt, maar // voorbij moet treden
        Al naar // de stad
        Daar haar // hert is, en haren schat.
      
      
        Gheacht zijn zij // als vreemdelinghen
        Of wij voorbij // door een stad ginghen,
        Daar wij // schier niet
        Ons aantrekken watter gheschiedt.
      
      
        Of men hier hoort veel vreugd verwekken
        Wij moeten voort // recht henen trekken
        Na 't woord // van hem
        Die ons roept na Ierusalem.
      
      
        Tis groot abuys // hier lanterfanten,
        Om vlieden 't cruys // en blyven planten,
        Zijn huis // opt zand,
        Buiten 't eeuwighe Vaderlandt.
      
      
        'S Heren dag zal // als strik bevanghen
        Die op dit dal // wonen en hanghen
        Met al // den sin,
        Op de dingen die hier zijn in.
      
      
        Want vyerigh moet // vergaen does stede,
        Iae al haar goed // en borghers mede,
        Dus doet // hij wel
        Die uit midden van haar wijkt snel.
      
      
        Want die hier geen wooning en maken,
        Dan op de been // om voort te raken,
        Alleen // door vlijt,
        Die blyven in 'sHeeren bevrijdt.
      
      
        'tGhemoedt geschort // op, onversteenight
        Het herte wordt, met God verenigd,
        Ghegort // in band
        Der liefde die vyerigh brandt.
      
      
        'sLams goedheid meest // de zulke smaken
        Ter Paesschen-feest // daar zij vermaken
        Den gheest // met vreugd,
        Die de ziele van binnen verheugt.
      
      
        Door d'ongheree // en onghebaende
        Woestijn oft Zee // recht henen gaande,
        Gheen wee // noch leer
        En leide ons af van den Heer.
      
      
        Zij zullen met // hem t'huys verblyden
        Die na zijn Wet // verwinningh stryden,
        Ook het // hier zal
        Wel haast wezen volstreden al.
      
      
        Dan zal men hoen // en cranssen delen,
        Des levens groen // en den gheheelen
        Roof koen // Sioen
        Zal onsterffelijckheydt aandoen.
