
    
      
        EEr, Loff, Prijs, int Hollants pleyn
        Zij u o Liefste schoon
        Door de zedicheyt ghemeyn
        En aerdicheyt ydoon
        Gratieus // zoet en heus
        Zijt gij maechdelijcke zaad
        Waarom trou // Eel jongvrou
        Ik u Min in Lieffdes graet
        O gij reine // Maagd
        Die alleyne // plaecht
        Hem / die Lieffd' naar jaecht.
      
      
        Menich lieven zoeten praet
        Wij hebben tsaem ghepleecht
        Lieffd' waarbij de wereld staat
        V nooit en heeft beweecht
        Mijn begheer // was op eer
        Door een onghevalste Min
        Maar gij vroet // Liefgen zoet
        Van mij keerden uwen sin
        Ik betrouden // mijn
        Op u goude // schijn
        Maar die bracht does pijn.
      
      
        Als Aurora in claerheyt
        V rein Versiersel blinckt
        Waarin hert en sinnen leit
        Die tsamen gij bedwinckt
        Als de Son,, Helicon
        Overstraelet wel van vart
        V ghesicht // alzo licht
        Ook inwendich me het hart
        Dat mits dezen // dan
        Het niet wezen // can
        V te wijken van.
      
      
        Met des wijsheyts deuchden croon
        Ik chieren zal u hoofd
        En dan stellen opten troon
        Van Pallas, dit geloofd
        D'in gheplant // cloeck verstand
        Sulcx in alles Meriteert
        Dan het schijnt // diet zo pijnt
        Gij in Lieffden niet begeert
        Mijn verheven // dier
        V trots leven // fier
        Can verandren schier.
      
      
        Mijn Princes vol hoocheyts aard
        Ik wensch u meerder vreugd
        Dan gij voor u Dienaar spaert
        Die u niet jont dan deucht
        Die gij loont // onverschoont
        Doch ik u niet haten // zal
        Dies voort lest // op zijn best
        Moetmen ook wat praten // mal
        Hout u waeren // gij
        En laat varen // mij
        Adieu segh ik dy.
