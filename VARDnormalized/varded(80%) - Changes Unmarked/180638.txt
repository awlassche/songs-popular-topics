
    
      
        O Broeders rein // looft den Here ghestade,
        Die hier int pleyn // laat vloeien zijn ghenade,
        Als een Fonteyn // voor die af staan van t’quaede,
        En voort alleyn // met Christum gaan te rade,
        Die zo het graen // in go aerd’ ontfaen
        Worden van dezen sad,
        Kinderen vroet // van God den Vader goed
        Door zijns zoons bloed // vergoten overvloed
        Ligt onder voet // dood en hell vertreden
        Lof hem zijn moet // van zijn ghenade zoet
        Wie boet // hier doet // zij staan noch heden.
      
      
        Eeuwich houdt stant // s’Heeren goedheid ghepresen,
        Hij heeft gheplant // zijn gemeind uytgelesen
        Want triumphant // ten Hemel opgheresen
        Ter rechter hand // zijns vaders court na dezen
        Sant zijnen gheest // met een winds tempeest
        Tonghen ghedeelt int wezen
        Vierich en claer // op yegelijck van haar
        T’ghetal aldaar // in zijnen naam te gaer
        Dat en was maar, ontrent twintich en honderd,
        Den Gheest hier naar // werkende wonderbaer
        Heeft zwaar // de schaer // des Volks verwondert.
      
      
        Van Zuyt oft Noort // uit menighe lantpal
        Heeft daar gehoord // een yeghelijck zijn tale
        Betuigen t’woort // en Gods werken loyale
        Vrees en discoort // hadde t’volck generale
        Want deze zijn // vol zoeten Wijn
        Ghedroncken al te male
        Spraken rebel // eenigh’ uit spot en spel
        Op zulk vertel // antwoorde Petrus wel
        Dits gheen voortstel // van droncke lieden
        Maar door t’bevel // Gods na t’profeteersel
        Ioel // blijckt snel // hier dit gheschieden.
      
      
        O Prins o Heer // u macht liet gij aenschouwen,
        Van leden teer // een sterk huis ginght gij bouwen,
        Meer ende meer // lieten Mannen en Vrouwen,
        Op uwe leer // haar dopen met betrouwen,
        Maar door den nijt // der Joden wijt
        Verstroyt door zwaar benouwen,
        V dienaars jent // van t’nieuwe Testament,
        Tot s’weerelts ent // maakten u woord bekent,
        Heer excellent // lof prijs moet gij gewinnen,
        V woord ontrent // ons laat zijn ongheschent
        Present // gheprent // in onze sinnen.
