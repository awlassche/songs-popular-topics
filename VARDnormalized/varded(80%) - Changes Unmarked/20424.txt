
    
      
        Vrolijke geesten // die liever zijt
        Op Kermis-feesten // als in den strijdt,
        Ist u behaeghen // rijt vaerdigh uit,
        Treet op de wagen // of in de schuit,
        De vrienden van 't spaer // die nemen u waar,
        Weest wellekom gasten // Wel benje daar.
      
      
        Ziet wat hier binnen // Den kramer doet,
        Om wat te winnen // Veylt hij zijn goed,
        Den waeffel-kraemer // Ik heb zeid hij
        De beste kaemer // Gaat niet voorbij.
        Dus wordje van d'een // En de ander ghebeen,
        Om dat-je sout // Jouw geld besteen.
      
      
        V geld verhandeld // u tijd besteedt,
        Na buiten wandelt // En daar wat eet,
        Dan met u Liefje // daar weer in 't groen,
        Steelt Snobbel-diefje // dan soon op soon,
        Of graest u bemint // zoekt boompjes geblint,
        Of doet daar gij // vermaeck in vindt.
      
      
        Leit dan dat meysje // daar gij mee gaat,
        Na duyn een reysje // of soose praet,
        Haar zwakke leden // niet toe vertrouwt,
        Zo veer te treden // blijft dan in't hout,
        Of geeft ze haar stem // na Roomen 't heeft klem,
        In Emaus, of Jerusalem.
      
      
        Ziet daar de brackjes // van Spaeren-stadt,
        Drinken niet zwackjes // 't Haerlemsche nat,
        Bij drie vier glaesjes // van bier, of wijn,
        Nooit fraeyer baesjes // wout altijdt zijn,
        Een vijf of zes vaen // kan dan daar niet schaen,
        't Is kermis // 't moeter nu opstaan.
      
      
        De gheest moet woelen // van buiten weer,
        Stracx na de Doelen // elk set hem neer,
        Daar weer ghedroncken // tot datse sluit,
        Smorgens beschoncken // de poort weer uit,
        Daar menich in 't wout // in 't veld bedout,
        Of op een bank // zijn nacht-rust houdt.
      
      
        D'een zijnde wakker // geeuwt als een koe,
        De ander zijn macker // ziet wonder toe,
        Hoe duycker koom ik // dus op een bank?
        Wat drommel droom ik // 't is door den drank,
        Waar kwam ik dus krom // ik wed dat ik om
        Het jaar niet weer // te kermis kom.
