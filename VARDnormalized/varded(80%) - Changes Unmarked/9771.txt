
    
      
        MVsijck gheschal
        Heeft gevonden Iubal,
        Eerst en vooral // om te verblijden
        Int jammer dal,
        Vol lijdens en ongheval,
        Vinden men zal // van ouden tyden,
        T'ghebruyck vant singhen vrij,
        In Mesopotamy,
        Om vreugd t'hanteren saen,
        Want Laban op de baen,
        Van God beraen // tot Iacob zeide,
        Waerdy zo niet gegaan,
        Harpen hadd' ik doen slaan,
        En u gedaan // met sank gheleyde.
      
      
        Moses die sank,
        Naar Pharaons onderganck,
        Den Here danck // lustig om horen,
        Myriam dwanck
        Ook den gheest, dat zij eer lanck,
        Sank, ende spranck // die vroukens voren,
        Ook over een fonteyn,
        Sank Israel ghemeyn,
        D'een voor en de ander naar,
        Debora sank eerbaer,
        Barack met haar // God danck bewezen,
        Twas zoet om horen daar,
        Ook al der Vrouwen schaer,
        David zeer klaer met sanghe presen.
      
      
        Victory schoon,
        Brandtoffer, Coninghs ghecroon,
        Israel ghewoon // was te vercieren,
        Met zoeten soon,
        Der Musijcken sangh ydoon,
        Na 't Schrifts vertoon // op veel manieren:
        Die oude Vaders wijs
        Gheeft Iesus Syrach prijs
        Dat zij dichten playsant
        Gheestelijcke Lieden // want
        Groot haar verstand // was in Muzieke
        Haar ghedicht stond gheplant
        Al op maet egelant,
        Als nu int land // die Rethorijcke/
      
      
        Ghedichten zoet,
        Door 'sHeeren Gheest overvloed,
        Heeft gemaakt vroet // David Gods diender
        T'Christen ghemoet,
        Tot troost ende leeringhe goed,
        Ghelijck ook doet // Assaph den Siender,
        Salomon ook verlicht,
        Heeft menigh Lied ghedicht,
        Want hij was wijzer dan
        Ander dichters, Ethan,
        Chalchal, Heman // oft Derda waren,
        Dat Gods lof vroeg began
        In Liedekens men kan
        Bewijzen van // over veel Iaren
      
      
        Noch principael
        Salomon, na zijn verhaal,
        Sanghers loyael // en Sangheressen
        Hadde ter zael,
        Die hem songhen menigmaal
        Schoon musicael // verscheyden lessen,
        Vol Gheestelijck gheluydt,
        Van een Minnaer en Bruydt,
        Als men te recht in ziet
        Der Cantijcken bediet,
        Ook menigh Lied // in die Profeten,
        Tis al van God gheschiedt,
        Anders ten baet al niet.
        Daar zonder iet // doen, oft vermeten.
      
      
        Paulus devijn
        Schrijft, en suypt u niet vol Wijn,
        Maar elk termijn // vol geests wilt blyven
        Int herte dijn
        Laat overvloeyende zijn
        'tWoordt Christi fijn // om wel beclyven,
        Psalmen, Lofsanghen meest,
        Liedekens na den gheest,
        Wilt hebben t'elcker stond,
        Onderling in den mond,
        In 'sherten grondt // spelen en singhen
        Den Here die u jont
        Zijn vredelijck Verbond,
        T'leven ghesont // en alle dingen.
      
      
        Die Majesteyt,
        Ons Prinsen wijdt verbreydt
        Dijn lof gheseyt // ende ghesonghen,
        Met goed bescheyt,
        In Christen zedicheydt,
        Laat zijn bereid // ons hert en tonghen,
        O broeders, nacht en dag,
        Daerment wel horen magh,
        In des singhens voorstel
        Om met inwendig spel
        Ons ghemoedt snel // troostigh versaden,
        Want tis Paulus bevel.
        Dus en doen zij niet wel,
        Die ons rebel // 'tsinghen versmaden.
