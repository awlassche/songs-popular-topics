
    
      
        Ia des Meys tijd, verblijd // nu int ghemeene
        Zonder respijt, subijt, die jonghe jeucht
        Met groot jolijt // en vlijt // is elken ene
        In daertsche krijt // want zijt al doen met vreugd
        Om alzo met gheneucht // te treden na de wey:
        Want Bloemkens rein // int aertsche dal
        Groot ende cleyn // staan over al
        Lieflick certain // breet ende smal
        Met dezen coelen Mey.
      
      
        Aurora goed // die moet // als dan ook blijcken
        De kruiden zoet, die doet // zij groeyen wel,
        Den water vloet // met spoet // vliet lancks de dijcken:
        Die God behoet // zeer Vroet // int loopen snel
        Der Amoreusen spel // Hoortmen in die country
        Als Luiten // Fluyten // zoeten toon
        Ontsluyten // wten // thert ydoon
        Daar buiten // ruyten // voglen schoon
        Met dezen coelen Mey.
      
      
        Hoe triumphant // playsant // moetet daar wezen
        Waar met verstand // die hand // een wordt gedaan:
        Die lancx den cant // van tlant // gaan om genesen
        der liefde brand // en vant // door zoet vermaen
        Daar twee gheliefkens staan, plucken Acoley:
        Die in eeren zo paren // dan
        Onzen God wel vergaren // can
        Zo krijcht een jonckvrou haren // man
        Stracx in den coelen Mey.
      
      
        Prinsen eerbaer // te gaer // wilt diviseeren
        Liefs schoonheid claer // van haar // niet en veracht:
        Oft groot misbaer // voorwaar // sou u causeeren
        Daar gij eenpaer // minnaer // af wordt verwacht,
        Tis goed wel overdacht // worden menigerley
        Zo gij u lief gewinnen // meucht
        Suldy dan zelfs bekennen // deucht:
        Waar door u thert van binnen // vreugd
        Gheeft met den coelen Mey.
