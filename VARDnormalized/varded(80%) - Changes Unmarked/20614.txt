
    
      
        ‘,,Ghelijck stadich
        ‘,,In de zee//t'roer loos schip/
        ‘,,In perijckel sweeft dach ende nacht;
        ‘,,D'welck smijt schadich
        ‘,,Licht in twee// op een clip/
        ‘,,Oft het zingt door t'groot onweders kracht/
        ‘,,De baren heffent op/en neer/
        ‘,,Nu hooghe // uit d'ooghe/
        ‘,,Dan schier ten gronde weer.
      
      
        Zo ist even
        Met den mensch // soo't hier blijckt
        Aan Eudotia weert ghelauw'riert/
        Wyens leven/
        Ende wensch // recht ghelijckt/
        T'schip/ dat door de baren wordt ghestiert!
        Dats door Fortun', diet hoochst / meest treft/
        Die weder // worpt neder
        Al 't ghene zij verheft.
      
      
        Deze Maghet
        Arm van staat/en gheslacht/
        Die Atheneas eerst was genoemd/
        Heeft behaghet
        Om haar daad // hoog gheacht/
        Wijsheid/deucht/en schoonheid wijt beroemt
        Den Keyser Theosiom,
        Die houde // en troude
        Met deze weerde blom.
      
      
        Als nu prachtich
        Was in weldt // roem/ en eer
        Does die boven wensch verheven sadt/
        Die groot machtich/
        Scheen ghesteldt // tot een leer/
        Schier op 't hoogste van Fortunas radt/
        Als Keyserinne boven al
        Tot haren // zeer zwaren/
        En al te droeven val.
      
      
        Met beliefte
        Heeft den Prins // op een tijd
        Haar/een schone raere vrucht vereert/
        Welke giefte
        Onversins // heeft vol nijt/
        s'Keysers min in jelousy verkeert/
        Midts zij die aan een ander gaf:
        ,,Hoe slechte // t'onrechte
        ,,Lijt menich smaet en straf.
      
      
        Dezen goeden
        Heer/ liep ras // tot den stoel
        Vanden Keyser/en schonck hem den Vorst:
        Die vermoeden
        Strackx dat 't was // haren boel/
        Die zij dezen Appel geven dorst/
        En naar wat ondersoechkingh/knap/
        Verlietfe: // en stietse
        In droeve ballinckschap.
      
      
        ,,Leert hier mensen!
        ,,Dat de snoo // wereld is
        ,,Recht een schaduwe/want als gij meynt/
        ,,Dat gh'u wenschen
        ,,Hebbet/zo // sijt gij wis
        ,,Niet/ gansch niet/ja min dan niet vercleynt.
        ,,Schout does vermomde streelster dan
        ,,Dient mere // den Here
        ,,Diet eeuwich loonen can.
