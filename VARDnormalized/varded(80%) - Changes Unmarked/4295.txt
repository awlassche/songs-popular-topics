
    
      
        MInsaem beeldeken amoureus //
        Die t'heyrcracht overwint,
        Troostbaer herteken gratieus //
        Gij zijt een seedbaer kint,
        Gemint // en wtgelesen //
        Int wezen // geen als gij //
        Daeromme //
        Rein blomme //
        Ik comme // tot dy //
        Mijn plagen //
        en clagen //
        Helpt dragen // met mij.
      
      
        Aerdich, edel, besneden // rijs,
        Gesproten magnifijck,
        Hester zydy in seden // wijs,
        Waar is doch ws gelijk?
        Aardrijk // vervult dijn fame,
        Eersame // maget smal:
        O puere,
        Figuere,
        De fluere // van al,
        O schone
        Persoone,
        De krone // zijn zal.
      
      
        Reyct mij heden dijn witte // hand,
        Dus lange niet vertouft,
        Mijn teer herte deur hitte // brand,
        D'welck meer dan troost behouft:
        Bedrouft // ligh ik ter neder,
        Keert weder // zoet aenschijn:
        Oorzake?
        Dijn sprake,
        Heeft smake // zo fijn:
        Dat heden,
        Dijn reden,
        Gebeden // moet zijn.
      
      
        Is bewezen zulk vrouwen // recht,
        Aan mij, in voller macht,
        Dat ik uwen getrouwen // knecht,
        Ben aangenaam verwacht:
        Den nacht // is dan verdwenen,
        Vaart henen // vals crackeel:
        Geen perten
        En smerten,
        Twee herten // geheel,
        Met sinnen
        Wij winnen
        Der minnen // casteel.
      
      
        Adieu, minnelick // cordiael,
        Jonck herte, heusch, en vrij:
        Die goddinnelick // teenemael
        Mijn vreugd zijt even bij,
        Zo bly // als wij nu scheiden,
        Verbeyden // wilt mij saen:
        
          T'onvroede
        
        Int goede
        Ten bloede // neemt aan,
        Houdt stede,
        In vrede,
        Hier mede // wij gaan.
