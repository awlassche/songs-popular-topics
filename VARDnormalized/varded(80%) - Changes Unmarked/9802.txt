
    
      
        T'mijden schijnt waar // al klaer,
        Voort oordeel, maar // daar naar,
        Met den sondaer // eenpaer
        De sommige wel brast,
        T'lidt eerst is afghesneden
        Is iet aent lichaam vast,
        Maar heeft het straffe leden,
        Daar aan het niet en past,
        Tot dat het weder wast,
        Door boet oft zulk berouw,
        Dat men ghewichtigh // plicht
        Voor elcx ghesicht // een licht,
        Zo wordet dicht, // hersticht
        Aen't gheestelijck ghebouw.
      
      
        Al zonder spijt // oft nijd,
        Eerst men met vlijdt // afsnijdt,
        En na der tijd // men mijdt
        Zulk deessem suer beslagh,
        Ander schijn wijze // temen
        En hebben hier gheen dag,
        Want men gheen spijze // nemen
        Noch handelen en mach,
        In tijdelijck bejagh,
        Met t'afghesondert lid,
        Op dat het vroet // tot Boet,
        Door schaamte moet // en goed,
        Leer doen, tot zoet // behoed
        Der zielen dient doch dit.
      
      
        T'ghebodt staat kort // begort,
        Wie daar aan hort // ten wordt
        Niet omghestort // 'tverdort
        Niet dat Gods hand // zo plant,
        Vergheefs pooght hem die boze
        Met zijn dienaren, want
        Te maken vremde gloose
        Over 'tGhebodts verstandt,
        Niet goed wezen en kant,
        Tis nietigh en onnut,
        Ia schalcken wind // men vindt
        Die als een kind // zo int
        Begrijp begint // zijn blindt,
        Hij raeckt wel in den put.
      
      
        Menigh al sacht // gebracht
        Zo in der gracht // versmacht,
        Die minder acht // de kracht
        Der Schrift, dan dat hij voort
        Brengt in gheschreven // boecken
        Wtlegh daar t'niet behoort:
        Maar wiltmen 'tleven // soecken
        Men blijft al onghestoort
        Bij Gods ghebodt en Woord,
        Dat en bedrieght niemand,
        Hij doet niet wel // die snel
        Met Gods bevel // houdt spel
        En achtet fel // instel,
        Of raad vanden vyandt.
      
      
        Elk vrome Christen // mist
        Verkeerdheid, twist // en list,
        Gheen wijsheid ist // hij ghist,
        En bouwt op waerheyts grondt,
        Hij laat hem blijde // raden
        Met leeringhe ghesondt,
        En die zo wijde // paden
        Leren, wijkt hij terstondt,
        Hij hout hem aan 't Verbond
        Vast in den nauwen pad,
        Hemel-wijsheyt // hem leidt,
        Zo God hem zeid // bescheydt,
        Hij tot arbeid // bereid
        Is om volbrenghen dat.
      
      
        Maar ruymer baen // te gaan,
        'tVleesch lust wel saen // daar aan,
        Om onderdaen // te staan,
        Te stijf is synen neck,
        Hem dunkt Gods leer // is schandigh,
        En vindt daar aan ghebreck,
        Om dat hem zeer mishandigh
        Is te doen zulk ontreck,
        Het loopt al buyter steck
        Met laster en verwijt,
        Van al te stijf // ghedrijf,
        Tot wederkijf // van 't lijf,
        Dat Man en Wijf // 't gherijf
        Van 't Houwelijck gaan kwijt.
      
      
        'tVleesch hadd' al bet // belet
        In zijn opset // doe het
        In Moses Wet // zag met
        Stenen doen sulcken Ban,
        Misschien van grouw'lijck // scheiden,
        Zo tussen Wijf en Man
        Brekende 't Houw'lijck // zeiden,
        Ghelijck als nu, ook dan
        Eenighe veel daar van,
        Het schad' al meer 'tlichaem,
        Doe men zo bloot // gebood,
        Te stenen dood // was groot
        Des vleeschs aenstoot // want nood'
        Ist den Heer ghehoorsaem.
      
      
        Princess Bruyt // rein Cruydt,
        Hof int besluyt // laat uit
        Al 'tvreemt gheluydt // en spruyt,
        Houdt uwen goeden rom,
        Al die't kastijden, haten
        Achten u leden dom,
        Die haar besnijden // laten
        Om 't Evangelium:
        Op dat u Bruydegom
        V nemmermeer versaeckt,
        Zo blijft al bij // Christi
        Simpelheyt, gij // komt vrij
        In heerschappy // als hij
        Vindt dat gij vlijtigh waeckt.
