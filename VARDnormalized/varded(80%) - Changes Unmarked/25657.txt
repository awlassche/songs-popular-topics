
    
      
        In onze roepen ende kermen
        Dat God hem onzer wil ontfermen,
        Zo laat // die maet // ons doch betrachten
        In ons ghedachten.
      
      
        Aenmercken waar toe hij gaat keren,
        En zijn barmherticheyt vermeeren,
        Dat wij // zo vrij // die met verlanghen
        Moghen ontvangen.
      
      
        Crachtich de zonden hier beclaghen,
        Want die nu leed en rouwe draghen,
        Haar is // ghewis // de Here goedich,
        En zeer lanckmoedich.
      
      
        Op Ninive was God verbolghen
        Maar doe haar beteringh deed’ volghen,
        Was God // doe tot // haar zeer beradich,
        Ende ghenadich.
      
      
        Bij den teghenspoet int benouwen
        Staat God barmhertioch int aenschouwen,
        Is groot // in noot // zijn zijn ghenaden
        Groter in daden.
      
      
        Groot ist aan Israel ghebleken,
        Deur t’roepen en t’ghebedt uytspreken,
        Doe zij // na bij // nu den dood waren
        Naar Schrifts verclaren.
      
      
        En die den Armen hier bedincken,
        Int cleeden, spijzen ende drinken,
        Does zijn // nu fijn // weerdt te verbeyden
        Barmherticheyden.
      
      
        Naar Christi Iesu eyghen reden,
        Die de barmherticheyt hier deden,
        Zij zal // voor al // haar ook gheschieden
        Die’t zo berieden.
      
      
        Cortlijck zal hij barmhertich wezen
        Den ghehoorsamen in Gods vresen,
        Erbermt // ontfermt // hem haarder spoedich
        Nu overvloedich.
      
      
        Recht mach men dit aan Paulo merken,
        Dat God barmhertich in zijn werken
        Is ziet // t’gheschiet // doe hij sprak coene
        Wat zal ik doene.
      
      
        In Godes vreze te aenverden,
        Betrouwende, daar in volherden,
        Verwerft // beerft // ook vroeg en spade
        Hier Gods ghenade.
      
      
        Int Boek der Psalmen staat geschreven
        Dat God hem ontfermt hoog verheven,
        Als een // ghemeen // vader niet minder
        Over zijn kinder.
      
      
        Naelatende nu die zondaren
        Die tegen ons doen, niet beswaren,
        Maar kwijt // altijd // schelden van herten
        Al zonder smerten.
      
      
        En vergheven al zonder trueren
        Onzen Naasten tot elker uren,
        Wie vroet // dus doet // zal God vercooren
        Nu gheern verhoren.
      
      
        Neemt dit van mij liefste int goede,
        Met enen Christlijcken ghemoede,
        Ik groet // u zoet // vrij onverholen
        Weest God bevolen.
