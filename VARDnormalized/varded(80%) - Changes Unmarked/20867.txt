
    
      
        Ik wil gaan loven // den wond'ren Held,
        Sint Joost, een Konings Soon,
        Die heeft hier boven // in 's Hemels veld,
        Ontvangen zijne loon,
        Om dat // de schat // des Koninkrijks Hij liet,
        Om dat Hij heeft gevliet
        Het tijdelijck gebiedt,
        't Welk Hem geven sou zijn Broe'r,
        Van wien heym'lijck weg Hij voer;
        Waar voor eeuwig God Hij ziet.
      
      
        Zegt om wat reden // hebt Gij, Sint Joost!
        Verlaten all' u goed,
        En kost'lijckheden? // zegt tot wat troost
        Wierd Gij enen armen bloed?
        Gij docht // Gij zocht // te volgen uwen Heer,
        Die op does aerd' kwam ne'er,
        Verlatende zijn eer,
        En zijn heerlijcke prael,
        En zijn schoon' gestarde zael,
        Voor ons all' uit liefde teer.
      
      
        O Wat gedachten // hebt Gij gehad
        Als Gij bij nacht weg liep,
        En als Gij jachten // na 'n ander stad,
        Terwijl uwe Broeder sliep!
        Ei! zegt // en legt // ons uit eens deze daed,
        Waarom uwe hoge staat,
        En Broeder, Gij verlaat?
        Op dat yder prijzen magh
        Uwen eersten vluchtens dag,
        En die niet en acht als kwaad.

        (Die uwen Broeder was)
        Wou zijne wooningh // ja altemaal
        Zijn goed, verlaten ras:
        Want Hij // wou vrij // van de aerdtsche goed'ren zijn.
        Des wereldts zoet fenijn,
        Was Hem een grote pijn:
        Hij koos U, maar Gij slough 't af
        Als Hij 't U te kennen gaf;
        Gij woud' volgen Christi lijn.
      
      
        Gij naemt acht dagen // om te bezien
        Wat Gij dewijl sou doen.
        Gij kreeght behagen // om weg te vlie'n,
        Gij ging U henen spoe'n:
        Het Rijk // als slijk // Gij achten anders niet;
        De Kroon was U verdriet;
        Gij beter U versiet.
        Op den Heer had Gij uwe sin
        Gants gesteld, en uwe min;
        Gij op Jesu U verliet.
      
      
        Den Heer der Heren ,, (gelijk ik merck)
        Heeft dit zo aengestaen,
        Dat Hij vereren // dit schone werk
        Wou op does werelts baen.
        Gekroont // geloont // heeft God zijn lieve vrinde,
        Die Hem zo was gesint,
        Die Hem zo had bemindt,
        Dat Hij om zijn 's Heren wil
        't Rijke Koninghs-hof af-vil,
        En aan God zich gants verbindt.
      
      
        De wreede beesten // de fell Leeuw,
        Hem waren onderdaen:
        De boze gheesten // met haar geschreeuw,
        Gheen vrees Hem jougen aan:
        De Hell' // nam snel // van Hem de vuyle vlucht;
        De Hell' was gants beducht
        Om Sinte Joosts gesucht.
        De Eng'len uit des Hemels zael
        Hem besochten menichmael,
        En Zij de'en Hem aan genucht.
      
      
        Daar was 't aenschouwen // aan Goods Outaer,
        (Waer-aen Goodsdienst Hij de'e)
        Dat wierdt gehouwen // in 't openbaer
        Een hand zo wit als snee,
        Een hand // met brand // en helder licht verciert,
        Die heerelijck Hem vyert,
        Van God aldaar gestiert:
        Deze heeft gebenedijdt
        Sint Judocum op dien tijd,
        En is opwaarts weg geswierdt.
      
      
        Als dit geschieden // zo is dit woord,
        't Welk neder kwam op Hem,
        Van all' die lieden // daar aen-gehoort,
        En deze blijde stem:
        Mijn zael ,, mijn maal ,, heb ik U toegeseydt,
        ,, Mijn hof is U bereid,
        ,, Gij wordt daar in verbeydt,
        ,, Om dat Gij des Konincks hof
        ,, Hebt veracht als slijk en stof,
        Om dat Gij dat hebt ontleydt.
      
      
        Och of ik horen // ten laatsten mocht
        Ook deze stem van God,
        Mijn uytverkooren // gij wordt verzocht
        Tot 's Hemels hoge lot:
        Wel-aen // komt gaan // komt binnen mijn Paleys,
        Gij die na 's redens eysch
        Hebt vroom getemt u vleysch.
        O Iudoce Konincxs Soon!
        Slaat u oogh uit 's hemels Troon,
        Brengt mij daar 't is eeuwig peys.
