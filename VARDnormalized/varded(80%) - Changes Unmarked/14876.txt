
    
      
        MArten maet // ik weetje raad
        Te comen bij een Vryster,
        Com gaat wat snel// je hoortse wel
        Want zij zingt as ien lyster.
      
      
        Ras neemt u Poock // eerst uit de rook
        En wiltse vetjes smeeren,
        Wat zo men vaer // nou binje klaer
        Je moet je lustich weren.
      
      
        Wangt voor de deur // van deze fleur
        Staan Snyers ende Wevers:
        Ja Klaes , en Kloen // en Jan Kapoen,
        Met noch meer and’ren strevers.
      
      
        Asje dan komt // hiel strafjes bromt
        En toontje vrij wat wreetjes
        Zo loopt t’ghespuys // flucx weg na huis,
        Dus wind gij wis de steetjens.
      
      
        De nieuwe Mart // en is niet vart
        Tis dunkt myn daar ontrentjens,
        Dit diertje woont // en haar vertoond
        Daar loopen zo veel ventjens.
      
      
        Hier Barent Brom // en Arent Krom
        Met Koentje, Kloentje, Douwen
        Longs vette Hanghs // en mag’re Franghs,
        Ik kant niet al onthouwen.
      
      
        Nu Marten maet // ik gaa tis laat
        Gij binter bij hielgaaren,
        Nou gaeter an // vrij as ien Man
        Maakt datje haast meucht paaren.
