
    
      
        O Zoete Nymph', die in mijn jong hert'
        Zo veel veghifs ghespouwen kont,
        Die mij staegh martelt met zo veel smert',
        En wederom ghedurigh wondt;
        Daar nooit verdwijnen // de pijnen // die gij
        Eerst in mijne ziele // ded' krielen te ly,
        Maar vast'lijck blijven // en drijven // niet uit,
        Dan elk wel eerder // in meerder // besluyt.
      
      
        Want als de vogh'len s'jaers altijdt bro'en,
        Esen en kippen, de jonghen we'er vlugh
        Ook 't zelve daar naar weerder doen,
        Zo gaeter seld' een wee te rugh';
        Maar, ziet, zij paren // en baren // altijdt,
        En we'er vermeeren // mijn seeren // bij tijd:
        Dus wildy dat 'k gh'nese // u wezen // om-keert,
        En mij van binnen // door 't minnen // ontseert.
      
      
        't Is zeker, wanneer dat gij u vier
        Eens in mijn ziele blasen woudt,
        Het soud' zo menighen, dat nooit dier
        Zo vruchtbaar is geweest in 't wout,
        En liefde de plagen // verjaghen // heel sou,
        En zo mijn wonden // ontgronden // van 't zeer:
        Nu dan Goddinne // wilt minnen // ter trouw,
        En ik zal slaven // en draven// tot eer.
