
    
      
        Liefde zoet, mijn ghemoet // can u niet volprijsen,
        vreugd gaat gij bewijzen // dien die u eeren hier,
        Door u lieflijck bestier,
        Want gij doet dat ik moet // mij nu verjolijsen,
        Doet in mij rasen // als gij zoet van manier,
        T’hart berooft met u vier:
        O God der Goden // die u geboden
        Hielden waren verblijd,
        Maar die u vloden // ginck gij ontbloden,
        Van vreugd in corter tijd,
        Dies ik zal // met gheschal // uwen lof nu zingen,
        Dan voor al // over al // u eer voort te bringhen.
        Oorspronc der dingen // gij cont dus dwinghen.
      
      
        Zonder u // can men nu // gheen vreuchde voort halen,
        Dies sietmen falen // en leven heel verblent,
        Daar gij zijt onbekent,
        Ik verspu // al die schu // van u Godtheydt dwalen,
        want zij zullen halen // niet dan druc en rotment,
        Twist en tweedracht int ent:
        Door u verpenen // doet gij vercleenen
        Does Goddinne eerbaer,
        Dats haar vereenen // en troost verleenen
        Die zulks soecken aan haar,
        Wie u macht // dan veracht // moet onrelijck wezen,
        want u kracht // altijd tracht // om ons te genesen,
        Dus zat door dezen // van mij ghepresen.
      
      
        V ghewelt // gantschlijck stelt // die weerelt in vreden,
        Die aers altijd streden, om t’aertsche goed met vlijt,
        Stilt gij alleen subijt,
        Die gij quelt // werd ghevelt // iae men vindt geen wreden
        Die u couder treden // want gij dic sterkste zijt,
        En dwingtse in dit krijt,
        Gij doet vergaen // en lieflijck paren
        Al t’ghediert op der Aard,
        Door druc weg varen voor al t’beswaren,
        Als zij zo zijn ghepaert,
        wat verheucht, met gheneucht // doet gij ons useeren,
        Alle vreugd // gaat de Ieucht // ook altijdt begeren,
        Om u in eeren // nu te hanteeren.
      
      
        D’arbeyt groot // alle noot // doet gij hier versachten,
        Dies alle gheslachten // nu prijzen met mij zeer,
        Houden u voor een Heer,
        T’lijen snoot // gij verstoot // dat anders met clachten
        Bij ons soud’ vernachten // verjaagd gij van ons veer
        Dies prijs ik u noch weer:
        Alle twist bloedich // doet gij zeer spoedich
        Van ons verstooten vert,
        Eenvredich, goedich, groeyt overvloedich
        Daar gij hier geëerd werd,
        Alle nijt // gij verbijt // doet ons vreugd verwerven,
        Ook den spijt // gij versmijt // en doet onrust sterven
        Door s’werelts erven // u noode derven.
      
      
        Prinse schoon // zeer ydoon // maakt gij myn ghedichten,
        Die ik heb gaan stichten // tot uwer eeren fijn
        Dats’u aenghenaem zyn:
        Gheenen croon // tot myn loon // wensch ic, maar u schichten
        Laat myn druck verlichten // en mij redden uit pijn,
        Laat die de Liefste myn
        Doorschieten crachtich // dat zij gedachtich
        Myner doch wezen mach,
        Zo salt hart clachtich // in vreugd warachtich
        Leven nacht ende dach:
        Door t’verbout // mij ghejont // redt zij mij uit sneven
        Dies myn mond // u terstont // zal heerlijck verheven
        End’ ere geven // voor al die leven.
