
    
      
        GOds wijsheid slechtich // voor s'wee-relts oogh,
        Ghedaelt oprechtich // van bouen hoog,
        Hoog disputeren niet en behoeft:
        Zulk argueren den gheest bedroeft,
        T'recht simpel verstand
        In de waarheid gheplant,
        Wel begrijpen cant,
        T'wordt door den Gheest gheproeft.
      
      
        Al wat niet geestich // en smaakt gesont,
        Dat moet tempeestich // als onghegront
        Op't zand verdwijnen // neuen den steen,
        Donker bruwijnen  // en moghen gheen,
        De waarheid perfeckt
        Houden bedekt,
        Oft schoon vertrekt,
        Zij verwint int ghemeen.
      
      
        S'Heeren ghemeente // met hert en lust,
        In go vereente // staat vast gherust
        Op de rein waarheid // als een pilaer,
        Van suyuer claerheyt // die hier noch daar
        Niet henen en draeyt
        Wat wind dat waeyt,
        In haar ghesaeyt
        Gods woord, versterckts' eenpaer.
      
      
        Broeders eendrachtich // elk in zijn graet,
        Laat ons aendachtich // na Petrus raad,
        Onder den heyden // ghelijck betaamt,
        Go wandel leiden // dat elk hem schaemt
        Tot beterschap
        Die met ghesnap
        Door achterclap
        Ons hebben zal gheblaemt.
      
      
        T'recht vromer leven // Van Abel aan,
        Staat al voorschreuen // tot ons vermaen
        Schoeriem, oft pleytken // van eenich draad,
        T'bleetende geytken // daar ook al staat:
        Niet schuldig en zijt
        Dan liefden vlijt,
        Die t'gheener tijd
        Den Naasten en doet kwaad.
      
      
        In does que tijden // voorhanden bloot,
        Laat ons vermijden // in cleen oft groot,
        Dat met besinnen // zelfs niemand wordt
        Buiten oft binnen // van ons vercort
        Dat zoete lucht
        Van goed gherucht,
        Door goede vrucht
        Van ons zij uytghestort.
      
      
        T'zy vremd' oft vrienden // wie dat ons borght,
        Zo wel wij dienden // te zijn bezorgt:
        Want ziet den name // den welken wij
        Aenroepen t'same // dezen wordt vrij
        Voor borghe betrout,
        En wie hier op bout,
        Hoe zal doch stout
        Dezen te court doen hij.
      
      
        Wie en zal pooghen // om laten niet
        Enen zo hooghen // borgh int verdriet,
        Oft zo in schanden // dat men ghestoort
        Daar door versmaden, zijnen Naam hoort,
        En t'volck vermaert
        Van zijner aard,
        Iae recht als waert
        Bedrieghlijck in zijn woord.
      
      
        Die dus laat steken // zijn borgh in last,
        Godloze treken // doet hij, t'is vast:
        Ook staat gheschreuen // hoe dat in al
        T'slecht ouerbleuen // Israels ghetal,
        Gheen leughenen doch,
        Iae tonghe noch,
        Doende bedroch,
        Gevonden zijn en zal.
      
      
        Dus moet den Tempel // Gods staan gesticht,
        Christi exempel, is claer en licht,
        Te gheenen stonden // was nooit onvroet
        Bedroch gevonden // in zijn mond zoet:
        Wie valsheid bedrijft,
        Oft leughen verstijft,
        Niet bij hem blijft,
        Maar uit zijn huis hij moet.
      
      
        Niemand bedrieghen // gij niet en zult,
        Sprak die van lieghen // nooit hadde schuld:
        Ziet, wilt aenschouwen // een Israelijt
        Oprecht vol trouwen // daar gheenen tijd,
        T'zy meer oft min
        Bedroch en is in,
        Des Heren sin
        Die blijckt hier breedt en wijt.
      
      
        Och gaat niet qualijck // als vriend omgaet,
        Met vriend verralijck // ist ook niet kwaad:
        Dat Broeders nouwe // en zouden zeer
        Moghen betrouwe // den Broeders meer:
        T'welck haestich gheviel,
        Soo't straffen ophiel,
        T'waer tot verniel
        Van s'Heeren volken leer.
      
      
        T'waer een oncieren // der leer Christy,
        Zo qua manieren // te lijden vrij:
        Daar soud' uytgroeyen // een kwaad geclanck
        Elk soud' verfoeyen // zulk volk eer lanck
        Ontstichtich het waar,
        Iae duysterheyt zwaar
        Voor licht, ook maar
        Voor reucke vuylen stank.
      
      
        Als wij wel schaffen, op waerheyts plicht,
        Zulk kwaad te straffen // is meerder licht.
        Na s'vromer wenschen // en s'Heeren Wet,
        Voor God en mensen // t'antwoorden bet,
        Dan datment becroont,
        Oft iet verschoont,
        Want't hem vertoond
        Weerstrijdich Gods opset.
      
      
        De Liefd' om sluyten // nu dit propoost,
        Wat daar is buiten // deucht niet altoost:
        Zij denckt gheen schade // te doen yemandt,
        Zij slaat wel gade // eens anders pand:
        Gheen vreugd zij en lecht
        In eenich onrecht,
        De waarheid slecht
        Verblijdt al haar verstandt.
      
      
        Broeders met enen // wilt noch verstaen,
        Men leest in gheenen // Brief van te saen
        Ghestraft te wezen // wat God mishaeght,
        Maar wel wij lezen // van te vertraagt,
        En hoe dat, dat
        Vercoren vat
        Paulus, vrij wat
        Daar over heeft gheclaeght.
      
      
        Prinsen bevroedich // elk int ghemeen
        Door een ootmoedich // hert, acht elk een
        Hoogher een ander, // dan hij self is,
        Onder malcander // gij zult ghewis
        In vredich ghespan
        Ghegordt zijn dan,
        Met den band van
        Rechter volcomenis.
