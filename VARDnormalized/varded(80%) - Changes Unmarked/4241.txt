
    
      
        CRachtich // nooit
        Geen gesont // hert gewont,
        Als het mijne onverschoont:
        Clachtich // oyt
        Mijnen mond // gans den gront
        Mijnder sielen heeft verthoont,
        Aan een ionghe maagd,
        Die mij iaeght // onversaeght,
        En gans desperaet // laat,
        Tis om niet geclaeght,
        Haar behaagt
        Dezen mijnen droeven staat.
      
      
        Al dat in
        Mijnen geest // onbevreest
        Plach te leven zonder noot:
        Heeft den sin
        Aldermeest // met tempeest
        Vast benout, tot in de dood:
        Eylaes! wie zal mij
        Maken vrij // nu dat zij
        Mij houdt in den last // vast?
        die mij gaan voor bij
        Roupen zij,
        Over mij ellendich gast.
      
      
        T Herte mijn
        Sucht en treurt // angstich scheurt,
        In benoutheyt schier versmacht:
        Moet het zijn
        Dus besuert // mij gebuert
        (Arme weze) meer dan kracht:
        Heb ik metter daad
        Haren raad // oyt versmaet,
        Of geacht te cleen: // neen?
        Maar als delicaet
        
          Toeverlaet
        
        Haar ter noot bekent alleen.
      
      
        Anders geen
        Troost zo zoet // die t'gemoet
        Geeft de ruste principael:
        Dan dat reen
        Edel bloed // die mij doet
        s'Daegs zuchten thien dusentmael,
        Valt zij mij wat zwaar
        Courts hier naar // werd troostbaer
        Al wat zij beleeft // geeft:
        Sulcx betrouw' ik haar
        Die voorwaar
        Een goed hert te my-waert heeft.
      
      
        Rustich zal
        Ik met vlijt // zijn altijd
        Dienstbaar vast in haar verbond,
        T'sy of al
        Mijn profijt // storm, en strijd,
        Wezen zal, ter laester stond:
        Daar mij anders niet
        Dan verdriet // en geschiet,
        Wacht ik goed accoort // voort:
        Zo zij mij aanziet,
        Ende biet
        Een gesontmakende woord.
      
      
        In mijn hert
        Schoonder kint // nooit gesint,
        Niet aanziende druck, of noot,
        Thisbe werd
        Vierich int // hert bemint
        Van Piramus totter dood:
        Maar dit excellent,
        Partinent // bloemken ient,
        Climt in hooger graet // iaet!
        Ben ik een moment
        Haar absent
        Tis een leven desolaet.
      
      
        Naar dien zij
        Even vast // groijt en wast
        In mijn hert zonder af gaan,
        Vind' ik mij
        Aengetast // overlast
        (Boven reden) aangedaan:
        Ik moet even wel
        Mijn gequel // achten spel,
        Wat baet veel geclach // ach!
        Al valt zij rebel,
        Niemand el
        Mijn versuchten einden mach.
