
    
      
        WT louter kracht der minnen,
        Claegh ik al mijn verdriet:
        Een maagd doet mij ontsinnen,
        Die mijnen druck aanziet // niet,
        Hoe wel mij hert-sweer // meer
        Dan te veel geschiet.
      
      
        Ik dan aldus ellendich
        Gestoten van der hand,
        Is mij arm knecht nootwendich
        Een troostbaer onderstant // want,
        Niet en coemt hier van // dan
        Steruen in den brand.
      
      
        Laat zij mij dus (met pijnen)
        Zuchten in tegenspoet:
        Mach mij eens troost om-schijnen,
        Laeth' costen vlees, en bloed // t'moet
        Eerst zijn ongeval // zal
        T'eynde werden zoet.
      
      
        Leven, en dood, achermen,
        Is al aan haar besteet:
        T ooght zij mij geen ontfermen,
        (Naerdien zij mijn secreet // weet)
        Ik blijf een arm schu // nu
        Zij mij dus vergeet.
      
      
        Een rouwich wederstrijden,
        Beclemt vast mijnen geest:
        Al hoop ik naar verblijden,
        T'wert al stormich tempeest // meest,
        Droufheyt, en geween // geen
        Herte zo bevreesd.
      
      
        Moet ik de over-schoone
        Princess, eigen zijn,
        Niet el wensch ik te loone,
        Dan t'vriendelick aenschijn // fijn:
        T is den weerdtsten schat // dat
        Beeldeken divijn.
