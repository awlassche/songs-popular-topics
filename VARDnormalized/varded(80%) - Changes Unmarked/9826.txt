
    
      
        SChoon duyvinne // die rein van sinne
        Mij meer bemint dan uws zelfs leven
        Want gij inne // 't vier mijner minne
        Verslonden zijt, van hier ghedreven,
        Door mijnen gheest // uit 'tAerdtsch foreest,
        Tot bij mij int Hemelse wezen,
        In mijn slaapkamer uytghelesen,
        Rust u hier dan // tot dat voorbij
        De hitte van // den middag zij.
      
      
        Zonder scheiden // wilt u vermeyden
        Met mij, en vast verenigd blyven,
        Gheen verleiden // hoe zij arbeiden,
        V deeren mach, gij zult beklijven,
        En vreest niet want // elken vyandt
        Zal ik met krachte neder vellen,
        En legghen tot ons voerschabellen,
        Dan werd altoost // ghewisselt dijn
        Droefheyt in troost, haar vreugd in pijn.
      
      
        Openbaren // met mijn Heirscharen
        Zal ik eer lanck, tot u verblyden,
        Den zondaren // tot een vervaren,
        Die mijn aenschijn zullen vermijden,
        Vluchten beducht // zonder ontvlucht
        Werden benout van herten sprekende',
        Och ziet, hoe lief en weerdt gherekent
        Is zij, die wij // hielden voor dwaes,
        Zij is nu bly // wij droef eylaes.
      
      
        O Princiere // blijft goedertiere
        Lijdtsamigh tot behoudt der zielen,
        Met den viere // uwe bloedtgiere
        Vervolghers rijk zal ik vernielen,
        Den nieuwen Wijn // int rijke mijn
        Zal u herte vreuchdigh ontwaken,
        En uwen mond vol lachens maken,
        Bij mij u steed' // eeuwig zijn zal,
        Troost u hier meed, int aertsche dal.
