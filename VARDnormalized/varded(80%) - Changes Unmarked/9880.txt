
    
      
        WIe zal ons gheraden // vroet,
        Welken voghel snel
        Does Landen veel weldaden // doet?
        Hij en heeft een veder // niet,
        Doch al heen en weder // ziet
        Men hem vlieghen wel.
      
      
        En is dit gheen wonder // werk?
        Mochtmen zeggen dan,
        Dat zonder pluym, en zonder // vlerck,
        Enig voghel wezen // mach,
        Ter wereld op dezen // dach,
        Die ghevlieghen kan.
      
      
        Hij heeft vlees noch benen // weert,
        Is te raden dit,
        En dan en heeft hij gheenen // steert,
        Maar hij heeft twee bekken // stijf,
        Lanck is hij van neck en // lijf,
        'tInghewant is wit.
      
      
        Somtijts graeu, oft ander // verf,
        Naar het wijlen past,
        Men kander huis, men kander // erf,
        Deucht en ere winnen // met,
        Die daar aan zijn sinnen // set,
        Hem te jaghen vast.
      
      
        Door een smal bosschage // dicht,
        Die naar hem toenijpt,
        Daar neemt hij zijn passage // licht,
        Aan elk endt om vanghen // gaan,
        Vijf altijdts haar ganghen // saen,
        En elk naar hem grijpt.
      
      
        'tSchijnt een kromme drooghe // schuit,
        Men wind van den buik
        Den darrem hem door d'ooghe // uit,
        En dat van den darrem // coemt,
        Die men rijk oft arrem // noemt.
        Dienet tot ghebruyck.
      
      
        'tIs weerdig verkoren // wis,
        Tot des mensen nood,
        Van als men kind geboren // is,
        En moghen wy't derven // niet,
        Iae tot dat wij sterven // ziet,
        En noch na den dood.
      
      
        'tLeert ons ook den rechten // pad,
        Tot dat eeuwig Land:
        Dus en ist gheenen slechten // schat,
        Die hier is te krijghen // bij,
        Van dit Raedtsel swijghen // wij
        Niet langher 't verstandt.
      
      
        'tVlasbloemken ter eeren // zoet,
        De Schietspoele rein,
        Die meen ik, does ons leren // moet,
        Hoe voorbij ons dagen // smal
        Gaan, zonder vertraghen // al,
        Haastig hier int pleyn.
      
      
        Den lijdsamen vromen // Iob
        Van haar snelheydt leert,
        Hiskia zonder schromen // op
        Den Wever ghedachtigh // werd,
        Doe hij leed onsachtigh // smert,
        Na de wandt ghekeert.
      
      
        Ons dagen vervlieghen // doch
        Als de Spoele hier,
        Maar vele haar bedrieghen // noch,
        Met te meenen leven // lanck,
        Die haast doodigh sneven // krank,
        Int sondigh bestier.
      
      
        Tot eeuwighen moyen // loon,
        Doe hier elk een vlijt,
        Den enen zal int lopen // Croon,
        De ander 'tSweerdt ghenaecken // maar,
        Hierom elk met waecken // waar
        Neme zijnen tijd.
      
      
        Pallas-gheesten konstigh // laat
        V niet vallen zwaer,
        Als yemandt u afjonstigh // smaedt,
        Als d'Eyptsche knapen // koen,
        Die voor t'vromer schapen // hoen
        Grouwelden eenpaer.
      
      
        Oorlof, u te samen // draagt,
        Vroom en wel ghemoedt,
        Den Volghel zonder schamen // jaagt
        Door het woudt, met lust en // hand,
        Laat hem weinig rusten // want
        Lust den arbeid voedt.
