
    
      
        Aenhoort wat Marcus schrijft // van Iesus wandelingh
        Na ‘t Doopsel hij beklijft, in Sathans bekooringh,
        Symon zijn Net afging // onreine gheest moet vrezen,
        Die koorts’ oorlof ontving Melaetsen wordt ghenesen.
      
      
        Boven door ‘t dak een Gicht // afdaelt en werd gesont,
        Levi zijn voeten licht // medecijn krancken vond:
        Vasten is gheen verbond // Sabboth is heel ghebleven
        Als ‘t Kooren met den mond // door honger werd gewreven.
      
      
        Cleyn tijd hier na geneest // hij een verdorde hand,
        Raad is daarom geweest // doch hij wijkt aan een kant:
        Verkiest tvvaelef Ghesan’ // en des Belzebubs treken
        Duyden zij hem tot schant // zijn Moeder wil hem spreken.
      
      
        Door zaads ghelijckenis // hij ‘t rijke Gods voorstelt,
        Driederley hinder is // van Mostert hij vertelt,
        ‘t Licht nummermeer bewelt // wie heeft, die wordt gegeven
        Tempeest het schipkan quelt // het welk hij heeft verdreven.
      
      
        En komend’ aan het Land // verdrijft hij ‘t Legioen,
        Die haast hebben vermant // de verckens in het groen:
        Den bloedt-ganck krijgt versoen // Dochter van tvvalef jaren
        Des Oversten als doen // brocht hij uit ‘s doots bezwaren.
      
      
        Flauw was zijn kracht bekent // bij zijn Lants-volck verdooft
        Apostels hij uit zendt // Iohannes wordt onthooft:
        Den hongher wordt berooft // van vijf duizend. Hij vvandelt
        Op zee: en wie geloofd // in zieckt zijn Zoom behandelt.
      
      
        Ghevvasch van hand en mond // van binnen niet en baet,
        En door der mensen vond // men Gods gebod verlaat:
        Hondeken wordt verzaedt // van d’afghevallen kruymen:
        Den doven hij ontslaet // van zijn ghehoor ‘t versuymen.
      
      
        Hier na vier duizend Mann’ // verzadinghe gheniet:
        Die teiken soecken an // van haren deessem vlied;
        De blinde bomen ziet // Petrus komt hem belyden,
        Die hem hier schamen niet // zal hij naemaels bevryden.
      
      
        In klaerheydt hij vertoond // Elias komen zal:
        Geseten hij verschoont // die schuyn’ en wenteld’ al,
        Zij ‘t Kindeken houdt stal // en leert ootmoedigheden,
        Niemand brenget ten val // door ergherlijcke leden.
      
      
        Kracht heeft nu geen Scheyd-brief // Kinders hij benedijt:
        Iongh’lingh heeft rijkdom lief // loon vint men in does tijd:
        ‘t Hoog sitten maakt veel spijt // dus zijnt Heydensche werk
        Bartimeus met vlijt // het dachlicht kan bemercken.
      
      
        Looft Syons Koning dan // die zit op d’Edelin,
        Den Vygheboom en kan // gheen vruchten brengen in,
        Iesus en heeft gheen sin // aan Wisselaers in Tempel,
        Zij eisen machts begin // hij wijst des Doops exempel.
      
      
        Met de Wijngaerdeniers // haar ontrou hij bewijst,
        De Phariseen vol viers // vraeg’ waarom men Exyst,
        Hoe naemaels men verrijst // ook na de hoogste Leer,
        d’Arme Wedyvve hij prijst // Davids Soon is zijn Here.
      
      
        Neder op d’olijf berg // zittende hij voorzeid,
        Van zeer veel secten ergh // die menich Mensch verleydt,
        Ook Oorlogh uytghebreyd // eer synen dach zal naecken,
        Dus met voorsichtigheyd // beveelt hij haar te vraecken,
      
      
        Oop hem men Zalve giet // Brood brekingh en Paaslam,
        Hij met de zijn gheniet // Droefheyt hem over kwam,
        Gevangen werd hij tam // voor Cayphas lijdt hij schennis,
        Die hem te quellen nam // Petrus ontkent zijn kennis.
      
      
        Pilatus saed den haat // doen men hem ‘t kruyce boot.
        Hij woud’ hem doen genaed’ // maar de ander hoop was groot
        Tussen tvvee Moorders snoot // is hij ghekruyst ghestorven,
        Hij heeft ons dood, ghedoot // en ons verderf verdorven.
      
      
        Qualijck den dach aenbrack // de Vrouwen zijn verschrikt,
        Den Engel met haar sprak // de klaarheid Iesu blickt,
        Doch werden zij verkwikt // als hij met haar kwam spreken,
        Als hij ‘t al hadd’ beschikt // is ten Henel gheweken.
