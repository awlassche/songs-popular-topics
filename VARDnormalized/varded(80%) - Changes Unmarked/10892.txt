
    
      
        Bloeyende jeucht, Edelen aard der vrouwen
        Wanneer gij meucht // zo denckt, toch eens om mij
        Maect mij verheucht, deur u goedheid vol trouwen
        Door sulcken deucht // zal ik dan wezen bly
        Een zoete melody // ontfang ic dan eenpaer,
        Den troost eender jonc-vrouwen zoet
        Mij langhen tijd onthouwen // doet
        Deur hope ickt vertrouwen // moet
        Met dezen Nieuwen Iaer.
      
      
        Menige reys // Lief komt gij mij te voren
        Met zwaar ghepeys // ben ic dicwils gekweld
        Der liefden eyes // doet mij dit al oorboren,
        Waar door mijn vleys // menichmael is ontstelt
        Wat hier wordt vertelt // ghebeur alzo voorwaar
        Daar jonkvrouwen // met eeren, zijn
        Machmen trouwen // verkeren // zijn
        Wilt onthouwen // dit leren // mijn
        Met dezen nieuwen Iaer
      
      
        Mocht ik verstaen // schoon lief wt uwen monde
        Sulc een vermaen // daar ic na heb gewacht
        Dan waar gedaan // in alzo corten stoned,
        mijn droefheyt saen // die ic heb dach en nacht,
        V liefde heeft gewracht, in mij een lijden zwaar
        schoon Bloemen mijns gedachtich // zijt
        Hoort hoe v dienaar clachtich // lijt
        Thoont hem vriendschap warachtich // wijt
        Met deze nieuwen Iaer
      
      
        Maechdelic zaad // wel verciert met veel deuchden
        Komt mij te baet // en geeft mij doch confoort,
        denct op den praet // die wi tsamen in vreuchden
        Vroeg ende laat // hadden zonder discoort,
        Laat ons ongestoort // worden een vrientlick paar
        Zo u persoon // mij de hand // biet,
        En alst met schoon // verstand // gheschiet,
        Alsulcken loon // verplant // ik niet,
        Met dezen nieuwen Iaer.
      
      
        Neemt toch in dance // den sanc // van mijn ghedichte
        Schier met bedwanc // zo schanc // ic u dit liet:
        deur liefde franc // den ganc // viel mij nz lichte
        Met sinnen kranc // d’aenvanc // is zo geschiet
        Laat gij mij tot verdriet // Edel Princes eerbaer
        Zo lijd’ ik dan van rouwe // pijn
        Wilt gij mij niet ghetrouwe // zijn
        Ghedenckt mij schoon jonckvrouwe // fijn
        Met dezen nieuwen Iaer.
