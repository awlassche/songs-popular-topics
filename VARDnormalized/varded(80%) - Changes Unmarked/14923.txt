
    
      
        O Gij heyloose schyn // vol vuyle list kwaad
        Hoe pynicht gij met pyn // mijn zo ghetrouwe daad,
        Ik die mijn Lief wel eer gheclaecht heb mijn ellent
        Werd nu veracht versmaet als ik haar com omtrent.
      
      
        De schyn myn laes! bedrooch // ghelijck ik nu bevin
        Mijn hert zij mij ontoogh // door heur beveynsde min,
        Zij spot nu met myn clacht, ja geckt met myn verdriet
        En in de plaets van mij een ander jonst zij biet.
      
      
        Maar ik, ik ducht het kwaad // dat zij mij nu bewyst
        Ghehoude zij te raad // om werde mee ghespyst,
        Wanneer zij braecke zal, en walghen zal van die
        Waaromme dat ik heur, laes! myn verlaete zie.
      
      
        De heel Wereld creet // volmondich overluyt
        Zo yder noch wel weet // t’Meysje was de Bruyt,
        De een had het Speck al weg, en d’aer die wist het wel
        Nu zal een ander heur, en ik, ik had de quel.
      
      
        Ik woude dat ik con // does hupsche fijne quant
        Die haar zo haast verwon // met glory triumphant,
        Ik vlechten hem van Miert, een braeve Crancelijn
        En schonck hem uit een borst, de frisse coele Wijn.
      
      
        Prinse daar synder meer // als ik door heur ghesicht
        Ghedwaelt verleid wel eer // fy toverende licht,
        Dat nooit ik speuren con, dyn schalcke listicheyt
        Dat zal mij moeyen nu, en inder eeuwicheyt.
