
    
      
        SILVIA myn schone en trotse Goddin,
        Alleen den roem // en den bloem // van alle schone,
        Zegt hoe lang zult gij noch in uwen zin
        Mij u slaef, och! och! met veel ellents gaan loonen?
        Zal ten geener tijden dan
        Op houden does plaaghen,
        Die ‘k niet langer lijden en kan
        Noch int hart verdraaghen:
        V gebaer // geeft mij vaer // en zal mij doen sterven
        Wat suld gij // dan t’onty // meer van mij verwerven.
      
      
        Merct ô myn schoon // opt vertoon // van myn min
        En t’eeuwich besluyt van myn getrouwicheden,
        Die te allen tijden achter haar Goddin
        V myn lusten haar Goddelijck aengebeeden,
        Zult gij die verlaten gaan
        Ook myn u arm slaave:
        Dat en kan ik niet verstaen
        Want u goede gave
        Neyghen naa // het ghenaa // der barmherticheden,
        En u oogh // tot medoogh // van liefden vreeden.
      
      
        Al wat de Son // gestralen kon // hier om laagh,
        Is maar schijn // zo dunkt myn, van u Hemels wezen?
        Ist dan gheen reen dat ik eerbiedich draagh
        V naams lof boven alle Sterren ghepresen,
        Schoon Goddin, myn troost, myn hert
        Al laat gij mij in lijden,
        K’hoop niet dat gij in myn smert
        V noch zult verblijden,
        Maar de vreuchds // syn myn jeuchds // die mij hebt gegeven
        In myn smart // t’meeste hart // dat is u leven.
