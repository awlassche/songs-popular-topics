
    
      
        Bloeyende jeucht / eedelen aard der vrouwen
        Wanneer gij meucht / zo denckt toch eens op mij
        Maakt mij verheucht / door v goedheid vol trouwen
        Door sulcken deucht / zal ik dan wezen bly
        Een zoete melody / ontfangh ik dan eenpaer
        Den troost eender jonkvrouwen / zoet
        Mij langhen tyt onthouwen / doet
        Door hope ickt vertrouwen / moet
        Met dezen nieuwe jaar
      
      
        Menighen reys / lief compt gij mij te voren
        Met zwaar ghepeys / ben ik dickwyls ghequelt
        Der liefden eysch / doet mij dit al oorboren
        Waar door myn vleysch / menichmael is ontstelt
        Wat hier werd vertelt / gebeurt alzo voorwaar
        Daar jonkvrouwen / met eeren / zyn
        Machmen trouwen / verkeren / fyn
        Wilt onthouwen / dit leren / myn
        Met dezen nieuwe jaar
      
      
        Mocht ik verstaen / schoon lief wt vwen monde
        Zulk een vermaen / daar ik na heb ghewacht
        Zo waar gedaan / in alzo corten stoned
        Myn droefheyt saen / die ik heb dach en nacht
        V liefde heeft gewracht / in mij een lyden zwaar
        Schoon blomken myns gedachtich / zyt
        Hoort hoe v dienaar clachtich / lyt
        Toont hem vriendschap waerachtich / wit
        Met dezen nieuwe jaar
      
      
        Al myn gemoet / heeft na v een verlangen
        Door liefde zoet / die mij tot v verwekt
        Myn vlees en bloed / is met liefde beuanghen
        Hierom lief doet / eenich teken perfeckt
        Het welke strekt / tot rein liefde eenpaer
        Doet als een go vriendinne / plach
        Op dat ickt zo verwinne / mach
        Twelck mij opt hert van binne / lach
        Met dezen nieuwe jaar
      
      
        Neempt toch in danck / den sank / van myn gezichte
        Schier met bedwanck / zo schanck / ik v dit liet
        Deur liefde stranck / den ganck / viel mij niet lichte
        Met sinnen cranck / d'aenvanck / is zo geschiet
        Laat gij mij int verdriet / eedel princes eerbaer
        Zo lydt ik dan van rouwe / pyn
        Wilt gij mij niet getrouwe / zyn
        Gedenkt mij schoon jonckvrouwe / fyn
        Met dezen nieuwe jaar
