
    
      
        VVAck’re Herders vrolijc // luchtich aender heyde
        Laat den grijsen molijck // ’t Vee allenich weide,
        Want tis tijd // dat ghyder syt
        Daar men S wijngaards loote snyt.
      
      
        Ziet hoe dicht de heuvelen // vol syn van Naejaden
        Laat u Nymph niet sneuvelen // t’mocht haar erghens schaden,
        Want het stof // te zwaar en grof
        Plucken zij t’ghereetste lof.
      
      
        Tyter laat u Schaepjes  // onder d’holle eycken
        Voecht u bij de knaepies // die vast opwaarts reiken,
        Na de kuyn // de pople bruin
        Ghelijck d’Hard’rinnetjes int Duyn.
      
      
        Vant gewas der velden // gaan ons schueren zwanger
        Dus dat wij uit stelden // onze Jaer-feest langher
        Waar een schant // komt hand aan hand
        Moet den Oost-May zyn gheplant.
      
      
        Coridon is wakker // ziet hem in de kuypen
        Parsschen op de akker // dat de beesje druypen,
        Van het nat // t’gheen Jaerlijckx wat
        Bachus ons geschonken had.
