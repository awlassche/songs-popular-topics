
    
      
        HOe treurt mijn zieltje vol smert,
        Om de schone Rosemondt
        Die mijn verlieven- de hart,
        Heeft geprickelt en gewont,
        Door haar glans // die bykans,
        Aan den dans // lock alle de Jonghmans.
      
      
        Hoe was mijn hartje verheugt,
        Doen ik eerstmael haar aensach:
        Maar ach! die schielijcke vreugd
        Is verandert ingheklach,
        Nu dat zij // geen gevry // wil van mij,
        Maar schimp en smaedt ik ly.
      
      
        Doen zij in't Bos, aende kant
        Van de klare Water-beeck
        Haar Wooningh hadde gheplant,
        Satse meninghmael en keeck
        In de vloet // wel gemoet // vlocht een hoed
        Van Kruid, en Blomtjes zoet.
      
      
        Daar heb ik 't aerdige beeld
        Menigmaal ghesproken aan,
        En op mijn Pijpje ghespeelt,
        Als ik kwam langs 't Water gaan,
        Haar ter eer // maar veel meer // neemt zij weer
        Van mij nu een afkeer.
      
      
        Ny zij haar Schaepjes en vee,
        En het Boersche leven laat:
        En neemt haar Wooningh in stee,
        Schijntet datse mij ook haat.
        Steetse pracht // zij meer acht // dan mijn klacht
        In droefheydt ik versmacht.
      
      
        Het krenckt mijn jeughdighe bloed,
        Als ik denck dat zij met vreugd
        Aan Thyrsis offeren moet,
        't Aldereelste van haar jeugd:
        Hayrtjes blondt // Ooghjes, Mond // Borsjes rondt,
        En, laes! ik blijf ghewondt.
      
      
        O Rosemondt: ach! vriendin
        Waerder noch wat hoop voor mijn,
        Ik knield' en bad om u Min,
        Tot verlichting van mijn pijn;
        Ik leyd' af // en vergaf // d'Harders staf,
        En koos de stad, voort Graf.
