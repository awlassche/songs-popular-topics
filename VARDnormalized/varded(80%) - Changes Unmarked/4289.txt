
    
      
        E En excellent // en jent //
        Bloemken playsant //
        Heeft mij eylaes gewont
        Schoonder en kent // noch sent //
        Venus niemand //
        Noch is op swe- relts gront,
        Al doet zij mij veel lijdens on- gesont //
        Noch blijft in mij haar minne //
        Want zij van den beginne //
        Oyt int herte stond.
      
      
        T'comt nu zo bij // dat zij
        Luttel verschoont,
        Mijn doloreus vermaen:
        Maar als zij mij // een bly
        Gezichte thoont,
        Is al mijn druck gedaan.
        Daar heb ik zulk een vast betrouwen aan,
        Al moet ickt wat besueren,
        Zij zal (tot allen uren)
        Vast int herte staan.
      
      
        Certain, zo waar // als daar
        Den hemel schijnt,
        Zij heeft mij gans deurtast:
        Dies t'herte zwaar // in haar
        Liefde verdwijnt,
        Dits cause vanden last,
        Den brand (die in mij noch veel meerder wast)
        Zal mij (ducht ik) verteren,
        Nochtans blijft zij met eeren,
        In mijn herte vast.
      
      
        Deur niemandts daad // noch raad,
        Does liefde fier,
        Nimmermeer te onder drijft,
        Hoe wel den haat // die kwaad
        Is int bestier,
        Veel schanden van mij schrijft,
        Nochtans werd' ik duer haar weder gestijft,
        Zij is de weerdste vrouwe,
        Deur haar liefde getrouwe,
        Zij int herte blijft.
      
      
        Princess vroet // zijn moet
        (En niemand el)
        Mijn liefken elegant:
        Die mij den gloet // onsoet
        Der minnen fel
        Vierich int herte sant.
        Mach haar (dit mijn gedich) comen ter hand,
        Zo ben ik vrij van smerte,
        En blijve in haar herte
        Eewich vast geplant.
