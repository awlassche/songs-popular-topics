
    
      
        Wat last is liefd' // ja recht een sware last
        Voor myn ‘tondiefd’ // dit spreeckwoort gaat vast,
        Dat voel ik nu wel in // myn sin
        Want ik moetse nu draghen:
        Dus durf ik het claghen // vrij wagen
        Van does vervloekte min.
      
      
        O Raserny // 't onrecht men u liefd' noemt,
        U dwaeserny // met schoonen schijn verbloemt
        Nu heel het jonghe hert // dat werd
        Door uwe glans bedroghen
        En door dijn vermoghen // ghetoghen
        In does siel-sieckt zo vert.
      
      
        Dat schijn bedriegt // dat werd ic nu wel waar,
        U faem die vliecht // van u deucht hier en daar
        Ik gh'loofden u ghelaet // en praet;
        Maar foey myn licht vertrouwen
        Nu mach ik 't onthouwen // dat vrouwen
        Zijn te licht van beraet.
      
      
        't Scheen of u hert // aen't mijne waar gehecht,
        Maar wat een pert // hebt gij nu aengherecht,
        Gij verlaat mijn arm bloed // om 't goed
        Moet ik myn Lief verliesen:
        Dus moet ik van biezen // nu kiesen
        Een dor ghevlochten hoed.
      
      
        U Mins ghedaant // heb ik te haast gelooft
        Vergheefs gewaant // en voor niet al geslooft
        Geen groter ongheluck // of stuk
        En mach myn overkomen:
        Want 't is mij door schromen // ontnomen
        En ik blijf in de druck.
      
      
        Maar noch Goddin // kan ik u haten // niet
        Door ware Min // al ben ik verlaten / ziet;
        Maar weest hier op niet trots // noch spots
        Ik denck het Kan verkeren,
        Dat u den tijd zal leren // begheeren
        Die gij nu weyghert schots.
      
      
        Princes ik roem // van u grote kracht,
        Een Jongebloem // hebt gij te ondergebracht,
        Myn voetsel is vercort // verdort
        Alleenlijck door u te derven,
        Ik wens menich werven // om 't sterven,
        Of dat het beter wordt.
