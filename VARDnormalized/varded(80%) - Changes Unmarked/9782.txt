
    
      
        O Bruydegom, mijn Vorst, mijn Capiteyn,
        Waar ik mij kere, hier in 'swereldts pleyn,
        Trouwe noch troost, en vind' ik int ghemeyn,
        Dan in uwen heylighen Naam alleyn,
        V rein // troostbaer // woord klaer
        Dat sitcht // verlicht // en doet
        Verblyden mijn ghemoedt.
      
      
        Te recht, o Heer, hebt gij mij wel gezeid,
        Ziet toe, op dat u niemand en verleydt,
        Want die slange met haar dienaars planteyt
        Ghebruyckt nu alzo grote listicheydt,
        Maar scheid // noch vliet // doch niet
        Van mij // o gij // of ik
        Moet vallen in haar strik.
      
      
        Gij toondet mij enen weg metter hand
        Om wandelen na het beloofde Land,
        Die wordt veracht, door menschelijc verstand,
        Enen besijden dezen aan den kant,
        Playsant // oprecht // men zegt
        Hem fijn // te zijn // en dat
        Hij ook loopt na u Stad.
      
      
        Met zulke stormwinden schalc verdraeyt
        Van alle zyden word' ik nu bewaeyt,
        O Heer, in vreden houdt mij wel ghepaeyt,
        En u vrucht der gherechticheydt mij zaeyt,
        Verfraeyt // mij saen // om staan,
        Standtvast // wat last // oft cruys
        Daar komt tegen u huis.
      
      
        Prinse, 'tbeproeven valt menighertier,
        Maar gij castijt, al die gij lief hebt hier,
        Want t'moet doch al gesouten zijn met vier,
        Maar verlost mij van al dit aertsch dangier,
        Dat schier // ik magh // den dag
        Ontvlien // en zien // altijdt
        V aenschijn, met jolijt.
