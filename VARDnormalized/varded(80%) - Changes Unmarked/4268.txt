
    
      
        PRompt, en volmaect //
        Is mijn wtvercoren //
        Zo wel geraect //
        En was nooit Held geboren //
        Alvoren // can
        Hij weg, en regel // schrijven //
        Op t'herte zal dan //
        Does jongen man //
        Voortan // als segel // blijven.
      
      
        In een loyal
        Hert, heb ik besloten:
        Nu dat den strael
        Der minnen is geschoten,
        Genoten // zijn
        Moet met hem vreugd' // en // wenen,
        Ken achte gepijn,
        Ik zal t'sanguijn
        Aenschijn // in deugden // nemen.
      
      
        Eenpaerlick staat
        T'concept mijner sinnen,
        Dat ik (naar raad)
        Hem niet en zal beginnen
        Te minnen // small,
        Maar als mijn here // croonen:
        In spijte t'getal
        Der tongen al
        Ik zal // hem ere // thoonen.
      
      
        Troostbaerder // geen
        Inder Goden troone,
        Al waerder // een
        Adonis in persoone,
        Zijn schone // glans
        Moet moet noch menich // buigen:
        Want, hij draecht den crans
        Der ionger mans,
        Dits gans // mijn eenich // tuigen.
      
      
        Een Paragoen
        Is hij, vroom gesegent:
        Van dien Baroen,
        Mij alle troost beiegent,
        Genegent- // heyt
        Theimwaerts, can geuen // smerte,
        Maar laat met bescheyt,
        V zijn gezeid,
        Hij leit, mij neuen // t'herte.
      
      
        Rustich, en fray,
        Moet mijn Prinse varen:
        Den Rijnschen bay
        Wensch ik hem zonder sparen:
        Geen zwaren // moet
        Wil ik ten fijne // maken:
        Dit gracelick bloed,
        Mij altijd doet,
        Meer zoet // dan pijne // smaken.
