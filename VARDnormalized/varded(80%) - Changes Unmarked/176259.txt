
    
      
        SChoon Nymphelijn // ach mindje mijn // wat soud ik al versieren,
        Om na mijn wens // does ledetjens // zo wel gemaakt, te cieren?
        Met blinckend goud // of perlen sout // gij voelen ras belasten
        Vw halfgen zoet // dat crael als bloed // daarom niet beter pasten.
      
      
        Ik soud u cleen // met keursgens reen // van lichte verwen blijtjens.
        Die zouden staan // gheschildert aan // u breedachtighe zijdtjens.
        Vw voetjens mit // haar schoentjens wit // daar bij gheval, in't bokken,
        Het incarnaet // zo wel bij staat // van hoosgens net ghetrocken.
      
      
        Vw armkens mee // zo wit als snee // soud ik corael om schikken.
        Does vlechgens blondt // op nieuwe vond // soud ik u leren strikken,
        Met snoertgens veel // nu groen en geel // bij lodderlijcke beurtgens;
        Voor wat jent haar // of cruyvendt daar // en duysendt zoete leurtgens.
      
      
        'Tperruyckgen sou // ik trekken nou // wat laechjens, dan wat hoochjens,
        En als het claer // ghefutselt waar // mij spieglen in u oochjens.
        Dan werpen licht // nu mijn ghesicht // op't ene, nu op't ander:
        Dan nemen raem // hoe't altesaem // sou voeghen bij elckander.
      
      
        Indien dat gij // uw oochjens bly // en lieffelijcke zeeden,
        Zo vreuchderijck // zo vriendelijck // zo vol bevallijckheden,
        En u aenschijn // vernoecht, tot mijn // dan, met een lachen wenden:
        Zo soud ik, hiel // in u mijn siel // gaan, metter wone, zenden.
      
      
        En als ik wat // beloncket had // den brand van alle knechgens,
        Vw zedetjens // uw ledetjens // uw fray ghetoyde vlechgens;
        Zo soud ik streng // met armen eng // uw jente lijfjen pranghen,
        Tot ik, daar uit // creech, buit om buit // u sieltjen weer gevangen.
      
      
        De verve van // mijn lippen, an // uw wangsjens, soud' ik plecken,
        Door soentgens sacht // en met haar kracht // u siel te mond uit trekken.
        Ik wed, ik weet // gij dan beleedt // dat niemand van u susjens,
        Daar gij, soo't schijnt // nu smaak in vijndt // gaf oyt zo zoete kusjens.
