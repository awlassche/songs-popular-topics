
    
      
        ROemt u des Heren
        Gij die de waarheid // kent,
        God in verseeren
        Looft, ghelijck in voorspoet,
        Van u bekeeren
        Bewijst danckbaerheyt // jent,
        Die u laat leren,
        En kennen overvloed
        Synen wille zeer goed:
        Doch uit rechter ootmoet
        Wilt merken, vrienden // gij,
        Wat een ghenade zoet
        Dat ons de Here doet
        Die met ons werken nooit en verdienden // wij.
      
      
        Zo wij wat weten
        Van Gods volkomen // heesch,
        Die schoon secreten,
        Wt ghenaden gheschiet,
        Wat wil vermeten
        Oft hem beroemen // vlees?
        T'welck na 'tPropheten
        Woord, is gras ende riet
        Indien wij dencken ziet,
        Meer te wezen als niet,
        Ny worden bedroghen // saen,
        Zo wij om ons verdriet,
        Op onze wijsheydt iet,
        Verdiensten, werken ofte vermogen // staan.
      
      
        Laat ons God prijzen,
        Ook yeghelijcken dan
        Sachtmoedt bewijzen,
        En lasteren niemand:
        Op dat 't verrijsen
        Des gheests recht blijcken // kan,
        Met laven, spijzen,
        En weldoen den vyandt,
        Door een liefde die brandt
        Christum na volghen, want
        Wij waren t'alder // tijd
        Ook verblendt int verstandt,
        Vol van boosheydt gheplant,
        Draghend' int herte op malkander // nijd.
      
      
        Maar noch behoeven
        Wij Gods ghenade wel,
        Als wij beproeven
        Ons ghewercken all'gaer,
        Dus zonder toeven,
        Laat ons ghestade snel,
        Vrij van bedroeven,
        Neersticheydt doen eenpaer,
        In als te nemen waar
        Ons beroepinghe klaer,
        Tot voldoeninghe // fijn,
        Zo is dan Christus daar,
        Voor die hem volghen naar,
        Wilt hij zeer milt, tot een verzoeninge // zijn.
