
    
      
        OP 't rantje // aan 't kantje van een Fonteyn,
        Met vochtighe // lochtighe Mos bedekt,
        Daar elk Adertje in het pleyn,
        Kwam krommelingh, dwalend' afgheleckt,
        Een Harder zat // en in het nat
        Hem spiegh'lende, zong op zijn Fluyt,
        Laat een Dag af // o schone! eer straf
        De wreede Dood mijn ogen sluit.
      
      
        Kan 't wezen // dat dezen groten pijn
        Die ik u om u Minne ly,
        Indien de Goon rechtvaerdich zijn
        Niet komen tot een zoet ghevry?
        Kan hartens gunst // en Minne-kunst,
        Ja Liefd' die 't al te boven gaat,
        U heus ghemoedt // niet tot het zoet
        Medooghen dwinghen toeverlaet?
      
      
        Die pronckend' // en lonckende ooghjes zoet
        Lief-glurich // doch durich vol veynsery,
        Dwongen mijn dikwijls tot goeden moed,
        Willen zij 't nu versweeren aan mij?
        Haar hart in 't endt // zijn strafheydt went
        Hebben ze mij zo vaeck gezeid:
        Sou zij does spraeck // een valse spaeck
        Wel lijden van haar schoonicheydt?
      
      
        Hoe! zouden die ogen // ghedooghen dat zij
        Verradertjes waren? lockend de ziel,
        Mindeling door haar bedrieghery?
        Bos-Nimphjes, indien het zo gheviel,
        Al draagt gij dan // 't blancketsel van
        Gheveynstheydt op u voorhoofd niet,
        't Schuylt in u hart // daar gheleert werd:
        Paeyt haar met woorden, en anders niet.
      
      
        In 't ende // wilt wenden u strafheydt / o schone
        Hard'rinne // Vriendinne 't is meer als tijd,
        En geloofd zo de schoonheydt niet meer wone
        Houdt jeuchdich vermaeck, en soeticheydt lijdt.
        Is ghelijck 't ghesicht dat (laes) zijn licht
        Moet missen, ja de jeugd hoe fris
        Zonder liefs aard // is als te onwaerd'
        De Lichaemen daar de Ziel uit is.
