
    
      
        T'Valt mijn // een pijn
        Aldus verkleynt te wezen,
        Bij der Ieucht fijn
        Ben ik vergheten heel,
        Tfenijn // voor wijn
        Werd zeer van haar ghepresen,
        Het drinken dunkt hen eel,
        Daar schade schijnt voordeel
        Zo dom is jeuchts ghespeel,
        Welks bloem valt van haar steel:
        Ziet hoe dat hier onachtsaemheyt
        Den tijd verquist ende afleyt,
        Zulk zeid // ontseyt
        Goed oordeel diet misgreyt.
      
      
        Komt Ieucht // en veucht
        V onder mijnen leren,
        Go oorden en deucht
        Leert // die sijt goed van aard,
        V vreugd // gheneucht
        En wilt van u niet weren,
        Ontvangt mijn liefde waert:
        Hierom gheen moeite spaert,
        Schout vliet // wat traecheyt baert
        Zij niet, dat schaedt vergaert:
        Wilt onlust van u jaghen dan,
        V tijd dan wel ghebruyckt voortan,
        Den strijd // met vlijt
        Aenveert om dien gij vrijt.
      
      
        Ziet toe // ik doe
        Een yghelijck ten goede,
        Die op mijn hebben // go,
        Acht met hert sin, ghedacht
        Denckt hoe // ik spoe
        Als die niet werde moede,
        Mij te ghebruycken tracht
        Den tijd na niemand wacht,
        Die sluymt, bij daech oft nacht
        Versuym, baet gheen na clacht,
        'Tverlies tot zijn der groter schant
        Van een zo costelijcken pant,
        Niemand // t'weer vant
        Dus oefent u verstand.
      
      
        Die Wet // gheset
        Om weren wat mach schaden,
        Neemt aan zonder verlet,
        Sijt oorden onderdaen,
        Te bet // gij met
        Den tijd zo zonder smaden,
        Des wijsheyts pat zult gaan
        Haar in den gront verstaen,
        Dit is, die rechte baen
        Ghewis, gheen ydel waen,
        Diet oyt te recht verstonden ziet
        'Tverbont viel hem zo zwaer doch niet,
        Gij cont // goed rond
        Vermeerderen uwe pont.
      
      
        Nu groeyt // en bloeyt
        Mijn gheestkens uit ghelesen,
        Met s'Gheests natuer besproeyt
        Weest neerstich int ghebou,
        Lust voeyt // cloeck roeyt
        En laat u niet verknesen
        Ik wijsheid uwe Vrou,
        Zal geven u mijn trou
        Dijn Naam // ik voor behou
        Den faem // diet oyt vergou,
        Met rijkdom, goed en eer belaen
        Ver van haar salse zijn gedaan,
        Die teer // in leer
        Maar int versuymen meer.
      
      
        Mijn zee // ook mee
        De jonckheyt vordel dede,
        Wilt gij niet zijn ghehoont
        Mij voor u tuchtster croont,
        Rust free // in stee
        Van tweedracht staan alrede,
        Zo mij met reden toont
        Dat oorden bij u woont,
        Met const, werd gij verschoont,
        Tijds jonst, u wel beloont,
        Hierom u tijd doch wel besteet
        Tot deucht en eer zijt snel bereet,
        D'ontucht // ontvlucht
        Zijt gierich tot leer sucht.
