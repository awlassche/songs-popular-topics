
    
      
        Wilt  niet vrezen, gij Christen verheuen
        Mer wilt aencleuen // dat Goddelic woord
        Al wordt gij daarom veriaecht, verdreuen
        Al costet v leven // weest niet verstoort
        Denckt dat gij moet gaan  door de nauwe poort
        In lijden voort
        Want Christus dat selue heeft gheseyt
        Wilt v verblijen // en vromelijck strijen
        Als gij moet lijen // om de waarheid.
      
      
        Om de waarheid leden, alle Gods Profeten
        Alzo wij weten // claerlijcken wel
        Zij zijn veruolcht, en dood ghesmeten
        Zij waren verbeten // van Jesabel
        Wie niet en prees haar gierich vals opstel
        Moest int ghequel
        Met moede fel // dreef syse wijt en breyt
        Wilt v verblijen // en vromelijck strijen
        Als gij moet lijen // om de waarheid.
      
      
        Om dat Sacharias, dwaerheyt de wtbreken
        Daarom is hij versteken // gesteenicht saen
        Om dat Micheas geen luegenen wou spreken
        Soot is ghebleken // hoort mijn vermaen
        Daarom gincmen hem voor zijn wangen slaan
        Hij worde gheuaen
        In den kercker is hij vast gheleyt
        Wilt v verblijen // en vromelijck strijen
        Als gij moet lijen // om de waarheid.
      
      
        Daniel die ginck den Afgod bederuen
        Hij deed ook steruen // den gierigen Draeck
        Doen gingen Bels dienaars een placcaet verweruen
        Om hem te doen sweruen // zo hielden zij spraeck
        Inder Leeuwen cuyl moest hij om deze saeck
        Hij gaf God de wraeck
        Die hem heeft bewaard, hoort na bescheyt
        Wilt v verblijen // en vromelijck strijen
        Als gij moet lijen // om de waarheid.
      
      
        Al coemt gij om de waarheid in schanden
        Dat gij achter landen // sweruen moet
        Al wilmen v daarom doden en branden
        Dat gij geen offeranden // den Afgodt en doet
        De drie Jongelingen en vielen hem niet te voet
        Om lijf noch om goed
        Om te storten haar bloed // waren wij bereid
        Wilt v verblijen // ten vromelijck strijen
        Als gij moet lijen // om de waarheid.
      
      
        Hoort wat Christus den zijnen gaat gewagen
        Gij zult wenen en clagen // wilt dit beuroon
        Wt haar Synagogen zullen zij v iaghen
        En gheuen v slaghen // met coorden en roen
        Om v te doden, zullen zij haar spoen
        Om God dienst te doen
        De wereld zal lachen, als gij schreyt
        Wilt v verblijuen // en vromelijck strijen
        Als gij moet lijen // om de waarheid.
      
      
        Leest doch hoe Paulus is gheuaren
        Ende de dienaren // Gods ook al me
        Doen Joannes de waarheid ginck verclaren
        Vrij zonder sparen //  men zijn hoofd af sne
        Doen Stephanus de waarheid claer ontde
        Doen waren zij re
        Om hem te stenigen zonder eenich verbeyt
        Wilt v verblijen // en vromelijck strijen
        Als gij moet lijen // om de waarheid.
      
      
        De Princelijcke waarheid, heeft zelfs geleden
        En voor ons gestreden // met veel lijdens zwaar
        Hij heeft voor zijn vianden ghebeden
        Om te brenghen in vreden // ons allegaer
        Hij is voorghegaen in groten vaer
        Volcht hem troulijck naar
        Zijn armen heeft hij voor ons wtghespreyt
        Wilt v verblijen // ende vromelijck strijen
        Als gij moet lijen // om de waarheid.
