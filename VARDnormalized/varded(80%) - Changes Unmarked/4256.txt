
    
      
        EErbare, zoete // ieught,
        Bloemken weert alder eeren:
        Dat u ontmoete // vreugd,
        Wensch' ik, int salueeren:
        Gij zijt voorwaar
        Een Peerle onder vele:
        T'elcken avond spele
        Is uwen naam lofbaer.
      
      
        Laat mij dijn name // gaan
        Vol prijzen naar haar weerde,
        Ziende bekwame // aan
        Hoe gij betreet de eerde,
        Zeer minnelick
        Moet mijnen geest ontspringen:
        Gij zijt in u dingen
        Volmaect, godinnelick.
      
      
        Ionge volmaecte // bruyt,
        Beleeft in u faconde<
        Nooit man en smaecte // fruyt
        Zo zoet, tot zijn gesonde,
        Veel smerte // dect
        Gij, deur troostbaer genesen,
        V amoureus wezen
        Liefs hert aan t'herte // trect.
      
      
        Ziende dijn wangen // root,
        Int herte ik verschiete:
        Gij zult den strangen // noot
        Dijns liefs, haast doen te niete:
        Als hij zulks merct,
        Gij bluscht hem al zijn pijne,
        Zo de medecijne
        Een crancke siele sterct.
      
      
        Acht dit de waarheid // al,
        Noch meer mocht hier ontbreken:
        Niemand de claerheyt // zal
        (Dijns lofs) geheel wtspreken:
        Doch even wel
        Al zijt gij weert de krone,
        Maagden fris, en schone,
        Zijn meest haar lief rebel.
      
      
        Bloemkens van geure // zoet,
        Konnen veel schoonheyts dragen:
        Maar u coleure // moet
        Mij bouen al behagen,
        Vaste pilaer,
        Manierich, ende stille,
        Coemt u liefkens wille
        In alle heusheyt naar.
      
      
        Es t'uwen hove // rijk,
        Aangenaam dit gedichte:
        Zo ist wel love // lijck
        Dat ik een schoon gezichte
        (Deur dit cleyn werk)
        Int minste mach believen:
        Ik zond anders brieven,
        Waar ik ter cunsten sterk.
      
      
        T'is tijd om scheiden // nu,
        Dits dan mijn supplieeren,
        God wil geleiden // u:
        Tot dat wij wederkeren,
        Princess net
        Wilt ons vriendschap niet mincken,
        Alst u zal gedincken,
        Peynst om mij altemet.
