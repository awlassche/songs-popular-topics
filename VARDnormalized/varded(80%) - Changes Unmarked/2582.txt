
    
      
        Hou, Herder slecht // waar trekt de loop dus heen?
        (De Liefst oprecht // te Off'ren myn gebeen
        Ja hert en sin // in will'ghe slaverny)
        Weg met de Min // t'is niet dan kasserry.
      
      
        Het is wel waar // ik ken myn cleene macht,
        Dat niet een haar // Ik, bij heur ben geacht,
        Maar t'out gepeys // van deze schoon Goddin
        Ziet int Fourneys // myn Ziel door heete Min.
      
      
        Want als ik weer // aensie does waarde Vrouw
        Vermeert 'tbegeer // en 'tverdubbelt myn rouw,
        Haar schone glans // en lieffelijck ghedaant
        Mij nu althans // tot d'ouwe Liefd' vermaant.
      
      
        Door u ghesicht // zo lijd ik Garen Brandt,
        Wiens Fackels licht // verlicht myn dom verstand
        Hoe wel ik swerf // na 't gene dat myn schaat
        Ja leevend' sterf // veel duizend doden quaat.
      
      
        Doch in ootmoedt // werp ik mij sellifs neer
        Voor u te voet // verschoont myn trots begeer
        U heuschen aardt // laat blijcken op het minst
        Ik bendt niet waardt // al wens ik deze winst.
      
      
        Maar belght myn Vrouw // dat een zo slechten slaaf
        Als ik mij houw // staat na de waartste gaaf:
        Maar niet dat gij // maar ik zal u ontsien
        Heerscht Liefste vrij // wat wilt gij mij gebien.
      
      
        Wat u Hert lust // gij crijcht wat u behaacht
        Van die d'Aard kust // die uwe Voeten draacht
        K'droech u op d'hand' // nam ghy't Princes int goed:
        Komt sulcken brand' // ws dienaars te gemoet.
      
      
        Princes te hoog // ik myn begeer gevoel
        Voorwaar zat ooch // waart gij myn wit myn doel,
        Och soot syn mocht // maar soot niet mach geschien?
        Nemt u genocht // in mij dat ik u dien.
