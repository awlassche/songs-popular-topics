
    
      
        Het is een schoon dierbaer zake
        Dat men maecke // onzes Gods Naam
        Groot, met Lofzangh ende spraecke,
        Zoet van smaecke // is hij bequaem;
        Iae vriendelijck ende minsaem
        Laat ons zijn faem // verbreyden wijdt,
        Looft hem ten rechten // die zijn knechten, zijt
        Met groot jolijt
        Looft den Schepper ghebenedijt,
        Die alle dingen heeft gemaakt subijt,
        Alle ere // tot hem kere,
        Nu end’ t’aller tijd.
      
      
        Synen Stoel is boven schreven,
        Hoog verheven // wijdt ontdaen:
        Veel duizend Enghelen beven,
        Die beneven // voor hem staan:
        Zijn gramschap doet berghen vergaen,
        Die diepten saen // verdroghen vrij,
        Voor zijn aenschouwen // mans en vrouwen, gij
        Looft hem, waar ‘t zij,
        Meerder dan alle lof is hij:
        God is overal, verr’ en bij,
        In hem leven // ende zweven
        Allegader wij.
      
      
        Laat ons God een nieuw Lied singhen,
        Grote dingen // doet hij bloot:
        Voor hem beeft ‘s werelds omringhen,
        Ook springhen // de steenrotsen groot
        In stukken, voor zijn toornig exploot,
        Maar in den noot // is hij zeer goedertier,
        Die betrouwen // op hem bouwen hier,
        Hij helptse schier,
        Wt lyden ende groot dangier,
        Hij gaat met haar door water ende vier,
        Hij kan merken // al ons werken,
        Hij proeft hert en nier.
      
      
        Hij weet al ‘s mensen ghedachten,
        Al haar trachten // en verstandt:
        Klaer als dag zijn hem de nachten,
        Groot van machten // is zijn hand,
        Hij hoort en ziet wel alle dingen, want
        D’oore gheplant // hebbend’ als wijs engien,
        Soud al voren // ook niet horen, dien
        Zoude, door wien
        D’ooghe gemaakt is, ook niet konnen zien
        Hij weet alle dingen eer zij gheschien,
        Na Schrifts getuygen // voor hem buigen
        Moeten alle knieën.
      
      
        God is een Coninck ghepresen,
        Ende dezen // woont in een licht
        Daar niemand toe komt gheresen:
        Zijn Wezen // van mensen ghesicht
        Is verborghen: hij heeft ghesticht
        Al het ghewicht // der aarden zwaer,
        Zeer groot bevanghen // en ghehangen, daar
        Op ‘t water klaer,
        Met zynen woorde krachtig ende waar,
        Ende hij onderhoudet ook all’ gaer,
        Zijn vermoghen // ongheloghen
        Blijckt wel openbaer.
      
      
        Al zijn werken groot bevonden,
        Die men gronden // niet en mach,
        Van zijn heerlijckheyt oorkonden,
        te allen stonden // enen dach,
        Doet den anderen daar af ghewach.
        Maar nooit en sach // iemand God vanden troon,
        Maar zijn verkoren // een-gheboren Soon,
        Heeft ons zeer schoon,
        Dat verkondigd // Lofprijs zij hem te loon,
        Die op den stoel zit, weerdig alle Kroon,
        Eeuwich ghepresen // moet hij wezen,
        En dat Lam ydoon.
      
      
        Doe wij Gods Vijanden waren,
        Iae zondaren // boos en fel,
        Ginck hij ons zijn Liefde baren,
        Zonder sparen // gaf hij snel,
        Voor ons allen in doots ghequel,
        Synen zeer wel // beminden Zone zoet,
        Die ons in noode // met zijn rode bloed,
        Verloste vroet,
        Ziet, wat een liefd’ ons God den Vader doet,
        Dat hij ons als kinderen zijn goed,
        Maakt deelachtig // hem eendrachtigh,
        Eeuwich Lof zijn moet.
      
      
        Zijn ghenade onverdweenen,
        Is verscheenen // over al,
        Tot profijt van den gheenen,
        Die recht heenen // in dit dal,
        In Christo wandelt den weg smal,
        Maar heten zal // na Schrifts bediet,
        Den obstinaten // zonder baten, niet
        Hem en gheschiedt
        Gheen ghenade // maar onghenade, ziet
        Toorn, anghst, druck en zwaer verdriet,
        Wie te voren // synen tooren
        Hier niet en ontvliedt.
      
      
        Tot quijt-scheld van ‘t sondigh leven
        Is ghegheven // die geloofd
        Iesus den name verheven,
        Tot ons leven // ende hoofd,
        Des Vijands macht heeft hij berooft,
        Blijft niet verdooft // wie oren heeft die hoort,
        Wilt ontvangen, met verlanghen, ‘t woord
        Rein onghestoort,
        Schickt u te gaan door d’enghe poort,
        Met liefd’ en broederlijck accoort,
        Brengt den name // Gods bekwame,
        Vrucht der lippen voort.
      
      
        Vreest God al gij Mensen teere,
        Gheeft hem ere // want het naakt,
        Nu den tijd zijns oordeels seer,
        Na zijn Leer // altijdt waeckt,
        Aanbidt hem, en niet en versaeckt,
        Die heeft gemaakt // Hemel en ‘t aertsche veld,
        Zee en Fonteinen // dezen reinen, held
        Zij toe ghestelt,
        Eeuwig lof, ere, prijs, ende gheweldt,
        Gij Conincklijck Priesterdom, hier vertelt,
        Vrij met breughden // al zijn deughden,
        Al wordt gij ghequelt.
      
      
        In Gods lof wilt niet verkrancken,
        Christen ranken // onversteent,
        Wilt in alle dingen dancken,
        Spijs en drancken // hij verleend,
        Geen mensch en wordt voor hem vergeten, neent
        Sorgt noch en weent // niet, om dat vlees aangaat,
        Maar wel gade // Gods ghenade, slaat,
        Daar in bestaat
        Met vaster herten blijft vroeg ende laat
        Bij den Here, volgt zijnen raad,
        Om niet verdolen // hem bevolen
        Blijft, Christelijck zaad.
