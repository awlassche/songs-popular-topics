
    
      
        OCh ! hoe vast staat hij, o Here, Die op u betrouwt
        Op u, Rosteen, t'uwer ere // Heb ik vast gebouwd;
        Hierom zal ik niet beswijcken // Want in allen noot
        Zal mijn vaste bouwsel blijcken // Tegen al aenstoot.
      
      
        Gij beschaemt al mijn Vijanden // En haar wreed' besluyt
        Maect gij in 't geheel te schanden // En redt mij daar uit:
        Doen mij meende uit te rooden // Mijne snoo party,
        Ging den Heer mijn Vyandt doden // En ik raecte vrij.
      
      
        Die u liefde mach genieten // O ! wat vreed', en vreugd,
        Gaat gij stadigh op hem gieten // Die hem zo verheucht,
        Dat hij moet met aendacht zingen // Vrolijck uwen lof,
        Hier toe voel ik mij ook dringen // Hier toe heb ik stof.
      
      
        Komt mijn tongh de kracht t'ontbreken // 't Herte blijft vervult,
        Stadigh wil ik van u spreken // Want ik weet gij zult
        Altijdt zyn in mijne wegen // In vreugd, angst en nood:
        Ik vrees' niet, al heb ik tegen // Wereld, Duyvel, Dood.
      
      
        Treet tot u, al mijn gedachten // Maakt mij, Heer, bereid,
        Ik wil stadigh tot u trachten // Met eerbiedigheyt:
        En wil vrolijck in u weghen // Tot u ere gaan,
        Och ! maakt mij bereid te degen // Dan zal ik vast staan.
      
      
        Kond' de vreuchd' in 't herte blijven // (Door u goedigheid)
        Die soud' uit mijn herte drijven   's Werelts ijdelheid.
        't Herte kan het al niet vaten // 't Wil niet zijn gemoeyt
        Om qua lusten in te laten // Want het overvloeyt.
      
      
        Soud' men, Heer, u niet beminnen ? Die u soetheyt smaect
        Komt de grootste Rijcdom winnen // Die recht is volmaect
        Want gij kont het hert vernoegen, Maar nooit staat, of eer.
        't Beste gaat gij mij toe-voegen // Lof zij u, mijn Heer.
