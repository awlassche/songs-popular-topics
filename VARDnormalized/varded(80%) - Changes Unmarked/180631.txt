
    
      
        Rein van manieren // is den Echten staat,
        Die hem hantieren // in den rechten graet,
        T’is slechten raad // die hem verbieden, want
        Beter is trouwen // dan te lijden brand:
        Want die hoereren // mal,
        God condemneren // zal,
        Des wij lauderen // al,
        T’huwelijcx bant.
      
      
        T’is waar dat vrienden // in malcanders noot,
        Dikwijls wel dienden // als bystanders bloot,
        Maar anders groot // met een affecty vrij,
        Staan man en wijf ter noot // malcander bij:
        Wie een go Vrouwe // heeft,
        Die God ghetrouwe // gheeft
        Bevrijdt van rouwe // leeft
        Te langher hij.
      
      
        Een wijf die deuchdich, is van wezen vroet
        s’Mans herte vreuchdich // zij genesen doet,
        Ghepresen moet zij zijn om haar bedrijf,
        Een edel gave Gods is zulk een Wijf
        Tot troost van haren // Man
        Die wel bewaren // can,
        Haar tongh en sparen // van
        Weder ghekijf.
      
      
        Zo lief bekwame // op der eerden niet
        Als een eersame // Wijf vol weerden ziet,
        Met zweerden vliet // een Roover in der nacht,
        En brengt den roof haar die boven geslacht
        Hij voor vriendinne // coos
        Veel zijn door minne // boos
        Geworden sinne-//loos
        En omghebracht.
      
      
        Een vroom Wijf schone // is als brande daar
        De lamp ydoone // op’t heylich Kandelaar,
        Zijnde bestande haar // eerbaer ghemoet
        Een gulden Pijlaer op een zilveren voet,
        Is zij gheleken // wel
        Om niet te breken // snel
        Oft te ontsteken // fel
        Door bozen gloet.
      
      
        Die een Wijf zedich // voor zijn proven hout
        Huylich, cuysch, zedich // machse loven stout,
        Al boven goud // en boven edel steyn,
        Haar sieraad is wat neerstich is en rein
        Al moets’ hier lijden // smert
        Haar vreugd in tijden // werd
        Met een zo blijden // hert
        In s’Hemels pleyn.
      
      
        t’Wijf zal door baren // ende kinder dracht,
        Ten Hemel varen // datse ginder lacht
        Zo in der kracht // ghleoove, liefde jent,
        Heiligheid, eerbaerheyt, haar blijft ontrent
        Voor des versijcke // wat
        Werd zij in Gods rijke // stad
        d’Enghels ghelijcke // dat
        Werd excellent.
      
      
        Prinse d’oprechte // zo zij dede neer,
        Zal men in Echte // niet besteden meer,
        Te vreden zeer // wel eeuwich zonder pijn
        Zullen zij in s’Lams bruyloft feeste zijn
        Zo vrolijck maken // schier
        Elk mach wel haken // fier
        En neerstich waken // hier
        Om daar te zijn.
