
    
      
        Myn tijd // verslijt, ik lief ghepresen
        Met vresen, gants en gaer
        die lijt // subijt, wilt dienstich wezen
        Genesen // zijn gevaer, maakt dat hij werd gewaar
        V hulp tot hem genegen
        Door praet // ghelaet // zijn hart versleghen
        Met raad // bystaet // in deze nieuwe iaer
      
      
        V heeft // beleeft, gediend Jonkvrouwe
        Ghetrouwe // does dienaar
        Hem gheeft // beleeft, blyschap voor rouwe
        Door trouwe // vreugd eerbaer
        Maect u nv eenpaer // zo zal zijn druc cesseren
        Met spoet // zeer zoet // al u begeren
        Hem doet // ootmoet // in deze nieuwe iaer.
      
      
        Dat hij onvry // u troost moet derven
        Doet sterven // vreuchde maar,
        Zeer bly // hoept hij, u hart te erven
        T’verwerven, zal voorwaar, doen slijten t’lijden zwaar
        Dat hem nu quelt ghestadich
        Dus vrou // zijn rou, verdrijft ghenadich
        Die rou // dient trou // in deze nieuwe iaer.
      
      
        V vlijdt // zeer blijdt // Princes verheven
        Beneven u dienaar
        Wien spijt // benijt // laat zijn verdreven
        Het sneveen, en misbaer, twelck hen quelt voor als naar
        Wilt doch zijn pijn versachten
        Dat hij // lief bij // u mach vernachten
        Heel vrij // en bly // in deze nieuwe iaer.
