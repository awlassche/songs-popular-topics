
    
      
        OP op mijn geest // en wakker weest,
        Om mijn ghedacht te stieren.
        Al wat men eert // ter Wereld leert
        V bet mijn Lief te vieren.
        Want alles swicht // het voor het licht
        Dat ik moet eeuwich minnen.
        Voor't eenich goed // dat mij voldoet
        Mijn al te kiessche sinnen.
      
      
        Het grof geslacht // des vollicx acht
        Van andere waerdt te booghen.
        O domme tijd // die mij verslijt,
        Gij hebt noch breyn, noch ogen!
        De gheen die gij // hoog in waerdy
        Verheft, en zijn gheen Vrouwen.
        Of zijn't, ik kan // de mine dan
        Voor een Mannin niet houwen.
      
      
        Die ghylie viert // zijn wel verciert
        Met schoon en aerdicheden;
        Maar bij den dach // en (dunctme) mach
        De claerste nacht niet treden.
        De mine toog // (want jong zij sooch
        Heltinne moeders speenen)
        De wise deucht // die't al wel veucht
        In't merrech van haar benen.
      
      
        Wie boven al // op d'aerdsche bal
        Doorluchtich is verschenen,
        Wies deur gevaer // en commer, daar
        Veel' in versmachten, heenen.
        Geen mensen hand // mijn dierste pand,
        Wt grousame' onghelocken,
        Maar loutre wacht // van hoger macht
        Des Hemels, heeft ghetrocken.
      
      
        Den Hemel droeg // wel laat en vroeg,
        Met reden, sorgh gheladen,
        Voor't schoonste waert // daar hij vergaert
        In had, zo veel cieraden.
        Van yder daad // die hij aenvaet,
        'Tmisvoeghen is ghebannen:
        Zij zingt of praet // zij lacht zij gaat
        Een strickjen staat // ghespannen.
      
      
        Een strickjen daar // verwarren haar
        In, al die kennis draghen.
        O wonder hoog! // O werelds ooch!
        O glans van onze dagen.
        De clare Son // niet op gaan con,
        Zo gij u aenschijn heelde:
        Alleen want hij // verrijst, voor dy,
        Mijn lief, mijn licht, mijn weelde.
      
      
        voor u de May // nu maxels fray
        Verziert, van kruid en bloemen.
        Maar als gij scheyt // uit menschlijcheyt,
        Sal't aan u staan, te noemen
        Wat, dat gij wilt // of Pallas schilt,
        Of Iunoos scepter draghen,
        Op dat u wijck // de suyverlijcke'
        Venus uit haar wagen.
