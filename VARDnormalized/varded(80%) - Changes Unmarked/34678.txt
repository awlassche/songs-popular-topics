
    
      
        NV Coridon schept moet,
        Men gaat den Peys verkonden;
        Den Vrede-maecker zoet,
        Die is ons af-ghesonden.
        Op dezen blijden nacht,
        Niet verr' van Davids stê,
        Messias heeft gebracht
        Den langh-ghewenschten Vrê,
        Hij gheeft ons peys // Voor deze reys:
        Voor deze reys // Hij gheeft ons peys,
        Peys, peys, peys, peys //
        Voor deze reys.
        Peys, peys, peys, peys //
        Voor deze reys.
      
      
        In't stal van Bethleëm,
        Te middernacht ghebooren,
        Gheeft uit een kinder-stem,
        Die God was van te voren:
        Hij ligt op stro verrot,
        Die is dat Vaders-Woordt;
        Om tussen ons en God
        Te maecken een accoordt.
        Hij gheeft ons peys // Voor deze reys:
        Voor deze reys // Hij gheeft ons peys:
        Peys, peys, peys, peys //
        Voor deze reys.
        Peys, peys, peys, peys //
        Voor deze reys.
      
      
        Voor Adam, en zijn zaad
        Komt hij de schuld betaelen,
        En op zijn schouders laat
        Al onze straffen daelen:
        Den Vader valt hij vrij
        Voor ons misdaad te voet,
        Voor onzen peys wilt hij
        Verghieten ook zijn Bloed.
        Hij gheeft ons peys // Voor deze reys:
        Voor deze reys // Hij gheeft ons peys:
        Peys, peys, peys, peys //
        Voor deze reys.
        Peys, peys, peys, peys //
        Voor deze reys.
      
      
        O Moyse gaat het zo !
        Hoe moeten wij hem minnen,
        Die ligt op hoy, en stro:
        Laat ons den stal gaan binnen,
        En kussen hem een reys
        Aan zijnen rooden mond;
        En singhen peys, peys, peys
        Voor eeuwig in't Verbond.
        Nu kust hem peys // Noch deze reys:
        Noch deze reys // Nu kust hem peys;
        Peys peys, peys, peys //
        Noch deze reys.
        Peys, peys, peys, peys //
        Noch deze reys.
