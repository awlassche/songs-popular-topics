
    
      
        O! Mijn Agnietje ,, mijn honingbietje,
        Die 't Lam gevolgt hebt naar
        Een kind van darthien jaar.
        Gij hebt gebeden // gij hebt streden
        Volhardigh voor uwe eer
        Door d'hulp van God uwe Heer.
        U jonge hoofd // door 't swaert is afgeklooft,
        Des hemels stad // Gij doen in-tradt,
        O waerde Hemels schat!
      
      
        Gij zijt gekomen // Gij zijt vernomen,
        Gij liet U weder zien
        Den achtsten dach na dien.
        Uwe lieve Ouders // uwe onderhouders
        Die bleven bij u Graf,
        Zij gingen daar niet af.
        Als U de dood // van 't leven had ontbloot,
        Hebt G' hun vertoond // waar dat Gij woont,
        En hoe Gij zijt beloont.
      
      
        Een Choor der Maagden // die God behaeghden
        Was doen met U verzelt,
        G'lijck als ons wordt vertelt.
        Gij tot haar zeide // die daar verbeyden,
        En wilt niet drouvich zijn,
        Wij hebben druck, noch pijn;
        Met 's Hemels vreughd // zijn alle wij verneught,
        Wij zijn bevrijdt // van alle strijd,
        Wij eeuwich zijn verblijdt.
      
      
        O Patronersse // mijn Voorspraekersse,
        Gods uytverkoore Maagd,
        Jonckvrouw, die God behaagt
        Die hebt vertreden // met recht en reden,
        Des lichaams dartelheyd,
        Die menich mensch verleid:
        Na redens eysch // wil dwingen ik mijn vleysch,
        Ik ben begaan // ik roep U aan,
        Laat mij uwe hulp ontfaen.
      
      
        In all' mijn lijden // in all' mijn strijden
        Versoeck ik u gebed,
        Ik bidd' U, op mij let.
        Wilt mij erbermen // wilt mij beschermen
        Op 't laetste 's levens end
        U oogh dan tot mij wendt.
        Verkrijgt dat ik // voor Satan niet en schrik,
        Die mij hier quelt // op 's werelts veld.
        Ach! breeckt toch zijn geweld.
