
    
      
        Waeckt op, waeckt op ‘t is meer dan tijd
        En laat de wereld varen  //  ja varen,
        Maakt u des bozen levens kwijt,
        En wordt uit God ghebooren  //  ghebooren.
      
      
        De lust des werelts moet vergaen,
        Heeft Ioannes gheschreven  //  gheschreven.
        Maar die den wille des Vaders doet,
        Die zal eeuwichlijck leven  //  ja leven.
      
      
        Geld ende goed  //  en daar toe moet,
        Dat moet men weynich achten  //  ja achten,
        En volghen den Here van herten goed,
        Dat eeuwich goed te betrachten  //  betrachten,
      
      
        Wie enen Thoren bouwen fel,
        Die mach hem wel versinnen  //  versinnen,
        Overdencken eerst de kosten wel,
        Eer hij dat gaat beginnen  //  beginnen.
      
      
        Vlees ende bloed mach niet bestaan,
        Dat moet men daarom wagen  //  ja wagen
        Want het moet al zeer korts ondergaen,
        En zijn kruys moet men dragen  //  ja dragen.
      
      
        Het is den vlees een sware pijn,
        Het en kan ‘t niet wel lijden  //  ja lijden,
        Als ‘t van de wereld veracht zal zijn,
        Dan zal de Gheest verblijden  //  verblijden.
      
      
        Nu gheeft mij u ghenade Heer,
        En wilt mij wijsheid geven  //  ja geven,
        Dat ik mach wandelen na uwe leer,
        Haten mijn eyghen leven  //  ja leven.
      
      
        De Stad is zeer lustig en fijn,
        Daar in ik hope te komen  //  te komen,
        Maar daar moet veel om ghestreden zijn,
        Dat heb ik wel vernomen  //  vernomen.
      
      
        Veel Vijanden aan elken kant,
        Zijn op does enghe straten  //  ja straten,
        O Heer doet mij nu onderstant,
        En wilt mij niet verlaten  //  verlaten.
      
      
        Vleeschelijck ghesint zijn  //  is de dood,
        Zo betuycht Paulus leer  //  ja leer,
        Gheestelijck ghesint zijn is ‘t leven bloot,
        En vrede in den Here  //  ja Here.
      
      
        Daarom heb ik mij korts bedacht,
        De waarheid na te jaghen  //  te jaghen,
        Niet meer te wandelen in de nacht,
        God alleen te behaghen  //  behaghen.
      
      
        Maar het staat niet in mijnder magt,
        Doen kan ik niet allene  //  allene,
        Daarom, O Heer neemt mijnder acht,
        Maakte mij dat herte reene  //  ja reene.
      
      
        Ik zal ‘t wagen door u ghenae,
        Op u beloften schone  //  ja schone,
        Op u beloften schone  //  ja schone,
        ‘t Lichaam moghen zij doden  //  ja doden.
      
      
        Dit Liedeken is uyt-ghericht,
        Tot den prijse des Heren  //  des Heren,
        Is ‘t dat daar iemand door werd ghesticht,
        Dat is wel mijn begheeren  //  begheeren.
