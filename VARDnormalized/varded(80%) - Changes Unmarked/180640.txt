
    
      
        EYlaes hoe saen // gaan // hier voorbij
        s’Menschen dagen gelijc int veld daar wij
        Zien schoon ontdaen // staan // die lely
        Die t’sander daags verdorven zij
        Alzo gheschiet // het, met ons ziet,
        Haast wordt hier iet // niet // door s’doodts cruys,
        Menschelijck riet // vliet // boos abuys
        Hier is u stad niet, gij hoort elders t’huys.
      
      
        Merckt hoe den tijd // lijt // hier present
        En alle dingen comen tot den ent
        Hierom met vlijt // zijt // diligent
        Om God te kennen, eerst u zelven kent
        En weest niet prat // op goed noch schat
        Ellendich vat // wat // gij begint
        Een ander stad // dat // zoekt en vint
        Of het is al verlies wat gij hier wind.
      
      
        Wat batet al // dal // berg oft land
        Iae die weerelt met haar lusten playsant
        Zo in den val // zal // lijden brand
        De ziele verloren t’costelijck pant
        O Mensch vermaert // dwinght uwen aard,
        Want onghebaert // waert // beter zijn
        Dan dat men vaart // daert // doots gepijn
        En schreyen zal dueren eeuwich termijn.
      
      
        Tegen den dood // noot // wendich raad
        Gebruikt, dat gij niet eeuwich en vergaet
        Wt Gods woord bloot // t’groot // weerdig zaad
        De rechte form des geloofs ontfaet,
        Baert liefden kracht // tot Gods gheslacht
        Voor die tweedracht // sacht // loos aenricht
        V wijs bedacht // wacht // met opzicht,
        Blijft op den gront der waarheid vast ghesticht.
      
      
        Prinse gemaakt // naakt // hier van slijk,
        Een schepsel den Schepper te zijn ghelijck,
        In deuchden waeckt // haect // na Gods rijk
        Om bij hem t’huys te zijn doet geen afwijc,
        Houdt in t’ghemoet // als honig vloet
        s’Heeren woord zoet // vroet // en vreest God
        Latet kwaad, doet // goed // na t’ghebodt,
        Gij crijght uit genaden t’Een noodich lot.
