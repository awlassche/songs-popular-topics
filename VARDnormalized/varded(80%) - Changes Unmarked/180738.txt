
    
      
        EEn wintersche nacht
        Heeft mij zo langhen tijd verdroten,
        Lanck ende onsacht,
        Waar in de Mane staat besloten,
        Vochtich en cout // zeer menichfout
        Waren de velden zeer begoten,
        Met haghel snee // dat doet mij wee,
        Waar door de velden ploten,
        Groot nochte cleen // en vandt ik gheen
        Cruydt nochte noten.
      
      
        Den nacht zeer stout,
        Maakt mij geheel bedroeft van binnen,
        Den Winter cout
        Benout ons arme huisgezinnen,
        Mijn blijschap smelt // van gaan door t’velt,
        Coudt en nat was ons gewinnen,
        Int donker vrij // zo liepen wij,
        Dolende buiten weghe,
        Maar wij hebben eer lanck // Gode zij danck
        Een licht ghecreghen.
      
      
        Den dagheraet,
        Een morghensterre licht van schijne,
        De Zonne op gaat,
        Den nacht is weg en al ons pijne,
        Door t’soet geclanck // der vogelen sank
        Den Winter die moeste verdwijnen,
        Want schoon // ydoon // staat daar de croon
        Van den Hemelschen bouwe,
        Het komt al uit // loovers en cruyt,
        Blommekerssouwe.
      
      
        Een blomme int dal,
        Een Roose tot Saron ghepresen,
        Die over al
        Haar reucke gheeft rein uytghelesen,
        Een lelye zoet // vol van ootmoet
        Was tussen twee doornen gheresen,
        Dien Held // in t’velt // bedrijft ghewelt,
        Als enen Leeuw ontsprongen,
        Den stok vol haat // des drijvers kwaad
        Heeft hij ontwronghen.
      
      
        Een bloyende rijs,
        Dat droeg amandelen bekwame,
        Crachtich en wijs,
        Dat zand God zelve in zijnen Name,
        Enen Mey met blaen // zeer wijt ontdaen
        Die droeg zo goede vrucht eersame,
        Hij groeyt // en bloeyt // zeer vriend’lijck besproeyt,
        Door des Somers voortbringhen,
        Int zoete dal // der voghelen gheschal,
        Fonteinen springhen.
      
      
        Noch vandt ik daar
        Een Maghet schoon rein uytvercooren,
        Haar stemme claer
        Was mij vertoond // zeer zoet om horen,
        Haar ogen laeft, // met melk begaeft,
        Haren hals was als Davids tooren,
        Nooit mey // zo vey // na mijnen grey,
        Die mij zo wel gheriefde,
        Wt s’Vaders schoot // haar moeder groot
        Dat was de liefde.
      
      
        Komt hier mijn vriend,
        Sprak daar die suyverlijcke schone,
        Eer dat u dient,
        Wijn ende melk gheef ik te loone,
        Een cofferken goed // vol druiven zoet
        Is mijn vriend zelve in persoone,
        Hij verblijd // int crijt // en maakt jolijt,
        Hij gaat boven de Vorsten,
        Hij is van verre // Een busselken Mirrhe
        Tussen mijn borsten.
      
      
        Ik comme mijn Bruyt,
        Laat ons te samen gaan vermeyden,
        En horen t’gheluyt,
        V Moeder die wil ons gheleyden,
        Die u hier zand // als beloofden pand,
        En bezien hoe die wijngaarden schreyden,
        Laat fijn // rosijn // en zoeten wijn
        Wt uwe borsten druypen,
        Mijn herte snaeckt // na vreuchden haeckt,
        Gheeft mij te suypen.
      
      
        Een vroome zoet,
        Heeft mij ghelaeft met haren wijne,
        Haar woorden goed
        Zijn mijner Ziele een medecijne,
        Na haren danck // ik van vreuchden clanck,
        Zij had belooft het was al mijne,
        Ist dat // ik wat // vergare schat,
        En loop de vrome tegen,
        Tot dat ik mijn Lief // al met gherief
        Hebbe ghecreghen.
      
      
        Ik hebbe haar
        In eenicheyt beloeft mijn trouwe,
        Vroom ende eerbaer,
        Zij is die alderliefste Vrouwe,
        Zij is verciert // rein ghemaniert,
        Haren name gaat boven den gouwe,
        Zij zegt // t’is recht // o beminden knecht,
        Gij moet vast bij mij blijven,
        Of gij en meucht // in mijnder vreugd
        Doch niet beclijven.
      
      
        Mijn vrome saet
        Heeft zeven berghen schoon becleven,
        Die Voskens kwaad
        En hebben daar niet in bedreven,
        Zij staan zeer rein // onder t’Hemels pleyn
        Met Roosen, Lelyen verheven,
        Dates // cypres // loof, cruydt en gres,
        Aan mijn Lief te betrapen,
        Niet ik alleen // maar groot en cleen,
        Ende al haar knapen.
      
      
        Princess eerbaer,
        Van den Zomer int ghemeene
        Zeer verre en naar
        Zijn wij ghebrocht int gheweene,
        Want // zijn verstandt // is abundant,
        Wij zijn verlost uit liefden reene,
        Elk moet // zeer vroet // vallen te voet,
        Met deze Bruydt ydoone,
        Binnen t’Somers tijd // u zelven kwijt,
        Staat na de krone.
