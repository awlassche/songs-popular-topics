
    
      
        Ik moet een lied, in's Heren naam beginnen,
        Op dat ik jeet // mocht an u Ziel gewinnen:
        Al mocht ik dy // de gantse Wereld geven,
        't Heeft geen weerdy // bij een Godtsaligh leven:
        Der Zielen Goed // is't hoogste goed geprezen,
        Een rein gemoed // ontvangt Gods Heylich wezen.
      
      
        Wilt gij zijn een // verkoren Bruyt des Heren,
        Tot hem alleen // moet gij u liefde keren,
        Vergetet al // u Vaders huis en magen,
        Den Koning zal // in u hebben behagen,
        Maar ist dat gij // Egipten wilt gedenken,
        Zo en zal hij // zijn liefde u niet schencken:
      
      
        Ieloers is God // en met liefde ontsteken,
        Zelfs wil hij tot // ons Ziel inwendig spreken,
        Zijn Bede-huys // ter eeren zijnen name,
        Moet rein en kuys // zijn, an Ziel en lichame:
        Geen sond' onreyn // staat hier vrij om te dragen,
        Daar God alleyn // regeert na zijn behagen.
      
      
        Scheydet u van // de Wereltlijcke lusten,
        Roertse niet an // die u Ziele ontrusten,
        Laat zijne Geest // een woningh in u maken,
        de Hemelse feest // zult gij dan lieflijck smaken.
        Zijn Avondmaal // zult gij met hem genieten,
        Zo gij zijn tael // u niet en laat verdrieten.
      
      
        Zulk een beloft, heeft ons den Heer gegeven,
        Wie dit versloft // die mist het eeuwig leven,
        In duysternis // wandelt hij als de blinden,
        Zeer ongewis // waeyt hij met alle winden:
        Maar die verlaat // des werelts woeste wegen,
        En doet Gods raad // die heeft de rust verkregen.
