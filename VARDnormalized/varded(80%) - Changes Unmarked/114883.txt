
    
      
        IN mijn leden // Hoe fris en sterk,
        Komt noch op heden // Geen treck tot werk,
        Om wat te winnen // heb ik geen lust,
        En in 't beginnen // Soeck ik de rust:
        Daar valt te slaven // Blijf ik van daen,
        Als and're draven // Houd ik van staan,
        En soeck het gemak // Doch niet aan de back,
        Daar kan ik laven, Mijn mage-sack.
      
      
        Ik weet van Sayen // Of Ploegens tijd,
        Noch ook van Mayen // Als zeer verblijd,
        Mijn Buren voeren // Den Oogst van 't Veld,
        Dan moet ik loeren // De slaap mij quelt.
        Voor mij te woelen // Des Somers heet,
        Dan moest ik voelen // mijn eigen zweet,
        Dat mij niet en past // Want dit is een last,
        Best mis te doelen // Die harde quast.
      
      
        Om zwaar te dragen // En dient mij niet,
        En veel te sagen // Is ook verdriet;
        Die mij komt spreecken // Om wat te doen,
        Vindt mij besweecken // En dapper groen:
        Want al mijn vreugde // Wordt dan verdriet,
        Die mij verheugde // En sprak zo niet;
        Zie ik een kloeck man // Die d'handen slaat an,
        Van zulke deughden // Zie ik dwars van.
      
      
        Ik houd van pralen // Het gantse Iaer,
        En veel te onthalen // Dan hier dan daar,
        Een lecker leven // Dat is mijn sin,
        Niet veel te geven // Om klein gewin,
        Net in de Kleeren // Vroeg in 't gelach
        Bij grote Heren // Den gantsen dag;
        Zo langh dit wil staan // Zal 't alles wel gaan,
        Maar licht kan 't keren // Dan is't gedaan.
      
      
        Die staat wil houwen // Met luttel geld,
        En hoog wil bouwen // Op 't bolle Veld,
        Vint hem bedroogen // Al eer hij 't gist,
        't Is weg gevloogen // Dat hij nood mist;
        Dies moet ik sorgen // Dat mijn geluk
        Van daegh of morgen // Verkeer in druck:
        Daar van dat alree // Mijn hertje doet wee,
        Die plach te borgen // Roept geld geree.
      
      
        Mijn goede Vrienden // zien van mij af,
        En mijn Beminde // Wordt droef en straf;
        Mijn lecker kluyven // Is gans verkeert,
        Men bakt geen struyven // Meer aan mijn Heert,
        Geen Suycker-wijntjes // Geen rare Vis,
        Geen Haes of Knijntjen // Komt op mijn Dis;
        Mijn Kleedingh wordt vael // mijn Beursjen is kaal
        En in mijn Spijntjen // Klinckt geen Reael.
      
      
        Mijn Huis versleeten // Kan kwalijk staan,
        Het Dak gespleeten // En heel vergaen:
        De Kat en Muysen // Loopen van hier,
        Daar ik moet kluysen // Met klein playsier,
        Ik segh u heden // En klaegh ook dit,
        Dat Frank mijn leden // Doorgaans bezit,
        En maakt dag aan dag // Dat ik niet een slach
        Van and're mede // Aenschouwen mach.
      
      
        Nu moet ik zuigen // Aan duim en poot,
        Om te betuigen // Mijn hongers noot;
        Nu moet ik sparen // Met groot verdriet,
        Om te vergaren // En woud' ik niet:
        Daar baet geen klagen // Die 't willens mist,
        Mijn schone dagen // Heb ik verquist,
        Al voel ik verdriet // 't En schaet mij doch niet,
        't Is met behagen // Van mij geschiet.
      
      
        Maar ik zal moeten // Door grote nood
        Met and're wroeten // Om 't daeghlijcks brood;
        Dat zal ook wezen // De rechte daad,
        Hij wordt geprezen // Die hier zo gaat:
        God geeft zijn zegen // Hem die zo doet,
        Loopt hem wat tegen // Het keert tot goed;
        Hij stuurt door zijn hand // De Lucht, Zee, en Land,
        Zijn mild Regen // Verheugt de plant.
