
    
      
        Ik heb bedacht // van het gheslacht
        Dat dach en nacht // neerstich verwacht
        Gij nieuw kracht // van God te cryghen:
        In haar ghevecht // recht ende slecht
        Ghelijck Iob secht // haar hope lecht
        Op hem wiens recht // niemand mach bedriegen
        Als met Arends vleugelen zij opvlieghen;
        Zij loopen en wandelen en worden niet moede:
        Maar haar vijand // die raeckt in schant
        Aan elken kant // na Schrifts verstand;
        Doch Godes hand // is haar behoede.
      
      
        Op Noachs daad // nu dencken gaat
        Hij vont ghenaet // door Godes raad
        Voor zynen saet // en wordt verheven:
        Die wereld rond // die wederstondt
        Des Heren mond // bleef in haar zond
        Ginghen te grondt // verlooren het leven;
        Die heeft God beide door't water bedreven:
        Dat vier deed ook die vrome gheen hinder,
        Sodoma ziet // en die Lot riet
        En hoorden niet // bleef in't verdriet
        Maar hij ontvliet // met zyne kinder.
      
      
        Heel Israel // leefden rebel
        Godes bevel // hij verdeed snel
        Haar met ghequel // zes honderd duysen
        D'een afgodt eert // de ander hoereert,
        Die derd' temteert // vierd' murmureert,
        Die vijd' begheert // menich abuysen,
        Zo verginghen ook al haar gheslachten en huizen:
        Maar Iosua en Caleb en de ander haar jonghen
        Zijn wel ghespaert // van God bewaard,
        Van bozen aard // zo Schrift verklaart,
        En zij vergaert // van vreuchde songhen.
      
      
        Al die niet vreest // in dit foreest
        Midts dat men leest // dat altijd meest
        Kwaad zijn geweest // zijn buiten ghesloten,
        Die God bemint // en recht begint
        Gheestelijck ghesint // hem zelfs verwint
        Blijft hij ontrint // haar medeghenoten
        Die vrome te bewaren heeft hem nooit verdroten:
        Die boze brengt hij tot straffe ook klaerlijck
        Nu 'tlicht doorbreckt // 'tvuyl wordt ontdekt,
        Nu ruymt die sect // Gods volk begeckt.
        En maakt suspect, gants openbaerlijck.
      
      
        Neemt hier op acht // hoe dat elk wacht,
        Met list bij nacht // bij daech met kracht,
        Gods volc met smacht // te brengen in sneven
        Die Salmists reen // tneghentich een,
        Acht dit niet cleen // troostet ghemeen,
        Allen den gheen // die haar begheven,
        Onder die bescherminge des almachtigen te leven
        En onder zijn schaduwe hem aldus aenroepen
        Mijn Heer mijn God // mijn burch mijn slot,
        Mijn deel mijn lot // wij Iacobs rot,
        . Na u ghebodt // op u wij hoepen.
      
      
        Niet een vergaet // int strik ghevaet
        Die hem verlanct // op Godes raad,
        Maar t'sijnder baet // zal men verheffen,
        Al viel int sant // aan u sijdts cant,
        Duizend ter schant // ter rechter hand,
        Tien duizend want // t'sal u niet treffen,
        Maar zult u lust zien hier beneffen,
        Ook hoed den Godloosen werd vergouden,
        V zal gheen schaet // door iemands daedt,
        T'eenigher maet // vroeg nochte spaet,
        Aencomen kwaad // wilt dit onthouden.
      
      
        Elk neem noch waar // gans openbaer,
        Dat bij Gods schaer // druck angst en vaer,
        Bangicheyt naar // vaeck is t'werven,
        Maar zulk verdriet // werd niet bediedt.
        Schriftuer aanziet // dat het gheschiet,
        Om haar tot niet // gants te bederven,
        Wij hadden besloten zeid Paulus te sterven
        Voor den druck die ons in Asien bedekte,
        T'welck ons gheschach // op dat elk mach
        Dencken die dach // doen hij aensach,
        Al ons gheclach // die de doden verwekte.
      
      
        Sulcken dood bank // in haar aenvanck,
        Stijf ende stranck // al duerdse lanck,
        God noch bedwanck // tot onze leer,
        Ghedenckt ook meed' // een wijze reed',
        Die Iudith deed' // in haar ghebeed',
        Ook t'hooft af sneed // van den Velt-heere,
        Tot die ouders des volks nam zij haar kere,
        Men mach den Heer tijd noch dagen aenreden
        Lijd' wij ghequel // vermaentse snel
        Exempel stel // Abraham wel,
        Isack Israhel // en Moyses veel leden.
      
      
        Ia allegaer // die God lief waar,
        Perijckel zwaar // kwam over haar,
        En bleven daar // trou zonder crakielen,
        Maar die in noot' t'zy cleyn of groot,
        Door elk aenstoot // oft valse boodt,
        Ook in den dood // van God vervielen,
        Ginck hij door den verderver vernielen,
        En zal hem noch aan zulke wreken,
        Daarom zo zult // gij met ghedult,
        Kennen u schuld // op dat Gods hult,
        In u vervult // nimmermeer ghebreecken.
      
      
        Al die God vreest // ghedachtich weest
        Dit zwaar tempeest // minst ende meest
        Hoe 't Davids gheest // dus gaat belyden:
        Ik prijs u Heer // dat gij mij weer
        Verhooghet zeer // door ws-naems eer
        Mijn vyandt neer // niet laat verblyden;
        Doen ik u aanriep Heer gij waert t'mijnder zijden
        En ghinckt mijn siel voor't verderven bewaren:
        En hebbet mij // levendich gij
        Behouden bly // daar alle zij
        Ter Hellen zij // zijn neder ghevaren.
      
      
        Komt Heyl'ghen teer // lof zingt den Heer
        Dankt prijst zijn eer // want zijn toorn zeer
        Die duert gheen meer // maar ons verheuchde
        Den avond lanck // gheduerde stranck.
        Het wenen bank // maar nam een ganck
        Smorgens vrij vranck // zo kwam die vreuchde:
        Maar ik sprac doent mij wel ginck in deuchde:
        Ik zal nimmermeer ter neder vallen;
        Want Here goed, door u behoet
        Met rechter spoet // hebt gij mijn moet
        Als een berg zoet // sterk maakt met allen.
      
      
        Och doen gij dijn // heylich aenschijn
        Berchde voor mijn // och watte pijn
        Tot dien termijn // verschrikte ik seer;
        Och here God // wij Iacobs rot
        Waren tot spot // zonder u lot
        Na u ghebodt // wij smeecken den Here
        Wat is er nut aan mijn bloed nu mere?
        Want na mijn dood zal u 'tstof ook dancken:
        Here toch hoort // na u accoort
        En heylich Woord // zo helpt ons voort
        Tot aan dien oort // daar wij na jancken.
      
      
        Boven dit al // mijn plaech misval
        Veel in't ghetal // meer dan ik kal
        Hier in dit dal // hebt gij verandert,
        In rey verhoogd, want gij vermoocht
        En mij uit toocht // den zak verdroocht
        Begort niet vroocht // als een vast standert
        Op dat al die in deuchde wandert
        Vwen lof singhen met eerbare zeden;
        En uwen eer // toch nimmermeer
        Stilswyghen weer // zo zal ik zeer
        Dancken mijn Heer // in eeuwicheden.
      
      
        Wie dit nu vaet // en meent hij staat
        In rechter maet // in deucht voortgaet
        Ziet toe met raad // dat hij niet  valle
        Niet stout van gheest // nu zijt maar vreest
        En zelver leeft // hij is geweest
        Streng over meest // die natuerlijcke alle
        Die poort is eng' die weg is small,
        Hij zal ons niet ontsien noch sparen
        Lucas derthien // gaat wel bezien
        Van deze twien // Galcese lien,
        En noch achtien // die tot Syloa waren.
      
      
        Zo wie gheen boet // oprecht en doet,
        Hem neerstich spoet // altijd int goed,
        Die selfde moet // zo gaan te gronde,
        Leeft noch voort daar // hij zeide haar,
        Doen allegaer // een byspil zwaar,
        Op dat elk claer // die zake verstonde,
        Een had een vijgeboom geplant hij vermonde
        In zijn wijngaard daar hij vruchten of sochte,
        En vanter gheen // sprak hij ghemeen,
        Met cloecke reen / drie jaar aan een,
        Met kwade zeen // gheen vruchten hij brochte.
      
      
        Dus hout hem of, dat wilde stof
        Is veel te grof / te hebben t'lof,
        In s'Heeren hof // als vruchtbare bomen,
        Merckt hoe ik las // hij antwoordt ras,
        Laat hem dit pas // staan als hij was
        Ende in cas // gheen vruchten en comen,
        Na dat ik hem neerstich heb waar genomen
        Met graven en messen na t'recht der vromen
        Daar na met vlijt // maakt hem u kwijt,
        Zonder respijt // op dat gheen tijd
        Elk ypocrijt // met bladeren staat roemen.
      
      
        Ei neemt nu waar // elk ghetals jaar,
        Zeven en vijftich daar // twee sessen claer,
        Twee achten zwaar // wat gingher te gronden
        Komt nu iemand // roemrijck ter hand
        En zeide want // zij zijn te schant,
        Ook al van cant // door hare zonden,
        En wij zijn noch oprecht bevonden,
        T'blijckt wij staan in des Heren gaerde,
        Daar stond hij meed // verhaelt alreed,
        Drie jaar men leed // noch een ter been,
        Vervult een steed // maar niet en baarde.
      
      
        Dat wij noch staan // den dau ontfaen,
        Is niet gedaan // om ons recht gaan,
        Maar God ziet aan // hoe wij ons bekeeren,
        Die nu vertraecht // t'zy knecht oft maagd,
        Ionck oft bedaecht // God niet behaecht,
        Maar van hem jaecht // na Salomons leren
        Daarom laat ons die lanckmoedicheydt des Heren
        Voor onze meeste zaligheid betrachte
        Wie niet is cloeck // in elken hoeck,
        Die crijcht een vloek // dit zelfs na soeck,
        Int propheets boec // Ieremias veertich achte.
      
      
        Iammer waert groot // sou gene noot,
        Noch zwaar aenstoot // hoe naakt en bloot,
        Ia veler dood // ons maken verleghen,
        Boven dit al  ons eyghen mis val,
        Bitter als gal // clinckt met gheschal,
        En't ons niet zal // meughen beweghen,
        Och vraagt na die voorgaande weghen,
        En wandeltse trou zo suldy ontvangen,
        Der sielen rust // in dien gij blust
        Stilt ende sust // des vleysches lust
        Die waarheid kust // al met verlanghen.
      
      
        Tot een besluyt // laat dit gheluyt
        Als een goed kruid // als Christus bruyt
        Niet een minuyt // uit hert en sinnen;
        Al u ghedacht // bedwingt met kracht
        V neerstich wacht // dach ende nacht
        Voor't Helsche gracht // om te verwinnen
        Als oft gij van nieuws aan wildet beginnen
        Dient God met vrezen verblijd u met beven
        Die Heer is zoet // barmhertich goed
        Ook metter spoet // toornich verwoet
        Want hij elk doet // zulk loon zulk leven.
      
      
        Memorie hout // op nieu en oud
        Christum betrout // hebt altijdt fout
        Dat u niet rout // 't eeuwighen tyden;
        Niet stout maar vreest // ghedachtich weest
        En zelver leest // hoe noch het beest
        Noch listich gheest // niet viert van stryden;
        Dus wilt gij eeuwich met God verblyden
        Omgort u lenden u licht laat branden
        En verlangt zeer // na uwen Heer
        Want hij komt neer // met groter eer
        Beveelt u teer // in zynen handen.
      
      
        Eer God // mijn Heer // het lot // ws eer
        Dus bot // sou weer // als spot // ter neer
        Tot elks begheer // duert kwaad allensken
        Sou zijn // ghewent // goud fijn, hoe jent
        Noch pijn // ellent // ten fijn // bekent
        Kwam mijn // ontrent // oft ik sou wensken
        Den dood te sterven // als alle mensken
        Om te verwerven // de ruste stilswyghen;
        Profijt // hier doet // zo ghy't // zijt vroet
        Altijd // tot boet // met vlijt // u spoet
        Verblijd // om 't goed // eeuwich te kryghen.
      
      
        Prins // wins // niet kwaad // suleys // eyes // noch raad
        't Welk // elk // aenvaet // maar // daar // toe gaat
        't Woord // hoort // verstaat // in zijn valeure
        Tot duecht // juecht // bly // tot goed // spoet // dy
        Ook oud // cout // gij // nieman // kan // vrij
        Gods Gheest // leest // hij // sell // wel // ter keure
        Bespien // deursien // ellick zijn labeure;
        Ons dagen // jaghen // snellijck // na te lest
        Dus u tijd // slijt // waeckt bidt strijd // mijt
        Som u smijt // lijt // ghedoocht spijt // nijt
        Noch verblijd // zijt // diet is die rest.
