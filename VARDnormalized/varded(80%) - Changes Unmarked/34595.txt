
    
      
        EI zegt ons lieven vriend // wie gij dient,
        En tot wie gij het meest zyt ghesint?
        Strydtbaer knechten // moeten vechten,
        En zetten 't zweerdt op sné  // voor Condé,
        Daarom ziet lieven vriend // wie gij dient,
        En tot wie gij het meest zyt ghesint.
      
      
        Gaat buigt den Franschen neck // met een treck,
        En verschaft hem een lustighe leck,
        Op dat Licht-voet // synen plicht doet;
        En ziet wie hij verjaagd // en mis-haeght,
        Gaat buigt den Franschen neck  // met een treck,
        En verschaft hem een lustighe leck.
      
      
        't En is niet al ghenoegh // spaed' en vroeg
        Enen Prins zo te zyn tot misnoegh,
        Wiens laurieren // 't land door-zwieren,
        En voor gheen blixems gaan // uit de baen,
        't En is niet al ghenoegh //  spaed' en vroeg
        Enen Prins zo te zyn tot misnoegh.
      
      
        Treedt Leopoldus vrij // op syn zij,
        Want dien Heldt heeft een aerdtjen als gij;
        Vlecht u kranssen // als twee Hanssen,
        In een // tot Spagniens eer // en vermeer,
        Treedt Leopoldus vrij // op syn zij,
        Want dien Heldt heeft een aerdtjen als gij.
      
      
        Doet den on-rusten Haen // doch eens gaan
        Langs den kant van een vredighe baen,
        Dat d'Olyven // groeyen blyven
        Tot 't heil van 't Christen-bloedt // ende goed;
        Doet den on-rusten Haen // doch eens gaan
        Langs den kant van de vredighe baen.
