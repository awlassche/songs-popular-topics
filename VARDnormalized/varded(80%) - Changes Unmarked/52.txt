        Waent gij dat u ontrou // voor mij versust zal blijven.
      
      
        Neen neen ik weet het wel // wat dat gij hebt bedreven,
        Ook mee tot wat gesel // dat gij u hebt begeven.
      
      
        Gedenck die zwaren eet // die u en mij verbonden
        Te laeten 't geen gij weet // ik aan u heb bevonden.
      
      
        Was Frederick niet genoech // u weeldich lijf te stillen,
        't Scheen neen Want 't geen gij droeg // had noch een meerder willen.
      
      
        't Was Lucidor wiens brand // u redens kas zo sengde,
        Dat gij ons waerde bant // als glants vergeeten plengde.
      
      
        Is hij 't dan die u ziel // zo machtich heeft beseeten,
        Dat gij hem voor mij hiel // laat niet mij te vergeten.
      
      
        En vly, u met u brunt // ia te dickste der Bosschagie,
        Daar eensaemheydt u gunt // een middel ter boelagie.
      
      
        Of wel u lust betracht // in duyst're naar spelonke,
        Of in de swarte Nacht // als ieder lecht te roncke.
      
      
        Ga schuyl u vrij voor 't ooch // van redelijcke zielen,
        Doch 't geen gij nooit bedrooch // treet u steeds op de hielen.
      
      
        Een knagende geweet // zal u gedeurich quellen,
        En 't swaerste harten leed // zal u gestaech versellen.
      
      
        Waar heen dan wroegent hert // tot decking uwre daden,
        Stick in u eigen smart // om dree te zijn ontladen.
      
      
        Doch echter zal u ziel // voor onze ogen waren,
        Eer gij gedoemt de kiel // van Charon zult bevaren.
