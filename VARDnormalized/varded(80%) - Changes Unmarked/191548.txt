
    
      
        O Maagd, mijn Bruyt, zegt wat lust u te gaan?
        Lanck dralen, dwalen hier over s'werelts baen,
        In duizend lagen //
        Zo soetelijck wagen
        U Ziele, die mij heeft zo dier gestaan.
        Wie spreekt hier dus, en wondt heel mijn gemoet?
        En Jesu zijde gij het, die het doet?
        Die met u straelen //
        Mijn hert komt by-haelen,
        En trekken tot het ongeschapen goed.
        Wereld vlied //
        't Is al niet //
        dat gij mij kont geven:
        Jesus min //
        Heeft mijn sin //
        Nu tot hem gedreven.
        Wel hoe Maagd moet ik dan weg?
        Iae ge wereld, zo ik segh.
        Ziet wel dat't u niet en rouwt.
        Hoe kan't als mij Jesus trouwt?
        Die met hem paert //
        Is wel bewaard,
        'tIs een Lam zeer rijk en zoet van aard.
      
      
        Verlaat gij dan dit zoete aerdtsche dal,
        Vol Bloemkens schoon, en vol vreugden over al:
        Alwaar de Roosen //
        Zo fraey staan en bloosen,
        Daar ik u (zo gij wilt) med' kroonen zal.
        Neen wereld. Ik ken dien bedriegelijcken vond;
        Gij stryckt ons hier rechts het seemken aan den mond,
        En zoekt met streelen //
        Ons hert te steelen;
        Daar naar laat gij ons liggen zwaar gewont.
        Op u vreugd //
        Op u jeugd //
        Op u lieflijck praeten,
        Op u pracht //
        Op u kracht //
        magh m'hem niet verlaeten:
        Och men doet, dat is gemist.
        Iae dat ik niet beter wist.
        Houdt mij vrij van hogen schick,
        Om u slechten ogenblik //
        Eer men aanziet //
        'tGeen gij ons biedt,
        Is't weg, en verandert in verdriet.
      
      
        Wel aan, nu gij Maagd dit slechte Poppe-goedt
        Van s'werelts kraem, stoot en trapt het met den voet:
        Zo zult gij gaan wezen //
        mijn Bruyt uyt-gelesen,
        En hebben dat u lust ten vollen doet.
        Wat is dat Heer Jesus? Ei komt toch en beduyt,
        Wat voor Iuweelen gij ons wilt reiken uit,
        Ik moet mij verstouten //
        Wat samen te kouten;
        Want u spraeck gaat te boven alle geluid //
        Als gij lonckt //
        Zo ontvonckt
        Mij met menigh herte //
        met een woord //
        Iaeght gij voort
        Droefheyt, pijn, en smerte //
        Blijft mij bij, o Bruydegom.
        Zegt mij Liefste toch waarom //
        Zijn der niet wel duizend reen:
        Nu begint dan seght'er een.
        Waar toe ge'eet //
        Als gij wel weet,
        Wat'er in het hert verborgen leed?
      
      
        En hoe kan een ziele, die krachtig u bemindt,
        Uytspreken alle schatten, diese vindt //
        In dat schoon wezen,
        Waar uit komt geresen //
        Een vier, dat hier lancks meer en meer verslint.
        Hoort hier nu eens, voorwaar gelovet vrij,
        O Bruyt, gelijk u sinnen staan op mij,
        Zo is mijn wenschen //
        Te zijn bij de mensen:
        Haar aangenaam geselschap maakt mij bly.
        Jesu vriend,
        hoe veel dient,
        die hier voor ons beiden,
        Is't u sin //
        Ik begin //
        Ziet, laat ons nooit scheiden,
        Bij u is't een Paradijs.
        Bruyt, g'en zijt het stuk niet wijs.
        Neen. Ik weet wel wat ik kies,
        't Geen gij minst acht, is verlies//
        Volgt mijnen raad,
        God komt ende gaat //
        Ziet dat gij altijdt gelaeten staat.
      
      
        Wel Jesu ik zal doen na dat gij mij hier zegt,
        Het is mij goed zo te worden onderrecht //
        bestiert mijne wegen,
        En stort uwen zegen //
        In mijn hert, dat het blijft aan u gehecht.
        Al wie op mij zijn hoop vrymoedigh set,
        En daar bij zoekt t'onderhouden mijne Wet,
        Van alle zijden,
        zal ik hem bevrijden
        dat hem geen kwaad en let.
        Jesu zoet //
        ook ik moet //
        U hier op wat vragen;
        Ziet men niet //
        het verdriet //
        Meest u vrienden dragen?
        Die mij mint weet van geen kwaad.
        Hoe kom 't dat dit dan zo gaat,
        Bruyt zo thoon ik mijne kracht //
        Als die lijdt, geen lijden acht.
        Wel God en Heer //
        'k Segh nu niet meer,
        Stiert mij, zo gij wilt, tot uwer eer.
