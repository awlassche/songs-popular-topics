
    
      
        EEn Lied eerbaer // van de liefste claer
        Voorwaar, Wil ik beginnen,
        Want bij haar // Zo rust wel zwaar
        Eenpaer, met hart en sinnen,
        Zij is zeer rein, In sWeerelts pleyn,
        Men mach gheen schoonder vinnen,
        Want zij alleyn, Is mijn shartsen greyn
        Dus moet ik haar beminnen.
      
      
        Haar wezen fier // En weet ik hier
        Bij niemand te ghelijcken,
        En haar manier // Zeer goedertier,
        Heeft mij doen blijcken:
        Maar zeer rebel // Past zij in't spel,
        En mij gheen trou doet blijcken
        Al lijd' ik ghequel, door dit opstel,
        Ik zal haar niet ontwijcken.
      
      
        Dat zij mij boot // Haar lipkens root
        Mijn moet, waar gheresen,
        Door haar liefde groot, Ik zegt v bloot //
        Van die dood, Waar ik ghenesen,
        Om dat zij went, Van mij ontrent,
        Dus leef ik in vrezen,
        Maar mijn Liefken jent // Zeer excellent
        Moet altijdt zijn ghepresen.
      
      
        Princess goed, die mijn vlees en bloed
        Ja doet mij mineren,
        Om dat ik moet // V wezen zoet,
        Weest dit wel vroet // Dus lang ontbeeren,
        Nochtans ik zal // Tot mijn verhaal
        In liefde glorieren,
        Spijt nijders geschal // Liefd' boven al,
        v lief exalteren.
