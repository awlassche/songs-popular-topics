
    
      
        Schoon lief playsant,
        Dit nieuwe liet ontvangt
        in liefden triumphant,
        Zal ik v dienaar wezen
        Door Venus brand
        Ontgaet mij het verstand,
        Ontbint mijn dezen bant
        Gij muecht mij ghenesen
        mijn druck versacht,
        Gij hebt die macht,
        Door liefden kracht
        Schoon Lief daar ik na bey,
        Slaat op mij acht
        Verhoord toch mijn gheclacht
        Daar ik na wacht,
        In deze coele Mey.
      
      
        Waert gij eens vroet,
        Wat pijn ik lijden moet
        Gij sijt diet mij doet //
        Nochtans ik v beminne //
        Mijn Lijf, mijn Goed,
        Ja ook mijn t'sherten bloed //
        Is voor v Liefken zoet //
        Hoe wel ik niet en winne
        Maar voelde gij
        eens wat ik ly,
        Zo waar ik vrij //
        Van alle druck in dezen country //
        Dan zouden mij,
        Die fantasy,
        heel maecken bly
        In dezen toekomende Mey.
      
      
        Schoon Lief minioot //
        Ik zal ontdecken bloot //
        Mijn lijden zwaar als loot,
        Gij muecht mij verblijden //
        Troost mij tis noot //
        Mijn smart en pijn is groot
        Leuendich ben ik dood,
        Ik sterft tallen tijden //
        Want doen wij iet,
        Het werd bespiet //
        Al eer dat het gheschiet
        Door nyders ingheuen,
        Doet toch te niet,
        Dit zwaar verdriet //
        Welk gij al aan mijn ziet //
        In dezen toekomende Mey.
      
      
        Hoort en verstaat //
        Princess delicaet,
        Oft v ter herten ghaet,
        Dat gij mij dus laat wachten,
        Niet en versmaet //
        V dienaar vroeg ofte laedt //
        Het is uwen eyghen baet,
        Gij muecht mijn druck versachten
        Zegt niet meer neen //
        Maar ia alleen,
        Maakt van ons tween //
        Een paar in dit gheschraey //
        alle gheween
        Zal hem ontspreen,
        Van ons alzo ic meen
        In deze toekomende Mey.
