
    
      
        CLouck ingenieus //
        volmaectste aller flueren //
        Vroom, en gratieus //
        in gauen der natueren //
        Eenich boeten //
        en versoeten //
        daert al bij verheugt //
        u t'ontmoeten //
        ende groeten //
        is een zoete vreugd.
      
      
        Al wat zonder // u
        Geschiet, en can niet treffen:
        Ligh ik onder // nu,
        Gij cont mij haast verheffen,
        Bij gedoogen,
        Wilt mij thoogen
        In mijn wederspoet,
        V bruin ogen,
        Of verdroogen
        Moet mijn vlees, en bloed.
      
      
        Te  waar wel reden // dat
        Ik u van minnen / sprake:
        Als gestreden // zat
        Ik brandich binnen // blake //
        Wtgelesen,
        Hoog geprezen,
        Beeldeken geeert,
        V bij wezen
        Can genesen,
        T'swaerste dat mij deert.
      
      
        Hebt in handen // gij
        De macht, dat (naar t'volherden)
        Al does banden // mij
        Een vreugd konnen gewerden,
        Wijt vermaarde,
        Jonck geiaerde,
        Zijn mij (alst u lust)
        Een vermaarde
        Sauegaerde,
        Daar mijn herte rust.
      
      
        Als ik schade // loos
        Aan u vinde ontsetten,
        Alle kwade // boos,
        En konnen geenszins letten:
        Wat verscheyden
        Tongen zeiden,
        Acht voor schimp, en spijt,
        Wilt verbeyden
        V bereiden
        Dienaar, taller tijd.
      
      
        Ramp, en schade // dan
        Acht ik voor windigh blasen,
        V genade // can
        Mijn droufheyt versolacen:
        T'soet beluchten
        Can doen vluchten,
        Does bruinen zwaar
        Dat mijn duchten,
        Treuren, zuchten,
        Moet verdwijnen // claer.
      
      
        Int vertrouwen // vast,
        Van u deught-saem beloonen,
        Geen berouwen // wast,
        Van mijn gedienstich thoonen,
        Hadde konnen
        Zijn gewonnen,
        Van u, iet gesienst,
        Onder zonnen
        Waar begonnen
        Nooit geen meerder dienst.
      
      
        Neemt (O weerde // maagd)
        Dit danckbaer int gehele,
        Al wat d'eerde // draagt,
        Schinck ik tot uwen dele
        Daar beneven
        Lijf, en leven,
        Is u, metter daad,
        Toegeschreven,
        
          Overbleuen
        
        Niet, dat mij aangaat.
      
      
        Adieu vreden // borght,
        T'scheyden heeft nu oorzake,
        Voor u heden // zorgt
        Den geest, altijd in wake,
        Dit bedrijven!
        Laat nu blijven
        V contentement,
        Voor te schrijven
        Veel meessijven,
        Wensch ik tzijn present.
