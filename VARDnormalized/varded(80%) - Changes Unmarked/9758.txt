
    
      
        Myn Ziele sucht in ellende,
        Zo ik mijn elders waert wende,
        Oft keer // dan tot u Heer.
      
      
        Ik vinde mij ter Woestijne
        Daert vreesselijck is te zijne,
        'tDangier // is menighertier
      
      
        Int midden van baren en golven,
        Van Leeuwen en grijpende Wolven,
        Ben ik // in groot verschrick.
      
      
        Waar ik mijn voeten zette,
        Het is er al strik oft net,
        V baen // Heer, leert mij gaan.
      
      
        Door deze Werelts bosschagie
        Neemt yeghelijck zijn passagie,
        En jaagt, dat hem behaagt.
      
      
        Nemrod was hier te velde,
        Een Iagher groot van ghewelde
        Maar vinck // al ydel dinck.
      
      
        Wat vindtmen al Iaghers met hopen?
        Die hier met Esau loopen
        Int wout // wilt ende stout.
      
      
        Zo dat zij den zegen verliesen,
        Voor d'eerste gheboorte, verkiesen
        Den kroes // oft schotel moes
      
      
        Die hier versuymen Gods gracy,
        Met 'swerelts dominacy,
        Eylaes // die zijn wel dwaes.
      
      
        Wie zal mijn leydtsman wezen?
        Volgh' ik de Wereld in dezen,
        Verblindt, is zij ghesint.
      
      
        Ga ik het vlees te rade,
        Het is gheneghen ten kwade,
        Ten dient // mij niet als vriend.
      
      
        Volgh ik mijn eigen lusten
        Zo kom' ik in zwaar onrusten,
        De dood // in haren schoot.
      
      
        Consentere ik de zonde,
        Zo gheeftse mij zulk een wonde,
        Niemand // helen en kant.
      
      
        Waar op zal ik mij verlaten,
        Want die mij smeecken, mij haten,
        O God // weest gij mijn lot.
      
      
        Niet beter dan mijn betrouwen,
        Op u, o Here te bouwen,
        Mijn troost // weest doch altoost.
      
      
        O Here, verhoord mijn zuchten,
        Want ik en mach niet ontvluchten,
        Ten zij // dat gij helpt mij.
      
      
        Heb ik u, o Here vol waerde,
        Naar Hemel, noch naar Aarde
        Ik niet // en vraghe iet.
