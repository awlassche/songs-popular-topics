
    
      
        Weg Cupido, weg Venus kind,
        Vlied van mij duizend mylen //
        schiet elders uwe pylen,
        'k Ben anders ghesindt;
        Ik haat het minnen //
        en verkies de kan,
        In plaets van een Vriendinne //
        Bacchus is den man,
        Aan wie ik mij verpan.
      
      
        Een volle fluyt vol Rhynschen wyn,
        Die spoel ik door myn kele //
        dat verdryft het quelen
        Van der minnen pyn;
        In plaets van kussen //
        't Liefken aan den mond,
        Kan ik den brandt wel blussen //
        aan den roemers grondt,
        Oft 't gat daar hij op stond.

        bibat ter vel quater,
        Dat en valt niet strengh:
        Dan stracx aan 't smooren //
        't is een vreugd om zien,
        En een gheneught om horen //
        al does satte li'en,
        En raest zo veel als thien.
      
      
        Niet een zo schone Maeghden-bloem
        En sou mij konnen wecken //
        oft tot liefde trekken,
        'k Houd' mij aan den broem;
        Ik houd' van brassen //
        en verhef den dis,
        Daar lustig valt te kassen //
        en te schranssen is,
        Daar vind' ik laeffenis.
      
      
        Ik weende eertydts wel van min,
        Maar nu schier als heel beken //
        de bier-tranen leken:
        Langs myn dobbel kin;
        Kom ik te schuylen //
        in den hoeck des eerdt,
        Ik drink dat d'ooghen puylen //
        en druypen op de eerd',
        Zo wordt de borst ghesmeert.
      
      
        Komt dan gij nobel Knuyskens drinkt,
        Wilt Cupido verlaten //
        en Vrouw Venus haten,
        Bacchus ons beschinckt,
        Hij heeft syn tonnen //
        Vers voor ons ghevolt;
        Sa, 't suypen eens begonnen //
        dat elk suysebolt,
        En van de trappen rolt.
