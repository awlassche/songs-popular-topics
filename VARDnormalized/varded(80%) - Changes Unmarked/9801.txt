
    
      
        O Heer wanneer ik met aendacht
        De grote kracht, der liefden mercke,
        Die gij aent vervallen gheslacht
        Van Adam hebt ghestelt te werke,
        En hoe gij noch daar onder // doet
        Grote ghenade wonder // goed,
        Ik uwen Naam loven bijzonder // moet
      
      
        Wat wonderlijck verkleenen // slecht,
        Hebt gij oprecht, ghekroonste // Coningh,
        Bewezen dat gij enen // knecht
        Kwaamt zijn van uit u schoonste wooningh
        En naemt op u der zonden // last,
        Van mij arem ghewonden // gast,
        In lief' aan u ben ik verbonden // vast.
      
      
        Hoe laat gij slechte weten // bloot,
        Dat wise duizenden verborghen
        Is, al die schoon secreten // groot
        Ws wills, hoe lust u zo te sorghen
        Voor kleen verworpen scharen // milt,
        Dat gij haar openbaren // wilt,
        Het welc gij voor den zeer ervaren // stilt.
      
      
        Hoe spraeckt gij al int duister // maar
        De Wereld in ghelijckenissen,
        En niet met enen luyster // klaer,
        Moest iemand van u Rijke // missen
        De kennis? om zo hooghen // schat,
        Behoortmen wel te pooghen // dat,
        Men vint, al sou men daarom doogen // wat
      
      
        Zo wie begint, aan u bevel,
        En boete snel, doet goederhandigh,
        In u kennis en leering' wel,
        Zal hij worden allencx verstandig,
        Maar zonder dat, hij even // bot
        Blijft in het sondigh leven // zot,
        Waar in men niet en kan aankleven // God.
      
      
        Een zeggen int aenhooren // schrickt,
        Als of den Vader tegen // 'tghene
        Hij heren heeft te voren // schickt,
        Dat boze gaan haar weghen // hene,
        Doch den zaeyer van't kwade // zaad,
        Zijn zelven maar te rade // gaat,
        Geen bouman goed na eigen schade // staat.
      
      
        Den Huysvader voorwistet // ziet,
        Maar voorschickte nooit kwaad oft zonden,
        Dijn zaad willens gij quistet // niet,
        Al valet weinig in go gronden,
        Waarom? om dat tot dezen // zij
        Niet doen 't gheloove wezen // bij,
        Ghelijck gij wilt, om haar ghenesen // vrij.
      
      
        Gij sijt een Medecijn, en elk
        Roept gij en biedt ghesonde // gaven,
        Levende Water, Wyyn // en Melk,
        Maar der zonden ghewonde // slaven
        Weygheren laes te kommen // meer,
        Haastende na 't verdommen // zeer,
        In onschult zij moeten verstommen // Heer.
      
      
        Ziet, mist men u rechtveerdich, kleed
        Opt voorbeschicken Gods te // maken,
        Gheen onschult men dan weerdig // weet,
        Van onbereyt zijn, oft niet te // waken,
        Als men handen en voeten // bindt,
        Dus nu men tijd ter boeten //vindt,
        Al willens wij niet dwalen moeten // blindt
      
      
        Een plaatse van begin // bereid
        Is er, ja vant begin der // Wereld,
        Daar werden zij al in // gheleyt,
        Die met deuchden werden bepeerelt,
        Maar de ander werd ghebleken // hier,
        Int eeuwig fel ontsteken // vier,
        Och men schromt daar van te spreken // schier.
      
      
        Gods voorschicking', en willen // noch
        In eenderlei ten mach niet lieghen,
        Maar waar in zoudt verschillen // doch?
        Hij wil altijdt, zonder bedrieghen,
        Dat alle Mensen zouden // zijn
        Inder waarheid behouden // fijn,
        Och of zij al daar toe doen wouden // pijn.
