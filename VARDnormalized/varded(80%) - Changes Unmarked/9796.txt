
    
      
        O Broeders rein looft den Here gestade
        Die hier in't pleyn // laat bloeyen zijn ghenade,
        Als een Fonteyn // voor die afstaen van't kwade
        En voort alleyn // met Christum gaan te rade
        Die zo het graen // in go Aerd' ontfaen,
        Worden van dezen zade,
        Kinderen vroet // van God den Vader goed,
        Door zijns zoons bloed // vergoten overvloed
        Ligt onder voet // Dood en Helle vertreden
        Lof hem zijn moet // van zijn ghenade zoet,
        Wie boet // hier doet // zij staan noch heden.
      
      
        Eeuwig houdt standt // 'sHeeren goedheid ghepresen,
        Hij heeft geplant // zijn ghemeynt' uytgelesen
        Want triumphant // ten Hemel op gheresen
        Ter rechterhand // zijns Vaders court na dezen
        Zand zynen geest // met een winds tempeest
        Tonghen ghedeelt // int wezen
        Vyerigh en klaer // op yeghelijck van haar,
        T'ghetal aldaar // in synen naam te gaer
        Dat en was maar // ontrent twintig en honderd,
        Den Gheest hier naar // werkende wonderbaer,
        Heeft zwaar // de schaer // des Volks verwondert.
      
      
        Van zuyt oft Noort // uit menige landpale,
        Heeft daar gehoord // een yegelijck zijn tale
        Betuigen 'twoort // en Gods werken loyale
        Vrees en discoort // hadde 'tvolck generale,
        Want deze zijn // vol zoeten Wijn
        Ghedroncken altemale,
        Spraken rebel // eenigh' uit spot en spel,
        Op zulk vertel // antwoorde Petrus wel,
        Dits gheen voorstel // van droncke lieden,
        Maar doort bevel // Gods, na 'tprofeteersel
        Ioel // blijckt snel // hier dit gheschieden.
      
      
        O Prins, o Heer // u macht liet gij aenschouwen,
        Van leden teer // een sterk huis gingt gij bouwen
        Meer ende meer // lieten Mannen en Vrouwen
        Op uwen leer // haar dopen, met betrouwen
        Maar door den nijd // der Joden wijdt
        Verstroyt, door zwaar benouwen,
        V dienaars jent, vant nieuwe Testament,
        Tot 'swerelts endt // maakten u woord bekent,
        Heer excellent // lof prijs moet gij gewinnen,
        V woord ontrent // ons laat zijn ongeschent
        Present // gheprent // in onze sinnen.
