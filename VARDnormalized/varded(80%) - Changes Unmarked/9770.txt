
    
      
        MIjn Lief devijn // is mijn,
        En ik ben zijn // na dijn
        Lieflijck aenschijn // met pijn
        Verlangt mij, liefste greyn:
        Ghelijck een hert bevonden
        Dorstich na die fonteyn
        Gij zijt die mij van zonden,
        Wt ghenaden alleyn,
        Schoon hebt ghewasschen rein,
        Ende gemaakt ghemeyn,
        Al door u bloed // u goed,
        V loven vroet // ik moet,
        Dat gij mij doet // zo zoet
        Vloeien u gracy pleyn.
      
      
        O Minnaer groot // minjoot,
        Zoet Hemels brood // ter noot,
        Die van der dood // mij bloot
        Puer hebt ghenesen al,
        O lief, heilig, almachtig,
        Gij zijt die komen zal,
        Wilt mij wezen ghedachtigh
        Hier in dit Aertsche dal,
        Leidt mij op den weg smal,
        Bevrijdt hier u Schaepstal,
        Vwen Wijngaard // bewaard,
        V Bruydt vermaert // verklaart,
        Na uwen aard // ghevaert,
        Beschermt voor den val.
      
      
        Biedt mij u hand // vailliant,
        Brengt mij int land // playsant,
        Doet mij bystandt // lief, want
        Op u ik mij verliet,
        Wt mijn eyghen vermoghen
        Ben ik ghelijck een riet,
        Ik zoude zijn bedroghen,
        Waert dat ik dachte, ziet,
        Ik ben al wat of iet,
        Want ik en ben doch niet
        Dan een broosch vat // saen mat,
        Om gaan den pad // ter stad,
        Maar deugh' ik wat // mij dat
        Door uwe kracht gheschiet.
      
      
        Mijn toost gij zijt // altijd,
        Want mynen strijdt, int krijt,
        Zal in jolijt // subijt
        Veranderen all' gaer:
        Maar waerder gheen verrysen,
        Ghelijck als er is klaer,
        Mij mochte wel afgrysen
        Van mijn ellende, maar
        O ghetrouwe Minnaer,
        Ik weet wel openbaer,
        V armen staan // ontdaen,
        Om mij t'ontfaen, zeer saen,
        Dus wil ik gaan // u baen,
        V last en is niet zwaar.
      
      
        Tot u ik keer // o Heer,
        Krank zijnde zeer // en teer,
        Gheeft dat ik meer // u leer,
        Volghe zonder afwijck:
        Want als ik ga bedincken
        Die krone magnifijck,
        Daar ik mede zal blincken,
        Die klaer Zonne ghelijck,
        Al in ws Vader rijk,
        Dan acht ick't hier al slijk,
        Vleeschs, lust, ghewelt // en ghelt,
        Dat mij voor stelt // int veld,
        Satan den heldt // want 'tsmelt
        Al in een oogenblijck.
      
      
        O goedertier // Princier,
        Gij die met vier // zult schier,
        Tot haar dangier // alhier
        Straffen die boze stranck:
        En legghen u vijanden
        Tot uwer voeten bank,
        Daar knerssinghe der tanden
        Werd huylingh ende stank,
        O lief, boet mijn verlanck,
        Wilt mij behouden vranck
        Van't onbekent // torment,
        V exellent // en jent
        Aenschijn present // mij went,
        In vreugd zonder verganck.
