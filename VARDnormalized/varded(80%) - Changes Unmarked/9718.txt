
    
      
        HEt is een schoon dierbaer zake,
        Dat men make // onzes Gods Naam
        Groot met lofsanck ende sprake,
        Zoet van smake // is hij bequaem,
        Iae vriendelijck ende minsaem,
        Laat ons zijn faem // verbreyden wijt
        Looft hem te rechten // die zijn knechten // zijt
        Met groot jolijt,
        Looft den Schepper ghebenedijt,
        Die alle dingen heeft gemaakt subijt,
        Alle ere // tot hem kere // nu ende te aller tijd.
      
      
        Synen stoel is boven schreven,
        Hoog verheven // wijdt ontdaen,
        Veel duizend Enghelen beven,
        Die beneven // voor hem staan,
        Zijn gramschap saen, doet bergen vergaen,
        De diepten saen // verdrooghen vrij,
        Voor zijn aenschouwen // mans en vrouwen // gij
        Looft hem waert zij,
        Meerder dan alle lof is hij,
        God is over al, verr' en bij,
        In hem leven // ende sweven // allegader wij.
      
      
        Laat ons God een nieu lied singhen,
        Grote dingen // doet hij bloot
        Voor hem beeft swerelts omringhen,
        Ook springhen // die steenrotsen groot
        In stukken, voor zijn toornig exploot,
        Maar in den nood // is hij zeer goedertier,
        Die betrouwen // op hem bouwen // hier,
        Hij helptse schier
        Wt lijden ende groot dangier,
        Hij gaat met haar door water ende vier,
        Hij can merken // al ons werken, hij proeft hert' en nier.
      
      
        Hij weet al 'sMenschen ghedachten,
        Al haar trachten // en verstand,
        Klaer als dag zijn hem de nachten,
        Groot van machten // is zijn hand,
        Hij hoort en ziet wel alle dingen, want
        D'oore geplant // hebbend', als wijs engien,
        Soud' al voren // ook niet horen // dien?
        Zoude, door wien
        D'ooge gemaakt is, ook niet konnen zien?
        Hij weet alle dingen eer zij gheschien,
        Na Schrifts getuygen // voor hem buigen moet alle knyen.
      
      
        God is een Coninck ghepresen,
        Ende dezen // woont in een licht,
        Daar niemand toekomt gheresen,
        Zijn wezen // van Mensen ghesicht
        Is verborghen, hij heeft ghesticht
        Al het ghewicht // der Aarden zwaar,
        Zeer groot bevanghen // en gehangen // daar
        Opt water klaer,
        Met synen woorde krachtig ende waar,
        Ende hij onderhoudet ook all'gaer,
        Zijn vermogen // ongheloghen // blijckt wel openbaer.
      
      
        Al zijn werken groot bevonden,
        Die men gronden // niet en magh,
        Van zijn heerlijckheyt oorconden,
        Te allen stonden // enen dag
        Doet den anderen daar af ghewagh,
        Maar nooit en zag // iemand God vanden troon,
        Maar zijn verkoren // eengheboren // Soon
        Heeft ons zeer schoon
        Dat verkondight, lof prijs zij hem te loon,
        Die op den stoel zit, weerdig alle croon,
        Eeuwig ghepresen // moet hij wezen // en dat Lam ydoon.
      
      
        Doe wij Gods vijanden waren,
        Iae zondaren // boos en fel,
        Ginck hij ons zijn liefde baren,
        Zonder sparen // gaf hij snel
        Voor ons allen, in doots ghequel,
        Zijnen zeer wel // beminden Zone zoet,
        Die ons in noode // met, zijn rode // bloed
        Verloste vroet,
        Ziet, wat een liefd' ons God de Vader doet
        Dat hij ons als kinderen zijn goed
        Maect deelachtig // hem eendrachtigh // eeuwig lof zijn moet.
      
      
        Zijn ghenade onverdwenen,
        Is verschenen // over al,
        Tot profijte vanden ghenen,
        Die recht henen // in dit dal
        In Christo wandelt den weg smal,
        Maar het en zal // na Schrifts bediet,
        Den obstinaten // sondaer baten // niet,
        Hem en gheschiet
        Gheen ghenade, maar onghenade, ziet:
        Toorn, angst, druck, en zwaar verdriet,
        Wie te voren // synen tooren // hier niet en ontvliet.
      
      
        Tot quijtschelt van't sondigh sneven,
        Is ghegheven // die geloofd,
        Iesus den name verheven,
        Tot ons leven // ende Hoofd,
        Des vijands macht heeft hij berooft,
        Blijft niet verdooft // wie oren heeft die hoort,
        Wilt ontvangen // met verlangen // t'woort,
        Rein onghestoort,
        Schickt u te gaan door d'enghe poort,
        Met liefd' en broederlijck accoort,
        Brengt den Name // Gods bekwame, vrucht der lippen voort.
      
      
        Vreest God al gij Mensen teere,
        Gheeft hem ere // want het naakt
        Nu den tijd zijns oordeels seer,
        Na zijn leer // altijdt waeckt,
        Aanbidt hem, en niet en versaeckt,
        Die heeft gemaakt // Hemel en 'tAertsche veld,
        Zee en Fonteinen // dezen reinen // Heldt
        Zij toeghestelt
        Eeuwich lof, ere, prijs ende ghewelt,
        Gij Conincklijck Priesterdom, hier vertelt
        Vrij met vreuchden // al zijn deuchden // al wordt gij ghequelt.
      
      
        In Gods lof wilt niet verkrancken,
        Christen ranken // onversteent,
        Wilt in alle dingen dancken,
        Spijs en drancken // hij verleend,
        Gheen Musch en wordt voor hem vergeten neent,
        Sorgt noch en weent // niet, om dat vlees aangaat
        Maar wel gade // Gods ghenade // slaat
        Daar in bestaat,
        Met vaster herten, blijft vroeg ende laat
        Bij den Here, volgt zijnen raad
        Om niet verdolen // hem bevolen // blijft Chrsitelijck zaad.
