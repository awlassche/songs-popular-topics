
    
      
        DEn coelen Mey
        Staat Lieffelijck voor ogen
        In does Country
        Betreden wordt de baen
        In elk valley
        De Zonne gaat verhooghen
        Om tsoet gheschrey
        De Minnaers buiten gaan
        Met haar gespelen in Lieffden schoon ontdaen
        Om therte te vermaken //iet
        Zij bij malcander raken // ziet
        Zulk vreuchde is te laken // niet
        Want t'mach in Lieffd' bestaan.
      
      
        Menich jong paar
        Liefflijcken om vercieren
        Wand'len te gaer
        Al langs den watercant
        Zien zwemmen daar
        Viskens in de Rivieren
        Gaande voorwaar
        Vriendelijck hand aan hand
        En openbaren haar Lieffde vol verstand
        Therte dat zwaer te sorghen // plach
        Door Lieffd' die lang verborgen // plach
        Haat ende nyt verworghen // mach
        Door Lieffde abondant.
      
      
        De weiden staan
        Heel cierelijck ontloken
        Met groene blaen
        T'dor hout becleedet is
        Der Nymphs paen
        Staan Lieffelijck vol roken
        Deur bloemkens saen
        Die Flora sent ghewis
        Die gaat verchieren Der Amoureusen dis
        Daar zij om drucx vergheten // rat
        Zoet Lieffelijcken eten / wat
        Bacchus schenckt onghemeten // t'nat
        Der Lieffden zoet en fris.
      
      
        Aurora vocht
        Door shemels dans besproeyen
        Een zoete locht
        Brengt aan het firmament
        De jeucht bedocht
        Zich haestelijck gaat spoeyen
        Met handen vlocht
        Veel Cranssen excellent
        Van Rode Roosen / en bloemkens pertinent
        Phoebus het Loof voortspruyten // doet
        In kracht en vol virtuyten // zoet
        Zo dat de jeucht na buiten // vroet
        Hun daghelijcken went.
      
      
        Prinslijcke rein
        Minnaers ende jonckvrouvven
        Int Nymphs pleyn
        Vermaeckt u allebey
        Gaat ter Fonteyn
        Vyt Lieffdes gloet beschouwen
        Hoort zo ghemeyn
        Des Voghels zoet gheschrey
        Want uwe vreuchde / groeyt op deze valey
        Laat Lieffdes lust ontspringhen // bly
        Wilt Lieffelijcken singhen // vrij
        Rein Lieffd' wilt niet bedwinghen // gij
        Trou Lieffgens in den Mey.
