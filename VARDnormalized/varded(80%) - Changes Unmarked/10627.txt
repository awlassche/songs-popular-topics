
    
      
        CVpido God, deur uwe minne stralen,
        V scherp ghebodt // doet mij in liefden dwalen,
        Ik ben versodt
        Op een bruin smaechdelijn smalen
        Die met mij spodt.
      
      
        Venus bestier // vint ik zeer dubbelt van treken
        Der minnen vier // heeft zij in mij ontsteken
        Die liefste Rosier
        Ic machse zien noch spreken,
        t'is groot dangier
      
      
        T'is groot verdriet // de Liefste moet ik derven,
        Die trouwe biet // en niet en can verwerven
        Om dat zij v liet,
        Moet ic van rouwe sterven
        Int meer geschiet.
      
      
        Haar zoet samblant // was mij een medecijne
        Zeer triumphant, Plach ik bij haar te zijne
        Die liefste playsant
        laat mij in grote pijne
        Aan mijn verstand.
      
      
        Als ic aensach // haar vriendelijcke ymagie
        Nacht en dach // schiep ik in haar coragie
        Dies ic wel mach
        clagen om haar vysagie
        Zonder verdrach.
      
      
        Zonder confoort, laat zij mij nu in weyne,
        Dus wij accoort // hielden int ghemeyne
        Dus roep ik moord,
        Over mijn lief alleyne
        die mij verstoort.
      
      
        Adieu eylaes // Adieu Princess schone
        Ik was te d'waes // Te vrijen v persoone,
        Dies ik nu raes
        Dat ik mij zelven zo hoone,
        op dit relaes.
