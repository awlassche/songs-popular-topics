
    
      
        Almachtich God, komt mij te baten,
        Gheeft mij in mijn herte te vaten
        Wat u // doch nu // heer doet behaghen
        In mijne dagen.
      
      
        En hout mij doch in uwer vresen,
        Wat mijn doen en laten zal wezen,
        Dat gheeft // beleeft // mij doch nu spoedich,
        O Here goedich.
      
      
        Christus u lieve kind zal comen
        Te rechten die boze en vromen,
        Voorwaar // aldaar // zal hij verclaren
        En openbaren.
      
      
        Het ghen’ dat hem eer doet bewijzen,
        Is die hongherighe te spijzen
        Met brood // in noot // om zo te werken
        Wilt mij verstercken.
      
      
        Troulijck den Armen te bedincken,
        In dorst te laven, hem te schincken,
        Met troost // altoost // zijn hert te laven
        Gheeft mij die gaven.
      
      
        Blijd’lijck die gasten te logieren
        In mijn huis, Here goedertieren,
        Arm, rijk, ghelijck, tot elker uren
        Al zonder trueren.
      
      
        En wilt mijn herte zo bereiden
        Om nu den naakten te becleyden,
        Met moet // en spoet // hem te verhoghen
        Na mijn vermoghen.
      
      
        Noch meer die krank ligghen in hoeken
        Gheeft mij lief Here te bezoeken,
        Met raad // en daad // soo’t is gheleghen,
        Haar vriend’lijck pleghen.
      
      
        En stiert mijn herte met verlanghen
        Tot hem die swaerlijck is gevangen,
        Dat mijn // const fijn // hem mach verblijden
        In al zijn lijden.
      
      
        Dat ik hier bidde iin dit singhen,
        Laat aan mijn ziele ook volbringhen,
        Spijst, drinkt // en schinckjt // haar cleeder schone
        Wt uwen Throone.
      
      
        In u herberghe wiltse leiden,
        Om d’eeuwighe rust ye verbeyden,
        Mijn zeel // geheel // met dezen zaken
        Wilt zo vermaken
      
      
        Krank leit mijn ziel, wilt tot haar comen
        O lieve Here uytghenomen
        Ghesont // terstont // maakt ze gherade
        Door u ghenade.
      
      
        T’ghevanckenis wilt Here breken,
        Daarin mijn ziele schier bezweken
        Leit vast // den last // wilt mij verlichten,
        Hier uit oprichten.
      
      
        Van vreuchden zal mijn hert opspringen
        Als ik bevind’, lief Heer, does dingen,
        Voor al // ik zal // u God hier boven
        Dancken en loven.
      
      
        Zo wil ik dezen sank besluyten,
        Crachtighe God, groot van virtuyten,
        Staat bij // gheeft mij // dit al te samen,
        Lief Here, Amen.
