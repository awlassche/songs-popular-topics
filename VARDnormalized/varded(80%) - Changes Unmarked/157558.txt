
    
      
        O Heer God Hemels Vader // goed
        Die rechte Fonteyn-ader // zoet,
        Sijt gij zo Schrift oorcont
        Hier in de Wereld rond
        Zijt mijn zonden ontlader // vroet
        Want ik als een misdader // moet
        Bekennen mijne zondt
        Voor u uit sHerten gront
        Die daar in onbetamen
        Van mij wel zijn geschiet
        Dies moet ik mij nu schamen
        Voor u Heer diet al ziet
        Van in mijn jeucht // zo had ic vreugd
        Tot alle kwaad, gedreven // milt
        o Heer vol deucht // maect mij verheucht
        Mijn schuld vergeven // wilt
      
      
        Gij hebt bereid een schone // stad
        Den weg is ongewone // wat
        Te gaan, vol van dangier,
        Van water, en van vier
        O Here wilt mij jone // dat
        Zo den Verloren zone // bad
        Om vergevinge hier
        O Vader goedertier
        Want groot zijn mijn misdaden
        Die u Heer zijn bekent
        Vader vol van genaden
        Hoe was ik zo verblent
        Als ic weg ginc // zonder bedinck,
        en mij tot boze werken // gaf
        Maar u geschinc // was kleed en Rinc
        Beter dant Vercken // draf
      
      
        Den Herder naert behoren // riep
        doen ik lange verloren // sliep
        Gelijk een Schaep verdwaelt
        Van Gods Rijke gefaelt
        Ik int zondich versmoren // liep
        En lag zo in Gods toren // diep
        Maar u liefde doorstraelt
        Heeft mij weder gehaelt
        Wat zijn wij arme mensen
        Dat gij ons zo gedinckt
        Al watmen goeds kan wenschen
        Wt genaed gij ons schinckt
        tot ons welvaert // u niet en spaert,
        Hebt ons met uwen bloede // root
        Gekocht vergaert // sijnde bezwaert
        Al met armoede // groot.
      
      
        Prinse alder Princieren // dan
        Koning diet al regieren // kan
        In Hemel en op Eerdt
        Gij sijt wel prijsens weert
        En loff, die ons zo deer // wan
        Al komt ons qua maniere // an
        Keert van ons dat ons deert
        dit wordt van ons begeert
        Om niet meer zo te ondegen
        Te loopen voort en voort
        Maar hier op uwe wegen
        sterckt ons doch met u woord,
        Dwelc is u spijs // goed en propijs
        Tot onzer Ziel bekwame // fijn
        O Hertoogh wijs // vol loff en prijs
        Moet uwen name // zijn.
