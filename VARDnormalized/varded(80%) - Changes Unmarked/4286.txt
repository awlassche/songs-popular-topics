
    
      
        RAs neemt vertreck  gij duister nacht //
        Licht op o dach van vre- den,
        Die heden // zal hier bij // den rey,
        Beschijnt nu de  lieflickste dracht //
        Volmaect van lijf en le- den,
        Vreugdich bringen den blij // den // Mey,
        Haast u gij jongers komt inne //
        Voor mijn schoon Coningin- ne.
      
      
        Eeuwich blijven verwonnen // gans
        Haar vijanden verseerlick:
        Haar aenschijn (als der zonnen // glans)
        Blinct in den morgen heerlick,
        Eerlick // draagt zij ter wereld, hier
        Klederen net bepeerelt // fier,
        Een gulden tak draegh ik wt minne,
        Voor mijn schoon Koninginne.
      
      
        T'haerwaerts is geheel t'lant // gehelt,
        Om breken fantasie:
        En Orpheus heeft zijn hand // gesteld
        Tot spel, vol melodie:
        De blie // Bergen, springen // hoog,
        De Nachtegael int zingen // vloogh,
        Met liedekens, zoet van gewinne,
        Voor mijn schoon Koninginne.
      
      
        Ontdect u, Eesters velden // wijt,
        Sent herwaarts Violieren:
        Ontspringht die cloecke helden // zijt,
        Croont haar met schoon Laurieren:
        Vercieren // wilt u feestich // dan,
        Elk roepe als een geestich // man,
        Buigt Iuno hemelse Goddinne,
        Voor mijn schoon Koninginne.
      
      
        Rieckende bloemkens abundant,
        Verlicht nu de contreye,
        De liefste coemt zeer triumphant,
        Plucken den blijden Meye:
        Elk spreye // cruyt, en bladen // cleyn,
        Op haar wegen, en paden // rein,
        Thooght u Flora, o gesellinne,
        Voor mijn schoon Koninginne.
      
      
        In dezen Mey vol iunsten, bly:
        Schinct haar schone presenten:
        Een lid der seven cunsten // vrij,
        Is zij niet om verjenten,
        Haar tenten // ik vercieren // moet,
        Coemt maagden met manieren // zoet,
        Verschijnt elk als een Cherubinne,
        Voor mijn schoon Koninginne.
      
      
        Koningen, Vorsten, Prinsen // wreet,
        Buigt u, voor mijn Deesse:
        Alle Landen, Provincen // breet
        Houden haar als Princess,
        Expresse // liefkens onder- // set
        Den Mey, voor mijn lief zonder // smet:
        Gemerct ik haar altijd bekinne,
        Voor mijn schoon Koninginne.
      
      
        Adieu, edel gecroonde // ieught,
        Rethorica met name:
        Neemt doch deze gethoonde // deugd,
        T'uwer eer' zonder blame:
        Eersame // doch faveurich // zijt,
        V dienaar, die doleurich // stijt,
        Zo houd' ik u met hert, en zinne,
        Voor mijn schoon Koninginne.
