
    
      
        O Galathea, mijn zoete Goddinne,
        O schoonste Nymph van dit wout,
        Waarom is ’t dat gij u weder-minne
        Van mij zo langh af-keerigh houdt?
        Daar gij van binnen // uwe sinnen wel ziet
        Dat u van Phyllis,, nooit gril is geschied,
        Daar ik van landen // van panden, van wey,
        Ben on-gelijcker // en rijcker dan gij.
      
      
        Ei! rijcken Phyllis, wat zal hier af komen?
        Houdt vrij uwe Ossen en Schapen op stal,
        Al komt gij over u kudde te roemen,
        Daarom ik u niet minnen en zal.
        Hierom dan Herder // niet verder en denckt,
        Eer gij mede // u reden verkrenckt:
        Want ieder Maget // bestaget zo vrij
        Yemandt met sinnen // te minnen als dy.
      
      
        Stadigh ontfangh ik u trotsige reden,
        Wat ik u oyt heb voor-geleydt?
        Gelijk gij over dry dagen geleden
        Een arrem kusjen mij hebt ontseydt,
        Als Amarillis, naar Phyllis vertreck,
        Met uwen lust // u kuste uit geck,
        Daar ik gave // als slave mij hand,
        Dat ik mijn leven // sou geven tot pand.
      
      
        Is dat, ô Phyllis! uwe eerste ontmoeten,
        Dat gij mij voor belooningh thoont,
        Dat Amarillis mij zelve kwam groeten,
        Dan een soentjen mij heeft gejont?
        Daar ik doch immer // hem nimmer en zag,
        Hoe soud’ ik konnen // hem jonnen een lach?
        Daarom, ô Phyllis,, mijn wil is dat gij
        Laat does gepeysen // nu reizen op zijd.
      
      
        Heb ik u Galathea doen spreken,
        ‘k Bidd’ u neemt dat niet qualijck af.
        O neen, ô Phyllis! maar wildy u wreken,
        Omhelst mij, dat is voor een straf.
        O Herderinne // uwe minne beleeft
        Maakt dat u Strijder // wat blijder nu leeft.
        Herder volherdy // zo werdy bemindt,
        Want het verblijden // het strijden verwint.
