
    
      
        MEt een versuchten // zwaar //
        Moet ik beclagen //
        Mijn drouven staat ellendich::
        Ik en weet vluchten // waar //
        Veel duizend plagen //
        Draegt mijn jonck hert inwendich:
        Zeer schendich // al mijn sinnen //
        Van binnen //
        Zijn eylaes verstroyt //
        Den camp der minnen //
        Heb ik gaan beginnen //
        Daar mij t'overwinnen //
        Falgeer- de oyt.
      
      
        Een eloquent // ieught,
        Wil mij niet haesten,
        Siel, ende lijf, verteren:
        Met excelente // vreugd,
        Waend' ik ten laatsten,
        Met haar te triumpheren:
        Doleren // dus onspoedich
        Int gloedich
        Ethna, is den noot:
        Haar handen bloedich,
        Geven overvloedich
        In mijn hert mismoedich
        Wonden ter dood.
      
      
        Zal nu te gronde // slaan
        Mijn vaste hope,
        In dit treurich gemoeten:
        En niet te monde // gaan
        Troostbaer Cirope,
        Om mijnen druck versoeten:
        Mijn bloed // en // al mijn crachten,
        Versmachten:
        Dat een yder deert,
        Nochtans mijn clachten
        Wil zij geenszins achten,
        Maar hout mijn gedachten,
        Gans verturbeert.
      
      
        Van haar moet mij // nochtans,
        Den troost verschijnen,
        In dit malheurich // strijden,
        Hoe wel dat zij // bycans,
        (Met duizend pijnen)
        Mij doet dus treurich // lijden,
        Dien blijden // dach der vreden,
        Moet heden
        Lichten, met Jolijt:
        Dat den voorleden
        Strijd, eens zij volstreden,
        Daar ik om gebeden
        Heb langen tijd.
      
      
        En heeft nooit stout Baroen
        Dien strijd verwonnen,
        Zonder liefs assisteren,
        Hoe zal ik iongh plantsoen
        Enigszins konnen
        Victorieus floreren:
        Ter eeren // moet die schone
        
          Persoone
        
        Mij seluen present
        Scepter, noch krone
        En crijgh ik te loone,
        Voor dat zij haar thoone
        Als assistent.
      
      
        Rustich, beleeft, Princier,
        Die vroom, en vlijtich,
        Strijd om v schoon Princess:
        En acht al u dangier
        Te zijn profijtich,
        Een amoureuse less:
        Noblesse // tooght die puere,
        Figuere:
        Prijsweerdich geacht:
        Op auontuere,
        Zij (t'uwen favuere)
        Den prijs willecuere,
        Die gij verwacht.
