
    
      
        KOmt hier; bij mij vergaert;
        Komt, die ik heb gebaard
        In den Heer // wonder teer // door mijn leeringh,
        Die Gh' in u hart bewaard:
        Vervordert hier uwe gang,
        Aenhoort does zoeten zang,
        Neemt die aan // van De Swaen // wiens regering
        U zoet viel jaeren lang.
      
      
        Gij zijt die ik bemin
        Naast God meest van 't begin,
        Tot het end // zo bekent // is u alle,
        Gij zijt mijn hart, mijn zin.
        Ik draegh U op dit werk,
        U, Le'en van Godes Kerck,
        Die mij sint // die mij mint // zonder galle,
        G'lijck dag'lijcx ik aenmerck.
      
      
        Met Venus vuyl fenijn
        Vervult veel Boecxkens zijn,
        Die aan kant // uit de hand // gij moet leggen,
        Of gij smaakt de helsche pijn.
        Neemt aan dit Boek van mij,
        't Welk is van vuylheyd vrij;
        't Minste Lied // dat gij ziet // zal u zeggen
        Dat ik de vuylheyd vly.
      
      
        Niet als van eerbaerheyd,
        Niet als van suyverheyd,
        Handelt hier // dit papier // ('t is waeractich)
        En 's levens heyligheyd.
        Tot deughd' ik u vermaen,
        Tot God doe ik u gaan;
        Te allen tijd // 'k bidd' U, zijt // toch gedachtich
        Does liefelijcke Swaen.
      
      
        Vaart wel, de deughd bemindt,
        Aan God u vast verbindt;
        Vliet het kwaad, en verlaat // vuyle Liedtjes,
        Tot suyv're zijt gesint.
        Begint den Hemel hier,
        Zingt Liedtjes met playsier;
        Ei! zijt all' // op dit dal // zoet als Bietjes;
        Volgt 's hemelrijcx manier.
