
    
      
        O clagen een paar // En zuchten zwaar
        Met groter vaer // Mijn harte doorboren
        Mijn Lief eerbaer, Dien ik voorwaar,
        Bouen goud zeer claer Had wtvercoren,
        Valt mijn rebel, O Venus fel //
        V wreet op stel, doet mijn gequel
        Door dit bevel // blijf ik verlooren.
      
      
        Der minnen kracht, Met groter macht
        Compt mijn onsacht // Zeer fel bestrijen
        Al mijn ghedacht // Dach ende Nacht,
        Heeft zij voorpacht // Tot allen tijen,
        Zij is cortuees // Lustich en prues,
        Zeer Amoreus // En gratieus,
        Haar wezen heus // mijn Hart verblijen.
      
      
        Haar ionge ieucht // Nooit meer ver vreugd
        Sietmen in deucht // Lieflick floreren
        t'Welck mijn verhuecht, met zoet geneucht
        En tHarte veucht // Om haar te eeren,
        In swerlts troon // Spant zij die croon
        Wel duizend done // Crijch ik te loon
        Zal ik die schoon // moeten ontberen.
      
      
        Zal ik met schant // Des diamant,
        Zeer triumphant // Nu moeten laten
        Venus playsant // Goddin vailgiant
        Der minnen brand // mach ik wel haten.
        Zij is certain // die Liefste greyn,
        Dit mijn alleyn // Verheucht ghemeyn
        O vrouwelic greyn // Coemt mi te baten.
      
      
        Princess fiet // Zoet van manier,
        Al mijn dangier // Doet gij weg stake
        V zoet bestier // Vrouwelick rosier
        Der minnen vier, Int hart doet blaecken
        Met v eenpaer // Schoon Lief eerbaer
        Hoop ik voorwaar // Zonder beswaer,
        Mijn leven claer, Teynde te maken.
