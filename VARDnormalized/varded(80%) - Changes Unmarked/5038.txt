
    
      
        Lief uytverkoren // mijns herten vreugd,
        Wt God geboren // vol van gheneucht,
        ‘t Inwendich wezen // met uwen douw,
        Kond gij ghenesen // van druck en rouw,
        Gij laevet mijn // met Melk en Wijn.
        Gheen Medecijn // men beter wenschen sou.
      
      
        Ben ik armoedigh // van tijd’lijck goed,
        Zeer overvloedigh // mijn Ziel gij voedt:
        Moet ik hier derven // des vleys ghemack,
        Uwe Conserven // zijn mijn gherack:
        Word ik versmaedt // door nijd, en haat,
        Gij geeft mij raad // hoe ik zal dragen ‘t pack.
      
      
        V wijsheydts Reden // mij onderwijst,
        Door de ghebeden // mijn Ziel gij spijst:
        Gij komt mij leren // tot aller stond,
        ‘k Hebb’ maar te keren // in ‘s herten grondt,
        Want van ‘t begin // waert gij daar in:
        Gheen soeter Minn’ // ter wereld oyt men vond.
      
      
        ‘t Is waar, ‘s vleys lusten // vertonen schoon,
        Maar met onrusten // komt haren loon:
        Des Gheestes minne // blijft altijdt staan,
        Iae met ghewinne // zal zij op gaan,
        Ghelijck een vier // zo brandtse schier,
        Wanneer ze hier // niet en werd uytghedaen.
      
      
        ‘t Is niet te spreecken // hoe dat een hert
        Hier wordt ontsteken // als ‘t niet en werd
        ‘t Voedtsel ontooghen // door valschen schijn,
        En dat de ogen // gheen strikken zijn,
        Als men de wacht // houdt voor ‘t ghedacht,
        Den Geest voelt kracht // die anders leyd in pijn.
      
      
        Prins toont u krachten // in ons ghemoedt,
        Dat de ghedachten // het rechte goed
        Moghen aenhanghen // en vinden rust,
        Voor het ontvangen // van d’aerdtsche rust:
        Blijft gij ons bij // en wij bij dy,
        Door gheen party // en werd ons Liefd geblust.
