
    
      
        Niemand volkomen // Moses Wet
        En maakt // maar noemen // altemet
        Men haar wel hoort
        Enen Dienst, die 't verdoemen
        Maar predikt, of een donker woord.
      
      
        Christi voldoening // ons verlicht,
        'tWoordt der versoening // opghericht
        Is nu, God danck,
        Den Wijnstok gheeft vergroening,
        En vochticheydt, zijn vruchtbaar rank.
      
      
        Voor die ghelooven // salicheydt
        Der zielen boven // toegheseyt
        Is daar den schat,
        Gheen Dief en kan beroven,
        OCh vrienden, wat ghenaed' is dat.
      
      
        Deze ghenade // wijt vermaert,
        Is nu zeer spade // gh'openbaert,
        Voor ieder man,
        Die Profeten ghestade
        Hebben daar veel ghesproken van.
      
      
        Wij moghen horen // op dit pas,
        'tGhene te voren // langhe was
        Verholen stil,
        Hoe dat, vrij van Gods tooren,
        Magh zaligh worden elk die wil.
      
      
        Maar wat des Heren // grote schoon
        Ghenaden leren // elk persoon
        Komt in dit dal,
        Dat is, dat men afkeeren
        Van alle 'swerelts lusten zal.
      
      
        Sober, rechtveerdig // en voor God
        Wandelen weerdig, na 't ghebodt,
        't Godloos bedrijf
        Versaken, en volheerdigh
        Den Heer altijdt aenhanghen stijf.
      
      
        Stijf aan te houden // is zeer goed,
        Want niet verkouden // int ghemoet
        De liefde magh,
        Oft onbekent wij zouden
        Zijn, voor den Heer in zijnen dag.
      
      
        Ghenaed' ontvangen // ydel waar,
        Bleven wij hanghen // niet aan haar,
        Maar vielen af,
        Ons dorste niet verlanghen
        Na 'sHeeren dag, en oordeel straf.
      
      
        Zo uytghelesen // tijd, niet meer
        En zal der wezen // tis zo zeer
        Ghesworen dier,
        'tGhenadigh woord na dezen,
        Hoe zeer ghesocht en vindtmen hier.
      
      
        Daarom verkoren // Christen hoop,
        Maakt dat verloren // dijnen loop
        Doch niet en zij,
        God daar gij uit geboren
        Zijt, blijft altijts in liefden bij.
      
      
        Does ongheringhe // liefde groot,
        Moet onderlinghe, klaer en bloot,
        Blijcken ghewis,
        Elk een wie hoort of singhe
        Zoekt altijdts een dat noodigh is.
