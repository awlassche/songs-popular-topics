
    
      
        ALs d'Harders haar Schaepjes bewaeckten, zo scheen
        De Stralen // te dalen // op d'Aerde beneen,
        Den Engel vertelt // de Harders in 't Veld
        Gods Soon is geboren, dus maakt u te been.
      
      
        Dus loofden en songhen de Enghelen rein,
        Treedt vaerdigh // 't is waerdigh // gij Harders gemeyn,
        Verhoord onze stem // te Bethlehem,
        Daar zult gij vinden dat Kindeken klein.
      
      
        De Hardertjes liepen, en lieten haar Vee,
        Met singhen // en springhen // vol vreugde al-ree;
        Dus liepen zij al // en vonden de Stal;
        Daar brochten zij Iesus veel soetigheydt mee.
      
      
        Daar zagen zij Iesus bij Beesten geleid,
        Ghewonden // en vonden // als haar was gezeid:
        Doen ginghen zij weer // en loofden den Heer;
        Zij hebben Gods grote ghenaden verbreydt.
      
      
        Nou laten wij als deze Harders bij nacht,
        Ons leven // begheven // om houden go wacht,
        En soecken Gods Rijk // op dat wij ghelijck,
        Door Iesus, Gods Kinderen worden gheacht.
