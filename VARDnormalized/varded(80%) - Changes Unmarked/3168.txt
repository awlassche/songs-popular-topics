
    
      
        God verlaat zijn Dienaars niet,
        Schoon hij haar laat proeven veel verdriet:
        Na den druck // geeft hij haar weer geluk, Heil en lust // troost en
        Heil en lust // troost en zegen, En de Hemelrust.
        Op't Pinxter-feest,, bleeck'et meest // aan de Geest,
        Aan de tael // van'er juychen en verhaal;
        't Harte sprong // en de tongh // vrolijck
        zong Godes lof // met een stijl en hoog achtbre stof.
      
      
        O! wat vrolijckheydt en vreugd,
        Wat vermaeck heeft haar de ziel verheugt,
        Toen de wind // over haar kwam gheswindt,
        Kwam gedaald // en 'er zielen heilig heeft bestraelt.
        't Volk zeer verbaasd //heeft geraest // en geblaest,
        En gezeid // 't is de Wijn die haar verleydt:
        Maar een deel // waren veel // in't geheel
        Zeer bevreesd // door 't gehoor van'er grote Geest.
      
      
        Maar zij zeiden overluydt,
        't Is gheen drank die onze lipp' ontsluyt;
        't Is geen most // die 't verstand maakt verbost,
        't Is gheen sap // van de Druiven, maar 'tis wetenschap,
        Die Godes kracht // en zijn macht // heeft gebracht
        In ons hert // dat'et vol van tonghen werd;
        God is zoet // God is goed // die dit doet
        Tot u baet // na zijn Wil en Al-wijsen raad.
      
      
        Daar vloeyt in uytheemsche Tael
        Voor het volk een Goddelijck verhaal,
        Dat'er ziel // en 'er hart wel beviel,
        En dee staan // datse vraagden, acht! wat dient gedaan?
        Zij zeiden: doet // in ootmoet // recht boet,
        Valt den Heer // voor zijn heyl'ghe Voeten neer,
        En dan voort // alst behoort // op zijn woord
        Laat dan toe // damen u't Heilig Doopsel doe.
      
      
        Yder, als dit was gezeid,
        Was terstondt tot dezen dienst bereid,
        En't ghetal // groeyde strack dat 'et al
        Kwam terstondt // op drie duizend/ die in't Heyl-verbondt
        Waren ghetre'en // onder een // wel te vre'en,
        Wel gherust // met een ziel- en harten-lust.
        Heilig God,, leidt ons tot // u Ghebodt
        Ook met vreugd // dat wij zo blijven in de Deugd.
