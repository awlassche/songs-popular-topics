
    
      
        Komt ter feesten // suyver gheesten
        Gods dienaren groot,
        Die daar boven // in Gods hoven
        Zijnen lof verbreyt // tot alder tijd.
      
      
        Helpt ons singhen // grote dingen
        Van een suyver Maagd,
        Die ons’ leven // heeft ghegheven
        Brenghende hier voort // Gods eeuwig woord.
      
      
        Hoog ghepresen // uytghelesen
        Maagd en moeder rein,
        Naar u wenscyhen // alle mensen,
        V zoekt groot en cleyn // in d’aerdsche pleyn.
      
      
        Laat doch dalen // uwe stralen
        Der bermherticheyt,
        Wilt u ogen // naar ons booghen
        Keert van ons het kwaad // o toeverlaet.
      
      
        Heel deurwonden // deur de zonden,
        Maakt ons al ghesont
        En u daeden // vol ghenaeden
        Toont ons al ghemeyn // als medicijn.
      
      
        Al ons crachten // en ghedachten
        Zijn tot uwen lof,
        Elk een prijst // en bewijst
        V allerhoogste eer // rein maghet teer.
      
      
        In de sure // laetste ure
        Als men sterven moet
        Bij ons staat // metter daad
        En neemt in u behoet // Maria zoet.
