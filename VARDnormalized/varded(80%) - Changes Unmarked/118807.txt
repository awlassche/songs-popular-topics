
    
      
        JEsus Godes Zoon // Wiens verheven Troon,
        Aan Gods rechterhand // Eeuwig is geplant.
        Na gij hemel, aard // Krachtig hebt herbaard,
        Daar u alle' die leven // De eer' af moeten geven.
      
      
        Want gij sterke Held // Heb door uw geweld
        Duivel, zond, en dood // Gants van kragt ontbloot
        Als u 't boos gespuis // Heeft gedood aan 't kruis,
        Daar gij hingt te bloeden // 't Menschdom all' ten goeden.
      
      
        Dies nu de Eng'len schaar // Al te zamen haar,
        Voor u, haren Heer // Willig buigen neer,
        Ia op uw gebod // Als haar Hoofd en God,
        Willig met verblyen // Alle dienst toewyen.
      
      
        't Gansche menschendom // Valle des alom
        Neer op uw bevel // Voor uw voet-schabell',
        Om dat gij uwe bloed, Achte niet te goed,
        Voor haar weg te plassen // Tot het zielen-wassen.
      
      
        O! gekruisten Heer // Liefdens zucht loopt veer,
        Dat gij met uwe bloed // 's Mensen ziele voed,
        En als offer-lam // Haar zond' op u nam:
        Door uw dood is 't leeven // Onze ziel gegeeven.
      
      
        Geef nu, Iesus, Vorst // Dat elks ziele dorst,
        Met een liefde-kracht // Die de dood veracht,
        Dat elk met u ly, En ter dood toe stry,
        Op dat zij ons 't leven // Mocht voor eeuwig geven.
