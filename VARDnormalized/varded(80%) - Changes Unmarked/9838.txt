
    
      
        TObias tot sterven gheneghen,
        Riep synen soon, en sprak daar tegen,
        Mijn kind // bemint,
        Hoort met accoorden // alle mijn woorden.
      
      
        Als God mijn ziel heeft wechghenomen,
        Mijn lichaam in een graf laat komen,
        En wilt // ook milt,
        V Moeder draghen // jonst al u dagen.
      
      
        Houdt haar in en weerden end' in eeren,
        Denckt opt perijckel en 'tverseeren,
        Hoe zij // droegh, dy,
        Met grote smerte // onder haar herte.
      
      
        End' als zij zal ghestorven wezen:
        Begraeftse nevens mij, bij dezen,
        Altijd // met vlijt
        En neerstigh poogen // hebt God voor ogen
      
      
        Draagt in u herte God almachtig,
        Zo dat gij hem altijdt ghedachtigh
        Zijn meugt // van jeugd,
        Tot 'slevens enden // t'hemwaerts u wenden.
      
      
        Ziet dat gij niet t'eenighen stoned
        V en bewillight in gheen zonde
        T'ghebodt // van God
        En wilt versteken // noch gheensins breken.
      
      
        Van uwen goede helpt den armen,
        Aanziet hen-lieden met ontfarmen,
        Bij dien // aanzien,
        Zal u ghestade // God met ghenade.
      
      
        Helpt neerstigh die behoeftigh leven,
        Hebt gij vele // wilt vele geven,
        En doet // ook goed
        Met herten reynigh // al hebt gij weinig.
      
      
        Gheeft trouwelijck al zonder sparen,
        Zo zult gij rechten loon vergaren,
        Ter nood // die dood
        En zonden lossen // de alemossen.
      
      
        De Aelmossen vol charitate,
        Is in der noot een grote bate,
        En troost // altoost,
        Zeer groot beseven // voor God verheven.
      
      
        Mijne zone wilt u ook vermijden,
        Van hoerery, laat die besijden,
        Hebt een // Wijf reen,
        Wilt daar beneven // gheen meer aankleven.
      
      
        De hooveerdy en gheeft gheen stede
        In u herte, noch woorden mede,
        Van in // 'tbegin,
        Wracht zij 'tbederven // daar wij om sterven.
      
      
        End' alle die om u arbeiden,
        Terstont wilt haren loon bereiden,
        En houdt // niet stout,
        Van gheen personen // verdiende loonen.
      
      
        En doet ook geen mensen geschieden,
        Dat gij nood' had van ander lieden,
        Ter nood // u brood,
        Deelt te allen tijden // die hongher lijden.
      
      
        Wilt om den naakten te bedekken,
        Van uwe klederen aantrekken,
        Maar meest // diegheest-
        Lijck zijn begraven // deelt uwe gaven.
      
      
        Met den Sondaeren vol vermeten
        Zo en wilt ook drinken noch eten,
        Maar gaat // om raad,
        Den Wijzen vraghen // tot allen dagen.
      
      
        Danct altijdt God, en bidt eendrachtigh,
        Dat hij u wil regieren krachtig
        Dat fijn // al dijn
        Voornemen blijcke // zijn woord ghelijcke.
      
      
        En zorgt niet, mijn Zone verheven,
        Al leiden wij een arem leven,
        Voorwaar // hier naar
        Wij hebben zullen // van als te vullen.
      
      
        Indien dat wij tot allen tyden
        God vrezen en de zonde myden,
        En goed // doen vroet,
        Na recht wij zouden // dit wel onthouden.
