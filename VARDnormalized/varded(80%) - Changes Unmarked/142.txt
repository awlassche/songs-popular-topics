
    
      
        Van achtentachtich, Het nieuwe iaer
        dat is warachtich, Heel wonderbaer
        Diet can geloouen // Het zal gheschien
        Dat blinde douen // Horen noch zien
        En tisser gezeid // Dat menige meyt
        Zonder bescheyt // Zal na een vryer vlien.
      
      
        Dus weest vrij vrolijc, Ghi knaepkens eel
        Oft bent gij een molic // Tis even veel
        Men zal v vryen // Om tgroote goed
        En ook somtijen, Om hoge moet
        Dus hout v wat fris, Tgeluc onwis,
        Waar datse doch is, Diet lach betalen moet
      
      
        Wij consenteeren // De meysgens schu
        Dat zijt met eeren, Gaat vragen nu
        En mogen vryen // Wie dat haar lust
        Wil hijt niet lijen // Dat zij hem cust
        Zo blijft hij in schant, Die onachtbaer quant
        En crijcht int lanc // Geen lief die bij hem rust
      
      
        Princess schone // Tis nu v beurt
        schenct hem die crone, Daert hert om treurt
        Het moet doch wezen // Dat men bemint
        Dus wilt niet vresen // Wie eerst begint
        Die liefste en eer // verachtmen niet meer
        Maar maectse weer // An beide zien blint
