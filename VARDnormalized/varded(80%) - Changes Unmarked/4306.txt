
    
      
        NObel divijn //
        Wezen sanguijn //
        Prijs naar ge- dienstich poogen,
        Deurluchtich fijn //
        Als Christalijn //
        Rayen u Claer bruin ogen,
        Lichten als twee //
        Sterren ter zee //
        Die mij mijn wee // verthoogen //
        Ondancx ik moeds gedoogen.
      
      
        Edel gezet
        Staat lijf // en ledt,
        T'mach zulk een herte vromen:
        Was Venus net,
        Gij meugt u met
        Veel meerder recht beromen:
        Nooit en sach man
        Volmaecter an,
        Al watmen can genomen,
        Gij draagt in al t'volcomen.
      
      
        Enig gewin,
        Die van t'begin,
        Waert cause van versijcke,
        Maar niet te min,
        Herte en sin,
        Prent u in mijn Chronijcke,
        Tot veel gepijns,
        Geen medecijns
        En weet ik dijns // gelijke,
        Op geheel aerden-rijcke.
      
      
        Leven, en dood,
        Vreugd, wederstoot,
        Staat in dijn welberaden:
        Gelijk als brood
        Bluscht hongers noot,
        Condy mijns drucx versaden,
        Dus welgemint,
        Soethertich kint,
        Weest mij doch int // beschaden,
        Een voorbeeld der genaden.
      
      
        Kies' ik niet best
        In dit arest,
        Aflaet van mijn doleren,
        Hier aengevest,
        Volgt een request,
        Om u te remonstreren:
        Den suppliant,
        Versoect bystant,
        Wilt metter hand // schriberen
        Fiat, naar t'supplieren.
      
      
        Een sterk bewijs
        Van lof en prijs,
        Zal u bij dien gewerden:
        Schoon ieughdich rijs,
        Gestaan propijs,
        Ik moet u naarder terden,
        Waar mij ontseyt
        Dijn mogentheyt,
        En gaaft afscheyt ten derden,
        Noch soud' ik t'stuck volherden.
      
      
        Niet el schoon lief,
        Heyscht dezen brief,
        Bij mijnder hand geschreven:
        Doen ik besief,
        Dat al mijn grief,
        Bij u mocht zijn verdreven,
        Heb ik mijn leed,
        Voor u ontcleet,
        Dus wilt gereed // doch geven,
        T'behouden van mijn leven.
