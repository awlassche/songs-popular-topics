
    
      
        WYck, wijck, alle Wereldts schat,
        Met al 't gheen dat uwe prael omvat,
        Met u Goud // hoe verciert, en gebouwd,
        Hoe bepraelt // en hoe konstigh, en hoe schoon bemaelt;
        Hoe machtig rijk // 't is als slijk // in 't gelijk,
        Van de Jeugd // raer vereent, met witte deugd;
        Die u naam // heel bequaem // op de Faem
        Henen voert // daar altijdt yders oogh na loert.
      
      
        Waerde pronck van 't ruchtb're Spaer,
        Aerdigh Beeldt, Ã´ die gheduerigh waar
        Bij het licht // bij de glans, van 't ghesicht,
        Van de gleur // die het alderdighste Hart boord deur;
        Die met 'er strael // en onthael // menigmaal
        Pordt mij vaeck // dat ik na u schoonheydt haeck;
        Dat ik sta'egh // alle daegh // even graegh
        In mij vin // grote treck steeds tot u Goddin.
      
      
        Want uwe schone lieflijckhe'en,
        En uwe stem, en pit vol zoete re'en,
        Houdt mijn Gheest // stille staan, heel bevreesd,
        Dat ik dick' // dus mijn bede na den Hemel schick:
        O goede Goo'n // watten kroon // wat al schoon,
        Wat al zoet // heeft does Maagd in overvloed:
        't Schijnt u kracht // en u macht // heeft gewracht
        In dit Beeld // daar geheel uwe gunst in speelt.
      
      
        Wat al blos en aerdigheydt
        Dat er is op 't heel lijf ghespreydt,
        Kan mijn tong // hoe volmaakt datse zong
        Niet ten deel // openbaren, en noch min geheel.
        Ay! zoete Bloem // dien ik roem // die ik noem,
        Dien ik toon // voor het allerschoonste schoon;
        Schiet een lonck // schiet een flonck // en een vonck
        In mijn Hart // dat ik smeul in de zoete smart.
