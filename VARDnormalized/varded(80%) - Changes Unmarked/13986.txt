
    
      
        EEn roede groen // uit Jesse Stam gesproten,
        's Winters saysoen // wij 't vruchtbaar hout genoten,
        Wiens vrucht is spijs // om de ziel te verquicken,
        Heylig en wijs // met noch meer ander sticken,
        Die door zijn kracht // levendig overvloeyen,
        In ons gebracht // als rancxkens op te groeyen.
      
      
        Dit vruchtbaar hout // staat aan een schoon Reviere,
        't Brengt twaalf fout // vruchten van waerden deer:
        En op een Jaar // alle maanden zijn ooghsten
        Voor al die daar // van te rapen begosten
        Zijn geresolveert // voor alle vremde qualen
        Ongesommeert // wie kan zijn deugd betalen.
      
      
        Wie hier de smaak // van zijn vrucht heeft gekregen,
        't Geeft hem oorsaeck // om te meer zijn genegen:
        't Is in den mond // boven spijze der sterken,
        Het maakt gesont // in den Lust-hof te werken:
        Men wordt daar door // voorsichtig en kloeckmoedigh,
        En om wel voor // 't kwade te zijn verhoedigh.
      
      
        Prins uwen lust // zij in Gods Wet gestadigh
        Vierigh gerust // in Gode zeer genadig,
        Gebruikt dit hout // tot een staf in u handen,
        Het maekt u stout // te gaan door de waranden
        Van 't Jammer-dal // daar men veel verdriet merkte,
        Het leit u al // van d'een tot de ander stercte.
