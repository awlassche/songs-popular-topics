
    
      
        O Christus lief minnare // zoet,
        Die mij schinckt u dierbare // bloed,
        Na u verlangt mij te allen uren,
        Zo den Hert sijnd' in var // doet,
        Na de Fonteine clare // vloet,
        In dorstich pijnlick besueren,
        Ik leve in sware dolueren
        Hoe langhe, Lief, zal dit ghedueren
        Wanneer zal ik uit rouwen, zijn,
        Wanneer zal ik aenschouwen, dijn
        Aenschijn, schoon boven naturen?
      
      
        O lief, in s werelts wijcke // niet
        Van mij in gheen versijcke // vliet,
        Maar helpt mij, dat mijn teere leden,
        Niet en buuyghen ghelijcke // riet,
        Maar van u zoete Rijke, ziet
        Mij hier in dit dal beneden.
        Want ik vinde mij bestreden,
        Van den Prinse der duysterheden,
        De wereld in te verstranghen // kwaad,
        Alle dat nydich Slanghen // zaad,
        En laat mij niet met vrede.
      
      
        Jonstich voor mij gheneghen // hecht,
        Leidet mij in uwe weghen // recht,
        door uwen Gheest, o Lief verheven,
        Want ziet ik ben bedeghen // slecht,
        Een teken daar elk tegen // zegt,
        Voor raserye acht men mijn leven,
        En ik nochtans becleven
        Wt uwe zyde zonder sneven,
        U lief, u Bruydt, u Kercke // reen
        Ik doe door u, O sterke // steen,
        Die Helsche poorten beven.
      
      
        O Prinse der Princieren //jent,
        Gij die herten en nieren // kent,
        Aenhoort mijn bitter claghen,
        Uwen Gheest goedertieren // zendt,
        Dat hij door zijn regieren // went
        Mijn ledekens na u behaghen,
        Want te zijn nu kwade dagen,
        Hierom, o Lief moet ik u vraghen,
        Zal ik van al dit lyden // vrij,
        Met u niet haast verblyden // mij,
        En s Levens Krone draghen?
