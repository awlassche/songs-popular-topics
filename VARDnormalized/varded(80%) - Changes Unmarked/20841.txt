
    
      
        OCh hoe schemert mijn gesicht!
        Wat is dit? wat zeid dit grote Licht,
        't Welk komt aan // als een klaer held're Maen
        Hier ter Land // 't welk (zo 't schijnt) hier stoken komt een brand.
        't Is Willibroord // die Goods Woord // zo 't behoort
        Preecken komt: 't is een Man in deughd vernomt.
        Goods gebod // O! wat lot // brengt Hij tot
        Onze pael // hier toe roept Hij ons altemaal.
      
      
        Ons geeft Eng'land dezen Man:
        Ziet! Hij komt, Hij gaat voort wakker an;
        Als een pijl // vlieght Hij voort, met een bijl
        In de hand // Hij wil 't kwaad hier helpen aan een kant.
        En met de mond // wil terstond // Hij de zond
        Straffen gaan // waar nie'e wij zijn zeer bela'en:
        Hij niet traag // maar zeer graegh // predikt staegh
        Godes Wet // Hij bevrijdt ons van Satans net.
      
      
        O wat groten Yveraer
        Heeft ons God gegeven tot een Va'er!
        Die met vlijdt // ende sorgh, ons bevrijdt
        Van de Hell' // die nooit ophoudt met zijn loos gequel.
        Does Man ons leert // en regeert // en begeert
        Dat wij all' // komen tot des Hemels wall'.
        Wijst Hem eer // valt ter ne'er // voor dien Heer,
        Die met lust // soecken komt onzer allen rust.
      
      
        Wilbroord heeft den waren God
        Ons gepreeckt, en Godes zoet gebod:
        Ons gebaard // aan onze God, en bewaard
        Heeft dien Held // en gebroken heeft Hij 'thelsch' geweld.
        Does is den een // dien ik meen // anders geen,
        Die geleerd // Holland heeft, en eerst bekeert.
        Hij geleid // Hij bereid // Hij verbreydt
        Heeft het pad // 't welk ons brengt tot des Hemels stad.
      
      
        Wat wilt gij noch hebben meer
        Holland? spreekt, zegt uit toch u begeer.
        Weet gij niet // dat door Hem // gij geniet
        Het geloof // zonder 't welk gij waert des Duyvels roof?
        Wilt dit verstaen // neemt dit aan // van dien Haen,
        Die dit kraeyt // die Goods Woord in u nu zaeyt.
        Gij en kent // de ellend // zonder end,
        Niet ontgaen // 't en zij gij dit Geloof neemt aan.
      
      
        Denckt dan wat gij schuldig zijt
        Aan Willibroord, Holland! en weest verblijdt,
        Wijst aan Hem // met u hert, met uwe stem,
        Grote eer // die bekeert heeft u tot God wel eer.
        Spant met mij t'saem // wilt de naam // en de faem
        Maecken groot // van die ons help uit de dood.
        Gij hebt stof // geeft Hem lof // die Goods hof
        Nu bewoond // die zo rijckelijck is geloont.
      
      
        Alderwaerdste Godes vrind
        Sint Willibroord! Die metter hart ons mindt:
        Slaat van hoog // op ons all' toch uwe oogh,
        Ziet hoe wij // door de zonden zijn in grote ly.
        Vrijdt van den brand // ende schand' // door U hand,
        Uwe Gemeent' // die nu bitter tot U weent;
        Helpt ons voort // Willibroord! // en verhoord
        Ons gebe'en // hebt meedoogen met ons gesteen.
