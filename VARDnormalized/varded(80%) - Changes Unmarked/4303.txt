
    
      
        CVpido god der minnen // stout //
        Gij // geeft mij //
        Int hert menich wonde,
        En in bedwanck mijn sinnen // hout //
        Dat // ik mat //
        Ben versmacht te gronde,
        Nooit en conde // mij ontmoeten //
        Troost voor een verslegen // moet //
        Ander ionde // gij t'versoeten //
        Mij, veel drucx en tegen // spoet.
      
      
        Als gij te mywaerts wrimpich // ziet,
        Lijdt // en strijdt
        Mijn ieught overstreden:
        Gij zijt die mij zo schimpich // liet
        Van // elk man,
        Als een worm vertreden:
        Bij wat reden // hebdy voren
        Mij te doden zonder schuld:
        Laer verbeden // zijn dijn toren,
        Naar u goddelick gedult?
      
      
        Tuwaerts moest te allen dagen // oyt
        Mijn // aenschijn
        Hem subject verschonen:
        Nochtans gij mij veel lagen // strooyt
        Ach // ik mach
        Zuchten voor t'beloonen:
        Om zo schoonen // kind der eeren
        Lijd ik veel, die mij can doen
        Deur haar Croonen // triumpheeren,
        Als deurstreden Campioen.
      
      
        Heb ik Apollo dan // geslacht,
        Met // opset,
        V geschut verschoven:
        O neen ik niet! wie can // de // kracht
        Gaan // weerstaen,
        Diet al gaat te boven?
        Prinsen hoven // u geboden
        Nemen met groot wonder // aan:
        Dies u loven // alle goden,
        En u macht zijn onder- // daen.
      
      
        Al lijd ik dan een smerte // wreet,
        Lof // hier of
        Moet u zijn gezongen:
        Venus (t'moederlick herte // heet)
        Heeft // beleeft,
        Vwen schicht deurdrongen:
        Hoe bedwongen // gij elk slave,
        Int prisoen van zuchten leidt:
        Als ontsprongen // wt den grave,
        Geeft mij voetstaens liberteyt.
      
      
        Reden? beveelt met billich // recht
        V // alls nu
        Vroom te adoreeren:
        Ik dan, als een goetwillich // knecht,
        Naar // d'Altaer
        Wil ik t'aenschijn keren:
        Supplieeren // daar soetmondich,
        Om gehoor, tot mijn propoost,
        En begeren // zeer hertgrondich,
        Eens in liefden t'sijn getroost.
      
      
        Ist dat u gramschap neder // daalt,
        Is // gewis
        T'eynden mijn vernoyen:
        Mijn aenschijn bleeck // dan weder // haalt
        Een // zeer cleen
        Schoon rosierich bloyen:
        Weeldich groyen // is voorhanden,
        Al was ik zeer straf // gevaen:
        Kerkers, boyen // ende banden,
        Zijn subyt mij af // gedaan.
      
      
        Naar dat gij (als machtich // princier)
        Slaat // en laat
        Wederom genesen:
        Regiert deur u crachtich // bestier,
        Haar // eerbaer
        Princesselick wezen,
        Die geprezen // ieught bepeerelt
        Heb ik wtvercoren // ras,
        Wtgelesen // nooit ter weerelt
        Soeter kind geboren // was.
