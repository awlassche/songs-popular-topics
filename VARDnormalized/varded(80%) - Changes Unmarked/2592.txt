
    
      
        O Droeve tijd // die ik verslyt
        Met vruchteloos beclaghen
        O aertsche Hel // dood mijn ghequel,
        Door t'snoeyen van mijn dagen.
      
      
        Ik ben alleen // verd van de geen
        En tegen danck gebannen,
        Helaes! van die // ik 't liefste zie
        Van alle jonghe Mannen.
      
      
        Ach ongheluck // ach leide druck
        Zult nimmer gij ophouwe,
        Dit hart ghewelt // dat nu dus knelt,
        Does teer benaude Vrouwe?
      
      
        O schraele dood // komt en door stoot
        Does vrouwelijcke Borsten,
        Ach! strengh gemoet // dat zo verwoet,
        Na u Kinds bloed gaat dorsten.
      
      
        Mijn Moeder wreet // doet mij dit leed
        En gij versufte Grijser:
        Een Keysers kint // om dat het mint,
        Besluytmen straf int Ijzer.
      
      
        U fel ghegrim // u wreedheid slim
        U dreyghen, noch u zweren,
        Geen schric noch noot // noch duyvel dood
        En can myn Liefde deeren.
      
      
        Maar als ik zal // int traenen dal
        Door ongheneught verdwynen,
        Mijn schim bevreesd // of bleecke gheest,
        Zal na myn dood verschynen.
      
      
        En quellen zwaar // myn Ouders maar
        Mijn Echte Man ghepreesen,
        Die zal ik dan // waar in ik can,
        Noch even dienstich wezen.
      
      
        Mijn Prinsch vertoeft // weest niet bedroeft
        Bedaert, u wilt verlanghen,
        Mijn hart blyft u // al ist lyf nu,
        In een Kercker gevangen.
      
      
        De trou en eed // dien ik u deed
        Die zullen geen sints lieghen
        Hoopt, en vaart wel // de dingen snel,
        Op aarden licht vervliegen.
