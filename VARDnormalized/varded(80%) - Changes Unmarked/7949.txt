
    
      
        Laest ghinck ic in de Mey spaceren /
        Wandelen met lusten // fyn
        Waar door myn hert moest confirmeren
        Ja geheel in rusten // zijn:
        Vrou Venus karen excellent /
        Sach ik bij paren / daar ontrent /
        Hand aan hand // zeer playsant /
        Danssen en springhen /
        Wentelen in't groene gras /
        Sout een quant // syn verstand
        Konnen bedwinghen /
        Die't ghewent te doene was.
      
      
        Terstont kwamen zij voort ghedronghen /
        Met Herp'n / Luiten / Velen goed /
        Zij dansten / speelden ende songhen /
        Sulcx 'tGhehoor ontsluyten doet.
        Musica schoon was daar bekent /
        Elc hiel den thoon op't Instrument /
        Delicaet / op zijn maet
        Seer manierich /
        Pas a pas voort danssende /
        Nijders kwaad // wordt versmaet
        Maar goedertierich
        Met Blomkens beschanssende.
      
      
        Does Maechdekens lustich van leven
        Stelden mij de Roosen // bij /
        Waarom soud'ic gheen vreugd aankleven
        Die tot zulks verkoosen // zij:
        Veel Blomkens ook suyver en klaer /
        Lieflijck van Reuc brochten zij daar /
        Sulcken vreugd // mij verheucht
        Zeer boven maeten
        En't Jonck gheselschap schoon / eerbaer /
        tSa gij Jeucht // maect gheneucht
        Met zoet praten /
        tIs couragie voor en naar.
      
      
        Terstont quaemen zij zeer gheladen
        Om den last te boeten // meer
        Met taerten / Vladen en Ghebraden /
        Wijn om druc versoeten // zeer
        Ik dacht ik haeck / na sulcken geur /
        Doen werd den Smaec noch Governeur /
        Heus en bly // willen wij
        Nu Bancketteren /
        Sprak een van does Maechden fier /
        Hout v vrij // wie sou dy
        Nu moghen deren /
        Spyt al die't benijden hier.
      
      
        Als ic sach al does vreugd voor ogen /
        's Hertsen bloed verwermden // zij
        Terstont kwamen zij toeghevlooghen
        Helsden / kusten n'armden // mij
        Ic hiel mij koel op dit termijn
        Maar het Ghevoel dede mij pijn
        Die nu mocht // als hij docht
        Naar synen wille
        In Venus hof graven ziet
        Sulcken tocht // mocht ghewrocht
        Syn al stille /
        Maar elc heeft de gracy niet.
