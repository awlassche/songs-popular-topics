
    
      
        Wilt nv al // Groot en smal //
        Den Mey helpen stichten,
        Want Auroras lichten //
        Zijn nu an tfirmament,
        Van ons heel vroeg bekent,
        Komt int dal, Met geschal //
        Aenschout Phoebus schichten,
        Vulcanus moet swichten //
        Die was ons assistent //
        In den Winter absent //
        Weg Mars dienaren // De noch beswaren,
        Ons vreugd alsomen ziet //
        Neptunus baren // Boreas scharen,
        Maect nv qua stromen // niet,
        Want haar kracht // is versacht
        Ziet Saturnus bouwen
        Die haar macht // niet en acht
        Doet Ceres ontfouwen
        Deur smeys bedouwen // Meucht gij anschouwen.
      
      
        Men ziet daar // openbaer //
        Venus residenci
        Pallas deur sienci //
        Geeft reden te verstaen
        Voor menich cortisaen
        Flora claer // De doet haar //
        bloemkens assistenci,
        Bewijst reuerenci //
        Mercurius verslaen
        Retoricas vermaen
        Dianas dieren // Schoon met manieren
        Sietmen versellen, bij
        Haar hoofden chieren // Met lauwerieren
        Luna opstellen, vrij
        Pan om hoen // Gaat in tgroen
        Lammerkens versaden
        Om te voen // Gaat hem spoen
        Ter velden ontladen
        Zeer groen van bladen // Al deur smeys daden.
      
      
        Gaat ten troon // hoort ydoon //
        De Nimphen verheuen,
        Vreugd werd daar bedreuen //
        Deur Cupidos verstand,
        Van Apollo playsant,
        Enen schoon // Lauwer croon //
        Werd hem daar gegeuen,
        Dus zit hij beneuen //
        Bij Venus eligant //
        Als een God triumphant,
        Musicas keelen // Hoortmen daar quelen,
        En accorderen // zoet //
        Harpen en Veelen, Lieflijke spelen //
        Den Mey ter eeren // goed
        Noch tgheluyt // Van tgemuyt //
        t'Gevogelt ghedoken,
        Alle cruyt // Ook ontspruyt
        Lieflick van roken
        Noch staat ontloocken // Den Mey voorsproken.
      
      
        Metter daad delicaet //
        Meucht ghi prinsen merken,
        Deur des Musels werker,
        Zijn alle deze goon //
        Bij haar Fonteyn gewoon //
        Nu ontslaet // Zonder haat //
        Jupiter zijn // Vlercken //
        Zo dat Bachus Clercken //
        Bij Venus zijn ontboon,
        Deur Copido haar Soon //
        De deur zijn stralen // Plutos doet dalen //
        Deur kracht der minnen, groot //
        Lieffelick talen // Gaat hij verhalen //
        Om te verwinnen, bloot
        Dat niet deucht // Werd gheneucht
        Hymenius blaecken,
        En deur vreugd met ghenuecht
        Gaat Momus versaecken
        Die doch moet laecken // Des Meys ontwaecken.
