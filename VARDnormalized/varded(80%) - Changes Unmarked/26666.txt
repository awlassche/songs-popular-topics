
    
      
        Aedde naturlic leven mijn,
        Add mijn trost, add mijn vrucht,
        Add het moet gescheijden zijn,
        Add schoen bloem, reijn suver jucht,
        Add dije mij duck heeft verhoecht,
        Add nu sietdij mij onthoegen,
        Dat heft gedaan vrou Venus ontucht:
        Bij wael betruwen ben ik bedrogen.
