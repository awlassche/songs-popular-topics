
    
      
        O Roosgen root // o peerle jent
        Liefgen minioot // daar ik om lijd' torment,
        Schone Godinne ik blijf geschend,
        Want mijn lijden is zonder ent,
        Nacht ende dach, doe ik geclach, voorwaar,
        Om dat ik scheyd' van v, o maagd eerbaer.
      
      
        Moet ik nu scheiden // ist nu den tijd
        Zonder verbeyden // ofte respijt
        Mijn hart van droefheyt // int lichaam splijt
        Mijn oochskens ik in sticken crijt
        O scheiden fel // die in mijn ghequel // doet aan,
        Ken weet hoe dat icket zal konnen bestaan.
      
      
        O wreede dach // ongheluckighe uer,
        Dien ik oyt sach // ende gij valt mij zo suer
        Want t'dunct mij te zijn tegen natuer,
        Dat ik nu moet o maechdeken puer,
        Zeggen adieu // op een nieu // ik blijf in rou
        O bittere dood waar na zo beydy nou.
      
      
        Princess schone // dit wel versint
        Gij spant die krone // en zijt bemint,
        Want ik ben niet ghelijck den wind,
        Gij staat altijd int harte gheprint
        Ik zal ghetrou // o schoon kersou // nu zijn,
        Al valt mij t'scheyden alzo grote pijn.
