
    
      
        O Venus fier // zoet van manier,
        Rein lieflijck dier // gij doet mij sneven
        Doort minnen vier // myn sinen schier
        In zwaar dangier // Nu moeten leven
        Wilt gij mij gaan // Gantschlijc versmaden
        Zo is gedaan // mijn vreugd voortaan
        Ik zal belaen, droefheyt aencleven
      
      
        Want dach en nacht, t’hart schier versmacht
        door pijn onsacht // in u afwesen
        Hier op hebt acht // schoon liefken tracht
        Met al u macht // mij te ghenesen,
        En mij verblijd, die hier nu lijdt
        Veel pijn subijt // wilt hem met vlijdt
        Spijt wiet benijt // Helpen uit vrezen.
      
      
        V schoonheid jent // spraeck eloquent
        Doet mij verblijen // in liefden strijen
        V gang behent // zeer excellent
        Brengt mij torment // aan allen zyen
        want den brand hiet // der liefden ziet
        En can ik niet in myn verdriet
        Al ist mij liet // gheensins vermijen.
      
      
        Gij zijt alleen // die mij gheween
        Door liefde een // nu cont ontladen
        Dus wilt ontspreen. t’wert u ghebeen
        V armken cleen // Lief uit ghenaden
        En u dienaar // dien u voorwaar
        Lijdt pine zwaar // int openbaer
        Troost lief eerbaer // zonder versmaden.
      
      
        Princes u keert // en u vermeert
        Hem die u eer // met vint t’aenghanghen
        Zijn troost vermeert // Na zijn begeert
        Ongheblameert // bluscht myn verlanghen
        En laat hem zijn, op dit termijn,
        Die liefste dijn // O schone robijn
        Wt liefden fijn // wilt hem ontvangen.
