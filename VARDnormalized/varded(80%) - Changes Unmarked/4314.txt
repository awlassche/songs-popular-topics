
    
      
        Komt Aura lief //
        V soetheyt nu ontmoete // mij,
        In dezen brief //
        Val ik absent te voete // dy,
        Een groete // zij //
        V van herten daar neven //
        Geschreven //
        Laat geschieden //
        Mijn ontbieden //
        Die mij smerten // cont //
        Wilt af vlieden //
        Zulke lieden //
        Die u rieden //
        Mij te wieden //
        Wt der herten // gront.
      
      
        O f gij mij als
        Ballingh, van u gezichte // drijft,
        Op mijnen hals
        Dit jok (zwaar van gewichte) // blijft,
        In dichte // schrijft
        Mijn hand noch even seer,
        V ere,
        Wat can schaden
        Der genaden
        Tijd, verwachten // langh
        Ben beraden,
        Met versmaden
        Dus geladen,
        T'gaen den kwaden
        Ongeachten // gang.
      
      
        Rayende licht
        Ontsprongen in den morgen // vrough,
        Mijn siel verplicht
        Aan u, altijd met sorgen // iough,
        Verborgen // drough
        V overgroote minne:
        Goddinne,
        Laat der schanden,
        Niet verbranden,
        Mij, met groot // verwijt:
        Spijt vijanden:
        Kerckers banden,
        Deur u handen.
        Werd' ik vanden
        Swaersten noot // bevrijt.
      
      
        Naar redens recht
        V lief, Laurier, en krone // vought:
        Wee zij den knecht,
        Die iet op dijn persoone // wrought:
        Hoe schone // lought
        Gij, in ons eerst genaken,
        Wij spraken
        Zo besneden,
        Zoeten reden,
        Zonder schamen // iet:
        Waar is heden
        Dien voorleden
        Tijd getreden,
        Die met vreden,
        Ons versamen // liet?
      
      
        Een uerken court
        Dunkt mij gelijk veel iaren // zijn)
        Wanneer mij wordt
        Ontvreemt zo hemelsch claren // schijn:
        Hoe zwaren // pijn,
        Ik draagt in uwen name:
        Eersame,
        Wat gij dochte,
        Ik volbroghte,
        Oyt al even // vroet:
        Wtgesochte,
        Welgewrochte,
        Diergecochte,
        Peerle, mochte
        Gij mij gheuen // spoet.
      
      
        Leef ik int vier,
        Deur u, als den Salmander // doet,
        Cupido schier
        T'hert af, met zijnen brander // wroet,
        Leander // moet
        Ik volgen, niet gedincken
        T'verdrincken:
        Deur mijn poogen
        Zal ik bogen
        S'waters sterken // vloet:
        Wilt medoogen
        Vanden hogen
        Toren, toogen,
        Dat mijn ogen,
        V licht merkten // zoet.
      
      
        Iason verslough
        Den Draeck, die hem fel tegen // blies,
        Heeft in den plough
        De Stieren wreet geslegen // dies
        Vercregen // t'vlies:
        Cause was d'uytvercoren
        
          Alvoren
        
        Dat gij ionde
        Zulk ghesonde,
        Zonder scheiden // van,
        Quets', noch wonde,
        Deeren conde,
        D'aertsche ronde,
        Dienstbaar stoned
        Onzer beiden // dan.
      
      
        Als magnifijck
        Princess, gij ten thoone // zit,
        Ja puer gelijk
        een duyfken zijt: idoone // wit,
        Mignoone // dit
        Ontvangt (van mij arm crancke)
        In dancke:
        Zo zal dezen
        Brief, int lezen,
        V behagen // meest:
        Dijn geprezen,
        Hoog geresen,
        Zoet bij wezen,
        Zal genesen,
        Mijn verslagen // geest.
