
    
      
        IN mijnen gheest // moet ik verjubileren,
        Nooit soeter feest // dan den Sabboth des Heren,
        Wie hem met lust // viert van zijn kwade werken,
        van zonden rust // in God die hem kan sterken
        die heeft zeer pleyn // den Sabboth recht gehouden,
        God heeft hem rein // zijn zonden kwijt geschouden.
      
      
        Ook saligh dan// is hij tot allen stonden,
        Alsulcken man // en rekent God gheen zonden,
        In wiens geest fris // geen bedrog noch qua perten
        Bevonden is // ook niet dobbel van herten,
        Maar als Gods knecht, de vrome Iob ghepresen,
        Slecht ende recht, volheerdigh in Gods vresen.
      
      
        Ook die geen geld, op woeker heeft gegeven
        Noch gheen ghewelt // aan yemandt heeft bedreven
        Maar leent met vlijt, den arme t'elcker ure,
        Zonder profijt // alleen uit liefden puere,
        En geeft haar al // die hem niet konnen loonen,
        dien sulcken zal // in Godes Tempel wonen.
      
      
        Zo wie met vliet // uit lust hier in beklijven,
        Zij zullen niet // ledig onvruchtbaar blijven
        In kentenis // van Gods volmaakte leer,
        die heylsaem is // in Christo onzen Here,
        God die vermeert // de gaef zijns gheests beseven,
        Wie hem verneert, die wordt van God verheven.
      
      
        'Tis wel geviert // die zo den Sabboth rusten
        Zulk een verciert // Gods leeringh zoet met lusten,
        die weeskens klein // beschermt gelijk een vader
        Voor 't ghewelt pleyn // en die als wijs berader
        Hem neerstigh keert // dat weeskens onbedrogen
        Geschickt, gheleert, in Gods woord opghetogen.
      
      
        Is een in noot // die hem dan troost van herten,
        En breeckt zijn brood // een ziel vol hongers smerten,
        die niet en kan // rusten eten noch drinken,
        Als hij een man // nootdruftigh kan bedincken
        Maar hem eerst spijst // uit liefden na en verre,
        Zijn licht dat rijst // op als de morghen-sterre,
      
      
        Als hij besorcht // de Weeukens rein van zede,
        Gheeft, leent, en borght // elk na gheleghenthede
        Die haar zo kan // voorstaen in alle zaken,
        Recht, als haar man // raad tot bidden en waken,
        Geestigh gesint // sacht, stil tot allen tijden
        Dat zij verwint, haar druck en 'svyants strijden.
      
      
        Niemand en can // tot dezen Sabboth komen,
        Oft ten zij dan // dat hij tot sijnder vromen,
        Gantschlijck versaeckt // de wereld met haar lusten,
        En niet en smaakt // dat aertsch vol van onrusten,
        Maar zijn gemoet, gants draegt in God verheven
        Ia suer en zoet // danckbaerlijcken te leven.
      
      
        Zeer onbesmet // is hij voor God eerweerdigh
        Die niet en zit // bij de spotters lichtveerdigh
        In haren raad // noch in haar kromme wegen
        Geenszins en gaat // maar contrari daar tegen
        Al zijnen gang // heeft in het Hemelsch wezen,
        Met lof en danck // als een met God verresen.
      
      
        Zoet ende fijn // moet 't eynd' aller geboden
        De Sabboth zijn // daar maar een is van noden,
        Dits 'teenigh een, stil houden God gelaten
        Als Magdaleen, die koos tot haarder baten
        'Tbest enig goed // d'welck haar bij is ghebleven,
        D'oprechte vroet, en is gheen wet gegeven.
      
      
        O Gods geest net, gij kont ons wel regeren
        Gij zijt de wet, niemand heeft u te leren,
        God heeft u voort // den heylighen gegeven,
        En na zijn woord // in haar herten geschreven,
        Gij zijt een licht // lanteerne t'haerder voeten,
        Na Davids dicht // hetwelk zij navolgen moeten.
      
      
        Och liefden graet // gij band aller secreten,
        In u bestaat // de Wet en de Profeten,
        Gij zijt niet smal // het eynd' end' de hooftsomme,
        Der wetten al // haar oorsaeck en waaromme,
        Na Schrifts uytlegh // groot boven alle gaven,
        De hoogste weg // de rust-dagh aller slagen.
      
      
        Nu Prinse God // die 't alles werkt in allen
        Gij zijt mijn lot // laat mij van u niet vallen,
        Werkt nacht en dag, wilt al mijn kracht regieren
        Op dat ik magh // in u den Sabboth vieren,
        V jok is zoet, u last is licht om dragen.
        Wie leert ootmoet // zijn ziel rust t 'allen dagen.
