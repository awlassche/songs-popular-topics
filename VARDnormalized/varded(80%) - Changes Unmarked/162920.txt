
    
      
        DIe vred' Gods die te boven // gaat
        Diens name hoog te loven // staat,
        Wensch ik u al te samen Waar dat gij
        woont, met namen,  Int groene land // aan
        elken cant, Wandelt na u betamen,
        De Heer biedt u zijn hand.
      
      
        Al schijnt gij nu verlaten // schier,
        De Heer com u te baten // hier, Och wilt dat
        wel bewaren, Dat hij u ginck verclaren,
        Zijn heylich woord // dat gij gehoord
        Hebt, van synen dienaren,
        Brengt doch goed' vruchten voort.
      
      
        Mijn liefste uytvercoren // al,
        Wel hem die does stem horen // zal, Komt
        hier mijn uytvercoren, Mijn Rijk staat u te
        voren, Ontvangt het loon // in 'sHemels
        troon, Gij zult vreuchden orbooren,
        Ontvangen 'slevens Croon.
      
      
        Ziet, niemand zal verblyden // saen,
        Dan die nu deur het lyden // gaan, En die
        als rechte leden, Christi voetstappen treden,
        Den smallen ganck // haar leven lanck
        Met Christelijcken zeden,
        Ghelijck een vruchtbaar rank.
      
      
        Treurt nu niet om den tegen // spoet
        Den welken gij verdraghen // moet,
        Ist zo Gods wel-behaghen, Zo wilt hem
        willich draghen, Lijdtsamich zijt // in uwen
        tijd, Het duyrt hier weynich dagen
        Gij zult noch zijn verblijd.
      
      
        En wilt doch des vergheten // niet
        Dat u de Heer noch weten // liet,
        Zijn heylich woord geprezen Wilt spreken
        ende lezen Te samen hout // elckander bout,
        Elk zij ghetrou in dezen,
        Dat hem hier is betrout.
      
      
        Recht houdt doch u becoepen // vrij,
        Gij Ouden, dit ontknoopen // wij, Wilt
        een goed voorbeeld geven, Den Iongen met
        u leven, 'Tmoet zijn gewaeght, weest niet
        vertsaagd Gij zult worden verheven,
        Zo gij u dapper draagt.
      
      
        Ziet toe gij jonghe Ieuchde // bloot,
        En hebt doch gheen gheneuchde // groot
        In s'weerelts lust voor ogen Dat gij niet
        wordt bedroogen, Van 'svyandts strijd // die
        u benijt, Zijn doen is al ghelooghen
        Wacht u daar voor met vlijt.
      
      
        Den Weduwen tot reender // eer,
        Gheeft Paulus schoon tot eender // leer
        Dat zij haar hope maken, Op God voor
        alle zaken, Want hij beleeft // d'uytcomen gheeft
        Wel hem die dit can smaken,
        Gods Gheest niet tegen streeft.
      
      
        Iaeght doch nu na den vreden // zoet,
        En sout in al u reden // doet, Elk wil zijn
        tongh' bewaren, Om niemand te beswaren,
        Ook leughens kwaad // de Here haat,
        Hier in als Gods dienaren,
        Bruyckt doch de rechte maet.
      
      
        Tezamen wilt doch nu draghen // 'tlast,
        Elckanders na't ghewaghen // vast,
        'Tis Christus Wet waerachtich, Weest des
        altijdt ghedachtich, Al dat gij saen // gheern
        had gedaan, Doe na dat gij zijt machtich,
        Zo zult gij wel bestaan.
      
      
        Vrienden gij staat int herte // mijn,
        Het zoude mij groot smerte // zijn,
        Dat ik van u moest horen, Anders als hier
        te voren, Doe gij ghesont // bekendet rond
        Dat gij als uytvercooren,
        Woudt houden Gods verbond.
      
      
        Gezonden is dit cleen // ghedicht
        Tot u, dat gij wordt reen // ghesticht,
        En dat gij ook meucht vaten, Dat gij niet
        zijt verlaten, Maar dach en nacht // in ons ghedacht,
        De Heer com' u te baten,
        Bewaer u deur zijn kracht.
