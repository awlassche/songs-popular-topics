
    
      
        GIJ schone Roos // die onder 't lijden
        Hier zijt altoos, wilt u verblijden
        Dat u is opgeleydt // tot zaligheid:
        Dat gij niet en zoudt // door lust menigfout;
        Dat gij niet en zoudt, verliesen
        V kleur, door 's  wereldts zot verkiesen.
      
      
        V kleur is schoon, door het verdrukken
        Van het uytroon // ende uytplucken:
        Van al u lusten kwaad, en overdaet;
        Van 's wereldts hoogheid // en haar ydelheydt;
        Van 's wereldts hoogheid, en pralen,
        Die in de afgrond moet neerdalen.
      
      
        Gij Bruydt Christi // laat vrij verachten,
        V naam, zijt bly // in u gedachten,
        Lijt stil het steken wreet // die u doen leed:
        Toont u zachte moet, en diepe ootmoet
        Toont u zachte moet, gestadigh,
        Vergelt het kwaad met goed, weldadigh.
      
      
        Wijkt hier niet uit // wilt liever sterven
        Dan lijdens kruid // te doen verderven:
        Want gij moet dragen 't kruys // zonder abuys,
        Wilt gij dat niet doen // maar tot wreken spoen;
        Wilt gij dat niet doen // wilt horen:
        Zo is u arrebeydt verlooren.
      
      
        V Bruydegom // heeft veel geleden,
        En heeft daarom // voor haar gebeden,
        Die hem deden verdriet // hij wreeckte niet:
        O exempel schoon // van Gods enig Soon:
        O exempel schoon // te aenschouwen,
        Volgt hem na met een vast betrouwen.
      
      
        Met hem zult gij // eeuwig be-erven
        Zo gij hoort mij // met hem wilt sterven
        Dan zult gij in zijn troon ontfaen de Croon,
        En aenschouwen zoet // 't lieflijck wezen goed
        En aenschouwen zoet // verheven,
        Gods aenschijn in dat eeuwig leven.
      
      
        In 's Hemels hof // zult gij vermeyden
        Daar men Gods lof // lieflijck verbreyden
        Met nieuwe tongen bly // een melody,
        Zoet zonder verdriet // want daar ghebreeckt niet:
        Zoet zonder verdriet, zal wezen
        't Lam Gods met zijne Bruyt geprezen.
      
      
        O Bruydt Christi // set u betrouwen
        In u Heer vrij // wilt doch aenschouwen
        Wat hij u schencken zal, na 't lijden al:
        Hebt dan geen verdriet, in u lijden ziet
        Hebt dan geen verdriet, lijdt stille:
        Vraagt met gedult uws Heren wille.
