
    
      
        O Mense // gij
        Die nu Gods heilig woord // hoort,
        Ten wensche // vrij
        Brengt vruchten in accoort // voort
        Verduldigh // lijdt ghehuldigh // strijdt,
        Verblijdt in 'sHeeren // kruyce vol eeren:
        Als God zijn kind, bemindt, sorchvuldigh smijt,
        Onschuldig kwijt, hij 't door 'tverseeren,
        Van 't Helsch zwaer // misbaer maar den zondaer
        Die moet hier naar // lijden eenpaer
        Int vier, tis klaer // dus wee die haar,
        Niet en bekeeren.
      
      
        Onlanghe duert,
        Der beproevinghe vier // hier,
        Die ganghe truert,
        Een lachen goedertier // schier,
        Met vreuchden // bloed // in jeuchden // zoet
        Eeuwig na dezen // salt al ghenesen,
        Bidt en waect // en maect // verheuchdenmoet
        In deuchden // goed, looft God ghepresen,
        Int foreest, en vreest, eest, geen tempeest,
        Gods woord ghekeest // ter herten meest
        Alzo wel heeft // laat in den gheest
        Vruchtbarich wezen.
      
      
        Den sterken // schilt
        Des gheloofs neemt nu an // dan,
        Aenmercken // wilt
        Christus uwen Hoofdman // van,
        Zijnen weghen // bloot, door tegen // stoot,
        En wilt niet wijken // noch ook gelijcken
        Loths Wijf // wiens lijf // heeft cregen // noot,
        Versleghen // dood, door't ommekijcken
        Werd zij een, soutsteen, gheen teken kleen,
        hetwelk zij ghemeen // wel moghen reen,
        Trekken te been // om int gheween
        Niet te beswijcken.
      
      
        Int Aarden // vat,
        Zo den Prinse behaagt // draagt
        Den waerden // schat,
        Na vrede onvertsaeght // jaagt,
        In smeecken // zijn, wilt weken dijn
        Bedd' en slaepstede // in den ghebede:
        Als Paulus // doet dus, int reecken, pijn
        Na teken // fijn, met neerstichede
        Dag en nacht, heb acht, wacht, na u macht
        Die met heyrkracht // komt onverdacht,
        Op dat gij lacht // bij hem gebracht,
        Eeuwig in vrede.
