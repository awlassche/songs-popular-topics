
    
      
        ONlanghs ik savonts ginck
        Ik weet serteyn wel waar,
        Luistert hier naar // ik bid u kaer
        Ik sach te gaer // twee Meyskens daar
        Zij praten // wanlaten, van Vryers zoet,
        En bedreven zo vreugd,
        Ik dachte // en lachte
        Ja wel hoe zoet is ook, is ook de jeucht.
      
      
        Ach! Trijntjen sey de een
        Ik ben nu zo ontstelt,
        Want mij strackx quelt // een jongher Heldt
        ’t Zij u vertelt // want met ghewelt
        Wil trouwen // en houwen, myn voor zyn Wijf,
        Tot dat de bleecke dood
        Zal scheyen // ons beyen
        Dus ly ik staech een zwaar, een zwaar aenstoot.
      
      
        Wel Marritjen myn moer
        Ben gij daarom belaen,
        Zo laat hem staan, en heenen gaan,
        Doet dit vermaen // of ist maar waen
        Wilt proeven // niet toeven // of hij oprecht
        Zijn Ambacht heel wel kan,
        Gij weet wel // secreetsel
        Bij myn verholen blyven, blyven dan.
      
      
        Maar Trijntje lieve sus
        Speelnoot t’sy u gheclaecht,
        Ik heb ghewaecht // ken ben gheen Maagd
        Want ziet ik draecht // myn Moeder vraagt
        Wat schort-jou, quyl-beck, dat gij dus tiert,
        En ik die ben dan stom
        Mijn herte // vol smerte
        Want ziet mijn tijd die is, die is haast om.
      
      
        Mijn rekeningh is uit
        Eylaes! ik sterf van rouw
        Want Jan syn trouw // niet houden wouw,
        Foey hem ik spou // hoe garen sou
        Ik wreecken // en spreecken // hem hier op straet,
        Dat hij myn zo bedrieght,
        Met looghen // bedrooghen
        Heeft hij myn strackx in slaap, in slaap gewiecht.
      
      
        Maar Marritjen myn moer
        Gij bint voorwaar niet dood,
        Al gady groot // dat is gheen noot,
        ‘K lach ook eens bloot // en Keesjen Root
        Kwam lustich // en rustich // als op myn bet
        Hij weerden hem zo dat
        Een Soon // te loone
        Ik binnen neghen Maanden, Maanden had.
      
      
        Hoort doch toe hoe ik voer
        Ik ginck besteden ‘tkint
        Dit wel versint // want ziet gij bint
        In de saeck blint // want ik bevint
        t’Kints leven // maar even // twee Maanden was,
        De Heer die gaf die namt
        De Vryer // de stryer
        Betaelden d’oncosten ghelijck, ghelijck betaalt.
      
      
        Ach! Tryntje tis zo langh
        Ik meen gheleden schier // acht maanden hier,
        Dat ik go chier // met Kees Jaep vier
        Eens was te bier,
        Hij soende // al doende, ô myn wat vreugd
        Ik dencker stadich om,
        Myn Billen // die drillen
        Ik sterf eylaes! vraagt niet // vraagt niet waarom.
      
      
        Prinsen ik scheyde doen
        Ende zij ginghen voort,
        Ik hebt gehoord // en was bekoort
        Doch onghestoort // noch rechtevoort
        Myn dochte // zij sochte, en Vryer fray
        Ghelijck veel Meyskens doen,
        Ik laete // die haete
        En wensch myn eyghen lief, myn lief een soon.
