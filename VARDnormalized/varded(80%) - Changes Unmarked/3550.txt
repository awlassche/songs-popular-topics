
    
      
        LIefftallich groen prieel
        Daar smaecklijck enen dis
        Was bereidet na ghewoon
        Daar ik met de Liefste schoon
        Het hert vermaecten fris
        Bacchides wijn zeer eel
        En Ceres smaecklijck brood
        Bracht ons int bij een ghevoech
        Een zulk hertelijck ghenoech
        Dat ons gheen tijd verdroot
        Phoebus was met haast gegaan
        Op zijns waghens Gouden raen
        Ginck beschijnen // wijt
        D'Medecijnen // blijt
        D'onbesinde // wicht
        Schoot een blinde // schicht
        Op dezen zelven tijd.
      
      
        Trou Lieffdes heten brand
        In mij werd stracx ghevoelt
        Die in alles zeer beleeft
        Mijns Lieffs Hert ontsteken heeft
        En twert niet haast ghecoelt
        Voor dat rein Liefdes bant
        Ghebonden werd zo vast
        Dat t'Lieff van de Liefste niet
        Wijken can / oft doen verdriet
        Door Venus overlast
        O Cupido hoog vermaert
        Stoort gheen Lieffd' zo zoet ghepaert
        Met opmercken // saen
        Wilt verstercken // gaan
        Onze jeuchden // dan
        Komt meer vreuchden // an
        Deur schichten die niet schaen.
      
      
        Mijn Flora schone Bloem
        Ons Lusthoff is gheciert
        Met twee spruiten die ghemeen
        Zullen zijn gheent aan een
        Heel vrientlijck ghemaniert
        Rein Lieffde draecht gheen Roem
        Want tloose nijders zaad
        Woud' het werd zo verd' gebracht
        Dat het quame int veracht
        En eeuwich enen haat
        Doch dat selfst zal niet gheschien
        Dat d'een wil van de ander vlien
        Het trou Minnen // zal
        Beyd' ons sinnen // al
        Gouverneren // wel
        Gheen tempteren // sell
        Ons brenghen ongheval.
      
      
        Nu Lieff stelt u gherust
        Verbindet hert aent mijn
        Dan zo zullen zeer bequaem
        Beyd' ons ziels in een lichaam
        Wel vast ghesloten zijn
        Zo werd te recht gheblust
        Het langh ghewachte goed
        Tgheen de jeucht Couragieus
        Deur de Lieffde Amoureus
        In vreugd ontspringen doet
        Nectars drank zal principael
        Doen verdrijven d'oude quael
        Die veel jaren // wreed
        Ons tbeswaren // deed
        Met een zoete // vlaech
        Komt zo groeten // staech
        Ons met een langen vreed'.
      
      
        Wel loffelijcken held
        Ik prijs u Venus kint
        Om dat al tgheleen ghetreur
        Is verkeert in goed faveur
        Men seylt recht voor den wind
        Oft Momus hem nu quelt
        En wezen wil party
        Dat zal comen veel te laat
        Niet voortspruyten can zijn zaad
        Hoed ook begonnen zij
        Want rein Lieffd' die eerbaer blijckt
        Nemmermeer uit therte wijkt
        Maar int groeyen // hoort
        Wil die bloeyen // voort
        Met ghenuchten // maar
        Zonder zuchten // zwaar
        Door tvriendelijcke woord.
      
      
        Prinslijcke Reine Maagd
        Vaart wel mijns herten vreugd
        Dencken Liefste wilt op mijn
        Als ik hoop te doen op dijn
        Alleen in eer en deucht /
        Nu mij dan Lieffde draecht /
        Die u ben toegheveucht
        En de wereld int ghemeen
        Nemmermeer en can ghescheen
        Ten zij met ongheneucht
        Maar wat Lieffde heeft voorsien
        Moet ten lesten zo gheschien
        Spijt de haters // loos
        Die met snaters // boos
        Door den tonghen // raad
        Onbedwonghen // kwaad
        Zijn werkende altoos.
