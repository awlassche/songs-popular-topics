
    
      
        Alzo den tijd // alles verslijt,
        Vreugd en jolijt // en treurich zuchten,
        Denckend’ op den voorleden stad
        Ons verlichten door Schrifts geruchten
        In tijd ons jeuchts met ijvers vruchten,
        Om des doots schaduwe t’ontvluchten,
        T’licht last ghelaen // om voort te gaan,
        d’Hant aan ploegh slaan // vrij zonder duchten.
      
      
        Van liefden gloet // gheen water vloet,
        Noch drucx onspoet // uytblusschen dede
        Altijts gheneyght tot dienste zoet
        Van zijnen naasten en tot vrede
        Was al ons zagen rein van zede,
        Ghestadich int ghebedt ook mede
        In perijckel groot // ons leven dood
        Door veel exploot // in Dorp en Stede.
      
      
        Den dach // men niet vergheten mach
        Doe men eerst sach // zijn dode werken,
        En dat men ook zo vyerich plach
        Te jaghen na des gheests verstercken
        s’Gheloofs bewijs in alle percken
        Verlatend’ aller Seckten Kercken
        Vreesende vroet // noch zweert noch gloet
        Noch geen onspoet // hoe groot te merken.
      
      
        Door s’gheloofs aard // wederghebaert,
        Mij voort bewaard, o Heer eendrachtich
        En mij doch niet ook en bezwaard
        Boven mijn vermoghen onsachtich,
        Mijns onperfectheyts ben ik clachtich
        Heer uwen gheest maakt mij deelachtich
        Om gaan den pad // na s’Hemels stad
        Behalven dat // ben icx niet machtich.
      
      
        Petrus die zeid // met onderscheyt
        Dat wij bereid // wel zouden waken,
        En dat Satan daar op toeleyt,
        S’gheloofs tresoor ons kwijt te maken
        Werpt sommige in s’kerckers blaken
        Om totter proeve te gheraken,
        V trouwe groot // tot in der dood,
        Gheeftse ter noot // diet moeten smaken.
      
      
        Een vrees mij niet // geeft cleyn verdriet
        Dat men nu ziet in onze dagen,
        Daar druckich berooken gheschiet
        Der goeden als zwaar om verdraghen,
        Slaperich wijken met vertsaghen
        Als riet oft wiet moetmens’ uit vaghen,
        Werd ik zulks kwijt // in tijds verslijt
        Mocht elk subijt // mij welbeclaghen.
      
      
        S’Princen propoost // mij wedertroost
        Waerdoor altoost // men hoept ghestadich,
        Te schand’ en laat hij niet of loost,
        Schoon Satans vuist slaan zeer beladich
        Toetast en prickelt onversadich,
        O Here weestse doch ghenadich,
        Die mij onvroet // does smerte doet,
        Tot ware boet // zijt haar beradich.
      
      
        Voor een Nieuw Iarich present,
        Gij die bekent // t’Een noodichs crachten
        Met alle die mij zijn absent
        T’mywaerts ghewent // in haar ghedachten
        Int ghebedt suchtende met clachten
        Over mijns drucx, en groot verachten,
        En wil ook dijn // ghedachtich zijn,
        Dat Gods aenschijn // zij ons verwachten.
