
    
      
        IN mijnen noot, roep ik tot u o Here,
        Laat mij niet bloot // want ic ben swac en teere,
        Want ik doch niet // zonder u en vermagh,
        In dit verdriet // troost mij Heer nacht end' dag.
      
      
        Troost mij o Heer // en wilt mij niet verlaten,
        In dit verseer // komt mij Here te haten.
        Op u alleen // betrouw' ik Here mijn:
        Wie magh ons scheen // van u liefd' op dit termijn?
      
      
        Vw' liefde zoet // bewijst Here gheduerigh
        Watmen ons doet // dat wij niet wezen truerigh.
        Door uwen geest // o Heer Daniel bystaet,
        In Babel meest // is hij bij die Leeuwen kwaad.
      
      
        Die Leeuwen fel // zijn ook in de woestijne,
        En soecken snel // ons te brenghen in pijne.
        En slanghen fier // die spouwen vyerigh fenijn,
        O God zijt hier // onzen rechten medecijn.
      
      
        Geen kruiden zo goed // die men sou mogen lezen,
        Noch plaester zoet // en kan ons niet ghenesen,
        Maar Here gij die Israel bystont
        Door Christum vrij // maakt ons zielen ghesont.
      
      
        Ons zielen krank // zijn altijdt in dangiere
        Door peste strangh // en ziekten menigertiere,
        Zo dat hier vergaet // menigh met jammer groot
        Die haar gheen raad // soecken tegen den dood.
      
      
        Zij beminnen meer // dat tijtelijcke leven
        En 's werelts eer // van vlees en bloed gedreven,
        Zij zijn eylaes // zeer jammerlijck verblent,
        Als die Maagden dwaes // die door liefden zijn gheschent.
      
      
        Zij zijn inder nacht // ook slaperigh bedeghen,
        Als die gheen acht // op 't licht hebben ghedreghen,
        Als den Coningh haar // zal opwecken subijt,
        Dan zullen zij daar // kloppen maar ter ontijdt.
      
      
        Dan zullen zij // beschaemt daar buiten blijven
        Dat soecken daar bij // en zal haar niet beklijven.
        'Twert dan te spae // om na oly te zien
        Haar zal geen genae // vanden Bruyd'gom geschien.
      
      
        Och vrienden gemeen // wilt bidden ende waken,
        Omgort u leen // laat u licht vyerigh blaken,
        V sedigheyt, laat wezen openbaer,
        Weest zo bereid // als wijze maagden eerbaer.
      
      
        Als een reine bruyt wilt u alzo vercieren,
        Toont overluyt // u licht in all' manieren,
        Want 'tis nu d'uer // van den slaap op te staan,
        Door Christum puer, tot den Vader te gaan.
      
      
        Prinslijcke God // in eeuwigheydt verheven,
        Dit voor een slot // mijn vriendekens gegheven,
        Die van 't ghetal // zijn weergheboren nieu,
        Dit schenck ik u al // voor een vriend'lijck adieu.
