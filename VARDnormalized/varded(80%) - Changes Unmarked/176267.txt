
    
      
        VOor 't al te schittrich licht // dat Venus uit haar lampen
        Gheblasen, heeft ghesticht // in u hel aangezicht,
        Hemel en Aarde swicht // en afgrond met haar dampen.
      
      
        De StroomGodinnen gladt // die door den Aemstel swieren,
        O pracht van onze Stad // hebben een nijd ghevat,
        Op u aenschijn, en dat // met nygend voorhoofd vieren.
      
      
        Zij quellen haar ghemoedt // met honderd duizend smarten,
        Om dat u schoonheid doet // onsteecken, in haar vloet,
        Met onleschbare gloet // der WaterGoden harten.
      
      
        Met moedeloos gheween // does Meereminnen quelen,
        Dat uw volmaakte leen // van boven tot beneen,
        Met nieuwe konst ghesneen // hen al haar Vryers stelen.
      
      
        Een troosjen hebben zij // dat's hoop van te beleven,
        Dat door uw strafheyt gij // den tijd zult gaan voorbij,
        Van zoete vryery // en hebben niet bedreven.
