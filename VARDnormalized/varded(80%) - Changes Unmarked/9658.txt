
    
      
        DE Mense is te wonder // bot
        Zo langhe als hij zonder // God
        Met vlees en bloed te rade // gaat,
        Want hij tot alle kwade // daad
        Daar lust ende gheneghen // haalt,
        Zo dat hij in veel weghen // dwaalt.
      
      
        Alzo wordt hij te samen // kwijt
        Den zoeten aenghenamen // tijd
        Van God een uytghelesen // dag,
        In welken saligh wezen // magh
        Hij, die met recht bekeeren // hoort
        Openhertigh des Heren // woord.
      
      
        Het woord zijn Ziele maken // zal.
        Saligh, maar eerst versaken // al
        Hij zijn eyghen qua leven // moet,
        Aannemen en aankleven // zoet
        T woord, met een rein sachtmoedich // hert,
        Daar zijn vrucht overvloedigh // werd.
      
      
        Wie in des Heren reden // blijft,
        Sterk, als van t'lichaems leden // stijft,
        Die kennisse der waarheid // hij
        Ghekrijcht, en wordt in klaarheid // vrij
        Vant duister jok der zonden // daar
        Hij onder lag ghebonden // zwaar.
      
      
        Maar die gaan buiten dezen // ganck,
        Zullen soecken en lezen // lanck,
        Nochtans die waarheid vinden // niet
        Want zo met alle winden // riet
        Daar waeyt does onghestadich // zijn,
        Al hebben zij weldadigh // schijn.
      
      
        Maar om vlieden 'tverleyden // fel,
        Zo dientmen t'onderscheyden // wel
        Die Bruydt uit alle Secten // eest,
        Die door sHeeren perfecten // gheest
        Noch wandelt onghevloden // slecht
        In alle Gods geboden // recht.
      
      
        Rein liefde tot den naasten // milt,
        Die tot onderwijs haesten // wilt,
        En ook tot beteringhe // straft,
        Oyt woord al sonderlinghe // schaft,
        Dit veldt-teecken, ydoone // draagt
        Die onbevleckte schone // maagd.
      
      
        Deze staat op op vaster // gront,
        Dan die met haren laster // mond
        Den zone van haar moeder // slaat,
        En spreekt van zijnen broeder // kwaad:
        Maar die vernieude jonghe // Bruydt
        Honig vloeyt haar ter tonghe // uit.
      
      
        Zij heeft in dezen heunigh // lust.
        Tis twoort Gods daers' op steunigh // rust
        En ook om op te wassen // melk,
        Hier op behoeft te passen // elk,
        Dat hij niet en verstroye // maar
        Kome ter rechte kope // naar.
      
      
        Welk huis Christus loyaligh // waeckt,
        En sonderlinghe saligh // maakt,
        Als hij t'onkruyt verloren // doet,
        Te zijn tarwe verkoren // goed
        Daar gheeft ons God ghenade // toe,
        Die ons van allen kwade // hoe.
