
    
      
        MYn lief is mijn, en ik ben zijn,
        Die mij zo dicmael trueren // doet:
        Als hij absent // hem van mij wendt,
        Droefheydt mijn hert besueren // moet.
        Dikmaal tot hem // heff' ik mijn stem,
        Die mij nooit te versmaden // plagh,
        Met ongheluck // of zwaren druck,
        Als hij mij maar beladen // zag.
        Dus met gheklagh // en zwaar gewagh
        Soeck ik, die mij versaden // magh.
      
      
        Hij vrijdt mijn hert // van alle smert,
        En al mijn kranckheydt hij // verdraagt:
        Hij komt daar in // vernieuwt mijn sin,
        Verdrijft wat hem in mij // mishaeght,
        Dus is 't met spoet // geworden zoet,
        Dat ik de wereld derven // magh:
        De zonden kwaad // ik nu zeer haat,
        Die ik zo noo te sterven // plagh,
        Dat eerst mijn ziel // zo bitter viel,
        Om 't selv' ik nu te erven // lag.
      
      
        Nu ben ik bly // van sorghe vrij,
        Die mij eertijds benoude // zwaar:
        O wereldts eer // quelt mij niet meer,
        'k Wil mij van u nu houden // klaer:
        Wellust, voorspoet, al 't aertsche goed
        Magh nu mijn ziel vermeyden // niet.
        Door haren raad // en valschen praet
        Ik mij wel eer verleiden // liet:
        Nu metter daedt // ik vroeg en laat
        Wil God alleen verbeyden // ziet.
      
      
        Jesu mijn lief // al mijn gherief
        Gij blijven zult tot aller tijd:
        Met u alleen // ben ik te vre'en:
        Wilt mij toch helpen in den strijdt.
        Boven gheneught // en alle vreugd,
        De welke spruyt van vlees en bloed,
        Is het vermaeck // daar ik na haeck,
        Van uwer liefden zoeten vloed.
        Om dezen smaak // ik heel versaeck
        Mijn eyghen ziel, en 's wereldts goed.
