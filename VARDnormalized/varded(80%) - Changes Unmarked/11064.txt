
    
      
        EEn  danckbaar liet // der Heren Heer aenbiet
        Die op ons neder ziet // om weder te ontladen
        Van t'groot verdriet, dat velen bracht te niet
        Door zwaren siecte hiet // zeer kwalijk omversaden
        En merckt hoe zeer //
        Dat God den Heer //
        De Mensen teer // verblijden,
        Als hij zand neer //
        Ghenade weer //
        Wie zal zijn eer // in deesen nieuwe Iaar belijden.
      
      
        Wij die in noot // van hulpe scheenen bloot
        En telcker uur de dood //zeer broesselijck wacht
        Door vreze groot // veel volckr ons ontbloot
        Om sulcken zwaren stoot // t'ontgaen met haar geslachten
        Maar onverstant //
        Doet buiten t'lant
        Haar ook met schant // versuchten,
        Want Godes hand //
        En can niemand//
        Hoe zeer vayliant // tot geener tijd van t Iaar ontvluchten.
      
      
        De mensch, gemien // wel spoedich tracht t'ontvlien
        De plaatsen daarmede zie // dat God bereid zijn plagen
        Maar t'punt van dien // waar door de straffen schien
        Wij uit ons niet en wien // dat meest is te beclagen,
        Wou elk met boet,
        Het kwaad verwoet,
        Wt zijn ghemoet // verdrijven,
        De Here goed,
        Liet metter spoet,
        Die straf ontsoet // int nieuwe Iaar wel achter blijven
      
      
        O Mensch verheeft // zijn lof die eeuwich leeft
        Door dien ghi met en bleeft, in dotelick verstrangen
        Zijn hulp aencleeft // ootmoedich u begheeft
        Voor hem daert al voorbeest, dat leven heeft ontvangen
        De wijl hij staat,
        En u omvaet,
        Om troost en baet // te geven
        Die zich verlaat,
        Op zijn ghenaet,
        En metter daad // int nieuwe Iaar zijn woord beleven.
      
      
        Hierom besnijt // u hart en ziel met vlijt
        En wilt tot alder tijd // des Hemels Prins verbeyen
        U schuld belijdt // en maakt u alles kwijt
        Dat tegen t'ghoede strijdt // zo suldy u bereyen
        Om met bescheyt //
        De Zaligheid,
        Daar elk na beyt // te horen,
        Als Christus zeyt //
        Werd in gheleyt //
        T'rijc is bereid // voor u mijns vaders wt vercoren.
