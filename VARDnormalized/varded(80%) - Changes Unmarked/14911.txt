
    
      
        V Minne gunst // is t’overkunst
        O gey,e Herderinnetjes,
        En t’ooghenblick // is als een strik
        Voor ons verliefde sinnetjes.
      
      
        Als ik int groen // eens dertel soon
        V nat bedaude lipjes,
        Zo find ik mijn // gheseylt te syn
        Aan twee Corale Clipjes.
      
      
        Daar ik dan koom // tegen de stroom
        Van de ghebruysde baertjes,
        Zo dat ik voel // doort zoet ghewoel
        Een bevingh in mijn aertjes.
      
      
        Ik steeck het zweert // en span de gheert
        Nu styfjes dan weer slapjes,
        Dan clim ik op // tot bide top
        Van der Hooft-touwen trapjes.
      
      
        Daar ik de schoot // vast aende boot
        Belegh met strik en knoopjes,
        En deur het nat // maeck ik weer rat
        De stramme overloopjes.
      
      
        En zien ik weer // als ik laveer
        Nu hoochjes dan weer laechjes,
        Dat ik niet win // zo find ik in
        Een reefjen voor de vlaechjes.
      
      
        Tis tovery // siel rovery
        En zoete Boereknaepjes,
        Als wij met u // int Bosjen luw
        Gaan houden middag slaepjes.
      
      
        Daar gij met ons // den brand des Sons
        Ontschuylt in diepe holletjes,
        Daar gij ons solt // daar gij ons rolt
        Als kittelsiecke snolletjes.
      
      
        Het schijnt dat gij // gheen vryery
        Ghebruyckt, maar toverkunsjes,
        Dat gij kort // deelachtich wordt
        Ons lieve Minne gunsjes.
      
      
        En Coridon // myn Goude Son
        Ik haat al myn Vriendinnetjes,
        Wanneer u beeld // zo haestich speelt
        Int diepste van myn sinnetjes.
      
      
        Wanneer ik voel, het zoet ghewoel
        En twemfelen uwes tonghetjes,
        Myn hertjen teer // doet op en neer
        Veel Minnelijcke sprongetjes.
      
      
        Het nockt, het beeft // het lockt, het leeft
        Nu laechjes dan weer hoochjes,
        Het tuycht en spreekt // wat myn ghebreeckt
        Deurt dryven van myn ooghjes.
      
      
        Maar als ik wen // en beter ken
        Tussen door beide rotsjes,
        Zo peur ik an // ghelijck een Man
        Tot spyt van Thetys trotsjes.
      
      
        Daar myn de vloet // strackx dryven doet
        Tussen twee hooghe baeckjes,
        Waar ik met vlyt // strackx buiten smyt
        Het Ancker met twee haeckjes.
      
      
        En als myn hert // heel quackjes werd
        Doort wellen van de stroomjes,
        Zo legh ik aan // een sloeg te gaan
        Naar de belade Boomjes.
      
      
        Maar Ameril  // wat coude gril
        Cruypt door myn moedeleetjes,
        Als ik opt strant // ga deur het sant
        Met wanckelbare treetjes.
      
      
        Myn dunkt de Zee // waar in ik tree
        Staech welt onder myn voetjes,
        Dan vloetse zeer // dan ebtse weer
        En hippelt ook zeer soetjes.
      
      
        Het klaecht tot dat // ik weet niet wat
        Voor wonderlijcke cluchjes
        Door ons geslacht // zo weerdig acht
        Verdrijven al myn suchjes.
      
      
        O Coridon // mijn Goude Son
        Hoe syt gij nu dus stilletjes,
        Is dan u lust // zo haast gheblust
        Van u verhitte willetjes.
      
      
        Wiens inghewant // int eerste brand
        Zo vierichjes, zo graechjes
        Als of de vreugd // van uwe Jeucht
        Sou dueren duizend daechjes.
      
      
        Maar als terstont // u vast verbond
        En Ionckend zoete lachjes
        Verkeerden in // gheveynsde min
        En trouweloose clachjes.
      
      
        Eerst vluchten wij // doe zuchten gij
        Met meer minne sanghetjes,
        Nu vlucht gij schuw, nu soeck ik u
        Met nat betraende wanghetjes.
