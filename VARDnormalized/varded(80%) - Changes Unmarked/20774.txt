
    
      
        ALderwaerdste Jan Baptist!
        Mijn Patroon, ei! neemt doch weg den twist,
        En den haat // die daar is, door het kwaad,
        Door de sond' // tussen mij en God die 'k heb gewondt.
        Het is mij leed // bidt dat God niet vergeet
        Mijne ziel // die voor does in zonden viel.
        Bidt dat Hij // geeft aan mij // dat ik vly
        Queden raed // bid dat ik wel belief mijn staat.
      
      
        Ik val plat ter aarde ne'er,
        Jan Baptist! Gij waerdigh zijt does eer.
        Ach! ik ken // dat ik niet waerdigh ben
        Op te staan // want ik ben met grote sond' belae'en:
        Waar aan ik vast // legh gehecht als een quast.
        Maar ik hoop // dat Gij breken zult de knoop,
        Die omwart // en benart // heeft mijn hart,
        't Welk nu sucht // en tot U nu neemt zijne vlucht.
      
      
        Gij zijt d'alder-eelste Bloem,
        Jan Baptist! Gij zijt des Hemels Roem,
        't Liefste Kind // 't welk meest heeft God bemindt;
        Wiens gebed // weg kan nemen mijnder zonden smet.
        Dit weet ik wel // 'k hoop de Hell', die mij fel
        Staeghs tast aan // door u bidden te ontgaen.
        Uwe macht // uwe kracht // ik verwacht:
        Ei! die toont // maakt dat ik worden mach gekroont.
