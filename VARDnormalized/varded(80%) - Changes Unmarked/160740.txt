
    
      
        QUijtschel van zonden geeft de Heer
        Door zijn louter ghenaden
        Die hier af laten en niet meer
        Lust hebben tot den kwaden
        Als een groete // laat hij zoete
        In de Schrift orconden // dan
        Komt met moete // neemt u boete
        Goetwillich voor die zonden // an.
      
      
        Ik en begheere niet den dood
        (Zeid de Heer) der zondaren
        Maar dat zij haar bekeeren soot
        Behoort, en laat varen
        Tsondigh sneven // dat zij dreven,
        t Zij heyden of Jode // boos
        'K wil vergeven // dat zij leven
        Wie tot mij komt hoe gode // loos
      
      
        Op dien daar van het kwaad afstaet
        Wil ik niet als den scherpen
        Zeer nauwe straffen zijn misdaad
        Noch hem ook niet verwerpen
        Wilt doch komen // met den vromen,
        Zoekt u ziele bate // gij
        Hebt verschromen // voort verdomen,
        Eer dat het hier te late // zij
      
      
        Heb ic u niet gebeden // vriend,
        Dat gij u zout bekeeren
        Tot mij, en mij met vreden // dient
        Zo wil ik u recht leren
        Mijne wegen // zijt genegen
        U vant boos voorstellen // dwinght
        Dat mijn zegen // u verlegen
        Hier namaels uit der Hellen // bringt
      
      
        Waaromme wilt gij sterven // doch
        Vraagt God door die Profeten
        Keert weder om verwerven // noch
        Mijn goedheid ongemeten
        U onthouwe // en mijn trouwe
        Weder aan u warghde // reen
        Boet en rouwe // heeft die vrouwe
        Bewezen, g'naemt Maeghde // leen
      
      
        Ik op der aarden niet en kwam
        Voor die rechtveerdich waren
        Maar voor die daar ic meest vernam
        Lagen in 's doods bezwaren
        Met veel wonden // onverbonden
        Heb ik onverholen // daar
        Tzelven stonden // wel ghebonden
        Met Mattheus den Tolle // naar
      
      
        Ist dat met u noch heden // heet
        Wilt u doch naar mij schikken
        Zijn u zonden voorleden // leed
        Zo zal ic u verquicken
        Tot versmullen // wilt vervullen,
        Mijn Huis zeer ootmoedich // bloot
        Zonden zullen // wit als wullen
        Worden al waar zij bloedich // root
      
      
        tVrouken van Canane zeer droef
        Dat bat met zeer veel smerten
        Die Here gaf na haar behout,
        Hij sacht geloof int Herte,
        Christum wrochte // en volbrochte
        Dat niet en misluckte // maar
        Voor haar Dochter // wiert al sochter
        Want den Duyvel verdructe // haar
      
      
        Den Pharaseen en Publicaen,
        Hadden voren genomen
        In den Tempel om bidden gaan
        Die heeft hem gaan beromen
        De ander wende // hem niet ende,/
        En dede gestadich // vlijt,
        Hij bekende // zijn ellende
        En sprak Heer mijns genadich // sijt
      
      
        Die Publicaen quyt schelt ontfinck
        Van zijn Goddeloos wezen
        Rechtveerdich in zijn huis hij ginck
        Voor den ander mispresen
        Prinse cierich // goedertierich //
        U deucht doet versachten // milt
        Die manierich // tot u vierich
        Komt, gij niet verachten // wilt.
