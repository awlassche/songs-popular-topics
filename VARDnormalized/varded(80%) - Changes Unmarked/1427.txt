
    
      
        O Venus jent // een zwaer torment
        Een groot ellent lyd' ik bij dezen,
        Gij staat gheprent // int hert bekent,
        Zeer excellent // lieffelijck van wezen,
        Wilt gij mij niet // verlaten ziet
        Alle mijn verdriet // wat mij gheschiet
        Zal ik achten niet schoon lief geprezen.
      
      
        Mocht ic van haar // schoon aenschijn claer
        Al zonder vaer // eens troost verwerven,
        Geen Goud zo zwaar // coos ik voor haar
        Een lieffelijck paar // zouden wij sterven,
        Alle mijn jolijt, in dezen tijd,
        Mijn hert verblijd // int s'werelts krijt
        'Tis meer dan tijd // zal ik troost erven.
      
      
        O Maechdelijck zaad // komt mij te baet
        Eert valt te laat, wilt druck verjaghen,
        Zo zoet ghelaet // zeer delicaet,
        Doet Venus raad // schout nijders kwaad,
        Ghedult tot mij, die in lijden zij
        Zo waar ik bly // dat segh ik vrij
        Niemand dan gij // en derf ik clagen
      
      
        Princess rein // gij zijt alleyn,
        Lief dien ik meyn // schoon bloem verheven
        Dat weertste greyn // zijt gij certain,
        Int swerelts pleyn // wilt mij niet begheven
        Gheeft mij confoort, een troostelijck woord
        Lief rechtevoort // weest niet ghestoort
        Mijn beden aenhoort // en laat mij leven.
