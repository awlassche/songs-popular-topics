
    
      
        Onmin dede mij dwalen // zeer
        Lieffde komt mij halen // weer
        Zeid door reden // tgheen voorleden
        Is / uit uwe sinnen // stelt
        V niet int loose Minnen// quelt.
      
      
        Menighe in zijn jonghen // tijd
        Dickmaels onbedwonghen // vrijt
        Ist niet druckich // ongeluckich
        Want ook iemand voor of na
        Komt wel te vroeg oft beyt te spae.
      
      
        Can ik me courtoiseren // niet
        Dochters respecteren // iet
        Door wat zaken // can aanraken
        Ik zo niet als menich // doet
        Maar altoos leven eenich // moet.
      
      
        Heet reine Lieffd' niet vrij // versaemt
        Waarom maakt ze mij // beschaemt
        Want na tseggen // int weerleggen
        Die in Lieffd' versmaet // fijn
        Somtijts meest beschamet // zijn.
      
      
        T'bedencken mij tot rede // keert
        En den tijd ook mede // leert
        Midts aenvaten // tboertich praten
        Bij zijn Lieff / hem wachter // veur
        Settende schaemt aan d'achter // deur.
      
      
        Redene zeid / die ik verhieff
        Ieder crijcht zijn wederlieff
        Die hem even // wordt ghegheven
        Daarom woud' ik wel eenpaer
        Dat ik bij haar allene waar.
      
      
        Venus Goddin ydoone // zoet
        Haestich met u Soon // spoet
        Zonder beiden // gaat bereiden
        Tbed dat oyt vol lusten // lach
        Daar ik met mijn Lieff rusten // mach.
