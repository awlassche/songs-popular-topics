
    
      
        EEn waerde vrou // doet mij veel vreugd bedryuen,
        Al heeft zij rou // somtijts dat can niet schaen
        Op eer en trou // min ic die fleur der wyuen
        Als ickse aenschou, zo moet ik zingen gaan,
        Want zij mij nou // beuryt voor alle vaer
        Op dat zij mach ghedencken // mijn //
        Zal ik myn herteken schencken // fijn
        Myn lief eerbaer // tot haren nieuwen jaar.
      
      
        Al was geresen // tot in den dood vol vresen,
        Myn jonstich hart // ja in den gront versmoort,
        Zij cant genesen // als zij doctoor wil wezen
        Van alle smart // met een wellustich woord
        Dus wil ik voort // verlaten droefheyt zwaar
        Wil zij slechts lieffelijck wincken // neer
        En trou ghewillich schincken // weer
        Mij voor een aer // tot mynen nieuwen jaar.
      
      
        Venus kint // wilt haar int harte steken
        Dat zij bemint // een weynich tmijnder baet
        Op dat verwint // myn veel ootmoedich smeken
        Dat nu noch int // alderquaetste blaetgen staat
        Ik roep om raad // alle die goden te gaer
        Dat zij mij helpen minnen // toch,
        Op dat ik mach gewinnen // noch,
        Een blyde maar // voor te gehele nieuwe jaar.
      
      
        Grote geneucht // met danssen, en springen,
        Zal ik volbringen // om myn Princess goed,
        Of zij deur vreugd // oft deur wellustich zingen
        Eens wou ghehinghen // Dat ik die vruchten zoet,
        Plocke tot boet // van ons liefde eenpaer
        En ickse mocht verwarmen // lanck
        En rusten in haar aermen // blank,
        Dicht neuens haar // vant ene jaar int aer.
