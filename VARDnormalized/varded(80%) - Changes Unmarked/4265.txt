
    
      
        PRijs-weerdich rein vat //
        Rijnstroomich in gena- de,
        O verborgen schat //
        In t'slot van rijpen ra- de,
        Dat mij t'verlangen // wast //
        Om t te ontvangen // vast //
        T'veroorsaect bitter schro- men //
        En cans ontco- men //
        Of den strangen // last //
        Werde mij afgeno- men.
      
      
        Is niet mijnen // geest
        Benout met rouwich zuchten,
        Die met pijnen // meest
        Geniet der minnen vruchten,
        Dat ik mijn dagen // ach?
        Niet dan mishagen // sach,
        Cause is dijn gestoorte,
        Dus naar behoorte,
        Ik beclagen // mach
        Den dach mijner geboorte
      
      
        Enig gulden zaad,
        Grijpt zweert, en schilt in handen,
        En drijft mijnen staat
        Ter eeren, oft ter schanden,
        Ik arme crancke // zal
        Nemen in dancke // al,
        De daad, aan mij bereuen,
        Steruen, oft leven,
        Suyver ranke // smal,
        Gij uwen knecht moet gheuen.
      
      
        Tooght dan liberael
        Der rusten sepulture,
        Kwaad ist, hondertmael
        Te steruen t'eender ure:
        Gij acht mijn kermen // cleen,
        Noch hebt ontfermen // geen
        Der smerten, die mij prangen,
        Doodlijk verstranghen,
        Nu dijn ermen // een
        Aengenamer ontvangen.
      
      
        Eewich int verwaet
        Als van der hand geslegen,
        Ben ik desolaet
        Verdwaelt langs dorre wegen,
        Moet ik onschuldich // dan
        (Recht als eenvuldich // man,
        Zonder secours verweruen)
        Zien mijn bederuen:
        Zal verduldich // van
        V wreede handen steruen.
      
      
        Rebelle, adieu,
        Nu fineert mijn ellende:
        Welke nooit geen nieu
        Minnaer/ te recht en kende,
        Ziet (t'mijnen leede // groot)
        Mij de bereede // noot
        Mijn edel hert deurclieuen,
        Schenden, en grieuen,
        Tot de wreede // dood
        Volcome dijn belieuen.
