
    
      
        MInsaem zoete natuere //
        Die mij contrariert,
        Hoe zeer dijn willecuere //
        Ootmoedich werd begeert,
        Toogt heden // maar //
        Een reden // claer //
        Waarom gij rebelleert?
      
      
        Een wtgelesen // vrouwe
        Scat ik naar weerde // dy,
        Om wien ik dezen // rouwe,
        Lijdsaem aenveerde // vrij:
        Geen lieuer ieught,
        Die mij verheugt,
        Op geheel d'aerde // zij:
      
      
        Zal mij maken mismoedich,
        Der wonden een, of twee,
        O neen! ik strijde bloedich
        Tot in de laetste wee:
        Al briesschen som
        Vyandich om
        Mij, als een wilde zee.
      
      
        Verhaert vileyne // tongen
        Verbergt u wreet aenschijn,
        Vipers onreine // iongen,
        Serpentich boos fenijn:
        V paden // gaat,
        Geen kwaden // raad,
        Zal hier van weerde zijn.
      
      
        Edel, ia magnifijcke
        Beelde, pointich gewracht:
        Theseus ik mij gelijke,
        Doch Ariadne slacht:
        Zo werd ik uit
        Den Dool-hoff ruyt,
        Bij uwer hand gebracht.
      
      
        Roup ik aldus om bate,
        Princess, altijd meer:
        Sijt mij een honighrate,
        In al mijn cranckheyt teer,
        Daar voren zal,
        Ik over al,
        V schoonheid loven zeer.
