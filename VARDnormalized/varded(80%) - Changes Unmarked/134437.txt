
    
      
        d'VYtmutend Herderin,
        In schoonte, prachtige Hemels Engelin,
        Eerwaerde Vrou // 'tpronckje van Trou,
        Die ik door rechte liefde aenschou,
        Wiens bloosje // en roosjen // uit spruyt,
        In 't Zuyt // verr' boven 't Kruid,
        'tCierlijckst' sieraad // Onachtsaem staat,
        Ha! Nimphje vol jeughdigh gelaat.
      
      
        k'Ben met u Soetert vrij,
        Te velde, voegt u doch aan mijn groene zij,
        Ik heb geen rust // voor d'liefd' u kust,
        Dat het jonck herte, natuerlijck lust,
        Hier enig // allenigh // te zijn,
        Aenschijn // geneest mijn pijn,
        Cupido schiet // zijn schichten ziet,
        En stoockt een heten brand, om niet.
      
      
        Waar zijn u Schaepjes veel,
        Godinne, noch in de Stal, mijn zoet gestreel,
        Mijn dert'le vreugd, die 't hertje verheugt,
        Werelts cieraetje, in't best van u jeugd,
        Min viertjen // pleysiertjen // ter wey,
        't Is Mey // ei sinckter een Rey,
        Dryft 't Vee en boet // al u gemoet,
        Gedenkt bij u dat liefd' veel doet.
      
      
        In't groen begraesde gras,
        Gewentelt, Ik ben noch die ik eertijds was,
        In't kort gezeid // u dienst bereid,
        Nevens mijn Zijtje soetst' eenicheyt,
        Ei queestert // geneester // mijn smert,
        In't hert // lief gansch verwert,
        Door d'heete Min // verkeert de sin,
        Doch k'huysvest wel een Velt-Godin.
      
      
        Princes mijn Granida,
        Gansch geestigh, denckt Vrou een leeft op u gena,
        Van lijf, en leen // 'talbastert steen,
        Ja 't witst' yvoor, verdooft in't gemeen,
        V glansje // 'tgroen kransje // (o! Goon,)
        Ten toon // liefd' aengeboon,
        Nymph Amariel // Ei staat wat stil,
        k'Voel door mijn leven een wond're gril.
