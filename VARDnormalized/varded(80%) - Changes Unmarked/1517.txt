
    
      
        SChoon en playsant // aan elken kant
        Sietmen 's Meys kruiden groeyen // Bomen bloeyen,
        Lief triumphant // wilt met verstand
        U nu tot liefde spoeyen // twist verfoeyen,
        Want men ziet merckelijck certain
        Dat menigh zoet vogelken klein
        Met liefde bij een vergaren,
        Laat ons och te samen paren, mijt beswaren.
      
      
        Aanziet het kruid // dat nu komt uit
        Op dat wij zouden sporen // vreugd t'orboren
        Maar 't zoet gheluyt // hetwelk uit u spruyt
        Dat gaat mij meer bekoren // wilt doch horen
        Na uwen dienaar te aller tijd,
        Die om u zijnen tijd verslijt,
        Op dat hij u sou behaghen,
        Wilt hem rein liefde draghen // te allen dagen.
      
      
        Rein onbevleckt // zijt gij perfect
        Als Aurora zeer koen // komt int groen,
        V zal correct // worden ontdekt
        Dat ik heb troost van doen // int saysoen,
        Staat gij van Flore delicaet,
        Hero gij veer te boven gaat,
        In schoonheid en practijcken
        Wilt mij gheensins nu beswijcken // trou laat blijcken
      
      
        Aenschout met vreugd // Mijn jonghe jeugd
        Die met hert en sinnen // gaat beminnen
        Op eer en deugd // maakt mij verheugt,
        Zo zal ik vreugd // gewinnen // druck besinnen
        Want Cupido heeft mij doorwont,
        Zo dat ik nu ben onghesont,
        En altijd zal moeten wezen
        Tot dat gij mij lief ghepresen // komt ghenesen.
      
      
        Prins dochter prent // zeer excellent
        In u dit slecht ontfouwen // laat ik schouwen,
        Dat liefd' zeer jent // is 't fondament,
        Daar ik vast mach op bouwen // en betrouwen,
        Op u woordekens zoet en fray
        Die ik verwacht nu in does May
        Van u om te verschonen,
        Wilt doch liefd' met liefde loonen // mijn liefde kroonen.
