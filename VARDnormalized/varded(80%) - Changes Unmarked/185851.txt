
    
      
        Ave Maria // Gods liefste lief ydone
        Der maechden kronen // Maria weest gegroet
        Gratia plena // claer als de zonne schone
        Int Hemels troone // vat vol genaden zoet
        Der deuchden volle vloet // een suyver hert ootmoet
        Hebt gij gebaard // Gods soon ons hoogste goed.
      
      
        Dominus tecum // die heer zij met v ene
        Suyver fonteene // vrou alder wereld wijt
        Benedicta tu // in mulieribus allene
        Al waert gij cleene // gij zijt gebenedijt
        Geen vrou te geender tijd // zo zalig als gij zijt
        Noch zo verheven // van God tot vijands spijt.
      
      
        Et benedictus tuus // gedankt, gelooft, ghepresen,
        Moet gij altijd wezen // dat hoochweerdich pant
        Fructus ventris tui // dijn vrucht rein wt gelesen
        Wt Ieffa geresen // Iesus Christus playsant
        Die als een Heer valjant // ons brac der hellen bant
        Ende die gevangen brocht // in haar vaders land.
      
      
        Sancta Maria Princess // hoog verheven krone
        Edel geboren // van koninck Davidts saet
        Ora pro nobis // bidt voor ons die te voren
        Waren verloren // verdoemt door zonden kwaad
        Tot dat ons kwam te baet // Iesus onze advocaet
        Van u geboren // o schone dageraad.
