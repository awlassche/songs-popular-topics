
    
      
        O Mensen // die wenschen
        Om tydelyk goed, Daar de wormen
        hun bestormen, In Vlees en in Bloed:
        Pracht en staeten // Moeder laeten
        Hier in 's weireldsch dal, Rek'ning' geven,
        Van hun leven, Aan den Opper-Al.
      
      
        Daeromme // Gij domme
        Keert u doch tot hem, Dien de Joden,
        Eens ontboden // Tot Jeruzalem:
        Vals van herten // Tot duyst smerten
        Van dat god'lyk Lam // Want hij peynsde
        Hoe 't geveinsde // Volk hem haelen kwam.
      
      
        Voor 't laeste // Te gaste,
        Gaf hij zyn Lichaam // Aan zyn' vrienden,
        Die hem dienden // d'Apostels te saem,
        Tot gedenken // Gaf te drinken
        Zyn godelyk Bloed // Aan de Schaere,
        Een van hare // Bleef noch heel verwoed.
      
      
        Den Here // ging zeere
        Na 't huis van Annas, Daar ook kwamen
        Al-te-samen // 't Volk van Cayphas,
        Die hem sloegen // En joegen
        Tot Pilatus Raed // Maar z'en vonden
        Gene zonden // In hem nocht' misdaed.
      
      
        Dan liepen // En riepen
        't Gemeente te gaer // Laat hem doden,
        Dezen snooden // Kruyst den tooveraer,
        Geeft uw oordeel // t'Onzer voordeel,
        Gij blyft 's keyzers vriend // Laat hem sterven,
        Kappen, kerven // Hij heeft meer verdient.
      
      
        Daar stonden //Die honden,
        Die Joden bereyd, Om t'onteeren
        Duyzend keren // 's Hemels Majesteyt:
        En zij greepen // Vreede zweepen
        En ketens, en roen // Om zyn' letten
        Te verpletten // En pyn' aan te doen.
      
      
        Meer plaegen // En slaegen
        Heeft den Heer ontsaen // Als duyst mannen
        Van Tirannen // Oyt wierd aangedaan:
        Bloed met stoopen // Zag men loopen
        Door hun vreed bedryf // En zij trokken,
        Hem met brokken // 't heylig Vlees van 't lyf.
      
      
        Zij brogten // Gevlochten,
        Van doornen een krans // Die z'hem planten,
        te allen kanten // In d'herssens bykans,
        En zij kwamen // Zoo te samen
        Uit Pilatus huis // Daar zij toonden,
        Den gekroonden // Aan het Jodsch gespuys.
      
      
        Zyn' smerte // Verherte,
        Noch meer hun gemoed // Zij verblydden,
        In zyn lyden // En wilden meer Bloed;
        Dies zij deden // Op zyn' leden
        Een zwaer kruysse laen // En daar naar,
        Den kalvaer // Berg is opgegaan.
      
      
        Die vreeden // Ontkleeden
        Den Heer hunnen God // Riet tot schanden,
        In zyn' handen // Gaven zij tot spot;
        Alsdan was hij // In zyn' Passy
        Nooit zoo zeer belaen // Dan ten toone,
        Met ene krone // Naekt en bloot te staan.
      
      
        Dan stak hem // En trok hem
        Dat Joden gespuys // En zij spanden,
        Voeten, handen // Nag'lend' hem aan 't kruys.
        Van zyn' leden // Tot beneden
        Liep zyn dierbaer Bloed // Men zag leken,
        Als twee beken // Van zyn traenen vloed.
      
      
        Verslagen // Kwam klaegen
        Maria bedroefd // God sprak, Vrouwe,
        Stelt uw rouwe // U wel troost behoeft:
        Zij zag hangen // Die ontvangen
        Had haer' zuyv're Borst // Daar zij gaven
        Om te laeven // Galle voor den dorst.
      
      
        Door 't steken // Bezweken
        Wierd hij van ene lans // Zon en Maen,
        Bleven staan // Verliezende hun glans,
        Hij riep: Vader // Wilt te gader
        Vergeven hun quaed // Want zij weten
        Geen' secreten // Van hun groot misdaed.
      
      
        Zoo bad hij // Op dat hij
        De zondaers te saem // Mogte leren,
        Hoe elk eeren // Moet zyns Vaders naam,
        Om voor deugden // Duyzend vreugden
        Met Gods Majesteyt // Naar 't verdrieten
        Te genieten, Inder eeuwigheyd.
