
    
      
        ACh God O Heer // wie zal ik dit toeschrijven
        Die mijn hart teer // met lijden dus turbeert
        Hoe zal ik wrijven // haar wt mijn sinnen drijven,
        Een fleur der Wijven, die t'hert altijd begeert
        En hem vermeert, haar dienaar dach en nacht
        Want zij daar in eeren // staat,
        Waar deur t'hert tot haar keren // gaat,
        Liefde niet veracht
        Want reine Liefde baert kracht.
      
      
        Lieflijck is zij // ondeucht heeft haar begheven,
        Beleeft daar bij // barmhertich te aller tijd,
        Eerbaer van leven, reinigheid zonder sneven
        Is haar aencleven // claerlijc in sweerelts crijt
        Haat nochte nijt // nimmermeer in haar ghedacht
        Dus als ik om haar dencke // zoet
        Mijn hert haar tranen schencke // moet
        Liefde wel acht.
        Want reine liefde baert kracht.
      
      
        Cloec van verstand, Is zij die ik beminne
        Der deuchden brandt, heeft haar zeer hoog verciert,
        Wat ic beginne, t'is al tegens hart en sinne,
        Dus ik bevinne // dat zij mij gants regiert
        Wel ghemaniert // is zij daar ik na wacht,
        Als ik haar aenschouwe // ziet,
        Van mijn druck benouwe // vliet,
        Liefde wel acht
        Want reine Liefde baert kracht.
      
      
        Suyver en net // van aenschijn is zij schone
        Zeer fris gheset // van leden wel volmaakt,
        Zij is ydoone // om te spannen de krone
        In sweerelts troone, die mij in't herte blaect
        Want zij druc laect, en al mijn pijn versacht,
        Zij is die te allen uren // jent
        Mijn hart wt allen trueren // went,
        Liefde wel acht,
        Want reine Liefde baert kracht.
      
      
        Waerste Princes wilt reine liefde baren,
        Weest mijn Maistres, troost mij in dezen noot
        Laat ons vergaren // in liefden tezamen paren
        Zo zullen verharen // does nijders te aller stoot
        Die cleyn en groot soecken altijd tweedracht
        Maar lief wilt haar versmaden // gaan,
        En op der Liefden paden // staan,
        Liefde wel acht
        Want reine liefde baert kracht.
