
    
      
        Elk claeght dat hij // niet onbecommert heeft,
        Vrolijck en bly // den gulden tijd beleeft,
        Dies op het pas // al zijn wij vol gheweens,
        Is als zij was // maar t'volck is niet alleens.
      
      
        Tot nootdruft cleen // heeft dat Volk haar gevoecht,
        Daar nu elk een // aan overvloed misnoecht,
        Geeft dan de schuld // den tijd niet maar t'gemoet,
        Dat niet vervult // en werd in overvloed.
      
      
        Was in de mensch // begeerlijcheyt noch niet,
        Hij had zijn wensch // als in de gulde tijd,
        Nu leven wij // in vol op noch bedroeft,
        Maar t'comt daar bij dat elk te veel behoeft.
      
      
        Die dan gherust // wil leven zonder haat,
        Zoeke zijn lust // in een ghemene staat,
        Int heersen niet // van anderen t'ghebien,
        Dat men doch ziet // ghehaet altijdt gheschien.
      
      
        Doch die van Stof // is, dat hij gheern heerst
        Dwingh niemand of // bedwinght zich zelven eerst
        Want t'sal ghewis // niet eender beter gaan,
        Voor dat elk is // zijn eyghen onderdaen.
