
    
      
        MElancolijck,
        Smelt in versijk,
        Mijn ionckhert afgestooten in den gront,
        O mangnifijck,
        Fleur van aerdrijck,
        Gij zijt hier, die mijn lijden stelpen cont:
        Hoe fel Cupido mij deurwont,
        Noch even seer,
        Gij troostbaer ieught,
        Crachtich vermeught,
        Dat ik met vreugd,
        Victorieus floreere:
        Dies moet ik u mijn leven // geven // ere.
      
      
        Eeuwich gij zijt,
        Gebenedijt,
        Eerweerdich, als een wtgesondert // kint:
        Of iemand spijt,
        Mijn hert altijd,
        V alleen bouen duizend honderd // mint:
        Den geest hem gans verwondert // vint,
        Als ik aenschouwe,
        Hoe gij u draagt,
        Als eerbaer maagd:
        Mij en behaagt
        (Gesproken op mijn trouwe)
        Nergens geen zo idoonen // schoonen // vrouwe
      
      
        Zo menichmael,
        Maeck ik verhaal,
        Van u (schoon lief) als mijn ionck hert versucht:
        Gij maagd loyael,
        Zijt principael,
        In sulcken noot, mijn schilt, borght, en toevlucht
        Voor geen dangier mijn siele ducht,
        Als gij goddinne
        Geheel mijn wreet,
        Hertgrondich leed,
        Zo overcleet,
        Dat ik secours gewinne:
        Och gij draagt boven last // vaste // minne!
      
      
        Van u absent
        T'zijn een moment,
        Mij dit leven zeer doloreus bezwaard:
        Zo partinent,
        En excellent,
        Is u minsaem bywesen rein geaert:
        Daar u schoonheid haar niet verclaert,
        De vreugd is kleine:
        Gelijk een hert,
        Naer-geiaeght werd,
        En dorstich smert,
        Naar een zoete fonteine,
        S'gelijcx ik naar u touve // souve- // reine.
      
      
        Sucade // gij
        Voor schade // mij
        Bevrijt, als mij de boze weder- // staan:
        Is t'quade // bij?
        Te rade // dy
        Heb ik, wie zal mij konnen neder // slaan?
        Tot u moet ik (O cedar) // gaan,
        Om t'zijn genesen:
        V puere // hand
        De cuere // vant,
        Does uere // spant
        Gij de croon wtgelesen,
        Hoe can ergens zo reene // ene // wezen?
      
      
        Princess wijs,
        Gij zijt den prijs,
        Die mij elk dus langhe vertouven // sach:
        Gij dan (propijs,
        Bloyende rijs)
        Weet al dat tot mijn smert behouven // mach:
        Geeft (voor zo menigh drouven // dach)
        Een court verblijden:
        T'zy wat verdriet,
        Mij dan geschiet,
        Ken achtet niet,
        Dan een vast toevertijden:
        Al moet ic noch langdeurich // treurich, lijden.
