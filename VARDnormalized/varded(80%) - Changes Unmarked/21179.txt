
    
      
        Hoe boosaerdigh // zijn nu de Mensen kind'ren
        Van onze tijd?
        Die zo vaerdigh // eens anders eer vermind'ren,
        Door Haat en Nijd:
        Hoe verkeert, hoe nijdigh spreekt de mond?
        Veel valse dingen // ja beuselinghen,
        Om zo te bringhen // haar Naasten in 't verdriet,
        Verdriet, verdriet, ja te storten in den grondt.
      
      
        Loosen Klapper // Faem-rover, Deugt-verberger,
        Elks ongerief,
        Leugen-snapper // weer vrij dat gij veel erger
        Zijt, als een Dief:
        Want een Dief zoekt maar het tijd'lijck goed,
        Tot zijn behoeven // Maar zulke Boeven
        Zal God bedroeven, die haren vriend zijn eer,
        Zijn eer, zijn eer nijdigh treden onder voet.
      
      
        Doch de Vrome // die op Godes goedheid hopen,
        Zullen 't verdriet
        Boven komen // Want Gods genaed' staat open
        Die tot hem vlied:
        Laat de Booz' uytspouwen haar fenijn,
        Die God betrouwen // en op hem bouwen
        In haar benouwen // die zullen noch in 't endt,
        In 't endt, in 't endt wel gemoed en vrolijck zijn.
