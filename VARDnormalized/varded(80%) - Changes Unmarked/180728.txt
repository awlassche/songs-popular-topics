
    
      
        DEn Here triumphant // zij prijs en danck ghestade,
        Die bijzonder // wonder//lijck zijn volc bevrijdt,
        Want met zijn sterke hand //om Israel voor schade
        Te verlossen // rossen // wagen en subijt
        Wierp hij omme zonder sparen // zo dat d’Egyptenaren
        Dreven op de baren // aan den oever dood,
        Die Israelijten waren // dickmaels verlost van haren
        Vijanden, en beswaren // angst ende noot.
      
      
        Christus Hoochpriester groot // heeft om eeuwich te blijven,
        Tegen zonden // vonden // een Verlosser goed,
        Door zijn eyghen bloed root // tot een zalig beclijven
        Hier voor leven // welken // hem ontvangen zoet,
        Haar wordt de macht ghegheven // kinderen Gods te leven,
        En God heeft verheven // zijn nieu verbond
        Met die zo zijn ghedreven // om eenicheyt t’aencleven
        Na Gods Wet gheschreven // in s’herten gront.
      
      
        Al wijken onghesont // veel argherlijcke leden,
        Die verdwenen // henen // gaan, en d’eenicheyt
        Des Gheests verlaten rond // ja en den bant des vreden,
        Dat haar laetste // t’quaetste // wordt // zo die Schrift zeid,
        God blijft noch zonder mijden // bij zijn volk te allen tijden,
        Die hem oprecht belijden // voor deze zal
        Hij schermen en strijden // zij en zullen niet glijden,
        Al viel ook besijden // een groot ghetal.
      
      
        Al waert dat heel end’ al // die Weerelt t’onder ginghe,
        Al verdronken // zonken // bergen in Zee,
        Zo zal // vrij van den val // de stad, s’Heeren wooninghe
        Noch gerustich // lustich // blijven zonder wee,
        Het Wijse-mans ghestichte // Onbeweghelijck dichte,
        Op des steens gewichte // niet en vergaen
        Dat vredelijcke ghesichte // Zijnde vol claren lichte,
        Met oordeel, gherichte // eeuwich bestaan.
      
      
        Dat huis Gods delicaet // lieflijck met zoet behaghen,
        Davids hert en sinnen // binnen // waren daar.
        Iae bad zijn toeverlaet // om daar zijn levens dagen
        Met verheugen // meugen // blijven eenpaer
        Om daar zonder verflouwen // den schoonen Godsdienst t’aenschouwen,
        Met een vast betrouwen // in God vervroyt
        Maar die besijden bouwen // haar aan t’Hooft niet en houwen,
        Zij comen in rouwen // zeer wijt verstroyt.
      
      
        Princelijck Herder nooit // en verliet gij u Schapen,
        Die u woorden // hoorden // gehoorzaam zijn
        Met u te zijn ghekoyt // al haar ghenoechte rapen,
        Doort volherden // werden // eeuwig termijn
        Gheluckich en ghepresen // van alle druck ghenesen,
        Een weyd’ uytghelesen // in s’Hemels pleyn
        Vinden zij boven dezen // vol rust en onverknesen,
        Haren drank zal wezen // s’Levens Fonteyn.
