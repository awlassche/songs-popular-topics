
    
      
        AVe Maria, Gods liefste Lief ydone,
        Der Maegden krone // Maria weest gegroet,
        Gratia plena, klaer als de Zonne schone,
        In 's Hemels troone // vol van ghenade zoet,
        Der deughden volle vloet // in suyverheyt ootmoet,
        Hebt gij gebaard // Gods Soon ons hoogste goed.
      
      
        Dominus tecum, de Heer zij met u reine,
        Suyver Fonteine // vrucht aller Wereld wijt:
        Benedicta tu in mulieribus, alleyne,
        Al waert gij kleine, gij waert gebenedijt.
        Geen Vrouw tot eniger tijd // zo salig als gij sijt,
        Nochte zo verheven, van God tot 's vijands spijt.
      
      
        't Benedictus, gedankt, gelooft gepreesen,
        Moet altijd wezen // dat hooghweerdigh pant:
        Fructus Ventris tui, uwe vrucht rein uytgelesen,
        Vyt Iesse geresen, Jesus Christus playsant,
        Die als een Leeuw valjant // ons brak der hellen bant,
        Ende die gevangen bracht // in zijns Vaders land.
      
      
        Sancta Maria, Princess hoog verkooren,
        Edel geboren // van Koning Davids Zaad,
        Ora pro nobis, bidt voor ons die te voren
        Waren verloren, verdoemt door zonden kwaad,
        Tot dat ons kwam te baet, Jesus ons toeverlaet,
        Van u geboren // o schone dageraad!
