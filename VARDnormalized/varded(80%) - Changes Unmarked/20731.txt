
    
      
        Laat ons van zonden op gaan staan
        In does tijd // als verblijdt,
        Wel-geneught // en verheugt
        Alle mensen zijn op does aerd';
        Om dat op de-ses werelts baen
        God onze Heer // nu komt we'er // uit zijn gracht,
        't Welk den gehele nacht
        Pilatus had be-waert.
      
      
        Wij hebben droef voor does geweest
        Als onze God // stond tot spot
        Van de Joo'n // die Goods-Soon
        Hebben bitterlijck zeer gekruisd.
        Maar waar voor zijn wij nu bevreesd?
        Waarom treurt // waarom scheurt // nu van smart
        Ons zo bedroefde hart,
        Daar de Heer uit 't graf verhuyst?
      
      
        Of houdt ons noch bedruckt de sond',
        Die wel eer // met begeer
        Heeft gelae'n // en gedaan
        Onze ziel met voordachte sin?
        Laat ons verlaten die terstond,
        G'lijck verliet // zo gij ziet // zijne staat
        Onze een'gen Toeverlaet,
        Laat ons ook doen niet min.
      
      
        Laat ons verrijsen g'lijck den Heer,
        Die niet sterft // na dat erft
        't Leven Hij // maar blijft vrij
        Van 't we'er-sterven ten allen tijd.
        Dat wij ook zo niet sterven we'er
        Na dat w' all' // naar onze val // zullen zijn
        Bevrijdt van 's ziels fenijn;
        Laat ons dit doen met vlijdt.
      
      
        De Dood heeft mensen veel voor does
        Met haar bijl // met haar pijl
        Onverbaest // metter haast
        Vaeck geworpen ter aarde ne'er.
        Maar weinig van die (zo ik lees)
        Zijn daar na // door gena // van onze God
        Gekeert op 's werelts schot;
        Zeer weynich kwamen we'er.
      
      
        En alhoewel verwekt die zijn,
        Evenwel // heeft zeer snel
        Naderhand // aan een kant
        Die geholpen de fell Dood
        Onze een'ge Heil, en Medicijn,
        Heeft de Dood // die ontbloot // Hem eens had,
        Daar na nooit aengevat,
        O wonder boven groot!

        Ei! verlaat // alle kwaad,
