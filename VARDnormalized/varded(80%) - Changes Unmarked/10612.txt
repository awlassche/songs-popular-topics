
    
      
        GHebenedijt // Weest altijd // O Here almachtich
        Die al dat leeft // voetsel gheeft // door v ghenaden
        In Christi naam // ons lichaam // niet alleen sijt ghedachtich
        Maar met v woord // rechte voort // ons siel wilt versaden
        Van eyghen rust // en wellust // ons hebt ontladen,
        O Here wijs // gheeft drank en spijs // die met v vereent,
        Sulcx dat wij niet // misbruycken iet // van de weldaden
        Die gij goedich // overvloedich // ons mildlijck verleend.
