
    
      
        O Blaeckerende Son
        Die mij myn vrijheid con
        Benemen door u clare glans
        Dat ik verwonne schyn altans,
        Hoe strale // en male // voor myn ghesicht
        De voncken van u kracht,
        Hoe schyne // met pine
        V lonckjes in mijn herte zo onsacht.
      
      
        Hoe can u schoonheid doch
        Zo vrevel syn, dat noch
        Myn klachten of ghebieden u
        Niet maecken van de wreedheid schuw,
        Syn heden // de reden // gans niet geacht,
        Of wilt gij die versmaen,
        En heetent // onwetent
        Mijn Engheltje bij u mach wel gedaan.
      
      
        Ten kan myn hertje neen,
        Neen t’ken niet zo ik meen
        Van u voor goed werden ghekent
        Want myn ghemoet dat tuycht gij bent
        Bedarich // mewarich // en goedertier,
        Weldadich en volmaakt,
        Ik mercke // u werke
        Begint gij licht doch vaerdich gij die staeckt.
      
      
        V schoonheid onghemeen
        Die doen mij u alleen
        Beminne in does Werelts dal
        Ik brommen uit met groot gheschal,
        Volmondich // en bondich // door reens beleyt
        Dat ghy’t hier al verwint
        In gave // zo brave
        Als gij men niemand meer op aarde vint
      
      
        Veel hebbender vermaert
        Gheschreven van de vaart
        Van Helena de Griekxsche Maagd
        Daar Grieckelant noch roem van draecht,
        Heur wezen // dat presen // zij volmaakt
        ‘tWas een zo schone Vrouw,
        Doch waerder // vermaerder
        En schoonder als Helena ik u houw.
      
      
        Princes myn lieve lust
        Myn aenghename rust
        Myn leven en myn ondergangh
        Myn vrijheid en mijn staff dwangh,
        Te spade // ghenade // myn niet betoond
        Blust doch mijn vlamme heet
        Die d’ooghe // uit vlooghe
        Van u Mijn Zonne laes! zoet en wreet.
