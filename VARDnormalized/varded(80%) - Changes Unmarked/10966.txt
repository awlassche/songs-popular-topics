
    
      
        Musica zoet // gesanck, clanck, claer end’ hel,
        Wie dat die hoort // doet zij verblijen
        Hert ende moet // door t’lieflijck snaren-spel,
        Men zoet accoort, vol melodijen,
        Met harmonijen // zij al wat leeft verkeucht
        T’gheclanck // van sank // maakt go partijen,
        Musijck, Musijck, Musijck volmaakt die vreugd.
      
      
        Van het gheslacht // Jubal // is Musica
        Wt wien dat spruyt // const sonderlinghen
        Syrenes sacht // int land Anthenusa
        Met zoet gheluyt // int water springen,
        Die haar hoort singhen // met lieffelijcker stem,
        Niet voor // t’ghehoor // die druck can dwingen,
        Musijck/ Musijck/ Musijck brengt vreugd met hem.
      
      
        D’edel Musijck // is een Meestersse jent
        Waar Leeck en Clerk die const beminnen,
        Want haar pratijck, is eerbaer excellent,
        Haar lieflijck werk // men zal bevinnen,
        Ghestelt met sinnen // op moet noot, ende toon,
        Den mensch // na wensch // droefheyt verslinnen,
        Musijck, Musijck, Musijck maect vreugd ydoon
      
      
        Musica eel // is een Princess fier,
        Een rechte grondt // van Constenaren,
        Want wat gespeel // van Instrumenten hier
        Die men oyt vond van duizend Iaren,
        Doet zij verclaren // geen Rethorica leert,
        Men ziet // druck vliet // door t’spel der snaren,
        Musijck, musijck, musijck doch vreugd vermeert
