
    
      
        BLijdschap van mij vlied //
        laat ik mij bereyen,
        Om mijns ziels verdriet //
        droevig te beschreyen:
        Want ik in mijn jeugd //
        heb mijn uyt-verkooren,
        Al mijn hoogste vreugd //
        en vermaeck verlooren:
        Berst uit brack dauw //
        ach! ik smelt in rouw,
        Als ik over-denck met smert,
        Dat de schone Vrouw //
        die den hemel wouw
        Mij te plaatsen in mijn hert,
        Daar ik op had geslagen //
        ’t opperst van mijn behagen;
        Die ik hiel voor al mijn lust,
        Is in haar jonghe dagen //
        mij van de dood ontdraghen
        Naar d’on-eyndelijcke rust.
      
      
        Ach! als ik haar deugt //
        haar roem-weerde zeden,
        Eerbare gheneught //
        wel-ghestelde leden,
        Vriendelijck onthael //
        lieffelijcke woorden,
        Daar met menigmaal //
        zij mijn ziel bekoorden,
        Bij mij zelfs betracht //
        en op alles acht
        Dat mij is van haar gheschiedt,
        Klaegh’ ik klacht op klacht //
        ja swem dag en nacht
        In een born van verdriet,
        Ach! dat de dood zo spoede //
        en eer wij ’t beyd’ vermoeden,
        V van mij heeft af-gheruckt!
        Waarom, o wreede roede //
        hebt gij niet in het woeden
        Mij met een in ’t graf ghedruckt?
      
      
        Wat ic doe oft laat //
        ‘k voed’ noch steeds mijn klachten,
        Mijn beminde gaat //
        nooit uit mijn gedachten,
        Zit ik aan den dis //
        (naar mijn oude wijze)
        Dad’lijck ik haar mis //
        weg dan met de spijze,
        Wil ik daar van daen //
        naar mijn kamer gaan,
        Spreken mijn gedachten weer:
        Hier zag ik haar aan //
        daar plagh zij te staan,
        Ginder zat zij bij mij neer,
        En mijn Lief, mijn waerde //
        mijn deugdelijck vermaarde
        Troostertje, leidt nu, helaes!
        In de swarte aarde //
        bij die haar eerst baarde,
        En is daar den wormen aas.
      
      
        Dan ist oft mijn hert //
        berst in duizend stukken,
        Dan vernieuwt mijn smert //
        met mijn ongelucken;
        Ga ik tot mijn rust //
        dan op ’t bedde leggen,
        Mijn verlopen lust //
        port mij stracx te zeggen:
        Hier genood ik vaeck //
        eerbaer mijn vermaeck,
        Soete-lam, in d’Echt, met u,
        En (o droeve saeck!)
        Lief, mijns levens baeck,
        Waar berust uwe lichaam nu?
        Ging d’Echt ons niet bereiden //
        doen ’t ons te samen leiden,
        Een geluk, en een gevaer?
        Waarom, in ’t verscheyden //
        hadden wij niet beiden
        Insgelijcx ook ene baer?
      
      
        Doch, bedacht ik recht //
        ik en moest niet klagen,
        d’Aerd’ was doch te slecht //
        om dat beeldt te dragen,
        Zij was al te vroom //
        deughdigh en Godvruchtigh,
        Loffelijck van roem //
        heusch, beleeft en tuchtigh,
        Dies heeft haar den Heer //
        op-getogen t’eer
        In zijn hemelse gebouw,
        Daar zij haar veel meer //
        zal vermaken weer,
        Als zij hier op d’aerde sou,
        Zij leeft in genuchten //
        daar ik hier in duchten,
        Wachtend’ tezamen naar de uer,
        Die niet is t’ontvluchten //
        ’t einde van mijn zuchten,
        Tot dat God ons tezamen vuer.
