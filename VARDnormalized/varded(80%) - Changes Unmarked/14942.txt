
    
      
        O Schoon vergoode beeld
        Die vol glans en pracht een yders herte steelt,
        O waerde Vrouw // wiers schoonheid nouw
        Al Venus roem bedekken souw,
        Goddinne // vol minne // vol jeucht // vol vreugd
        Wiens oogh verheucht,
        Al watter leeft
        En adem heeft
        Ook slaefbaer in u juck hem gheeft.
      
      
        Gansch niet is Phoebus glans
        Als gij myn schone zo verweent komt ten dans,
        Gansch niet ook is // de beeltenis
        Van Iuno praet wat Pallas wis,
        Int ende // bekende // dat gij // niet zij
        Noch een van dry,
        Den prijs behoort
        Die Paris hoort
        In Cytherea had bekoort.
