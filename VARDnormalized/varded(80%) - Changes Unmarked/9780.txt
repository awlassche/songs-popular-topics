
    
      
        O Christen leden,
        Verstroyt breedt en wijdt,
        Gij die in vreden
        Van God gheroepen zijt,
        Al ist // dat twist // oprijst // bewijst,
        Den Christen aard altijdt.
      
      
        Die u wil spreken
        Oft iet aenbrenghen wilt
        Van die ghebreken
        Zijns naasten, dezen stilt,
        Zegt maet recht gaat, spreect dien, misschien
        Veel in de zake schilt.
      
      
        Handelt bij orden,
        Die u Schriftuere leert
        Alzo mach worden
        Een dinck te recht ghekeert,
        Daar bij // elk vrij // int licht // hem sticht,
        En Gods Naam wordt ghe-eert.
      
      
        Maar lust op kyven
        Is gheenen Christen aard,
        Die zulks bedryven
        Zijn uit God niet gebaard,
        Waar nijd // of strijdt // is, daar // is zwaar
        De boosheydt onghespaert.
      
      
        Want dien geen rustich
        Die de Christenen leidt,
        En is niet lustich
        Tot haat, oft toornigheid,
        Maar heeft // en gheeft, Al zeer // veel meer
        Ghenade, zo Schrift zeid.
      
      
        Hierom wilt merken
        Op die hier achten klein,
        Discoort te werken
        Neffens de leer rein,
        Afsnijdt // en mijdt, Zulk suer // en stuer
        Onsuyver deeg ghemeyn.
      
      
        Wilt vyerigh passen
        Op Gods geboden zoet,
        Zo zal dan wassen,
        Ghelijck een watervloedt,
        Den vre // hier me // Zal zijn // ook dijn
        Gherechticheydt, vol spoet.
      
      
        Oorlof eendrachtich,
        Christen ghesinde wel,
        God laat u krachtich
        Sathan verwinnen snel,
        Op dat // de stad, Beclijft // en blijft,
        Die lijdt veel verstoots fel.
