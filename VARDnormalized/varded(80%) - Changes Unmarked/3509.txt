
    
      
        ALs ik om rusten sacht // bedacht
        Gheleghen was te nacht // in kracht
        Doen peynsden ik ydoone
        Hoe dat de wijsheid schone
        Tot mij kwam zelver gaan // zeer saen.
      
      
        Laat u sinnen niet slecht // ghelecht
        Zijn tot een maagd in echt // oprecht
        Die ik u wel zal wijzen
        En weerdig is om prijzen
        Ziet dat gij deze maagd // behaecht.
      
      
        Ik sprak wijsheyts Goddin// de min
        Brengt ook veel droefheyts in // den sin
        Wist zij tot dezen tijen
        Dat ickse dacht te vrijen
        Zo waar mijn Lieffde ziet // verdriet.
      
      
        Pallas sprak / opter daad // met raad
        Zal ik u zijn te baet // hoed gaat
        Ook al die eerlick minnen
        Vyt reynder hert en sinnen
        Wie vrijt in eer en deucht // wind vreugd.
      
      
        Overschoone greyn // zeer rein
        Thert denckt om u certain // alleyn
        Zo wel int van u scheiden
        Als tsamen gaan vermeyden
        Die Lieffd' tot t'liefste kint // vast vint
      
      
        Prinslijcke Maget teer // vol eer
        Thert daechlijcx met begeer / strekt meer
        Na u wt Lieffde trouwe
        Als tot eenich Jonkvrouwe
        En wacht na tsoet propoost // van troost.
