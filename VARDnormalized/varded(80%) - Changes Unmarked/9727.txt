
    
      
        IN kromme weghen, duister en zwaar,
        Vol dode vruchten, en werken onvruchtbaar,
        Daar namen wij ons ydel wandelinghe,
        Maar uit ghenade werden wij ghewaer
        Enen zoeten reucke, stem, ende licht klaer,
        Wt 'sHeeren Hof ende wooninghe,
        Dies wij gheringhe // merkten saen,
        Dat wij de baen // misten int gaan,
        Maar wij vonden daar staan // van dezen // Hof
        De deur ontdaen // end' hem moet wezen // lof
        Die ons daar heeft zo vriendelijck ontfaen.
      
      
        Wij vonden daar in des Hofs besluyt
        Van alderhande welriekende kruid,
        Maar boven al // hooghe verheven,
        Een roos int dal // die door haar zoet virtuyt
        Syon des Heren uytvercoren Bruydt
        Geeft troost en vreugd, welvaert en leven,
        En vast beneven // der beecken kant,
        Wierden gheplant // zeer abundant,
        Groen spruytkens playsant // die als leden, goed
        Opwiessen in den band // van vreden // zoet,
        In deucht, en jeucht, zeer schoon en triumphant.
      
      
        Menigh was vroylijck hier ter feest,
        Zo lang als in dit Cruyde-hofs foreest
        Den Zuiden wind waeyde zeer stille,
        Maar werden door den Noorden wind bevreesd,
        Want daar een bitter wortel is ghekeest
        Door enen gheest // vol van gheschille,
        En vol moedtwille // ginghen daar voort,
        Vele verstoort // buiten de poort,
        Want dezen wind van Noordt // doet verkonden // haar,
        Alle die aent woord // niet en houden // maar
        Haar laten omwaeyen met fel discoort.
      
      
        Princelijcke Bruydegom, tot wien
        Zullen wij ons keren, oft henen vlien,
        Dan bij u in u Hof ons wooningh' maken,
        Ghemerckt dat wij zo klaerlijcken zien,
        Die bij u niet volherden in dien,
        Waar zij int einde toegheraken:
        O Heer ons waken // helpt met vlijt,
        In allen strijdt // zijt ons bevrijdt,
        Dat niemand ons in zijd-wegen dwaligh // leidt,
        Maar wat profijt // is t'onser zaligheid,
        Geeft ons, want een is noodigh altijd.
