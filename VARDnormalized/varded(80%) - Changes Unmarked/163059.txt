
    
      
        O Verenigd Land, Gesegent
        Zydy boven al vermaerd
        Door de vlugge Faem befaemt // hoog vernaemt
        En als de gulden Sonn' verschoont:
        Want met' Hemels Dauw, beregent
        Wordt, bevochtight uwe Aerd'
        Popilier, Palm, en Laurier // Is 't vercier
        Daar u Hoofd med' is gekroont
        Of 't strijden // 't benijden // 't ontglijden
        Bij tijden // brengt 't lijden
        't Verblijden geeft u God,
        Dit heeft oyt als ook nu gebleken
        Dat u viel het beste Lot,
        Want 't bommen // 't geklommen
        Al 't trommen // al 't krommen // met 't schrommen
        Is gekommen al tot spot,
        En zo aerdighlijck uit gestreken
        Dus verlooren Stad en Slot.
      
      
        Hoe geblixemt, en gedondert
        Hoe geplondert en vernield,
        Door het ysselijck ergh getergh // van den Berg
        En den Koeckoecks droef gefluyt
        Elk versnelt, gesteld verwondert
        Uytgesondert veel ontzielt,
        Elk verschrikt, beraest, verbaasd // als verdwaest
        Zo heeft daar den Hoorn getuyt,
        Door 't geruchte // elk pluchte,
        Elk duchte // elk vluchte // elk suchte
        Dan vloed elk overliep,
        Goede Hoop was om leegh gezonken,
        Als versmacht ten gronde diep,
        't Gedeckte // 't bevleckte // 't bepleckte
        Hem strekte // 't ontweckte
        Dat hier bedrieghlijck sliep,
        't Schijn gebluste ging weer ontfoncken
        Als verblijd, verwinningh riep.
      
      
        Maar doen 't scheen 't gemeen verlooren
        In 't verstooren heel gebracht
        Al 't gequeel // en 't straf gespeel // O te veel,
        O te veel ons toe gesent,
        Onverwacht en vreemdt om horen,
        Van te voren nooit bedacht
        Kwam een glans en schonen Krans // met gedans
        Door een Weselken bekent
        Door 't spieden // 't Uytwieden // 't gebieden
        Van Dieden // geschieden
        Saghmen does vroome daed
        Dat in 't Bos werd den reuk geroocken,
        Ja verdorret Plant en Zaad,
        't Beschicken // 't recht micken // 't verklicken
        Bracht 't schricken // in strikken
        't Verquicken te onzer baet
        Draecken steert, Arents-beck gebroocken,
        En gevestigd onzen Staat.
      
      
        Nu 't verlooren is gevonden
        En 't vervallen op-gerecht,
        En den Yssel-Zoom en stroom // Bos in toom,
        Mits 't beduchte is bewaard,
        Nu 't benoude is ontbonden,
        Onze gronden vast gelegd
        Onze Hope vast gepast // Onzen last
        Al vrij wat verlicht, ontswaert
        't Verwezen // 't mispresen // 't verknesen
        Door dezen // genesen
        Geresen 't laegh vermoeyd
        En gekroockt nu den Staf van Spangien
        de Arents-veeren af-gesnoeyt,
        Zijn Jongen // ontwrongen // ontdrongen
        Ontsprongen // met Tongen
        Gezongen onvermoeyt
        Tot Gods Lof en de schoon Orangien
        Moet dan zijn daar Liefde groeit.
