
    
      
        Met wandelen in ‘t groen
        Ginck ic mijn bedroefde geest wat vreugt aandoen,
        In bos en wey // Door sant en hey,
        Ginck ik betreen in de zoete Mey,
        Geneughden // En vreugden,
        Dit was // In ‘t gras,
        Elk beesje las,
        Zijn behoef kruipt // Van ‘t groen gespruyt,
        Daar pluckte het zijn voetsel uit.
      
      
        Het zoet parigh gediert,
        Dat in de bossen en door de bomen swiert,
        Voelden de min // In hare sin,
        Want de telens lust was daar in,
        Zij zongen // En sprongen,
        Met vreugd // Verheugt,
        In zoet geneucht,
        In zoete lust // Geheel gerust,
        Hebben zij haar zoete min geblust.
      
      
        De Herders in het veld,
        Hebben met geley haar Herderins verfelt,
        Het zoet gevley // Was veelderley,
        Dat menige zoete tongh daar s
        Tot minne // Te winne,
        De sin // Daar in,
        Te doen gewin,
        Had ider voor // Die hij verkoor,
        Elk een volghde des liefdens spoor.
      
      
        Menige Man en Vrou,
        Sach ik daar gaan o zoet vermaeck van de trou,
        Die altemaal // Met zoet onthael,
        Malkander vermaeckte, voor quael,
        Veel vreugde // Verheugde,
        Haar hert // Dat werd,
        Geheel van smart,
        Door dien zij al // Met zoet geval,
        Zijn lief kuste zonder getal.
      
      
        Ik sach menigh zoet paar,
        Gaan wandelen door de velden hier en daar,
        Ik sach daar treen // Veel met haar tween,
        Niemand als ik most gaan alleen,
        De rouwe // kwam houwe,
        Mijn hert // Dat werd,
        Vol grote smart,
        Dit was voor mijn // Een grote pijn,
        Ik dacht waar mach de mijne zijn.
