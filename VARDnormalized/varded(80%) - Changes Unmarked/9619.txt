
    
      
        AL die hebben tot jaghen // sin
        Staat op, en gaat int veld,
        En hebt niet meer behaghen // in
        Den slaap, die menigh quelt,
        Daar is te vaen, o Iagher, dat u gheldt,
        T'eeuwigh welvaren // zoet,
        T'eeuwigh leven is ons voorghesteldt,
        Elk als Heldt // vlijt en gheweldt
        Hier toe openbaren // moet.
      
      
        DRY dingen zijn bevonden // noch
        Bequaem tot dezer jacht,
        Ten zijn netten noch honden // doch
        Tis 't gheloove vol kracht,
        Hope, en de Liefde boven al geacht,
        Om wel te vanghen // maar
        Niet hier // een dier // onder wildt geslacht,
        Maar een sacht // Lam vol eendracht,
        Wel weerdt om verlanghen // naar.
      
      
        AAN dit suyver ghepresen // Lam
        Al ons welvaren kleeft
        Elk Iagher zonder wezen // gram,
        Volgt hem, en niet en sneeft,
        In zonden meer, maar schrickt daar voor, en beeft,
        Den Heer weest onder // daen,
        Iae Iagher u hem gevangen gheeft,
        Die hem heeft // eeuwig hij leeft,
        Dus kleeft hem bijzonder // aan.
      
      
        HOEFven wij in't ellendigh  // wout
        Iets beters t'onser noot?
        Och neen wij: maar die blendigh // stout
        Van hem afwijcken bloot,
        Haar lusten jaeghend, en niet achten groot
        T'lieflijck vermonden // schoon,
        Maar spelen hier met der Zielen brood,
        In den schoot // vanghen den dood,
        Want dat is der zonden // loon.
      
      
        NA 'tbeste moet men speuren // dan
        Meer als na een wildt dier,
        Dewijl het ons gebeuren // kan
        Ontvlieden t'eeuwigh vier,
        Der ghenaden tijd niet door licht' bestier
        Willens verliesen // al,
        De dood fel, of t'eeuwigh leven schier.
        Van tween hier // om 't dangier
        Men 't beste verkiesen // zal.
      
      
        GHELijcken niet den rouwen // knecht
        Esau, en moeten wij,
        Maar God altijdt betrouwen // recht,
        Als Iacob, dat als hij
        Ons deel der ghebenedijdinghe zij,
        Maar zo ik proefve // snel,
        Om te vanghen dat Een noodigh vrij,
        Find ik mij // zeer zwak: waar bij
        Gods hulp ik behoefve // wel.
