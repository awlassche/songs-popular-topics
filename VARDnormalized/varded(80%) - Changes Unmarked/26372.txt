
    
      
        DEn Here triumphant // zij prijs en danck ghestade,
        Die bijzonder wonderlijck zijn volk bevrijt,
        Want met zijn sterke hand // om Israel voor schade
        Te verlossen // rossen // wagenen subijt,
        Wierp hij omme zonder sparen // zo dat d'Egyptenaren
        Dreven op de  baren // aan den oever dood,
        Die Israeliten waren // dickmaels verlost van haren
        Vijanden, end' beswaren // angst ende noot.
      
      
        Christus Hoochpriester groot // heeft om eeuwich te blijven,
        Tegen zonden // vonden // een verlossen goed,
        Door zijn eyghen bloed, root // tot een zalig beclijven,
        Hier voor elken // welken // hem ontvangen zoet,
        Haar wordt de macht gegeven // kinderen Gods te leven,
        En God heeft verheven // zijn nieu verbond,
        Met die zo zijn gedreven // om eenicheyt t'aencleven,
        Na Gods Wet geschreven // in't herten grondt.
      
      
        Al wijken ongesont // veel argerlijcke leden,
        Die verdwenen // henen // gaan en d'eenicheyt
        Des Geests verlaten rond // ja en den band des vreden,
        Dat haar laetste // t'quaetste // wordt zo die Schrift zeid,
        God blijft noch zonder mijden // bij zijn volk te allen tijden,
        Die hem oprecht belijden // voor deze zal
        Hij schermen en strijden // zij en zullen niet glijden,
        Al viel ook bezijden // een groot ghetal.
      
      
        Al waert dat heel end' al // die weerelt te onder ginghe,
        Al verdronken // zonken // berghen ind' Zee,
        Zo zal, vrij van den val // de stad, s'Heeren wooninghe,
        Noch gherustich // lustich // blijven zonder wee,
        Het Wijse-mans ghestichte // onbeweghelijck dichte,
        Op des steens gewichte // niet en vergaet,
        Dat vredelijc gezichte // zijnde vol claren lichte,
        Met oordeel, gherichte // eeuwich bestaat.
      
      
        Dat huis Gods delicaet // lieflijck met zoet behaghen,
        Davids hert en sinnen // binnen // waren daar
        Ja had zijn toeverlaet // om daar zijns levens dagen,
        Met verheughen // meughen // blijven eenpaer,
        Om daar zonder verflouwen // den schoonen Godsdienst t'aenschouwen,
        Met een vast betrouwen // in God vervroyt,
        Maar die bezijden bouwen // haar aen't hoofd niet en houwen,
        Zij comen in rouwen // zeer wijdt verstroyt.
      
      
        Princelijck Herder nooit // en verliet gij u Schapen,
        Die u woorden // hoorden // ghehoorsaem fijn,
        Met u te zijn gekoyt // al haar genoechte rapen,
        Door't volherden // werden // eeuwich termijn,
        Gheluckigh en ghepresen // van alle druck ghenesen,
        Een weyd' uytghelesen // in s'Hemels pleyn,
        Vinden zij boven dezen // vol rusten onverknesen,
        Haren drank zal wezen // s'Levens fonteyn.
