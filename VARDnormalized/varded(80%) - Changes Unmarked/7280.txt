
    
      
        Wij Spoelders jonc van dagen // teer
        Moghen als arme wichten
        Van ongheluck wel klaghen // zeer,
        Men hoordt singhen noch dichten,
        Van onze Konst verheven // moet,
        Ghelijckmen wel van ’t Weven // doet.
        Men acht ons slecht als Boeren
        Daar wij nochtans, als Edelmans
        De Kroon in Hollandt voeren.
      
      
        Niemand doet ons verlanghen // zot
        Het mach ons wel verdrieten
        Wij moeten altijdt vanghen // bot
        En gheenen lof ghenieten
        ’t Is Guyten hier, en Leckers // daar
        Ten zijn Spoelders of treckers // maar
        Wat is er aan bedreven
        Gheeftse den gons // doch zonder ons
        Wat zoudet zijn van ’t Weven.
      
      
        Maar dit doet ons tot leren // spoen
        Het weven jong van Iaren,
        Want wevers dat zijn Heren // koen
        Zo haast zij laten varen
        Het Spoel-wiel als een snoo refuys,
        Men kander niet met houden huis
        Zij willen zijn het mantjen
        Verdrincken ’t gheldt, gaan uytghestelt
        Ghelijck een aerdich quantjen.
      
      
        Het mutsken hangt om vallen // van
        Het hoofd als wilt van zede,
        Met een wanbays van smallen // an
        Ghebreyde Kousen mede
        Den mantel op de een schouder hangt,
        Die veel te kort, wel dient ghelanght
        Den maendagh moetmen vieren
        En ’s avonds laat // gaan achter straet
        Al singhend’ en al tieren.
      
      
        Rethorica leidt haar // in ’t hoofd
        Om Refereyn te spreecken,
        Men schijnt van sinnen gaer // berooft
        Men wil ook snijen en steecken
        Al is men enen Turf maar hoog
        Men wil niet houden zijn ghedoodgh,
        Men willet al te kere
        Den Borgher vroet // zelfs wijken moet
        Iae schande dunkt haar Ere.
      
      
        Den maendag is niet schaed’ // genoech
        Meer dagen in de weke
        Moetmen noch drinken spaed’n en vroeg
        Tot dat men van ghebreke
        Zeer ’t achterkant aan ’t weven // slaat
        Als dan ons wel te beven // staat
        Oyt vreze te ghevoelen
        Den Deunstock stijf // al op ons Lijf
        Hoe seer dat wij spoelen.
      
      
        Want zij gaan weven nacht en // dag
        Ofzy uytsunnich waren,
        Na Spoelen men niet wachten // mach
        Men vloekt ook zeer het garen
        De Reeders hebben ’t aan den hals
        Die wenschtmen ook veelonghevals
        Door ’t breken van de draden
        ’t Is gheenen raad // maar hoe dat gaat
        Wij zijn met Kruys beladen.
      
      
        Soud’ men aan spoelen grijpen // sin
        ’t Waar om te zijn verwondert
        Men hoefde schier gheen pijpen // min
        Dan achte oft neghen honderd
        Men kan niet zijn te voren // saen
        Men wil ons om ons oren // slaan
        Als men moet laten wachten
        Al sietmen wel // dus ons ghequel
        Noch wilmen ons verachten.
      
      
        Oorlof ghesellen Zuyt en Oost
        Wij moeten ons wat lijden,
        Op hope leven wel ghetroost
        Dat men van oude tijden
        Van zulke wevers queken // plocht,
        Dat ik wat wijder reken // mocht
        Niet langher soud’ ik blijven
        Met ’t wiel geplaegt // meen ik, ik saegt
        In ’t Spare liever dryven.
