
    
      
        KOmt Schepper Geest beneden,
        Dit soecken onze gebeden,
        Komt Heer // komt ne'er // dit is onze hart begeer.
        Wilt in de herten treden
        Van Jesu Christi leden;
        Doet meer // geeft we'er // des ziels verloore eer.
        Wilt ons uwe gra-ty geven,
        Met gaven ons versiet.
        Troost ons in dit leven
        Gij, die den Trooster hiet.
        Gij die zijt des levens schoon' fonteyn,
        Maakt ons rein // ons, die zijn in deughden klein.
      
      
        Gij Liefde zijt geheeten,
        Gij zijt die ons doet weten,
        't Gebod // van God // den Heer van Sabaoth.
        Gij hebt als vier geseten
        Op de hoofden der Profeten,
        Wat lot // komt tot // die waren 's werelts spot!
        Gij seventvout in gaven
        Geeft ons uwe gaven schoon,
        Geeft aan uwe slaven,
        Uwe salvinghe tot loon.
        Gij de vinger zijt van Godes hand,
        's Liefdens brand // Gij de zonden helpt aan kant.
      
      
        Wilt onze ziel genaecken,
        God Heyl'gen Geest wilt maecken
        Dat zij // dat Gij // zijt elk met de ander bly.
        Verrijckt met tael onze kaecken,
        Regeert zo onze zaken,
        Dat wij // zijn vrij // van zonden te allen ty.
        God Heyl'gen Geest laat branden
        In ons, u heylich vier,
        Maakt dat onze verstanden
        U kennend' werden hier.
        Plant uwe Liefd' in onzer allen hart,
        't Welk beswart // door de sond' is, en verwart.
      
      
        Wilt 's lichaams swackheyd stijven,
        Onze Vyand wilt verdrijven,
        Die quelt // die stelt // gestadich groot geweld.
        Maakt dat w' uwe vrinden blijven,
        Wilt ons aan free gerijven,
        Verzelt // en schelt // ons all' in 's Hemels veld.
        Wilt onzen Leydts-man wezen,
        Bevrijdt ons van het kwaad;
        Maakt dat wij verresen
        Bekomen 's Hemels staat.
        Maakt dat wij, die aarde zijn en stof,
        Godes lof // zingen mogen in zijn Hof.
