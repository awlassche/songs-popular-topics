
    
      
        LOf prijs en eer, zij v O Heer // en mild Vader
        Die voor den morgen, niet te sorgen, hebt bevolen,
        Dewijl gij alleen // En anders gheen // ons allegader
        Lijf en Siel voet // ende doet // hongher en dorst vlieden
        Voor de weldaden // en ghenaeden, die ons gheschieden,
        Eyscht gij maar // Een hart danckbaer // willich en bereid
        Dus te allen ty // Motten wij // Ook alle lieden,
        God hier boven // danckelijck loven // inder eeuwicheyt.
