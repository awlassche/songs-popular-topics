
    
      
        VRou-Voester van mijn jeughd,
        Meesteresse van mijn zinnen,
        Mijn hoop, mijn troost, mijn vreughd
        Mijn suyvere Goddinne,
        Mijn licht, mijn heil, mijn rust,
        Schat-kist van mijn ghedachten,
        Springh-ader van mijn lust,
        Reviere van mijn klachten:
        Ziet (bid ik) hoe // ik ben te moe,
        Ziet hoe ik quijn // deur druck en pijn,
        En dat om dy // ach lief lost mij // in 't ende
        Doen eens uit does ellende.
      
      
        O oorsaeck van mijn min:
        Hoe laat gij mij dus klaghen?
        Ik hoop niet dat gij in
        Mijn lijen hebt behaghen?
        Ziet hoe ghewilligh dat
        Ik ben om u te dienen,
        Ach: wilt mij daar voor wat
        Doch van u gonst verlienen.
        Nooit minnaer heeft // den dach gheleeft,
        Die zijn Goddin // met zulke min,
        Stantvasticheyt // en trouwicheyt // beminde
        Als gij mijn lief zult vinde.
      
      
        Zult gij o: al mijn vreughd,
        O glory der Goddinnen,
        O overschoone jeugd,
        Dus slijten zonder minnen?
        Denckt bij u zelfs met gonst
        Dat deze brave leden
        Die uit de hoogste konst
        Natuer dus heeft besneden,
        De snelle tijd // die niemand mijt
        Door d'ouderdom // zal maken krom
        En gij zult dan // het bloeyende van // u jeughde
        Kwijt wezen zonder vreuchde.
      
      
        Iupijn was nooit zo zeer
        Verhit in snoeperijen,
        Of ik ben noch veel meer
        Verhit int eerbaer vrijen.
        Nooit Herculus ook me
        Heeft zo ghebrant int minnen
        Om zijnen Omphale
        Doen zij hem leerde spinnen:
        Als ik om u, ach: weest niet schu
        Maar mint mij weer, want niet ik meer
        Begheer tot loon, van u (mijn schoon) goddinne
        Als min voor wederminne.
      
      
        En zo gij mij die gonst
        (Mijn Engel) wilt verlienen
        Ik zal met alle konst
        V al mijn leven dienen,
        Dan zien u blijtste dagen
        Gij zult o schone maeghd
        Want al wat u behaagt
        Lief dat zal mij behaghen,
        Wat u verheuight // zal zijn mijn vreugd.
        En zo daar iet // Goddin gheschiet
        Waar over gij, int minste mij, meugt klaghen
        Dat zal mijn siel mishaghen.
