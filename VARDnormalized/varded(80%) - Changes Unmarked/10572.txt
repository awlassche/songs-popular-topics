
    
      
        SChoon lief mij niet versmaet // maar laat // v oude weghen,
        Dat gij dus obstinaet // mijn scaet // in liefden tegen.
        Vernieut uwes sins ghemoet // en doet // mij blyde mare
        Zo sing ik v met spoet // een groet Int nieuwe Jare.
      
      
        Wilt gij in mijn gheween // na reen // myn drucknontladen
        Zo wordt myn hart besneen // alleen // door v ghenaden,
        Indien gij niet aanziet // t'verdriet // van v dienare
        Zo wordt een droeuich liet // bespiet Int nieuwe Jare.
      
      
        Zo gij v niet verneert // en keert, v stramme sinnen,
        Zo wordt ik door begheert // verteerd int vier der minnen
        Ik maeck aan Venus macht // mijn clacht voor den Outare
        Dat gij mij dach en nacht // veracht, Int nieuwe Jare.
      
      
        Princes van dit rappoort // en stoort v niet in dezen,
        Maar spreekt eens met accoort // een woord, ik zal ghenesen
        Gunt dat ik voor mijn vrou // v hou, int openbare,
        Zo lost gij mijne trou // van rou Int nieuwe Jare.
