
    
      
        Joden men hiet // die nacy, ziet,
        Van Iacob afghekomen,
        Maar ten baet niet // namen noch iet
        Zonder boete met schromen,
        Men moet besneen // aan 'sherten leen
        Inwendig zijn een Jode,
        Christum (voorheen // zijnde) na treen,
        Als navolghers van Gode.
      
      
        Ontstaen is vrij // t'Antiochy
        Den naam der Christen vaten,
        Gij maakt al bij // dat ik ook zij
        Een Christen door u praten,
        Sprak Agrippa // Paulus daar na
        Sprak, ik zoude wel wenschen,
        Vrij zonder scha // dat zo waar ja
        Met u en alle Mensen.
      
      
        Christen oprecht // te zijn zo slecht
        Gheen saeck en is beseven,
        Doch menigh knecht // der zonden lecht
        Hem toe in zijn boos leven
        Den name van // een Christen Man,
        Maar ziet lieve beminde,
        Christen en kan // niemand zijn dan,
        De gheestelijck ghesinde.
      
      
        Het eigen woord // Christen brengt voort
        Een ghesalfde midts dezen
        Wedergheboort // hem toebehoort
        Wt den Gheest, die wil wezen
        Een Christen vroet // wezen hij moet
        'sGheests salvinghe deelachtich,
        Met wandel goed // den roke zoet
        Dat wij uytstorten krachtich.
      
      
        Ioannes zeid // gij hebt planteyt
        De salvinghe des Heren,
        Die met bescheyt // ter zaligheid
        V alle dinck kan leren,
        Niemandts verstand // behoeft gij, want
        Verleyders Antechristen
        Zijn voor de hand // aan elken kant
        Verwekkende veel twisten.
      
      
        Mattheus saen // wijst ons ook aan
        De valse Christen lieden
        Stille opstaan // met zoet vermaen
        Het volk wonder bedieden,
        Door haar practijck // waert moghelijck,
        Zij zouden d'uytverkoren
        Buiten Gods rijk // al door afwijck
        Ook doen dwalen verloren.
      
      
        Antechrist gheeft // in dit foreest,
        Met achterklap en smaden
        Maect veel tempeest // maar die God vreest
        En mach dit vier niet schaden.
        Hoe menigh hoop // buiten den knoop
        Der liefden stelt te werke
        Avondmaal, Doop // haren aenloop
        Is tegen Christi Kercke.
      
      
        En gaat niet uit // na 'tvreemt gheluyt,
        Vrienden, al doen zij horen,
        Hier is de Bruydt // in ons besluyt
        Is Christus, want te voren,
        Gij breet en wijdt // ghewaerschout zijt,
        Elk wacht hem van verdolen,
        Tis nu den tijd // dat elk met vlijt
        Go toesicht is te bevolen.
      
      
        Recht zonder mis // te kennen is
        Een Christen bij onchristen,
        Klaer licht ghewis // bij duysternis
        Vol nevelen en misten,
        Men wel bekent // ook 't different
        Van gheest ende vlees vruchten,
        'tRecht fondament, der waarheid jent
        En zal voor niemand vluchten.
      
      
        Ten baet gheen rom // op Christendom
        Alwaar 's Gheests vruchten falen,
        Liefde hooftsom // ik voren nom,
        Als van den principalen
        Vruchten ook me // vreugd ende vre,
        Gheloovigh, goedertierich,
        Lanckmoediche // en kuyssche ze,
        Een gants wezen manierigh.
      
      
        Zulke zeer schoon // vruchten, als kroon
        Hebben gherust ook zelve
        Op den Gods Soon // die spruyt ydoon,
        In Esaiam elve,
        Want op dit Lam // den Gheest afquam
        In ghedaent' ener Duiven,
        Does Wijngaards stam // zijn ranken tam
        Draghen ook zoete druiven.
      
      
        Onchristen zijn // al die den Wijn-
        stok Christi niet aankleven,
        Die elk termijn // niet worden fijn
        Van den Gheest Gods ghedreven,
        Manck gaande slap // der deughden stap,
        Oft ook levend' in zonden,
        Van ghemeenschap // met God // 'tgheklap
        Is al loghen bevonden.
      
      
        Onze ghemeynsaemhede pleyn
        Bestaat in wandelinghe,
        Ghelijck hij rein // int licht certain
        Is voorghehaen gheringhe,
        Maar zwak wij zien // in ons gheschien
        Ghebreken met onvreuchden,
        Dan doch indien // wij recht belien,
        God reynicht van ondeuchden.
      
      
        Neemt dit in danck // elk Christen rank,
        Laat den gheest niet verkouden,
        Neffens den sank // is meest belanck
        Bidden zonder ophouden,
        Dat ons betaem // den Christen naam
        In swackheyt, die wij romen
        God maeck ons tsaem // voor hem bequaem
        Te zijn oprecht volkomen.
