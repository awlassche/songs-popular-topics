
    
      
        VVAer is de lieve Maagd, ghebleven
        Die myn jonck herte zo behaecht,
        Waar is de deucht // die mij verheucht
        Ja die verslijt myns levens vreugd:
        Die schone // persoone, wiens kracht
        En macht // bij dach en nacht
        Haar vriendelijckheyt
        En minnelijckheyt
        Haar zoet ghesicht, myn staech verleid.
      
      
        Eylaes! wat is de hoop, een zorge
        Een strijd ghestaech in s’weerelts loop,
        Alzo men ziet // en anders niet,
        Ja dickmaels volchter noch verdriet:
        Al zuchten // en duchten, heel zwaar
        Daar naar volcht dikwijls claer,
        Na Dochters manier
        Zo groot ghetier
        D’its doch den aard, van t’minnevier.
      
      
        Hoe dicmaels zuchten ic, met tranen
        Wanneer ik viel in deze strik,
        O lijden groot // k’wens om de dood
        Zo zij mij laat in deze noot,
        Want Maghet // gij jaghet // myn zeer
        Om veer // hoe langh hoe meer,
        Helpt mynder, doch nu,
        En weest niet schu
        K’en lief gheen ander lief dan u.
      
      
        Gheen vreugd ter Wereld schoon, verwacht ik
        Zo het niet is van u persoon,
        Een uur, een Jaar // dunkt mijn voorwaar,
        Maar sprak ik een reys tegen haar,
        Hoe luckigh // ondruckigh // mijn ziel
        Hem hiel // dat zulks gheviel,
        Zo haddet een endt
        Myn groot torment,
        Zo was gedaan mijn zwaar ellendt.
      
      
        De vrees bevanght mij staech // o wonder
        Hoe dat mijn ziele is zo graech
        Om een Jonckvrouw // om een Kersouw
        Op wien ik stelle mijn vertrouw
        Maar lacy // ter spacy // ik quyn,
        Verdwyn // eylaes om dijn
        O Edele Maagd
        V ist beclaeght
        Gij zijt de Perel die mijn behaagt.
      
      
        Princes Amstels-puyck, in schoonheid
        En weet ik gants niet uw’s ghelijck
        Het Amstel-lant // roem wydt bekant
        Mach hem roemen heel abonbant,
        Princess,, Matresse // mijn schat,
        O dat // ghyer noch wat
        Mij kwaamt ter baet,
        Want Nymph ik laat
        De tonghen boos, en alle de haat.
