
    
      
        Cupidoos schicht // mz recht bedicht
        Die kracht der minnen,
        Int hard brand sticht // en mij aan een verplicht,
        Wiens lieffelijck ghesicht // mij als der Zonne licht
        Doorschijnt van binnen,
        In gaven rijk, ik nooit en vandt
        Een haars ghelijck // in cloeck verstandt,
        Maar laes ik find in mij gheplant
        Twee die partyen,
        Om d’overhandt // beyd’ even vaillant,
        Dats hope en vreze, want // zij steken vrij den brand
        Int harte om stryen.
      
      
        De hope zoet // wil in t’ghemoet
        Een starckte bouwen,
        Maar vreze ontdoet // en werpts’ onder de voet,
        De een tuyght kwaad dander goed, gheluck oft teghenspoet
        Wordt mij ghebrouwen,
        Och of tgheluck // mij gunstich waar,
        En vreugd na druck // wou schencken claer
        Gheen pyn zo groot die mij te zwaar
        Soud zyn om draghen,
        Mocht ik met haar // myn liefste Lief eerbaer
        In echt verlost van vaer // als een getrou Minnaer,
        Einden myn dagen.
      
      
        Wat vreuchde can doen // dit te vermoen
        Alleen inwendich?
        Ic wil mij spoen // bij haar, om myn behoen
        En maken mij zo coen // dat ik haar deed bevroen
        Myn druck ellendich:
        Maar schaemtens macht // verbiedet mij
        En liefkens kracht // zeid, doet het vrij,
        Op hoop van troost wil ik mij bij
        De Liefste voeghen,
        Weg schaamte gij // en vreze aan de een zij
        gij doz slechs dz ic ly // o venus maect dz wt
        Ons beide ghenoeghten.
      
      
        Princes mijn noot // en lijden groot
        Wilt doch ontfermen,
        Levende dood // verwacht schoon Lief minjoot
        Wt u mondeken root // een troostelijc woord alsoot
        Vereyscht myn kermen:
        Mijn dienst oprecht // ik presenteer
        V inder Echt // op deucht end’ eer,
        Gij zijt alleen die ik begheer,
        Mijns hartsen leven,
        En werpt met veer // hem jonstich te zien weer,
        wiens pine is veel meer // dan hij can clagen zeer,
        Oft can aengheven.
