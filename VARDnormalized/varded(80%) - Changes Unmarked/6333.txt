
    
      
        Ik moet u verklaren wis,
        Wat dat mij is ghebeurt
        En wat mij ervaren is
        Waar door ic heb ghetreurt
        Bij den kant van het Land,
        Het Casteel sancte Margriet
        Den Spangiaert // zeer vervaert
        Door ons lustelijcke gheschiet,
        Zo zij zeiden al // vijftich int ghetal
        Van zijn Mannen liet.
      
      
        Bij den enen na den Noen
        Ons Schipper boven kwam
        Sprac Gasten hier ist te doen,
        Elc daar zijn Wapen nam,
        Om te slaan // onbelaen
        Tegen die Spangierden fel,
        Elk zey blie // Schepen drie
        Zie ic komen alzo snel,
        met zo menich man // dus gij geesten dan,
        Elc weer hem nu wel.
      
      
        Ons schans kleeren schoeren wij
        Vinghen de groot Ree
        Slaan zouden wij leren bly
        Al deet den Spangiaert wee,
        Als een guyt // achter uit
        Weken zij driemaal voorwaar
        maar aenboort // door een woord
        Van enen Constapel daar
        Kwamen met geweld // dry honderd ghetelt
        Dat viel ons te zwaer.
      
      
        Onzen Timmerman daar sprak
        Geeft mij de Lonte hier
        Zo zal ik den Spangiaert strack,
        Doen vlieghen met het Vier
        En ons me, hier ter Ste
        Onzen Capiteyn bleef dood
        Den Commijs // ook propijs,
        Onzen Barbier zeer minjoot:
        Den Constapel met, onzen goeden smet,
        Een Goutsoecker bloot.
      
      
        Dus zagen wij ons vermant
        Veel honden kleen en groot,
        (Soomen zegt aan elken kant)
        Brenghen den Haes in noot,
        Onzen knecht // doort ghevecht
        Die verloos daar zijn een Been
        Hij sprangh uit // de Ky-uyt,
        En hij sprac met woorden reen,
        Nimmermeer en kan // Ik worden een Man,
        Ic blijf int gheween.
      
      
        Snachts ontrent de Middernacht,
        Lach deze Knecht en kreet
        Want de pijne was onsacht,
        Gheeft Water hij beleet
        Des brants smert // slaat aent Hert
        Ic en kant verdraghen niet
        Een Soldaet // tzijnder baet
        Ginght zijn Meester klaghen ziet,
        Die sprac zeer gheree, Werpt hem in de Zee,
        Twelck zo is gheschiet.
      
      
        Maar eer hij kwam over boord
        Dat zoet ellendich dier
        Stac hem enen Spangiaert voort
        Dood met een Oud Rappier,
        smeten hem // alzo slem,
        Oft geweest had' enen hond,
        zeiden saen // Lutriaen
        Licht daar in de Zee ghewont
        Al ons hoopken krees, soomen hem verwees,
        Het ging al te bont.
      
      
        Enen groten Spaenschen Heer
        Sprak savonts op ons Schip
        Wat spaert gij does dieven meer,
        Zij zijn doch in den knip
        Hanghtse op // bij den krop,
        wij zagen wel alzo seur
        Naar den dach // met gheklach
        En met alzo groot getreur
        Die verscheen eenpaer // door Gods werken daar
        Tot der zelver eur.
      
      
        Sanderendaechs een groot Schip kwam,
        Het scheen wel enen Geus,
        Doen sagen zij als een Lam
        Orange, Blance, Bleus,
        Twaeyder op // in den top
        En de honden blaften claer
        Al te maal // van metael
        En van Ijzer alzo zwaer,
        Den Spangiaert vermant // liep vast onder tlant
        En zweetten van vaer.
      
      
        Doen brochten zij ons zeer rat,
        met vreze zeer vervaert
        Na Sancte Margriet de stad
        met Soldaten bewaard
        Geforiert, ong'hantiert
        In Schouts Huizen en Bailjuw
        Elc verscheen // maar bij een
        Kwamen wij met vrees en gruw
        die doen werd gezien // ik moet u bedien
        Kreech menigen duw.
      
      
        Ontrent acht dagen het voort
        Wierden wij opghesocht
        Van de Spangiaerden verstoort
        Weder aan Boord ghebrocht
        Som te Voet // metter spoet
        Maar die daar waren ghewont
        Voerden zij // droef en bly,
        want zij waren niet ghesont
        Ons Trompetter bleef // want men hem beschreef
        Gouverneurs knecht terstont.
      
      
        Doen zetten zij ons verscheen
        Op elk Schip vier of vijf
        Mochten niet komen bij een,
        Oft zij sloegen op't lijf
        Storm vrij // Kreghen wij
        Verlooren de grote Mast:
        Zonder fael // d'Amirael,
        wiert ooc me alzo verrast
        Noch een Galioen // bleef op dat saysoen
        tSchip met al zijn last.
      
      
        Doen wij kwamen aan den hoeck
        Van Calis neemt dit merck
        Thijs Mossel ons Schipper kloeck
        Die nam daar een wijnberck
        Seylder me // Na de Ste,
        Amsterdam in Hollant schoon
        Hij was vrij // Lacy wij,
        Kreghen doen zo straffe loon,
        Zeer wreet aengetast // in de Boeyen vast,
        Twas een sober Croon.
      
      
        Princelijcke Gheesten wilt
        Verdragen Ouders raad
        V doch in moers Keucken stilt,
        Het zal wezen u baet
        Blijft daar bij // 'Ksecht u vrij,
        Houdet Peerd vast bij den Toom
        Eert ontspringht // lustich dwingh
        Tis gheen boter tot den boom
        Dus onthout dit al // van het Soetendal.
        Acht dit voor gheen droom.
