
    
      
        Does Goddelijcke Nymph is zo gesint,
        Dat zij nooit Liefde weerlief mint
        Want d’Harder stout // van zoete Wout,
        Die zij al schempende gaande hout,
        Kranste met Laure kruid // does Bruyt,
        Knielende met ghebeen
        Of Tyter klaecht // om hulp haar vraagt
        D’afkeerige Maagd // zeid altijd neen.
      
      
        Hoe dikwijls Tyter heeft verlaten ’t Vee,
        Om deze schone Galete,
        Quelent van min // om does Goddin,
        Vluchten haar na ten Boswaert in
        Doch as zij hem komen sach // t’ghelach
        Klonck door het Bosjen heen
        Of Tyter klaecht // om hulp haar vraagt,
        Dafkeerighe Maagd zeid altijdt neen.
      
      
        Lest vont ik haar alleyn bij een Fonteyn,
        Wies haar Albaste Handen rein
        Kuerschuylde ras // wat vreugd myn was,
        Als zij int Waeter lobberent was:
        Zo haast zij myn wiert gewaar // liet daar
        Myn doen alleen,
        Of Tyter klaecht // om hulp haar vraagt,
        Dafkeerighe Maagd, zeid altijd neen.
      
      
        Hadieu versteende Vrou // helaes ik schou
        Duer u stuershe-den van u! Trou,
        Does pijn onsoet // die jy myn doet
        Gij self in ent lijden moet,
        Dit lichte Lofje teer // heeft meer
        Beweghen van u stuerheen,
        ‘K dael grafwaert in // vaart wel Goddin
        In plaets van Min, zegt ja voor neen.
