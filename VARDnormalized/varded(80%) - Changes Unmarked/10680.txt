
    
      
        O Venus fier // Zoet van manier,
        Rein lieflijc dier, gij doet mij sneven
        Doort minnen vier, mijn sinnen schier
        In zwaar dangier // Nu moeten leven
        Wilt gij mij gaan // Gantschlijck versmaen
        Zo is gedaan // mijn vreugd voortaan
        Ik zal belaen // Droefheyt aencleven.
      
      
        Want dach en nacht // t'hart schier versmacht
        Door pijn onsacht // In v afwesen
        Hier op hebt acht, schoon lief en tracht
        Met al v macht // Mij te ghenesen,
        En mij verblijd // die hier nu lijdt
        Veel pijn subijt // Wilt hem met vlijt,
        Spijt wie't benijt // Helpen wt vrezen.
      
      
        V schoonheid jent // Spraeck eloquent
        Doet mij verblent // In liefde strijen:
        V ganck behent // Zeer excellent
        Brengt mij torment // Aan allen zijen
        Want den brand hiet // der liefden ziet
        En can ik niet // In mijn verdriet,
        Al ist mij liet // Gheensins vermijen.
      
      
        Gij zijt alleen // Die mijn gheween
        Door liefde reen // Nu komt ontladen,
        Dus wilt ontspreen, T'wert v ghebeen
        V armken cleen // Lief wt ghenaden,
        En v dienaar // die nu voorwaar
        Lijdt pijne zwaar // In't openbaer
        Troost lief eerbaer // Zonder versmaden.
      
      
        Princes v keert // En v verneert
        Hem die v eert // Met vlijt t'aenhanghen,
        Zijn troost vermeert // Na zijn begheert,
        Ongheblameert // Bluscht zijn verlanghen
        En laat hem zijn // Op dit termijn
        Die liefste dijn // O schone robijn,
        Wt liefde fijn // Wilt hem ontvangen.
