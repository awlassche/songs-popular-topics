
    
      
        PRoper ende // ient,
        Maegdeken amoureus //
        Mijn ellende // kent,
        Maect mij victotieus //
        Niemand dan gij //
        Rein bloemken wtgelesen //
        can mij maken vrij //
        Die ben ter dood verwezen.
      
      
        In mijn lijden // weet
        Apollo genen raad:
        Niet besijden // treet,
        Die zijt mijn toeverlaet,
        Want gij vermeught
        Deur troostbaer medecijne,
        Dat ik ben verheugt
        Al laegh ik dood in schrijne.
      
      
        Es mijn qual // dan
        Curabel deur dijn daad:
        Zo dijn tale // can,
        Verlichten mijnen staat,
        Tooght dat in dy
        Noch leeft een vrouwen herte,
        Op dat gij in mij
        Geneest een bitter smerte.
      
      
        Touft doch lange // niet
        Eer mij, den geest ontwijct,
        Ik ben bange // ziet,
        Dat mij t'herte beswijct:
        Zal mij de dood
        Met zijnen vliem deursteken,
        Eylaes t'sal ter noot
        Aan uwen troost gebreken.
      
      
        Est dijn wille // dus,
        Verheugt u dan hier af,
        Dat ik stille // vlus
        Beclimme Iphis graf,
        Wilt u gerust
        O herte! daar toe geuen,
        Want voorwaar mij lust,
        T'steruen voor zulk een leven.
      
      
        Rein sucade // zoet,
        Gij valt mij veel te wreet,
        V genade // moet,
        Een ander zijn gereed,
        Mij ziende nu
        Int lijden dus verduldich,
        Weet dat ik voor u,
        Sterue de dood onschuldich.
