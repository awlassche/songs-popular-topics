
    
      
        EEn nieuwe Liet // vaet dit bediet
        Tantwerpen geschiet // al in den Jare
        Van tsestich ziet // en verwondert v niet
        Een Broeder met vliet
        Met groter schare
        Van diefleyders hoort // gebracht al voort
        Van de Steenpoort
        In de Vierschare.
      
      
        Willem hiet hij // och hij was zo bly
        Een Cleermaker vrij // binnen Stede
        Als hij nv bij // de Heren tfy
        Kwam, ende zij
        Hem vraechden mede
        Oft hij was herdoopt // ons dit ontknoopt
        Waar gij op hoept
        De Schout dede de reden.
      
      
        Hi sprac tot haar // waarom vraechdy maar
        Na mijnen doop claer // ende niet na mijn leven
        Op dat elk voorwaar // en gij allegaer
        Een ordeel daar
        Na mochte gheuen
        Want ic ben een roof // vraagt na mijn geloof
        Op dat does wereld doof
        Daar na doe zonder sneuen .
      
      
        De Schout zonder deer // die vraechde zeer
        Of hi herdoopt weer // als hi eens had beleden
        Wat is v begeer // gij begeert niet meer
        Dan een woord, O Heer
        Om mij te vertreden
        Hij sprak als een goed knecht // ordeelt doch recht
        En ouerlecht
        Aldus gingen zijn reden.
      
      
        De Schout int gemeen // hert als een steen
        Was niet te vreen // vraagt wederom met lagen
        Willem sprac hoort reen // gij hebt voorheen
        Mij geuraecht alleen
        Mach ik ook nv niet vraghen
        De Schout sprak ontbeyt // geeft eerst bescheyt
        Al van v feyt
        Als dan zal ickt ghewaghen .
      
      
        Ik wilde wel // sprak Willem snel
        Met goed opstel // dat gij eens dachte
        Op den dach zeer fel // die branden sell
        Ghelijk een Oven rebel
        V doch daar voor wilt wachten
        Als de godloose snoo // verbranden zal als stro
        Al doet hijt noo
        Dus wilt daar doch op achten.
      
      
        Willem sprac zonder vaer // in Marco claer
        In zijn sesthiende daar // muecht ghijt lezen
        Al wie gelooft voorwaar // en wordt ghedoopt eenpaer
        Die zal hier naar
        Zalig wezen
        Men vraagt, hoort mijn // sprac de Schout zeer fijn
        Op dat termijn
        V niet na dezen.
      
      
        Willem sprac voortaan // als hij had gedaan
        De kinder onbequaem // en konnen niet geloouen
        Noch sprak hij saen // ook onbelaen
        Wilt dit verstaen
        Nochtans voor de doouen
        Op mijn gelooue rein // verstaat dit pleyn
        Ben ik ghedoopt alleyn
        Hierom ben ik verschouen.
      
      
        De Heren onuroet // al metter spoet
        Maecten hem op de voet // en gingen binnen vlieden
        Met goeden moet // sprak hij zeer zoet
        Gij Borghers doet boet
        De Schout ginct hem verbieden
        Willem sprac: Heer Schout // laat mij spreken onuerflout
        Niet lange (ik hout)
        Zal mogen spreken van v lieden.
      
      
        Terstont verstaat // de Heren kwaad
        Met haren raad // kwamen wilt horen
        Willem sprac, vaet // mijn Heren het doet v baet
        En wijst niet na tPlaccaet
        Wilt gij nit zijn verloren
        Denct doch opten dach // die niemand voorbij mach
        Als gij zult roepen ach
        Oft gij moet zijn herboren.
      
      
        Zij hebben terstont // al gelijk int rond
        Hem met enen mond // verwezen
        O Heer doet cont // haren kwaden vont
        Dat zij worden ghesont
        En de Schriftuer lezen
        Maect dat haar hert // hart als een berd
        Doch soeter werd
        Dat bid ik v mits dezen.
      
      
        Smorgens vroeg tijd // op de merct wijt
        Met cleyn iolijt // zij hem brachten
        Aan den staeck zonder strijd // worgen met nijt
        Zonder langer respijt
        Gingen zij hem versmachten
        Och God, O Heer // ziet doch eens neer
        Na v leden teer
        Want zij haar zeer verachten.
      
      
        Prinse ient // mijn liet bekent
        Is nv ten ent // gezongen
        Ik wasser present // en hem ontrent
        Als hij was ghesent
        Voor den ouden ende iongen
        Daar hij beleet // zijn geloof gereed
        Voor dOuerheet
        God moeten louen alle tonghen.
