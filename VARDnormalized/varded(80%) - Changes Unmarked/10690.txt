
    
      
        T'Valt zwaar // zonder stadighen hoep
        Van troost te winnen // yemandt te minnen
        Voorwaar, in dezen baen ik loop,
        T'vlas moet ic spinnen, t'cost mij mijn sinnen
        Want die mijn hart begheert, Met liefd' bevangen
        Mijn bidden en smeecken weert
        Hoer oochskens van mij keert,
        En zoekt t'verlanghen.
      
      
        Eertijdt // scheen mij Cupidos gonst
        Na mijn behagen // self na te iaghen,
        Verblijd // doch ik door slechte const,
        En raad van magen // haar liefd' te draghen.
        Maar t'radt is zeer verdraeyt, Nu blijft die ogen,
        Doort schijn ben ik verleid
        Die liefste van mij scheyt
        Ik ben bedroghen.
      
      
        Mijn hart // zal ik van haar oprecht
        Al eer niet wenden // t'leven zal enden,
        Met smart, mach mij haar trouwe knecht
        En veel ellenden, Venus toesenden,
        Ic blijf mijn lief getrou, t'hart zal niet sneven
        Al treur ic nu in rou //
        ic zal does weerde vrou
        Noch niet begheven.
      
      
        Zij is // mijn vreugd, Mijn troost, mijn hoop
        Naast God mijn Here, die ic begeere,
        Ghewis // om does Princes ik loop,
        Ende t'bloet vertere, t'neemt eens een kere
        Zo sou ik mij t'geluck // geheel toeschrijven
        Verwerpen t'sware juck,
        Bevrijdt van pijn en druck,
        Ghenuecht bedrijven.
