
    
      
        Elk medeghenoot // int gheloove minioot,
        In veel droef aenstoot // met verduldicheyt groot,
        Stelt u vreze besijden,
        Al doen haar exploot // weerelt Hel ende dood
        Hoe menigen cloot // dat elk oyt op u schoot
        Victory zult gij strijden,
        Voor Christum alleyn,
        Wilt u verblijden,
        Want hij int pleyn,
        Is te allen tijden,
        Met zijn volk rein
        Als Capiteyn
        Om haar bevrijden.
      
      
        Wie oyt strijd aennam // in den name van t’Lam
        Oyt hij te boven clam // exempel Abraham
        Hij verwan die Coninghen,
        David Goliam // den Reuse fel en gram,
        Die in den name kwam // der Philistijnsche stam,
        En d’Isaelijten ginghen
        Met trompetten slaan
        Iericho omringhen,
        De mueren vielen saen,
        Veel wonder dingen
        Hebben gedaan,
        Al die vast aan
        Den Here hinghen.
      
      
        Petrus die eerst maar // voor een maagd hadde vaer,
        Dat hij tegen haar // niet dorst belijden claer
        Die sietmen nu voort treden
        Voor een grote schaer // door Gods gheest wonderbaer,
        Sterc als een pijlaer // en heeft den Here daar
        Voor elk en de een beleden,
        Zijn antwoorde tot
        Diets verbodt deden
        Gaf dus een slot,
        Oordeelt ist reden
        Dat zelve God
        Voor u ghebodt
        Zal wijken heden?
      
      
        Dat s’Heeren geest // sterk maakt en onbevreest,
        Stephanus beweest // als groot vervolgh tempeest,
        Stormich op kwam gheresen,
        Iae Paulus die eest // vervolchde, heeft noch meest,
        Gedaan int aertsch foreest // menighe wilde beest
        Weder vochten t’Ephesen,
        Iae met goed verdrach,
        In al zijn wezen
        Op Christum sach,
        En sprak, door dezen
        Ickt al vermach,
        In zijnen dach
        Werd’ ik ghepresen.
      
      
        Van mijns Prinsen hand // wordt mij int eeuwich land
        De troon gheschoncken want // hij heeft tot onderpant,
        Ons zijnen Gheest ghegheven,
        Dus Broeders vailliant // en laat doch u niemand
        Wt sin en verstand // tonen dat God gheplant
        Heeft en daar in gheschreven
        Door zijnen gheest vroet,
        Maar zonder sneven
        V beste doet
        Om vast aencleven,
        Hebt goeden moet
        Gij crijght den hoed
        Van t’eeuwich leven.
