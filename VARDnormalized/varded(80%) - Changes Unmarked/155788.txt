
    
      
        O Altoos heet Autaer,
        Ik volgh uwe Off'rend naar,
        Want duyck ik in den Amstel-vliet,
        Mijn heete vlam verkoelter niet.
        Mijn zuchten // mijn kluchten // voor die does smert
        Eerst schiep in mijn jonck hert.
        Want schreyd' // ik // of vleyd' // ik,
        Zij hield het voor een loose minnaers pert.
      
      
        'k Nam Cupido te baet,
        Die gaf mij dezen raad,
        Voor mijn onlijdelijcke plaegh:
        Gaat Venus off ren alle daegh,
        En smoocken // en roocken // doch steeds voor haar,
        Op 't blaeckende Autaer,
        En treuren // 't zal beuren
        Dat Venus haar, uw zal doen volghen naar.
      
      
        En 't is ook zo gheschiet,
        Want ik deed' anders niet
        Als off ren, van den dag begon,
        Tot 's avonds naar den Avondt-son:
        Ik schreyden // ik vleyden, aan de Goddin,
        De Moeder van de Min,
        't Verdroot mij // met schoot zij
        Haar heete kracht, mijn Lief, ter boezem in.
      
      
        Nu volgt zij mij zo naar,
        G'lijck ik eerst deed' om haar,
        Met tranen in haar droef ghesicht,
        En klaeghd' om 't plagend Venus wicht:
        Nu groeyt ik // nu moey ik, mij gantschlijck niet
        Met haar // noch haar verdriet:
        Al vleyt zij // al schreyt zij,
        Ik houd haar tranen voor een Minne-vliet.
