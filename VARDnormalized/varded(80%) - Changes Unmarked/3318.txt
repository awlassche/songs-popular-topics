
    
      
        Broeders al geprezen // wilt met u verkoen,
        Bystant doen // den armen vray dat aertsche goed,
        Zo zal Christus, wezen // die u zal verghoen
        (Wilt haar voen) in u ellent en tegenspoet,
        En bevrijden altijd van 'tvyanden ghebroet,
        Zo sterckt dan Gods zegen (door loon die elk behaecht // en raeckt)
        Al uwe weiden // en 'tmilt delen niet vertraecht.
      
      
        Wel dien die den armen // ontvangen met vlijt,
        En verblijd // mildelijck deel uit s' hertsen gront,
        Zo zal u ontfermen God ghebenedijt,
        Dus doch zijt // milt, Christen broeders te allen stond,
        V zonden al hij u vergheeft, zijt dit oorkont,
        'tSal God niet vergeten // mer van hem zijn gesint // bemint
        Mocht dit te recht weten // als, een moeder haar kint
      
      
        Wilt den armen geven // van u aertsche schadt
        Want zo wat // gij doet 'tgheschiet in Christi naam:
        Onzen Heer verheven // gheeft u vol u vadt,
        In Dorp, Stad, deelt dan mildich zonder blaem,
        veriaecht gierigheid zo vercrijcht gij d'eeuwighe faem,
        SSoo zal u licht vene / eeuwich opgaan // en staan
        Als d'Hemelsch stene // want gij doet Gods vermaen.
      
      
        Goods troost overvloedich // zal beloonen ient,
        Zonder ent // onze werken vroeg en spade,
        Ent wilt zeer goedich, den armen ontrent
        Zijn u went // tot hun, betrout Goods ghenade,
        Op dat Christus ons van 'tHelsche Serpent ontlade,
        Zo blijven veel zonden // daar mede wij zijn bevlect // bedect,
        Dies 'tghemeen stonden // 'thert vanden armen trect.
      
      
        Nu oorlof al ghelijck // Eedel Princieren
        Regieren // doch u leven wilt te recht,
        Sneeft niet doort aertsche rijk // zijt goedertieren,
        Wilt vieren // Christum, haat giericheyts ghevecht,
        Sulcx werd VVt liefd' bestaan, u misdaden hij slecht,
        Tot God neemt uwen keer // en bemint Christum rein, alleyn
        Geeft milt den armen teer // dus volcht hem groot en cleyn.
