
    
      
        SChoon da- geraet //
        Die als Aurora claer opgaat //
        En alle duysterheyt verclaer- de //
        V amoureus seedbaer gelaat //
        Passeert zeer wijt der maeg- den staat //
        O alderschoonste Rosegaer- de! //
        Ten leeft van aarde //
        Zo wreden man op aerderijck,
        Die u zoet by-zijn oyt beswaerde //
        Dies ik u in mijn hert bewaarde //
        Als enen bruyt- schat weerdelick.
      
      
        V weerdicheyt?
        Drijft mij tot meerder veerdicheyt,
        Als t'quaet gespuys tempeestich dondert:
        Den loon van mijn volherdicheyt,
        Schat ik in zulker eerdicheyt,
        Al had' ik Iouis hof geplondert:
        Gans wtgesondert,
        Vermaert, boven wtlanders // wijn,
        Zydy schoon lief: wt duizend honderd,
        Dat het de goden zelfs verwondert:
        Hoe zoudt ook konnen anders // zijn.
      
      
        Reden ist dan,
        Dat ik u (als vroom edelman)
        Bouen mijn eigen herte minne:
        Deur u, ickt al verwinnen can,
        Gelijk als Iason t'vlies gewan
        Bij raad van zijn liefste vriendinne:
        Als ik versinne,
        Wat gij mij doet, naar wenschen // meest:
        Zo dunct mij dat een Cherubinne,
        Mij voetstaens daalt ter herten inne,
        De vreugd begreep nooit mensen // geest.
      
      
        T is zonder noot,
        Dat ik in vreze van der dood,
        Als Schaepken zonder herder dwale:
        T'vertrouwen van dijn hulpe groot,
        Veriaeght den angst van tegenstoot
        Der helscher geesten t'eenemale:
        Schoonste Omphale,
        Geen tonge dy vollouen // zal:
        Als ik der maagden lof verhale,
        En comen moet ten Principal,
        Zo triumpheert gij boven // al.
      
      
        En of ik stout
        Als Midas conde wenschen goud,
        Al wat ik mij ter handen vogue:
        Ja dat elk Cederboom int wout
        Peerlen, Robijnen, menichfout,
        T'mijnen beliefste altijd drouge:
        En dat ik slouge
        (Deur een tempeest) u van der // hand
        Waert niet al ydel wat ik iouge,
        Naar dien ik mij altijd vernouge
        Met u, voor enig ander // pant?
      
      
        Princess lief,
        Troostbaerste mijnder herten grief,
        Hier mede sluit ik mijn missijue:
        Hoe weerdig ik v oyt verhief,
        Getuigt den laesten minnebrief,
        Waar bij, ik noch gestadich blijue:
        Hoe veel ik schrijue,
        Ten is noch maar een naecte // deucht,
        Maar als ik kome met den lijue,
        Ghelijck een man tot zijnen wijue,
        Zo werdet een volmaecte // vreugd.
