
    
      
        MYn aengenamen vrindt is dood;
        Ach! roep uit // overluydt,
        Ende segh // hij is weg,
        Uit onze ogen is hij vervoert.
        Zijn lichaam is van ziel ontbloot
        Wonder snel // wonder snel // Godes wil
        Wou dat hij neder vil,
        Ons hart is zeer ontroert.
      
      
        Hij weynich tijd was krank geweest,
        Ach! niet meer // als den Heer
        Hem ontbood' // door de Dood
        Uit does werelts bedroefde baen.
        Wie zal niet wezen dan bevreesd,
        Als haar pijl // en haar bijl // zo dree set
        De Dood, die op ons let,
        Die niemand kan ontgaen.
      
      
        Daarom zo oefent u in deughd
        Gij u all' // op dit dal,
        Eer dat komt // en verdomt
        Om uwe zonden, u, God den Heer.
        Dan zult gij wezen g'heel verheugt,
        En verblijdt // als den tijd // komen zal
        Dat binnen 's Hemels wall'
        Gij komen zult met eer.
      
      
        Hij was een man zeer kloeck en vroom
        Zo van re'en // als van le'en,
        G'lijck bekent // tot het end
        Hij geweest is aan yder-een:
        Zo dat het scheen te zijn een droom,
        Dat ons vrind // als een wind // als een rook
        Uit deze wereld doock,
        En haestelijck verdween.
      
      
        Zo dan en kan geleertheyd niet
        Ons de nood // die de Dood
        Ons brengt aan // doen ontgaen;
        Want de Dood yder-een vertreedt.
        Daarom ik bidd' u, voor u ziet,
        En betrouwt // noch op goud // noch op kracht
        U oyt; noch ook op macht;
        De Dood komt eer gij 't weet.
      
      
        En g'lijckerwijs rond omme-gaet
        Satan staechs // 's nachts en daags,
        Om den mensch // na zijn wensch
        Te vervoeren na de helschen brand:
        Zo ook de Dood' nooit stil en staat,
        Maar zij tracht // dach en nacht // in 't gemeen
        Te Moorden yder-een,
        En 't halen uit dit Land.
      
      
        Zo dan gelukkig is die zeer
        Die in strijd // dien hij lijdt,
        Dien hij heeft // niet en sneeft,
        Die ten allen tijd hout go wacht.
        Des toe ziet, vrinden, slaapt niet meer
        In u kwaad // maar verlaat // uwe zond,
        Ei! doet toch dit terstond,
        En na de deughden tracht.
      
      
        O vrinden! dit wordt u gezeid
        Van die geen // wiens gesteen
        Na u jacht // ende tracht,
        Om dat gij daar na leven soud.
        Ach! ach! uwe zonden hier beschreyt,
        Zo zult gij // eeuwich bly // zijn bij God:
        'k Wensch dat gij zijn gebod
        In alles onderhoudt.
      
      
        Bidt dat onze vrind haast vrij mach zijn
        Van 't gesteen // van 't geween,
        Van 't verdrijt // dat hij lijdt
        Om zijn zonden in 't Vagevyer:
        Dan zal verlost uit all' zijn pijn
        Onzen vrind // die ons mindt // bidden we'er
        Ons algemeenen Heer,
        Dit 's de Hemelse manier.
