
    
      
        BLoeyende Jeucht // gij meucht // wel zijn verheucht,
        Want dat u lange heeft ontbroken,
        Is nu beweucht // met vreugd // uit goeder deucht,
        Dat in den mey zeer zoet van roken,
        Mey-bloemken liefelick ontloken,
        Die winter lanc // zeer wranc // met stranc // bedwanck,
        Is vrij en vranc voor ons gedoken.
      
      
        Schoon melodyen // ook wyen // zoude dit zien,
        En hem naar dezen mey niet poogen,
        Melancolien // moet vlien // men ziet geschien,
        Dat de Vyghboomen cnopen toogen,
        En den wijnstoc crijght nu ogen,
        Nooit oogh opslach // en sach // den dach // die mach,
        te Winters geclach dus doen verdrogen.
      
      
        Nooit mey zo zoet // en stoet // gebloeyt int bloed
        Compt hier mijn Bruyt mijn Coninghinne,
        Valt mij te voet // met spoet // storm regen vloet
        Is al vergaen lieve Vriendinne,
        Aanziet hoe de Tortelduyvinne,
        Int Lust-hof zingt // ontspringt // com drinkt //
        Hier drank die blinckt uit reynder minne.
      
      
        Vercooren graen // zeer saen // wilt snellic gaan,
        Tot de Fonteyn zonder verdrieten,
        Den dorst belaen // verstaen // claer als een traen
        Sietmen den vloet daar heenen vlieten,
        Van alles meucht gij daar ghenieten,
        Rijeckende cruyt, en fruyt, den buit, deelt uit,
        Des Meys viertuyt, doet al opschieten.
      
      
        Een Roosken root // mijnioot // staat daar zeer bloot,
        Onder de Doorens uytgesprooten,
        Van 't hemels brood // ter noot // uit s'Vaders schoot,
        Is over al het Veld begooten,
        t'Lam Syons heeft Boek ontslooten,
        t'Hantschrift in tween, gescheen, tgeween, niet cleen,
        Is al gemeen, daar deur verschooten.
      
      
        Princess fier // Rosier // lief goedertier,
        Wilt voor den Winter niet meer schromen,
        Maar maect u schier // tot hier // uwen princier,
        Heeft zijnen loon met hem genomen,
        Hij zal haastelijk tot u comen,
        In s'Hemels Troon, ydoon, de croon, zeer schoon,
        Werd uwen loon met alle vromen.
