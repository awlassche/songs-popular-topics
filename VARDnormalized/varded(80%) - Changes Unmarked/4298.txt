
    
      
        PErsse vol drucx //
        Die ik met drouven geeste tre- de //
        Nu moet ik flucx //
        Drinken den kelck des ongelucx //
        Tot afganck van mijn le- vens vrede,
        Ik die nooit mense iet misde- de //
        Ben van een maegt t'leven ontseyt //
        O woord van onbermhertichede //
        Moordadich zweert zeer scherp van snede //
        Laat d'ure van mij zijn beschreyt //
        Eer ik moet nemen mijn af scheyt.
      
      
        Ik arm verdwaelt
        Ballinck, dole in veel gewesten:
        Wat dinck mij faelt,
        Is haar zo beeldich afgemaelt,
        Daar can niet el dan t'wesen resten:
        Des waters druppen, breect ten lesten
        Een steen: maar aan dit kind // verdrayt,
        En can God moet, noch reden vesten,
        Al is mijn dienst tot haren besten,
        Nimmermeer zij haar vindt // gepaeyt,
        T'zijn pluymen in den wind // gesaeyt.
      
      
        Een sulcken wee
        Heeft mijn iongh herte gans deurtloven:
        Gelijk de zee
        Passeert der drupkens een of twee,
        Gaat ook mijn smert, elcx smert te boven:
        Eylaes! deur Venus vals beloven,
        Ik voor een lam, does Tigre coos:
        Mij ziende nu deerlick verschoven,
        Inwendich branden als een Oven,
        Mishandelt als quaetdoender boos,
        Laat zij mij sterven troosteloos.
      
      
        Tot cleyn verblien
        Is t'sien, t'gehoor, t'gevoel, volhandich:
        Wat cant bedien
        Dat ik de liefste mach aanzien?
        Of horen haar woorden verstandich:
        Den middel smal, wt liefde brandich,
        T'omreyken, met behendicheyt,
        Sulcx maakt mijn sinnen meer wtlandich,
        Verstaande haren haat vyandich,
        T'is een rechte ellendicheyt,
        Blijft ongetroost d'inwendicheyt.
      
      
        Eerloos gespuys,
        Die vrouwen herten cont beceuren,
        En laat confuys,
        Als gij haar ziet opnemen t'cruys,
        T'sy! t'is daad boven nateuren,
        Mocht mij van t'lief, zulks eens gebeuren,
        Met blijschap waar ik saen // becleet:
        Maar neen! veel wenschen werkt doleuren,
        Ik arm gevangen moet nu treuren
        Met anghst, en commer, staan // besweet,
        Den rechter is om slaan // bereet.
      
      
        Rust, mijn gemoet,
        Niet meer suldy bange // sneven?
        V naakt voorspoet,
        Want gij zult achten t'sterven zoet,
        Ten aanzien van dit strange // leven
        Waar sijdy dood, dus lange // bleuen,
        Coemt, dat ik mach omhelzen dy:
        Mijn Cus zal aan dijn wange // cleven,
        Niet goeds can mij een slange // geuen,
        Die mij wil schenden, zulcx dat gij
        Mij aengenamer zijt, dan zij.
