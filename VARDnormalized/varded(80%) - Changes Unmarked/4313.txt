
    
      
        Rein maagd eerbaer // Pilaer //
        Op wien mijn her- te rust:
        O we- sen claer // Waar naar //
        Mijn siele vie- rich lust:
        Och blust // mijn bitter lij- den //
        T'strijden zal zijn ge-ent //
        Sent doch besijden //
        Al die mij benijden //
        Jont mij een court verblij- den //
        Voor langhduerich torment.
      
      
        Eylaes den stoot
        Is groot // helpt mij winnen t'gevecht:
        Blijf ik in noot,
        De dood // sterf ik met groot onrecht:
        Dijn knecht // wilt niet vercleenen,
        V stenen // hert versoet,
        Spoet // u met enen,
        Secours te verleenen,
        Hem, die (met anghstich wenen)
        Verteerd zijns herten bloed.
      
      
        Troostbare bruydt,
        Ontsluyt // doch eens does banden stijf,
        Nergens en spruyt
        Zulk cruyt // (als gij) confortatijf,
        Mijn lijf // al waerder zeven,
        Daar neven // goed, en schat,
        Dat // soud' ik geven,
        Mocht ik u aencleven,
        Ik heb u (als mijn leven)
        Oyt lief, en weerdt, gehad.
      
      
        Es dezen rom
        Oyt stom // beschaemt, gij weet het wel:
        Tooghd' ik oyt om
        Rijcdom // t'uwaerts eenich voorstel,
        Niet el // dan van mijn ermen,
        T'beschermen // ik vercoos:
        Boos // zijn u termen:
        Wie zal mijns erbermen?
        Bij u is cleen ontfermen
        Nu sterf ik troosteloos.
      
      
        In mijnen strijd
        Verblijd // gij u, dat dunct mij vrij:
        Hoe wreet gij zijt,
        Altijd // blijft u mijn herte bij:
        Wee mij // dat ict beginne,
        Ik minne // (met gepijn)
        Mijn // viandinne:
        Die (met blijden sinne)
        Mij (als een valse spin)
        Deurstraelt met haar fenijn.
      
      
        Princess stout,
        Aenschout // doch eens mijns herten gru:
        Waerdy niet cout,
        Gij sout // mij noch ontfermen nu:
        Voor u // blijf ik te pande,
        Met schande // vast gepaelt,
        Haalt // mij te lande
        Of (zo ik verbrandde)
        Mijn leven is d'offrande
        Die mijn onschult betaalt.
