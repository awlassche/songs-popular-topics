
    
      
        O Venus ient, een zwaar torment,
        Een groot allent // lijdt ic bij dezen
        Gij staat geprent, int hart bekent
        Zeer excellent // lieffelick van wezen
        Wilt gij mij niet, verlaten ziedt,
        Alle mijn verdriet // wat mij gheschiet
        Zal ic achten niet // schoon Lief ghepresen.
      
      
        Mocht ic van haar, schoon anschijn claer
        Al zonder vaer, eens troost verweruen,
        Geen goud zo zwaar, koos ik voor haar,
        Een lieflick paar // zouden wij steruen
        Alle mijn iolijt // in dezen tijd
        Mijn Hart verblijd int swerlts crijt,
        Tis meer dan tijd, zal ik troost eruen.
      
      
        O Maechdelick saet, compt mij te baet
        Eert valt te laat, helpt wilt druck veriagen
        Zo zoet ghelaet // zeer delicaet
        Doet Venus raad // schout nyders kwaad
        Ghedult tot mij, die in lijden zij
        Zo waar ik bly, dat seg ik vrij,
        Niemand dan gij // en derf ik claghen.
      
      
        Princess rein, gij zijt alleyn,
        Lief dien ic meyn // schoon bloem verheuen
        Dat weerste greyn // gij zijt certain,
        Int swerlts pleyn // wilt mij niet begeuen
        Gheeft mij confoor // een troostelic woord
        Lief rechte voort, weest niet verstoort
        Mijn beden anhoort // en laat mij leven.
