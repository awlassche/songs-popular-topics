
    
      
        Ik arme kint // waar zal ik keren
        Dan mij verneeren // onder Gods hand
        Dat mij den wind // der valscher leren
        Niet en mach deeren // Heer doet mij bystant.
      
      
        O Heer God Vader // vol van ghenade
        Van alle kwade // mij toch bewaard
        Gij Fonteyn ader // vloeyt vroeg en spade
        Op dijnen zade // van dijnder aard.
      
      
        O God doet mij // veriubileren
        Ik zal lauderen // v name groot
        Al soumen vrij // mij persequeren
        En tribuleren // tot inder dood.
      
      
        Zal ik niet haast // met v verblijden
        In witter zijden // O Vader ient
        Tleuen verbaasd // is niet dan strijden
        Tot allen tijden // druck en ellent.
      
      
        Tot v O Here // wil ik mij gheuen
        Mijn steen verheuen //  mijn toeuerlaet
        Want in v leer // vindick het leven
        Hoe soudick beuen // als gij mij bystaet.
      
      
        Ik bid v seer // om uwe gracy
        Tot elker spacy // verhoord v bruyt
        Vertroost v teere // door iubilacy
        Als swerelts nacy // zal roepen luid.
      
      
        Als haar de mensen // zeer verblijden
        Jaghen en rijden // na swerelts loop
        Dan is mijn wenschen // en mijn belijden
        Wilt mij beurijden // voor sulcken coop.
      
      
        Na v allene // zijn mijne ganghen
        Ik ben behanghen // met zonde snoot
        Na anders gheene //  staat mijn verlanghen
        Wilt mij ontvangen // Heer inder noot.
      
      
        Zal does turbacy // noch langhe dueren
        Dit doet mij trueren // troost mij O Heer
        Want door blamacy // zij veel vervueren
        Door het versueren // in uwer leer.
      
      
        Zo wilt bewaren // sparen en wachten
        Mij door v crachten // mijn  middelaer
        Wilt mij vergaren // leiden en achten
        Bij die gheslachten // onder toutaer.
      
      
        God mijn herder // doet mij toch groeyen
        Door liefde bloeyen //  als rank vruchtbaar
        Ik weet geen weerder // wilt mij besproeyen
        Laat op mij vloeien // v gheest eerbaer.
      
      
        O gij Fonteine // vol goeds ghepresen
        Daar wt gheresen // is tleuen ient
        Gij kont alleyne // ons siel ghenesen
        Wilt bij ons wezen // tot in den ent.
      
      
        Nu bid ic  Prinse // mijn God almachtich
        Weest nu toch achtich // v schaepkens al
        O Heer, bemintse // wt liefden crachtich
        Weest ons ghedachtich // int iammerdal.
