
    
      
        NU zyt // verblyt // tis tyt // te singhen
        Wij bringhen, goede maar
        t'Quaat myt // dit slyt // u stryt // gheringhen
        t'Sal dwinghen // d helsche schaar
        Gij hebt een middelaar
        (Is u wil ghoed) verkreghen
        U moed // tot boed // dies zij gheneghen
        En doet // het ghoed // in dezen nieuwen Iaar.
      
      
        Want deugd // verheugt // met vreugt de sinnen
        Die minnen // zijn woord claar
        O Jeugd // gij meugt // gheneught // hier winnen
        Van binnen // trachter naar
        Volgt Christus wet eenpaar
        Al in u hart gheschreven
        Bestaat // zijn raad // nu te beleven
        En laat // het quaad // in dezen nieuwen Iaar.
      
      
        Elk held // zich stelt // om gheld // te erven
        t'Verwerven // valt hem swaar
        t'Hart swelt // beknelt // oft tquelt // doort derven
        Wilt sterven // zulk misbaar
        Gods ghoeden zijn voorwaar
        Ghemeen huyden en morgen
        Maar t'mijn // en dijn // dat doet u sorghen
        Schouwt fijn // t'fenijn // in dezen nieuwen Iaar.
      
      
        Prins acht // en tracht // met macht // na goetheyd
        Gods soetheyd, proefdy daar
        Zijn kracht // versacht // t'ghedacht // met vroedheyd
        Vermoedheyd //schuwt te ghaar
        Christ helpt u wt ghebaar
        Wildy niets niet verkiezen
        Dit kind // men find // door zelfs verliesen
        Wiet'mint //verwint // in dezen nieuwen Iaar.
