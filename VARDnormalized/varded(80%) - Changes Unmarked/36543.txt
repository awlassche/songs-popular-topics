
    
      
        Cupido hoort myn klaegen,
        Op dit termyn,
        Bij nachten en bij dagen
        Lyd' ik gepyn:
        Myn lief heeft mij begeven //
        Om 't grote goed,
        Och Cupido verheven //
        Mij bystand doet.
      
      
        Wilt mij uw bystand tonen
        Vrouw Venuw kind,
        Myn lief wilt mij niet loonen,
        Zoo nu 't wel schynt,
        Door die grote rykdommen //
        Word zij verblend
        Hierom is 't dat ik kome //
        In dit torment.
      
      
        't Goed is de meeste cause
        Van myn getreur //
        Dus beginn' ik myn' clause
        Met groot douleur:
        Liefde plagt te floreren //
        Aan elken kant,
        Maar 't goed heeft nu te zeere //
        De overhand.
      
      
        Waar ik van goede ryke,
        Ik waar verheugd,
        Zij en zoud' van mij niet wyken,
        Die zuyv're jeugd,
        Ik zoud' wel troost gewinnen,
        Nu treuren ik moet,
        Want zij stelt all' haer' zinnen //
        Op't grote goed.
      
      
        't Goed neemt (laecen) zyn' gangen,
        Als sneeuw op 't veld //
        Daar zij al met verlangen
        Haar zin op stelt //
        Maar daar is liefde puere,
        Die zij klein acht,
        Groeyt van ure tot ure,
        Dag ende nacht.
      
      
        Wat mag ter weireld baeten,
        Goed oft schat //
        Zij moeten 't hier al laeten,
        't Zij dorp ofte stad,
        Wat baet haar grond oft erven
        In 't aardse dal?
        Zoo wel als zij moet sterven //
        Het blyft hier al.
