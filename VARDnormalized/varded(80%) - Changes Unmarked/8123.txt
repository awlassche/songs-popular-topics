
    
      
        O Ionckheydt groene, Bloeyende Ieught,
        In dit saysoene, Schickt u tot deugd,
        Wilt u begheven, tot sedigheyt,
        Alle u leven // daar toe bereid,
        Met sout doch menght Wat gij voortbrenght,
        Spreekt ofte zingt // alzo ons Paulus zeid.
      
      
        Laat u doch raden, o Iongheydt fier
        Teder van bladen zoet van manier,
        Wilt u doch keren, van 's werelts vreugd,
        Vreest doch den Here // terwijl gij meugt,
        Want ziet die dood, spaert klein noch groot,
        Hoe jong, hoe oud, 't wordt al tot hem geveught.
      
      
        En wilt niet treden // om wat nieus te zien,
        Als Dyna deden // ten ouden tien
        Als zij aenschoude, des Lands Heer,
        Kwam zij in rouwe, en zwaar verseer,
        Haar schoonheydt jent // heeft hij gheschent,
        V hier van went // gij jonghe dochters teer.
      
      
        Maar wilt aenschouwen, Het vroom ghemoet
        Van de ghetrouwe // dochter Iephta zoet,
        Die ging versmaden // al 'swereldts eer,
        En liet verbraden, haar lichaam teer
        Op den Outaer // van haren Vaer
        Spieghelt u klaer // en houdt dit voor een leer.
      
      
        Dit differenti, wilt doch aanzien,
        En maakt doch menti // al van des twien
        Hoe d'een bedorven // ende beschaemt,
        De ander ghestorven // heel ghehoorsaemt,
        Dus wilt doch fijn // de ouders dijn
        Ghehoorsaem zijn // sedich en bequaem.
      
      
        Op dat gij princelijcke Dochters fijn,
        Mocht mede van de wijze Maagden zijn,
        Die metten Here, werd ingheleyt,
        Want tot de oneere // zal worden gheseyt,
        Dit droevig Lied // ik ken u niet,
        Gaat in 't verdriet // day voor u is bereid.
      
      
        Daar zal gheschieden // zo groot dangier,
        Na Schrifts bedieden // in 't Helsche vier,
        Wringhen der handen // druck ende pijn,
        Knerssen der tanden // t'eeuwighen tijn,
        Hierom o jeugd // schickt u tot deugd,
        Op dat gij meugt // hier van bevrijdet zijn,
      
      
        En dan meugt horen, het zoet gheluyt,
        Mijn uytvercooren // mijn lieve Bruydt,
        Wilt tot mij treden, ontvangt de kroon,
        En gaat in vreden // in 's Hemels troon,
        Blincken hoort mijn // als der Zonnen schijn,
        Dat zal dan zijn // der uytvercoren loon.
      
      
        Dus wilt vercieren, u Lampen subijt,
        Met go manieren // waar dat gij zijt,
        Laat u licht rijsen, wilt liefde schoon
        Altijdt bewijzen, zo zal Gods Soon,
        V blijde ontfaen // en voor u staan,
        En dienen saen // al in zijn Vaders Troon.
      
      
        Oorlof hier mede, o Ieught eerbaer,
        Gods eeuwighe vrede wensch ik u klaer,
        Dat bij ons blijve // tot 't ende goed
        En ons verstijve // in een vroom ghemoet,
        En na dezen tijd, te zijn verblijdt,
        Wensch ik met vlijt // de Ieught tot enen groet.
