
    
      
        DE swoele Lent die 't al ontfonckt
        Wt haar Moeders schoot,
        'tRood' Angier staat op-ghepronckt
        Met het Roosjen Root.
        d'Lely Wit,, ontspruyt snel //
        Aenschout dit // Pijn-Appel,, komt uit
        En does Groene Laurier-spruyt;
        Ziet den Coorenaer,, staat nu weer vruchtbaar,
        Door den Gheest begloost // en does Opper-Oogen-troost.
      
      
        Wij dancken den Heer die 't al gheeft,
        En ook voor ons strijd;
        Zo dat Orangjen nu verd' streeft,
        Tot des Spanjaerts spijt;
        Vromen Heldt,, zijt ghegroet //
        Die daar stelt // lijf en bloed // te pant,
        Voor ons lieve Vaderland:
        V manhaftich thoont // gij bent waerdt ghekroont
        Van u jonckheyt of // met een Laurier vol Lof.
      
      
        Die Staten die ons Land ghebien,
        Groeten wij nu mee:
        Den Tromp die hem niet gaan ontsien,
        Den Vyandt op Zee,
        Altijdt tracht // om te slaan //
        Dach en nacht // den Maraen // tot niet,
        Zo onder Duyns is gheschiet,
        Daar hij hem manlijck queet // zo ieder wel weet,
        Met zijn macht zeer groot // en verwon de Spaensche vloot.
      
      
        Den Edelen Heer van Warmont
        Groeten mede wij,
        En wenschen dat hij langh ghesont
        Bedien does Oud' Abdy:
        Den Bailjou,, zeer bequaem //
        Groeten nou // wij eersaem // voorwaar,
        Wel-ghebooren allegaer,
        Al die 't Dorp bedient // ik wensch, God verlient
        V al 't saem wijsheid // dat gij goed en kwaad verscheyt.

        Van Konst-Redenrijck,
        Die voor Zoylus niet is vertsaagd,
        Zoo 't nu klaer gheeft blijck,
        Onze lust // dat wou staech //
        't Had gheen rust // alle daech // noch macht
        Om te zien u Woord vol Kracht:
        Nu komen wij aan // met Trommel en Vaen;
        Voor de Maagd een jeugd // van wegen des Oestgeest-vreught.
