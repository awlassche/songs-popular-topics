
    
      
        HEt plach te zijn,
        Als Christenen malcanderen zagen,
        Met vreuchden fijn,
        Hoe staat 'tghemoet? was doen het vragen,
        En gaven vroet // malkander moed,
        Om den Heer te behaghen,
        Hoe goed // hoe zoet // als heunigh vloet,
        Vloeyden daar des Gheests beecken,
        T'hert werd zeer dicht // ghesticht // verlicht
        Vyerigh ontsteken.
      
      
        Vreuchdigh confoort
        Wast bij malkander doens te wezen,
        D'een uit Gods woord
        Heeft een Capittelken ghelesen,
        En de ander sank // een Lied dat klanck,
        Om der Zielen ghenesen,
        Daar spranck // de drank // uit elken rank,
        Aan den Wijnstok devyne,
        Die berghen daar // dropen eenpaer
        Van zoeten Wijne.
      
      
        Maar nu eylaes
        Schijnt elk te wezen d'onversaetste
        Na dit gheraes
        Van synen naasten, op de quaetste,
        Wijs oft manier // te spreken hier,
        Dit is nu t'eerste en 'tlaetste
        Dit vier // dood schier // wel zo veel dier
        Edel ghecochte Zielen,
        Als ter woestijn // door het fenijn
        Der slanghen vielen.
      
      
        Weinig de Schrift,
        Tot beteringhe heeft nu stede,
        Maar kwaad vergift
        Des achterklaps, vol van onvrede,
        Daar openbaer // de lieden maar
        Argher en worden mede,
        Dat haar // valt zwaar // te jaghen naar
        Den vrede Gods voorhanden,
        Die in den graet // te boven gaat
        Alle verstanden.
      
      
        Maar gij die draagt,
        Als ghesalfde // enen Christen Name,
        Na vrede jaagt,
        En verbreydt niemandts kwade fame,
        Want dat verstoort // liefd' en accoort,
        Dus mijdt laster en blame,
        Gaat voort // na 't woord // alsoot behoort,
        Zijn't ergherlijcke leden,
        Snijdt af, steekt uit // doet nae't gheluyt
        Van Christus reden.
      
      
        Maakt overlegh,
        O Broeders, wilt considereren,
        Hoe dat den weg
        Des Heren // die gij moet passeeren,
        Zeer wynigh tijd // is zonder strijd,
        Doch zonder reculeren,
        Met vlijt // doorlijt // volherdich zijt,
        En wandelt anders gheenen,
        Want daar en leit // ter zalicheyt
        Maar dezen enen.
