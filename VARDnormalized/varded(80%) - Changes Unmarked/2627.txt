
    
      
        Het weten en wijs syn, sou mij in alle delen // zeer veel verschelen,
        Het weten leert kunst-greep, ampt, recht spreken en helen // na bootsen spelen,
        Veel krijgt die schrand'righ leest // verschiet van boecken,
        Maar achtmen wijsheyd meest // men moetse in den Geest // bij Gode soecken.
      
      
        De pronck-kennis en yd'le roem van veel te weten // maakt ons vermeten:
        En leyder! om goet-wordingh wij niet eens en sweten // maar God vergeten,
        Wij soecken 't valse schijn // voor 't goede wezen,
        Al werd van ik en mijn // de schaduw niet maar 't zijn // voor 't best geprezen.
      
      
        De liefde loose harsen-kunt pooght steeds te pralle // met schoon te kallen,
        Doch ware wijsheyts hoogste punt, zoekt God in alle // meest te gevalle,
        Wie die in zich bevijnt // die zal bemercken,
        Dat woorden-kunst is wijnt // want 't meeste nut verschijnt // in goede werken.
      
      
        De eersucht drijft verwaende lien tot hoog te spreken // voor slechte leecken,
        Doch eenvoudelijck slecht, en recht berou te spreken // doet meer af-breecken,
        En schricken van het kwaad // ja gants verlaten,
        Daar de ander 't vollick laat // verwondert; want 't verstaat // niet wat ze praten.
      
      
        Wat baet dat men zo zeer verkeert // de tijd verquisten // in schoolsche twisten?
        Al waar men schoon zo schrift geleerd // dat wij 't al wisten // ten maakt geen Christen;
        Maar wel die sabbatteert // van eigen wanen,
        Van zonden en begeert // die zelfs doet 't geen hij leert // die kan vermanen.
      
      
        Het mont-geswets kijflusts, woord-strijd noch 't bitter schelden // die stichten selden,
        Maar baren fell nijt, weer-wraeck en straf vergelden // van 't geen hen quelden,
        Verkeerden van 't gemoet // die last'ren garen
        Het geen een ander doet // het zij dan kwaad of goed // syn dat leeraren?
      
      
        Gij meesters kloeck, vloeijend en rijk in wetenheden // van rijm en reden,
        Dijn pont, dijn licht, dijn Hemel-gaef, wilt toch besteden // tot nut en vreden,
        Jaagt uit dijn eigen hert // dijn huys-vyanden,
        Maar meest ziet elk te vert // en blijft in 't aertsch verwert // met onverstanden.
