
    
      
        Verblijd // met vlijt // in deze Meye tijd
        En spoet u nu int groen, en spoet u nu int groen,
        Vermijt // en slijt // de vileyny en spijt,
        Rein liefde wilt bevroen, rein liefde wilt bevroen
        Haast u int veld zeer coen // om de Mey eer te doen,
        In vriendelijckheyt vergaert // doch met u Liefken paert,
        Bedaert // eer gij bezwaard,
        Verclaert // u aard // en liefd’ bij u bewaard
        In deze Mey vermaert, in deze Mey vermaert
      
      
        Int veld // u stelt // met droefheyt u niet quelt,
        Maar vreuchde doch aenvaert, maar vreuchde doch aenvaert,
        Druck smelt, met ghewelt, weest vrolijck als een Held,
        Toont liefde onbeswaert // toont liefde onbeswaert
        V liefste Lief openbaart // zo werd u liefd’ verclaert,
        Dan suldy troost ontfaen // als gij doet zulk vermaen
        Tot die u hert heeft bevaen,
        Does paen // wilt gaan // met vreugd wilt nemen aan
        Does Mey zeer zoet ontdaen, does Mey zeer zoet ontdaenmmdees
      
      
        In deucht // verheucht // met u Lief als gij meucht,
        Int wout zeer schoon geplant, int wout zeer schoon geplant
        En veucht // geneucht // lieflijck als jonghe jeucht,
        Neemt vrolijckheyt ter hand, neemt vrolijckheyt ter hand
        Spoet u vaerdich int land // daar de voghelkens triumphant
        Zingen met stemmen veel // en prijzen met luider keel
        Ghelijck die boven al is eel,
        Quereel // t’ghescheel // doet van onze Taneel,
        Zo valter geen crackeel, zo valter geen crackeel,
      
      
        Prins tracht // met macht // na haar die u verwacht,
        In liefd’ bloeyende zoet, in liefd’ bloeyende zoet
        u clacht // versacht // haestich rijt op de jacht
        Tot dat gij haar ontmoet, tot dat gij haar ontmoet,
        Verthoonr u haar met der spoet // als een recht minnaer doet
        Die tot haar lieffde draecht // en haar geerne behaecht,
        Niet vertsaecht // te comen bij die maagd
        maar waecht // en vraagt, weest weg eer dat het daecht,
        dat gij u niet beclaecht, dat gij u niet beclaecht.
