
    
      
        JOngvrouwe, schoon Image // zoet,
        Die in u ieught floreert:
        Ik bid' u, geen oultiage // doet
        Aan mij, die suplieert,
        Tot u vriendinne,
        Dat eens u minne
        Mij mocht zijn ontdect:
        Deur anders geen courage // moet
        Dijn knecht zijn opgewect.
      
      
        Coemt nu, en ouervloedich // telt
        Mijn wonden, diep getreft:
        Nu mij Cupido bloedigh // quelt,
        Mij metter hand opheft:
        Of zonder // falen,
        Moet onder // dalen
        Mijn ionck hert // belaen,
        Dwelck hem voor u oodmoedich // stelt
        Om alle smerte // t'ontgaen.
      
      
        Bewijst als goederhande // maagd,
        Compassieus gedult:
        Dat gij mij achter lande // saeght
        Dolen, bij uwer schuld,
        V soud' ontfermen
        Dat ik moet kermen,
        Zonder iet misdoen:
        Als oft gij zelfs te pande // laeght
        Gevangen int prisoen.
      
      
        Verstandich gij van seden // zijt:
        Perfect in reynder aard:
        Te meer gans tegen reden // strijd
        Dat gij u wreet gebaard:
        Wat baet vercleenen,
        Ik ben doch enen
        Knecht, zonder bedroch:
        Gelijk in den voorleden // tijd,
        Ben ik u dienstbaar noch.
      
      
        Weerdig zijdy gestadich // vat
        Princesselick gemoijt:
        Want gij (als een sucadich // bat)
        Van soetheyt overvloijt:
        Als ik u nake,
        Find ik een smake,
        Boven al exquijs.
        Schoon lief, die u genadich // schat,
        Geeft u dijn eigen prijs.
