
    
      
        AL die met Noe in d’Arcke // zijt
        Ghetreden zonder mincken
        En zoekt geen vremde parcken // wijt
        Op Gods toorn wilt ghedincken
        Want die ten zelven stonden // ras
        Buiten die Arcke bevoden // was
        Die moesten tezamen verdrincken.
      
      
        Laat u niet troosten met woorden // maar
        Ghedenckt dat gruwelijck sneven,
        Dat alle Mensen versmoorden // daar
        Die buiten d’Arcke bleven,
        D’welck ons als een figuere // schijnt,
        Alzo men wel in Schriftuere // vijnt
        Niet te vergeefs en ist gheschreven.
      
      
        In Noachs Arcke wilt blijven // vroet,
        Maar buiten u niet begheven,
        D’welc is Gods huis na t’beschrijven // goed,
        Van die Apostelen verheven,
        Men mach daar niet uytwijcken // saen
        Als Dina zoude om kijcken // gaan
        Zij werd gheschent in haar leven.
      
      
        Rahab woonachtich in Iericho 7
        Zij werd bewaard van sorghen,
        Met al haar s’Vaders huis ghelijck // zo
        Maar die bespieders verborghen
        Zeiden, om dat weldadich // werk,
        Zo gaven zij een ghenadich // merck,
        Om haar zo niet te verworghen.
      
      
        Op dat zij in des doots noode // heel
        Niet weg sou worden genomen
        Spraken zij, bindt dat rode // zeel
        Aan u veynster zonder schromen,
        En blijft daar in de dagen // fijn,
        Of wij willen ontslaghen // zijn
        Des eets, van die daar uit comen.
      
      
        Rahab die hield in ghedachten // snel
        Dat zeel aan t’veynster te knopen 7
        Zij wistent haar al te wachten // wel,
        Wt dat huis niet te loopen,
        Want wijf kinders en mannen // ziet,
        In Iericho al verbannen // hiet,
        Zij mostent met die dood becoopen.
      
      
        In huis te blijven gheraden // goed,
        Om niet te comen in blamen
        Als die van Ay met kwaden // moet
        Liepen ter Stad uit al tezamen,
        En werden zo wij weten // bloot
        Van haar vijanden ghesmeten // dood,
        Dat er gheen weder en kwamen.
      
      
        Benjamin hadde in voorleden // tijd,
        Op Israel kwaad benoeghen,
        Zij namen ook zonder reden // strijd,
        Wel veertich duizend versloegen,
        Dat kwam door den verzoeke // boos,
        Van die van Gybia roecke-//loos,
        En al die rest zij verjoeghen.
      
      
        De Israeliten toerusten // haar,
        Voor Gybia met droefhede,
        Die Benjamiten met lusten // naar,
        Liepen meest al uit die Stede
        Maar gene weder in stad // ghekeert,
        Hier door zijn wij al wat // geleerd
        Wt loopen is een kwade zede.
      
      
        Semi ontfinck van Salomon
        Een bevel, hij zoude maken
        Een huis, dat ginck hij te male // doen
        Om zijn vloeckende spraken,
        Binnen Ierusalem // ghepast
        Niet uit te gaan zo werd hem // belast
        Of die dood zoude hij smaken.
      
      
        Een goede meeninghe zeid Semi
        Heeft den Coninck ghesproken,
        Een huis ghetimmert, dit al de // hij,
        En heeft hem daar in beloken,
        Hij woonde daar zonder beswaren // stranck,
        Wel den tijd van drie Iaren // lanck
        Maar zelve heeft hy’t noch verbroken.
      
      
        Daar ontliepen uit den huize // hem
        Twee knechten zonder falen,
        Semi die ginck uit Ierusalem
        Na Gadt om haar daar te halen,
        Niet eens bedenckend’ aan den ent
        Semi na s’Conincx Mandament,
        Most dat met die dood betalen.
      
      
        Gehasy den Knecht van Elisa
        Liep uit den huize met list,
        Achter Naeman, maar niet bij // ra
        Van den Man Gods die wel wiste,
        Zijn hert in die gheschincken // zat
        Zo ghinck hij leughens bedincken // wat
        Om zo te vullen zijn kiste.
      
      
        Als hij na zijnen begeere, dat
        Goed nu al hadde versteken,
        Zo vraechde hem synen Here // plat
        Van waar hij kwam, hij ginck spreken,
        Nerghens en heeft geweest u // knecht
        Daar op bat hem den Man Gods nu // zegt
        En openbaart hem zijn treken.
      
      
        Nu gij tot uwen solase // die
        Gheschincken aan hebt gaan vaten
        Zo zal u naemaels laseris
        Aenhanghen zonder verlaten
        Op u en u zaad, hij ditte // sey
        En werd melaets ghelijck witte // sney,
        Wat mochte hem uit loopen baten?
      
      
        Wtloopen gheschiet zeer menich // fout
        Dat er veel moeten om treuren
        Als mensen worden een weynich // stout
        Zij lichte afdeelen in scheuren,
        Want naar Ioannes luid // vermaen
        Zegt hij, zij zijn van ons uit // gegaan
        Zij hebbent ook wel zien ghebeuren.
      
      
        Op Zee Petrus handelen // lach
        Als Vissers zonder verdolen
        Maar als hij Christum wandelen // sach,
        Op Zee bij t’schip onverholen,
        Zo heeft hij dat begrepen // niet,
        Te treden buiten den schepen // iet
        Eer dat hem worde bevolen.
      
      
        Gedenct wat Pauus uit Rome // schreef
        Aan Timotheum vailliandich,
        Dat er zo weynighe vroome // bleef,
        In Asien, en volstandich,
        Zij keerden af bij vele, dus
        Met Iermogeny en Phigelius
        Op enen onwegh zeer schandelich.
      
      
        Prinse laat ons altijd blijven // vast
        In s’Heeren Tempel met vreden
        Dat t’lichaem tot zijn verstijven // wast
        Ghelijck die van Romen deden
        Om gheen druck noch ellende // scheyt,
        Denckt dat de krone int ende // leit
        Als de strijd worde ghestreden.
