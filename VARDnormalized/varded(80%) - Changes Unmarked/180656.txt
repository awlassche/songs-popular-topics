
    
      
        LOoft al den Here // snel,
        Maar met wandel bekwame
        Volgt na zijn leer // wel
        Zo eert gij zijnen Name,
        Zijn eersame // deucht // gij meucht
        Zo onder den Heyden // breyden,
        Leiden // zo wandel met vreugd
        Door nieu gheestelijcke jeucht.
      
      
        Ieucht ende nieuwe kracht
        Zal in den Mense varen
        Die op den Here wacht,
        Dat hij zonder beswaren,
        Op zal varen // dat // niet mat
        In loopen gheresen // dezen
        Wezen // en zal iet of wat,
        Noch ook moede op den pat.
      
      
        Eertijds veel moede zijn
        Op s’Heeren weg bedeghen
        Gaande door die Woestijn,
        Waren kwaad om beweghen,
        Loon zij creghen // maar // zeer zwaar,
        Laat ons deze werken // merken
        Sterken // onzen roep eerbaer,
        Christum vierich volghen naar.
      
      
        Vierich zijn in den gheest
        Moeten wij te allen tijde
        Voor den Here bevreesd
        En in die hope blijde,
        Wie hier lijde // iet // verdriet
        Neerstich tot den Here // kere
        Leer // aan hem ende ziet
        Wat ons voorbeeld is gheschiet.
      
      
        Een voorbeeld op den Berg
        Dat was Moysem bevolen
        Te volghen zonder ergh,
        Dus om niet te verdolen
        Een recht school-kint // begint
        Na zijns Meesters zeden // treden
        Heden // ook wie God bemint
        Moet hem ghelijck zijn ghesint.
      
      
        Nieuwe ghesint oprecht
        Laat ons Christelijck blijcken,
        Want t’is ghenoech den knecht,
        Zijn Meester te ghelijcken
        Zonder wijken // vant // verbant
        Ik ons in ghenade // rade
        Stadelijck te blijven, want
        Niemand can ontvlien Gods hand.
      
      
        Pooghen om te ontvlien
        Moeten wij nu te voren
        Neerstich zijn, en toe zien,
        Dat wij na het behooren
        Den Gods tooren // saen // ontgaen,
        Ons na zijn behaghen // draghen,
        Iaghen // en na vrede staan,
        Al zijn woorden onderdaen.
      
      
        Al zijn woorden zijn waar,
        En ons zeer wel van noode,
        Om bij te leven, maar
        Anders waren wij dode,
        Zo van brood // t’lijf // wordt stijf
        Gods woorden ghemoeden // voeden,
        Spoeden // en geven beclijf
        In Godsalicheyts bedrijf.
      
      
        Nut is Godtsalicheyt
        Voor al die haar aencleven
        Vrij zonder dwalicheyt
        Hier en int ander leven
        Toe gheschreven // is // ghewis
        Loon die wij ghepresen // lezen,
        Dezen // schat heeft zonder mis
        Een grote beloftenis.
      
      
        T’is ons ghewis belooft
        Zo wij in alle stukken,
        Wassen aan t’rechte Hoofd,
        Niet wijken door verdrukken
        T’sal ghelucken // wel // want snel
        Zal hem God verschijnen // zijnen,
        Pijnen // al die zijn rebel
        Zijn Euangelisch bevel.
      
      
        Een bevel boven al,
        Laat ons int herte vaten,
        Maar t’minste niet en zal
        Men gheensins achter laten
        Al mocht baten // in // ghewin,
        Dat ons ziele schade // dade,
        Kwade // Coopmanschap met min
        Doen wij tegen Christi sin.
      
      
        Neemt dezen sank // in danck,
        Broeders, laat ons vailliandich
        Wandelen cranck // noch manck
        Al is druck menigherhandich
        Die volstandich strien // henlien,
        Zal gheschien // d’ydoone // schone,
        Krone // t’Rijck der heerschappien,
        Boven in s’Hemels verblien.
