
    
      
        PEerle der vrouwen // zoet //
        Verciert met aller deugden //
        In dijn aenschouwen // moet //
        Den geest vrolick verjeugden //
        Noch even zeer //
        ben ik dijn knecht volveerdich //
        Gij zijt het mij weerdig //
        Al waert duizend maal meer.
      
      
        Inwendich lijd' ik // grief,
        Van buiten toog ik vreugde:
        Selden verblijd' ik // lief,
        Want gij (die mij verheugde)
        In mijn dangier,
        Zijt als Daphne voor // vluchtich,
        Nochtans gij door // luchtich,
        Zult blijuen mijn laurier,
      
      
        Eertijds dijn herte // louch
        Op mij, als lief tot lieve:
        Ik (die nooit smerte // drough)
        Ben nu vervult met grieve:
        Mijn bloed verschrict,
        Mijn ieught versmelt in pijne,
        Cruyt, noch Medecijne,
        Mijn droevich hert verquict.
      
      
        Tis doch verloren // werk
        Geclaeght, ik ben in handen:
        Ik zie dijn thoren // sterk
        Te my-waerts vierich branden:
        Onsalich man,
        Wat deed' ik oyt geboren,
        Zij wil mij versmoren
        Die mij meest helpen can.
      
      
        Es mijn geen wrake // noot
        Van tgeen ik nu verhale:
        V is d'oorsake // groot,
        Vileyn en teenemale:
        Gij waert gelooft
        Te spreken u belieuen,
        Wreede mensen dieuen,
        Gij hebt mijn lief geroofd.
      
      
        Ras op, het scheiden // naect,
        Adieu, lief ongestadich,
        Die int verbeyden // haect
        Naar troost, weest hem genadich,
        Neemdy nu aan
        Te zijn eens anders vrouwe,
        Toocht hem meerder trouwe
        Dan gij mij hebt gedaan.
