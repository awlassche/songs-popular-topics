
    
      
        WAerde Kuysheyt, Hemels Kint,
        Van den Heer en yder een bemint:
        Wiens gepronck // wiens gegluer //
        En ghelonck, En gelaat
        Is der Deughden Allerhooghsten graedt:
        Het is een Deugt,, 't Is een vreugt //
        Die de Jeugd Meer bepraeldt,
        Als het Goud, Hoe zeer 'et straeldt:
        't Is een schat // Die het pad //
        Na de Stad henen leidt,
        Daar men nooit ween-ooght, snickt, of schreyt.
      
      
        Ioseph, Spiegel van de Ieught,
        Deftigh beeldt van eerbaerheydt en Deugd;
        Sedigh hert // Die de schimp
        En de smert // Van de Vrouw'
        Vaeck wou lijden // En noch blijven trouw;
        Die Liever vlood // In de nood,
        Totte dood // Dan u Heer
        Gants te brenghen // Tot on-eer:
        Die u zinn'// Op 'er Minn'
        En ghewinn'// Niet en hecht',
        Maar u droegh // Trouw, ghelijck een Knecht.
      
      
        Lof eerwaerde Kuyscheydts glans,
        Gij verdient een versse Rose-krans,
        En een Croon // Op u hoofd,
        Tot een loon // Voor u strijd:
        Om dat gij ons // Zo een Voorbeeldt zijt,
        Op Kuyscheyts baen // Die wij gaan
        Nu voortaan // Stadigh voort
        Moeten treden // Alst behoort:
