
    
      
        EEn is alleen // mijn reen
        Duive minjoot,
        Hier gheacht kleen // maar gheen
        Voor mij zo groot,
        Sterk als de dood // is liefde stranck,
        Ik was ter noot // van minnen krank,
        Als ik u schanck // den drank zeer excellent,
        Schone Bruydt jent // bekent,
        Boet mijn verlanck // den ganck
        Te mywaerts went.
      
      
        Schoon onghelaeckt // volmaakt
        Is mijnen Man,
        Mijn herte waeckt // en blaeckt
        In liefden dan.
        Gij dochters van // Ierusalem,
        Spreekt mijn vriend an // en segghet hem,
        O lief u stem // zeer tem // maect mij verheugt,
        Gheeft mij kracht, deugd // en jeugd
        Dat ik op clem // niet strem,
        Bij u in vreugd.
      
      
        Toont mij nu dijn // aenschijn,
        Vriendinne zoet,
        V borsten zijn // als Wijn,
        Lieffelijck goed,
        Der liefden gloet // heeft groot virtuyt,
        Gheen watervloed // en bluscht haar uit,
        Komt uit dat ruyt // besluyt // der Leeuwen wreedt,
        Doet aan u kleed // bereedt,
        Maakt u rein spruyt // mijn Bruydt
        Wt liefden heet.
      
      
        Met wanghen vocht // u zocht
        Ik lief minnaer,
        Och of ik mocht // ontknocht
        Van lyden zwaer,
        Vrij kussen daar // buiten versint,
        V die eenpaer // mijn ziel bemint,
        Want als die kint // begint // te baren wel,
        Mijn pijn is fel // maar snel,
        Bij u troost vint // men int
        Cruys end' ghequel.
      
      
        Heft op u hoofd // geloofd
        Mijn woord ydoon,
        V vlees verdooft // zo rooft
        Niemand u Croon,
        In mynen troon // werd gij gebracht,
        Tis daar zo schoon // van mij gewracht,
        Nooit oogh en sacht // dus lacht // inwendich meest,
        Blijd' in den gheest // die beest,
        Met haar gheslacht, haar macht
        Doch niet en vreest.
      
      
        O Bruydegom // mijn rom
        Zijt gij altijdt,
        V eyghendom // daarom
        Werd zeer benijdt,
        Maar in den strijd // doet mij bystant,
        Breedt ende wijt // u sterke hand,
        Men vint int land // niemand // getrou als gij
        Dus blijf ik bij // u vrij,
        Vast ingheplant // ja want
        Twee een zijn wij.
      
      
        Princess saen // vergaen
        Zal u verdriet,
        Want elken traen // af dwaen
        Zal ik u, ziet,
        En twijffelt niet // maar op mij bout,
        Dat gij als riet // niet en verflout,
        Mijn woord aenschout // betrout // op mij gepast,
        Wie in u tast // vreugd vast,
        Quelt oft benout // zeer stout,
        Mijn oogh' aentast.
      
      
        Princelijck Heer // wanneer
        Oordeelt gij recht,
        Met valse leer // men zeer
        Tegen mij vecht,
        Elk wederseght // mij als ghestoort,
        Veel listen legt // die Slangh' aen't boord,
        Maar lief u woord // accoort // is een licht vat
        Op mynen pat // op dat
        Ik door de poort // ga voort,
        Binnen de Stad.
