
    
      
        In ‘s Heren Naam moet ik beginnen,
        Een troostbaer Lied voor uwe sinnen,
        Waar bij // dat gij,
        Door dit beschencken // ons meucht ghedencken.
      
      
        Al zijt gij wijt van ons gheseten,
        Nochtans en zijt gij niet vergheten,
        Veel min // noch in,
        Des Heren ogen // die ‘t al vermoghen.
      
      
        Nu ‘t gheen dat gij hebt te aenschouwen,
        Dat is ‘t exempel van de Vrouwen,
        Die God // tot lot
        Hebben verkooren // zijn wil te horen.
      
      
        Neemt eerst aan Sara deze Leer,
        Die haren Man genoemd heeft Here,
        Haar wil // was stil
        Overghegheven // in haar Mans leven.
      
      
        Een Vrouw kan door ootmoedicheden,
        Met haren Man leven in vreden,
        Niet zij // maar hij,
        Komt toe ‘t regieren // en al ‘t bestieren.
      
      
        Konden de Vrouwen dit bevaten,
        Het soud’ haar dick veel onrust baten,
        En ‘t sou // de trouw,
        Met liefd’ bespoeyen // den Geest doen bloeyen.
      
      
        Elkanas Wijf leert hoe aendachtigh,
        Een Vrouw haar nood moet wezen klachtigh,
        Aan hem // die stem,
        Noch woord behoevet // als ‘t hert bedroevet.
      
      
        Neerstich en in armoed’ gheduldich,
        Bewees haar Hanna eenvuldich,
        Zij was // te pas,
        En wel te vreden // in kleynicheden,
      
      
        Koninghen zijn het hert ghewonnen,
        Van Vrouwen, als wij lezen konnen,
        Zo heeft // beleeft,
        Hester verkreghen // haar Mans beweghen.
      
      
        Abigail was een wise Vrouwe,
        Zij sweech in tijd van druck en rouwe,
        Met goed // doen vroet,
        Kon zij ‘t gheresen // onheyl ghenesen.
      
      
        Rein was Maria in haar leven,
        Deur ghehoorsamich ondergheven,
        In vlucht // ghesucht,
        Boog zij haar herte // tot pijn en smerte.
      
      
        Een voorbeeld van Godvruchticheden,
        Is Priscilla, wiens kloeke reden,
        Verwon // en kon,
        Leren manierich // Apollo vierich.
      
      
        Laat does exempels u behaghen,
        En wilt u zelven daar na draghen,
        De Kroon // en ‘t loon,
        Van deze vromen // zult gij bekomen.
      
      
        Sijt God bevolen nu in vreden,
        Draagt ons ghemeynschap in ghebeden,
        Dit deel // blijft heel,
        En onghescheyden // in eeuwicheyden.
