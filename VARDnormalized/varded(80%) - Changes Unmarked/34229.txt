
    
      
        Gij amoureuse geesten,
        Verquist den tijd als beesten,
        En versmacht // dag en nacht,
        Onbedacht // door de kracht
        Van Venus zoeten brandt,
        Als Cupido ter hand
        Zijnen gulden boghe spant,
        En komt u hert door-wonden,
        Hoewel hij heeft gebonden
        Zijn ogen toe,
        Wie weet hoe // al is ’t moe;
        Nochtans kan hij geraken
        V gesicht // door zijn schicht,
        Want dit wicht // vlieght zeer licht,
        Die ’t al in ’t vier doet blaken.
      
      
        Al waert gij duizend mijlen
        Van hem, nochtans zijn pijlen
        Zullen ’t hert // wonder pert
        Met een smert // wonden hert,
        Niemand hij en vergeeft;
        Wie heeft ‘er oyt geleefd,
        Die hij niet door-schoten en heeft?
        Daarom draagt hij de vlogels:
        Want hij schiet zonder kogels:
        Maar het gepeys,
        Zydy ’t wijs // houdt den peys,
        Waar door hij wordt verwonnen,
        Schout verdriet // en gaat niet
        Dat gij ziet // dat van ’t riet
        Klein garen wordt gesponnen.
      
      
        Die haastig zijn verbolgen,
        En hunnen kop dul volgen,
        Hoort doch wel // wat een spel
        Door ’t gequel // ik vertel,
        Wat den mensch dickmaels naakt,
        Als hij somtijdts aan die raeckt,
        Hoe inwendig hij dan blaeckt,
        Aen-siende dan geen reden
        Door ongeruste zeden,
        En mallen raad,
        Doet veel kwaad // ende schaedt:
        Ik wil te kennen geven,
        Neerstigh hoort // ongestoort,
        Hier komt voort // hoe vermoort
        Elivia liet haar leven.
      
      
        Onkuysheydt doet bederven,
        En menigh mensch vroeg sterven
        Wel gesondt // ’t hert gewondt:
        Schouwt die sond’ // t’aller stond;
        Op dat gij niet en vaart,
        Gelijk u hier wordt verklaart,
        Van den onkuyschen aard,
        En wilt hier uit wel leren,
        Hoe de Romeynsche Heren
        Zijn wel zo saen
        Al vergaen // in de baen,
        Die Venus spel hanteren.
        Dezen sank // neemt in danck,
        Al is ’t lanck // zonder dwanck,
        Zwijgt stil, laat ons geberen.
