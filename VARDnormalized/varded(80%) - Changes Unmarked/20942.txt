
    
      
        Maria Maagd, en Moeder
        Van Iesus onze Behoeder,
        Aanziet // 't verdriet // dat hier aan ons geschied.
        Den Helschen loosen loerder,
        Der zielen boos vervoerder
        Wilt niet // verbiedt // te doen dat God gebiedt:
        Hij komt gestadigh quellen,
        Ontrusten ons gemoed;
        Hij wil ons verzellen
        Bij 't vuyle helsch' gebroedt.
        Wij gedurig zijn in grote strijdt,
        Hij ons bijt // hij ons aendoet alle spijt.
      
      
        Tot u w' onze stem verheffen,
        Belet gij Satans keffen.
        Uwe kracht // uwe macht // hij vreest, en uwe wacht.
        Wilt vroom onze saeck betreffen,
        Maakt onze wegen effen;
        Ei! jacht // ei! tracht // ons t'helpen dag en nacht.
        Gij zijt die hebt vertreden
        Het hoofd van 't oud' Serpent;
        Gij zijt tot wien heden
        Onze ogen zijn gewend.
        Moeder! wilt verhoren onze gebe'en,
        Ons gesteen // ei! drooght af ons drouf geween.
      
      
        Gij zijt die Vrouw bevonden,
        Die Satan heeft verslonden;
        Wel-aen // wilt staan // ons bij dan in does baen.
        Wij hebben meenigh wonden
        Gekregen door onze zonden;
        Belae'n // vergaen // zijn wij van Christus Vaen:
        Wij willen weder keren
        Tot Christum onzen Heer;
        Wij door u begeren
        Te komen tot hem we'er.

        Let, Maget! op onze zielen,
        Want Satan op onze hielen
        Staeghs let // zijn net // heeft hij voor ons gezet.
        De Hell' wil ons vernielen;
        Des wij voor u ne'er knielen,
        Belet // verplet // zijn list door u gebedt.
        O waerdeste Vrouw der Vrouwen!
        Uwe hulp wij roepen aan:
        Maakt dat wij behouwen
        Door u, in d'Hemel gaan.
        W'hopen dat 'et door u zal geschie'n
        Dat verblie'n // wij ons zullen, en God zien.
