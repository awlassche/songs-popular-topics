
    
      
        SMorghens vroeg al in het kriecken
        Zo als het Vogeltje sweeft op syn Wiecken,
        En alst syn krop // met gesang zoet gaan steken op,
        Zo teegh ik ras is wt // met d’hangel roe inne schuit,
        En hebme ghestelt // neder int Veld,
        Doch ik vingh niet dus creech het een stuyt.
      
      
        Waar over ik doort langh staeren
        After een Bosje ben ghevaeren,
        Daar ik wat sliep // korts daar na een Meysje riep,
        Hou hangelaer vangje wat // ik schoot op wel wie is dat,
        Riep ik terstont // doen sprackse heel pront,
        Hoe leghje daar zo afghemat.
      
      
        Om dat ik gheen hoeck can krijghen,
        Ay seyse k’bid jou wilt dat swijghen,
        Set u bij men neer // want het is het braefste weer,
        Het heete Sonnetje breckt // het Wintje naet zuiden trekt,
        Ghewenster dach // jy niet en mach
        Becomen daarom heb ik u gheweckt.
      
      
        Stracx nam ik myn Anghel roetje
        En kwam bij dat Suycker soetje,
        En say wijstmen een plaets // zo binne wij de beste maets,
        Doen sprackse met woortjes schuw // weg weg ik geck met uw,
        Ik boert slechs was // wel ach jy dat
        Ik ben ghemeenelijck toch zo nuw.
      
      
        Wel seyd ik dat lijckt wel scheren,
        Ik loof ik seltje dat nu verleeren,
        Doen bloos zij quamshuys // en riep Ach laat myn huis,
        Ik verstond wel uit haar reen // waar datse strackx wou heen,
        Ik vraechdent haar dra // zij sey neen of ja
        Maar heetfet stilswijghens gheleen.
      
      
        Naet stoeyen hiet en warmpjes
        Namse mijn datelijck in haar narmpjes,
        En gaf myn een soon // stoutert gij moetet niet weer doen
        Of jy raeckt weer in den hel // ja seyd ik verstaje wel,
        Wat datje meent // wij syn vereent
        Als jy begheert ik stracx weer sell.
      
      
        Gheeft myn hier op een kusje
        Say zij en bruyckt vrij weer u lusje,
        Ghelijck wij terstont // deden soetjes mond aan mond,
        Stracx heb ik op haar bescheyt // weer fraytjes aengeleyt,
        Een flautje zij kreech // waar doorse neer seech
        Ja haar oochjes stonden verdreyt.
      
      
        Ik zonder langh te lijmen,
        Riep ach! ach! myn lief is aent swijmen,
        En voorts neerwaerts clam // tot ic een hand vol waters nam
        En sprengdēt in haar gesicht // dat heefter voorts verlicht,
        Doen sprac does schoon //  jy soutmen wel doen
        Eer myn rechte sterf-dach kwam.
      
      
        Voort nam ik dit ziecke Bruytje,
        Settense achter in myn Schuytje,
        En voort myn koers nā // tot ik aēt grote loofvelt kwam,
        Doen spracse met stemmige reen // ic moet na Amsterveen
        Daarom laat myn geen // zeer garen gedaan
        [...] en liet ze kuyeren heen.
