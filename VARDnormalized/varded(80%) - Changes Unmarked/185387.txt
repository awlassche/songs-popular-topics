
    
      
        Gij Christelijcke lieve jeucht,
        Cleyn spruytjes, jonck van jaren,
        Laat varen // alle ydel vreugd
        Verheucht // in deucht
        Zo veel gij meucht,
        En wilt u Ziel bewaren.
      
      
        Bewaard u Ziel en zorge draagt
        Dat gij niet wordt bevanghen,
        Van slangen list die menich plaeght,
        Maar jaagt // en vraagt
        Wat God behaagt,
        Beveelt hem al u ganghen.
      
      
        Let op u ganck, en zijt geen kint
        Al in ‘t verstand onvroedigh,
        Maar spoedigh // zoekt en ondervint
        Hoe blint // zij mint //
        en heeft ghesint
        De wel-lust wanckel-moedigh.
      
      
        Wel-lust bedrieght als zij verblijd,
        ‘t Iolijt // is niet ghedurich
        Doleurigh // haren dienaar krijt
        Als tijd // is kwijt //
        dan heeft hij strijd
        In zijn gemoet zeer treurich.
      
      
        En daar ‘t gemoet zo knaegt en wroet
        Onsoet // is sulcken leven,
        Bedreven // en vol anghstigh bloed,
        Maar doet // gij goed //
        in u gemoet
        Werd hogen troost gegeven.
      
      
        Wie dezen troost is toe ghevloeyt
        Besproeyt // van ‘s Heren zegen,
        Ter degen // hij in deughden blpoeyt
        Hij roeyt // en snoeyt //
        al watter groeyt
        Door ‘t vleysschelijck gheneghen.
      
      
        Gheneghentheyt is den aenvanck
        Des sonds, dus wie het kwade
        heeft stadelijck in zijn bedwanck
        Lofsanck // en danck //
        van die ontfanck
        Bewijs hij vroeg en spade.
      
      
        Neemt aan in danck dit cleyn gedicht,
        Zijt den Leeraer der scholen
        Bevolen // die u onder-ricjt
        Hij sticht // verlicht //
        en geeft ghesicht
        Om nimmermeer te dolen.
