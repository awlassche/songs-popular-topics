
    
      
        DEurluchtich weertste lief //
        Troostbaer Rosemarine,
        Bereid der minnen // drank,
        Tot mijn inwendich grief //
        Weet gij de medecin,
        Tot al mijn sinnen // cranck,
        Gemerct mijns herten brief //
        Spreect van veel smert en pijne.
      
      
        Aerdich schoon gulden vat,
        Dierbaerder weet ik gene:
        Zoetgeurich rosen blat,
        Die ik met herten men,
        Al t'goudt in een // gewracht,
        Is bij mij cleen // geacht,
        Boven al s'weerelts schat
        Bemin ik dy allene.
      
      
        Vriendelijk eerbaer saet
        In mij is cleen beromen:
        V amoureus gelaat
        Heeft mij mijn hert genomen:
        De wonde bloedich // smert:
        Voor mijn weemoedich // hert
        Vind' ik den besten raad,
        Tot u om troost te comen.
      
      
        Iont een verworpen caf
        Honig, voor bitter bruine:
        Gemerct ik mij begaf
        (V lief) eigen te zijne,
        Mint gij een ander // pand,
        Mij slaande van der // hand,
        Zo vaer ik dan int graf
        Al waar de weerelt mijne.
      
      
        Den brand wordt mij te groot,
        Hoe can dijn hert vervriesen?
        Och lief! laat mij ter noot,
        Dijn hulpe niet verliesen:
        Spreect een verheuen // woord,
        Zo mach ik leven // voort,
        Want ik tot in den dood,
        Geen ander zal verkiesen.
      
      
        Troostbare, gij vermeught
        Veel meer, dan nijders wanen,
        Thien iaer ben ik verheugt,
        Mach ik dijns eens vermanen:
        Deur u by-wesen // zoet,
        Ik gans genesen // moet,
        Maar t'scheyden (zoet ieught)
        Cost mij een zee vol tranen.
