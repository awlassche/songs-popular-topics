
    
      
        Gij Ionghmans fray
        Die vreugd zoekt te hanteren
        Gaat plant de Mey
        Met u liefste vol eeren,
        Op dat u liefd' macht groeyen // daar
        En met haar in liefd' bloeyen // klaer
        Wilt snoeyen // haar, wilt snoeyen // haar
        Die nijdigheid useren.
      
      
        Als gij dan koen
        Met u Lief zijt gheseten
        Buiten int groen
        Laat droefheyt zijn vergeten,
        En brengt Harpen en Luiten // bij
        Ook Rethrozijnsche kluyten // bly
        Wilt uyten // vrij, wilt uyten // vrij
        Om haar als lief te heten.
      
      
        Loer-ooght en winckt
        Minlijck aan beide zijden
        Kuskens bedinckt
        Onder u lieflijck vrijen,
        Op datse u mach kroonen // wis
        En liefd' met liefd' te loonen // is
        Wilt tonen // fris, wilt tonen // fris
        Dat gij garen sout bedijen.
      
      
        Zo gij dan weet
        Dat zij u gaat toe draghen
        Haar liefd secreet
        Zo wilt u lust bejaghen
        Om Venus te verfraeyen // hoort
        Haar akker te besaeyen // spoort
        En maeyen // voort, en maeyen // voort
        Vruchten na u behaghen.
      
      
        Maar nimmermeer
        Wilt brenghen in benouwen
        Noch om haar eer
        V fleur van alle Vrouwen,
        Want zij is haast daar van // berooft
        maar op ere haar dan // belooft
        Als man // en hoofd, als man // en hoofd
        Om voor u lief te houwen.
      
      
        Princess jent
        Ontvangt op deugd // verheven
        Des May bekent
        Die ons veel vreugd // gaat geven,
        Want door des Mays virtuyten // zoet
        Ons vele bloemkens spruiten // goed
        U buiten // spoet, u buiten // spoet
        Daar wordt gheneught // bedreven.
