
    
      
        BRoeders houdt vast aan Gods gebot,
        Al wordt gij hier der mensen spot,
        Haar fabel gerel // haar liet, haar spel,
        Mensen saen // al vergaen,
        Maar in dien men can
        Volherdich God aencleven // vroet,
        Men crijght dat eeuwich leven // zoet,
        Wat noot ist dan?
      
      
        Die Christen Mensen zijn of zij
        Ter Weerelt slechs reisden voorbij
        Ghetroost int cruys, om wezen t’huys
        Bij den Heer // wat verseer
        Wij in dit ghespan
        Lijden, zo wij gheraken // daar,
        Zo staan geheel ons zaken // claer
        Wat noot ist dan?
      
      
        T’schijnt wel een treurich dingen kwaad,
        Als s’Heeren castijdinghe slaat,
        Doch t’aertsch verdriet // gaat haast te niet,
        Maar den loon // wonder schoon
        Is zeer groot hier van,
        Als naar zo weynich lijden // court
        Ons deel t’eeuwich verblijden // wordt
        Wat noot ist dan?
      
      
        Mishaghen wij de Mensen // fel,
        Dat zij ons niet en wenschen // el,
        Dan lijden, druck // en ongheluck
        Als die weert // zijn het zweert,
        Iae dood oft ban
        Oft zegt in ons veel blamatie // naar
        Staan wij wel in Gods gratie // maar
        Wat noot ist dan?
      
      
        En hadde s’Heeren woord gedaan
        David waar in ellendt vergaen,
        Want t’eenen troost // hem God altoost
        Ter noot gaf // dezen staf,
        Daar hij op lan,
        Dus als ons komt benouwen // last,
        Zo wij Gods woord betrouwen // vast,
        Wat noot ist dan?
      
      
        De Here groot zonder ghelijck
        Ginck door veel lijdens in zijn Rijk,
        Wie nu ter tijd // des lijdens strijdt
        Wel ghesint // ook verwint
        Zo hij verwan,
        Die werd ghecleedt met witten // al.
        Alle dinck hij bezitten // zal,
        Wat noot ist dan?
      
      
        Paulus hadde victory, want
        T’gheloof als enen weerden pant
        Behouden rein // had hij int pleyn,
        Ongheschent // tot den ent,
        Zo hij began,
        De croon was zijn verwachten // schier,
        Konnen wij hem gheslachten // hier,
        Wat noot ist dan?
      
      
        Prinse wat noot ist oft het vlees
        Niet al en gaat na zijnen heesch?
        Als na Gods wensch // den binnen mensch
        T’alder stond // is ghesont
        En zo cloeck Man?
        Dat men tot s’Heeren glory // recht
        In zijnen Naam victory // vecht,
        Wat noot ist dan?
