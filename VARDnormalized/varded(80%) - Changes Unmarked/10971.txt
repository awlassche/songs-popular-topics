
    
      
        Als liefde blent // myn ghemoedt had bevanghen,
        Heeft mij ghewent // rontsom nu hier en daar
        End’ met rotment // mij ghesocht te verhanghen,
        Maar reden jent // heeft mij gherecht voorwaar
        Tot dat zij mij bracht bij u licht zeer claer:
        Maar zo haast uwen ogen // schijn
        Mij int hert dede dooghen // pijn,
        Wast niet in tvermoghen // myn
        Te weerstaen zijn kracht zwaar.
      
      
        Cupidoos licht // meynt hem aan mij te comen,
        Van tvier ghesticht // in myn amoureus hert,
        Maar tis tghesticht // van u Lief uytghenomen,
        Twelck mij verplicht // aan u nu houdt met smert
        Dies u die eer alleen gegeven werd
        Weg liefd’ met u ontspresen vier,
        Gij condt mij niet doen vresen // hier,
        Maar myn liefkens wezen fier
        Was in die const expert.
      
      
        Een ghestadighen strijdt // sedert ik u beminde,
        Bevingh subijt // myn gemoet zeer bezwaard
        Liefde somtijt // doet mij wat vreuchts ontbinden,
        Maar wanhoops krijt // gij zyt te wreedt van aard,
        deen in myn vreugd, en dander lijden baert:
        Zo dat ik int verdriet // belaen
        Int einde gants te niet // zal gaan,
        Een rijk dat men ziet staan
        In tween, is zwak van aard.
      
      
        Tegens myn danck, zeer wreet gij u gaat tonen,
        moet ik met dwanck, u minnen dach en nacht
        Met lyden strack // gaat gij myn Lief beloonen,
        Als gij verlanck // zoekt in tgheen dat ik wacht
        Dies Goddin Lief, myn liefde nu betracht
        V fierheyt gants besyden // stelt,
        Aanziet hoe mij het lyden // quelt,
        En dat myn verblyden // smelt
        Als gij mij Lief veracht.
