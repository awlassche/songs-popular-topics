
    
      
        TOt maechdelijcke spruiten
        Mijn dicht // ik richt // eersaem,
        Die binnen ende buiten,
        Aan gheeft // ook leest // lichaam,
        Heylich rein en bequaem,
        Ook meer // den Heer // te achten,
        Dan t'recht // van d'echt // betracht,
        Dit leert Paulus te saem.
      
      
        Ik wensch u voor een groete,
        Veel deucht // o jeucht // verlicht,
        Ghenade vrede boete,
        Een waar // een claer // ghesicht,
        Een wandel wel ghesticht,
        Bedacht // dach nacht // met smeecken,
        Voort val // der smal // ghebreecken,
        Dat doch t'ghemoet niet swicht.
      
      
        Ziet deze salutacy,
        Tot goed // met spoet // om meest,
        Zij u van Godes gracy,
        Ghewracht // met kracht // des gheest,
        Door Christum onbevreest,
        Op dat // ons schat // gheladen,
        Verdriet // gheschiet // noch schaden,
        Die Schrift doorzoekt en leest.
      
      
        Ghelijck gij zijt natuerlijck,
        Elk maghet // waghet // sacht,
        Met Gods Heylighen puerlijck,
        Dat gij // en wij // veracht,
        Maechden zijn inder kracht,
        Die t'Laem // vreetsaem // navolghen,
        Noch list // noch twist // verbolghen,
        Moet zijn van ons bedacht.
      
      
        Eerstelinghen vercoren,
        Geen goed // maar bloed // ghecost,
        Kinders weder geboren,
        Die elk // met melk // en most
        Ghelaeft zijn en verlost,
        Does prijs // als wijs // van moede,
        Does naam // bequaem // en goede,
        V neme brood' noch post.
      
      
        Roem van naam of waerdye
        Niet vaet // is daad // absent,
        S'gheestes of vleysch hoverye,
        Slaat dood // hoe groot / present
        God bewaer ongheschent
        Die recht // en slecht // hier leven,
        Met vreugd // tot deucht // haar geven
        In blyschap en torment.
      
      
        Rein heylich van manieren,
        Zulk maechden // haechden // t'lam,
        Die haar met deucht verchieren,
        Niet stout / met goud // aan kwam,
        Noch onrechtveerdich mam,
        T'gheslacht // hier sacht // van sinnen,
        Dit stil // Gods wil // beminnen,
        Zijn vanden rechten stam.
      
      
        In Exody // wij // leren,
        Tien paar // wel claer // subijt
        Vader en moeder t'eeren,
        Ephesen // lezen // wy't
        Sierach drie dreycht met vlijt,
        Die kinder // hinder // t'senden
        Die d'ouden // zouden // schenden,
        Als nu in dezer tijd.
      
      
        T'wort in Matheo vonden
        Ik mien // vyfthien // die vaet,
        In Marco ook does gronden
        Int zeven // schreven // staat,
        Collossen drie ook gaat,
        Die meer // begheer // om weten
        Besie ook die Profeten
        Wat hier is Godes raad.
      
      
        Spieghelt hier in u werken,
        T'is wel // t'bevel // van God,
        Maar weynich die 't bemercken,
        Met hopen // loopen // zot,
        Die jeucht acht Gods ghebodt
        Voor niet // besiet // haar ganghen,
        Zij dichte // lichte // sanghen,
        Dit is des werelts rodt.
      
      
        Sara in der stad mede,
        Zie zeid // bescheyt // aenhoort,
        Met die ydel van zeden,
        En meen // ik gheen // accoort,
        En die Psalmist opboort,
        Wel dien // die vliet // den kwaden,
        Altijd // met vlijt // beladen,
        In Godes heylich woord.
      
      
        Ook heeft Salomons leren
        Een schone // krone // toe zeid,
        Die d'ouders obedieren,
        Der looser // bozer // zeid,
        Gheensins en zijn bereid,
        Door woeden // spoeden // waerlijck,
        Tot quaeden // schaden // vaerlijck,
        T'verderven haar verbeyt.
      
      
        Och waar gaer ik mij keren,
        Wat zake // maeck // ik hier,
        Met vermanen en leren,
        Diens vaer // om zwaar // dangier
        Te weren, heeft regier,
        Zijn huis // t'abuys // te letten,
        Gods gheest // dits meest // ginck zetten,
        Hem over zijn bestier.
      
      
        Neemt mij doch dit ten besten,
        Want ik // verschrick // aan een,
        Nu ik bedenck ten lesten,
        Tot wien // gheschien // mijn reen,
        Want mijn wijsheid is cleen,
        Gheen const // maar gonst // en minne,
        Wt richt // en dicht // den sinne,
        Denckt mij in u ghebeen.
      
      
        Ik was sorchd ik verbonden,
        V cloeck // versoeck // zulks dee,
        Door u laetste vermonden,
        T gheschenck // ghedenck // ik mee,
        Voort liefde trou en free,
        Die drijven // t'schrijven // tezamen,
        Maar ziet // t'gheschiet // met schamen,
        Hoewel met danckbaerhee.
      
      
        Die dochters die ik men,
        Haar vaer // wordt claer // ghenaemt,
        Eerste letters bij ene,
        Elcx vers // zo dwers // versaemt,
        Tys Gherritsen het raemt,
        Waar door // ter coor // bedacht // ziet,
        Met haast // het naast // die clacht // schiet,
        Memory wordt gheblaemt.
