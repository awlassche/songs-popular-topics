
    
      
        Zeer minnelijcke groet // zijn moet
        Alle die doet // het goed
        Elk minnaer vroet // zeer zoet
        Des gheests coragieus
        Die met een goed verlanghen
        Dezen Mey precieus
        Vierich begheert tontfanghen
        Van zijn lief amoreus
        Dats Christus glorieus
        De Mey zeer wijt ontdaen
        Schoonder is nooit int land // gheplant
        Zeer triumphant // playsant
        Met sGheests verstand // vailliant
        Wilt dezen Mey ontfaen.
      
      
        Dezen Mey delicaet // ontfaet
        En niet en laat // maar gaat
        Want ziet hij staat // ornaet
        Zeer lieffelijck ten toon
        Lustich ient bloemkens cierich
        En die vruchten ydoon
        Veel rooskens balsemierich
        Onder des Hemels troon
        En vintmen gheen zo schoon
        Als dezen Mey propijs
        Bouen Calmus, Caneel, geheel
        Ist tbeste deel // perceel
        Tlustich iuweel // geheel
        Recht als een Paradijs.
      
      
        Zo gaat nu al ghemeyn // certain
        Tot dedel greyn // int pleyn
        Sleuens Fonteyn // alleyn
        Alle die zijn vermoeid
        Om te vermaecken vruechdich
        Recht als de Palmboom bloeyt
        Ende olie duechdich
        Ziet hoe dezen Mey groeyt
        Met sGheestes dau besproeyt
        Zijn bloemkens Dauid zeid
        Lieffelijck int aenschijn // deuijn
        Een Medecijn // zeer fijn
        Tallen termijn // zij zijn
        Der Heyden ghesontheyt.
      
      
        Princess excellent // bekent
        En reverent // v went
        Dezen Mey ient //  die sent
        Ons God ghebenedijt
        Wt liefden liberalich
        Ontvangt hem met iolijt
        Zijn vruchten triumphalich
        Gheeft hij ter rechter tijd
        Gheestich in hem verblijd
        Voor tslot met melody
        Dezen Mey exalteert // en eert
        Glorificeert // laudeert
        Onghecesseert // vermeert
        Altijd Gods heerschappy.
