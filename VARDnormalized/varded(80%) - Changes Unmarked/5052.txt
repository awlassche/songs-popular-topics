
    
      
        Gij schone Roos // die onder ‘t lyden,
        Hier zijt altoos // wilt u verblyden,
        Dat u is opgheleydt // tot saligheydt,
        Dat gij niet en sout // door lust menichfout;
        Dat gij niet en sout // verliesen,
        V kleur, door ‘s Werelts zot verkiesen.
      
      
        V kleur is schoon // door het verdrukken,
        Van het uit roon // ende uytplucken,
        Van al u lusten kwaad // en overdaet;
        Van ‘s werelts hoocheydt // en haar ydelheydt;
        Van ‘s werelts hoocheydt, ende pralen,
        Die in de afgront moet neerdalen.
      
      
        Gij Bruyt Christi // laat vrij verachten
        V naam // sijt bly // in u ghedachten,
        Lijdt stil het steecken wreet // die u doen leed:
        Toont u zachte moet // en diepe ootmoet;
        Toont u zachte moet, ghestadich,
        Vergelt het kwaad met goed, weldadich.
      
      
        Wijkt hier niet uit // wilt liever sterven,
        Dan lydens kruid // te doen verderven:
        Want gij moet dragen ‘t kruys // zonder abuys;
        Wilt gij dat niet doen // maar tot wreecken spoen;
        Wilt gij dat niet doen, wilt horen:
        Zo is u arbeid verlooren.
      
      
        V Bruyd’gom // heeft veel gheleden,
        En heeft daarom // voor haar ghebeden,
        Die hem deden verdriet // hij wreeckte niet:
        O exempel schoon // van Gods eenich Soon;
        O exempel schoon, t’aenschouwen,
        Volcht hem na met een vast betrouwen.
      
      
        Met hem zult gij // eeuwich be-erven,
        Zo gij hoort mij // met hem wilt sterven:
        Dan zult gij in zijn troon // ontfaen de Kroon,
        En aenschouwen zoet // ’t lieff’lijck goed;
        En aenschouwen soet’ verheven,
        Gods aenschijn in dat eeuwich leven.
      
      
        In ‘s Hemels hoff // zult gij vermeyden,
        Daar men Gods loff // lieff’lijck verbreyden,
        Met nieuwe tonghen bly // een melody,
        Zoet, zonder verdriet // want daar ghebreeckt niet;
        Zoet zonder verdriet // zal wezen,
        ‘t Lam Gods met zijne Bruydt ghepresen.
      
      
        O Bruydt Christi // ziet u betrouwen
        In u Heer vrij // wilt doch aenschouwen
        Wat hij u schencken zal // na ‘t lyden al:
        Hebt dan gheen verdriet // in u lyden ziet;
        Hebt dan gheen verdriet // lijdt stille:
        Draagt met ghedult uws Heren wille.
