
    
      
        Lest ginck ik in de mey spanceren
        Wandelen met Lusten fyn
        Waar door het hert most conforteren
        Ja geheel in Rusten syn
        Vrou venus kaeren excellent
        Sach ik bij paeren daar ontrent
        Hand aan hand // zeer plaisant // dansen en springen
        Wentelen int groene // gras
        Zoudt een quandt // syn verstandt // konnen dwingen
        Die gewoon te doene // was.
      
      
        Terstondt quaemen zij voorts gedrongen
        Herpen veelen Luiten goed
        Zij dansten speelden ende zongen
        Sulckx tgehoor ontsluyten doet
        Musyka schoon was daar ontrent
        Elk hield syn thoon opt instrument
        Delykaet op syn maet // zo manierich
        Pas a pas voorts dansende
        Nyders Raad // werd versmaet // puttertierich
        Met vreugd bloomkens schansende.
      
      
        Does machdekens Lustich van leven
        Stelden mij die Rosen bij
        Waarom soud'ick geen vreugd aencleuen
        Die tot sulckx verkosen zij
        Veel bloemkens ook suyuer ende Claer
        Liefflick van Reuk brochten zij daar
        Sulcken vreugd // mij verheucht // bouen maetten
        Dacht ik hier end most horen daar
        Tza gij Ieucht // maakt geneucht // met zoet praeten
        Tis courage voor ende naar.
      
      
        Terstont quaemen zij voorts gelaeden
        Omden Lust te boeten mere
        Met vlaeden taerten ende gebraeden
        Wyn om druckx versoeten zeer
        Myn docht ik haeck naar sulcken geur
        Doen werd de smaak noch gouuerneur
        Heus en bly // willen wij // bancketteren
        Sprak een van does maechden fier
        Hout v vrij // wie soudy // moghen deeren
        Spyt al diet mishaechden hier.
      
      
        Dus ziende alle does vreugd voor ogen
        Therten bloed verwermden zij
        Terstondt syn zij toe gevlogen
        Helsden custen narmden mij
        Ik hiel mij coele op dit termyn
        Maar het gevoele deed mij pyn
        Dye nu mocht als hij docht na syn wille
        Venus hoff omgraeuen ziet
        Sulcken tocht moet gewrocht syn al stille
        Maar elk heeft de gaeuen niet
