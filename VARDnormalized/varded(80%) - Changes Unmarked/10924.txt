
    
      
        De liefde blent // gaat mij bekent
        Met groot torment // Nu swaerlijck plagen,
        waar ic mij went // Zijnde absent
        van u lief jent // Zo moet ik claghen,
        In verdriet // Door liefde hiet men mij ziet,
        gaan te niet // En gansch verjaghen
        dus suyver greyn // Acht u zo cleyn
        t’aensien certain // Mijn liefde rein
        wilt mijn gheweyn // Doch helpen draghen.
      
      
        Want u licht claer // wezen eerbaer
        Doet u dienaar // in liefden derven‘
        V sank eenpaer, uytspraeck voorwaar
        Brengt mij beswaer // en doet mij sneven
        V bestier // brengt mij hier, veel dangier,
        Dat mij schier // rust gaat beweghen.
        Dies mij doch jont // Lief u verbond
        Zo zal terstont // mijn hert doorwondt
        Wezen ghesont // in vreuchden leven.
      
      
        Goddin u keert // en u verneert,
        Hem die u eert // laat troost verwerven
        Liefde useert // quae raad afkeert.
        Na u begheert // suldy vreugd erven
        Des nijders macht // nu veracht
        Want haar kracht // alleen tracht’V te bederven
        dies nu verlaat // Door mijnen raad
        does valschen praet // T’gheschiet door haat
        en mij bystaet, V druck zal sterven.
      
      
        Princes zoet // hart lijf en goed
        In suer in zoet // zal u standich eeren,
        Zo gij wel doet // Lief met ootmoet
        Dat gij u spoet // mij te salveeren
        Reine vrucht // en geneucht, zal u jeucht
        Door mine deucht // altijd useeren
        Hartseer en pijn // nijders fenijn
        Met valschen schijn // zal van u zijn
        Gij lief myn // troost gaat vermeeren.
