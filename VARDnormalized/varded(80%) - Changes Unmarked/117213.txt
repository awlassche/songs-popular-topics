
    
      
        JAn Kooptmen een Kermis,
        Moy Meysje ik heb ‘er geen geld,
        Ik sou u wel een Kermis kopen,
        Maar ’t geld is door myn broek gedropen,
        Ian koopt men een Kermis,
        Ach Meysjen ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Moy Meysje ik heb ‘er geen geld,
        Ik sou uw Cupidootje geven,
        Maar de krachten zyn verdreven,
        Ian kooptmen een Kermis,
        Ach Meysje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Moy Meysje ik heb ‘er geen geld,
        Ik sou u wel een Roosje plukken,
        Maar ’t Bloempje mocht verongelukken,
        Ian kooptmen een Kermis,
        Ach Meysje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Moy Meysje ik heb ‘er geen geld,
        Ik wou uw liever Carizeeren,
        En met uw Roosje Kwinkeleeren,
        Ian kooptmen een Kermis,
        Ach Meysje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Moy Meysje ik heb ‘er geen geld,
        Myn beste Pandje is verzet,
        Ach Liefje op myn woorden let,
        Ian kooptmen een Kermis,
        Ach Meysje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Moy Meysje ik heb ‘er geen geld,
        De Waaffel Kramer wil niet borgen,
        De Tapper seyd komt overmorgen:
        Ian kooptmen een Kermis,
        Ach Meysje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Moy Meysje ik heb ‘er geen geld,
        Ik sou met u wel eens vrolik zyn,
        Maar de Schyven leiden pijn,
        Ian kooptmen een Kermis,
        Ach Meysje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Moy Meysje ik heb ‘er geen geld,
        Ik sou u wel een Fluytje kopen,
        Maar ’t geld is gisteren al verzopen:
        Ian kooptmen een Kermis,
        Ag Meysje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Moy Meysje ik heb ‘er geen geld,
        Ik sou u wel een Popje schenken,
        Maar het mochte uw Leden krenken:
        Ian kooptmen een Kermis,
        Ach Meysje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Moy Meysje ik heb ‘er geen geld,
        Want ik moet haast na Indiēn varen,
        Al door Nephtunis woeste Baren:
        Ian kooptmen een Kermis,
        Ach Meysje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Moy Meysje ik heb ‘er geen geld,
        Ik moet een Zieltje op gaan soeken,
        Ian Calis zit in al myn broeken:
        Ian kooptmen een Kermis,
        Ach Meysje ik heb ‘er geen geld.
      
      
        Ian kooptmen een Kermis,
        Moy Meysje ik heb ‘er geen geld,
        Ian Credit en wil niet borgen,
        Moy Meysje laat myn niet meer sorgen:
        Ian kooptmen een Kermis,
        Ach Meysje ik heb ‘er geen geld.
