
    
      
        DEn aenghenamen // tijd
        Die is alsnoch voorhanden,
        Och doet te samen // vlijt,
        Mensen in alle landen,
        Om vanden slaap der zonden,
        Als nu ten stonden
        Opstaan, tis tijd bevonden,
        Laat zijn ontbonden,
        De ogen, die van binnen // int
        Herte zijn, laat u sinnen // blint,
        Met des Gheests uytgelesen
        Ogen zalve ghepresen,
        Zalven, genesen,
        Men hoeft klaer ghesichte // wis,
        Anders grijptmen lichte // mis,
        Iae de schaduw' voor t'rechte wezen.
      
      
        Blinde leydslieden // kent
        Aan haar vruchten en werken,
        V snel te vlieden // went
        Bedrieghelijcke clercken,
        Opmercken, is bevolen,
        Om niet te dolen,
        Want men vindt onverholen,
        Veel Satans scholen,
        Alwaar men valsch' afgoden // eert,
        Oft der Mensen geboden // leert,
        Zulke loghen Leeraren
        Stellen haar, op het waren
        Christi dienaren,
        Zij zouden verloren // snel,
        Leiden d'uytverkoren // wel,
        Waert moghelijck door haar ghebaren.
      
      
        S'vleeschs ydel wenschen // kwaad
        Moetmen al van hem jaghen,
        Niet doen na Mensen // raad,
        Noch voor mensen vertsaghen,
        Noch ook behaghen mede,
        Noch houden vrede,
        Zo Bosors zone dede,
        Het zij door bede,
        Oft beloften, maar onderdaen
        Moetmen Christum byzonder // staan
        Voor dezen Heer verheven,
        Moet ieder, na dit leven,
        Rekenschap geven,
        Die de goede kroonen // zal,
        Maar zonder verschonen // al
        Straffen die kwaad hebben bedreven.
      
      
        Al die ghelooven // ziet
        Verwachten te verwerven
        Schat, die men rooven // niet
        Can, noch zal in die erven
        Verderven, noch verrotten,
        Door roest, noch motten
        God laat hem niet bespotten,
        Tis niet te zotten,
        Al wat de Mensen zaaien // hier,
        Zullen zij namaels maeyen // schier,
        Elk wil te waken pijnen,
        De Here zal den zijnen
        Zeer haast verschijnen,
        Den tijd der ghenaden // zoet,
        Die veel nu versmaden, moet
        Voor eeuwigheid geheel verdwijnen.
