
    
      
        O Cierelijck sieraad!
        Wt-muntend’ in deugd en schoonheid van gelaat,
        Die ik bemin // met hert en sin,
        En die ik houd’ voor mijn Goddin,
        Princess // Vooghdesse, die meest // mijn geest
        Wondt en geneest,
        Door ’t lieve licht // van u gesicht,
        Ontfermt u eens, voldoet u plicht.
      
      
        Hoe lange zal ik doch
        (Dit vraegh’ ik) zo wreed gemartelt worden noch?
        V strafheydt wendt // o Lief, en endt
        In blijde vreugd mijn droef ellend’:
        Met reden // doch heden aanziet // ’t verdriet
        Dat mij geschiet,
        Daar ik zo zeer // u schoonheydt eer,
        Ja vier’ de zelve langs hoe meer.
      
      
        Zal ik dan stadich leidt
        Genieten, tot loon van al de dienstbaerheydt,
        Die ik met lust // nooit uyt-geblust,
        Aan u betoond hebb’ zonder rust?
        O schone // persoone, laat af // de straf,
        Eer gij in ’t graf
        Vervoert de geen’ // daar gij alleen
        Met yver wordt van aen-ghebeen.
      
      
        Mijn Liefste, zo gij maar
        Vyt liefde mij aen-neemt tot uwe trouw dienaar,
        Ik soud’ u staegh // bij nacht, bij daegh,
        Gedienstigh zijn met herten graegh:
        Wat lusten // wat rusten, wat vreugd, uwe jeugd
        Tot haar geneught
        Bedenken mocht // oft oyt verzocht,
        Dat soud’ gheschieden zo gij ’t docht.
      
      
        Ik wedd’ mijn Engel dan
        Soud’ zeggen, dat nooit op aarden enig man
        Zijn liefste vrouw // was oyt zo trouw,
        Als ik u stadigh wezen sou;
        Dus enig // alleenigh te zijn // is pijn
        Voor u en mijn:
        Dus laat ons beyd’ // o zoete Meydt,
        Versamen eens in vrolijckheydt.
      
      
        Wij zullen dan voor al
        Zo leven, dat de een wenscht de ander wenschen zal,
        Wat de een bemindt // oft goed bevindt,
        Daar toe zal de ander zijn ghesindt.
        Wat droomt gij? beraedt gij doch snel // ’t uyt-stel
        Komt selden wel:
        Segh; laat voort-aen // u droefheyt staan,
        Ik neem u tot mijne dienaar aan.
