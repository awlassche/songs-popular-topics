
    
      
        Ach dach! wenschelijcke dach,
        Vol schoon gheleghentheyt,
        Dat ik menschelijcke mach
        Met mijn gheneghentheyt
        Mijn Goddin doen bescheyt
        Op 't gheen dat zij mij zeid
        Deur haar praat // of in daat
        Hoe het gaat // in wat staat
        Mij 't gheluck rechte voort leit.
      
      
        Zijn dit wrevelinge // niet?
        Neent, droomend Fantasij?
        Flucx, weg revellinge // vliet
        En maakt u knap van mij,
        Weg droeve dampen gij,
        De schone dach naackt bly
        't Zuchten vlie // ik door die
        'K weet wel wie // dien ik zie
        Hier sitten an mijn groene zij.
      
      
        Sijt ghy't, Egtillen // jent,
        Ik bid u dat gij spreekt;
        Sijn't weer dolle grille // blent
        Die de begeert ontsteekt?
        Mijn oogjens onbevleckt
        Zien u, mijn lief perfeckt,
        Ja voorwaar // schoon en klaer,
        En geen aer // het schort maar
        Aan u dat gij mij dit ontdekt.
      
      
        In sulcken cierelijcken staat
        Gheloof ik nauw te zijn,
        Dijn zoet manierelijcke praet
        Die ik staech an u vijn,
        Nochtans is dit geen schijn,
        Ik ziet voor d'ooghen mijn:
        Ik twijffel // met ghequel
        Of ik wel // die ghesel,
        Die ik thans was, wel souw zijn.
      
      
        In d'onbegrijpelijcke // vreugd
        Wentelt mijn wuft ghedacht
        Te zien u rijpelijcke // deucht
        En jonst dus onverwacht,
        Dat ik mij gh'luckich acht
        Boven 't mensch'lijck gheslacht:
        Want Keyser braaf // Prins noch Graaf
        Heeft sulcken gaaf // als u slaaf
        Die 't gheluck zo zoet toelacht.
      
      
        Och of gij deelachtich // waert
        Die vrucht die ik gheniet,
        Die in mij zo crachtich // vaart
        Dat mij 't vernuft ontvliet,
        Laes! een dinck mij verdriet,
        Dat u gheen eer gheschiet
        Deur vreuchden zoet // mijn ghemoet
        U niet groet // als 't herte doet
        Dat gij naar wensch ghebiet.
      
      
        Cleen souw de danckbaerheyt // zijn
        Indien dat ik verstout;
        Dat inder waarheid // fijn,
        Te spreecken was goed rond
        Met menschelijcker mond,
        Die weldaet mij ghejont
        Die 'k wardeer // hoog en zeer,
        Eeuwich eer // na 't begeer
        Ben ik, helaes! te zwak ghegront.
      
      
        Spreeck ik beroemlijck // iet
        Princes myn schuit vergeeft:
        Voorwaar onnoemelijck // ziet
        Syn u deuchden beleeft,
        U lof ten Hemel sweeft
        Daar 't zonder twijffel leeft.
        Ik bid God // dat hij tot
        Volle slot // na 't ghebodt
        Ons met trou liefd 't samen cleeft.
