
    
      
        Staat op, en wordt verlicht
        O Mensen! staat op, en gaat na 't eeuwich Licht,
        Verlaat uwe sond' // staat op terstond,
        En wordt na u-we ziel gesondt.
        Gekomen // vernomen // is God, Ã´ lot!
        In 't beesten schot;
        Wien soecken gaan // (ziet 't werk eens aan)
        Does Koningen met Goud bela'en.
      
      
        Uit Oosten komen Zij
        (O wonder!) door vorst, en hagel, even bly.
        Zij volgen na // de Sterre dra,
        Zij vinden haarder ziels gena.
        Zij alle // ne'ervallen // te voet // (hoe zoet!)
        Het Opper-Goedt:
        't Welk in een stal // op 't aerdtsche dal
        Ne'er was gekomen voor ons all'.
      
      
        Ziet aan wat grote eer
        Does Prinsen, bewijzen aan dien groten Heer.
        Haar zoet ghesucht // vlieght tot de lucht,
        Haar schreyen is aan God ghenucht,
        Zij kussen // Zij sussen // het Kind // 't welk sindt,
        En Haar bemint.
        Zij geven Goud ,, aan die verkoudt
        Op 't hooy leit ne'er, die d'Hemel boudt.
      
      
        Het Wierook wordt gebrocht
        En Myrrhe, wie had zulks oyt van Haar gedocht.
        Met groot' ootmoed // Zij all' dit goed
        God dragen op, ja ook haar bloed.
        'k Wil mede // gaan treden // bij Haar // en naar
        Does tijd voorwaar
        Aan dezen Heer // uit liefde teer
        Mijn hart op dragen 't sijnder eer.
      
      
        Zij groeten ook de Vrouw
        MARIAM, die ons verlost heeft uit den rouw.
        't Is aanziens waerd // 't moet zijn bewaard
        In 't hart, hoe dat does totter aerd'
        Haar buyghen, en tuyghen // met le'en // en re'en,
        Met bly geween,
        Dat Zij de kroon // en de Opper-loon
        Verdient van Christo haren Soon.
      
      
        Laat ons nu desgelijck
        Ook knielen, die anders-niet en zijn als slijk.
        Laat zeggen uit // ons overluydt,
        MARIA! Gij zijt Godes Bruyd.
        G'ontfermt ons // G'erbermt ons // in strijd // Gij zijt
        Die ons verblijdt.
        Ei ! segh ons aan // dat Gh' ons ontfaen
        Zult naemaels binnen 's Hemels baen.
