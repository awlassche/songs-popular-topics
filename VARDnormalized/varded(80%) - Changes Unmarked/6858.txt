        Der Maechden krone // Maria weest gegroet,
        Gratia plena, claer als die Zonne schone
        In sâ€™Hemels troone // vat vol genaden zoet
        Der deuchden volle vloet // een suyver hert ootmoet
        Hebt gij gebaard // Gods Soon ons hoogste goed.
      
      
        Dominus tecum, die Heer zij met u ene
        Suyver Fonteene // Vrou alder wereld wijt
        Benedicta tu in mulieribus, allene
        Al waert gij cleene, gij sijt gebenedijt,
        Gheen vrou te geender tijd // zo zalig als gij sijt
        Noch zo verheven, van God tot vijands spijt.
      
      
        Et benedictus, gedankt, gelooft, geprezen,
        Moet gij altijd wezen // dat hoochweerdich pant
        Fructus ventris tui, dijn vrucht rein wtgelesen
        Wt Iesse geresen, Iesus Christus playsant
        Die als een Heer valiant // ons brak der hellen bant
        Ende die gevangen brocht // in haars Vaders land.
      
      
        Sancta Maria Princes verheven krone
        Edel geboren // van Koning Davids saet
        Ora pro nobis, bidt voor ons die te voren,
        Waren verloren, verdoemt door zonden kwaad
        Tot dat ons kwam te baet, Iesus ons Advocaet,
        Van u geboren // o schone dageraad.
