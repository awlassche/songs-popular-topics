
    
      
        CRachtich gewont // den gront //
        Mijns herten ziet,
        Tonge, noch mond // bestond //
        Nooit drouver liet,
        Eylacen diet // mij heeft gedaan //
        Ziet mijn verdriet // zeer luttel aan //
        Mij zal ontstaen // zeer saen //
        Een meerder last //
        Mijn lijden wast // noch vast //
        Waar zal ik gaan //
        Gevaen // ellendich gast.
      
      
        O droeuen dach // u mach
        Ik vloucken schier,
        Dat ik oyt zach // t'opslach
        Der Oogskens fier,
        Van t'vroulick dier // diet al passeert,
        Wiens goedertier // bij zijn falieert,
        Gans gematteert // doleert
        Al mijn verstand,
        Wie bluscht den brand? // niemand,
        Want diet moveert
        Mij keert // flucx van der hand.
      
      
        Rein zedebaer // volcht haar
        Der vrouwen // aard:
        Een gulden iaer // voorwaar
        T'aenschouwen // waert:
        In trouwen // baert // zij liefde heet,
        T'berouwen // vaart // henen gereed,
        Van alle leed // hoe wreet,
        En straf, het zij:
        T'vriendelick // bij // zijn, vrij
        Elk (met bescheet)
        Becleet // bijzonder mij.
      
      
        Nooit tooghd' ik geen // onreen
        Dorper gelaat:
        Of dat te cleen // zulk een
        Beelde toestaet:
        Of haren haat // mij dus bevecht,
        Veracht, versmaet // onweerdich slecht,
        Mij wederleght // ontseght
        Een woord, hoe cleyn,
        Mijn hert is rein // certain
        Ik ben oprecht,
        Haar knecht // en geen vileyn.
      
      
        Eylacen och // merckt doch
        Mijn cuysch opset:
        Met schande, noch // bedroch,
        Geenszins besmet:
        Vroom heb ik met // liefde gewrocht,
        Naar S'heeren wet // te zijn verknocht
        Met haar, hoe mocht // verzocht
        Zijn meerder deugd?
        Mijn hert verheugt // does ieught
        Gans wtgesocht,
        Mij brocht // oyt d'ooghste vreugd.
      
      
        Laat zij dan mijn // gepijn
        Opwassen groot:
        Wat medecijn // zal zijn
        Voor mij ter noot?
        Dat mij de dood // niet en versmacht,
        Die mij verstoot // heeft meerder macht:
        Dus naar does clacht // verwacht
        ik, binnen // t'perck:
        Der sinnen // merck // die t'werck
        Dus heeft gewracht,
        Wt kracht // der minnen // sterk.
      
      
        Is haar bekent // den endt
        Van mijn ghetreur,
        En mijn torment // afwent:
        Haar straf rigeur,
        Weg dan malheur // vyandich gloet,
        Haar wreedheid (duer // mijn groot oodtmoet)
        Zal zijn versoet // sent spoet
        In dezen bant,
        Cupido, want // dit pant
        Cost vlees, en bloed,
        Dus doet // mij onderstant.
      
      
        Aldus ik dan // hier van
        Make finael,
        Op aarden can // geen man
        Blusschen den strael,
        Die teenemael // mijn hert deurviert,
        Dan speciael // (wel gemaniert)
        Diet al regiert // en stiert
        Met zulk virtuyt:
        Al stoot zij uit // mij ruyt
        Dus gehantiert,
        Verciert // blijft zij mijn bruyt.
