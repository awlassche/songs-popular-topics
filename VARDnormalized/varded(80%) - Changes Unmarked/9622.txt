
    
      
        ALle vlees is hoy, na Schrifts belijdt,
        Kort is den tijd, van smenschen leven,
        Als bloem in't veld, ontploken wijdt,
        Moet hier subijt, zijn glory sneven,
        Wat is met den Mense bedreven?
        Een uyr en is hem niet ghegheven,
        Hoe langhe hij // zal leven vrij,
        Zijn perck hebt gij // o God beschreven.
      
      
        Dat oud verbond // is stervens nood,
        End' na den dood // wij al verwachten
        Noch enen dag des oordeels groot,
        Daar werden bloot // alle ghedachten,
        Als Christus komt met zijn heyrkrachten,
        Zullen huylen alle gheslachten
        Der Aarden kwaad // veel drucx toestaet,
        Al die den raad // Christi verachten.
      
      
        Voor den Rechtstoel Christi ydoon,
        Op den persoon // werd gheen aenmercken,
        Die 't herte kent, zonder verschoon,
        Zal geven loon // elk na zijn werken,
        Kracht en zal niet baten den sterken,
        Noch geen kloeckheyt den wijzen klercken,
        Recht oordeel klaer // zal openbaer
        Strijken aldaar // dat hoofd der Kercken.
      
      
        Zeer vreeslijck God hem wreken zal
        Over 't getal // syner vijanden,
        Die de Wereldt beminnen al,
        Want haren val // werd groot ter schanden,
        Haarlieder vier zal eeuwig branden,
        Haarlieder worm die blijft voorhanden,
        Rouwe, gheween // werd daar ghemeen,
        Angst ende een // knerssen der tanden.
      
      
        Dan zullen die rechtveerdich zijn,
        Vrij van ghepijn // ofte weeklaghen,
        Blincken ghelijck der Zonnen schijn,
        Met vreuchden fijn // niet om ghewaghen,
        Int Rijke Gods, na haar behaghen,
        Onsterffelijcke Croonen draghen
        T'Lammeken zoet // zal haar tot spoedt,
        Der tranen vloed // van't aenschijn vaghen.
      
      
        Een Princelijcke Majesteyt
        Heeft God bereid // zijn uytverkoren,
        Dus is des loons groot onderscheyt,
        Vast toegezeid // ende ghesworen,
        Tussen die Godloos God tegen sporen,
        En die hem dienen wedergheboren,
        Wat neerstigheid // men hier toe leit,
        Gheenen arbeid // en werd verloren.
