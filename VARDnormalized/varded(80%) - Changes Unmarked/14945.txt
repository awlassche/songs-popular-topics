
    
      
        EEn vreemde Maar // ik u bringhen zal,
        Doch schier ruchtbaar // acht ik het over al:
        ’T is dat een Maaght // een zeer aenneming Dier
        Haar Eer beclaaght // door het bekende vier.
      
      
        Wiens hoofd cieraat // uytvluchtich stond gheset,
        Trat over straat // trots als een Pauw te net,
        S’is nu gheschent // neer leit haar hooghe moet,
        Zulk loon gheeft t’ent // haar die het selfde doet.
      
      
        Ach blinde Min // en wulpsheyt onbedocht,
        Gij hebt haar sin in doolinghe ghebrocht,
        En ‘thert verleid // tot does begheerten snoot,
        Door welks feyt // haar ere is ontbloot.
      
      
        Wie had ghemeent // dat een zo schoon ghelaat
        Haar sou vercleent // hebben en ’t best versmaat,
        Ja ’t alderbest // gheacht ter Wereld wijt,
        ‘tWelck zij op’t lest // helaas! te laat becryt.
      
      
        Geen Maagt in stee // bij does mocht zijn geacht,
        Haar Cleedinghe // was uytmuntende dracht,
        Cierlijck van stof // ghestrickt , ghekoort, ghesneen
        Volgden soo’t Hof // meer als d’anderen deen.
      
      
        Wat ist nu weer // of sijt schoon alles heeft
        Zij mist haar eer // daar t’alder meest aan cleeft,
        Gheniet een lot // dat yder haar begeckt
        Nu hem de pot // dus schandelijck ondeckt.
      
      
        Waerdich waar zij // noch ere aan ghedaan
        In dien zijt vrij // does feyt hadden bestaan,
        Niemand en sout // acht ik haar spreecken schant
        Want zij ghetrout // mocht werden na der hand.
      
      
        Maar nu eylaes // ist te beduchten wel,
        Dat doort gheraes // van does gheruchten snel,
        Haar niemand nu // aenspreecken zal op deucht
        Hier aan gij u // ô Maechden spiegelen meucht.
      
      
        Cleen lof hij wind // bij die vroom zyn van aart,
        Dien vroom mans Kint // niet te bedrieghen spaart,
        Maar steelt als dief // in schijn van ware trouw
        T’hert uit syn lief // en laatse na in rouw.
      
      
        Doch een vroom Prins,, alsulcke daden // vliet
        En tracht geenszins // iemand te schaden // iet,
        Maar dient als Knecht // die hij van herten meent,
        Cuysch en oprecht // tot dats hem God verleend.
