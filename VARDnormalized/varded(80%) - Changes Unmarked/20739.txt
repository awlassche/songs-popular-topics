
    
      
        DE groene Meye // brengt ons den Feest
        Van Sint Philip' we'erom;
        wiens eer verbreyen // wil on-se geest,
        Door 't gantse Christendom.
        Hij is // 't is wis // geweest een vromen Held,
        Die Satan met ge-weld
        Ge-dreven heeft uit 't veld;
        Hij is 't die in onze druck
        Ons verkrijgen zal geluk,
        Hier toe is Sint Ph'lip gesteld.
      
      
        Des laat ons prijzen // does groten man,
        Does groot' Apostel Goods,
        Die zal verdrijven // ons zwaar gestan
        In de laetste uyr des doods;
        Die sall' // ons all' // beschermen op het lest,
        Die voor ons doet zijn best;
        En vrijdt van Satans nest;
        Die is oorsaeck van onze vreughd,
        Die ons voordert in de deughd,
        Die ons brengt tot 's Hemels vest.
      
      
        Hij is verkooren // geweest van God
        Tot ene Capiteyn,
        Eer dat gebooren // (ziet wat een lot)
        Hij was op 's werelts pleyn:
        Hij zocht // Hij brocht // tot Christvm zijnen Heer,
        De zielen meer en meer
        Door zijne trouwe leer'.
        Hij geweest is enen Tolck
        Bij den Heer, voor 't Heydens volk,
        Wat is dit een grote eer!
      
      
        In swarigheden // te geven raed
        Was Ph'lippus wijs en vroed:
        Zijn goede reden // de'en meenich baet,
        Zij vrochten meenich goed.
        In nood // van brood // vraagt Christus Hem bescheyt
        Waar me'e does swarigheyd
        Sou worden afgeleyt?
        Van waar zullen kopen wij,
        't Welken mogen eten zij?
        Heeft den Heer tot Hem gezeid.
      
      
        Maar veel geleden // en uytgestaen
        Heeft dezen groten Sant,
        En vroom gestreden // op 's werelts baen
        Voor 's Hemels Vaderland.
        Hij lijdt // Hij strijdt // wel twintich jaren langh
        Voor Christo met verlangh;
        Voor 't sterven geenszins bangh.
        Hij heeft Asia bekeert,
        Hij heeft Schytia geleerd
        Den oprechten Christen-gangh.
      
      
        Toen is gevangen // van 't vuyl gespuys
        Te Aleppo in de Stad,
        En opgehangen // aan 't houte kruys
        Dit Hemel-waerde Vat:
        Om-hoogh // zijn oogh // Hij opsloegh na de lucht,
        Hij bad, Hij de'e noch vrucht,
        Hij was voor 't Volk beducht.
        Maar eylaes! Hij wordt gesteent,
        D'wijl voor 't volk Hij heeft geweent,
        All' dit was Hem groot genucht.
      
      
        De aarde scheurden // met groot gedruys,
        Aleppo was in nood;
        De beulen treurden // na Plutoos huis
        Zij daelden voor haar dood.
        Dit gaf // tot straf // die 't all' geschapen heeft,
        Bij Wien nu Ph'lippus leeft,
        Voor Wien den Duyvel beeft.
        O hoe wonder is den Heer,
        In het voorstaen van de eer,
        Van die lof aan Hem hier geeft!
      
      
        O Sint Philipp! // wij groeten U,
        Met grote ned'righeyd:
        Uwe mond, en lippen // doet open nu
        Tot onzer saligheyd.
        Verwerft // dat erft // onze ziel het eeuwig wel;
        Bevrijdt ons van de Hell',
        En Satans boos gequel:
        Ach! uw ogen tot ons wendt,
        Staat ons bij tot 's levens end
        Aldertrouste Goods Gesel.
