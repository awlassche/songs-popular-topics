
    
      
        Elk medegenoot // int gelove minjoot,
        In veel droef aenstoot // met verduldicheyt groot,
        Stelt u vreze bezijden,
        Al doen haar exploot // Wereld, Hel ende dood,
        Hoe menigen kloot // dat elc oyt op u schoot
        Victory zult gij stryden,
        Door Christum alleyn,
        Wilt u verblyden,
        Want hij uit pleyn
        Is te allen tijden
        Met zijn volk rein,
        Als Capiteyn,
        Om haar bevrijden
      
      
        Wie oyt strijd aennam // in den name van 'tLam,
        Oyt hij te boven klam // exempel Abraham,
        Hij verwan die Koninghen,
        David Goliam // den Reuse fel en gram.
        Die in den name kwam // der Philistijnsche stam,
        En d'Israelijten ginghen
        Met trompetten slaan,
        Iericho omringhen,
        De mueren vielen saen,
        Veel wonder dingen
        Hebben gedaan,
        Al die vast aan
        Den Here hinghen.
      
      
        Petrus die eerst maar // voor een maagd hadde vaer,
        Dat hij tegen haar // niet dorst belijden claer
        Die sietmen nu voort treden
        Voor een grote schaer // door Gods Geest wonderbaer,
        sterk als een pylaer // en heeft den Here daar
        Voor elk end' een beleden,
        Zijn antwoorde tot
        Diets verbodt deden
        Gaf dus een slot,
        Oordeelt, ist reden,
        Dat zelve God
        Door u ghebodt
        Zal wijken heden?
      
      
        Dat 'sHeeren Geest // sterc maakt en onbevreest,
        Stephanus beweest // als groot vervolgh tempeest
        Stormigh op kwam geresen,
        Iae Paulus die eest // vervolgde, heeft noch meest
        Gedaan int aertsch foreest // menige wilde beest
        Weder vochten t'Ephesen,
        Iae met goed verdragh,
        In al zijn wezen
        Op Christum zag,
        En sprak, door dezen
        Ik 't al vermagh
        In zijnen dag
        Werd' ik ghepresen.
      
      
        Van mijns Prinsen hand // werd mij int eeuwich land
        De Croon geschonken, want // hij heeft tot onderpandt
        Ons synen Gheest ghegheven,
        Dus Broeders vailliant // en laat doch u niemand
        Wt sin en verstand // rooven dat God geplant
        Heeft, en daar in gheschreven,
        Door zijnen gheest vroet,
        Maar zonder sneven
        V beste doet,
        Om vast aankleven,
        Hebt goeden moet
        Gij krijgt den hoedt
        Van 't eeuwich leven.
