
    
      
        T'vertoont zich Vrouwe // in alder ootmoet
        U Lief ghetrouwe // en Dienaar goed,
        Met weenend' ogen // bedenkt, beschaamt
        Versoeck meedoogen // Prinses vernaamt,
        Mijns lyder smart
        Does wetten hart
        Zijn veels te vart,
        Van een goed Maacht befaamt.
      
      
        Ik onderdanen // heb nacht, op nacht
        Veel sucht en traanen // ja clacht, op clacht,
        Ghestort, ghegooten // met bitter getreur
        Maar nooit ontslooten // u wreede deur,
        Dit claach ik God
        Want treed' ik tot
        Het staale slot,
        En grendels synder veur.
      
      
        Koom ik behendich // ontrent u gelas
        Mij ontmoet schendich // snel het ghebas
        En t'gerucht der mense // dat ik zeer haat
        Mijn Vrou ik wensche // en bid ghenaad
        O quamt // ik namt
        Vant t'hart dat vlamt
        Gheeft dat dit ampt,
        Maar voor mij oopen staat.
      
      
        Zo uit ghenaade // does jonst gheschiet,
        Ik zal die weldaade // vergheeten niet,
        Maar die beloone // met danckbaarheyt
        Met Lof betoone // ben ik bereid,
        Tot alder stond
        Doort Heil verbond,
        Met hert en mond
        Werd uwen naam verbreyt.
      
      
        Voor u wel vaare // Prinses eerbaar
        Stelt hij t'syn Gar // brand u Dienaar
        In jonst dus vierich // weg arech vermoen
        Syt gij eergierich // zo wilt u spoen
        Na dit request
        Zo doet u best
        Ik sech int lest:
        Dus doende zult wel doen.
