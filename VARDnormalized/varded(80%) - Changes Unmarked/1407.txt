
    
      
        Ik weet een Peerle jent
        Geprent in schoonheid rijke,
        Nooit man en  heeft present
        Bekent // haarder ghelijcke.
      
      
        Diamant ofte Robijn
        Sangwijn verciert in gouwe
        Men vint tot geen termijn
        Zo fijn // in geen Lantsdouwe.
      
      
        Ghelijck die Zonne klaer
        Eenpaer, in's Hemels Throone,
        Zo is mijn Lief voorwaar
        Aldaar //blinkende schone.
      
      
        Waar ik een Coninck vailjant,
        Playsant // rijk van person,
        Ik versette voor haar mijn Land
        En Sant // en ook mijn Krone.
      
      
        Dat ik mijn ghedacht
        Dach nacht // op haar set als onwijze,
        Zij is mij te hoge van macht
        Geacht // en ook te groot van prijse.
      
      
        Does peerle daar mij dat leed
        Zeer wreet // om moet ghebeuren,
        'tIs mij lief dat ik weet
        Hoe zij heet // de fleur der fleuren.
      
      
        Wist ik met mijne herten bloed
        Onsoet // haar noch te winnen,
        zo soud' ik tot een boet
        Mijn bloed uit laten rinnen.
      
      
        Storve ik van rouwe groot
        Den dood // om haar door dezen,
        Zo waren mijn wondekens root
        Ter noot // geheel genesen.
      
      
        Ik geve elk minnaer wel
        Bevel // dat hij der liefden draghe,
        Maar dat hij der liefden ghequel
        Zeer fel // niet en beklage.
      
      
        Die hem spiegelt vrij
        Aan mij // die spiegelt hem zachte,
        Elk minnaer verre ofte bij
        Wie hetzij // slaat hier wel op achte.
      
      
        Prinse mindy een edel greyn
        Int pleyn // als ik boven maten,
        Zo en mooghdy der liefde treyn,
        Certain // met recht niet haten.
