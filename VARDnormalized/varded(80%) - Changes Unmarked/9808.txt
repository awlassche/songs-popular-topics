
    
      
        O Christus lief minnare zoet,
        Die mij schenckt u dierbare bloed,
        Na u verlangt mij te allen uren,
        Zo den Hert zijnd' in var //doet,
        Na de Fonteine klare // vloet,
        In stortich pijnelijck besueren,
        Ik leve in zwaer dolueren,
        Hoe langhe, Lief, zal dit ghedueren?
        Wanneer zal ik uit rouwen // zijn?
        Wanneer zal ik aenschouwen // dijn
        Aenschijn, schoon boven natueren.
      
      
        O Lief in 'sWerelts wijcke // niet
        Van mij in gheen versijcke // vlied,
        Maar helpt mij, dat mijn teere leden
        Niet en buyghen ghelijcke // riet,
        Maar van u zoete Rijke // ziet
        Mij hier in dit dal beneden:
        Want ik vinde mij bestreden
        Van den Prinse der duysterheden,
        De Wereld int verstranghen // kwaad,
        Alle dat nijdich Slanghen // zaad
        En laat, mij niet met vreden.
      
      
        Ionstigh voor mij gheneghen // vecht
        Leidt mij in uwe weghen // recht,
        Door uwen Gheest, o Lief verheven,
        Want ziet, ik ben bedeghen // slecht,
        Een teken daar elk tegen // zegt,
        Voor rasernije acht men mijn leven:
        En ik nochtans becleven
        Wt uwe sijde zonder sneven,
        V Lief, u Bruydt, u Kercke // reen,
        Ik doe door u o sterke // steen
        De Helsche poorten beven.
      
      
        O Prinse der Princieren // jent,
        Gij die herten en nieren // kent,
        Aenhoort mijn bitter klaghen,
        Vwen Gheest goedertieren // zendt,
        Dat hij door zijn regieren // wendt
        Mijn ledekens na u behaghen,
        Want 'tzijn nu kwade dagen,
        Hierom Lief moet ik u vraghen,
        Zal ik van al dit lijden // vrij,
        Met u niet haast verblyden // mij,
        En 'sLevens Krone draghen.
