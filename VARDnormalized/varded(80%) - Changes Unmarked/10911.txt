
    
      
        Den mey komt ons lieffelijck aan
        de baen // wordt nu betreden, uyter steden,
        Van Minnaers die daar buiten gaan,
        Zeer faen // volcht haar lieff meden // meest in vreden,
        Met harpen, luiten zoet gheclanck
        Violoneen, Fluyten ende sank
        Over dz schoon groen heye,
        Met dezen coelen meye.
      
      
        Daar gaan geliefkens hand aan hand
        Opt land // langs de Rivieren // zo playsieren,
        Bevangen metter liefden brand,
        Playsandt // daar al de dieren // den Mey chieren,
        de Vischkens in het water zoet
        Spelen daar met een bly ghemoet
        In die zelve valleie,
        Met dezen coelen meye.
      
      
        Die Veldekens staan abondant
        Geplant // met Bloemkens reine // int ghemeyne
        d’Amoureusen zeer triumphant
        Vayllant // na de Fonteine // in dat pleine
        Treden zij dan // paar ende paar,
        Menich Iong’man, met zijn Lieff daar
        Zij plucken d’Acolye
        Daar in den coelen Meye.
      
      
        Menich Bloemken stater ghebloeyt
        Daar groeyt // in die landouwe // menich Karssouwe
        Zijnde met sHemels dou besproeyt
        Ooc vloeyt, dwater nouwe // schoon Ioncvrouwe
        Phebus gheeft alle cruyden kracht
        Aurora vochticht dach en nacht
        Tgewas in menigherleye
        Met dezen coelen Meye.
      
      
        Gij Princelicke minnaers hoort
        Treet voort // condyse vinde // u beminde
        Maakt met u tween een accoort.
        Oerboort // als wel ghesinde // onder een Linde
        Zo Amoreusen schuldich // zijn
        Wilt uwen tijd gheduldich // zijn
        Wie can, dat hij verbeye,
        Tot in den coelen Meye.
