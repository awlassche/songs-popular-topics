
    
      
        AL die hebben tot jaghen // sin
        Staat op en gaat int veld,
        En hebt niet meer behaghen // in
        Den slaap, die menich quelt
        Daar is te vaen, o Iagher dat u ghelt
        T’eeuwich welvaren // zoet
        T’eeuwich leven is ons voorghestelt
        Elk als Held // vlijt en ghewelt
        Hier toe openbaren // moet.
      
      
        DRIe dingen zijn bevonden // noch
        Bequaem tot dezer jacht,
        Ten zijn netten noch honden // doch
        T’is t’gheloove vol kracht,
        Hope, en de Liefde boven al gheacht
        Om wel te vanghen // maar
        Niet hier // een dier // onder wilt gheslacht,
        Maar een sacht // Lam vol eendracht
        Wel weerdt om verlanghen naar.
      
      
        AAN dit suyver ghepresen // lam
        Al ons welvaren cleeft,
        Elk Iagher zonder wezen // gram
        Volgt hem en niet en sneeft,
        In zonden meer, maar schrickt daar vaen
        Den Heer weest onder // daen,
        Iae Iagher u hem gevangen gheeft
        Die hem heeft // eeuwich hij leeft,
        Dus cleeft // hem bijzonder // aan.
      
      
        HOEFven wij int ellendich // wout
        Iet beters t’onser noot?
        Och neen wij, maar die blendich // stout
        Van hem afwijcken bloot,
        Haar lusten jaghend’, en niet achten groot
        T’lieflijck vermonden // schoon,
        Maar speelen hier met der zielen brood
        In den schoot // vanghen den dood
        Want dat is der zonden // loon.
      
      
        NA t’beste moetmen speuren // dan,
        Meer als na een wildt dier,
        Dewijl het ons ghebeuren // can
        Ontvlieden t’eeuwich vier,
        Der ghenaden tijd niet door licht bestier
        Willens verliesen // al,
        De dood fel oft t’eeuwich leven schier
        Van tween hier // om t’dangier
        Men t’beste verkiesen // zal.
      
      
        GELijcken niet den rouwen // knecht,
        Esau moeten wij,
        Maar God altijd betrouwen // recht
        Als Iacob, dat als hij
        Ons deel der ghebenedijdinghe zij
        Maar zo ik proeve // snel
        Om te vanghen dat Een noodich vrij
        Vind’ ik mij zeer zwak, waar bij
        Gods hulp ik behoefve // wel.
