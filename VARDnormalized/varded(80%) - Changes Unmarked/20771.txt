
    
      
        LOwys Gonsagen! van kindsche dagen
        Hebt Gij den Heer gediendt,
        Als zijnen trouwen vriend.
        God U beminden // God U wel sinden,
        En uit het eeuwig wee
        Help U Maria dree:
        In barens-noodt // waert Gij na-by uwe dood:
        Maar deze Vrouw // verdreef den rouw,
        Zij was uwe ziels behouw.
      
      
        U eerste spreken // g'lijck heeft gebleken,
        Was Jesus zoete Naam,
        Wien Gij waert aangenaam.
        En hier beneven, hebt Gij gegeven
        Goods Moeder 't ander woord,
        Daar wierdt Mari gehoord.
        O! zoete Kind // hoe hebt Gij God bemindt,
        Die m' in uwe mond // ten allen stond,
        En in uwe woorden vond.
      
      
        In uwe voort-stappen, op alle trappen
        Hebt Gij does Maeghd gegroet;
        Zij lag in u gemoed.
        Met goed behagen // ging G' Haar op-dragen
        Uwe witte Suyverheyd,
        Die G' Haar had toegezeid.
        Gij suyver zijt // geweest ten allen tijd,
        Nooit heeft beswart // nooit heeft verwart
        De vuyligheyd u hart.
      
      
        In uwe gebeden // en vyerigheden,
        Waert Gij getrokken op
        Tot 's Hemels hooghsten top.
        Gij zijt ontsteken // van liefd' bezweken,
        Als Gij uit 's Priesters hand
        Kreeg de Heyl'ge Offerhand.
        Gij wijt-beroemt // een Engel wierdt genoemd,
        Gij wierdt vermaerdt // op deze aerd',
        Gij was den Hemel waerd.
      
      
        Als G' waert gestorven // hebt Gij verworven
        Des Hemels gulde Zael,
        Waar Gij nut Godes maal.
        Bidt dat genaecken // bidt dat geraecken
        Ik mach bij God mijn Heer,
        Die U doet deze eer.
        Och of ik kwam // door U bij Godes Lam!
        Dan zouden wij // van zonden vrij,
        Met God zijn eeuwich bly.
