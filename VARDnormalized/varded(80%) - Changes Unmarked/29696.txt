
    
      
        Alle myn betrouwen // stel ik op God den heer
        Alleen op v ik bouwen // verlaat myn nimmermeer
        Gij syt myn troost myn toeuerlaet
        Daar alle myn hoop op staat
        Voor schande en oneer // behoet mij doch o heer
      
      
        Want gij o God geprezen // helpt mij uut alle noot
        En gij cont mij genesen // door u genade groot
        Dus bid ik vroeg ende laat
        Mij toch met uwen hulp bystaet
        Voor schande en oneer // behoet mij doch o Heer
      
      
        Wilt mij doch regeren // v arme schaepken teer
        Dat ik mij mach verneren // ootmoedich na u leer
        Zo zal ik verlaten saen
        Die wereld met haar boos vermaen
        Voor schande en oneer // behoet mij doch o heer
      
      
        O God almachtich // verhoord mij metter daad
        Want ik ben v zeer clachtich // mij doch niet en versmaet
        Laat mij doch beschamen niet
        Maar helpt mij doch uut dit verdriet
        Voor schande en oneer // behoet mij doch o Heer
      
      
        O God der prinschen // laat mij syn onvertsaecht
        Want al myn winschen // van v arme dienstmaecht
        Syn u bekent o heer divyn //
        Verlost mij doch van druck en pyn
        Voor schande en oneer behoet mij doch o heer
