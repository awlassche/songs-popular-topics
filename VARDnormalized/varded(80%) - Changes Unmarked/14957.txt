
    
      
        Ziet hoe het wicht syn Sluyertje
        Mijn voor de ogen bynt,
        Lest ging ik om een kuyertje
        Met Luytghen Aelghens kynt,
        Maar door de gang // vielt mij zo bangh
        Vermits de dwerrel wynt.
      
      
        Wij namen ras van Stee de wijck
        ’t Was sa de poort ontsluyt
        Ons gang was recht na Sloterdijck
        Wij kwamen bruyntges uit,
        Luytghen wiert moe // en ik daar toe
        Dus creeghen wij een Schuit.
      
      
        Ik roeyden vast door ’t Slootgen // claer
        En Luytgen zat aan ‘tRoer
        Ei lieve kyck in’t bootgen // daar
        Zo riep een styve Boer,
        De Stuerman heeft // heel onbeleft
        Recht reetschap als myn Moer.
      
      
        Gij onbesuysde bulleback
        Sprak Aeltgens kynt schaem root
        Ze grauden dat ik niet en sprak
        Ik sey moer tis gien noot,
        Jouw deucht, jouw eer // vermach veel meer
        Als roemen na u dood.
      
      
        Daar na zo sloeg haar tongetjen
        Ziet voor u riep myn Roos,
        Daar dobbert een bol Jongetjen
        Bedoven in het croos,
        Het doock terstont // tot in de gront,
        Maar in een corte poos
      
      
        Sloeg het zijn hanghden aan ons stuer
        Syn beentgens op het boord,
        Myn Luytgen sach zo evel suer
        Ay Leendert roeyt doch voort,
        Hoe staat de geck // ei lieve treck
        Jou Riemen zo ’t behoort.
      
      
        Gangs donder heb ik nu gien kracht
        Daar speult de Nicker mee
        Wat dat ik roey met al myn macht
        ‘tSchip wil niet van zyn stee,
        ‘tRoer gaf een krack // de riem die brak
        ‘tGheen ons Cupido dee.
      
      
        Dat onbeleefde guytje // jong
        Greep Luytgen bij haar hangt,
        En ik voort uit het Schuytje // sprong
        Zeer spoedich op het Langht,
        Nooit zo bevreesd // heb ik geweest
        Ik viel door schrik int sanght.
      
      
        Ik wist niet wat ik dencken souw
        Zo klein waert van ghewas,
        Hey Vryer riep hij neemt het touw
        En steekt het hout in t’gras:
        Neen lansje ziet // gij spreekt voor niet
        Het gheeft mij nu gheen pas.
      
      
        Myn Luytgen seech int Schuytje neer
        Beswymde kout en stijf,
        Ik dacht benaut Och! lieven Heer
        Wat komt de Meyt opt lyf,
        Is t’popelsy // of tovery
        Ik weet niet waar ik blyf.
      
      
        Haar aanzicht was zo bleeck gheverft
        Ik teech het water in,
        Gangs lyden zo de Meyt hier sterft
        Wel wel weten dat ik bin
        De oorsaeck snoot // van Luytgens dood,
        Dus met een dolle sin.
      
      
        Gaf ik de benghel sulcken bos
        En stiet hem over zij
        Voort bont ik Luytgens Cousebant los
        Dus quamt een weynich bij,
        Ik was vol vrees // Luytghen die rees
        Gans stichters ‘k was zo bly,
      
      
        Prins eer dit lieve weckertje
        Vlucht willich van ons vlooch
        Zo greep dat slimme leckertje
        Twee Pylen met zyn Boog,
        De een raakten myn // vol druck en pyn,
        D’aer Luytgen tot meedooch.
