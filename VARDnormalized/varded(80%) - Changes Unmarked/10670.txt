
    
      
        MIjn sinnen zijn mij ontstelt // ghequelt
        Met fantasie // Waar deur mijn vreuchde veld, en smelt,
        tot allen tijen
      
      
        Absency valt mij te maal // regael,
        Lief wtghelesen // Toont jonst medicinael, mijn quael,
        Wilt doch ghenesen.
      
      
        Rasende schier van rou // Kersou,
        Mistroostich van sinnen // Loop ik om v o vrou // ghetrou,
        Laat mij troost winnen.
      
      
        In mijn verdriet, bystant // te hand,
        Doet mij verwerven // Of door der minnen brand // en bant
        Zo moet ik sterven.
      
      
        Aanziet v knechts ellent // torment,
        Princess schone, En laat hem zijn lief jent // in't ent
        Crijghen te loone.
