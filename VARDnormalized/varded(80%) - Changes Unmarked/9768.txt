
    
      
        MOyses Gods Knecht ghepresen,
        Leide leven en Dood
        Voren ghehoorsaem wezen
        Den Heer, is'tleven klaer // maar
        De dood loont den sondaer // zwaar,
        Dit onderscheyt is groot.
      
      
        Twee weghen staan te kiesen,
        Elk moet voor hem zien, want
        Zijn ziele te verliesen
        Namaels int eeuwig vier // schier,
        Wij behooren te dier // hier
        Toe achten dezen pand.
      
      
        Twee poorten, en twee weghen,
        Wie den wijden onreyn
        In duysterheyt wil pleghen
        Te gaan, komen ten val // zal
        Den anderen, die al // smal
        Is, leit in 'sHemels pleyn.
      
      
        Den Gheest na d'enghe poorte
        Ons wijst, en te gaan int
        Bad der wedergheboorte,
        Want vruchtbaar rechte boet // goed,
        Voor al gedaan zijn zoet // moet,
        Ootmoedich als een kind.
      
      
        Die hier niet gaan in erven
        Gods Rijk als kinders kleen,
        Die sullent namaels derven,
        Gods kinders onderdaen // saen,
        Met kruys de small baen // gaan
        Ten leven in alleen.
      
      
        Christus is tot den Vader,
        De deur, om ingaan vrij,
        En waar hij gheen ontlader,
        In al ons onghemack // zwak,
        Ghevallen onder 'tpack // vlack,
        Te gronde laghen wij.
      
      
        Weinig soecken en vinden,
        Den smallen weg oprecht,
        Door 't vleeschelijck verblinden
        En der lusten opstel // fel,
        Wordt menigh mense wel // snel
        Der sond' en 'swerelts knecht.
      
      
        Ter wijder poorten vele
        Loopen al over hoop,
        De Hell' ontdoet haar kele,
        Heer en ghemeenen Man // dan,
        Gaan in, 'tschijnt niemand can // van
        Daar keren zijnen loop.
      
      
        Maar wilt doch niet aflaten,
        Gods Kinders al ghelijck,
        Den Gheest komt u te vaten,
        Wat lijden u gheschiet // ziet,
        God laat u int verdriet // niet,
        Hij helpt u in zijn Rijk.
      
      
        Laat ons in tijden // mijden
        Den weg der duysternis,
        En liever strijden // lijden,
        Op 'slevens enghen pad // wat,
        Te komen in de stad // dat
        Enig ons noodigh is.
