
    
      
        HOe diep zijt gij ghesoncken // met u ghewenschte smart
        O zoete zachte voncken // en borrelt in mijn hart!
      
      
        Gij komt mij staegh vermanen // van myner sielen bruyt,
        Som parst gij blye tranen // ten droeven ogen uit,
      
      
        Gheen onweer u en blusten // uit mijn verheert ghedacht,
        En of gij som wat rusten // 'twas ga'ren van u kracht.
      
      
        V crachten die mij glissen // door't merrech in't ghebeent,
        Soud' ik niet willen missen // om 'sdagheraets ghesteent.
      
      
        Al doedy somtijts quynen // als't lichaams ooch ontbeert,
        Mijn siel ziet lijckwel schynen // de schoonheid die zij eert.
      
      
        En schept zo groot behagen // in't Goddelijck sieraad,
        Dat zij na smart te draghen // voor sulcken wonder staat.
      
      
        Een wonder dat verflaeuwen // alle ander schoonheid doet,
        En dalen haar wijnbraeuwen // en zinken haren moet.
      
      
        De leden en de zeden // en't vierighe verstandt
        Op Pallas leeft ghesneden // van Pallas eyghen hand.
      
      
        Omhelst mijn gheest vol minnen;, en zij, van hare kant,
        Omhelzen al mijn sinnen // met even stracken band.
      
      
        Had mij dit heil begheven // zo was mijn weelde voos;
        En al het doffe leven // ontseech mij smakeloos.
      
      
        Des die van 't overwelven // des Werelts Bouwheer is,
        Gheef dat ik mis mij zelven // eer ik u liefde mis.
