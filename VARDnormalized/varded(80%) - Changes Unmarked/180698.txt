
    
      
        NImrod een Iagher van groter fame,
        Was uyter stamme // van Ham present,
        T’beginsel van zijn Rijke eersame
        Dat was met name // Babel bekent,
        Dit was een Heer // van zeer // groot reghement,
        Enen wilden Iagher wijdt uit ghebreyt,
        Ionck met coragie // in s’Weerelts Bosschagie,
        Maar wat hij vinck dat was ijdelheyt.
      
      
        Abraham is comen naar dezen
        Een zaad ghepresen // uit Sem hij sproot
        Die joeg al anders // alzo wij lezen,
        Den eeuwich wezen // bij God devoot,
        Hij kwam // en nam // Loth uit ghevangens noot,
        Van Kedor Laomor, die ook planteyt
        Wel joeg uit spijten // die Sodomijten,
        Maar dat hij vinck het was ijdelheyt.
      
      
        Esau wesende Iacobs broeder
        Van ener Moeder // Rebecca fijn,
        Iacob vercoos als een wijs bevroeder
        Enen hoeder // der schapen te zijn
        Maar Esau // zeer ru // op dat termijn
        Was een Iager, zo die schrift zeid,
        Meest al zijn leven // daar in bedreven,
        Maar dat hij vinck het was ijdelheyt.
      
      
        Iosua wesende licht ter velde
        Heeft met ghewelde // ook vele ghejaeght
        Die Coninghen eenendertich men telde,
        Die als rebelde // waren gheplaeght,
        Binnen Canaan // verwan // zeer onversaeght
        Iosua al haar Majesteyt,
        Diet anders wagen // zonder Gods behagen
        Al wat ze vanghen is ijdelheyt.
      
      
        Gedion naar des Heren beslechten,
        Is wel ten rechten // ter jacht gegaan
        Vergadert met zijn drie honderd knechten
        Ghestelt om vechten // en joegen saen
        Na haar begeer // dat heer // van Madiaen,
        Die int land waren uit ghespreyt,
        Om te verderven // des Heren erven,
        Maar wat ze vingen was ijdelheyt.
      
      
        Ionathan zijnde zijns levens wager
        Was een Iager // niet van Conijn
        Hij track op met zijnen Wapendrager,
        Wel een verslager // der Philistijn
        Hij bracht // met kracht // en deet in doods ghepijn
        De ander en hebben hem niet verbeyt,
        Elk henen vluchte // door dit gheruchte
        Maar datse vingen was ijdelheyt.
      
      
        David die claeght dat hem zijn vijanden,
        Strikken en banden // leiden ghenoech,
        Om hem te vanghen tot zijner schanden,
        Maar door zijn handen // den Here verjoech
        Doen hij zo plat // den Goliath // versloech
        Wie hoorde ook van vromer feyt
        Die ander hopen // moesten gaan loopen,
        Al wat ze vinghen was ijdelheyt.
      
      
        Nebucad Nezar, na Schrifts oorconden,
        Heeft ook gezonden // een Heir zeer snel
        Die hebben Landen en lieden verslonden
        Als Leeuwen monden // wreet en fel,
        Zij kwamen // tezamen // tot Israel
        Hebben Bethulia beleyt
        T’was haar bedenken // haar al te krencken,
        Maar datse vinghen was ijdelheyt.
      
      
        Veel wilde Iaghers na Schrifts verclaren,
        Hebben veel Iaren // dat volk verdooft,
        Zij troosten haar met vredighe maren,
        Oft leughens waren // het werd geloofd
        Men stak // tot ghemack // pueleins onder t’hooft,
        Dat heeft den Here quaelijck ghegreyt,
        Soecken de Zielen // zo te vernielen,
        Maar dat zij vinghen was ijdelheyt.
      
      
        Aan wilde Iaghers het nooit en faelde
        Elk vierich straelde // liep diligent,
        Waar door zo Ieremias verhaalde
        Dat volk zeer dwaalde // maar ongesent
        Vol van bedroch // zijn doch // als Herders blent,
        Hebben die schapen qualijck gheweyt
        Die dus omginghen // met zulke dingen
        Al wat ze vinghen // t’was ijdelheyt.
      
      
        Paulus joeg maar noch niet ghegrepen
        Want tot Philippen // ghetuyght hy’t claer,
        Zijn vaste hoep is hem niet ontsepen,
        T’stont zo gheschepen // hij joeghder naar
        En hadde lust // met rust // te zijn bij haar
        Maar liever bij den Here ghecleyt,
        Die anders leren // tot groot afkeeren,
        Al wat zij vanghen t’is ijdelheyt
      
      
        Die wilde Iaghers wilt dit recht vaten,
        Hoe schoon zij praten // ghelooftse niet,
        Laat u niet vanghen, als tot Galaten
        Ter cleyner baten // wel is gheschiet,
        Want dat verwekt // en strekt // tot groot verdriet,
        Hier na salt noch wel worden beschreyt,
        Die tot verscheuren // altijd labeuren
        Al wat zij vanghen t’is ijdelheyt.
