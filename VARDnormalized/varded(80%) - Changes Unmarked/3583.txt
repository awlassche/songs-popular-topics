
    
      
        Komt gij Syrenen al
        Met u helder gheluyt
        In het Pegasydes pleyn
        Daar de Lieffde gaat in treyn
        Heel crachtich van firtuyt:
        Hoe wel het Loovren dal
        Door Boreas verhard
        In hem zelven zo verdort
        Dat het schier vernieticht wordt
        Den Nimphen tot een smart
        Door een Christalynich ijs
        Na des winters coude wijs
        Wilt voort bringhen // daar
        Helder singhen // claer
        V stem niet en crenct
        Maar een Lietgen schenckt
        Mijn Lief tot een Nieu Iaer.
      
      
        Apollo zal ook wel
        Vermaecklijck doen het best
        Als Pan op Siringes fluyt
        Accordeeren int gheluyt
        Met Amphion opt lest
        Orpheus liefflijck spel
        De dieren wel vergaert
        Zo dat door de zoete vreugd
        Menich herte is verheucht
        Vyt jonstelijcken aard /
        Liefdes crachten maect becant.
        Ghelijck als men Meya plant
        Tot t'vervreughen // maar
        En 'tlangh heughen // naar
        Dat een Minnaer heeft
        Dit zijn Lieffgen gheeft
        Tot enen Nieuwen Iaer.
      
      
        Vernieut u Liefste zoet
        In een volcomen rust
        Waar na menich herte haect
        Dat door lieffde hittich blaect
        Al eert gheniet zijn lust
        O gij mijn Nimphe vroet
        Op u rust legher sacht
        Vyt een vrientlijck vrij bestaan
        Werde door rein Lieffd' gedaÄ“
        Dit Nieu Iaer hier gebracht
        Liefste die in Liefden slaapt
        Hier uit wat genuchten raept
        Met tbegheeren // zwaar
        Mits t'onberen // haar
        Die partydich // fijn
        Geheel nydich // zijn.
        Twee Liefgens int Nieu Iaer.
      
      
        Princess Liefste schoon
        Mijn waerde ghesellin
        Overlegghet menichwerf
        Wat moeyt op des werelts erf
        Mijn aenbragt deze Min
        Om u Lieff zoet persoon
        Die deuchdich zijt begaeft
        Met veel minsaemheden fier
        En een oversoet manier
        Waernae mijn herte slaeft
        Om te winnen int ghemoet
        Eens de alderliefste zoet
        Does rein schone // caer
        Recht mijn loone // waar
        Int volheyrden // trou
        Soo's aenveyrden // wou
        Mij voor een Nieuwe Iaer.
