
    
      
        BRoeders eersame // als gij present
        In 's Heren name // vergadert bent
        En daar anhoren, zijns woords vermaen
        Zo komt weer voren, het scheiden gaan
        Ziet dat gij 't woord // dan niet versmoort,
        Maar zo 't behoort // vierig zijt onderdaen.
      
      
        Want die 't woord horen // en zijn voor God
        Niet uytverkoren // maar die 't gebod
        Hoort en bewaren // is saligheydt
        Na Schrifts verklaren // dan toegeseydt:
        Dus yder een // vrinden gemeen,
        Zijn herte reen // daar toe make bereid.
      
      
        Want in een stenen // herte of land,
        Wil Gods woord reene // niet zijn geplant;
        Het woord des Heren // als ploegh bequaem,
        Moet 't eerst omkeren // dat aangenaam
        Het steenigh hert // vermurwet werd,
        En 't woord anvert // vruchtbarig na 't betaem
      
      
        Die 't woord ang'name // ontvangt en plant
        In zulk bekwame // een hert of land,
        Dien zal het dragen // in overvloed
        Naar Gods behagen // 's Geests vruchten zoet,
        En naar does tien // zal dien geschien
        Eeuwig verblien // aan hert, sin, en gemoet.
      
      
        Broeders in 't scheiden // het woord vol free
        In soo'n bereiden // hert, draagt doch mee,
        't Zal u vrucht bringen // die 't boven gaat
        All' aertsche dingen // hoe delicaet,
        Ia niet daar bij // te gelijcken zij;
        Broeders tot dy // zo ook ons hope staat.
      
      
        Anders waar 't horen // des woords vermaen
        Niet dan verloren // kosten gedaan,
        Ia om bespotten // van yder een,
        En 't doen eens zotten // was men gemeen,
        Die los op 't sant // gants onbestant
        Timmerde, want // zijn werk dat viel daar heen
      
      
        Van diergelijcken // Ezechiël
        Ons ook doet blijcken // hoe Israël
        Hem daar te voren // des Heren woord
        Zouden anhooren // maar nochtans voort
        Doen na haar raad // en opset kwaad;
        Maar op does daad // was de Here gestoort.
      
      
        Schoon wij zo spreken // O Broeders dy,
        Hier bij geleken // gants niet en zij,
        te Uwaerts betrouwen // al beter wij,
        Dan te een waerschouwen // is dus van mij
        Gedaan alleen // of wij naar 't scheen
        Eeuwig bij een // mochten vergaren bly.
      
      
        Nu ik hiermede // beveel u God,
        Dan noch mijn bede // tot u in 't slot,
        Is dat vermeere // u liefd' en deugt,
        Dat te allen kere // gij suyver meugt
        Wezen ter noot // zonder anstoot,
        Voor klein en groot // zo suldy zijn verheugt.
