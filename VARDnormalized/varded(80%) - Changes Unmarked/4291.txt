
    
      
        IOngh besneden // vat //
        Bloemken weerdig geeert,
        Och! ist reden // dat //
        Gij mij dus tormenteert,
        Bevroet het: // gij //
        Lief, doet het // mij,
        Versoet het // vrij, want geen //
        En mach al mijn geween //
        Blusschen dan gij alleen.
      
      
        ACO leye // baert
        T'genadich herte schoon:
        In den Meye // waert
        Gij lief // mijns hoofts een croon,
        Genesen // doet
        T'verwesen // bloed,
        In dezen // moet gij zijn
        (Onder der zonnen schijn)
        Mijn enig medecijn.
      
      
        Beleeft, ende // wijs,
        Gestadich rein gelaat,
        Voor ellende // prijs
        Mij geeft, aanziet den staat,
        Mijn leven // bloot,
        Daar neven // dood,
        Is even // groot // geplant
        Liefste in uwer hand:
        Dus doet mij onderstant.
      
      
        MYNder sielen // vreugd,
        In als confortatijf:
        Die vernielen // meugt,
        En weder schincken t'lijf:
        Voor bloedich // strijd,
        Die gloedich // snijt,
        Sachtmoedich // zijt // oprecht,
        Die niet en wederleght
        Te zijn dijn eigen knecht.
      
      
        KENt int laetste // nu
        Dijn Prinse, teer, en mat:
        Die op t'naetste // u
        Begeert voor goed, of schat,
        T'vertouven // lanck,
        Is drouven // drank,
        T'behouven // dwanck // ter noot
        Te comen in dijn schoot,
        Dus thoont u hulpe groot.
