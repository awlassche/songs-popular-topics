
    
      
        COR ect // in heusscher seden,
        Weet ik een vrouwe // fyn:
        Perfect // en zo besneden,
        Mocht noit kersouwe // zijn:
        
          T'sanguijn
        
        Aenschijn,
        Divijn:
        Dunct mij zo glorieus:
        Mijn herte // gesont
        Deur smerte // gewont,
        Maect zij victorieus.
      
      
        Notabel // Rethorijcke,
        Haar edel tonge // waeght
        Zo Abel // in muzieke
        En was nooit Jonge // maagd,
        Wie claeght,
        Of knaeght,
        Zij draagt,
        In als haar even // wijs,
        Van claren // gesanck,
        Of snaren // geclanck,
        Moet ik haar geven // prijs.
      
      
        Een // jeugd // is zij bepeerelt,
        Geprezen // wonder // baer:
        Geen // vreugd // in gans de weerelt
        Can wezen // zonder // haar,
        Geen zwaar,
        Gevaer,
        Int iaer,
        Can mij ter noot // verslaen:
        Bij reden // gezeid,
        Ben heden // bereid,
        Voor haar ter dood // te gaan.
      
      
        Lust draegh ik // int beminnen,
        Van dezer weelden // vloet:
        Al saegh ik // duyst goddinnen,
        Verciert als beelden // zoet:
        Lijf, goed,
        En bloed,
        Zijn moet,
        Alleen in haarder // hand:
        Niet blijders // behaagt:
        De nijders // gevraagd,
        Waar is dierbaerder // pant?
      
      
        In vroetheyt // zo deurdroncken,
        Can geen gecroonder // staan:
        Van soetheyt // overschoncken,
        Sach ik nooit schoonder // aan,
        Zulk graen
        Werd saen,
        Misdaen,
        Die haar tooght cleynder // staat:
        Bij crachte // can // zij
        De clachte // van // mij
        Croonen, met reynder // daad.
      
      
        Als ik // verwondert // mercke
        Wat zij ter eeren // biedt:
        Den strik // van honderd // sterke.
        Vileyns, can deeren // niet,
        Dus vlied
        Verdriet:
        Want ziet
        Mij is den morgen // bij:
        Mijn // sinnen // bezwaard,
        Zijn // binnen // bewaard,
        Ja gans van sorgen // vrij.
