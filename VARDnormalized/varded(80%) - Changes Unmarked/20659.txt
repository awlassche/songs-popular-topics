
    
      
        ALder-soetste Jesu mijn!
        Mijnen Heer! mijn Heil, mijn Medicijn,
        Groot geluk // wensch ik U in den druck
        Die Gij lijdt // om mijn grote zonden op does tijd.
        U jonge bloed // trouw vol-doet// voor de boet
        Van mijn zond' // waar mee ik U heb gwondt.
        O! mijn Heer // ik begeer // nimmermeer
        Te-gen U // op te staan, dit beloof ik nu. U jonge.
      
      
        'k Wensch U een gelukkig Jaar
        JESU Lief! ik wensch dat w'allegaer
        Uwe eer // in dit Jaar // meer en meer
        Met de mond // wijt verbreyden mogen stond op stond.
        'k Wensch dat Gij gaf // van nu af // dat niet laf,
        Maar met vlijt // ik U dienden te allen tijd!
        'k Hef mijn oogh // na om-hoogh // en ik poh
        Van does uur // U te dienen in zoet en suur.
      
      
        Ik zal nu beginnen gaan
        JESU Lief! 'k zal mij van zond' ontslaen:
        Ik wil nu // staende-voet dienen U,
        Metter daad // ik nu mijne zonden gants verlaat.
        Ach! ik beschrey // het gevley // en 't verley
        Van mijn vleysch // 'k volgh voortaan des redens eysch,
        Ik besteen // ik between // all' het geen
        't Welk Gij haat // die gheen kwaad ongestraft en laat.
      
      
        Ik aenbidd' U, kleine Kind,
        Ik weet niet, wat dat uwe liefd' versint!
        Gij komt teer // in een stal, hier ter ne'er
        Op does aerd' // die nochtans des Hemels Troon zijt waerd.
        Op 't aerdtsche dal // in een stal // voor ons all',
        Komt Gij Heer// arm en naakt, en zonder eer.
        Gij ontsiet // geen verdriet // Gij vergiet
        't Jonge bloed // Gij dit all' om mijne zonden doet.
      
      
        Wat zal ik U geven we'er
        JESU Lief! voor deze grote eer?
        'k Zal mijn le'en // van den hoofd, tot bene'en,
        En mijn min // U opdragen met mijn hert en sin.
        'k Schenck met ootmoed // U mijn goed // en mijn bloed,
        Om dat Gij // dit geschonken hebt aan mij.
        Heer! ik stijf // ziel en lijf // tot gerijf
        Van uwe lof // die mij schencken zult 's Hemels Hof.
