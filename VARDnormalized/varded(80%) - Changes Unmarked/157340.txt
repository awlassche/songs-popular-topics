
    
      
        T'Drij enig wezen // het eeuwig ende heilig woord
        Can ghenesen // daar de siel leidt in versmoort
        Wie de deuchden // leiden uit dit traenigh dal
        Naar de vreuchden // daar de siel rust hebben zal.
        Die de zonden blusten // waar med' de siel besproeyt
        Brengt met lusten // in veel rusten
        Waar Gods ghenade eeuwig vloeyt.
      
      
        Die wilt smaecken // het hemels licht van onzen al
        En gheraecken // daar men noot eynt vinden zal
        Noot u sinnen // zetten wilt op t'swerels pracht
        Want van binnen // allenskens dan u siel versmacht
        En wilt dan noot lieghen // en u van d'onsuyver zond
        In slaap wieghen // en bedrieghen
        Want zij trekt de siel tot grondt.
      
      
        Wilt u dan keren // tot God den Heer die t'al beooght
        En vermeeren // zijnen naam ende verhoogd
        Die gheschapen // heeft den mensch tot zijne vreugd,
        Wilt die dan rapen // uit de waar en suyver deucht
        Verlaat s'werels quaelen // Gods glori dan hebben zult
        Die neer kwam daelen // om te betaelen
        Met de dood ons grove schuld.
