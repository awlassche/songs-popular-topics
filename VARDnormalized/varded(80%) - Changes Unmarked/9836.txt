
    
      
        T'Ghesichte, 't eerste der vijf sinnen,
        Doet ons onderscheyden // klaer
        Gods werken al ter Wereld binnen,
        Die zijn Macht verbreyden // maar
        Nooit niemand moghen // heeft ghewis
        God zien met ogen, zo hij is:
        Maar die tot // zijn ghebodt
        Keert met lusten,
        En volherdt in deuchden // vroet,
        Die zal God // t'saligh lot,
        Zien in rusten,
        T'wert een vreugd der vreuchden // zoet.
      
      
        T'ghehoor is den weg daar 'tgheloove
        Door komt, uit des Heren // woord,
        Daar 'thert' opmerckigh, niet als doove,
        Recht Gods bode leren // hoort:
        T'ghehoor oft lezen // salight niet,
        Self doender wezen // moetmen, ziet,
        Met verdragh // dat op dag
        Van Gods oordeel
        Van ons als verblijde // fijn,
        'tSoet ghewach // gehoord magh
        Zijn tot voordeel,
        Komt ghebenedijde // mijn.
      
      
        Den roke der salvinghe reine
        Van des Heren goeden // Gheest,
        Die neder vloeyt uit s'deuchts fonteine
        Hier in den ootmoeden // meest //
        Elk Christen geven // moet ook uit
        Zoet rokigh leven // als de Bruydt
        Van Sioen // zijnde koen
        Wtghesproken,
        Een kruydigh bekwame // hof,
        Altijdts groen // in wel doen,
        Zoet van roken
        Gheeft zij haar vreindts name // lof.
      
      
        De smake van Gods goedheid mild,
        En 'sHeeren woord goedertier,
        Twaer goed dat ieder daar aan wilde,
        Als kind dat de Moeder // hier
        Suyght om op groeyen // met zulk sogh,
        Daar niet int vloeien // is bedrog,
        Alzo haeck // na de smaak,
        Van de weerde
        Heylighe Schriftuere // elk,
        Zulk gheen saeck // tot vermaeck
        Hier op eerde,
        Als does voedtsaem puere // melk.
      
      
        Ghevoelen laat ons t'lyden smertigh,
        Dat hier onzen Naasten // dooght,
        Als Samaritaen die goethertigh,
        Hulp toe doen met haesten, pooght,
        'tQuaet laat ons mijden // met verstandt,
        En te allen tyden // goed doen, want
        s'Heeren raad // ons wel staat
        Te vervullen,
        Zonder te verkoelen // vrij,
        Want naar daedt // goed oft kwaad,
        Namaels zullen
        Int lichaam bevoelen // wij.
