
    
      
        MYn tijd // verslijt // ik lief ghepresen
        Met vrezen // gants en gaer
        Die lijt // subijt // wilt dienstich wezen
        Ghenesen // zyn gheuaer
        Maakt dat hij werd ghewaer,
        V hulp tot hem gheneghen
        Door praet // ghelaet // zyn hart versleghen
        Met raad // bystaet // in deze nieuwe Jaar.
      
      
        V heeft // beleeft // ghedient jonckvrouwe
        Ghetrouwe // does dienaar
        Hem geeft // hij beeft // blyschap voor rouwe
        Voor trouwe // vreugd eerbaer
        Maakt v en hem eenpaer
        Zo zal zijn druck cesseren,
        Met spoet // lief doet // al v begheren
        Hem zoet // ontmoet // in deze nieuwe Jaar.
      
      
        Dat hij onvry // v troost moet deruen
        Doet steruen // vreuchde maar,
        Zeer bly // hoopt hij // v hart te eruen
        T'verweruen // zal voorwaar,
        Doen slijten t'lyden zwaar
        Dat hem nu quelt gestadich
        Dus vrou // zyn rou // verdrijft genadich
        Die iou // dient trou // In deze nieuwe iaer.
      
      
        vlijdt // zeer blijdt // Princes verheuen
        Beneuen v dienaar
        Wient spijt // benijt // laat zijn verdreuen,
        Het sneuen // en misbaer
        T'welck hem quelt voor als naar
        Wilt doch zyn pyn versachten
        Dat hij // lief bij // v mach veruachten
        Heel vrij // en bly // in deze nieuwe Jaar.
