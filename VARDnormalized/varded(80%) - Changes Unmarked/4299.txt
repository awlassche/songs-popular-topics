
    
      
        MAegdelick schoon gestichte //
        Deurschijnich da- geraet //
        Claer amoureus gezichte //
        Die angst en noot verslaat //
        Rein saet //
        Gij gaat //
        In daad //
        Vesta te boven // wijt //
        In vreed' // en // accoort //
        Gij heden // en voort //
        Zeer hoog te lo- ven zijt.
      
      
        Aanziet mij knecht der minnen,
        Als een die deugd hantiert:
        Den Chaos mijnder sinnen,
        Verwert, ongeregiert,
        Maniert,
        Scholiert,
        En stiert,
        Naar uwen reden // toom:
        T'wert stadich // begeert,
        Genadich useert
        Aan mij, dijn seden // vroom.
      
      
        Rachel zijdy int wezen,
        Naar dat d'estime // luydt,
        Als Sara wt gelesen,
        O magnanime // bruydt,
        Die wt,
        Zulk cruyt,
        
          Virtuyt
        
        (Tot zijn gesonde) // haalt:
        Geen smerte // verstranght,
        Zijn herte // ontvangt
        Troost, die ten gronde // daalt.
      
      
        Ist, dat deur u ontfermen
        Ik vreugd gebruiken // can:
        Den lust-hof uwer ermen,
        Laat mij omluycken // dan:
        Geen man,
        Werd van,
        T'gespan,
        Der minnen // eer // gevrijt:
        Ben bange // bedrouft,
        Niet lange // vertouft,
        Maect mij eens zeer // verblijd.
      
      
        Adieu zoete Princess,
        Dijns houd ik goede wacht:
        Gij zijt een forteresse,
        Onwinnelick bij kracht,
        Geen macht,
        Geacht,
        Gij slacht
        Minerva, gans // en // al:
        O fiere // divijn
        Lauriere // die mijn
        Hoofd noch becranssen // zal.
