
    
      
        O Mey playsant // coutheyts vijand
        die weelde plant // zeer triumphant,
        Aan elken cant // u comen loven,
        want u aessem sacht // zeer Liefflijck lacht,
        Door u dous kracht // werd voortgebracht
        Dat lach versmacht // in coud’ bedoven,
        V oude fleur komt weer boven,
        Omhelzen doet // gij Lucht en Aarde,
        T’welck overvloed // voort brengen doet,
        Door vruchten zoet // zo dat men spoet,
        Nuttigheid goed // voorcomt in waerde.
      
      
        T’gheboomt neemt aan // cleedingh van blaen,
        Die groenlijck staan // tot lust verlaen,
        Gaat ooc ontfaen // zachte winds bewegen,
        Die velden groen // die wat leeft voen,
        Tot ruychte spoen // in dit saysoen,
        Door het toedoen // van uwen zegen:
        T’ghebloemt heeft me vercreghen
        Schoonheid, reuk, en cierlijck bloeyen,
        De Thuynen gemeyn // veelverwich rein
        Staan groot en cleyn // in onze pleyn,
        Door u certain // lieflijck en groeyen.
      
      
        Het pluymgediert // zeer schoon verciert,
        Zijn sang bestiert // en vreuchlijck tiert,
        Soot ghemaniert // is van natueren:
        Onrustich noch zwaar // zonder misbaer
        Is t’water claer // en lijdt voorwaar
        T’ghebruyck van haar // tot allen ueren,
        Gheen onghemacx besueren
        Wilt gij lustighe Mey lijen,
        Voorteling bespoort // gij aan allen oort
        Met zoet accoort, nu rechte voort,
        Ende oorboort // lieflijck verblijen.
      
      
        O vrolijcke tijd // diet’ al verblijd,
        Prinslijcke vlijt // gij waerdich zijt,
        Ghebenedijt // gaat men u achten
        Want Auroras gsicht // Flora u Nicht
        Phebus zijn licht // Venus ghericht,
        Cupido schicht // brengen haar crachten,
        Nuttelijckst zij altoos wrachten
        Wanneer gij Iaerlijcx zijt verschenen,
        Met snaer gheclanck // lieflijck gesanck,
        Deuchtlijcke danck // zijt hier in swanck
        Den Lauren Rank // wij nemen.
