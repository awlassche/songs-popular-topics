
    
      
        PArst uit 't ghesicht een droef ghetraen,
        Laat big'lend' langs does kaecken gaan
        Brack vloed // die ik moe
        Op d'aerde gieten
        Met naar ghesucht, helaes om een,
        Wiens harde hert blijft als een steen,
        En niet acht // na de klacht
        Van mijn verdrieten,
        't Gheen ik staegh // alle daegh
        Doe voorwaar // of ik haar // noch hier naar
        Hier door eens mocht ghenieten.
      
      
        Doch 't schijnt mijn smeeckingh zij begeckt,
        Of dat haar borst een rots verstreckt:
        Want wanneer // ik begeer
        Met zoete woorden
        Haar aan te spreecken, stuers en straf
        Draeyt zij haar wezen van mij af,
        En gaat heen // of mijn reen
        Zij niet en hoorden,
        Ja het schijnt // dat zij vijnt
        Hier in rust // en haar lust // werd gheblust
        Met dus mijn ziel te Moorden.
      
      
        Wel Amarille zegt mij doch
        Wat hebt gij voorghenomen noch
        Voor mijn trouw // waerde vrouw
        Mijn te doen erven?
        Wat is het gheen, zegt Harderin,
        Dat gij met mijn hebt in u sin?
        Sleghts ghebiedt // daar is niet,
        Lust u mijn sterven?
        Of begheer // gij noch meer,
        'k Ben te vreen // wilt mij leen // vrij van een,
        Van lid tot lid doen kerven.
      
      
        Is 't dat u wraeck dan meerder dorst:
        Zo komt en opent vrij does borst,
        Wt zijn stee // rockt vrij mee
        Mijn trouwe herte,
        Klooft het aan tween, en ziet Goddin
        V eyghen beeltenis daar in,
        't Gheen ghewis // d'oorsaeck is
        Mijns vreughds en smerte,
        Gij zult zien // dat er gien
        Minnaer trou // is Me-vrou // of ik sou
        In liefd' hem durven terten.
      
      
        Want wat gij wilt, ik ben te vreen,
        Ik vrees gheen straf, hoe groot, o neen:
        'k Ben gherust // als gheblust
        Werd u begeren,
        Want Lief, mijn leven en mijn dood
        Bestaat in u vermoghen groot,
        Wat gij doet // 't is al goed,
        't Kan mijn niet deeren,
        't Gheen u haeghd // waerde Maeghd
        Doet dat vrij // laes aan mij // 'k zal mij bly
        Na u behaghen keren.
