
    
      
        In ‘t geest’lijck Bloemen-hof // staat ene Blom gheplant,
        Waerdigh om hebben lof // hij staat in ‘t effen Land,
        Te Saron zeer playsant // lieflijck en uytghelesen,
        Boven all’ schoonheid: want // ’t is Godes Soon geprezen.
      
      
        Hij staat in ‘t herte: daar // niet is enig onreyn,
        Want deze hebben haar // ghekeert tot de Fonteyn
        Des levens, daar is pleyn // geen Was-bat meer te vinden,
        Om ‘t hert te wassen rein // dan does enig beminden:
      
      
        Druckt mij spreekt Christus zoet // als een Zegel op ‘t hert
        Een herte vol ootmoet // dat rouwe draagt met smert,
        Veel troost gegeven werd // een troost vol soeter minne,
        Komt al, ‘t zij na oft vert // gaat tot dit Blom-Hof inne.
      
      
        Spreect dan met blyden // spreect dan met ’s herten sin,
        O Name Iesu! goed // hoe zoet is uwe Min,
        Hoe groot is dit ghewin // hoe saligh zijn de ogen,
        Die dit te recht zien in // die werden niet bedrogen.
      
      
        O Name Iesu! wien // hoog zit ter rechter hand,
        Voor u moet alle knieën // ootmoedigh buyghen, want
        Gij verwont triumphant // alle onze vijanden,
        O Coningh zeer valiant // naar u moet ons liefd branden.
      
      
        O Iesu, Lam Gods eel // op Syons Berghe schoon,
        Daar gij de Scharen veel // opsetten zult de Kroon,
        Hoe heerlijck is u troon // hoe lieflijck is u wezen,
        Hoe groot is uwen loon // u Naam moet zijn ghepresen.
      
      
        O Iesu, u aenschijn // ist schoonste boven al,
        Hoe lieflijck salt daar zijn // daar men ‘t aenschouwen zal,
        Komt al gij Christen tal // komt laat ons vrolijck singhen,
        Komt maakt nu bly geschal, komt wilt Gods lof voortbringen
      
      
        Komt maar uit sondig huis, kleeft niet meer aan het kwaad,
        Verlaat ‘t sondigh abuys // met al des werelts staat,
        Niet meer met haar en gaat // op hare kromme weghen,
        Haar aerts gierigheyd haat // zijt tot wel doen gheneghen.
      
      
        Dan find gij ’s herten wens // dan vint gij wat gij wilt,
        Kont gij verstaen o mens // zo werd u hert ghestilt,
        Ik denck dat gij zwaer tilt // de wereld heel te laten,
        Maar of gij die behilt // wat sout u konnen baten.
      
      
        ‘t Is doch al ydelheyd // wat de wereld hoog acht,
        Rijkdom haast onder leyd // hoe hoog dat schijnt haar macht
        Al wast Konincx geslacht // ’t moet al ter Aarden onder,
        Dat dit niet meer bedacht // en werd, dat is groot wonder.
      
      
        Den onderganck is zwaer // van sulcken dwazen rijk;
        Van zulk een aerts dienaar // die ‘t hert van God afwijck,
        Al tot dat aertsche slijk // al wilt gij ‘t nu niet weten,
        n Zal haast komen, gelijk // een val-strick neer ghesmeten.
      
      
        Dan zult gij worpen heen // u gouden Goden rond,
        Die hier u troost wel scheen // maar dan in den afgrond
        Laat dalen u terstont // o mensch! wilt dit bedenken,
        Laat af u kwade vont // God wil u beter schencken.
      
      
        Wilt gij dat beter zien // dat God u schencken sell;
        Wilt gij weten in wien // gij dit zult vinden wel:
        In Christo, haast u snel // ziet zijn leven, manieren,
        Zijn woord, waerheydts bevel // daar in zal hij u stieren.
      
      
        Toeft niet, beyt niet, o mensch // in dit Blom-hof te gaan:
        Hier vint gij ’s levens wensch // haast u, wilt niet stil staan:
        ‘t Moet hier doch al vergaen // maar Gods rijk zal beklyven
        Gaat voort door d’enge baen // zo zult gij eeuwig blyven.
