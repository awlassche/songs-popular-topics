
    
      
        GOddinne van der Minnen
        Gij weet wie ik verhieff
        En wat int herte binnen
        De Lieffd' al liefs besieff
        Goddinne vroet // de Liefste zoet
        Doet mij me hebben lieff.
      
      
        Dat zij zonder weerlegghen
        Met een verheucht aenschijn
        Vrymoedich mach uytsegghen
        Ik mindren zal u pijn
        Ook anders gheen // dan gij alleen
        Iae / ja / gij zult hem zijn.
      
      
        Voor zulke go weldaden
        Gij Cypria ydoon
        Zal ik vroeg ende spade
        Volloven u persoon
        Door gonsticheyt // mij toegheleyt
        Van Cupido ydoon.
      
      
        Laat haar u schichten voelen
        Int binnenste verstand
        Zo mach dan haast vercoelen
        Inwendich dezen brand
        Die gij mij stuert // theeft lang geduert
        Door Lieffde abondant.
      
      
        De peerle mijnder ogen
        Is deze Nimphe rein
        Zal haar altijd verhooghen
        Vyt lieffden int ghemeyn
        Zo trou beclijft // en zij dan blijft
        Mijn lavende Fonteyn.
      
      
        Als Aurora int dagen
        De neveling verclaert
        Zo licht mijn Lief bij vlaghen
        tGhemoet dwelck was bezwaard
        Voor 'tbitter roet // zij suycker zoet
        Mij schenckt uit goeder aard.
      
      
        Tis reden dat ik ere
        Does eenichste in al
        Van leden zoet en teere
        Zij is / en blyven zal
        Den spieghel rein // om zien alleyn
        tVoorleden ongheval.
      
      
        Het herte jong in deughen
        Rein wederliefde biet /
        Wanneer dat het vol vreuchden
        Zijn alderlieffste ziet /
        Het is verknocht // vast tsaem gebrocht
        Om gants te wijken niet.
      
      
        Princess uytghelesen
        De goetheyts gaven groot
        Gheduerich zijn ghepresen
        Bij twee schoon Liefs minjoot
        Wanneer dat hij // en weder zij
        Aenbiet 'tmondelijn root.
