
    
      
        Rein van manieren // is den Echtenstaet,
        Die hem hantieren // in den rechten // graedt,
        Tis slechten raad, die hem verbieden, want
        Beter is trouwen, dan te lyden brand:
        Want die horeren // mal
        God condemneren // zal,
        Des wij lauderen // al
        T'Huwelijcx band.
      
      
        Tis waar dat vrienden // in malcanders noot
        Dikwijls wel dienden // al bystanders bloot
        Maar anders groot, met een affecty vrij
        Staan man en wijf ter noot // malkander bij
        Wie een go Vrouwe // heeft
        Die God ghetrouwe // gheeft,
        Bevrijt van rouwe // leeft
        Te langher hij.
      
      
        Een Wijf die deughdigh // is van wezen vroet
        'sMans herte vreuchdigh // zij genesen doet
        Ghepresen moet // zij zijn om haar bedrijf,
        Een edel gave Gods is zulk een Wijf,
        Tot troost van haren // Man,
        Die wel bewaren // kan,
        Haar tongh, en sparen // van
        Weder ghekijf.
      
      
        Zo lief bekwame // op der eerden niet
        Als een eersame // Wijf vol weerden ziet,
        Met zweerden vliet // een roover in der nacht
        En brengt den roof // haar die boven geslacht
        Hij voor vriendinne // koos,
        Veel zijn door minne // boos,
        Geworden sinne // loos,
        En omghebracht.
      
      
        Een vroom wijf schone // is als brandt daar
        De Lamp ydoone // opt heilig Kandelaar
        Zijnde bestande haar // eerbaer ghemoedt,
        Een gulden Pilaer op een zilver voet
        Is zij gheleken // wel
        Om niet te breken // snel.
        Oft te ontsteken // fel
        Door bozen gloet.
      
      
        Die een Wijf zedigh // voor zijn proven houdt,
        Huysich, cuysch, zedigh // maghse loven stout
        Al boven goud // en boven edel steyn,
        Haar sieraad is, wat neerstigh is en rein,
        Al moets' hier lijden // smert,
        Haar vreugd in tijden // werd
        Met een zo blijden // hert,
        In 'sHemels pleyn.
      
      
        'tWijf zal door baren // ende kinder dracht
        Ten Hemel varen // datse ginder lacht,
        Zo inder kracht // gheloove, liefde jent
        Heylicheydt, eerbaerheyt haar blijft ontrent
        Voor does versijcke // wat,
        Werd zij Gods rijke // stad,
        D'Enghels ghelijcke // dat
        Werd excellent.
      
      
        Prinse d'oprechte // zo zij deden eer,
        Zal men in Echte // niet besteden meer,
        Te vreden zeer // wel, eeuwig zonder pijn
        Zullen zij in 'sLams Bruylofts feeste zijn,
        Zo vrolijck maken // chier,
        Elk magh wel haken // fier,
        En neerstigh waken // hier,
        Om daar te zijn.
