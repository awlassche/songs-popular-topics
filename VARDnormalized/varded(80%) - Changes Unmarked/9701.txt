
    
      
        EEn ghebroken herte, na Gods behagen,
        Om na te jaghen // den vrede zoet,
        Voor de woorden Gods een vertsagen,
        Met een verslaghen // oprecht ghemoet,
        Wensch ik u tezamen voor enen groet,
        Daar toe een goed // ghesicht altijdt,
        Dat gij meugt merken // door 'sGheests verstercken,
        Waar af dat gij ghevallen zijt.
      
      
        Goud dat doort vier is zonder beswijcken,
        Coopt om verrijcken // doch niet en swicht,
        Coopt zalve om u ogen bestrijcken,
        Dat u mach blijcken // een klaer ghesicht,
        Dat u verschijne dat wonderlyck licht,
        Wordt zo ghesticht // dat gij subijt
        Mocht zien van binnen // met hert en sinnen
        Waar af dat gij ghevallen zijt.
      
      
        Gij die Gods timmeringhe reene,
        Met ons ghemeene // voortijdts waeckt,
        Keert weder tot den verworpen steene,
        Christus allene // gheopenbaert,
        Tot enen vasten gront vermaert,
        Met hem vergaert // dat hij niet wijt
        V en verstroye // keert t'sijner koye,
        Waar af dat gij ghevallen zijt.
      
      
        Komt ende smaakt de Hemelse gaven,
        En wilt u laven // aan de Fonteyn,
        En wilt u niet als arme slaven,
        Slijckputten graven // in swerelts pleyn
        Die zonder water zijn onreyn,
        Maar in den treyn // der waarheid bevrijt,
        Schiet hier u gangen // peyst met verlangen,
        Waar af dat gij ghevallen zijt.
      
      
        Hoe lieflijck ist, wanneer Gods zonen,
        En dochters wonen // in goed accoort,
        Dat wil ons God den Vader jonen,
        Met alle den gonen // diet gheern hoort,
        Dat Goddelijcke suyver woord,
        Dus van nu voort // om u profijt,
        Zo wilt ghedachtigh // zijn eendrachtigh,
        Waar af dat gij ghevallen zijt.
      
      
        Het Lam en heeft maar een ghetrouwe
        Bruydt voor huisvrouwe // in swerelts dal,
        Een is zijn Duive, zo ik aenschouwe,
        Het bed' is nouwe // en 'tdecksel is smal,
        Een kudde, een Herder, en een Schaepstal,
        Is er voor al // na Schrifs belijdt,
        End' een Ghemeente // op Syons gesteente
        Waar af dat gij ghevallen zijt.
      
      
        Verloren Schaepken krank ende teere,
        Ten rechten kere // gij wel behoeft,
        Verloren penninck een Vrouw met ere,
        Die is zo seer // om u bedroeft:
        Verloren Zone, niet en vertoeft,
        Hongherigh proeft // met groter vlijt,
        In uwer herten // met droever smerten,
        Waar af dat gij ghevallen zijt.
      
      
        Daar is noch plaatse voor den aftreder,
        Om keren weder // ter vreden band,
        Zijt gij verheven hoog als een Ceder,
        Buigt u doch neder // met u verstandt,
        Onder Gods almachtige hand,
        Dat in den band der liefden verslijt,
        Twistigh afgrijsen // dat vrede mach rijsen,
        Waar af dat gij ghevallen zijt.
      
      
        Gij hebt verlaten de eenichede,
        Den rechten vrede // hebt gij ghemist,
        Liefd' en ghenade en heeft gheen stede
        Bij u ook mede // door uwen twist,
        Dus hebt gij des Vaders goed ghequist,
        Door 'svyants list // geworden kwijt
        Zijt gij u krone // ende pand schone,
        Waar af dat gij ghevallen zijt.
      
      
        Wie bij hem zelven wel meent te stane,
        Heeft ga te slane // menighertier,
        Dat hij niet en valt uit de bane,
        Die om te gane // zo smal is hier,
        Aan d'een zijd' water, aan de ander zyd' vier,
        Vol van dangier // aenvecht en strijd,
        Dus den voorleden // weg wilt treden,
        Waar af dat gij ghevallen zijt.
      
      
        Dit zegt die Here, treedt op die wegen,
        Maar ziet ter deghen // voor u wel,
        Vraagt waar den goeden weg is gelegen
        Dien zijnde verkreghen // zonder uitstel,
        Wandelt daarin, tis Gods bevel,
        V Ziele snel // zal rusten verblijd,
        Helpt zo des Heren // ghetal vermeeren,
        Waar af dat gij ghevallen zijt.
      
      
        Moyses die heeft hem zelven geboden,
        Te laten roden // uit 'sLevens Boek,
        Ook Paulus van ghelijcker moden,
        Al voor den Joden // uit liefden kloeck,
        Om die te verlossen vanden vloek,
        In dat versoeck // u ook niet en mijt,
        Neemt aan u eeste // liefde, dits 'tmeeste,
        Waar af dat gij ghevallen zijt.
      
      
        Barmhertigheid wil ik van uwen hand
        En gheen offerande // dit woord insiet,
        Had gij gheweten, als de vailliande,
        Ten rechte verstande, zijn bediet,
        Den onschuldighen en had gij niet
        Zonder verdriet // vermaledijt,
        Maar had gebleven // Gods woord beneven
        Waar af dat gij ghevallen zijt.
      
      
        Oorlof hiermede, wilt des Heren
        Zachtmoedigheid leren // met herten ootmoet
        Tot rust der Zielen, vrij van verseeren,
        Wilt u bekeeren // tot zijn jok zoet,
        Blijft niet beraden met vlees en bloed,
        Neemt dit voor goed // tis uit gheen nijt:
        Maar om met tranen // u te vermanen,
        Waar af dat gij ghevallen zijt.
