
    
      
        Komt hier Nymphjes al te saem,
        Die tot vreugden altijdt zijn bequaem,
        Vlecht een Krans // voegt u bij dezen glans,
        Aan de Ry // bij de Harders, plucked nu Bloempjes vrij;
        Ay zoekt doch Kruid // van viertuyt, tot versoet
        Van does Maagd // die meneert dat haar behaagt:
        Noemt haar lof // nu zeer grof // spaert gheen stof,
        Maar verciert // nu haar hoofd met een Krans omzwiert.
      
      
        Tot sieraad al van haar Ieught,
        Die zij nu door brengt met grote vreugd:
        Zij ist waert // door haar deugd, dit zij baertmet geneught!
        Want zij haar door 't zoet vermaeck verheugt:
        Dus wilt haar eer // breyden veer, met gheschal
        Want 't gheluyt // speelt in 't dal doch op u Fluyt;
        Schept doch hier // goede chier // met mplaysier.
        Haar ontmoet // met Laurier haar does eer aan doet.
      
      
        Ik zal u in 't eerst voor gaan,
        Spoeydt u Nymphjes soetjens achter aan,
        Komt nu ras // hier ontrent // met ghewas:
        Want gij bent // met u kruiden Hymus wel bekent:
        Orpheus queelt // ende speelt met u Snaer
        Op Musijck, aan dit vloeyend' Beeckje klaer,
        Dat daar stroomt // door 't Gheboomt // niet en schroomt
        Voor 't Gediert // dat in 't Bos met gedruys zo tiert.
      
      
        Phoebus helder glanssend' licht,
        Brengt ons met zijn straeltjes in 't gesicht:
        Want de Zon // hier op d'Aerdt, aan does Bron,
        V bewaard // op dat vreughdigh, gij dan wordt verjaert,
        Gij wordt gekroont // en geloont // nu ter tijd,
        Dus weest gij in den Here nu t'saem verblijdt,
        Die ons loon // nu zeer schoon // stelt ten toon
        Voor ons oogh, dat het straelt, Tot den Hemel hoog
