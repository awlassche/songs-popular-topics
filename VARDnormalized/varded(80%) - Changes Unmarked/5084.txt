
    
      
        Ter wereld is geen meer ellent // dan in armoed te wezen,
        Maar als men die ten besten went // zo wordt de ziel genesen,
        Wanneer men wil bemercken // hoe zij het hert kan sterken,
        Hoe zij ‘t gemoet // noch voordeel doet,
        Meer dan den overvloed.
      
      
        Een arme die Godvrugtig is // en zijns gemoets een weter,
        Die vint het licht in duysternis // en spijze bij den eter,
        Hij suycht Honig, uit stenen // en blijschap uit het wenen,
        Hebbende niet // hij volheydt ziet,
        ‘t Welk deur Gods kracht gheschiet.
      
      
        Ten eersten heeft hij geenene noot // om ampten te bedienen,
        Burg’meester, ofte schepen groot // zal niemand op hem mienen
        Hij hoeft hem niet te moeyen // met Peerden, of met Koeien,
        Met dijc of dam // maar als een lam //
        mach hij hem houden tam.
      
      
        Wanneer den rijcken wordt belast // met tafelen vol eten,
        Zo blijft hij in zijn hutjen vast // met stilheyd neer gheseten,
        Hij mach wel gaan spanceren // als ander grote Heren,
        Op die rechtbanck // tegen haar danck //
        wel sitten uren lanck.
      
      
        Komter een oorloch in het land // hij heeft niet te verliesen,
        Oft zo daar schepen zijn gestrant // hij staat zonder verkiesen,
        Om schattinghe te geven // ist hem ghelijcke even,
        Die niet en heeft // ook niet en geeft //
        en evenwel hij leeft.
      
      
        Komt hij in grote steden gaan // men zal hem niet verdragen,
        De beurs mach hij wellaten staan // daar Kooplieden haar plagen,
        Wandelt hij op de velden // Hij is bekommert selden
        Met land of zand // maar zijn verstand //
        dat set hij na zijn hand.
      
      
        Van haat en nijt is hij bevrijt // in erf-goeden te delen,
        Zijn schoonen tijd hij niet en slijt // met jaren lanck krackelen,
        Hij mach zijn tijd besteden // in aendacht, en ghebeden,
        Van grote moeyt // is hij besnoeyt //
        op dat den geest herbloeyt.
      
      
        ‘t Geld brengt hem niet dan droefheydt in, Met dag en nacht in peysen,
        Hoe hij ‘t zal brengen tot gewin // zijn vreugt begint te deysen,
        Daar anderszins met zingen // en ander zoete dingen
        Hij op zijn werk // verheugt hem sterk,
        Ghelijck als in een Kerck.
      
      
        Als d’rijcken een huis bouwen zal // daar is zo veel te stellen,
        Dat hij mach kijcken boven al // daar gaat hij hem med’ quellen
        Schier boven zijn vermogen // dient hij den lust der ogen,
        Den armen recht // wel overlegt //
        hoe hij ‘t zal maken slecht.
      
      
        Ziet hoe den rijcken is bezweet // met vaeren ende vletten,
        Speel-jachten, en Karossen breet // hem niet dan moeite zetten,
        Veel onbetemde lusten // houden hem in onrusten,
        Sulck’ heerschappy // is slaverny //
        al scheen het noch zo vrij.
      
      
        Pytagoras heeft wel gheseyt // in ‘t verhuysen der zielen,
        Dat de een en de ander majesteyt // hem niet zo wel bevielen,
        Als ‘t vrolijck tierelieren // van klein ghe-achte dieren,
        Schoen-lappers stoel // na zijn ghevoel,
        Ginck boven ‘s Konincks gh’woel.
      
      
        Denckt als den rijcken hier verarmd // hoe zeer hij is verlegen,
        Hij is zo in het goed verwarmt // hij kan geen kleine wegen,
        Hij geeft hem dick tot lieghen // uytstrijcken, en bedrieghen,
        Een ambachts man // die werken kan //
        en weet hier al niet van.
      
      
        t’Aensien en hoeft hij in het kleed // noch huizen niet te soecken,
        Het is den rijcken dikwijls leed // dat hij in slechte hoeken,
        Wt nood hem moet gheneeren // en als zij veel verteren,
        Een groot gewin // moet komen in //
        den armen hoeft veel min.
      
      
        Wanneer ‘t gemoet den rijcken wijst // religie aan te nemen,
        En als er dan een kruysjen rijst // daar is zo veel te temen,
        Zijn eer en hoge staten // die vreest hij te verlaten,
        Den arme, hij // is hier van vrij //
        en leeft gerust en bly.
      
      
        Wanneer den rijcken hier misdoet, Niemand en derf hem straffen,
        Hier in heeft het den armen goed // want op hem gaetmen schaffen,
        Als hij niet wel en wandelt // of erghens in mishandelt:
        Dit is van God // een groot genot //
        want ‘t is der zielen lot.
      
      
        Wordt hier de’t rijcken ziek of kranc // de menicht’ van Doctoren,
        Helpen hem dikwijls van de planck // al is hij nood’ verloren,
        Den armen en geringen // gebruikt maar slechte dingen,
        En wordt ghesondt // met kleine vond,
        Wanneer het God hem jont.
      
      
        Maar noch is wel het aldermeest // als hier de Rijck’ moet scheiden
        Dat hij zo zeer het sterven vreest // of laat hij hem om leiden,
        Hij veynst hem op te geven // maar sou noch geeren leven,
        Liet God hem hier // in zijn playsier //
        hij name wel vertier.
      
      
        De dood is hem een bitter pijn // zo Syrach gaat vermonden,
        [...]en armen ist een Medecijn // te werden hier ontbonden.
        [...]e rijke komen waerlijck // in ‘t Hemelrijcke zwaerlijck,
        Want haren sin // staat meerof min //
        gehecht op ‘t aerts gewin.
      
      
        De arme hebben enen troost // aan Christi leer en leven,
        [...]ft men den rijcken al wat g’looft // ’t kan geen zekerheid geven,
        Want met den rijcken vrecken // zo moet hij hier vertrekken,
        Lazarus deel // moet hij geheel //
        aanzien als een tonneel.
      
      
        Als hier den armen lyden moet // gebreck aan kost, en kleren
        Hij heeft een groot geselschap zoet // die met hem pilgremeren,
        Die eertijdts ginghen dolen // in Berghen en in holen,
        En dezen druck // was haar geluk //
        gedenkt dit wonderstuk
      
      
        Den rijcken die zijn schueren houd’, Die moest van droefheydt huylen,
        Apostelen en hadden goud // noch zilver in haar buylen,
        Noch konden zij berlyden // in ‘t midden van haar lyden,
        Want al haar goed // was int gemoet //
        die God den armen doet.
      
      
        God heeft d’arme verkoren hier // die rijk zijn in ‘t gelove,
        Al schynen zij verlooren schier // en alleman ten roove,
        Zij zijn zalig gheheten // o wonderbaer secreten,
        Zij geven meer // met mijtjes teer //
        dan enen rijcken Heer.
      
      
        Is dit niet wel een voordeel groot // die d’arme mach geven,
        In deze tijd, en in de dood // bevrijdt te zijn van treuren,
        Oft schoon natuer moet lijden // ’t gemoet kan hem verblyden,
        Wenscht gheen rijkdom // ’t zijn weghen krom,
        Gij hoort nu den waarom.
