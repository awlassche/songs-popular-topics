
    
      
        MInnelick beeld,
        Bloemken van geure // zoet,
        Die boven kracht vermeught:
        In uwen teelt
        Gij haestich t'suere // doet
        Veranderen in vreugd:
        Gij zijt een Jeugd
        Zo delicaet,
        Daar niet dan deugd // en ere in bestaat.
      
      
        Aerdiger geen
        En was in honderd // iaer,
        Daar meer troost bij geschiet:
        Die mijn geween
        Gans wtgesondert // zwaar,
        Zo haestich bringht te niet,
        Nu gij u biedt
        Behulpich zeer,
        Hoe can verdriet // mij deeren immermeer:
      
      
        Rustich ghesint
        (Vrij aller plagen // last)
        Ben ik, bij uwen raad,
        Te zijn bemint
        In u behagen // vast,
        Schat-ick den hooghsten graet:
        Faelgeert u daad
        Op dezen // dach,
        Geen toeverlaet // op aarden wezen // mach.
      
      
        Int herte gij
        Mij als een plant // draegt,
        En hebt, van aenbegin,
        S'gelijcx ik dy
        Als een constant // maagd,
        Prente in mijnen sin:
        Zulk een ghewin
        Houd-ick in prijs,
        Als t'wesen in // der Goden Paradijs.
      
      
        Adieu, hier met
        Moet het gescheiden // zijn,
        Princess excellent:
        Wilt (met opset)
        Jonstich verbeyden // dijn
        Dienaar zeer diligent,
        Die niet en kent
        Vreugd, of iolijt,
        Dan u present // te wezen te aller tijd.
