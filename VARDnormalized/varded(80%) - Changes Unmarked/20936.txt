
    
      
        Weest verblijdt // te allen tijd
        's Hemels Koninginne
        Alleluya, alleluya,
        Ik u opdraegh all' mijn sinnen.
      
      
        Want gij hebt // die 't all' scheyt,
        Maagd! in u ged ragen,
        Alleluya // alleluya,
        God in u had groot behagen.
      
      
        Dezen Heer // heeft zijn keer
        Uit het graf genomen,
        Alleluya // alleluya,
        En is we'er op d'aerd' gekomen.
      
      
        Onverwacht // uit het gracht
        Is uwe Soon verresen;
        Alleluya // alleluya,
        G'lijck voorzeid hij had voor dezen.
      
      
        Weest verheugt // weest verneught,
        Waerdtste Godes Moeder!
        Alleluya // alleluya,
        Opgestaan is ons Behoeder.
      
      
        Maget bidt // dat wij 't Wit
        Van den Hemel raecken,
        Alleluya // alleluya,
        En dat wij ons God genaecken.
