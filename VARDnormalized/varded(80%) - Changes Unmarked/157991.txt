
    
      
        Hoog en duer // ist bestuer // van het vuer
        Levitici thien // t'sien
        Tot figuer // moestet doen gheschien,
        Maar heel puer // is den sin van dien,
        Liefde is het ware wezen,
        Some Canticorum lezen,
        Nemet merck // zij is sterk // in haar werk,
        Ghelijck dood en hel // fel
        Houdt haar perck // tegen veel ghequel,
        Leeck of clerk // maakt haar niet rebel.
      
      
        Een gheleert // hoog gheeert // die temteert,
        Christum onder 't schijn // fijn,
        Als begheert // onderwesen t'zijn,
        Maar verkeert // t'hert is vol fenijn,
        Hij en was niet verslaghen,
        Ginck nochtans Christo vraghen,
        Wat ghebodt // t'hoochste lot // heeft van God
        T'welck is in de Wet // set,
        Deze zot, onghereghelt tret,
        Christus tot // hem antwoorde net.
      
      
        Niet allien // zes en thien // maar ik mien
        Ook in 't dertichst plaen // t'staen
        T'vijfde boek // deed' Christus vermaen,
        Zijn versoeck // die 't recht can verstaen,
        Lief te hebben God u Here,
        Hert en siele daar toe kere,
        Ook 'tghemoet // hij is goed // die 't zo doet,
        Dit ist grootst bevel // wel,
        T'ander zoet // dit ghelijcken sell,
        D'naesten moet // gij ook beminnen snel.
      
      
        Deze twee // gants gheree // daar ter stee,
        Van Christo belast // vast,
        Wel ter snee // worter aenghepast,
        Wet ende mee // der Profeten wast,
        Ditte gaat ons onderwijzen
        Dat die liefd' is hoog te prijzen
        Wie misdoet // uit vleys oft bloed // om tijtlijc goed,
        Oft om anders wat // schat,
        Is onvroet // boven Gode dat,
        T'sijn onspoet // heeft hij lief gehad.
      
      
        Raad nu voort // met accoort // soo't behoort
        Die does liefd' uit twist // mist,
        Die verloort // al daar hij op gist,
        Telcker oort // oft hij wel al wist,
        Tonghen ende Prophecyen,
        Niet zal hyder mee bedyen,
        Gelove vaet // wonder daedt // hoort oft laat
        Zonder liefd ist niet // ziet
        Elker maet // t'goet den armen biet,
        Oock't ghewaet // t'lichaem branden liet.
      
      
        In haar zoet // is ootmoet // vriendschap goed
        Zij toont niemand spijt // nijt,
        Schalckheyt doet // zij tot gheender tijd,
        Noch verwoet // opgheblasen crijt,
        Zij bedriecht in ghenen hoeken,
        Zij en gaat t'haer niet soecken,
        Niet verwacht // verdriets clacht // erch ghedacht,
        Noch dats' om tweedracht // lacht,
        Maar met kracht // die waarheid betracht
        Dach en nacht // tdoet haar wee of sacht.
      
      
        Claer bescheyt // hier verbeyt // als gheseyt
        Wordt in Moyses Wet // net,
        Alle beyt // Gods en s'naesten met
        Ook bereid // u sin hier toe set,
        Siedy s'broeders vee gaan dolen
        Exo.23,4. V wordt neerstelijck bevolen,
        Niet ontreckt // u suspeckt // maar perfeckt
        Tot u broeders deer // keer,
        Isset sect // ofte valse leer
        Niet begect // maar helpet doch weer.
      
      
        Toont nu klaer // met gebaer // hier en daar,
        Dat in u God bemint // vint
        Is niet waar // s'werelts liefd' ontrint
        In u maar // zijn ghebodt versint,
        Die hier s'werelts vriend wil wezen
        Zal Gods vyandt zijn in dezen,
        Wat ghetier // oft manier // dat men hier
        Stellen tot een toon // schoon
        Voor het vier // gheltet met een boon
        Hert en nier // doorzocht elk zijn loon.
      
      
        Is er nu // vrees en schu // dat gij u
        Voor het kruys versaecht // draecht,
        Al ist ru // en u seer plaecht,
        Zulks verdu // ende van u jaecht:
        Iacob schrijft wilt verblyden
        Als gij veel becoring lyden,
        Dit versint // wie hem vint // een Gods kint
        Zal voor gheen kastien // vlien,
        Troost ontrint // is er te allen tien,
        Wie verwint // zal veel goeds gheschien.
      
      
        In den Heer // stelt nu teer // u begheer
        Om der liefden werk // sterk
        Langs zo meer // te brenghen in't perck, Esa.58,8.
        Na zijn leer // tot stichting der kerck,
        Hij verbiet // overspelen,
        Doodtslaen ziet // ende stelen,
        Ghetuyghe vals // geen van als // dies getals
        Nimmer doch begheer // meer
        Myr of te als // zijnde bitter zeer
        Met regals // neemt al hier zijn keer.
      
      
        Sonderling // in dit ding // zo ik sing,
        V naasten wel ghesint // mint
        Gants ghering, oft gij 't zelver bint,
        Zonder dwing // als die eerst begint:
        Die liefde doet niet ten quaesten
        Ymmermeer zynen naasten;
        Maar ghehult // zij vervult, alle schuld
        Van der gantser Wet // net,
        Met ghedult // u toch hier na set
        En gij zult // dan zijn onbesmet.
      
      
        Zaligheid // is toeseyt // met bescheyt
        Niet die alleen t'woort // hoort
        Maar bereid // tottet werk accoort,
        Twellick leit // inder liefden oordt,
        Liefde is die hooftsomme
        Der Geboden Gods alomme,
        Hoort ghemien // t'kan niet schien // dan allien,
        Alst een rein hert // werd,
        Ende vlien // dat consciÃ«ntie smert,
        Ook bij dien // int gheloof volhert.
      
      
        Och wat raad // menich kwaad // van does maet,
        Hebben veer ghefaelt // dwaalt
        Onnutte praet // bij haar toe ghehaelt,
        Zonder baet // cierlijck af ghemaelt,
        Als leeraars die wet vertellen,
        Verstaen niet wat ze zeggen oft stellen,
        De Wet is goed // tot behoet // van 't ghemoet,
        Ghebruycktse iemand recht // slecht,
        Maar men moet // weten die oprecht
        Is zeer zoet // gheen Wet opghelecht.
      
      
        OP dat dit // elk lit // onbesmit,
        Doch in rechter maet // vaet,
        Neerstich bidt // den Heer om ghenaet,
        Hier in zit // ledich vroeg noch spaet,
        Laat u leven claer zo blijcken,
        Dat er elk op mach kijcken,
        Metter kracht // met ghedacht // houdt go wacht
        Op dies' Heren gheeft // vreest
        Dit gheslacht // een goed voorbeeld weest,
        Inder nacht // gaan die mensen meest.
      
      
        Neemt in danck // dezen sank // gaat niet manck,
        Al waar schoon die noot // groot,
        V verlanck // zij in elk aenstoot,
        Eer iet lanck // na een ander boot,
        Komt nu gij ghebenedijde,
        Die bokken aan de lincker sijde,
        Leert dit voort uit Gods woord // met accoort,
        V zelfs onverschoont // toont,
        Onghestoort // wordt gij dan beloont,
        Wie u hoort // wordt met u ghecroont.
