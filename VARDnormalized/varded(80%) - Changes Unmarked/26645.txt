
    
      
        Gij Christen schaer // eerbaer,
        Neemt uwen roep claer // wel waar,
        Om verblijen hier naar // aldaar
        Gheen scheiden en werd meer:
        Als rechte Israelijten,
        Volghende Christus leer,
        Laat ons beloften quijten,
        Dat wij vaten ter eer
        Oorbaer tot elken Keer
        Wezen van den Huysheer,
        Dat deur goed ghereucht // beweucht
        Den blinden met vreugd // tot deucht,
        En op dat gij meucht // vol jeucht
        Wassen in Christo zeer.
      
      
        Wast nu ter tijd // met vlijt
        Als ranken wijt // int crijt
        Op dat gij zijt // bevrijt
        Van Gods toorn vertsaecht,
        Die hem zal openbaren
        Zo Paulus ghewaeght,
        Op alle boze scharen
        Die na God niet en vraagt,
        T'sal noch worden beclaeght:
        Wie in hem gheen vrucht en draagt,
        Zal hij na Schrifts bedien // afsnien,
        Dus wilt wel in tien // toesien,
        End' de boosheid vlien // verbien
        Zal hij, die God behaagt.
      
      
        De boosheid kwaad // weerstaet
        Niet met den praet // maar daad
        Met goeden raad // na gaat
        Christum den Herder fijn,
        Die ons heeft ghenesen
        Met olye, en met wijn,
        End' veel liefde bewezen,
        Als onzen Medecijn,
        Hij droeg al ons ghepijn:
        Deur zijn woorden deuijn
        Ghenas na Schrifts vertel // zeer snel,
        Israel // ook wel
        Van der slanghen fel // t'ghequel
        Was doodelijck fenijn.
      
      
        T'fenijn zeer bloot // was groot,
        Dat der slanghen hoot // uytschoot,
        Maar deur Christus dood // minjoot,
        Is dat tot niet gebracht:
        Satan die heeft verloren
        Zijn gheweldighe macht,
        Daarom is hij vol toorn,
        Soeckende dach ende nacht:
        Laat ons houden go wacht,
        Het uytvercoren gheslacht
        Gaat hij met behent // ontrent,
        Als een Leeu bekent // die schent,
        Op dat hij verblent // afwent,
        Van God hooghe gheacht.
      
      
        Gaat onghestoort // doch voort
        Door die enghe poort // soo't behoort,
        En maakt gheen discoort // int woord,
        Als eerlijcke leen:
        En wilt niet teghenstrijden
        Ghelijck de Phariseen,
        Die den Here benijden,
        En verstootten zijn reen,
        Dat was meest dat zij deen:
        Want zij waren van beneen,
        En hebben den Heylant // vailjant,
        Aan een cruys ghespant // met schant,
        En dien hij sant // int land,
        Die hebben zij ook bestreen.
      
      
        Satan bevecht, Gods knecht,
        Die nu t'onrecht, aflecht,
        Zo ons Iob zegt // die slecht
        God vreesde en zijn ghebodt:
        Zijn wijf die hem wel kende,
        Die heeft hem ook bespot,
        En sprak in zijn ellende,
        Blijft gij noch vast bij God:
        Iob die antwoorde tot
        Haar: gij spreekt als Wijven zot:
        Want hebben wij dat goed, zeer vroet,
        Met ouervloet, men moet
        Ook teghenspoet, onsoet
        Ontvangen voor ons lot.
      
      
        Prinse vermaert, t'welvaert,
        Met goed regaert, bewaard,
        Want onzen aard, bezwaard
        De Ziele menichfout,
        In dit leven dangierich
        Wel vast op God betrout,
        Op dat de liefde vyerich
        Nemmermeer en verflout:
        En wordt nemmermeer stout,
        Al wordt gij hier benout
        Met veel boos, verknijs, afgrijs,
        Zoekt met auijs, zeer wijs,
        Int Paradijs, u spijs,
        Van dat leuende hout.
