
    
      
        ORpheus met zijn snaren // spel
        Dieren doet vergaren // wel
        Aender heyden // en groen weiden
        Mocht ik zo in vreuchts ghespan
        Vrolijcken zijn bij mijn Lieff dan.
      
      
        Eender heeft mij tot Lieffd' gebracht
        Die de wereld luttel acht
        Nochtans zal // ik // heel bevallick
        Dencken om does Maghet zoet
        Die ik van herten dienen moet.
      
      
        T'soetelijck wezen heeft gheraeckt
        Mij / die daar verdriet om smaakt
        Want does schone // maagd ydoone
        Heeft mij met tghesicht bestraelt
        Lieffde wt tdiepst vant hert ghehaelt.
      
      
        Lieffste ghedenckt den cleynen tijd
        Dat wij tsamen waren blijt
        Doen u ogen / na tbetooghen
        Zo doorstraelden t'jonghe hert
        Dat ik Lieff uwen dienaar werd.
      
      
        Edel jonckvrou die ik Min
        Kende gij mijn hert en sin
        Zo sout Minnen // gij met sinnen
        Mij / die u in deuchden acht
        Wenschende biet u goede nacht.
      
      
        Nu dan Princess vaart ook wel
        Maar veracht gheen slecht ghesel
        Lieffde reine // nu alleyne
        Mij doet u / o jonckvrou fijn
        Achten voor d'alderliefste Mijn.
