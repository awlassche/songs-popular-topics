
    
      
        AL die hebben tot jagen // zin
        Staat op en gaat in het veld,
        En hebt niet meer behagen // in
        Den slaap, die menig quelt
        Daar is te vaan, o Iagher dat u geld
        T’eeuwich welvaren // zoet
        T’eeuwich leven is ons voorgesteld
        Elk als Held // vlijt en geweld
        Hier toe openbaren // moet.
      
      
        DRIe dingen zijn bevonden // noch
        Bekwaam tot dezer jacht,
        Ten zijn netten noch honden // doch
        T’is t’gheloove vol kracht,
        Hope, en de Liefde boven al geacht
        Om wel te vangen // maar
        Niet hier // een dier // onder wilt geslacht,
        Maar een zacht // Lam vol eendracht
        Wel werd om verlangen naar.
      
      
        AAN dit zuiver geprezen // lam
        Al ons welvaren kleeft,
        Elk Iagher zonder wezen // gram
        Volgt hem en niet en sneeft,
        In zonden meer, maar schrickt daar vaan
        Den Heer weest onder // dan,
        Iae Iagher u hem gevangen geeft
        Die hem heeft // eeuwig hij leeft,
        Dus kleeft // hem bijzonder // aan.
      
      
        HOEFven wij in het ellendig // wout
        Iet beters t’onser noot?
        Och neen wij, maar die blendich // stout
        Van hem afwijken bloot,
        Haar lusten jaghend’, en niet achten groot
        T’lieflijck vermonden // schoon,
        Maar spelen hier met der zielen brood
        In den schoot // vangen den dood
        Want dat is der zonden // loon.
      
      
        NA t’beste moetmen speuren // dan,
        Meer als na een wild dier,
        Dewijl het ons gebeuren // kan
        Ontvlieden t’eeuwich vier,
        Der ghenaden tijd niet door licht bestier
        Willens verliezen // al,
        De dood fel oft t’eeuwich leven schier
        Van twen hier // om t’dangier
        Men t’beste verkiesen // zal.
      
      
        gelijken niet den rouwen // knecht,
        Esau moeten wij,
        Maar God altijd betrouwen // recht
        Als Iacob, dat als hij
        Ons deel der ghebenedijdinghe zij
        Maar zo ik proeve // snel
        Om te vangen dat Een nodig vrij
        Vind’ ik mij zeer zwak, waar bij
        Gods hulp ik behoefve // wel.
