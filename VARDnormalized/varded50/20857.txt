
    
      
        O Hemels groot Sieraad,
        Xaveri! Met deugd versierd in de hoogste graad:
        Ik wil uwe deugd // en uwe vreugd
        Verhalen hier met des harts geneught:
        'k Wil schrijven // 'k wil stijven // uwe eer // uwe leer,
        Staeghs meer en meer;
        Want ik mijn min // en al mijn zin
        Op U gesteld heb van het begin.
      
      
        Van uwe gehoorsaemheyd,
        Xaveri! kan worden nooit genoeg gezeid:
        Met ene I // ging vlijtig Gij
        De wereld rondom even blij:
        Gij eerden // Gij leerden // te zam // de faam,
        En Jesus Naam,
        Met grote vlijt // tot des vijands spijt,
        Gheen tegenspreken Gij en lijdt.
      
      
        Gij had zo suyv'ren hart,
        Xavieri! dat vuil gedacht U was een smart:
        Want als bij nacht // een vuil gedacht
        Den Duivel U had voor-gebracht,
        Van minne // van sinne // Gij bloed // hoe zoet
        Was deze vloed!
        Het speet U dat // uw des lichaams vat
        Ook in de slaap zijn wet vergat.
      
      
        Uwe ijver was zo groot,
        Xaveri! dat Gij niet hebt ontzien de dood:
        U drouf gesteen // u drouf geween
        Vermurwden harten hart als steen.
        Gij smeeckten // Gij preeckten // staegs aan: // Gods baan
        Deed' Gij het volk gaan:
        Het speet U zeer, dat God uwe Heer
        Vergeten worden, en zijn eer.
      
      
        Gij waart Mariaes vrind
        Xaveri! Zij achten U te zijn haar kind;
        In all' uwe rouw // was deze Vrouw
        Een troost aan U, ja u Behouw:
        In het lijden, in het strijden stond Zij // U bij,
        Gij raakten vrij.
        Des helschs geweld // heeft S' ne'er geveld;
        Gij wiert door Haar zo vromen Held.
      
      
        Gij sijt geweest gesteld,
        Xaveri! een Prins in het Indiaensche veld,
        Dat land geleerd // dat land bekeert
        Hebt Gij, en met Gods Wet vereerd.
        Wat lont // wat krone // wat lot // geeft God
        Van Sabaoch,
        Aan U zijn vrind // die Hem zo mindt,
        Dat Gij aan Hem die Landen wind.
      
      
        God heeft U zo verheugt,
        Xaveri! en ingestort zo grote vreugd,
        Dat Gij riept, Heer! // geeft mij niet meer,
        Van waar komt mij toch deze eer?
        Est Satis, est Satis ,, ay mij! // ik lij
        Gods slavernij!
        Ik ben niet waard // dat mij ervaert,
        Ik bidd' U, Heer! uwe vreugde spart.
      
      
        Is het zake dat Gij zijt,
        Xaveri! geweest zo op des aerd' verblijd:
        Wat zal dat zijn // een zoete pijn
        Te aanschouwen eeuwig Gods aanschijn!
        Vernomen // bekomen // die kans // Gods glans,
        Hebt Gij nu gants:
        Gij slaat om-hoogh // op God U oog,
        Wien ik altijd te dienen pog.
      
      
        Maar aangezien ik ben,
        Xaveri! een zondig mens, g'lijck ik beken,
        In ziel gewondt // met zware sond',
        Zo roept uwe hulp mijn droeve mond.
        Gestadig // genadig // U toont // mij lont,
        Die boven woont:
        Dat is Xaveri! ,, bidt dat ik hier
        Wel-levende ontga het vier.
