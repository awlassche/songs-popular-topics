
    
      
        IN kromme wegen, duister en zwaar,
        Vol dode vruchten, en werken onvruchtbaar,
        Daar namen wij ons ijdel wandelinge,
        Maar uit genade werden wij ghewaer
        Enen zoeten reuke, stem, ende licht klaar,
        Wt 'sHeeren Hof ende woninge,
        Dies wij geringe // merkten zaan,
        Dat wij de baan // misten in het gaan,
        Maar wij vonden daar staan // van dezen // Hof
        De deur ontdaan // end' hem moet wezen // lof
        Die ons daar heeft zo vriendelijk ontfaen.
      
      
        Wij vonden daar in des Hofs besluit
        Van alderhande welriekende kruid,
        Maar boven al // hoge verheven,
        Een roos in het dal // die door haar zoet virtuyt
        Syon des Heren uitverkoren Bruid
        Geeft troost en vreugd, welvaart en leven,
        En vast beneven // der beken kant,
        Wierden gheplant // zeer abundant,
        Groen spruytkens playsant // die als leden, goed
        Opwiessen in den band // van vreden // zoet,
        In denkt, en jeugd, zeer schoon en triumphant.
      
      
        Menig was vroylijck hier ter feest,
        Zo lang als in dit Cruyde-hofs foreest
        Den Zuiden wind waeyde zeer stille,
        Maar werden door den Noorden wind bevreesd,
        Want daar een bitter wortel is ghekeest
        Door enen geest // vol van gheschille,
        En vol moedwille // gingen daar voort,
        Vele verstoort // buiten de poort,
        Want dezen wind van Noord // doet verkonden // haar,
        Alle die aent woord // niet en houden // maar
        Haar laten omwaaien met fel discoord.
      
      
        Princelijcke Bruidegom, tot wien
        Zullen wij ons keren, oft henen vlien,
        Dan bij u in u Hof ons wooningh' maken,
        Gemerkt dat wij zo klaerlijcken zien,
        Die bij u niet volherden in dien,
        Waar zij in het einde toegheraken:
        O Heer ons waken // helpt met vlijt,
        In allen strijdt // zijt ons bevrijd,
        Dat niemand ons in zijd-wegen dwaligh // leidt,
        Maar wat profijt // is t'onser zaligheid,
        Geeft ons, want een is nodig altijd.
