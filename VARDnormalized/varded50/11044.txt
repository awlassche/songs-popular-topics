
    
      
        NU zijt // verblijd // tis tyt // te zingen
        Wij bringhen, goede maar
        t'Quaat mijt // dit slyt // u strijd // geringen
        t'Sal dwingen // d helsche schaar
        Gij hebt een middelaar
        (Is u wil goed) verkregen
        U moed // tot boed // dies zij genegen
        En doet // het goed // in dezen nieuwen Iaar.
      
      
        Want deugd // verheugt // met vreugd de zinnen
        Die minnen // zijn woord klaar
        O Jeugd // gij meugt // gheneught // hier winnen
        Van binnen // trachter naar
        Volgt Christus wet eenpaar
        Al in u hart geschreven
        Bestaat // zijn raad // nu te beleven
        En laat // het kwaad // in dezen nieuwen Iaar.
      
      
        Elk held // zich stelt // om geld // te erven
        t'Verwerven // valt hem zwaar
        t'Hart swelt // beknelt // oft tquelt // doort derven
        Wilt sterven // zulk misbaar
        Gods goeden zijn voorwaar
        Gemeen huiden en morgen
        Maar t'mijn // en dijn // dat doet u zorgen
        Schouwt fijn // t'fenijn // in dezen nieuwen Iaar.
      
      
        Prins acht // en tracht // met macht // na goedheid
        Gods zoetheid, proefdy daar
        Zijn kracht // versacht // t'ghedacht // met vroedheid
        Vermoedheyd //schuwt te gaar
        Christ helpt u wt gebaar
        Wildy niets niet verkiezen
        Dit kind // men vind // door zelfs verliezen
        Wiet'mint //verwint // in dezen nieuwen Iaar.
