
    
      
        Zegt mij schoon wtvercoren // lief //
        Twij doetstu overlast //
        Heb ik deur een bezworen // brief //
        V niet getoond zeer vast //
        Volmaakte trouwe //
        om u Jonkvrouwe //
        te zijn onderdaan //
        Gij doet mij al verloren //
        grief// vast zal ik blijven staan.
      
      
        V heb ik (in het beginnen // vroom)
        Betrout mijn herte heel:
        Regiert doch minder zinnen // toom,
        Naar dien gij zijt mijn deel:
        Zeedbaerste blomme,
        Keert u eens omme,
        Tot u dienstbaar knecht:
        Ontwaakt mij wt der minnen // droom,
        En houdt mij staande recht.
      
      
        Rein heb ik v beuonden // maagd,
        Vol eerbaar zeden // ziet,
        Om u mijn hert veel wonden // draagt,
        doch zonder reden // niet:
        Zo schonen vrouwe
        Te houden trouwe //
        Is een eerbaar zin:
        Nooit hert gij zo verbonden // saeght,
        En was van aanbegin.
      
      
        Troost-baer, en wtgelesen // zijt,
        Gij lief, en dies te meer
        Heb ik u ooit bewezen // vlijt,
        En noch doe even zeer:
        Zoudt ik u laten
        Die mij verwaten,
        Zult tot gender stond,
        O neen, ten is op dezen // tijd
        Lieuer ter wereld rond.
      
      
        Eerweerdich zij dij in den // graat
        Van uwe digniteit.
        Zulk achter zijn zo blinden // daad
        Dat ik u ben bereid,
        Te zijn gestadig:
        Gij die sucadigh
        Zijt, en wezen zult:
        Hierom, uwen beminden // laat
        Met geen droefheid vervult.
      
      
        Princesse, schoon eersame // bruyt,
        Oorlof, in mijn vertrek:
        Zijt doch (als een bekwame // cruyt,
        Oorboorlick mijn gebrek:
        Gij weet in allen,
        Het zoet gevallen,
        Dwelck mij is ydoon:
        Vriendinne uwen name // luid
        Tot inder goden troon.
