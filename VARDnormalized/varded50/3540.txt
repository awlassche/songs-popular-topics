
    
      
        Onmin dede mij dwalen // zeer
        Liefde komt mij halen // weer
        Zeid door reden // tgheen voorleden
        Is / uit uwe zinnen // stelt
        V niet in het loose Minnen// quelt.
      
      
        Menighe in zijn jongen // tijd
        Dikmaals onbedwongen // vrijt
        Is het niet drukkig // ongelukkig
        Want ook iemand voor of na
        Komt wel te vroeg oft beit te spa.
      
      
        Kan ik me courtoiseren // niet
        Dochters respecteren // iet
        Door wat zaken // kan aanraken
        Ik zo niet als menig // doet
        Maar altoos leven enig // moet.
      
      
        Heet reine Lieffd' niet vrij // versaemt
        Waarom maakt ze mij // beschaemt
        Want na tseggen // in het weerleggen
        Die in Lieffd' versmaad // fijn
        Somtijds meest beschamet // zijn.
      
      
        T'bedencken mij tot rede // keert
        En den tijd ook mede // leert
        Mits aenvaten // tboertich praten
        Bij zijn Lief / hem wachter // veur
        Settende schaamt aan d'achter // deur.
      
      
        Redene zeid / die ik verhief
        Ieder krijgt zijn wederlieff
        Die hem enen // wordt gegeven
        Daarom woud' ik wel eenpaar
        Dat ik bij haar allene waar.
      
      
        Venus Goddin ydoone // zoet
        Haastig met u Zone // spoed
        Zonder beiden // gaat bereiden
        Tbed dat ooit vol lusten // lach
        Daar ik met mijn Lief rusten // mag.
