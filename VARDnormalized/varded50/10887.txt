
    
      
        Mijn tijd // verslijt, ik lief geprezen
        Met vrezen, gants en gaar
        die lijt // subijt, wilt dienstig wezen
        Genezen // zijn gevaar, maakt dat hij werd gewaar
        V hulp tot hem genegen
        Door praat // gelaat // zijn hart versleghen
        Met raad // bystaet // in deze nieuwe iaer
      
      
        V heeft // beleeft, gediend Jonkvrouwe
        Getrouwe // des dienaar
        Hem geeft // beleeft, blyschap voor rouwe
        Door trouwe // vreugd eerbaar
        Maakt u nv eenpaar // zo zal zijn druk cesseren
        Met spoed // zeer zoet // al u begeren
        Hem doet // ootmoed // in deze nieuwe iaer.
      
      
        Dat hij onvrij // u troost moet derven
        Doet sterven // vreugde maar,
        Zeer blij // hoept hij, u hart te erven
        T’verwerven, zal voorwaar, doen slijten t’lijden zwaar
        Dat hem nu quelt gestadig
        Dus vrou // zijn rou, verdrijft genadig
        Die rou // dient trou // in deze nieuwe iaer.
      
      
        V vlijdt // zeer blijdt // Prinses verheven
        Beneven u dienaar
        Wien spijt // benijt // laat zijn verdreven
        Het sneven, en misbaar, twelck hen quelt voor als naar
        Wilt doch zijn pijn verzachten
        Dat hij // lief bij // u mag vernachten
        Heel vrij // en blij // in deze nieuwe iaer.
