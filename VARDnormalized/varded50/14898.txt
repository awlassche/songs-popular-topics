
    
      
        Onlangs ik savonts ginck
        Ik weet serteyn wel waar,
        Luistert hier naar // ik bid u kaar
        Ik sach te gaar // twee Meyskens daar
        Zij praten // wanlaten, van Vryers zoet,
        En bedreven zo vreugd,
        Ik dachte // en lachte
        Ja wel hoe zoet is ook, is ook de jeugd.
      
      
        Ach! Trijntjen zei de een
        Ik ben nu zo ontstelt,
        Want mij strackx quelt // een jonger Held
        het Zij u vertelt // want met geweld
        Wil trouwen // en houwen, mijn voor zijn Wijf,
        Tot dat de bleecke dood
        Zal scheyen // ons beyen
        Dus lij ik staech een zwaar, een zwaar aanstoot.
      
      
        Wel Marritjen mijn moer
        Ben gij daarom belaen,
        Zo laat hem staan, en henen gaan,
        Doet dit vermaan // of is het maar waan
        Wilt proeven // niet toeven // of hij oprecht
        Zijn Ambacht heel wel kan,
        Gij weet wel // secreetsel
        Bij mijn verholen blijven, blijven dan.
      
      
        Maar Trijntje lieve sus
        Speelnoot t’sy u gheclaecht,
        Ik heb gewaagd // ken ben gheen Maagd
        Want ziet ik dracht // mijn Moeder vraagt
        Wat schort-jou, quyl-beck, dat gij dus tiert,
        En ik die ben dan stom
        Mijn herte // vol smerte
        Want ziet mijn tijd die is, die is haast om.
      
      
        Mijn rekening is uit
        Eilaas! ik sterf van rouw
        Want Jan syn trouw // niet houden wouw,
        Foei hem ik spon // hoe garen sou
        Ik wrekken // en spreken // hem hier op straat,
        Dat hij mijn zo bedrieght,
        Met logen // bedrogen
        Heeft hij mijn strackx in slaap, in slaap gewiekt.
      
      
        Maar Marritjen mijn moer
        Gij bint voorwaar niet dood,
        Al gady groot // dat is gheen noot,
        ‘K lach ook eens bloot // en Keesjen Root
        Kwam lustig // en rustig // als op mijn bet
        Hij werden hem zo dat
        Een Zone // te loone
        Ik binnen negen Maanden, Maanden had.
      
      
        Hoort doch toe hoe ik voer
        Ik ginck besteden ‘tkint
        Dit wel versint // want ziet gij bint
        In de sak blind // want ik bevind
        t’Kints leven // maar even // twee Maanden was,
        De Heer die gaf die namt
        De Vrijer // de stryer
        Betaelden de onkosten gelijk, gelijk betaalt.
      
      
        Ach! Tryntje tis zo lang
        Ik meen geleden schier // acht maanden hier,
        Dat ik goe chier // met Kees Jaap vier
        Eens was te bier,
        Hij soende // al doende, o mijn wat vreugd
        Ik denker stadig om,
        Mijn Billen // die drillen
        Ik sterf eilaas! vraagt niet // vraagt niet waarom.
      
      
        Prinsen ik scheyde doen
        Ende zij gingen voort,
        Ik hebt gehoord // en was bekoort
        Doch ongestoord // noch rechtevoort
        Mijn dochte // zij zochte, en Vrijer fray
        Gelijk veel Meyskens doen,
        Ik late // die haete
        En wens mijn eigen lief, mijn lief een zoen.
