
    
      
        Waakt op, waakt op ‘het is meer dan tijd
        En laat de wereld varen  //  ja varen,
        Maakt u des bozen levens kwijt,
        En wordt uit God geboren  //  geboren.
      
      
        De lust des werelds moet vergaan,
        Heeft Ioannes geschreven  //  geschreven.
        Maar die den wille des Vaders doet,
        Die zal eeuwiglijk leven  //  ja leven.
      
      
        Geld ende goed  //  en daar toe moet,
        Dat moet men weinig achten  //  ja achten,
        En volgen den Here van herten goed,
        Dat eeuwig goed te betrachten  //  betrachten,
      
      
        Wie enen Toren bouwen fel,
        Die mag hem wel verzinnen  //  verzinnen,
        Overdenken eerst de kosten wel,
        Eer hij dat gaat beginnen  //  beginnen.
      
      
        Vlees ende bloed mag niet bestaan,
        Dat moet men daarom wagen  //  ja wagen
        Want het moet al zeer korts ondergaan,
        En zijn kruis moet men dragen  //  ja dragen.
      
      
        Het is den vlees een zware pijn,
        Het en kan ‘het niet wel lijden  //  ja lijden,
        Als ‘het van de wereld veracht zal zijn,
        Dan zal de Geest verblijden  //  verblijden.
      
      
        Nu geeft mij u genade Heer,
        En wilt mij wijsheid geven  //  ja geven,
        Dat ik mag wandelen na uwe leer,
        Haten mijn eigen leven  //  ja leven.
      
      
        De Stad is zeer lustig en fijn,
        Daar in ik hope te komen  //  te komen,
        Maar daar moet veel om gestreden zijn,
        Dat heb ik wel vernomen  //  vernomen.
      
      
        Veel Vijanden aan elken kant,
        Zijn op des enge straten  //  ja straten,
        O Heer doet mij nu onderstand,
        En wilt mij niet verlaten  //  verlaten.
      
      
        Vleeschelijck ghesint zijn  //  is de dood,
        Zo betuigt Paulus lere  //  ja lere,
        Geestelijk ghesint zijn is ‘het leven bloot,
        En vrede in den Here  //  ja Here.
      
      
        Daarom heb ik mij korts bedacht,
        De waarheid na te jagen  //  te jagen,
        Niet meer te wandelen in de nacht,
        God alleen te behagen  //  behagen.
      
      
        Maar het staat niet in minder magt,
        Doen kan ik niet allene  //  allene,
        Daarom, O Heer neemt minder acht,
        Maakte mij dat herte reene  //  ja reene.
      
      
        Ik zal ‘het wagen door u ghenae,
        Op u beloften schone  //  ja schone,
        Op u beloften schone  //  ja schone,
        ‘het Lichaam mogen zij doden  //  ja doden.
      
      
        Dit Liedeken is uyt-ghericht,
        Tot den prijse des Heren  //  des Heren,
        Is ‘het dat daar iemand door werd gesticht,
        Dat is wel mijn begeren  //  begeren.
