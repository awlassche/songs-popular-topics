
    
      
        MIjn Lief devijn // is mijn,
        En ik ben zijn // na dijn
        Lieflijk aanschijn // met pijn
        Verlangt mij, liefste grein:
        Gelijk een hert bevonden
        Dorstig na die fontein
        Gij zijt die mij van zonden,
        Wt ghenaden alleyn,
        Schoon hebt gewassen rein,
        Ende gemaakt ghemeyn,
        Al door u bloed // u goed,
        V loven vroed // ik moet,
        Dat gij mij doet // zo zoet
        Vloeien u gracy plein.
      
      
        O Minnaar groot // minjoot,
        Zoet Hemels brood // ter noot,
        Die van der dood // mij bloot
        Puur hebt genezen al,
        O lief, heilig, almachtig,
        Gij zijt die komen zal,
        Wilt mij wezen gedachtig
        Hier in dit Aardse dal,
        Leidt mij op den weg smal,
        Bevrijd hier u Schaepstal,
        Vwen Wijngaard // bewaard,
        V Bruid vermaard // verklaart,
        Na uwen aard // ghevaert,
        Beschermt voor den val.
      
      
        Biedt mij u hand // vailliant,
        Brengt mij in het land // playsant,
        Doet mij bijstand // lief, want
        Op u ik mij verliet,
        Wt mijn eigen vermogen
        Ben ik gelijk een riet,
        Ik zoude zijn bedrogen,
        Waart dat ik dachte, ziet,
        Ik ben al wat of iet,
        Want ik en ben doch niet
        Dan een broos vat // zaan mat,
        Om gaan den pad // ter stad,
        Maar deugh' ik wat // mij dat
        Door uwe kracht gheschiet.
      
      
        Mijn toost gij zijt // altijd,
        Want mijnen strijdt, in het krijt,
        Zal in jolijt // subijt
        Veranderen all' gaar:
        Maar waarder gheen verrijzen,
        Gelijk als er is klaar,
        Mij mochte wel afgrijzen
        Van mijn ellende, maar
        O getrouwe Minnaar,
        Ik weet wel openbaar,
        V armen staan // ontdaan,
        Om mij t'ontfaen, zeer zaan,
        Dus wil ik gaan // u baan,
        V last en is niet zwaar.
      
      
        Tot u ik keer // o Heer,
        Krank zijnde zeer // en teer,
        Geeft dat ik meer // u leer,
        Volghe zonder afwijck:
        Want als ik ga bedincken
        Die krone magnifijck,
        Daar ik mede zal blinken,
        Die klaar Zonne gelijk,
        Al in ws Vader rijk,
        Dan acht ik het hier al slijk,
        Vleeschs, lust, geweld // en geld,
        Dat mij voor stelt // in het veld,
        Satan den held // want 'tsmelt
        Al in een ogenblik.
      
      
        O goedertier // Princier,
        Gij die met vier // zult schier,
        Tot haar dangier // alhier
        Straffen die boze stranck:
        En leggen u vijanden
        Tot uwer voeten bank,
        Daar knerssinghe der tanden
        Werd huylingh ende stank,
        O lief, boet mijn verlanck,
        Wilt mij behouden vranck
        Van het onbekend // torment,
        V exellent // en jent
        Aanschijn present // mij went,
        In vreugd zonder verganck.
