        Met fantasie // Waar deur mijn vreugde veld, en smelt,
        tot allen tien
      
      
        Absency valt mij te maal // regaal,
        Lief wtghelesen // Toont jonst medicinaal, mijn quael,
        Wilt doch genezen.
      
      
        Rasende schier van rou // Kersou,
        Mistroostig van zinnen // Loop ik om v o vrou // ghetrou,
        Laat mij troost winnen.
      
      
        In mijn verdriet, bijstand // te hand,
        Doet mij verwerven // Of door der minnen brand // en band
        Zo moet ik sterven.
      
      
        Aanziet v knechts ellent // torment,
        Princesse schone, En laat hem zijn lief jent // in het ent
        Krijgen te loone.
