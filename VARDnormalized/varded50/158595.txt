
    
      
        GIJ schone Roos // die onder het lijden
        Hier zijt altoos, wilt u verblijden
        Dat u is opgeleid // tot zaligheid:
        Dat gij niet en zoudt // door lust menigfout;
        Dat gij niet en zoudt, verliezen
        V kleur, door des  werelds zot verkiesen.
      
      
        V kleur is schoon, door het verdrukken
        Van het uytroon // ende uitplukken:
        Van al u lusten kwaad, en overdaad;
        Van des werelds hoogheid // en haar ijdelheid;
        Van des werelds hoogheid, en pralen,
        Die in de afgrond moet neerdalen.
      
      
        Gij Bruid Christi // laat vrij verachten,
        V naam, zijt blij // in u gedachten,
        Lijt stil het steken wreed // die u doen leed:
        Toont u zachte moet, en diepe ootmoed
        Toont u zachte moet, gestadig,
        Vergeld het kwaad met goed, weldadig.
      
      
        Wijkt hier niet uit // wilt liever sterven
        Dan lijdens kruid // te doen verderven:
        Want gij moet dragen het kruis // zonder abuis,
        Wilt gij dat niet doen // maar tot wreken spoen;
        Wilt gij dat niet doen // wilt horen:
        Zo is u arrebeydt verloren.
      
      
        V Bruidegom // heeft veel geleden,
        En heeft daarom // voor haar gebeden,
        Die hem deden verdriet // hij wreeckte niet:
        O exempel schoon // van Gods enig Soon:
        O exempel schoon // te aanschouwen,
        Volgt hem na met een vast betrouwen.
      
      
        Met hem zult gij // eeuwig be-erven
        Zo gij hoort mij // met hem wilt sterven
        Dan zult gij in zijn troon ontfaen de Kroon,
        En aanschouwen zoet // het lieflijk wezen goed
        En aanschouwen zoet // verheven,
        Gods aanschijn in dat eeuwig leven.
      
      
        In des Hemels hof // zult gij vermeyden
        Daar men Gods lof // lieflijk verbreiden
        Met nieuwe tongen blij // een melody,
        Zoet zonder verdriet // want daar ghebreeckt niet:
        Zoet zonder verdriet, zal wezen
        het Lam Gods met zijne Bruyt geprezen.
      
      
        O Bruid Christi // zet u betrouwen
        In u Heer vrij // wilt doch aanschouwen
        Wat hij u schenken zal, na het lijden al:
        Hebt dan geen verdriet, in u lijden ziet
        Hebt dan geen verdriet, lijdt stille:
        Vraagt met geduld uws Heren wille.
