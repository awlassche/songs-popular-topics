
    
      
        O Christen leden,
        Verstroyt breed en wijd,
        Gij die in vreden
        Van God geroepen zijt,
        Al is het // dat twist // oprijst // bewijst,
        Den Christen aard altijd.
      
      
        Die u wil spreken
        Oft iet aanbrengen wilt
        Van die gebreken
        Zijns naasten, dezen stilt,
        Zegt maet recht gaat, spreekt dien, misschien
        Veel in de zake schild.
      
      
        Handelt bij orden,
        Die u Schrifture leert
        Alzo mag worden
        Een dinck te recht gekeerd,

        Die zulks bedrijven
        Zijn uit God niet gebaard,
        Waar nijd // of strijdt // is, daar // is zwaar
        De boosheid onghespaert.
      
      
        Want dien geen rustig
        Die de Christenen leidt,
        En is niet lustig
        Tot haat, oft toornigheid,
        Maar heeft // en geeft, Al zeer // veel meer
        Ghenade, zo Schrift zeid.
      
      
        Hierom wilt merken
        Op die hier achten klein,
        Discoord te werken
        Neffens de lere rein,
        Afsnijdt // en mijdt, Zulk zuur // en stuur
        Onzuiver deeg ghemeyn.
      
      
        Wilt vyerigh passen
        Op Gods geboden zoet,
        Zo zal dan wassen,
        Gelijk een watervloed,
        Den vre // hier me // Zal zijn // ook dijn
        Gherechticheydt, vol spoed.
      
      
        Oorlof eendrachtig,
        Christen gezinde wel,
        God laat u krachtig
        Sathan verwinnen snel,
        Op dat // de stad, Beclijft // en blijft,
        Die lijdt veel verstoots fel.
