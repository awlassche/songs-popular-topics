
    
      
        KOR agieuse // violette //
        Reynste fleur // Op aerderijck,
        Amoureuse // schoon Jenette //
        In den geur // aromatijck,
        O zoetheid zonder gelijk //
        Zulk een jeugd //
        Gij zijt //
        Die vermeugt // Altijd //
        Smert en leed
        te scheu- ren //
        Met vreugd //
        Jolijt //
        Verheugt //
        Verblijd //
        Mijn langmoedig treuren.
      
      
        Nobel herte // borght van trooste,
        Acht doch cleen // twistig erreur:
        Doen mij smerte // d'alderbooste,
        Gij alleen // tooght mij faveur,
        Vwen lof, werd met honneur,
        Honderd fout
        Verklaart,
        En als goud
        Bewaard,
        Gants uwen persoone:
        Benout,
        Vervaard,
        Verflout,
        Bezwaard,
        Ik mij t'uwaert thoone.
      
      
        Enig wezen // hier ter wereld,
        Wel gespraect // en niet te min
        Hoog geprezen // rijk bepeerelt,
        Gans volmaakt // in mijnen zin:
        Wtvercoren vant begin,
        Alle mijn
        Verdriet,
        En gepijn
        Is niet,
        Troost gij mij behendig,
        Macht zijn,
        Mij biet,
        T'aenschijn,
        En ziet
        Mijnen staat ellendig.
      
      
        Lieuer steruen // naar sentency,
        (Dit zal elk // houden gewis)
        Dan te deruen // u presenty,
        Zonder welk // geen leun is:
        Welgeboren maget fris,
        Hoe krul
        Het zij,
        Enen wel,
        Zult gij
        Lief! zijn mijn egade:
        Niet fel
        Tooht dij,
        Omhel-
        zen mij
        Laat uwe genade.
      
      
        In het beminnen // ik hert-grondig
        Roupe voort // om hulpe groot,
        Want mijn zinnen // vast, en bondig,
        Zijn akkoord // tot in de dood,
        V te laten t'geender noot:
        O dierbaar
        
          Parceel
        
        
          Wederpaar
        
        En deel
        Tot mijn treurig klagen,
        Coemt naar,
        Geheel:
        Dit zwaar
        
          Goreel
        
        Helpt mij lijdzaam dragen.
      
      
        Aldervroomste // schoon Princesse,
        Niet onreins // en gaat u bij:
        Mijn voornoemste // forteresse,
        Van vileyns, en dorpers vrij
        Blijft nu voort: hier mede wij
        Scheiden wt,
        Dijn hof,
        Mijn virtuyt
        Hier of,
        Schijnt treurig verdrogen:
        Schoon bruid,
        Oorlof,
        Mij ruid
        En grof
        Neemt in u gedogen.
