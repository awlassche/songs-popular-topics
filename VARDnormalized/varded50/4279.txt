
    
      
        PRoperste van persoone // schone //
        Die ooit geest versinde,
        Weerdiger ik geen krone // thoone //
        Dan u wel beminde,
        Nimmer geschiet // niet
        Buiten u gebiet // iet //
        T'sy vreugd of verdriet.
      
      
        Enig zijt mijn troostbare // care,
        In mijn lijden dodig:
        Dat ik geen martelare // ware,
        Troost waar mij onnodig:
        Gij (die t'axcident // sent,
        En mij deligent // kent)
        Sterkt dijn patient.
      
      
        Troosteloos! met verzuchten // vluchten
        Is een droeuich strijden:
        Moet ik ook voor des vruchten // duchten,
        T'wert een deerlijk lijden,
        Zo gij mij wt haat // slaat,
        Zonder een misdaad // kwaad,
        Sterf ik desolaat.
      
      
        Redelijk tooght u zeden // heden,
        O mijn amoureuse:
        Alle mijn druk beneden // treden
        Wilt, compassieuse:
        In den swaersten noot // bloot
        Mij de wreede dood // stoot,
        Faelt dijn hulpe groot.
      
      
        Verre ghepeynst doet qualen // halen,
        Met veel ongelucken:
        Veneris laat haar stralen // dalen,
        Mij tot een verdrukken:
        Nu mij dezen last // wast,
        Werd' ik drukkig gast // vast
        Fortsich aengetast.
      
      
        Zonder u, moet mijn treuren // deuren,
        Aangenaam matresse,
        Wilt doch eens mijn doleuren // scheuren,
        O mijn schoon Princesse:
        T'ooght mij u aanschijn // fijn,
        Zo zal al t'gepijn // mijn
        Haast verandert zijn.
