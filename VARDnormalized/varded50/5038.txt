
    
      
        Lief uitverkoren // mijns herten vreugd,
        Wt God geboren // vol van gheneucht,
        ‘het Inwendig wezen // met uwen douw,
        Kond gij genezen // van druk en rouw,
        Gij laevet mijn // met Melk en Wijn.
        Gheen Medecijn // men beter wensen sou.
      
      
        Ben ik armoedig // van tijd’lijck goed,
        Zeer overvloedig // mijn Ziel gij voedt:
        Moet ik hier derven // des vleys gemak,
        Uwe Conserven // zijn mijn gerak:
        Word ik versmaad // door nijd, en haat,
        Gij geeft mij raad // hoe ik zal dragen ‘het pak.
      
      
        V wijsheydts Reden // mij onderwijst,
        Door de gebeden // mijn Ziel gij spijst:
        Gij komt mij leren // tot aller stond,
        ‘k Hebb’ maar te keren // in ‘s herten grond,
        Want van ‘het begin // waart gij daar in:
        Gheen soeter Minn’ // ter wereld ooit men vond.
      
      
        ‘het Is waar, ‘s vleys lusten // vertonen schoon,
        Maar met onrusten // komt haren loon:
        Des Geestes minne // blijft altijd staan,
        Iae met ghewinne // zal zij op gaan,
        Gelijk een vier // zo brandtse schier,
        Wanneer ze hier // niet en werd uytghedaen.
      
      
        ‘het Is niet te spreken // hoe dat een hert
        Hier wordt ontsteken // als ‘het niet en werd
        ‘het Voedsel ontogen // door valsen schijn,
        En dat de ogen // gheen strikken zijn,
        Als men de wacht // houdt voor ‘het gedacht,
        Den Geest voelt kracht // die anders leyd in pijn.
      
      
        Prins toont u krachten // in ons gemoed,
        Dat de gedachten // het rechte goed
        Mogen aanhangen // en vinden rust,
        Voor het ontvangen // van d’aerdtsche rust:
        Blijft gij ons bij // en wij bij dij,
        Door gheen partij // en werd ons Liefd geblust.
