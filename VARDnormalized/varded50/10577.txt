
    
      
        Mijn tijd // verslijt // ik lief geprezen
        Met vrezen // gants en gaar
        Die lijt // subijt // wilt dienstig wezen
        Genezen // zijn gheuaer
        Maakt dat hij werd ghewaer,
        V hulp tot hem genegen
        Door praat // gelaat // zijn hart versleghen
        Met raad // bystaet // in deze nieuwe Jaar.
      
      
        V heeft // beleeft // gediend jonkvrouwe
        Getrouwe // des dienaar
        Hem geeft // hij beeft // blyschap voor rouwe
        Voor trouwe // vreugd eerbaar
        Maakt v en hem eenpaar
        Zo zal zijn druk cesseren,
        Met spoed // lief doet // al v begeren
        Hem zoet // ontmoet // in deze nieuwe Jaar.
      
      
        Dat hij onvrij // v troost moet deruen
        Doet steruen // vreugde maar,
        Zeer blij // hoopt hij // v hart te eruen
        T'verweruen // zal voorwaar,
        Doen slijten t'lyden zwaar
        Dat hem nu quelt gestadig
        Dus vrou // zijn rou // verdrijft genadig
        Die ion // dient trou // In deze nieuwe iaer.
      
      
        vlijdt // zeer blijdt // Prinses verheuen
        Benenen v dienaar
        Wient spijt // benijt // laat zijn verdreuen,
        Het sneuen // en misbaar
        T'welck hem quelt voor als naar
        Wilt doch zijn pijn verzachten
        Dat hij // lief bij // v mag vernachten
        Heel vrij // en blij // in deze nieuwe Jaar.
