
    
      
        TObias tot sterven genegen,
        Riep zijnen soon, en sprak daar tegen,
        Mijn kind // bemint,
        Hoort met accoorden // alle mijn woorden.
      
      
        Als God mijn ziel heeft weggenomen,
        Mijn lichaam in een graf laat komen,
        En wilt // ook milt,
        V Moeder dragen // jonst al u dagen.
      
      
        Houdt haar in en werden end' in eren,
        Denkt opt perikel en 'tverseeren,
        Hoe zij // droeg, dij,
        Met grote smerte // onder haar herte.
      
      
        End' als zij zal gestorven wezen:
        Begraeftse nevens mij, bij dezen,
        Altijd // met vlijt
        En neerstigh pogen // hebt God voor ogen
      
      
        Draagt in u herte God almachtig,
        Zo dat gij hem altijd gedachtig
        Zijn meugt // van jeugd,
        Tot 'slevens enden // t'hemwaerts u wenden.
      
      
        Ziet dat gij niet t'eenighen stonde
        V en bewillight in gheen zonde
        T'ghebodt // van God
        En wilt versteken // noch geenszins breken.
      
      
        Van uwen goede helpt den armen,
        Aanziet hen-lieden met ontfarmen,
        Bij dien // aanzien,
        Zal u gestade // God met genade.
      
      
        Helpt neerstigh die behoeftig leven,
        Hebt gij vele // wilt vele geven,
        En doet // ook goed
        Met herten reinig // al hebt gij weinig.
      
      
        Geeft trouwelijk al zonder sparen,
        Zo zult gij rechten loon vergaren,
        Ter nood // die dood
        En zonden lossen // de alemossen.
      
      
        De Aelmossen vol charitate,
        Is in der noot een grote bate,
        En troost // altoost,
        Zeer groot beseven // voor God verheven.
      
      
        Mijne zone wilt u ook vermijden,
        Van hoererij, laat die bezijden,
        Hebt een // Wijf reen,
        Wilt daar beneven // gheen meer aankleven.
      
      
        De hooveerdy en geeft gheen stede
        In u herte, noch woorden mede,
        Van in // 'tbegin,
        Wracht zij 'tbederven // daar wij om sterven.
      
      
        End' alle die om u arbeiden,
        Terstond wilt haren loon bereiden,
        En houdt // niet stout,
        Van gheen personen // verdiende lonen.
      
      
        En doet ook geen mensen geschieden,
        Dat gij nood' had van ander lieden,
        Ter nood // u brood,
        Deelt te allen tijden // die honger lijden.
      
      
        Wilt om den naakten te bedekken,
        Van uwe klederen aantrekken,
        Maar meest // diegheest-
        Lijk zijn begraven // deelt uwe gaven.
      
      
        Met den Zondaren vol vermeten
        Zo en wilt ook drinken noch eten,
        Maar gaat // om raad,
        Den Wijzen vragen // tot allen dagen.
      
      
        Dankt altijd God, en bidt eendrachtig,
        Dat hij u wil regieren krachtig
        Dat fijn // al dijn
        Voornemen blijcke // zijn woord gelijke.
      
      
        En zorgt niet, mijn Zone verheven,
        Al leiden wij een arem leven,
        Voorwaar // hier naar
        Wij hebben zullen // van als te vullen.
      
      
        Indien dat wij tot allen tijden
        God vrezen en de zonde mijden,
        En goed // doen vroed,
        Na recht wij zouden // dit wel onthouden.
