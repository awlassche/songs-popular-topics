
    
      
        EEn waarde vrou // doet mij veel vreugd bedryuen,
        Al heeft zij rou // somtijds dat kan niet schaen
        Op eer en trou // min ik die fleur der wyuen
        Als ik ze aenschou, zo moet ik zingen gaan,
        Want zij mij non // beuryt voor alle vaar
        Op dat zij mag gedenken // mijn //
        Zal ik mijn herteken schenken // fijn
        Mijn lief eerbaar // tot haren nieuwen jaar.
      
      
        Al was geresen // tot in den dood vol vrezen,
        Mijn jonstich hart // ja in den gront versmoord,
        Zij kant genezen // als zij doctoor wil wezen
        Van alle smart // met een wellustig woord
        Dus wil ik voort // verlaten droefheid zwaar
        Wil zij slechts lieffelijck winken // neer
        En trou gewillig zinken // weer
        Mij voor een aar // tot mijnen nieuwen jaar.
      
      
        Venus kind // wilt haar in het harde steken
        Dat zij bemint // een weinig tmijnder baat
        Op dat verwint // mijn veel ootmoedig smeken
        Dat nu noch in het // alderquaetste blaetgen staat
        Ik roep om raad // alle die goden te gaar
        Dat zij mij helpen minnen // toch,
        Op dat ik mag gewinnen // noch,
        Een blijde maar // voor te gehele nieuwe jaar.
      
      
        Grote geneucht // met danssen, en springen,
        Zal ik volbringen // om mijn Princesse goed,
        Of zij deur vreugd // oft deur wellustig zingen
        Eens wou ghehinghen // Dat ik die vruchten zoet,
        Plocke tot boet // van ons liefde eenpaar
        En ik ze mocht verwarmen // lank
        En rusten in haar armen // blank,
        Dicht neuens haar // vant ene jaar in het aar.
