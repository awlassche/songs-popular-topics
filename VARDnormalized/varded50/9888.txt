
    
      
        Ontsleghen vrij // zijt gij,
        Door die roo Zee getreden,
        Al Sinai // voorbij,
        Tot dat gezicht des vreden.
      
      
        T'vredich gezicht // vol licht
        En klaarheid, boven maten,
        Nooit stad zo dicht // gesticht,
        'tBlinckt al mueren en straten.
      
      
        Die straten fijn, goud zijn,
        En midden daar op gater,
        Als Christalijn // vol schijn,
        Zo rein levende water.
      
      
        Tot 'sLevens rein // Fontein
        Gij dorstighe wilt draven,
        Komt vrij in het plein // hoe klein,
        Om niet meugt gij u laven.
      
      
        Komt, ziet, om niet // gheschiet
        Elk Melk, en Wijn bekwame,
        Daar vliet verdriet // voort Lied,
        Ter eren 'tLam eersame.
      
      
        Ere zij tLam // dat nam
        'sWereldts misdaden geladen,
        Op 'sCruycen stam // en kwam
        Als brood, om elk verzaden.
      
      
        Dit Hemels brood // minjoot
        De wereld 'tleven kan geven,
        Nooit troost zo groot // de dood
        En vloek heeft hij verdreven.
      
      
        Prinse, bewaard // en spart,
        V Bruid, geeft haar victory,
        Die zeer bezwaard // nu baard,
        Brengt haar hier naar in glory.
