
    
      
        PArst uit het gezicht een droef getraan,
        Laat big'lend' langs des kakken gaan
        Bracke vloed // die ik moe
        Op de aarde gieten
        Met naar gezucht, helaas om een,
        Wiens harde hert blijft als een steen,
        En niet acht // na de klacht
        Van mijn verdrieten,
        het Gheen ik staegh // alle dag
        Doe voorwaar // of ik haar // noch hier naar
        Hier door eens mocht genieten.
      
      
        Doch het schijnt mijn smeeckingh zij begeckt,
        Of dat haar borst een rots verstreckt:
        Want wanneer // ik begeer
        Met zoete woorden
        Haar aan te spreken, stuurs en straf
        Draeyt zij haar wezen van mij af,
        En gaat heen // of mijn reen
        Zij niet en hoorden,
        Ja het schijnt // dat zij vijnt
        Hier in rust // en haar lust // werd gheblust
        Met dus mijn ziel te Moorden.
      
      
        Wel Amarille zegt mij doch
        Wat hebt gij voorgenomen noch
        Voor mijn trouw // waarde vrouw
        Mijn te doen erven?
        Wat is het gheen, zegt Harderin,
        Dat gij met mijn hebt in u zin?
        Sleghts gebiedt // daar is niet,
        Lust u mijn sterven?
        Of begeer // gij noch meer,
        'k Ben te vreen // wilt mij leen // vrij van een,
        Van lid tot lid doen kerven.
      
      
        Is het dat u wraak dan meerder dorst:
        Zo komt en opent vrij des borst,
        Wt zijn stee // rockt vrij mee
        Mijn trouwe herte,
        Klooft het aan twen, en ziet Goddin
        V eigen beeltenis daar in,
        het Gheen gewis // d'oorsaeck is
        Mijns vreughds en smerte,
        Gij zult zien // dat er gien
        Minnaar trou // is Me-vrou // of ik sou
        In liefd' hem durven terten.
      
      
        Want wat gij wilt, ik ben te vreen,
        Ik vrees gheen straf, hoe groot, o neen:
        'k Ben gerust // als gheblust
        Werd u begeren,
        Want Lief, mijn leven en mijn dood
        Bestaat in u vermogen groot,
        Wat gij doet // het is al goed,
        het Kan mijn niet deren,
        het Gheen u haeghd // waarde Maagd
        Doet dat vrij // laas aan mij // 'k zal mij blij
        Na u behagen keren.
