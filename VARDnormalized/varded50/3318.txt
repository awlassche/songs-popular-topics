
    
      
        Broeders al geprezen // wilt met u verkoen,
        Bijstand doen // den armen vray dat aardse goed,
        Zo zal Christus, wezen // die u zal verghoen
        (Wilt haar voen) in u ellent en tegenspoed,
        En bevrijden altijd van 'tvyanden gebroed,
        Zo sterkt dan Gods zegen (door loon die elk behaagt // en raakt)
        Al uwe weiden // en 'tmilt delen niet vertraagt.
      
      
        Wel dien die den armen // ontvangen met vlijt,
        En verblijd // mildelijk deel uit s' hertsen gront,
        Zo zal u ontfermen God ghebenedijt,
        Dus doch zijt // milt, Christen broeders te allen stond,
        V zonden al hij u vergeeft, zijt dit oorkont,
        het zal God niet vergeten // mer van hem zijn gezind // bemint
        Mocht dit te recht weten // als, een moeder haar kind
      
      
        Wilt den armen geven // van u aardse schad
        Want zo wat // gij doet 'tgheschiet in Christi naam:
        Onzen Heer verheven // geeft u vol u vadt,
        In Dorp, Stad, deelt dan mildig zonder blaam,
        veriaecht gierigheid zo verkrijgt gij d'eeuwighe faam,
        SSoo zal u licht vene / eeuwig opgaan // en staan
        Als d'Hemelsch stene // want gij doet Gods vermaan.
      
      
        Gods troost overvloedig // zal belonen ient,
        Zonder ent // onze werken vroeg en spade,
        Ent wilt zeer goedig, den armen ontrent
        Zijn u went // tot hun, betrout Gods genade,
        Op dat Christus ons van 'tHelsche Serpent ontlade,
        Zo blijven veel zonden // daar mede wij zijn bevlekt // bedekt,
        Dies 'tghemeen stonden // 'thert vanden armen trekt.
      
      
        Nu oorlof al gelijk // Edel Princieren
        Regieren // doch u leven wilt te recht,
        Sneeft niet doort aardse rijk // zijt goedertieren,
        Wilt vieren // Christum, haat giericheyts gevecht,
        Sulcx werd VVt liefd' bestaan, u misdaden hij slecht,
        Tot God neemt uwen keer // en bemint Christum rein, alleyn
        Geeft milt den armen teer // dus volgt hem groot en klein.
