
    
      
        Gij Bataviers wilt nemen acht,
        Vuyt wiens geslachte // gij zijt gebracht,
        BATTVS alzo was hij geheten,
        SALANDERS broeder wel bedacht,
        Den welken placht // te wonen zacht,
        Bouen Hongarien, te weten,
        In twist en onvreden gezeten,
        Dies gingen des Broeders vermeden,
        Te zoeken t'eylant vanden Rijn,
        Dits hoort mijn // d'afcomts fijn // en moet zijn Holnederlant,
        Dus bruyct verstand // zonder verheffen,
        Ziet waar gij bent // accordeert jent // na t'slants naam effen.
      
      
        Menich oude Roomsche Cronijck,
        Zegt autentijck // dat uit duytsch-rijck,
        Neerlanders waren d'vroomste knechten,
        Snel, kloek, wakker, lanckmoech gelijk,
        Deen geen bezwijk // daar van sach blijk,
        Keizer OTTO GALBA,, in het vechten,
        Die zij dienden als een oprechten,
        Waren bemint strijdt me te slechten,
        Bij Keizers, Koningen altijd,
        Weest verblijd // doet u vlijt // die noch zijt // Bataviers vroom,
        Looflick in room // wilt trimpheren,
        Maar u verneert // zo accordeert // gij na t'slants leren,
      
      
        Des werelds vreugd is na verdriet,
        Diet wel in ziet // t'sy hoed gheschiet,
        Hoog geacht zal God verstoren.
        Doet wel, peynst om nederheid iet,
        Zoekt geen ghebiet // maar als knecht vliet,
        Denkt ik ben Batavier geboren,
        Als verdreuen // maar niet verloren,
        Die ooit de versmaadheid verkoren,
        Wonachtigh nederig // ende laech,
        Gods behaag // dt ik jak // en na vraag, zonder pracht al,
        Zulk een die zal // na zijn met vreden,
        Die verneert snel // doet gods wil wel // en na t'slants zeden.
      
      
        Leeft als een Geestlick Batavier,
        Volgt de manier // van CHRISTO fier,
        Zo overtreft gij d'ou Romeinen,
        Die in hem zelfs glorieert hier,
        Is verdwaald schier // door zulk bestier,
        Van CHTRISTO t'hooft der Capiteynen,
        Want God sticht zijn kerk op den kleinen,
        Roept den nedrigen, en den reinen,
        Van herten, ghebroocken van geest,
        Hij geneest minst en meest // zomen leeft // door t'geloof heel,
        Werd bekwaam el // te strijen goe strijden,
        Die hem verneert // doet zo God leert // na schrifts belijden.
      
      
        Een recht Batavische ghenoot,
        Heeft een strijd groot // te strijden bloot,
        Tegen die den mens haast beklemmen,
        t'Welck zijn syn eigen lusten snood,
        Sathan is dood // dies des in sloot,
        Die dikmaals mits t'vleysch bouen zwemmen,
        Metten den geest moet hij die betemmen,
        Ontvangen door Gods ware stemmen,
        Als deze strijd nu is volbrocht,
        Hij wordt zocht // wel bedocht, en gebrocht, in Gods rijk t'loon,
        Met triumph schoon // sidt gij in waerden,
        Al sulcken held // bij God meest geld // als den vermaerden.
