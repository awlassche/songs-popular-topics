
    
      
        PRoper ende // ient,
        Maegdeken amoureus //
        Mijn ellende // kent,
        Maakt mij victotieus //
        Niemand dan gij //
        Rein bloemken wtgelesen //
        kan mij maken vrij //
        Die ben ter dood verwezen.
      
      
        In mijn lijden // weet
        Apollo genen raad:
        Niet bezijden // treet,
        Die zijt mijn toeverlaat,
        Want gij vermeught
        Deur troostbaar medecijne,
        Dat ik ben verheugt
        Al laegh ik dood in schrijne.
      
      
        Es mijn quale // dan
        Curabel deur dijn daad:
        Zo dijn tale // kan,
        Verlichten mijnen staat,
        Tooght dat in dij
        Noch leeft een vrouwen herte,
        Op dat gij in mij
        Geneest een bitter smerte.
      
      
        Touft doch lange // niet
        Eer mij, den geest ontwijct,
        Ik ben bange // ziet,
        Dat mij t'herte bezwijkt:
        Zal mij de dood
        Met zijnen vliem deursteken,
        Eilaas t'sal ter noot
        Aan uwen troost gebreken.
      
      
        Est dijn wille // dus,
        Verheugt u dan hier af,
        Dat ik stille // vlus
        Beclimme Iphis graf,
        Wilt u gerust
        O herte! daar toe genen,
        Want voorwaar mij lust,
        T'steruen voor zulk een leun.
      
      
        Rein sukade // zoet,
        Gij valt mij veel te wreed,
        V genade // moet,
        Een ander zijn gereed,
        Mij ziende nu
        In het lijden dus verduldig,
        Weet dat ik voor u,
        Sterue de dood onschuldig.
