
    
      
        OCh hoe schemert mijn gezicht!
        Wat is dit? wat zeid dit grote Licht,
        het Welk komt aan // als een klaar held're Maan
        Hier ter Land // het welk (zo het schijnt) hier stoken komt een brand.
        het Is Willibroord // die Gods Woord // zo het behoort
        Preken komt: het is een Man in deugd vernomt.
        Gods gebod // O! wat lot // brengt Hij tot
        Onze paal // hier toe roept Hij ons altemaal.
      
      
        Ons geeft Eng'land dezen Man:
        Ziet! Hij komt, Hij gaat voort wakker an;
        Als een pijl // vlieght Hij voort, met een bijl
        In de hand // Hij wil het kwaad hier helpen aan een kant.
        En met de mond // wil terstond // Hij de zond
        Straffen gaan // waar nie'e wij zijn zeer bela'en:
        Hij niet traag // maar zeer graag // predikt staegh
        Godes Wet // Hij bevrijd ons van Satans net.
      
      
        O wat groten Ijveraar
        Heeft ons God gegeven tot een Va'er!
        Die met vlijdt // ende zorg, ons bevrijd
        Van de Hell' // die nooit ophoudt met zijn loos gekwel.
        Des Man ons leert // en regeert // en begeert
        Dat wij all' // komen tot des Hemels wall'.
        Wijst Hem eer // valt ter ne'er // voor dien Heer,
        Die met lust // zoeken komt onzer allen rust.
      
      
        Wilbroord heeft den waren God
        Ons gepreeckt, en Godes zoet gebod:
        Ons gebaard // aan onze God, en bewaard
        Heeft dien Held // en gebroken heeft Hij 'thelsch' geweld.
        Des is den een // dien ik meen // anders geen,
        Die geleerd // Holland heeft, en eerst bekeert.
        Hij geleid // Hij bereid // Hij verbreydt
        Heeft het pad // het welk ons brengt tot des Hemels stad.
      
      
        Wat wilt gij noch hebben meer
        Holland? spreekt, zegt uit toch u begeer.
        Weet gij niet // dat door Hem // gij geniet
        Het geloof // zonder het welk gij waart des Duivels roof?
        Wilt dit verstaan // neemt dit aan // van dien Haan,
        Die dit kraeyt // die Gods Woord in u nu zaeyt.
        Gij en kent // de ellend // zonder end,
        Niet ontgaan // het en zij gij dit Geloof neemt aan.
      
      
        Denkt dan wat gij schuldig zijt
        Aan Willibroord, Holland! en weest verblijd,
        Wijst aan Hem // met u hert, met uwe stem,
        Grote eer // die bekeert heeft u tot God wel eer.
        Spant met mij t'saem // wilt de naam // en de faam
        Maken groot // van die ons holp uit de dood.
        Gij hebt stof // geeft Hem lof // die Gods hof
        Nu bewoond // die zo rijkelijk is geloont.
      
      
        Alderwaerdste Godes vrind
        Sint Willibroord! Die metter hart ons mindt:
        Slaat van hoog // op ons all' toch uwe oog,
        Ziet hoe wij // door de zonden zijn in grote lij.
        Vrijdt van den brand // ende schand' // door U hand,
        Uwe Gemeent' // die nu bitter tot U weent;
        Helpt ons voort // Willibroord! // en verhoord
        Ons gebe'en // hebt meedoogen met ons gesteen.
