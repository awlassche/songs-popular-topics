        Op wien mijn her te rust:

        Sent doch bezijden //
        Al die mij benijden //
        Jont mij een cord verblij- den //
        Voor langhduerich torment.
      
      
        Eilaas den stoot
        Is groot // helpt mij winnen t'gevecht:
        Blijf ik in noot,
        De dood // sterf ik met groot onrecht:
        Dijn knecht // wilt niet vercleenen,
        V stenen // hert versoet,
        Spoed // u met enen,
        Secours te verlenen,
        Hem, die (met angstig wenen)
        Verteerd zijns herten bloed.
      
      
        Troostbare bruid,
        Ontsluyt // doch eens des banden stijf,
        Nergens en spruit
        Zulk cruyt // (als gij) confortatijf,
        Mijn lijf // al waarder zeven,
        Daar neven // goed, en schat,
        Dat // soud' ik geven,
        Mocht ik u aankleven,
        Ik heb u (als mijn leven)
        Ooit lief, en werd, gehad.
      
      
        Es dezen rom
        Ooit stom // beschaemt, gij weet het wel:
        Tooghd' ik ooit om
        Rijkdom // t'uwaerts enig voorstel,
        Niet el // dan van mijn ermen,
        T'beschermen // ik verkoos:
        Boos // zijn u termen:
        Wie zal mijns erbermen?
        Bij u is cleen ontfermen
        Nu sterf ik troosteloos.
      
      
        In mijnen strijd
        Verblijd // gij u, dat dunkt mij vrij:
        Hoe wreed gij zijt,
        Altijd // blijft u mijn herte bij:
        Wee mij // dat ict beginne,
        Ik minne // (met gepijn)
        Mijn // viandinne:
        Die (met blijden sinne)
        Mij (als een valse spinne)
        Deurstraelt met haar fenijn.
      
      
        Princesse stout,
        Aenschout // doch eens mijns herten gru:
        Waardij niet cout,
        Gij zout // mij noch ontfermen nu:
        Voor u // blijf ik te pande,
        Met schande // vast gepaelt,
        Haalt // mij te lande
        Of (zo ik verbrandde)
        Mijn leven is d'offrande
        Die mijn onschuld betaalt.
