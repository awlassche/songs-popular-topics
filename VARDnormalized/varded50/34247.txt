
    
      
        DEn Mei die komt ons bij // zeer blij,
        Den Winter kout ons nu verlaten heeft,
        het Geboomte dat komt uit // en spruit
        Door Phoebus, die ons nu zijn warmte geeft;
        Dianâs nachten lang // Boreas stuur en strang
        Heeft ons verlaten, in vrolijkheid leeft,
        Den Nachtegaal zingt // het Geytjen dat springht,
        Het Swaluwtjen sweeft.
      
      
        God Pan die zat in het veld // en teld’
        Zijn Schaepjens die daar weiden in het gras,
        Zijn Pijpjen hij doen nam // en klam
        Op enen berg, daar het zeer geneugh’lijck was,
        In de oog kreeg hij zeer dra // zijn zoete Sylvia,
        Die Coridon ontmoete zeer ras,
        Zij wierden verstoort // hij leide haar voort,
        En wierp ze in het gras.
      
      
        Coridon seyd’, komt rust // met lust
        Mijn zoete Sylvia, wat neffens mij.
        Een suchtjen haar ontvloodt // zeer groot;
        Nochtans lonckten haar ooghjens even blij.
        Coridon sprak haar toe // mijn zoete Beckjen, hoe,
        ‘k En bender niet verre, zucht gij om mij?
        Zegt Herderinne // quelt u de minne?
        Komt klaget mij vrij.
      
      
        Och! neen, het en is dat niet // maar ziet
        Mijn Schaepjens zijn verstrooyt hier in dit dal,
        Een Lammetjen ik hier mis // gewis
        het Is voor mijn herte een groot ongeval:
        Coridon laat ons gaan // Sylvia blijft wat staan;
        In het ruysschen der bossen is geen geschal,
        Niet dan gewoel // van bladertjens koel,
        Ei! maakt mij niet mal.
      
      
        Ik schencke dezen Mei // zeer vrij,
        Aan een Princesse die mijn hert door-wondt,
        Die zoete Veldt-goddin // der min,
        Die haar Minnaers vermaeckt tot allen stond:
        Maar ziet doch eens met lust // hoe dat Coridon kust,
        Zijn Sylvia mint, zijn herteken toont,
        Maeghdetjen reine // lost mij van pijne
        zo lef ik gezond
