
    
      
        Met wandelen in ‘het groen
        Ginck ik mijn bedroefde geest wat vreugd aandoen,
        In bos en wei // Door sant en hei,
        Ginck ik betreen in de zoete Mei,
        Geneughden // En vreugden,
        Dit was // In ‘het gras,
        Elk beesje las,
        Zijn behoef kruipt // Van ‘het groen gespruyt,
        Daar pluckte het zijn voedsel uit.
      
      
        Het zoet parig gediert,
        Dat in de bossen en door de bomen swiert,
        Voelden de min // In hare zin,
        Want de telens lust was daar in,
        Zij zongen // En sprongen,
        Met vreugd // Verheugt,
        In zoet geneucht,
        In zoete lust // Geheel gerust,
        Hebben zij haar zoete min geblust.
      
      
        De Herders in het veld,
        Hebben met gelei haar Herderins verfelt,
        Het zoet gevlei // Was veelderley,
        Dat menige zoete tong daar s
        Tot minne // Te winne,
        De zin // Daar in,
        Te doen gewin,
        Had ider voor // Die hij verkoor,
        Elk een volgde des liefdens spoor.
      
      
        Menige Man en Vrou,
        Sach ik daar gaan o zoet vermaak van de trou,
        Die altemaal // Met zoet onthaal,
        Malkander vermaeckte, voor quael,
        Veel vreugde // Verheugde,
        Haar hert // Dat werd,
        Geheel van smart,
        Door dien zij al // Met zoet geval,
        Zijn lief kuste zonder getal.
      
      
        Ik sach menig zoet paar,
        Gaan wandelen door de velden hier en daar,
        Ik sach daar treen // Veel met haar twen,
        Niemand als ik most gaan alleen,
        De rouwe // kwam houwe,
        Mijn hert // Dat werd,
        Vol grote smart,
        Dit was voor mijn // Een grote pijn,
        Ik dacht waar mag de mijne zijn.
