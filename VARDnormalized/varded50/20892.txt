
    
      
        WIlt Sint Steven // lof gaan geven, op des dag,
        Want op heden Hij den Hemel open zag.
        Wilt nu springen // wilt nu zingen // want het is nu tijd,
        het Is tijd dat alle man ten hoogsten zich verblijd.
      
      
        Wij de deughden // en de vreugden // willen zien
        Van des vromen held, wij willen die belie'n.
        Dit des den gene // wien ik meene // nu te prijzen gaan,
        Die Christum zijnen Heer, zag in den Hemel staan.
      
      
        Des wilt horen // na behoren // met aandacht,
        Hoe Sint Steven heeft zijns levens loop volbraght:
        Hij bekeerden // en Hij leerden // te Jerusalem
        Veel mensen, groot en klein, door kracht van zijne stem.
      
      
        Op die tijden // ging Hij strijden // wakker aan
        Met het Joodse School, het welk Hem wou tegen staan:
        Zij verzonnen // of zij konnen // Hem beletten niet;
        Dit grote werk te zien was haar een groot verdriet.
      
      
        Zij Hem zeiden // en opleiden // we'er en we'er,
        Dat Hij brak de Wet, en dat was vals zijn leer.
        Hij daar tegen // wel te degen // met den Geest vervult,
        Heeft haar gewezen aan dat bij haar was de schuld.
      
      
        Als zij sagen // dat zijn vragen // en bescheid,
        Met haar kloek verstand niet werden ne'er geleid;
        En zijn woorden // die zij hoorden // hadden grote kracht;
        Zo trachten zij dat Hij sou worden omgebracht.
      
      
        Zij bedochten // en verzochten // daar toe mans,
        Die Hem totter dood te helpen, wisten kans:
        En zij kregen // die genegen // waren tot dat kwaad,
        Die Hem opleiden het geen niet waar was inder daad.
      
      
        Deze Engels // deze bengels // zeiden, Dat
        Hij Moysen en God, gelasterd lelijk had.
        Des zo worden // met hun orden // dezen Held gebrocht
        In hare vollen Raad, die Hem te moorden docht.
      
      
        Als een Engel // ('k zeg gheen bengel) // stond Hij daar;
        Niet min als de Sonn', verscheen zijn aanzicht klaar:
        Hij heeft hare // wonderbare // haat en nijdigheid,
        Die s' hadden tegen God, hun daar voor het oog geleid.
      
      
        Ach! zij bersten // ja zij knersten // op haar tand',
        Als met vrome moed, dit uytsprack dezen Sant.
        Deze beelden // zeer verveelden // Stevens vromigheid,
        het Stond haar niet langer aan te horen zulk bescheid.
      
      
        Dan ziet, even // wel heeft Steven // voortgegaen,
        Roepend' overluyd', Ik zie Heer Jesvm staan:
        Mijn Voor-vechter // aan de rechter // hand zijns Vaders staat;
        Ziet op vrij na om-hoogh, geenszins mij God verlaat.
      
      
        Toen verlieten // en verstieten sy-luy Hem,
        Werpt Hem, werpt met steen, zo riep haar schorre stem.
        En zij stopten // ja zij propten // hare oren toe;
        Maar Sinte Steven was in Jesv wel te moe.
      
      
        Zij Hem leiden // zonder beiden // buiten stad,
        Waar met grote lust, aldus Sint Steven bad:
        ,, Wilt ontvangen // met verlangen // Jesu! mijnen geest,
        Ik hebbe nu lang-genoeg op deze aerd' geweest.
      
      
        Voor die genen // die met stenen // Hem de dood
        Aenbrochten, bad Hij met enen ijver groot:
        Voor de zielen // die vernielen // wilden dezen man,
        Boog totter aerd' zijn knieÃ«n Sint Steven t'stonden an.
      
      
        Wilt vergeven // sprak Sint Steven // Heer! de sond'
        Van die worpen mij; zo sprak, zo bad zijn mond:
        Toen gesturven // toen verwurven // heeft het Hemelrijk
        Des vromen Martelaar, die niemand heeft zijns g'lijck.
      
      
        Och! Sint Steven // hoog verheven, in het rijk Gods,
        Wilt mij beschermen in de ure minder doods!
        Wilt verkrijgen // dat bedrijgen // Satan mij niet mag,
        En dat volhardig ik blijf, tot mijn soncxten dag.
      
      
        G' hebt met reden // God gebeden // voor die U
        Gesteent hebben wreed, ei! bidt ook voor mij nu;
        Die mijn zinnen // die mijn minne // heb op U gezet,
        Uwe ogen op mij slaat, gedurig op mij let.
