
    
      
        EEn is alleen // mijn reen
        Duive minjoot,
        Hier geacht kleen // maar gheen
        Voor mij zo groot,
        Sterk als de dood // is liefde stranck,
        Ik was ter noot // van minnen krank,
        Als ik u schanck // den drank zeer excellent,
        Schone Bruid jent // bekent,
        Boet mijn verlanck // den ganck
        Te mywaerts went.
      
      
        Schoon onghelaeckt // volmaakt
        Is mijnen Man,
        Mijn herte waakt // en blaeckt
        In liefden dan.
        Gij dochters van // Ierusalem,
        Spreekt mijn vriend an // en zegget hem,
        O lief u stem // zeer tem // maakt mij verheugt,
        Geeft mij kracht, deugd // en jeugd
        Dat ik op klem // niet strem,
        Bij u in vreugd.
      
      
        Toont mij nu dijn // aanschijn,
        Vriendinne zoet,
        V borsten zijn // als Wijn,
        Lieffelijck goed,
        Der liefden gloed // heeft groot virtuyt,
        Gheen watervloed // en bluscht haar uit,
        Komt uit dat ruit // besluit // der Leeuwen wreed,
        Doet aan u kleed // bereedt,
        Maakt u rein spruit // mijn Bruid
        Wt liefden heet.
      
      
        Met wangen vocht // u zocht
        Ik lief minnaar,
        Och of ik mocht // ontknocht
        Van lijden zwaar,
        Vrij kussen daar // buiten versint,
        V die eenpaar // mijn ziel bemint,
        Want als die kind // begint // te baren wel,
        Mijn pijn is fel // maar snel,
        Bij u troost vint // men in het
        Kruis end' gekwel.
      
      
        Heft op u hoofd // geloofd
        Mijn woord ydoon,
        V vlees verdooft // zo rooft
        Niemand u Kroon,
        In mijnen troon // werd gij gebracht,
        Tis daar zo schoon // van mij gewracht,
        Nooit oog en zacht // dus lacht // inwendig meest,
        Blijd' in den geest // die beest,
        Met haar geslacht, haar macht
        Doch niet en vreest.
      
      
        O Bruidegom // mijn rom
        Zijt gij altijd,
        V eigendom // daarom
        Werd zeer benijdt,
        Maar in den strijd // doet mij bijstand,
        Breed ende wijt // u sterke hand,
        Men vint in het land // niemand // getrou als gij
        Dus blijf ik bij // u vrij,
        Vast ingheplant // ja want
        Twee een zijn wij.
      
      
        Princesse zaan // vergaan
        Zal u verdriet,
        Want elken traan // af dwaan
        Zal ik u, ziet,
        En twijffelt niet // maar op mij bout,
        Dat gij als riet // niet en verflout,
        Mijn woord aenschout // betrout // op mij gepast,
        Wie in u tast // vreugd vast,
        Quelt oft benout // zeer stout,
        Mijn oogh' aentast.
      
      
        Prinselijk Heer // wanneer
        Oordeelt gij recht,
        Met valse leer // men zeer
        Tegen mij vecht,
        Elk wederseght // mij als gestoord,
        Veel listen legt // die Slangh' aan het boord,
        Maar lief u woord // akkoord // is een licht vat
        Op mijnen pat // op dat
        Ik door de poort // ga voort,
        Binnen de Stad.
