
    
      
        DOor s'Meys virtuyt // rijst nu uit, rouwe // zwaar,
        Jonkvrouwe // klaar geprezen,
        O Syon volgt u Vorst getrouwe // naar,
        In het nouwe // maar, na dezen
        Zal wezen // u belonen // wijd,
        Als gij in het eeuwig kronen // zijt,
        Den schonen // tijd, des Meis deelachtig
        In volkomen wezen krachtig.
      
      
        Voorbij is al des winters tegen // stoot,
        Den regen // groot, verdreven,
        T'veldt staat gebloeid in aller wegen // root
        die plegen // dood, zijn leven,
        Het geven // groene rijzen // klank,
        Den Mey-boom groen // tot prijs en // dank
        Met wijzen // sanck, en nieuwe tongen,
        Wordt verwinninghe gezongen.
      
      
        Syon verblijd in dit verwinnen // want
        Tis binnen // tlandt, al vrede,
        De liefd' uws vriends in uwe zinnen // plant
        In minnen // brand, hij mede,
        De stede // is hij bereyen // gaan,
        Daar alzo schone Meien // staan,
        V schreien // zaan, stelt al bezijden,
        In den Mei wilt u verblijden.
      
      
        Princesse Bruid, met enen stijven // moet,
        Laat drijven // vloed, van vreugden,
        In uwen geest, des Meis inlijven // zoet,
        Beklijven // doet, in jeuchgden,
        Zijn deuchden // zult gij romen // al,
        Daar genen winter komen, zal,
        Tot blomen // val, noch heet verbroeien,
        En gij zult daar eewigh bloeien.
