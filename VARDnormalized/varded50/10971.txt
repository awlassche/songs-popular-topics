
    
      
        Als liefde blent // mijn gemoed had bevangen,
        Heeft mij gewend // rontsom nu hier en daar
        End’ met rotment // mij gezocht te verhangen,
        Maar reden jent // heeft mij gherecht voorwaar
        Tot dat zij mij bracht bij u licht zeer klaar:
        Maar zo haast uwen ogen // schijn
        Mij in het hert dede dogen // pijn,
        Wast niet in tvermoghen // mijn
        Te weerstaan zijn kracht zwaar.
      
      
        Cupidoos licht // meynt hem aan mij te komen,
        Van tvier gesticht // in mijn amoureus hert,
        Maar tis tghesticht // van u Lief uitgenomen,
        Twelck mij verplicht // aan u nu houdt met smert
        Dies u die eer alleen gegeven werd
        Weg liefd’ met u ontspresen vier,
        Gij condt mij niet doen vrezen // hier,
        Maar mijn liefkens wezen fier
        Was in die const expert.
      
      
        Een gestadigen strijdt // sedert ik u beminde,
        Beving subijt // mijn gemoet zeer bezwaard
        Liefde somtijt // doet mij wat vreuchts ontbinden,
        Maar wanhoops krijt // gij zijt te wreed van aard,
        deen in mijn vreugd, en dander lijden baard:
        Zo dat ik in het verdriet // belaen
        In het einde gants te niet // zal gaan,
        Een rijk dat men ziet staan
        In twen, is zwak van aard.
      
      
        Tegens mijn dank, zeer wreed gij u gaat tonen,
        moet ik met dwanck, u minnen dag en nacht
        Met lijden strak // gaat gij mijn Lief belonen,
        Als gij verlanck // zoekt in tgheen dat ik wacht
        Dies Goddin Lief, mijn liefde nu betracht
        V fierheid gants bezijden // stelt,
        Aanziet hoe mij het lijden // quelt,
        En dat mijn verblijden // smelt
        Als gij mij Lief veracht.
