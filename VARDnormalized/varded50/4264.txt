
    
      
        Nooit in het gemoet // mocht drouver //
        man // verzuchten:
        En diet mij doet, heeft touver //
        van // genuchten:
        Tot mijn verdriet, ver
        wekt zij schade // boos //
        ben ik dan niet //
        Een minnaar rade // loos ?
      
      
        Eertijds ik docht
        Vrueght wil ik mainteneren:
        Meende ik mocht
        Op liefs samblant fonderen:
        Maar zulk confoort
        Schat ik vervlogen // wind,
        Als die haar woord
        Nu gans gelogen // vint.
      
      
        Eigen ben ik
        Haars dienst, bij goed gedogen:
        T'sy in den strik,
        Cause zijn haar bruin ogen:
        Mijn herte toont
        Haar eerbaarheid, naar schuld:
        Is het ongeloond,
        Ik neme mijn geduld.
      
      
        Laestmael wij twee,
        De zoetste vrueght bedreven
        Droufheyt, en wee,
        Hadden mij doen begeven,
        Nu gaetse // bij
        Een ander treden // ziet,
        Och! haetse // mij,
        Ken weet de reden // niet.
      
      
        Keert zij mij t'hooft,
        De schoonste der iongvrouwen,
        Als niet gelooft
        Haar handekens te douwen,
        Staat mijn dangier
        Meer, en meer, voort // te // gaan,
        Al had' ik schier,
        Den wreetsten moord // gedaan.
      
      
        Eilaas, eilaas,
        Dit zijn der minnen // treken,
        Die mij (arm dwaas)
        Mijn kranke zinnen // breken:
        T'hert leit mij zwaar
        D'welck plocht verhueght // te // zijn
        Absent van haar
        Is alle vrueght // fenijn.
      
      
        Nu mag ik wel
        Mijn eigen trooster wezen,
        En mijn gekwel
        Met goed verdrag genezen,
        Niemand mij geen
        Bekwamer lesse // schrijft,
        Nu dat zij een
        S'anders Princesse // blijft.
