
    
      
        HEt plag te zijn,
        Als Christenen malkanderen zagen,
        Met vreugden fijn,
        Hoe staat 'tghemoet? was doen het vragen,
        En gaven vroed // malkander moed,
        Om den Heer te behagen,
        Hoe goed // hoe zoet // als heunigh vloed,
        Vloeyden daar des Geests beken,
        T'hert werd zeer dicht // gesticht // verlicht
        Vyerigh ontsteken.
      
      
        Vreugdig confoort
        Wast bij malkander doens te wezen,
        De een uit Gods woord
        Heeft een Capittelken gelezen,
        En de ander sanck // een Lied dat klank,
        Om der Zielen genezen,
        Daar sprank // de drank // uit elken rank,
        Aan den Wijnstok devyne,
        Die bergen daar // dropen eenpaar
        Van zoeten Wijne.
      
      
        Maar nu eilaas
        Schijnt elk te wezen d'onversaetste
        Na dit geraas
        Van zijnen naasten, op de quaetste,
        Wijs oft manier // te spreken hier,
        Dit is nu t'eerste en 'tlaetste
        Dit vier // dood schier // wel zo veel dier
        Edel gekochte Zielen,
        Als ter woestijn // door het fenijn
        Der slangen vielen.
      
      
        Weinig de Schrift,
        Tot beteringhe heeft nu stede,
        Maar kwaad vergift
        Des achterklaps, vol van onvrede,
        Daar openbaar // de lieden maar
        Argher en worden mede,
        Dat haar // valt zwaar // te jagen naar
        Den vrede Gods voorhanden,
        Die in den graat // te boven gaat
        Alle verstanden.
      
      
        Maar gij die draagt,
        Als gezalfde // enen Christen Name,
        Na vrede jaagt,
        En verbreydt niemandts kwade fame,
        Want dat verstoort // liefd' en akkoord,
        Dus mijdt laster en blame,
        Gaat voort // na het woord // alsoot behoort,
        Zijn het ergherlijcke leden,
        Snijdt af, steekt uit // doet na het geluid
        Van Christus reden.
      
      
        Maakt overleg,
        O Broeders, wilt considereren,
        Hoe dat den weg
        Des Heren // die gij moet passeren,
        Zeer wijnig tijd // is zonder strijd,
        Doch zonder reculeren,
        Met vlijt // doorlijt // volherdich zijt,
        En wandelt anders genen,
        Want daar en leit // ter zaligheid
        Maar dezen enen.
