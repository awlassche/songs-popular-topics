
    
      
        GHebenedijt // Weest altijd // O Here almachtig
        Die al dat leeft // voedsel geeft // door v ghenaden
        In Christi naam // ons lichaam // niet alleen sijt gedachtig
        Maar met v woord // rechte voort // ons ziel wilt verzaden
        Van eigen rust // en wellust // ons hebt ontladen,
        O Here wijs // geeft drank en spijs // die met v vereend,
        Sulcx dat wij niet // misbruiken iet // van de weldaden
        Die gij goedig // overvloedig // ons mildlijck verleend.
