
    
      
        NImrod een Iagher van groter fame,
        Was uiter stamme // van Ham present,
        T’beginsel van zijn Rijke eersame
        Dat was met name // Babel bekent,
        Dit was een Heer // van zeer // groot reghement,
        Enen wilden Iagher wijd uit ghebreyt,
        Jonk met coragie // in s’Weerelts Bosschagie,
        Maar wat hij vink dat was ijdelheid.
      
      
        Abraham is komen naar dezen
        Een zaad geprezen // uit Sem hij sprot
        Die joeg al anders // alzo wij lezen,
        Den eeuwig wezen // bij God devoot,
        Hij kwam // en nam // Loth uit gevangens noot,
        Van Kedor Laomor, die ook planteit
        Wel joeg uit spijten // die Sodomijten,
        Maar dat hij vink het was ijdelheid.
      
      
        Esau wesende Iacobs broeder
        Van ener Moeder // Rebecca fijn,
        Iacob verkoos als een wijs bevroeder
        Enen hoeder // der schapen te zijn
        Maar Esau // zeer ru // op dat termijn
        Was een Iager, zo die schrift zeid,
        Meest al zijn leven // daar in bedreven,
        Maar dat hij vink het was ijdelheid.
      
      
        Iosua wesende licht ter velde
        Heeft met ghewelde // ook vele gejaagd
        Die Koningen eenendertich men telde,
        Die als rebelde // waren geplaagd,
        Binnen Canaan // verwan // zeer onversaagd
        Iosua al haar Majesteit,
        Diet anders wagen // zonder Gods behagen
        Al wat ze vangen is ijdelheid.
      
      
        Gedion naar des Heren beslechten,
        Is wel ten rechten // ter jacht gegaan
        Vergadert met zijn drie honderd knechten
        Gesteld om vechten // en joegen zaan
        Na haar begeer // dat heer // van Madiaen,
        Die in het land waren uit ghespreyt,
        Om te verderven // des Heren erven,
        Maar wat ze vingen was ijdelheid.
      
      
        Ionathan zijnde zijns levens wager
        Was een Iager // niet van Konijn
        Hij track op met zijnen Wapendrager,
        Wel een verslager // der Philistijn
        Hij bracht // met kracht // en deed in doods gepijn
        De ander en hebben hem niet verbeid,
        Elk henen vluchte // door dit geruchte
        Maar dat ze vingen was ijdelheid.
      
      
        David die claeght dat hem zijn vijanden,
        Strikken en banden // leiden genoeg,
        Om hem te vangen tot zijner schanden,
        Maar door zijn handen // den Here verjoech
        Doen hij zo plat // den Goliath // versloeg
        Wie hoorde ook van vromer feit
        Die ander hopen // moesten gaan lopen,
        Al wat ze vingen was ijdelheid.
      
      
        Nebucad Nezar, na Schrifts oorkonden,
        Heeft ook gezonden // een Heir zeer snel
        Die hebben Landen en lieden verslonden
        Als Leeuwen monden // wreed en fel,
        Zij kwamen // tezamen // tot Israel
        Hebben Bethulia beleid
        T’was haar bedenken // haar al te krenken,
        Maar dat ze vingen was ijdelheid.
      
      
        Veel wilde Iaghers na Schrifts verklaren,
        Hebben veel Iaren // dat volk verdooft,
        Zij troosten haar met vredighe maren,
        Oft leugens waren // het werd geloofd
        Men stak // tot gemak // pueleins onder t’hooft,
        Dat heeft den Here kwalijk ghegreyt,
        Zoeken de Zielen // zo te vernielen,
        Maar dat zij vingen was ijdelheid.
      
      
        Aan wilde Iaghers het nooit en faelde
        Elk vierich straelde // liep diligent,
        Waar door zo Ieremias verhaalde
        Dat volk zeer dwaalde // maar ongesent
        Vol van bedrog // zijn doch // als Herders blent,
        Hebben die schapen kwalijk gheweyt
        Die dus omginghen // met zulke dingen
        Al wat ze vingen // t’was ijdelheid.
      
      
        Paulus joeg maar noch niet gegrepen
        Want tot Philippen // getuigt hy’t klaar,
        Zijn vaste hoep is hem niet ontsepen,
        T’stont zo gheschepen // hij joeghder naar
        En hadde lust // met rust // te zijn bij haar
        Maar liever bij den Here ghecleyt,
        Die anders leren // tot groot afkeren,
        Al wat zij vangen t’is ijdelheid
      
      
        Die wilde Iaghers wilt dit recht vaten,
        Hoe schoon zij praten // ghelooftse niet,
        Laat u niet vangen, als tot Galaten
        Ter cleyner baten // wel is gheschiet,
        Want dat verwekt // en strekt // tot groot verdriet,
        Hier na salt noch wel worden beschreyt,
        Die tot verscheuren // altijd labeuren
        Al wat zij vangen t’is ijdelheid.
