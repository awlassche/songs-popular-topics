
    
      
        Poliphemus ande Stranden // voelt branden
        Zijn wreed hart in heete min,
        Met een Seyse hij hem scheerde // ende hij keerde
        Met zijn een oog t'Zeewaert in.
      
      
        Zevenmaal sach hij rontsomme // is geklomme
        Op een steile rots aan Zee,
        Daar hij op zijn Rijspijp neurde // hoe hij treurde
        Om zijn Nimphe Galathe.
      
      
        Galathe heeft hij gesonghe, zal den jonge
        Poliphemus nooit zijn brand
        Met zijn water Nimf verkoelen // moet ik voelen
        Hoe den Etna mijn vermandt.
      
      
        Galathea kompt wat nader // denkt mijn Vader
        Is een Zee-god onbevreest,
        Voor Jupijn noch voor zijn donder // is het geen wonder
        Ben ik nooit vervaard geweest.
      
      
        Galathe braveert de wereld // geen geperelt
        Is zo el, noch gij Jupijn,
        Zij is bloosender als Flora // trots Aurora,
        Klaerder als een Kristalijn.
      
      
        Wt u ooghjes Paragonne // haalt de Zonne
        Al haar schijnsel die zij geeft,
        Gij verdooft de silv're mane // ja Diane
        Bij u schoonheid niet en heeft.
      
      
        Dies mijn schoonste Galathea // ooght mijn Vee na,
        Die de Sillops Berg bedekt,
        En mijn knods bedwinght de Landen // met de Stranden
        Alzo wijt ons Landschap strekt.
      
      
        Ik heb o! Goddin vol waarde // in mijn Garde
        Al de vruchten Pomene,
        Daar ik op mijn fluit zal spelen // en zal kwelen
        Aengenamer dan Orphe.
      
      
        Gij zegt ik heb maar een oge // ziet de hoge
        Phebus heefter ook maar een,
        Zonder d'welck dat Aard noch Hemel // en het gewemel,
        Van de Starren nooit en scheen.
      
      
        Nooit hield hij zijn oog in des Westen // dan ten lesten
        Keerde hij hem razend om,
        Om zijn Galathe beminde // te verslinden
        Gaf een yselijcke brom.
      
      
        Door dit gruwelijk naerbotsen // spleet de Rotsen
        Vanden Berg AEtna heel,
        Galathea teegh aan 'tvluchten // doch het zuchten
        Was tot Arsis achterdeel.
