
    
      
        Komt hier Nymphjes al te zam,
        Die tot vreugden altijd zijn bekwaam,
        Vlecht een Krans // voegt u bij dezen glans,
        Aan de Rij // bij de Harders, pluckt nu Bloempjes vrij;
        Ay zoekt doch Kruid // van viertuyt, tot versoet
        Van des Maagd // die meneert dat haar behaagt:
        Noemt haar lof // nu zeer grof // spart gheen stof,
        Maar versierd // nu haar hoofd met een Krans omzwiert.
      
      
        Tot sieraad al van haar Ieught,
        Die zij nu door brengt met grote vreugd:
        Zij is het waart // door haar deugd, dit zij baertmet geneught!
        Want zij haar door het zoet vermaak verheugt:
        Dus wilt haar eer // breiden veer, met geschal
        Want het geluid // spelt in het dal doch op u Fluit;
        Schept doch hier // goede chier // met mplaysier.
        Haar ontmoet // met Laurier haar des eer aan doet.
      
      
        Ik zal u in het eerst voor gaan,
        Spoeydt u Nymphjes soetjens achter aan,
        Komt nu ras // hier ontrent // met gewas:
        Want gij bent // met u kruiden Hymus wel bekent:
        Orpheus queelt // ende spelt met u Snaar
        Op Muziek, aan dit vloeyend' Bekje klaar,
        Dat daar stroomt // door het Gheboomt // niet en schroomt
        Voor het Gediert // dat in het Bos met gedruis zo tiert.
      
      
        Phoebus helder glanssend' licht,
        Brengt ons met zijn straeltjes in het gezicht:
        Want de Zon // hier op d'Aerdt, aan des Bron,
        V bewaard // op dat vreugdig, gij dan wordt verjaard,
        Gij wordt gekroond // en geloont // nu ter tijd,
        Dus weest gij in den Here nu t'saem verblijd,
        Die ons loon // nu zeer schoon // stelt ten toon
        Voor ons oog, dat het straelt, Tot den Hemel hoog
