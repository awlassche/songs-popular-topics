
    
      
        Krachtig // nooit
        Geen gezond // hert gewont,
        Als het mijne onverschoont:
        Klachtig // ooit
        Mijnen mond // gans den gront
        Minder zielen heeft vertoond,
        Aan een jonge maagd,
        Die mij iaeght // onversaagd,
        En gans desperaat // laat,
        Tis om niet geclaeght,
        Haar behaagt
        Dezen mijnen droeven staat.
      
      
        Al dat in
        Mijnen geest // onbevreest
        Plag te leven zonder noot:
        Heeft den zin
        Aldermeest // met tempeest
        Vast benout, tot in de dood:
        Eilaas! wie zal mij
        Maken vrij // nu dat zij
        Mij houdt in den last // vast?
        die mij gaan voor bij
        Roupen zij,
        Over mij ellendig gast.
      
      
        Het Herte mijn
        Zucht en treurt // angstig scheurt,
        In benoutheyt schier versmacht:
        Moet het zijn
        Dus besuert // mij gebuert
        (Arme weze) meer dan kracht:
        Heb ik metter daad
        Haren raad // ooit versmaad,
        Of geacht te cleen: // neen?
        Maar als delicaat
        
          Toeverlaat
        
        Haar ter noot bekent alleen.
      
      
        Anders geen
        Troost zo zoet // die t'gemoet
        Geeft de ruste principaal:
        Dan dat reen
        Edel bloed // die mij doet
        s'Daegs zuchten tien dusentmael,
        Valt zij mij wat zwaar
        Korts hier naar // werd troostbaar
        Al wat zij beleeft // geeft:
        Sulcx betrouw' ik haar
        Die voorwaar
        Een goed hert te my-waert heeft.
      
      
        Rustig zal
        Ik met vlijt // zijn altijd
        Dienstbaar vast in haar verbond,
        T'sy of al
        Mijn profijt // storm, en strijd,
        Wezen zal, ter laster stond:
        Daar mij anders niet
        Dan verdriet // en geschiet,
        Wacht ik goed akkoord // voort:
        Zo zij mij aanziet,
        Ende biet
        Een gesontmakende woord.
      
      
        In mijn hert
        Schoonder kind // nooit gezind,
        Niet aanziende druk, of noot,
        Thisbe werd
        Vierich in het // hert bemint
        Van Piramus totter dood:
        Maar dit excelent,
        Partinent // bloemken ient,
        Klimt in hoger graat // iaet!
        Ben ik een moment
        Haar absent
        Tis een leven desolaat.
      
      
        Naar dien zij
        Even vast // groijt en wast
        In mijn hert zonder af gaan,
        Vind' ik mij
        Aengetast // overlast
        (Boven reden) aangedaan:
        Ik moet even wel
        Mijn gekwel // achten spel,
        Wat baat veel geklag // ach!
        Al valt zij rebel,
        Niemand el
        Mijn verzuchten einden mag.
