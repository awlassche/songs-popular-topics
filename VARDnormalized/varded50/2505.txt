
    
      
        Mijn zinnen // ziet // die minnen // hiet,
        U overschoone Maakt,
        Met liefde vast // als een trouw gast,
        Die reine liefde draagt.
        U klaar aanschijn // verheuchtet harde mijn
        Als u klaarheid // inder waarheid
        Bij mij sleg mag zijn.
        Droefheede // vlucht, de wreede // zucht,
        Met al het verdriet en pijn
        Tot niet is het haast gebracht
        Mijn rouwe // snood,
        o vrouwe // groot,
        Als gij mij maar toelacht.
      
      
        Ik b'nijd // en haat, met spijd // en smaat,
        Dat gij zo bent verkuist
        Met die gesel // gij kent hem wel,
        Die t'uwent dikwijls ruyst
        Als ik bij u wil gaan // zie ik hem bij u staan,
        Ik ga weder // op en neder,
        Swaermoedich belaen,
        Mijn ijver // klaacht, die styver // jaagt,
        En port het hart vast aan
        Dat willend' niet en mag,
        Door tegenheid,
        
          gheleghentheyt
        
        Verwacht ik nacht en dag.
      
      
        Doch zelden // tijd // oft quelt // en spijt,
        Mij staag in mijn ghemoet
        Dat gij hier lijdt // of werd ghevrijdt,
        Van sulcken slechten bloed,
        Wiens roem is syn geslacht // en is zelfs niet geacht,
        Van ellick // een // het wellick // gheen,
        Kennis geeft de macht.
        De luiden // nu // zien huiden // nu
        Al na de weytse dracht,
        Doch is zij best bewaard
        Die merken // zet //
        op werken // let,
        En op goe geest en aard.
      
      
        Verkoren // schat // te voren // had
        Gedocht en dan de daat,
        Wel voor u ziet // alst is gheschiet //
        Berouwt het meest te laat,
        Al raed' ik u hier toe // gij meucht wel zeggen hoe,
        Dus beladen // met mijn schaden,
        Raakt u wat ik doe?
        Dan waarde // maakt // die de aarde draagt,
        Duit mijn goe zorg in te goe,
        U welvaart ik besin,
        Kend' u letten doch,
        Annetta, och
        Dat ik u dus bemin?
      
      
        U waardigheid en aardigheid
        Heb ik nooit eer gedaan,
        Als het wel betaamt, dies ik beschaamt
        Voor u altijd zal staan.
        Ik heb u, lief, gelakt, en ben zelfs niet volmaakt,
        Ik soud' wel laten, had zijn praten
        Mij eerst niet gheraeckt,
        O braven dier, het begraven vier
        Na u begeren haackt
        En ziet niet na de konst,
        Lief, ik ten dele,
        mij bevele
        In goede jonst.
