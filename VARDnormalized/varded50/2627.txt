
    
      
        Het weten en wijs syn, sou mij in alle delen // zeer veel verschelen,
        Het weten leert kunst-greep, ambt, recht spreken en helen // na bootsen spelen,
        Veel krijgt die schrand'righ leest // verschiet van boeken,
        Maar achtmen wijsheid meest // men moetse in den Geest // bij Gode zoeken.
      
      
        De pronk-kennis en yd'le roem van veel te weten // maakt ons vermeten:
        En leider! om goed-wording wij niet eens en zweten // maar God vergeten,
        Wij zoeken het valse schijn // voor het goede wezen,
        Al werd van ik en mijn // de schaduw niet maar het zijn // voor het best geprezen.
      
      
        De liefde loose harsen-kunt pooght steeds te pralle // met schoon te kallen,
        Doch ware wijsheyts hoogste punt, zoekt God in alle // meest te gevalle,
        Wie die in zich bevind // die zal bemerken,
        Dat woorden-kunst is wind // want het meeste nut verschijnt // in goede werken.
      
      
        De eerzucht drijft verwaende lien tot hoog te spreken // voor slechte lekken,
        Doch eenvoudelijck slecht, en recht berou te spreken // doet meer af-breken,
        En schrikken van het kwaad // ja gants verlaten,
        Daar de ander het vollick laat // verwonderd; want het verstaat // niet wat ze praten.
      
      
        Wat baat dat men zo zeer verkeert // de tijd verkwisten // in schoolsche twisten?
        Al waar men schoon zo schrift geleerd // dat wij het al wisten // ten maakt geen Christen;
        Maar wel die sabbatteert // van eigen wanen,
        Van zonden en begeert // die zelfs doet het geen hij leert // die kan vermanen.
      
      
        Het mond-gezwets kijflusts, woord-strijd noch het bitter schelden // die stichten zelden,
        Maar baren felle nijt, weer-wraeck en straf vergelden // van het geen hen quelden,
        Verkeerden van het gemoet // die last'ren garen
        Het geen een ander doet // het zij dan kwaad of goed // syn dat leraren?
      
      
        Gij meesters kloek, vloeiend en rijk in wetenheden // van rijm en reden,
        Dijn pont, dijn licht, dijn Hemel-gaef, wilt toch besteden // tot nut en vreden,
        Jaagt uit dijn eigen hert // dijn huis-vijanden,
        Maar meest ziet elk te vert // en blijft in het arts verwert // met onverstanden.
