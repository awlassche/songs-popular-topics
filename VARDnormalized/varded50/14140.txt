
    
      
        BLoeyende Jeugd // gij meucht // wel zijn verheugt,
        Want dat u lange heeft ontbroken,
        Is nu beweucht // met vreugd // uit goeder denkt,
        Dat in den mei zeer zoet van roken,
        Mey-bloemken liefelick ontloken,
        Die winter lank // zeer wranc // met stranc // bedwanck,
        Is vrij en vranc voor ons gedoken.
      
      
        Schoon melodyen // ook wyen // zoude dit zien,
        En hem naar dezen mei niet pogen,
        Melancolien // moet vlien // men ziet gezien,
        Dat de Vyghboomen knopen togen,
        En den wijnstok krijgt nu ogen,
        Nooit oog opslag // en sach // den dag // die mag,
        te Winters geklag dus doen verdrogen.
      
      
        Nooit mei zo zoet // en stoet // gebloeid in het bloed
        Compt hier mijn Bruyt mijn Koninginne,
        Valt mij te voet // met spoed // storm regen vloed
        Is al vergaan lieve Vriendinne,
        Aanziet hoe de Tortelduyvinne,
        In het Lust-hof zingt // ontspringt // kom drinkt //
        Hier drank die blinckt uit reynder minne.
      
      
        Verkoren graan // zeer zaan // wilt snellic gaan,
        Tot de Fontein zonder verdrieten,
        Den dorst belaen // verstaan // klaar als een traan
        Sietmen den vloed daar henen vlieten,
        Van alles meucht gij daar genieten,
        Rijeckende cruyt, en fruit, den buit, deelt uit,
        Des Meis viertuyt, doet al opschieten.
      
      
        Een Roosken root // minjoot // staat daar zeer bloot,
        Onder de Doorens uytgesprooten,
        Van het hemels brood // ter noot // uit s'Vaders schoot,
        Is over al het Veld begooten,
        t'Lam Syons heeft Boek ontslooten,
        t'Hantschrift in twen, gescheen, tgeween, niet cleen,
        Is al gemeen, daar deur verschoten.
      
      
        Princesse fier // Rozier // lief goedertier,
        Wilt voor den Winter niet meer schromen,
        Maar maakt u schier // tot hier // uwen princier,
        Heeft zijnen loon met hem genomen,
        Hij zal haastelijk tot u komen,
        In s'Hemels Troon, ydoon, de kroon, zeer schoon,
        Werd uwen loon met alle vromen.
