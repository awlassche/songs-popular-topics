
    
      
        HOe diep zijt gij gezonken // met u gewenste smart
        O zoete zachte vonken // en borrelt in mijn hart!
      
      
        Gij komt mij staegh vermanen // van mijner zielen bruyt,
        Som parst gij blye tranen // ten droeven ogen uit,
      
      
        Gheen onweer u en blusten // uit mijn verheerd gedacht,
        En of gij som wat rusten // 'twas ga'ren van u kracht.
      
      
        V krachten die mij glissen // door het merrech in het gebeend,
        Soud' ik niet willen missen // om 'sdagheraets ghesteent.
      
      
        Al doedy somtijds kwijnen // als't lichaams ooch ontbeert,
        Mijn ziel ziet lijckwel schynen // de schoonheid die zij eert.
      
      
        En schept zo groot behagen // in het Goddelijk sieraad,
        Dat zij na smart te dragen // voor sulcken wonder staat.
      
      
        Een wonder dat verflauwen // alle ander schoonheid doet,
        En dalen haar wijnbraeuwen // en zinken haren moet.
      
      
        De leden en de zeden // en't vierige verstand
        Op Pallas leeft gesneden // van Pallas eigen hand.
      
      
        Omhelst mijn geest vol minnen;, en zij, van hare kant,
        Omhelzen al mijn zinnen // met even stracken band.
      
      
        Had mij dit heil begeven // zo was mijn weelde voos;
        En al het doffe leven // ontseech mij smakeloos.
      
      
        Des die van het overwelven // des Werelds Bouwheer is,
        Geef dat ik mis mij zelven // eer ik u liefde mis.
