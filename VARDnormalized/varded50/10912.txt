
    
      
        Ia des Meis tijd, verblijd // nu in het gemene
        Zonder respijt, subijt, die jonge jeugd
        Met groot jolijt // en vlijt // is elken ene
        In daertsche krijt // want zijt al doen met vreugd
        Om alzo met gheneucht // te treden na de wei:
        Want Bloemkens rein // in het aardse dal
        Groot ende klein // staan over al
        Lieflijk certein // breet ende smal
        Met dezen koelen Mei.
      
      
        Aurora goed // die moet // als dan ook blijken
        De kruiden zoet, die doet // zij groeien wel,
        Den water vloed // met spoed // vliet lancks de dijken:
        Die God behoed // zeer Vroed // in het lopen snel
        Der Amoreusen spel // Hoortmen in die contrei
        Als Luiten // Fluiten // zoeten toon
        Ontsluiten // wten // thert ydoon
        Daar buiten // ruiten // voglen schoon
        Met dezen koelen Mei.
      
      
        Hoe triumphant // playsant // moetet daar wezen
        Waar met verstand // die hand // een wordt gedaan:
        Die lancx den kant // van tlant // gaan om genezen
        der liefde brand // en vant // door zoet vermaan
        Daar twee gheliefkens staan, plukken Acoley:
        Die in eren zo paren // dan
        Onzen God wel vergaren // kan
        Zo krijgt een jonckvrou haren // man
        Straks in den koelen Mei.
      
      
        Prinsen eerbaar // te gaar // wilt diviseren
        Liefs schoonheid klaar // van haar // niet en veracht:
        Oft groot misbaar // voorwaar // sou u causeren
        Daar gij eenpaar // minnaar // af wordt verwacht,
        Tis goed wel overdacht // worden menigerlei
        Zo gij u lief gewinnen // meucht
        Suldy dan zelfs bekennen // denkt:
        Waar door u thert van binnen // vreugd
        Geeft met den koelen Mei.
