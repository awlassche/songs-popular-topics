
    
      
        IN mijn leden // Hoe fris en sterk,
        Komt noch op heden // Geen trek tot werk,
        Om wat te winnen // heb ik geen lust,
        En in het beginnen // Zoek ik de rust:
        Daar valt te slaven // Blijf ik van dan,
        Als and're draven // Houd ik van staan,
        En zoek het gemak // Doch niet aan de bak,
        Daar kan ik laven, Mijn mage-sack.
      
      
        Ik weet van Sayen // Of Ploegens tijd,
        Noch ook van Mayen // Als zeer verblijd,
        Mijn Buren voeren // Den Oogst van het Veld,
        Dan moet ik loeren // De slaap mij quelt.
        Voor mij te woelen // Des Zomers heet,
        Dan moest ik voelen // mijn eigen zweet,
        Dat mij niet en past // Want dit is een last,
        Best mis te doelen // Die harde kwast.
      
      
        Om zwaar te dragen // En dient mij niet,
        En veel te sagen // Is ook verdriet;
        Die mij komt spreken // Om wat te doen,
        Vindt mij bezweken // En dapper groen:
        Want al mijn vreugde // Wordt dan verdriet,
        Die mij verheugde // En sprak zo niet;
        Zie ik een kloek man // Die d'handen slaat an,
        Van zulke deughden // Zie ik dwars van.
      
      
        Ik houd van pralen // Het gantse Iaer,
        En veel te onthalen // Dan hier dan daar,
        Een lekker leven // Dat is mijn zin,
        Niet veel te geven // Om klein gewin,
        Net in de Kleeren // Vroeg in het gelach
        Bij grote Heren // Den gantsen dag;
        Zo lang dit wil staan // Zal het alles wel gaan,
        Maar licht kan het keren // Dan is het gedaan.
      
      
        Die staat wil houwen // Met luttel geld,
        En hoog wil bouwen // Op het bolle Veld,
        Vint hem bedrogen // Al eer hij het gist,
        het Is weg gevloogen // Dat hij nood mist;
        Dies moet ik zorgen // Dat mijn geluk
        Van dag of morgen // Verkeer in druk:
        Daar van dat alree // Mijn hertje doet wee,
        Die plag te borgen // Roept geld geree.
      
      
        Mijn goede Vrienden // zien van mij af,
        En mijn Beminde // Wordt droef en straf;
        Mijn lekker kluiven // Is gans verkeert,
        Men bakt geen struyven // Meer aan mijn Hert,
        Geen Suycker-wijntjes // Geen rare Vis,
        Geen Haas of Knijntjen // Komt op mijn Dis;
        Mijn Kleding wordt vaal // mijn Beursjen is kaal
        En in mijn Spijntjen // Klinckt geen Reaal.
      
      
        Mijn Huis versleten // Kan kwalijk staan,
        Het Dak gespleeten // En heel vergaan:
        De Kat en Muizen // Lopen van hier,
        Daar ik moet kluizen // Met klein playsier,
        Ik zeg u heden // En klaegh ook dit,
        Dat Frank mijn leden // Doorgaans bezit,
        En maakt dag aan dag // Dat ik niet een slag
        Van and're mede // Aanschouwen mag.
      
      
        Nu moet ik zuigen // Aan duim en poot,
        Om te betuigen // Mijn hongers noot;
        Nu moet ik sparen // Met groot verdriet,
        Om te vergaren // En woud' ik niet:
        Daar baat geen klagen // Die het willens mist,
        Mijn schone dagen // Heb ik verquist,
        Al voel ik verdriet // het En schat mij doch niet,
        het Is met behagen // Van mij geschiet.
      
      
        Maar ik zal moeten // Door grote nood
        Met and're wroeten // Om het daeghlijcks brood;
        Dat zal ook wezen // De rechte daad,
        Hij wordt geprezen // Die hier zo gaat:
        God geeft zijn zegen // Hem die zo doet,
        Loopt hem wat tegen // Het keert tot goed;
        Hij stuurt door zijn hand // De Lucht, Zee, en Land,
        Zijn milde Regen // Verheugt de plant.
