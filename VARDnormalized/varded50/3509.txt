
    
      
        ALs ik om rusten zacht // bedacht
        Gelegen was te nacht // in kracht
        Doen peynsden ik ydoone
        Hoe dat de wijsheid schone
        Tot mij kwam zelver gaan // zeer zaan.
      
      
        Laat u zinnen niet slecht // ghelecht
        Zijn tot een maagd in echt // oprecht
        Die ik u wel zal wijzen
        En weerdig is om prijzen
        Ziet dat gij deze maagd // behaagt.
      
      
        Ik sprak wijsheyts Goddin// de min
        Brengt ook veel droefheyts in // den zin
        Wist zij tot dezen tien
        Dat ik ze dacht te vrijen
        Zo waar mijn Liefde ziet // verdriet.
      
      
        Pallas sprak / opter daad // met raad
        Zal ik u zijn te baat // hoed gaat
        Ook al die eerlijk minnen
        Vyt reynder hert en zinnen
        Wie vrijt in eer en denkt // wind vreugd.
      
      
        Overschoone grein // zeer rein
        Thert denkt om u certein // alleyn
        Zo wel in het van u scheiden
        Als tsamen gaan vermeyden
        Die Lieffd' tot t'liefste kind // vast vint
      
      
        Prinslijcke Maget teer // vol eer
        Thert daechlijcx met begeer / strekt meer
        Na u wt Liefde trouwe
        Als tot enig Jonkvrouwe
        En wacht na tsoet propoost // van troost.
