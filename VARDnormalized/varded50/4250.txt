
    
      
        Krachtig gewont // den gront //
        Mijns herten ziet,
        Tonge, noch mond // bestond //
        Nooit drouver liet,
        Eylacen diet // mij heeft gedaan //
        Ziet mijn verdriet // zeer luttel aan //
        Mij zal ontstaan // zeer zaan //
        Een meerder last //
        Mijn lijden wast // noch vast //
        Waar zal ik gaan //
        Gevaen // ellendig gast.
      
      
        O droeuen dag // u mag
        Ik vloucken schier,
        Dat ik ooit zach // t'opslach
        Der Oogskens fier,
        Van t'vroulick dier // diet al passeert,
        Wiens goedertier // bij zijn falieert,
        Gans gematteert // doleert
        Al mijn verstand,
        Wie bluscht den brand? // niemand,
        Want diet moveert
        Mij keert // fluks van der hand.
      
      
        Rein zedebaar // volgt haar
        Der vrouwen // aard:
        Een gulden iaer // voorwaar
        T'aenschouwen // waart:
        In trouwen // baard // zij liefde heet,
        T'berouwen // vaart // henen gereed,
        Van alle leed // hoe wreed,
        En straf, het zij:
        T'vriendelick // bij // zijn, vrij
        Elk (met bescheet)
        Bekleed // bijzonder mij.
      
      
        Nooit tooghd' ik geen // onreen
        Dorper gelaat:
        Of dat te cleen // zulk een
        Beelde toestaet:
        Of haren haat // mij dus bevecht,
        Veracht, versmaad // onweerdich slecht,
        Mij wederleght // ontseght
        Een woord, hoe klein,
        Mijn hert is rein // certein
        Ik ben oprecht,
        Haar knecht // en geen vilein.
      
      
        Eylacen och // merkt doch
        Mijn cuysch opzet:
        Met schande, noch // bedrog,
        Geenszins besmet:
        Vroom heb ik met // liefde gewrocht,
        Naar S'heeren wet // te zijn verknocht
        Met haar, hoe mocht // verzocht
        Zijn meerder deugd?
        Mijn hert verheugt // des ieught
        Gans wtgesocht,
        Mij brocht // ooit d'ooghste vreugd.
      
      
        Laat zij dan mijn // gepijn
        Opwassen groot:
        Wat medecijn // zal zijn
        Voor mij ter noot?
        Dat mij de dood // niet en versmacht,
        Die mij verstoot // heeft meerder macht:
        Dus naar des klacht // verwacht
        ik, binnen // t'perck:
        Der zinnen // merk // die t'werck
        Dus heeft gewracht,
        Wt kracht // der minnen // sterk.
      
      
        Is haar bekent // den endt
        Van mijn getreur,
        En mijn torment // afwent:
        Haar straf rigeur,
        Weg dan malheur // vijandig gloed,
        Haar wreedheid (duur // mijn groot oodtmoet)
        Zal zijn versoet // sent spoed
        In dezen band,
        Cupido, want // dit pand
        Kost vlees, en bloed,
        Dus doet // mij onderstand.
      
      
        Aldus ik dan // hier van
        Make finaal,
        Op aarden kan // geen man
        Blussen den straal,
        Die teenemael // mijn hert deurviert,
        Dan speciaal // (wel gemanierd)
        Diet al regiert // en stiert
        Met zulk virtuyt:
        Al stoot zij uit // mij ruit
        Dus gehantiert,
        Versierd // blijft zij mijn bruyt.
