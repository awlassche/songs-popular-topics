
    
      
        V Minne gunst // is t’overkunst
        O gei,e Herderinnetjes,
        En t’ooghenblick // is als een strik
        Voor ons verliefde sinnetjes.
      
      
        Als ik in het groen // eens dertel zoen
        V nat bedaude lipjes,
        Zo vind ik mijn // gheseylt te syn
        Aan twee Corale Clipjes.
      
      
        Daar ik dan kom // tegen de stroom
        Van de ghebruysde baertjes,
        Zo dat ik voel // doort zoet gewoel
        Een beving in mijn aertjes.
      
      
        Ik steek het zweert // en span de gheert
        Nu styfjes dan weer slapjes,
        Dan klim ik op // tot byde top
        Van der Hooft-touwen trapjes.
      
      
        Daar ik de schoot // vast ande boot
        Beleg met strik en knoopjes,
        En deur het nat // maak ik weer rat
        De stramme overloopjes.
      
      
        En zien ik weer // als ik laveer
        Nu hoochjes dan weer laechjes,
        Dat ik niet win // zo vind ik in
        Een reefjen voor de vlaechjes.
      
      
        Tis toverij // ziel roverij
        En zoete Boereknaepjes,
        Als wij met u // in het Bosjen luw
        Gaan houden middag slaepjes.
      
      
        Daar gij met ons // den brand des Sons
        Ontschuylt in diepe holletjes,
        Daar gij ons sold // daar gij ons rolt
        Als kittelsiecke snolletjes.
      
      
        Het schijnt dat gij // gheen vrijerij
        Gebruikt, maar toverkunsjes,
        Dat gij kort // deelachtig wordt
        Ons lieve Minne gunsjes.
      
      
        En Coridon // mijn Goude Zon
        Ik haat al mijn Vriendinnetjes,
        Wanneer u beeld // zo haastig spelt
        In het diepste van mijn sinnetjes.
      
      
        Wanneer ik voel, het zoet gewoel
        En twemfelen uwes tonghetjes,
        Mijn hertjen teer // doet op en neer
        Veel Minnelijcke sprongetjes.
      
      
        Het nockt, het beeft // het lockt, het leeft
        Nu laechjes dan weer hoochjes,
        Het tuycht en spreekt // wat mijn ghebreeckt
        Deurt drijven van mijn ooghjes.
      
      
        Maar als ik wen // en beter ken
        Tussen door beide rotsjes,
        Zo peur ik an // gelijk een Man
        Tot spijt van Thetys trotsjes.
      
      
        Daar mijn de vloed // strackx drijven doet
        Tussen twee hoge baeckjes,
        Waar ik met vlijt // strackx buiten smijt
        Het Anker met twee haeckjes.
      
      
        En als mijn hert // heel quackjes werd
        Doort wellen van de stroomjes,
        Zo leg ik aan // een sloeg te gaan
        Naar de belade Boomjes.
      
      
        Maar Ameril  // wat coude gril
        Kruipt door mijn moedeleetjes,
        Als ik opt strand // ga deur het sant
        Met wanckelbare treetjes.
      
      
        Mijn dunkt de Zee // waar in ik tree
        Staech weld onder mijn voetjes,
        Dan vloetse zeer // dan ebtse weer
        En hippelt ook zeer zoetjes.
      
      
        Het klacht tot dat // ik weet niet wat
        Voor wonderlijke cluchjes
        Door ons geslacht // zo weerdig acht
        Verdrijven al mijn suchjes.
      
      
        O Coridon // mijn Goude Zon
        Hoe zijt gij nu dus stilletjes,
        Is dan u lust // zo haast gheblust
        Van u verhitte willetjes.
      
      
        Wiens ingewand // in het eerste brand
        Zo vierichjes, zo graechjes
        Als of de vreugd // van uwe Jeugd
        Sou duren duizend daechjes.
      
      
        Maar als terstond // u vast verbond
        En Ionckend zoete lachjes
        Verkeerden in // geveinsde min
        En trouweloose clachjes.
      
      
        Eerst vluchten wij // doe zuchten gij
        Met meer minne sanghetjes,
        Nu vlucht gij schuw, nu zoek ik u
        Met nat betraende wanghetjes.
