
    
      
        IN mijnen noot, roep ik tot u o Here,
        Laat mij niet bloot // want ik ben zwak en teere,
        Want ik doch niet // zonder u en vermagh,
        In dit verdriet // troost mij Heer nacht end' dag.
      
      
        Troost mij o Heer // en wilt mij niet verlaten,
        In dit verzeer // komt mij Here te haten.
        Op u alleen // betrouw' ik Here mijn:
        Wie mag ons scheen // van u liefd' op dit termijn?
      
      
        Vw' liefde zoet // bewijst Here gedurig
        Watmen ons doet // dat wij niet wezen truerigh.
        Door uwen geest // o Heer Daniel bystaet,
        In Babel meest // is hij bij die Leeuwen kwaad.
      
      
        Die Leeuwen fel // zijn ook in de woestijne,
        En zoeken snel // ons te brengen in pijne.
        En slangen fier // die spouwen vyerigh fenijn,
        O God zijt hier // onzen rechten medecijn.
      
      
        Geen kruiden zo goed // die men sou mogen lezen,
        Noch plaester zoet // en kan ons niet genezen,
        Maar Here gij die Israel bystont
        Door Christum vrij // maakt ons zielen ghesont.
      
      
        Ons zielen krank // zijn altijd in dangiere
        Door peste strang // en ziekten menigertiere,
        Zo dat hier vergat // menig met jammer groot
        Die haar gheen raad // zoeken tegen den dood.
      
      
        Zij beminnen meer // dat tijtelijcke leven
        En des werelds eer // van vlees en bloed gedreven,
        Zij zijn eilaas // zeer jammerlijk verblent,
        Als die Maagden dwaas // die door liefden zijn geschend.
      
      
        Zij zijn inder nacht // ook slaperig bedeghen,
        Als die gheen acht // op het licht hebben ghedreghen,
        Als den Koning haar // zal opwekken subijt,
        Dan zullen zij daar // kloppen maar ter ontijd.
      
      
        Dan zullen zij // beschaemt daar buiten blijven
        Dat zoeken daar bij // en zal haar niet beklijven.
        'Twert dan te spa // om na oly te zien
        Haar zal geen genae // vanden Bruyd'gom gezien.
      
      
        Och vrienden gemeen // wilt bidden ende waken,
        Omgort u leen // laat u licht vyerigh blaken,
        V zedigheid, laat wezen openbaar,
        Weest zo bereid // als wijze maagden eerbaar.
      
      
        Als een reine bruyt wilt u alzo versieren,
        Toont overluyt // u licht in all' manieren,
        Want het is nu deur // van den slaap op te staan,
        Door Christum puur, tot den Vader te gaan.
      
      
        Prinslijcke God // in eeuwigheid verheven,
        Dit voor een slot // mijn vriendekens gegeven,
        Die van het getal // zijn weergheboren nieu,
        Dit schenk ik u al // voor een vriend'lijck adieu.
