
    
      
        Cupido hoort mijn klagen,
        Op dit termijn,
        Bij nachten en bij dagen
        Lyd' ik gepijn:
        Mijn lief heeft mij begeven //
        Om het grote goed,
        Och Cupido verheven //
        Mij bijstand doet.
      
      
        Wilt mij uw bijstand tonen
        Vrouw Venuw kind,
        Mijn lief wilt mij niet lonen,
        Zoo nu het wel schijnt,
        Door die grote rykdommen //
        Word zij verblend
        Hierom is het dat ik kome //
        In dit torment.
      
      
        het Goed is de meeste cause
        Van mijn getreur //
        Dus beginn' ik myn' clause
        Met groot douleur:
        Liefde plagt te floreren //
        Aan elken kant,
        Maar het goed heeft nu te zeere //
        De overhand.
      
      
        Waar ik van goede rijke,
        Ik waar verheugd,
        Zij en zoud' van mij niet wijken,
        Die zuyv're jeugd,
        Ik zoud' wel troost gewinnen,
        Nu treuren ik moet,
        Want zij stelt all' haer' zinnen //
        Op het grote goed.
      
      
        het Goed neemt (laken) zyn' gangen,
        Als sneeuw op het veld //
        Daar zij al met verlangen
        Haar zin op stelt //
        Maar daar is liefde puere,
        Die zij klein acht,
        Groeyt van ure tot ure,
        Dag ende nacht.
      
      
        Wat mag ter weireld baten,
        Goed oft schat //
        Zij moeten het hier al laten,
        het Zij dorp ofte stad,
        Wat baat haar grond oft erven
        In het aardse dal?
        Zoo wel als zij moet sterven //
        Het blijft hier al.
