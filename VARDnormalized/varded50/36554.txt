
    
      
        O Mensen // die wensen
        Om tijdelijk goed, Daar de wormen
        hun bestormen, In Vlees en in Bloed:
        Pracht en staeten // Moeder laten
        Hier in des weireldsch dal, Rek'ning' geven,
        Van hun leven, Aan den Opper-Al.
      
      
        Daeromme // Gij domme
        Keert u doch tot hem, Dien de Joden,
        Eens ontboden // Tot Jeruzalem:
        Vals van herten // Tot duist smerten
        Van dat god'lyk Lam // Want hij peynsde
        Hoe het geveinsde // Volk hem halen kwam.
      
      
        Voor het laeste // Te gaste,
        Gaf hij zijn Lichaam // Aan zyn' vrienden,
        Die hem dienden // d'Apostels te zam,
        Tot gedenken // Gaf te drinken
        Zijn godelyk Bloed // Aan de Schare,
        Een van hare // Bleef noch heel verwoed.
      
      
        Den Here // ging zeere
        Na het huis van Annas, Daar ook kwamen
        Al-te-samen // het Volk van Cayphas,
        Die hem sloegen // En joegen
        Tot Pilatus Raad // Maar z'en vonden
        Gene zonden // In hem nocht' misdaad.
      
      
        Dan liepen // En riepen
        het Gemeente te gaar // Laat hem doden,
        Dezen snooden // Kruyst den tooveraer,
        Geeft uw oordeel // t'Onzer voordeel,
        Gij blijft des keizers vriend // Laat hem sterven,
        Kappen, kerven // Hij heeft meer verdiend.
      
      
        Daar stonden //Die honden,
        Die Joden bereid, Om t'onteeren
        Duizend keren // des Hemels Majesteit:
        En zij grepen // Vrede zwepen
        En ketens, en roen // Om zyn' letten
        Te verpletten // En pyn' aan te doen.
      
      
        Meer plagen // En slagen
        Heeft den Heer ontsaen // Als duist mannen
        Van Tirannen // Ooit wierd aangedaan:
        Bloed met stoopen // Zag men lopen
        Door hun vreed bedrijf // En zij trokken,
        Hem met brokken // het heilig Vlees van het lijf.
      
      
        Zij brogten // Gevlochten,
        Van doornen een krans // Die z'hem planten,
        te allen kanten // In d'herssens bijkans,
        En zij kwamen // Zoo te samen
        Uit Pilatus huis // Daar zij toonden,
        Den gekroonden // Aan het Jodsch gespuis.
      
      
        Zyn' smerte // Verherte,
        Noch meer hun gemoed // Zij verblydden,
        In zijn lijden // En wilden meer Bloed;
        Dies zij deden // Op zyn' leden
        Een zwaar kruysse laan // En daar naar,
        Den kalvaer // Berg is opgegaan.
      
      
        Die vreden // Ontkleden
        Den Heer hunnen God // Riet tot schanden,
        In zyn' handen // Gaven zij tot spot;
        Alsdan was hij // In zyn' Passy
        Nooit zoo zeer belaen // Dan ten toone,
        Met ene krone // Naakt en bloot te staan.
      
      
        Dan stak hem // En trok hem
        Dat Joden gespuis // En zij spanden,
        Voeten, handen // Nag'lend' hem aan het kruis.
        Van zyn' leden // Tot beneden
        Liep zijn dierbaar Bloed // Men zag leken,
        Als twee beken // Van zijn tranen vloed.
      
      
        Verslagen // Kwam klagen
        Maria bedroefd // God sprak, Vrouwe,
        Stelt uw rouwe // U wel troost behoeft:
        Zij zag hangen // Die ontvangen
        Had haer' zuyv're Borst // Daar zij gaven
        Om te laven // Galle voor den dorst.
      
      
        Door het steken // Bezweken
        Wierd hij van ene lans // Zon en Maan,
        Bleven staan // Verliezende hun glans,
        Hij riep: Vader // Wilt te gader
        Vergeven hun kwaad // Want zij weten
        Geen' secreten // Van hun groot misdaad.
      
      
        Zoo bad hij // Op dat hij
        De zondaars te zam // Mogte leren,
        Hoe elk eren // Moet zijns Vaders naam,
        Om voor deugden // Duizend vreugden
        Met Gods Majesteit // Naar het verdrieten
        Te genieten, Inder eeuwigheid.
