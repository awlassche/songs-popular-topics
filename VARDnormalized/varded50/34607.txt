
    
      
        O Zoete Nymph', die in mijn jong hert'
        Zo veel veghifs ghespouwen kont,
        Die mij staegh martelt met zo veel smert',
        En wederom gedurig wond;
        Daar nooit verdwijnen // de pijnen // die gij
        Eerst in mijne ziele // ded' krielen te lij,
        Maar vast'lijck blijven // en drijven // niet uit,
        Dan elk wel eerder // in meerder // besluit.
      
      
        Want als de vogh'len s'jaers altijd bro'en,
        Esen en kippen, de jongen we'er vlug
        Ook het zelve daar naar weerder doen,
        Zo gaeter seld' een wee te rugh';
        Maar, ziet, zij paren // en baren // altijd,
        En we'er vermeren // mijn seeren // bij tijd:
        Dus wildy dat 'k gh'nese // u wezen // om-keert,
        En mij van binnen // door het minnen // ontseert.
      
      
        het Is zeker, wanneer dat gij u vier
        Eens in mijn ziele blazen woud,
        Het soud' zo menighen, dat nooit dier
        Zo vruchtbaar is geweest in het wout,
        En liefde de plagen // verjagen // heel sou,
        En zo mijn wonden // ontgronden // van het zeer:
        Nu dan Goddinne // wilt minnen // ter trouw,
        En ik zal slaven // en draven// tot eer.
