
    
      
        T'Ghesichte, het eerste der vijf zinnen,
        Doet ons onderscheiden // klaar
        Gods werken al ter Wereld binnen,
        Die zijn Macht verbreiden // maar
        Nooit niemand mogen // heeft gewis
        God zien met ogen, zo hij is:
        Maar die tot // zijn gebod
        Keert met lusten,
        En volherdt in deuchden // vroed,
        Die zal God // t'saligh lot,
        Zien in rusten,
        T'wert een vreugd der vreugden // zoet.
      
      
        T'ghehoor is den weg daar 'tgheloove
        Door komt, uit des Heren // woord,
        Daar 'thert' opmerckigh, niet als dove,
        Recht Gods bode leren // hoort:
        T'ghehoor oft lezen // salight niet,
        Zelf doender wezen // moetmen, ziet,
        Met verdrag // dat op dag
        Van Gods oordeel
        Van ons als verblijde // fijn,
        'tSoet gewag // gehoord mag
        Zijn tot voordeel,
        Komt ghebenedijde // mijn.
      
      
        Den roke der zalvinge reine
        Van des Heren goeden // Geest,
        Die neder vloeyt uit s'deuchts fonteine
        Hier in den ootmoeden // meest //
        Elk Christen geven // moet ook uit
        Zoet rokigh leven // als de Bruid
        Van Sioen // zijnde koen
        Wtghesproken,
        Een kruidig bekwame // hof,
        Altijdts groen // in wel doen,
        Zoet van roken
        Geeft zij haar vreindts name // lof.
      
      
        De smake van Gods goedheid milde,
        En 'sHeeren woord goedertier,
        Twaer goed dat ieder daar aan wilde,
        Als kind dat de Moeder // hier
        Suyght om op groeien // met zulk zog,
        Daar niet in het vloeien // is bedrog,
        Alzo haak // na de smaak,
        Van de werde
        Heilige Schrifture // elk,
        Zulk gheen sak // tot vermaak
        Hier op eerde,
        Als des voedtsaem puere // melk.
      
      
        Gevoelen laat ons t'lyden smertigh,
        Dat hier onzen Naasten // dooght,
        Als Samaritaen die goethertigh,
        Hulp toe doen met haasten, pooght,
        'tQuaet laat ons mijden // met verstand,
        En te allen tijden // goed doen, want
        s'Heeren raad // ons wel staat
        Te vervullen,
        Zonder te verkoelen // vrij,
        Want naar daad // goed oft kwaad,
        Namaals zullen
        In het lichaam bevoelen // wij.
