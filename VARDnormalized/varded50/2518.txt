
    
      
        Mijn sinlijckheyd // heb ik geleid // op een die ik moet derven,
        Die mij onthoudt // hetwelk mijn zo rond // dat ik vaak wens om sterven:
        Niet dat ik haar // mijn Lief eerbaar // iet kwaads souw willen gunnen,
        Ja woud' ik al // (dat ik niet zal) 'k en souw voorwaar niet kunnen.
      
      
        Want d'oogjens blij // die zij tot mij // eertijds zo lieflijk keerde,
        Die heeft zij noch // waar door ik, och! haar hertelijck begeerde.
        En haar wel-vaert // is lief en waart // in mijn getrouwe zinnen,
        Souw het hart van mijn // wel konnen zijn // zonder haar te beminnen?
      
      
        Neen't, zeker neent // het zucht en weent // als het mijn Lief ziet lijen,
        En in haar vreugd // schep ik geneucht // en over groot verblijen,
        Want zij besint // die haar bemint // soo't onberoemt mag blijken,
        Geen last of noot // zal voor de dood // mij van mijn Lief doen wijken.
      
      
        Hoe soud' ik // ziet, beminne niet // die zo men mag bemerken;
        Is wijs van Raad // arrem van praat // en rijk van goede werken:
        Hier door het oog viel // op d'ed'le Ziel // van mijn Lief uitgelezen,
        In wien ik, ach! docht dat ik sach // het alder-schoonste wezen.
      
      
        Klein is mijn roem // of ik schoon noem // haar wel-geschickte zeden,
        Want d'Ouders die // gaan zo ik zie // haar aan Rijcker besteden.
        En die haar eerst // scheen wel begeerst // gaat vluchtig zij verlaten,
        Maar die aen-neemt // voor eigen, vreemd // laat zich door het geld bepraten.
      
      
        Het duivels Geld // heeft ook geweld // zie ik in des mensen ziele,
        Want die quaat doet // kan metter spoed // de Liefde licht verniele.
        Geen Liefd maar schijn // want ziet an mijn // of het mij Kan verkeren,
        Daar men Jonst koopt // hoe dat afloopt // zal ons de Tijd wel leren.
