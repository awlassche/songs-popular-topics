
    
      
        Ach dag! wenschelijcke dag,
        Vol schoon gheleghentheyt,
        Dat ik menselijke mag
        Met mijn gheneghentheyt
        Mijn Goddin doen bescheid
        Op het gheen dat zij mij zeid
        Deur haar praat // of in daat
        Hoe het gaat // in wat staat
        Mij het geluk rechte voort leit.
      
      
        Zijn dit wrevelinge // niet?
        Neent, droomend Fantasij?
        Fluks, weg revellinge // vliet
        En maakt u knap van mij,
        Weg droeve dampen gij,
        De schone dag naakt blij
        het Zuchten vlie // ik door die
        'K weet wel wie // dien ik zie
        Hier zitten an mijn groene zij.
      
      
        Sijt gij het, Egtillen // jent,
        Ik bid u dat gij spreekt;
        Zijn het weer dolle grille // blent
        Die de begeert ontsteekt?
        Mijn oogjens onbevlekt
        Zien u, mijn lief perfeckt,
        Ja voorwaar // schoon en klaar,
        En geen aar // het schort maar
        Aan u dat gij mij dit ontdekt.
      
      
        In sulcken cierelijcken staat
        Geloof ik nauw te zijn,
        Dijn zoet manierelijcke praat
        Die ik staech an u vin,
        Nochtans is dit geen schijn,
        Ik ziet voor d'ooghen mijn:
        Ik twijfel // met gekwel
        Of ik wel // die ghesel,
        Die ik thans was, wel souw zijn.
      
      
        In d'onbegrijpelijcke // vreugd
        Wentelt mijn wuft gedacht
        Te zien u rijpelijcke // denkt
        En jonst dus onverwacht,
        Dat ik mij gh'luckich acht
        Boven het mensch'lijck geslacht:
        Want Keizer braaf // Prins noch Graaf
        Heeft sulcken gaaf // als u slaaf
        Die het geluk zo zoet toelacht.
      
      
        Och of gij deelachtig // waart
        Die vrucht die ik geniet,
        Die in mij zo krachtig // vaart
        Dat mij het vernuft ontvliet,
        Laas! een dinck mij verdriet,
        Dat u gheen eer gheschiet
        Deur vreugden zoet // mijn ghemoet
        U niet groet // als het herte doet
        Dat gij naar wens ghebiet.
      
      
        Cleen souw de danckbaerheyt // zijn
        Indien dat ik verstond;
        Dat inder waarheid // fijn,
        Te spreken was goed rond
        Met menschelijcker mond,
        Die weldaad mij ghejont
        Die 'k wardeer // hoog en zeer,
        Eeuwig eer // na het begeer
        Ben ik, helaas! te zwak gegrond.
      
      
        Spreeck ik beroemlijck // iet
        Prinses mijn schuit vergeeft:
        Voorwaar onnoemelijk // ziet
        Syn u deuchden beleeft,
        U lof ten Hemel sweeft
        Daar het zonder twijfel leeft.
        Ik bid God // dat hij tot
        Volle slot // na het gebod
        Ons met trou liefd het samen kleeft.
