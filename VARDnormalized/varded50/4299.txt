
    
      
        Maagdelijk schoon gestichte //
        Deurschijnich da- geraet //
        Klaar amoureus gezichte //
        Die angst en noot verslaat //
        Rein saet //
        Gij gaat //
        In daad //
        Vesta te boven // wijt //
        In vreed' // en // akkoord //
        Gij heden // en voort //
        Zeer hoog te lo- ven zijt.
      
      
        Aanziet mij knecht der minnen,
        Als een die deugd hantiert:
        Den Chaos minder zinnen,
        Verwert, ongeregiert,
        Maniert,
        Scholiert,
        En stiert,
        Naar uwen reden // toom:
        T'wert stadig // begeert,
        Genadig useert
        Aan mij, dijn zeden // vroom.
      
      
        Rachel zijdy in het wezen,
        Naar dat d'estime // luid,
        Als Sara wt gelezen,
        O magnanime // bruid,
        Die wt,
        Zulk cruyt,
        
          Virtuyt
        
        (Tot zijn gezonde) // haalt:
        Geen smerte // verstranght,
        Zijn herte // ontvangt
        Troost, die ten gronde // daalt.
      
      
        Is het, dat deur u ontfermen
        Ik vreugd gebruiken // kan:
        Den lust-hof uwer ermen,
        Laat mij omluycken // dan:
        Geen man,
        Werd van,
        T'gespan,
        Der minnen // eer // gevrijt:
        Ben bange // bedrouft,
        Niet lange // vertouft,
        Maakt mij eens zeer // verblijd.
      
      
        Adieu zoete Princesse,
        Dijns houd ik goede wacht:
        Gij zijt een forteresse,
        Onwinnelijk bij kracht,
        Geen macht,
        Geacht,
        Gij slacht
        Minerva, gans // en // al:
        O fiere // divijn
        Lauriere // die mijn
        Hoofd noch becranssen // zal.
