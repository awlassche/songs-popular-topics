
    
      
        Here zijt ons hender // bij
        En die u geven // dank
        dat wij van die schender // vrij
        Doch zijn ons leven // lank
        Dat hij niet // wat hij ziet
        en bespiet // in het verdriet
        Of houdt in zijnen dwanck.
        Dat wij Vader // vol genaden
        Altydt nader // in u paden
        Zijn bevryt // van den nyt
        En den strijd // worden quyt
        Dat naar mag zijn jolijt.
      
      
        Den mens van God geschapen // is
        Heilig goed groot // geacht
        door sDuyvels betrapen // wis
        Die heeft de dood // bewracht
        Doen hij plat // overtrat
        En ook wat // daar van at
        En hield zij geen goe wacht
        Want een bevel // Gods der goden
        Zij wistet wel // twas verboden,
        Dat zij wan // alleen van
        Den Satan // was den ban
        De dood die kwam haar an
      
      
        Alzo in doods gewelde // kwam
        De Mensen alle // gaar
        Den Held alder Helden // nam
        Ons ui den valle // zwaar
        Dien Heiland // heeft vermant
        Den Vijand // met der hand
        Deilt uit zijn roof aldaar
        Den Leen Juda // heeft verwonnen
        Dat goed' noch qua // doen en konnen
        In ootmoed // kwam hij zoet
        Wat hij doet // tis al goed
        Verlost ons met zijn bloed
      
      
        Dits den rechten Herder // fijn
        Die zijne Schapen // voedt:
        Daar en kan geen werder // zijn
        Die ons vant slapen // hoedt
        Hij kan leen // dat niet een
        Groot noch kleen // en moet scheen
        Tot sWolfs gapen // verwoed
        Dan alleen dat // kind verloren
        Daar David wat // zeid te voren
        Twas Judas // dat boos aas
        Dat eilaas // als een dwaas:
        Verriet zijn eigen baas.
      
      
        Prins alder Princieren // stam
        Van hoger werden // fier
        Als een goedertieren // Lam
        Kwaamt gij op Eerden // hier
        Hebt gehaelt // dat gefaelt
        Was verdwaald // en betaalt
        Met uwen bloede dier
        Den ingang tot // d'eeuwigh ruste
        Is van God // gemaakt tot luste
        Hij wil schoon // elk persoon
        Zeer ydoon // tot haar loon
        Zinken des levens Kroon.
