
    
      
        RAs neemt vertrek  gij duister nacht //
        Licht op o dag van vre- den,
        Die heden // zal hier bij // den rei,
        Beschijnt nu de  lieflickste dracht //
        Volmaakt van lijf en le- den,
        Vreugdig bringen den blij // den // Mei,
        Haast u gij jongers komt inne //
        Voor mijn schoon Koningin ne.
      
      
        Eeuwig blijven verwonnen // gans
        Haar vijanden verseerlick:
        Haar aanschijn (als der zonnen // glans)
        Blinct in den morgen heerlijk,
        Eerlijk // draagt zij ter wereld, hier
        Klederen net bepeerelt // fier,
        Een gulden tak draegh ik wt minne,
        Voor mijn schoon Koninginne.
      
      
        T'haerwaerts is geheel t'lant // gehelt,
        Om breken fantasie:
        En Orpheus heeft zijn hand // gesteld
        Tot spel, vol melodie:
        De blie // Bergen, springen // hoog,
        De Nachtegaal in het zingen // vloogh,
        Met liedekens, zoet van gewinne,
        Voor mijn schoon Koninginne.
      
      
        Ontdekt u, Eesters velden // wijt,
        Sent herwaarts Violieren:
        Ontspringht die kloeke helden // zijt,
        Croont haar met schoon Laurieren:
        Versieren // wilt u feestich // dan,
        Elk roepe als een geestig // man,
        Buigt Iuno hemelse Goddinne,
        Voor mijn schoon Koninginne.
      
      
        Rieckende bloemkens abundant,
        Verlicht nu de contreie,
        De liefste coemt zeer triumphant,
        Plukken den blijden Meye:
        Elk spreye // cruyt, en bladen // klein,
        Op haar wegen, en paden // rein,
        Thooght u Flora, o gesellinne,
        Voor mijn schoon Koninginne.
      
      
        In dezen Mei vol iunsten, blij:
        Schinct haar schone presenten:
        Een lid der seuen cunsten // vrij,
        Is zij niet om verjenten,
        Haar tenten // ik versieren // moet,
        Coemt maagden met manieren // zoet,
        Verschijnt elk als een Cherubinne,
        Voor mijn schoon Koninginne.
      
      
        Koningen, Vorsten, Prinsen // wreed,
        Buigt u, voor mijn Deze:
        Alle Landen, Provincen // breet
        Houden haar als Princesse,
        Expresse // liefkens onder // zet
        Den Mei, voor mijn lief zonder // smet:
        Gemerkt ik haar altijd bekinne,
        Voor mijn schoon Koninginne.
      
      
        Adieu, edel gecroonde // ieught,
        Retorica met name:
        Neemt doch deze gethoonde // deugd,
        T'uwer eer' zonder blame:
        Eersame // doch faveurich // zijt,
        V dienaar, die doleurich // stijt,
        Zo houd' ik u met hert, en zinne,
        Voor mijn schoon Koninginne.
