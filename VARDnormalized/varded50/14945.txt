
    
      
        EEn vreemde Maar // ik u bringhen zal,
        Doch schier ruchtbaar // acht ik het over al:
        HET is dat een Maagd // een zeer aanneming Dier
        Haar Eer beclaaght // door het bekende vier.
      
      
        Wiens hoofd cieraat // uytvluchtich stond gezet,
        Trat over straat // trots als een Pauw te net,
        S’is nu geschend // neer leit haar hoge moet,
        Zulk loon geeft t’ent // haar die het zelfde doet.
      
      
        Ach blinde Min // en wulpsheid onbedocht,
        Gij hebt haar zin in doolinghe ghebrocht,
        En ‘thert verleid // tot des begeerden snood,
        Door welks feit // haar ere is ontbloot.
      
      
        Wie had gemeend // dat een zo schoon gelaat
        Haar sou vercleent // hebben en het best versmaat,
        Ja het alderbest // geacht ter Wereld wijt,
        ‘tWelck zij op’t lest // helaas! te laat becryt.
      
      
        Geen Maagd in stee // bij des mocht zijn geacht,
        Haar Kledinge // was uytmuntende dracht,
        Sierlijk van stof // ghestrickt , gekoord, ghesneen
        Volgden soo’t Hof // meer als de anderen deen.
      
      
        Wat is het nu weer // of sijt schoon alles heeft
        Zij mist haar eer // daar t’alder meest aan kleeft,
        Geniet een lot // dat ieder haar begeckt
        Nu hem de pot // dus schandelijk ondeckt.
      
      
        Waardig waar zij // noch ere aan gedaan
        In dien zijt vrij // des feit hadden bestaan,
        Niemand en zout // acht ik haar spreken sant
        Want zij ghetrout // mocht werden na der hand.
      
      
        Maar nu eilaas // is het te beduchten wel,
        Dat doort geraas // van des geruchten snel,
        Haar niemand nu // aenspreecken zal op denkt
        Hier aan gij u // o Maagden spiegelen meucht.
      
      
        Cleen lof hij wind // bij die vroom zijn van aard,
        Dien vroom mans Kind // niet te bedriegen spaart,
        Maar steelt als dief // in schijn van ware trouw
        T’hert uit syn lief // en laatse na in rouw.
      
      
        Doch een vroom Prins,, alsulcke daden // vliet
        En tracht geenszins // iemand te schaden // iet,
        Maar dient als Knecht // die hij van herten meent,
        Cuysch en oprecht // tot dats hem God verleend.
