
    
      
        LOf prijs en eer, zij v O Heer // en milde Vader
        Die voor den morgen, niet te zorgen, hebt bevolen,
        Dewijl gij alleen // En anders gheen // ons allegader
        Lijf en Ziel voet // ende doet // honger en dorst vlieden
        Voor de weldaden // en ghenaeden, die ons geschieden,
        Eyscht gij maar // Een hart dankbaar // willig en bereid
        Dus te allen tij // Motten wij // Ook alle lieden,
        God hier boven // dankelijk loven // inder eeuwigheid.
