
    
      
        O Roosgen root // o peerle jent
        Liefgen minjoot // daar ik om lijd' torment,
        Schone Godinne ik blijf geschend,
        Want mijn lijden is zonder ent,
        Nacht ende dag, doe ik geklag, voorwaar,
        Om dat ik scheyd' van v, o maagd eerbaar.
      
      
        Moet ik nu scheiden // is het nu den tijd
        Zonder verbeiden // ofte respijt
        Mijn hart van droefheid // in het lichaam split
        Mijn oogskens ik in stikken krijt
        O scheiden fel // die in mijn gekwel // doet aan,
        Ken weet hoe dat icket zal konnen bestaan.
      
      
        O wreede dag // ongheluckighe uur,
        Dien ik ooit sach // ende gij valt mij zo zuur
        Want t'dunct mij te zijn tegen natuur,
        Dat ik nu moet o maechdeken puur,
        Zeggen adieu // op een nieu // ik blijf in rou
        O bittere dood waar na zo beydy non.
      
      
        Princesse schone // dit wel versint
        Gij spant die krone // en zijt bemint,
        Want ik ben niet gelijk den wind,
        Gij staat altijd in het harde gheprint
        Ik zal ghetrou // o schoon kersou // nu zijn,
        Al valt mij t'scheyden alzo grote pijn.
