
    
      
        O Here God van machte // groot
        Voor u zijn mijn gedachten // bloot
        Daar en is niet zo ergen // man,
        Die hem voor u verbergen // kan
        Want gij binnen en buiten // ziet
        Men kan voor u opsluiten // niet.
      
      
        Tis voor u als den lichten // dag
        Geeft in mij dat stichten // mag
        Enen nieuwen bekwamen // geest,
        Met al die uwen name // vreest
        Dat ik mag hier beneden // plein
        Wandelen in der vreden // rein
      
      
        Zonder de vrede zeer dier // baar
        En zal men mogen zien hier // naar
        God in zijn eeuwig wezen // fijn
        De knecht zal niet geprezen // zijn
        Die met onvrede kwellen // gaat
        En zijn medegesellen // slaat
      
      
        Met stok ofte met roeden // stijf
        En meen ik op dat goede // lijf
        maar met der tongen haasten // praat
        Spreken van zijnen naasten // kwaad
        Zo met der tonge griefde // rat
        Maar niet en leert de Liefde // dat
      
      
        Al trok gij ook een blodich // mes
        Tegen u vriend die nodig // es
        En u goed principalick // riet
        Zo doet gij noch zo kwalijk // niet
        Als dat gij met versmaden // maakt
        Hoe men in kromme paden // raakt
      
      
        Die zijnen mond tot laster // wendt
        Den vre hij veel tonvaster // kent
        Want na Syrachs gewagen // goed
        Hij doet van al zijn dagen // boet,
        Die hem tot kwalijk spreken // geeft
        En zelven vol gebreken // leeft
      
      
        Here bewaard mij in de // pat
        Naar die reine beminde // stad
        Mijn tong' en mij beneven // dien
        Dat ik hier na mag tleven // zien
        En goede dagen mede // voort,
        Och! Here mijne bede // hoort
      
      
        Heer mijn gebedt verhoren // wilt
        Dat daar door uwen toren // stilt
        Over alle mijn zonde // boos
        Door u genade gronde // loos
        Vergeeft mij dit altsamen // vroed
        Want ik mij voor u schamen // moet
      
      
        Laat mij Here voor u vrij // gaan
        Van zonden als den Publy // kaan
        Die voor u in den Tempel // bat
        Geeft mij na dat exempel // wat
        Dies gij na u verklaren // vast
        Kwijt maakte van den zwaren // last.
      
      
        Of ik u zo behaagde // reen,
        Here gelijk als Maegde // leen,
        Doen zij verkoos dat beste // deel
        Sold gij haar kwijt de reste // heel
        Der zonden tot verzoeten // dies
        Zij met tranen u voeten // wies.
      
      
        Veel biddens zij met tranen // de
        Ook dat vrouken van Cana // ne
        Zij bad met groot betrouwen // snel,
        Voor haar dochters benouwen // fel
        Ootmoedig en van herten // leech
        Van u zij haar begheerte // kreeg,
      
      
        Haar dochter deed gij Heer bijstand
        Zij was gekweld van den vijand
        Deed haar veel persequeren // an
        Maar gij zacht dat verneren // van
        sMoeders gelove machtig // sterk
        Zo toonde gij u krachtig // werk,
      
      
        Zeer veel krachtige daden // zoet
        Here gij deur genaden // doet
        Wie tot u met gelove // kwam
        Melaatse, kreupel, dode // lam,
        Den blinden sach, den stommen // sprak
        Gij hebt van haar genomen // tpack
      
      
        Och! Here God bermhertich // vaar
        Den man die achtendertich // jaar,
        Opt Bedd hadde gelegen // krank
        Hebt gij kloekheid en degen // ganck
        Wt liefden hem goed rond // ghejont,
        En maakte hem terstond // Gezond.
      
      
        Princelicke God devijn // sijt mijn
        Een eeuwig medecijn // met wijn
        En Oly mij inwendig // bint
        Waar dat ghyt Heer elledich // vindt
        In mij een geest verslagen // schept
        Daar in gij een goed behagen // hept.
