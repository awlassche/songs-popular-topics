
    
      
        DEs Werelds pronk is als een water-bel,
        Die ik zeer schoon den mens voor ogen stel:
        Olyme bedroefde Vrouw / u zielens rouw,
        Klaeght vrij aan de Goon / gist'ren stond gij schoon,
        Gist'ren stond gij schoon / verheven,
        En als een Bel gij nu moet sneven:
      
      
        Helaas wat baat mij nu de Egypsche kroon?
        Wat glory ken Alexandrias troon
        Nu Cleopatria zijn // meer als fenijn,
        Nu ik in de hand // Cesars mijn vijand,
        Nu ik in zijn hand moet glyen,
        Door het moedig en manhaftig stryen.
      
      
        O gij Jupijn verheft u Donder niet,
        Ooghd' op mijn staat, en pynelijck verdriet.
        Voorkomt de mijnen val // gij groten al //
        Ei straft mijn niet meer // ziet u Maget neer //
        Ziet u Maget neer gebogen //
        U treurig smeken om medoogen.
      
      
        Medoogen hebt bid, ik met een slavin,
        Die wel eertijds was des Werelds Monarchin:
        Medooghd' o Hemel mij // dat bidden wij //
        Spaat dit machtig rijk // mijn volk al gelijk //
        Spart dit machtig rijk // o Goden //
        Roeyt uit die ons pooght uit te roden.
