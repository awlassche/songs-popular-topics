
    
      
        Ik arme kind // waar zal ik keren
        Dan mij verneren // onder Gods hand
        Dat mij den wind // der valscher leren
        Niet en mag deren // Heer doet mij bijstand.
      
      
        O Heer God Vader // vol van genade
        Van alle kwade // mij toch bewaard
        Gij Fontein ader // vloeyt vroeg en spade
        Op dijnen zade // van dijnder aard.
      
      
        O God doet mij // veriubileren
        Ik zal lauderen // v name groot
        Al soumen vrij // mij persequeren
        En tribuleren // tot inder dood.
      
      
        Zal ik niet haast // met v verblijden
        In witter zijden // O Vader ient
        Tleuen verbaasd // is niet dan strijden
        Tot allen tijden // druk en ellent.
      
      
        Tot v O Here // wil ik mij genen
        Mijn steen verheuen //  mijn toeuerlaet
        Want in v lere // vindick het leun
        Hoe soudick beun // als gij mij bystaet.
      
      
        Ik bid v seere // om uwe gracy
        Tot elker spacy // verhoord v bruyt
        Vertroost v teere // door iubilacy
        Als swerelts nasi // zal roepen luid.
      
      
        Als haar de mensen // zeer verblijden
        Jagen en rijden // na swerelts loop
        Dan is mijn wensen // en mijn belijden
        Wilt mij beurijden // voor sulcken coop.
      
      
        Na v allene // zijn mijne gangen
        Ik ben behangen // met zonde snood
        Na anders gene //  staat mijn verlangen
        Wilt mij ontvangen // Heer inder noot.
      
      
        Zal des turbacy // noch lange duren
        Dit doet mij trueren // troost mij O Heer
        Want door blamacy // zij veel vervuren
        Door het verzuren // in uwer leer.
      
      
        Zo wilt bewaren // sparen en wachten
        Mij door v krachten // mijn  middelaar
        Wilt mij vergaren // leiden en achten
        Bij die geslachten // onder toutaer.
      
      
        God mijn herder // doet mij toch groeien
        Door liefde bloeien //  als rank vruchtbaar
        Ik weet geen weerder // wilt mij besproeien
        Laat op mij vloeien // v geest eerbaar.
      
      
        O gij Fonteine // vol goeds geprezen
        Daar wt gheresen // is tleuen ient
        Gij kont alleyne // ons ziel genezen
        Wilt bij ons wezen // tot in den ent.
      
      
        Nu bid ik  Prinse // mijn God almachtig
        Weest nu toch achtig // v schaepkens al
        O Heer, bemintse // wt liefden krachtig
        Weest ons gedachtig // in het iammerdal.
