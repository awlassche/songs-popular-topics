
    
      
        COrrect schoon beelde // rein,
        Volmaectste onder vele:
        Och! t'is een weelde // klein,
        Daar gij zijt buiten spele:
        Zo gratieus
        Zijt gij, O schoon vriendinne:
        Ten was beginne
        Geen maagd zo amoureus.
      
      
        Om u (met zinnen // recht)
        T'sijn dienstbaar, vrough en spade,
        Ben ik der minnen // knecht,
        Subiect uwe genade:
        Geheel constant,
        Wacht ik, zeer pacientich
        (Naar t'lijden tormentich)
        Den troost van uwer hand.
      
      
        Ras tot gezonde // helpt,
        Naar u genadig plegen:
        Mijn bitter wonde // stelpt,
        Die gij mij hebt geslegen:
        Gelijk gij snel
        Mij cont wonden, en grieven,
        Kan u wel belieuen
        Haast blussen mijn gekwel.
      
      
        Neemt in onweerde // niet
        Dit mijn ellendig klagen:
        Wist ik op d'eerde // iet
        Lief, dat u mocht behagen,
        T'soud' zijn verzocht,
        Het zij publijck, of stille,
        Tot dijn goeden wille
        Geheel ware volbrocht.
      
      
        Een zulk imagie // zoet
        Zal ik ten fijne // wachten:
        Deur u couragie // doet
        Gij mij geen pijne // achten:
        Bezwijken // straf
        Zal eer al s'weerelts trouwe,
        Dan ik u Jongvrouwe
        Een voet zal wijken af.
      
      
        Laat zulk dienstbaarheid // groot
        Niet onvergolden blijven:
        T'is inder waarheid // noot,
        Naar tuigen dit mijn schrijven:
        Liefde en smert
        Geen minnaar, tot versijcke
        Alst al met gelijke
        Liefde vergolden werd.
      
      
        Is mij in dezen // graat
        Ontseyt dijn troost accoordigh:
        De dood in het wezen // staat
        Mij altijd tegenwoordig:
        Mijns levens draad
        Werd haastig af gesneden,
        Atropos eyndt heden
        Mijn leven desolaat.
      
      
        Verthert hier inne // boos
        Dat gij beangst, en drukkig,
        Dijn knecht laat sinne loos
        Sterven, dus ongelukkig:
        Met sulcken vrueght
        Als gij mij doet dit lijden,
        Sterf ik (met verblijden)
        Voor u, O zoete Jeugd.
      
      
        Schone Princesse,, zaan
        Zal einden al mijn treuren:
        Ziet als meestresse // aan
        Mijn hert in het midden scheuren,
        Dwelck zonder schuld,
        Hem vrolijk geeft te pande,
        Op dat nu d'offrande
        (Naar heysch') werde vervult.
