
    
      
        OP op mijn geest // en wakker weest,
        Om mijn gedacht te stieren.
        Al wat men eert // ter Wereld leert
        V bet mijn Lief te vieren.
        Want alles zwicht // het voor het licht
        Dat ik moet eeuwig minnen.
        Voor het enig goed // dat mij voldoet
        Mijn al te kiessche zinnen.
      
      
        Het grof geslacht // des vollicx acht
        Van andere waard te bogen.
        O domme tijd // die mij verslijt,
        Gij hebt noch brein, noch ogen!
        De gheen die gij // hoog in waardij
        Verheft, en zijn gheen Vrouwen.
        Of zijn het, ik kan // de mijne dan
        Voor een Mannin niet houwen.
      
      
        Die ghylie viert // zijn wel versierd
        Met schoon en aerdicheden;
        Maar bij den dag // en (dunctme) mag
        De claerste nacht niet treden.
        De mijne toog // (want jong zij sok
        Heltinne moeders spenen)
        De wijze denkt // die het al wel veucht
        In het merrech van haar benen.
      
      
        Wie boven al // op d'aerdsche bal
        Doorluchtig is verschenen,
        Wies deur gevaar // en kommer, daar
        Veel' in versmachten, henen.
        Geen mensen hand // mijn dierste pand,
        Wt grousame' ongeloken,
        Maar loutre wacht // van hoger macht
        Des Hemels, heeft getrokken.
      
      
        Den Hemel droeg // wel laat en vroeg,
        Met reden, zorg geladen,
        Voor het schoonste waart // daar hij vergaard
        In had, zo veel cieraden.
        Van ieder daad // die hij aenvaet,
        'Tmisvoeghen is ghebannen:
        Zij zingt of praat // zij lacht zij gaat
        Een strickjen staat // gespannen.
      
      
        Een strickjen daar // verwarren haar
        In, al die kennis dragen.
        O wonder hoog! // O werelds ooch!
        O glans van onze dagen.
        De klare Zon // niet op gaan con,
        Zo gij u aanschijn heelde:
        Alleen want hij // verrijst, voor dij,
        Mijn lief, mijn licht, mijn weelde.
      
      
        voor u de May // nu maxels fray
        Versierd, van kruid en bloemen.
        Maar als gij scheid // uit menschlijcheyt,
        Zal het aan u staan, te noemen
        Wat, dat gij wilt // of Pallas schild,
        Of Iunoos scepter dragen,
        Op dat u wijk // de suyverlijcke'
        Venus uit haar wagen.
