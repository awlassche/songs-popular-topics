
    
      
        Minnelijk schoon // gelaat,
        Ranke mijns leuens weerdig:
        In Venus troon // bestaat
        Gij, als een blomken aardig:
        Mag mij (tot lof)
        Dijn schoonheid eens gelukken,
        Ik zal v af plukken
        Voor t'schoonste in den hof.
      
      
        Aerdiger gene // maagd
        En was, die smerten bluste:
        Dies gij allene // draagt
        T'confoort, der herten // ruste,
        Deurschoten // lief
        Heeft mij u zeed-baer wezen,
        Sent mij doch t'genesen,
        In een gesloten // brief.
      
      
        Ingelick weder // paar,
        Voor zwaar tempeest, en donder,
        Zijt gij den Ceder // daar
        Mijn ziel zal rusten onder:
        Der zonnen // schijn,
        Passeert gij in het gehele,
        Valt gij t'mijnen dele,
        T'sal veel gewonnen // zijn.
      
      
        Keert dan der pijnen // vloed
        In perfecte gezonde:
        Want gij van mijnen // moet
        T'secreet weet, tot den gronde,
        En kleeft altijd
        Op thert, als krachtig zegel:
        O volmaeckten spegel,
        Die zonder schaduw' zijt.
      
      
        Een zulk behagen // vast
        Draak ik u Jeugd // ter // eren,
        Geheel der plagen // last
        Cont gij in vreugd // verkeren,
        Ter schanden // laat
        V dienaar niet bederuen:
        Mijn leun, en steruen,
        In uwen handen // staat.
      
      
        Nobel deze // zoet,
        In dank, wilt dit aenvaten,
        Mijn schoon Princesse // moet
        Gij zijn, zonder aflaten:
        Hier mede // schrijft
        Mijn hand, dat u ontmoeten
        Honderd duizend groeten,
        En voorts in vrede // blijft.
