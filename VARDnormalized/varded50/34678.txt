
    
      
        NV Coridon schept moet,
        Men gaat den Peys verkonden;
        Den Vrede-maecker zoet,
        Die is ons af-ghesonden.
        Op dezen blijden nacht,
        Niet verr' van Davids stê,
        Messias heeft gebracht
        Den langh-ghewenschten Vrê,
        Hij geeft ons peys // Voor deze reis:
        Voor deze reis // Hij geeft ons peys,
        Peys, peys, peys, peys //
        Voor deze reis.
        Peys, peys, peys, peys //
        Voor deze reis.
      
      
        In het stal van Bethleëm,
        Te middernacht geboren,
        Geeft uit een kinderstem,
        Die God was van te voren:
        Hij ligt op stro verrot,
        Die is dat Vaders-Woordt;
        Om tussen ons en God
        Te maken een akkoord.
        Hij geeft ons peys // Voor deze reis:
        Voor deze reis // Hij geeft ons peys:
        Peys, peys, peys, peys //
        Voor deze reis.
        Peys, peys, peys, peys //
        Voor deze reis.
      
      
        Voor Adam, en zijn zaad
        Komt hij de schuld betalen,
        En op zijn schouders laat
        Al onze straffen dalen:
        Den Vader valt hij vrij
        Voor ons misdaad te voet,
        Voor onzen peys wilt hij
        Vergieten ook zijn Bloed.
        Hij geeft ons peys // Voor deze reis:
        Voor deze reis // Hij geeft ons peys:
        Peys, peys, peys, peys //
        Voor deze reis.
        Peys, peys, peys, peys //
        Voor deze reis.
      
      
        O Moyse gaat het zo !
        Hoe moeten wij hem minnen,
        Die ligt op hooi, en stro:
        Laat ons den stal gaan binnen,
        En kussen hem een reis
        Aan zijnen roden mond;
        En zingen peys, peys, peys
        Voor eeuwig in het Verbond.
        Nu kust hem peys // Noch deze reis:
        Noch deze reis // Nu kust hem peys;
        Peys peys, peys, peys //
        Noch deze reis.
        Peys, peys, peys, peys //
        Noch deze reis.
