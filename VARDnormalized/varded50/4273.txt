
    
      
        Minnelijk beeld,
        Bloemken van genre // zoet,
        Die boven kracht vermeught:
        In uwen teelt
        Gij haastig t'suere // doet
        Veranderen in vreugd:
        Gij zijt een Jeugd
        Zo delicaat,
        Daar niet dan deugd // en ere in bestaat.
      
      
        Aerdiger geen
        En was in honderd // iaer,
        Daar meer troost bij geschiet:
        Die mijn geween
        Gans wtgesondert // zwaar,
        Zo haastig bringht te niet,
        Nu gij u biedt
        Behulpig zeer,
        Hoe kan verdriet // mij deren immermeer:
      
      
        Rustig ghesint
        (Vrij aller plagen // last)
        Ben ik, bij uwen raad,
        Te zijn bemint
        In u behagen // vast,
        Schat-ik den hoogsten graat:
        Faelgeert u daad
        Op dezen // dag,
        Geen toeverlaat // op aarden wezen // mag.
      
      
        In het herte gij
        Mij als een plante // draagt,
        En hebt, van aanbegin,
        S'gelijcx ik dij
        Als een constante // maagd,
        Prente in mijnen zin:
        Zulk een gewin
        Houd-ick in prijs,
        Als t'wesen in // der Goden Paradijs.
      
      
        Adieu, hier met
        Moet het gescheiden // zijn,
        Princesse excellent:
        Wilt (met opzet)
        Jonstich verbeiden // dijn
        Dienaar zeer diligent,
        Die niet en kent
        Vreugd, of jolijt,
        Dan u present // te wezen te aller tijd.
