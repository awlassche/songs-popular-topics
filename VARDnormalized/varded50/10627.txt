
    
      
        CVpido God, deur uwe minne stralen,
        V scherp gebod // doet mij in liefden dwalen,
        Ik ben versodt
        Op een bruin smaechdelijn smalen
        Die met mij spodt.
      
      
        Venus bestier // vint ik zeer dubbelt van treken
        Der minnen vier // heeft zij in mij ontsteken
        Die liefste Rozier
        Ik machse zien noch spreken,
        t'is groot dangier
      
      
        T'is groot verdriet // de Liefste moet ik derven,
        Die trouwe biet // en niet en kan verwerven
        Om dat zij v liet,
        Moet ik van rouwe sterven
        In het meer geschiet.
      
      
        Haar zoet samblant // was mij een medecijne
        Zeer triumphant, Plag ik bij haar te zijne
        Die liefste playsant
        laat mij in grote pijne
        Aan mijn verstand.
      
      
        Als ik aensach // haar vriendelijcke ymagie
        Nacht en dag // schiep ik in haar coragie
        Dies ik wel mag
        klagen om haar vysagie
        Zonder verdrag.
      
      
        Zonder confoort, laat zij mij nu in weyne,
        Dus wij akkoord // hielden in het gemeine
        Dus roep ik moord,
        Over mijn lief alleyne
        die mij verstoort.
      
      
        Adieu eilaas // Adieu Princesse schone
        Ik was te d'waes // Te vrijen v persoone,
        Dies ik nu raas
        Dat ik mij zelven zo hoone,
        op dit relaas.
