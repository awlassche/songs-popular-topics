
    
      
        Kan wel een minnaers herte //
        verzwijgen al zijn smerte //
        zijnde in zorgen // groot //
        O neen, t'moet somtijds spreken //
        Of in quartieren breken //
        Deur den verborgen // noot.
      
      
        Alzo is ook mijn klagen
        T'versoeten van veel plagen,
        Welk deel noch seerder // wast,
        Te lijden (zonder woorden)
        Zulk heimelijk vermoorden,
        Waar vintmen meerder // last.
      
      
        Ter dood lijd' ik ontschuldich,
        Nochtans alzo verduldig,
        Dat menig tonge // kwaad
        Wiste, hoe patientich
        Ik drage t'iock tormentich,
        Ten waar niet onge- // haat.
      
      
        Hoe zwaar is zulk verzuchten,
        Als een belemmert duchten
        T'mergh wt den gronde // heft,
        T'moet zijn dat (uiter wete)
        Cupido in het secrete
        Hem menig wonde // treft.
      
      
        Aan wien zal ik verklaren
        Mijn inwendig bezwaren,
        Twelck nodig zwichten // heeft:
        Verbergh' ik al mijn pijne,
        Plaester noch medecijne,
        Mij geen verlichten // geeft.
      
      
        Ras ik mijn klacht beginne,
        Aan u, mijn schoon vriendinne,
        Van dezen bangen // strijd,
        Moet u t'secreet behanden,
        Of heimelijk verbranden
        Moet ik, eer langen // tijd.
      
      
        Is mij t'herte deursteken,
        Benaut, bijkans bezweken,
        Al waart verslegen // dood:
        En menig wonde // bleven,
        Gij cont gezonde // geven,
        Voor druk, en tegen // stoot.
      
      
        Neemt gij mij in genade,
        Geen hulpe coemt te spade,
        V seedbaer wezen // zal
        Mij (van t'lastich verwegen)
        Hoe heimelijk verswegen,
        Haastig genezen // al.
      
      
        Aanziet in mijn vertrekken,
        Mijn vrijmoedig ontdekken,
        Aan u meestresse // fijn,
        Der banden die mij drukken,
        Dus wilt (tot mijn gelukken)
        Een rein Princesse, zijn.
