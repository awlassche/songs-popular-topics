
    
      
        Musica zoet // gesanck, klank, klaar end’ hel,
        Wie dat die hoort // doet zij verblijen
        Hert ende moet // door t’lieflijck snaren-spel,
        Men zoet akkoord, vol melodijen,
        Met harmonijen // zij al wat leeft verkeucht
        T’gheclanck // van sanck // maakt goe partijen,
        Muziek, Muziek, Muziek volmaakt die vreugd.
      
      
        Van het geslacht // Jubal // is Musica
        Wt wien dat spruit // const sonderlinghen
        Syrenes zacht // in het land Anthenusa
        Met zoet geluid // in het water springen,
        Die haar hoort zingen // met lieffelijcker stem,
        Niet voor // t’ghehoor // die druk kan dwingen,
        Muziek/ Muziek/ Muziek brengt vreugd met hem.
      
      
        De edel Muziek // is een Meestersse jent
        Waar Leek en Klerk die const beminnen,
        Want haar pratijck, is eerbaar excellent,
        Haar lieflijk werk // men zal bevinnen,
        Gesteld met zinnen // op moet noot, ende toon,
        Den mens // na wens // droefheid verslinnen,
        Muziek, Muziek, Muziek maakt vreugd ydoon
      
      
        Musica el // is een Princesse fier,
        Een rechte grond // van Constenaren,
        Want wat gespeel // van Instrumenten hier
        Die men ooit vond van duizend Iaren,
        Doet zij verklaren // geen Retorica leert,
        Men ziet // druk vliet // door t’spel der snaren,
        Muziek, muziek, muziek doch vreugd vermeert
