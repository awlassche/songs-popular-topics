
    
      
        Alle mijn betrouwen // stel ik op God den heer
        Alleen op v ik bouwen // verlaat mijn nimmermeer
        Gij zijt mijn troost mijn toeuerlaet
        Daar alle mijn hoop op staat
        Voor schande en oneer // behoed mij doch o heer
      
      
        Want gij o God geprezen // helpt mij nut alle noot
        En gij cont mij genezen // door u genade groot
        Dus bid ik vroeg ende laat
        Mij toch met uwen hulp bystaet
        Voor schande en oneer // behoed mij doch o Heer
      
      
        Wilt mij doch regeren // v arme schaepken teer
        Dat ik mij mag verneren // ootmoedig na u leer
        Zo zal ik verlaten zaan
        Die wereld met haar boos vermaan
        Voor schande en oneer // behoed mij doch o heer
      
      
        O God almachtig // verhoord mij metter daad
        Want ik ben v zeer klachtig // mij doch niet en versmaad
        Laat mij doch beschamen niet
        Maar helpt mij doch nut dit verdriet
        Voor schande en oneer // behoed mij doch o Heer
      
      
        O God der prinsen // laat mij syn onvertsaecht
        Want al mijn winschen // van v arme dienstmaagd
        Syn u bekent o heer divyn //
        Verlost mij doch van druk en pijn
        Voor schande en oneer behoed mij doch o heer
