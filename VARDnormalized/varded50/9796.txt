
    
      
        O Broeders rein looft den Here gestade
        Die hier in het plein // laat bloeien zijn genade,
        Als een Fontein // voor die afstaan van het kwade
        En voort alleyn // met Christum gaan te rade
        Die zo het graan // in goe Aerd' ontfaen,
        Worden van dezen zade,
        Kinderen vroed // van God den Vader goed,
        Door zijns zoons bloed // vergoten overvloed
        Ligt onder voet // Dood en Helle vertreden
        Lof hem zijn moet // van zijn genade zoet,
        Wie boet // hier doet // zij staan noch heden.
      
      
        Eeuwig houdt stand // 'sHeeren goedheid geprezen,
        Hij heeft geplant // zijn ghemeynt' uitgelezen
        Want triumphant // ten Hemel op gheresen
        Ter rechterhand // zijns Vaders cord na dezen
        Zand zijnen geest // met een winds tempeest
        Tongen gedeeld // in het wezen
        Vyerigh en klaar // op iegelijk van haar,
        T'ghetal aldaar // in zijnen naam te gaar
        Dat en was maar // ontrent twintig en honderd,
        Den Geest hier naar // werkende wonderbaar,
        Heeft zwaar // de schaer // des Volks verwonderd.
      
      
        Van zuyt oft Noord // uit menige landpale,
        Heeft daar gehoord // een iegelijk zijn tale
        Betuigen 'twoort // en Gods werken loyale
        Vrees en discoord // hadde het volk generale,
        Want deze zijn // vol zoeten Wijn
        Gedronken altemale,
        Spraken rebel // eenigh' uit spot en spel,
        Op zulk vertel // antwoorde Petrus wel,
        Dits gheen voorstel // van droncke lieden,
        Maar doort bevel // Gods, na 'tprofeteersel
        Ioel // blijckt snel // hier dit geschieden.
      
      
        O Prins, o Heer // u macht liet gij aanschouwen,
        Van leden teer // een sterk huis gingt gij bouwen
        Meer ende meer // lieten Mannen en Vrouwen
        Op uwen leer // haar dopen, met betrouwen
        Maar door den nijd // der Joden wijd
        Verstroyt, door zwaar benouwen,
        V dienaars jent, vant nieuwe Testament,
        Tot 'swerelts endt // maakten u woord bekent,
        Heer excellent // lof prijs moet gij gewinnen,
        V woord ontrent // ons laat zijn ongeschend
        Present // gheprent // in onze zinnen.
