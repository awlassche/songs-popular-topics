
    
      
        EEn Liet // wil ik gaan beginnen,
        het Bediet // zult gij ook wel beminnen,
        Want al mijn hart en zinnen
        Die winnen // nu niet dan vreugd,
        Dus schenckt en brengt
        'tHert denkt // en wenckt // op u rein jeugd.
      
      
        Ik moet // dit nu schoontjens uit vegen,
        Het voet // de mens in alle wegen,
        Dus moet // ik dit te degen
        Gaan legen // en klaren uit,
        Volgt na // zegt ja,
        Het ga zeer dra al in mijn huyt.
      
      
        Dit glas is nu ledig gedronken,
        Zeer ras zat het worden gheschoncken,
        En niet lang laten pronken,
        Maar lonken // hier aan mijn zij,
        Op dat // het nat,
        In het vat // ons mag maken blij.
