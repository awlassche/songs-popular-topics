
    
      
        EEn gebroken herte, na Gods behagen,
        Om na te jagen // den vrede zoet,
        Voor de woorden Gods een vertsagen,
        Met een verslagen // oprecht ghemoet,
        Wens ik u tezamen voor enen groet,
        Daar toe een goed // gezicht altijd,
        Dat gij meugt merken // door 'sGheests versterken,
        Waar af dat gij gevallen zijt.
      
      
        Goud dat doort vier is zonder bezwijken,
        Coopt om verrijken // doch niet en zwicht,
        Coopt zalve om u ogen bestrijken,
        Dat u mag blijken // een klaar gezicht,
        Dat u verschijne dat wonderlijk licht,
        Wordt zo gesticht // dat gij subijt
        Mocht zien van binnen // met hert en zinnen
        Waar af dat gij gevallen zijt.
      
      
        Gij die Gods timmeringhe reene,
        Met ons gemene // voortijds waakt,
        Keert weder tot den verworpen steene,
        Christus allene // geopenbaard,
        Tot enen vasten gront vermaard,
        Met hem vergaard // dat hij niet wijt
        V en verstroye // keert t'sijner koye,
        Waar af dat gij gevallen zijt.
      
      
        Komt ende smaakt de Hemelse gaven,
        En wilt u laven // aan de Fontein,
        En wilt u niet als arme slaven,
        Slijckputten graven // in swerelts plein
        Die zonder water zijn onrein,
        Maar in den trein // der waarheid bevrijd,
        Ziet hier u gangen // peyst met verlangen,
        Waar af dat gij gevallen zijt.
      
      
        Hoe lieflijk is het, wanneer Gods zonen,
        En dochters wonen // in goed akkoord,
        Dat wil ons God den Vader jonen,
        Met alle den gonen // diet geern hoort,
        Dat Goddelijke zuiver woord,
        Dus van nu voort // om u profijt,
        Zo wilt gedachtig // zijn eendrachtig,
        Waar af dat gij gevallen zijt.
      
      
        Het Lam en heeft maar een getrouwe
        Bruid voor huisvrouwe // in swerelts dal,
        Een is zijn Duive, zo ik aenschouwe,
        Het bed' is nouwe // en 'tdecksel is smal,
        Een kudde, een Herder, en een Schaepstal,
        Is er voor al // na Schrifs belijd,
        End' een Gemeente // op Syons gesteente
        Waar af dat gij gevallen zijt.
      
      
        Verloren Schaepken krank ende teere,
        Ten rechten kere // gij wel behoeft,
        Verloren penninck een Vrouw met ere,
        Die is zo seere // om u bedroeft:
        Verloren Zone, niet en vertoeft,
        Hongerig proeft // met groter vlijt,
        In uwer herten // met droever smerten,
        Waar af dat gij gevallen zijt.
      
      
        Daar is noch plaatse voor den aftreder,
        Om keren weder // ter vreden band,
        Zijt gij verheven hoog als een Ceder,
        Buigt u doch neder // met u verstand,
        Onder Gods almachtige hand,
        Dat in den band der liefden verslijt,
        Twistig afgrijzen // dat vrede mag rijzen,
        Waar af dat gij gevallen zijt.
      
      
        Gij hebt verlaten de eenichede,
        Den rechten vrede // hebt gij gemist,
        Liefd' en genade en heeft gheen stede
        Bij u ook mede // door uwen twist,
        Dus hebt gij des Vaders goed ghequist,
        Door 'svyants list // geworden kwijt
        Zijt gij u krone // ende pand schone,
        Waar af dat gij gevallen zijt.
      
      
        Wie bij hem zelven wel meent te stane,
        Heeft ga te slane // menighertier,
        Dat hij niet en valt uit de bane,
        Die om te gane // zo smal is hier,
        Aan de een zijd' water, aan de ander zyd' vier,
        Vol van dangier // aenvecht en strijd,
        Dus den voorleden // weg wilt treden,
        Waar af dat gij gevallen zijt.
      
      
        Dit zegt die Here, treedt op die wegen,
        Maar ziet ter degen // voor u wel,
        Vraagt waar den goeden weg is gelegen
        Dien zijnde verkregen // zonder uitstel,
        Wandelt daarin, tis Gods bevel,
        V Ziele snel // zal rusten verblijd,
        Helpt zo des Heren // getal vermeren,
        Waar af dat gij gevallen zijt.
      
      
        Moyses die heeft hem zelven geboden,
        Te laten roden // uit 'sLevens Boek,
        Ook Paulus van ghelijcker moden,
        Al voor den Joden // uit liefden kloek,
        Om die te verlossen vanden vloek,
        In dat verzoek // u ook niet en mijt,
        Neemt aan u eeste // liefde, dits 'tmeeste,
        Waar af dat gij gevallen zijt.
      
      
        Barmhertigheid wil ik van uwen hande
        En gheen offerande // dit woord insiet,
        Had gij geweten, als de vailliande,
        Ten rechte verstande, zijn bediet,
        Den onschuldigen en had gij niet
        Zonder verdriet // vermaledijt,
        Maar had gebleven // Gods woord beneven
        Waar af dat gij gevallen zijt.
      
      
        Oorlof hiermede, wilt des Heren
        Zachtmoedigheid leren // met herten ootmoed
        Tot rust der Zielen, vrij van verseren,
        Wilt u bekeren // tot zijn jok zoet,
        Blijft niet beraden met vlees en bloed,
        Neemt dit voor goed // tis uit gheen nijt:
        Maar om met tranen // u te vermanen,
        Waar af dat gij gevallen zijt.
