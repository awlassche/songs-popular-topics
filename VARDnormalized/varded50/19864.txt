
    
      
        DOet u Oogjens open
        Schoon LIef het is dag
        Ik heb lang geroepen
        Dat ik u zien mag,
        Want u wezen schone
        Mijn hert doet verblijden // zeer,
        Komt ontvangt de krone,
        Gij en zult niet lijden // meer.
      
      
        Doet aan uwe kleeren,
        Hier is een kleed wit,
        In het bloed des Heren
        Is gewassen dit:
        Uit mijn dierbaar wonde
        Vloeyt een reine Beke // klaar,
        Die afwascht de zonde,
        Die u gaf doods stekken zwaar.
      
      
        Uit u zelfs wilt treden,
        Ende komt tot mij,
        Want wij zullen heden
        het Samen wezen blij
        In mijns Vaders Rijke
        Zullen wij gekroonet // zijn,
        Komet uit den Rijke,
        Uwe liefd betoonet // mijn.
      
      
        Ziet de Zonne schijnet
        Van mijns Liefs gezicht,
        En de nacht verdwijnet
        Lief gij zijt mijn licht:
        Uwe Zonne-ogen
        Mijne ziel verlichten // klaar,
        Ach ik moet verdrogen,
        LIefd komt mij doorschichten // gaar.
      
      
        Ik val aan u voeten,
        Lief van minne krank:
        Wilt den druk verzoeten,
        Schenckt mij liefdes drank:
        U bloed hoog verheven
        Is mijn Medecijne // goed,
        U dood doet mijn leven,
        En maakt de Helsche pijne // zoet.
      
      
        Werelds ijdelheden
        Ik ga ver van u,
        des Lands gerechtigheden
        Mij bedekken nu,
        Ik ga voor Gods Throone,
        Vrij van u besmetten // al,
        Ene Goude krone
        het Lam op mijn hoofd zetten // zal.
      
      
        Eer zij God den Vader,
        Ere het reien Lam,
        d'Heylge Geest te gader,
        Looft God heylge stam,
        Vallet voor Gods Throone,
        Ende roept met herten // blij,
        Eer zij Godes Zone
        Die ons maakt van smerten // vrij.
