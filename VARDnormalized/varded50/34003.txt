
    
      
        O Cierelijck sieraad!
        Wt-muntend’ in deugd en schoonheid van gelaat,
        Die ik bemin // met hert en zin,
        En die ik houd’ voor mijn Goddin,
        Princesse // Vooghdesse, die meest // mijn geest
        Wond en geneest,
        Door het lieve licht // van u gezicht,
        Ontfermt u eens, voldoet u plicht.
      
      
        Hoe lange zal ik doch
        (Dit vraegh’ ik) zo wreed gemartelt worden noch?
        V strafheid wendt // o Lief, en endt
        In blijde vreugd mijn droef ellend’:
        Met reden // doch heden aanziet // het verdriet
        Dat mij geschiet,
        Daar ik zo zeer // u schoonheid eer,
        Ja vier’ de zelve langs hoe meer.
      
      
        Zal ik dan stadig leidt
        Genieten, tot loon van al de dienstbaarheid,
        Die ik met lust // nooit uitgeblust,
        Aan u betoond hebb’ zonder rust?
        O schone // persoone, laat af // de straf,
        Eer gij in het graf
        Vervoerd de geen’ // daar gij alleen
        Met ijver wordt van aen-ghebeen.
      
      
        Mijn Liefste, zo gij maar
        Vyt liefde mij aen-neemt tot uwe trouw dienaar,
        Ik soud’ u staegh // bij nacht, bij dag,
        Gedienstig zijn met herten graag:
        Wat lusten // wat rusten, wat vreugd, uwe jeugd
        Tot haar geneught
        Bedenken mocht // oft ooit verzocht,
        Dat soud’ geschieden zo gij het docht.
      
      
        Ik wedd’ mijn Engel dan
        Soud’ zeggen, dat nooit op aarden enig man
        Zijn liefste vrouw // was ooit zo trouw,
        Als ik u stadig wezen sou;
        Dus enig // allenig te zijn // is pijn
        Voor u en mijn:
        Dus laat ons beyd’ // o zoete Meid,
        Verzamen eens in vrolijkheid.
      
      
        Wij zullen dan voor al
        Zo leven, dat de een wenst de ander wensen zal,
        Wat de een bemindt // oft goed bevind,
        Daar toe zal de ander zijn ghesindt.
        Wat droomt gij? beraad gij doch snel // het uyt-stel
        Komt zelden wel:
        Zeg; laat voort-aen // u droefheid staan,
        Ik neem u tot mijne dienaar aan.
