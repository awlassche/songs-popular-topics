
    
      
        Komt jonge Ieughden // u verlusticht hier,
        En schept eens vreugden // in den Hof van Lier,
        Maar wilt niet vluchten // tot een yders spijt
        D'onrijpe vruchten // door haat ende nijd.

        Die niet en weten // de gheleghentheyt,
        Hoe't dor werd groeyende // ziet dezen Hof
        Zeer aardig bloeyende // beschrijft den lof.
      
      
        Ziet nu Diana // met vreughdt komt hier bij
        Ops en Rosana // met veel melody
        Zij plukken geurig // en rieckende cruyt,
        Ialemum treurig // sluiten zij daar uit.
      
      
        Hoort nu Auterpe ,, zeer zoet zingen daar,
        Op de Luyd' Herpe // van Apollo klaar,
        Bij Nethe stromen // waar de Saters al
        Geyt-voetigh komen // in het Liersche dal.
      
      
        Daar Deycken groeien // sietmen lustig de
        Naiaden spoeyen // met t'soetbletigh vee,
        De Nymphen springen // uit haar malsche lis
        Armvlechtigh zingen // hoort wat vreugd daar is.
      
      
        Daar Blommen brochten // maakten enen dans,
        Ende zij vlochten // enen groenen Krans,
        Die nu Rhethorica ,, te samen bint
        Door-wroght van Laura ,, met een groene lint.
      
      
        De Eendrachs vrienden // verdoemen den haat
        Die zocht t'ontbinden // veel twist ende kwaad,
        Komen bezetten // dit groen reine land
        Met el Iennetten ,, en Eeckels beplant.
      
      
        Prinsen verheven // die wilt voeden wel
        V hong-righ leven // met zoet snaren spel,
        Wilt nijd versaecken // en in het Liersche Hof,
        Komt u vermakken // tot Gods meerder lof.
