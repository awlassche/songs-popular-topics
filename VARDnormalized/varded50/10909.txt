
    
      
        Verblijd // met vlijt // in deze Meye tijd
        En spoed u nu in het groen, en spoed u nu in het groen,
        Vermijt // en slijt // de vileyny en spijt,
        Rein liefde wilt bevroen, rein liefde wilt bevroen
        Haast u in het veld zeer koen // om de Mei eer te doen,
        In vriendelijkheid vergaard // doch met u Liefken part,
        Bedaard // eer gij bezwaard,
        Verklaart // u aard // en liefd’ bij u bewaard
        In deze Mei vermaard, in deze Mei vermaard
      
      
        In het veld // u stelt // met droefheid u niet quelt,
        Maar vreugde doch aanvaart, maar vreugde doch aanvaart,
        Druk smelt, met geweld, weest vrolijk als een Held,
        Toont liefde onbeswaert // toont liefde onbeswaert
        V liefste Lief openbaart // zo werd u liefd’ verklaart,
        Dan suldy troost ontfaen // als gij doet zulk vermaan
        Tot die u hert heeft bevaen,
        Des paan // wilt gaan // met vreugd wilt nemen aan
        Des Mei zeer zoet ontdaan, des Mei zeer zoet ontdaenmmdees
      
      
        In denkt // verheugt // met u Lief als gij meucht,
        In het wout zeer schoon geplant, in het wout zeer schoon geplant
        En veucht // geneucht // lieflijk als jonge jeugd,
        Neemt vrolijkheid ter hand, neemt vrolijkheid ter hand
        Spoed u vaardig in het land // daar de voghelkens triumphant
        Zingen met stemmen veel // en prijzen met luider keel
        Gelijk die boven al is el,
        Querel // t’ghescheel // doet van onze Taneel,
        Zo valter geen krakeel, zo valter geen krakeel,
      
      
        Prins tracht // met macht // na haar die u verwacht,
        In liefd’ bloeyende zoet, in liefd’ bloeyende zoet
        u klacht // versacht // haastig rijt op de jacht
        Tot dat gij haar ontmoet, tot dat gij haar ontmoet,
        Verthoonr u haar met der spoed // als een recht minnaar doet
        Die tot haar liefde dracht // en haar geerne behaagt,
        Niet vertsaagd // te komen bij die maagd
        maar wacht // en vraagt, weest weg eer dat het dacht,
        dat gij u niet beklacht, dat gij u niet beklacht.
