
    
      
        Waar neemt gij zo haastig henen uwen gang,
        Zegt mij Galathé door uwen zoeten zang?
        Bestiert u voetjens // en gaat wat soetjens,
        ’k Groet u met gewenste dagen lang.
        Wel wie spreekt mij hier zo minnelijcken aan,
        Zijt gij dat Tyter? Waar henen wildy gaan?
        Ik ga om melken // mijne Koeytjens, welke
        Ik wedd’ zij mij al loeyende wachten staan.
        Galathé,, leidt mij mee, moght het mij gebeuren.
        Tyter maet // het waar geen raad // iemand moght ons speuren.
        Wie zal ons hier zien bij een?
        het Veld heeft ogen in het gemeen.
        het Zijn al qua gedachten // maar.
        Maagden moeten wachten // haar.
        Laat mij u geleiden
        Door de groene weiden,
        Op dat ik u mijn trouw en min verklaar.
      
      
        O Galathea, mijn schone Herderin,
        V eerst aen-sien brocht mij het minnen in den zin,
        V schoon lichame // rein en bekwame,
        Heeft willig mij gebrocht tot uwe min.
        Ja Tyter, en let u anders niet?
        Gij en zoekt mij maar te brengen in het verdriet,
        Sulcken kluchten // die zal ik wel vluchten,
        ‘k En zal u Herder licht geloven niet.
        Schoon Jong-vrouw // kreeg ik non // troost tot minder wonden.
        Tyter ziet // geen verdriet // is ooit aan u bevonden.
        Gij weet beter dan gij praat.
        ‘k Raed’ u Tyter dat gij gaat.
        Galathé, zijt niet zo wreed,
        Ik en doe u doch geen leed.
        Is dat geen leed,
        het Is het meeste dat ik weet,
        Dat iemand ooit een sterfelijk mens aen-deedt.
      
      
        Hoe menigh-mael heb ik u verhaalt
        Vw’ deughden, en uwe schoonheid af-gemaelt?
        Hoe menigh-mael geprezen // u eerbaar wezen,
        Waar door gij mijn hert ende vrijheid staelt?
        Hoe menigh-mael liet ik mijn schaepkens al
        Gaan dolen, oft gestolen in den stal,
        Om u te spreken? zoudt gij dat wreken
        Met een on-waerdigh weer-geschal?
        Tyter vlied // het is om niet // gij moet mij verlaten.
        Ei! waarom // liefken kom // laat ons noch wat praten,
        Ik moet bij mijn Koeien // gaan.
        Laat die noch wat loeien // staan.
        Neen Tyter, men wacht naar mij.
        Dies te beter acht men dij.
        Och! ik moet scheiden.
        Neen, gij moet wat beiden,
        Den dag die is noch vroeg genoeg van tij.
      
      
        En zult gij dan noch nimmermeer
        V wreed gemoed tot mij-waarts keren weer?
        En zal ik u zinnen // niet eens verwinnen,
        Daar ik u bemin op deugd en eer?
        Ja Tyter, soud’ ik zo jonge teere Maagd
        Zo licht geloven de klachten die gij klaeght:
        Weg ware lusten // rouw dat waar mijn rusten;
        Want sulcken bedrog de herders meest behaagt.
        Schoon jongvrouw // eer ik sou // de geloften breken,
        Van mijn trouw // liever sou // ik mijn hert af-steken:
        Den Olifant sou vliegen eer,
        Eer ik mijn zinnen van u keer;
        Och mijn Lief, mijn troost, mijn al.
        Tyter, tiert u niet zo mal.
        Ei! en wilt niet klagen,
        Tot u behagen
        Ik noch een vrolijk Liedtjen zingen zal.
      
      
        Alhier in dit genoeghelijcke dal,
        Vol bloemetjens schoon, bij dezen groenen wal,
        Daar dit klaar stroomken // langs menig boomken
        In het midden wispelt met een zoete val;
        Hier zag ik eerst mijn schone Galathé,
        Wiens glants scheen helder als de nieuw-gevallen snee,
        Wiens eerbaerheden // ende brave zeden
        Mijn hert met heete min ontsteken de;
        Haar blond haar // gaf zij daar // den wind mee te spelen,
        Haar gelaat // metter daad // mijn vrij hert kwam stelen,
        Haar oogskens bloncken bruin van verr’,
        Veel schoonder als de klaerste sterr’:
        Och! vrouw Venus, hoog-geacht,
        Seyndt door Cupido uwe kracht,
        Dat mijn Galathea
        Nu eens zegge mee Ja,
        Voor al mijn droevig lijden voor en na.
      
      
        Ja Tyter, hoe zydy zo gewondt?
        O Schalk en Vos, hoe speeldy met den mond!
        Gut, hoe kondy kwelen // en de herten stelen
        Van de Maagden met een lieffelijcken vond!
        Ja, mijn Galathea, het is al het oude lied,
        Mijn zingen, en mijn springen, dat daar door geschiet,
        Is om te vermeren // uwe lof, prijs, en eren,
        Noch en wind de liefde uwe jonste niet,
        Boefjen snood // ik word’ rood // Gut hoe kan jij praten!
        Ei! waarom // Liefjen kom // ‘k zal u niet verlaten.
        ‘k Blijf te lang van huis, eilaas!
        Galathé maakt geen geraas.
        Wel het schijnt of het dond’ren wil.
        Neen, het weer is wonder stil;
        Och! mijn hoop’, mijn leven,
        Wilt mij nu troost geven,
        En maakt doch eens een eynd’ van ons geschil.
      
      
        Bemindt doch, o mijn Galathea braaf,
        Die u trouw dient als de aller-meeste slaaf,
        Gij krijgt in handen // alle mijn landen,
        Mijn Ossen, Geiten, Schapen, en mijn haef.
        O! Tyter, zwijgt, het is genoeg geklaeght,
        Gij wind victori’ op mij jonge Maagd,
        Ik ken de krachten // van mijn gedachten,
        Mijn Moeder moet eerst zijn om raad gevraagd.
        Neen, jongvrou // onze trou // knopen wij hier tezamen.
        Tyter, wat // zegt gij dat // gij hoort u te schamen.
        Het wordt op eer en deugd gebouwd.
        Tyter, stil uwe handen houdt.
        O mijn Galathea zoet.
        Ik zal roepen zo gij het doet.
        Roept zo ‘het u zeer // doet,
        Zwijgt zo ik het meer // moet,
        Een kusjens en ontneemt u eer noch goed.
      
      
        O! zoete Galathea, mogen wij
        Eens gedurig blijven zijd’ aan zij,
        En dat niemand scheiden // en mocht van ons beiden,
        het Waar een hemels Paradijs voor mij.
        Ja Tyter, hier mede zo ga ik deur,
        Gij moogt mij volgen: Adieu, dat gaat u veur;
        En al ons praten // wilt verborgen laten,
        Op dat wij niet en komen in het getreur.
        Galathé,, op wat stee // komen wij te samen?
        Tyter vriend // hier toe dient // deze wei bekwame:
        Adieu Tyter, goede rust.
        Galathé, noch eens gekust.
        Mijn Moeder die sou kijven // och!
        Wat sou zij meer bedrijven // doch?
        Och mijn Galathé,
        Liefjen ik ga mee,
        Ons liefde laat doch leven in peys en vree.
