
    
      
        EEn Lied eerbaar // van de liefste klaar
        Voorwaar, Wil ik beginnen,
        Want bij haar // Zo rust wel zwaar
        Eenpaar, met hart en zinnen,
        Zij is zeer rein, In sWeerelts plein,
        Men mag gheen schoonder vinnen,
        Want zij alleyn, Is mijn shartsen grein
        Dus moet ik haar beminnen.
      
      
        Haar wezen fier // En weet ik hier
        Bij niemand te gelijken,
        En haar manier // Zeer goedertier,
        Heeft mij doen blijken:
        Maar zeer rebel // Past zij in het spel,
        En mij gheen trou doet blijken
        Al lijd' ik gekwel, door dit opstel,
        Ik zal haar niet ontwijken.
      
      
        Dat zij mij boot // Haar lipkens root
        Mijn moet, waar gheresen,
        Door haar liefde groot, Ik zegt v bloot //
        Van die dood, Waar ik genezen,
        Om dat zij went, Van mij ontrent,
        Dus lef ik in vrezen,
        Maar mijn Liefken jent // Zeer excellent
        Moet altijd zijn geprezen.
      
      
        Princesse goed, die mijn vlees en bloed
        Ja doet mij mineren,
        Om dat ik moet // V wezen zoet,
        Weest dit wel vroed // Dus lang ontberen,
        Nochtans ik zal // Tot mijn verhaal
        In liefde glorieren,
        Spijt nijders geschal // Liefd' boven al,
        v lief exalteren.
