
    
      
        Ave Maria // Gods liefste lief ydone
        Der maagden kronen // Maria weest gegroet
        Gratia plena // klaar als de zonne schone
        In het Hemels troone // vat vol genaden zoet
        Der deuchden volle vloed // een zuiver hert ootmoed
        Hebt gij gebaard // Gods soon ons hoogste goed.
      
      
        Dominus tecum // die heer zij met v ene
        Zuiver fonteene // vrou alder wereld wijt
        Benedicta tu // in mulieribus allene
        Al waart gij cleene // gij zijt gebenedijt
        Geen vrou te gender tijd // zo zalig als gij zijt
        Noch zo verheven // van God tot vijands spijt.
      
      
        Et benedictus tuus // gedankt, gelooft, geprezen,
        Moet gij altijd wezen // dat hoochweerdich pand
        Fructus ventris tui // dijn vrucht rein wt gelezen
        Wt Ieffa geresen // Jezus Christus playsant
        Die als een Heer valjant // ons brak der hellen band
        Ende die gevangen brocht // in haar vaders land.
      
      
        Sancta Maria Princesse // hoog verheven krone
        Edel geboren // van koninck Davidts saet
        Ora pro nobis // bidt voor ons die te voren
        Waren verloren // verdoemt door zonden kwaad
        Tot dat ons kwam te baat // Jezus onze advocaat
        Van u geboren // o schone dageraad.
