
    
      
        Waarde Kuisheid, Hemels Kind,
        Van den Heer en ieder een bemint:
        Wiens gepronck // wiens gegluer //
        En gelonk, En gelaat
        Is der Deughden Allerhoogsten graad:
        Het is een Deugd,, het Is een vreugd //
        Die de Jeugd Meer bepraeldt,
        Als het Goud, Hoe zeer 'et straeldt:
        het Is een schat // Die het pad //
        Na de Stad henen leidt,
        Daar men nooit ween-ooght, snickt, of schreyt.
      
      
        Ioseph, Spiegel van de Ieught,
        Deftig beeldt van eerbaarheid en Deugd;
        Zedig hert // Die de schimp
        En de smert // Van de Vrouw'
        Vaak wou lijden // En noch blijven trouw;
        Die Liever vlood // In de nood,
        Totte dood // Dan u Heer
        Gants te brengen // Tot oneer:
        Die u zinn'// Op 'er Minn'
        En ghewinn'// Niet en hecht',
        Maar u droeg // Trouw, gelijk een Knecht.
      
      
        Lof eerwaarde Kuyscheydts glans,
        Gij verdiend een versse Roze-krans,
        En een Kroon // Op u hoofd,
        Tot een loon // Voor u strijd:
        Om dat gij ons // Zo een Voorbeeld zijt,
        Op Kuyscheyts baan // Die wij gaan
        Nu voortaan // Stadig voort
        Moeten treden // Alst behoort:
        Lof o! Held,, Die in het veld
        Hebt gesteld // Al u macht,
        En de lust,, Heel hebt t'onderbracht.
