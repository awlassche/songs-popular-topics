
    
      
        WEl zalig is hij, die den Here // kent
        En hem van al de valse lere // went
        Die veel en loos opstaat aan allen // sijden
        Zo dat hij haast sou om te vallen // glijden,
        Die niet en ziet met vlijt op 'sHeeren // woord,
        Twelck men eenpaar meer, meer te leren // hoort.
      
      
        'tWoordt Christi moet ons rijcklijck wezen // bij,
        Met wijsheid groot, op dat in dezen // wij,
        Door zijnen Geest, zijn vruchtbaar ranken // reinig,
        Die schuldig zijn hem niet te danken // weinig,
        Maar veel niet lust van zijn weldaden // goed,
        Die hij zo veel ons uit ghenaden // doet.
      
      
        Laat dag en nacht ons overleggen // dan
        Den rechten pad, en daar niet zeggen // van
        Alleen, maar daar in al ons gangen // schikken
        Zo zijn vergeefs om ons te vangen // strikken
        Vanden vijand geleid genegen // om
        Vervoeren ons in zijn zijdweghen // krom.
      
      
        Zo wie afwijkt, of niet herboren // leeft
        Maar voor zijn deel iet el verkoren // heeft,
        Kwijt is hij God, en zijn gemeente // mede,
        Dus ofmen schoon ons veel verkleente // dede,
        Laat ons geen kaf zijn, noch gelijken // riet,
        Dat wind verstroyt, oft dat men wijken // ziet:
      
      
        Met eewich vier zal men verbranden // kaf,
        En 'toncruyt ook, geknocht in banden // straf
        Wel hem die werd Tarw' in de leste // ure,
        Hij werd gheberght in d'alderbeste // schure,
        Dit zal gezien hem, die met 'tkrachtigh, zaad
        Gods woord bevrucht, vruchtbaar eendrachtig // gaat.
