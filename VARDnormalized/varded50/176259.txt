
    
      
        SChoon Nymphelijn // ach mindje mijn // wat zoudt ik al versieren,
        Om na mijn wens // des ledetjens // zo wel gemaakt, te cieren?
        Met blinckend goud // of perlen zout // gij voelen ras belasten
        Vw halfgen zoet // dat kraal als bloed // daarom niet beter pasten.
      
      
        Ik zoudt u cleen // met keursgens reen // van lichte verwen blijtjens.
        Die zouden staan // gheschildert aan // u breedachtighe zijdtjens.
        Vw voetjens mit // haar schoentjens wit // daar bij geval, in het bokken,
        Het inkarnaat // zo wel bij staat // van hoosgens net getrokken.
      
      
        Vw armkens mee // zo wit als snee // zoudt ik koraal om schikken.
        Des vlechgens blond // op nieuwe vond // zoudt ik u leren strikken,
        Met snoertgens veel // nu groen en geel // bij lodderlijcke beurtgens;
        Voor wat jent haar // of cruyvendt daar // en duysendt zoete leurtgens.
      
      
        'Tperruyckgen sou // ik trekken non // wat laechjens, dan wat hoochjens,
        En als het klaar // ghefutselt waar // mij spieglen in u oochjens.
        Dan werpen licht // nu mijn gezicht // op het ene, nu op het ander:
        Dan nemen raam // hoe't altesaem // sou voegen bij elkander.
      
      
        Indien dat gij // uw oochjens blij // en lieffelijcke zeeden,
        Zo vreugderijk // zo vriendelijk // zo vol bevallijckheden,
        En u aanschijn // vernoecht, tot mijn // dan, met een lachen wenden:
        Zo zoudt ik, hiel // in u mijn ziel // gaan, metter wone, zenden.
      
      
        En als ik wat // beloncket had // den brand van alle knechgens,
        Vw zedetjens // uw ledetjens // uw fray ghetoyde vlechgens;
        Zo zoudt ik streng // met armen eng // uw jente lijfjen prangen,
        Tot ik, daar uit // creech, buit om buit // u sieltjen weer gevangen.
      
      
        De verve van // mijn lippen, an // uw wangsjens, soud' ik plekken,
        Door soentgens zacht // en met haar kracht // u ziel te mond uit trekken.
        Ik wed, ik weet // gij dan beleedt // dat niemand van u susjens,
        Daar gij, soo't schijnt // nu smaak in vindt // gaf ooit zo zoete kusjens.
