
    
      
        IN Esaie // prophecye
        In het tweede staat geschreven
        Hoe dat het zij // ten laatsten tij
        Als dan zal hoog zijn verheven,
        Den berg gewis// op welker is
        Mijn groot gedruis // des Heren huis,
        Als dat er veel hopen // hene lopen,
        Met vlijt // verblijd
        Elk ander dus toe  kopen.
      
      
        Al afghelaen // komt laat ons gaan,
        Tot dezen berg geprezen,
        Die vast zal staan // nimmermeer vergaan,
        Maar eeuwig t'hoochste zal wezen
        In welk die God // van Iacobs rodt,
        Nu hout present // zijn schone tent,
        Dat hij ons leer te degen // zijne wegen,
        Zijn paan // te gaan,
        En in gheen vreemde stegen.
      
      
        Niemand aenhoort // noch genen oord
        Dan Syon uytghelesen,
        Heeft Godes woord // met goed akkoord
        Ierusalem geprezen,
        Voert Godes wet // die onbesmet,
        Oordelen zal // die Heidens al,
        In oprechtigher klaarheid // na der waarheid,
        Zijn recht // bevecht,
        Die volken zonder swaerheyt.
      
      
        In sulcken ganck // zal elk te dank
        Zijn zwart tot ploechysers breken,
        Die spietsen lank // tot sikkelen blank,
        En denken niet meer te wreken
        Van deze gheen // ja ook met een
        Die daar begeert // enige zweert
        Tegen iemand te luiken // en gebruiken,
        Gheen striedt // geschied
        Van haar noch zulke stuiken.
      
      
        Alzo des reet // van den Profeet
        Luid gatet in onze tijden,
        Want veel is leed // haar overtreet,
        Waar over haar nu verblijden
        Hier niet alleen // Christus gemeen,
        Maar ook zeer klaar // d'enghelen schaer,
        Want zo staat hier beneven // klaar geschreven
        Van een // onreen
        Die hem tot boet wil geven.
      
      
        Komt doch eens naar // aenschout die schaer
        Die nu na t'beste hakken,
        Die voormaals waar // s'werelts dienaar
        Betracht nu andere zaken,
        Eertijds was zij // eigen nu vrij,
        Gheneycht tot kwaad // nu tot weldaad,
        Gierig onkuis zij zaten // boven  maten,
        Met veel // zulk speel,
        T'welck zij nu al verlaten.
      
      
        Ook ziet hier meer // hoewel noch teer
        Die mee beginnen te komen,
        Al haar geweer // leggen ter neer,
        Aanzien die mensen als bomen,
        Wijn zonder alst // melk ongevalst,
        Begeren zij // als kinderen ny,
        Op dat al jonge ranken // door die drancken
        Aemacht // tot kracht,
        Zij komen en God danken.
      
      
        Peys en vree // is in des stee,
        Met God voor zijn genade,
        Dus wendt u tree // gij alle nu mee,
        Die vreest der zielen schade,
        Nu isset vrij // komt alle mee bij,
        Het is nu klaar // terecht gulden jaar
        na der Bazuinen clangen // wilt verlangen,
        Des dag // vermach
        lElck een zijn erf te langen.
      
      
        Zal nu des teyt // der zaligheid
        Des aangename dagen,
        Met goed bescheid // worden verbreyt,
        En dat na Gods behagen,
        Sulcx niet allien // hoort te gezien,
        Met groot begeer // der reyner leer,
        Maar wij moeten niet beven // ook ons leven
        Daar naar // gants klaar
        Met neerstigheid begeven.
      
      
        Op dat wij geen // der swacker leen,
        Tot ergernisse strekken,
        Want vele ween // staan over deen,
        Die zijn des lichaams vlekken,
        Haar straf met spoed // haars naasten bloed,
        Och t'is zeer zwaar // komt al op haar,
        Als vruchteloose bomen // en onvromen,
        Met vliet // toesiet
        Om sulckes te ontkomen.
      
      
        En wilt doch niet // den blinden iet,
        Tot enen aanstoot zetten,
        V leven biet // zonder verdriet,
        Dat elk daar op mag letten,
        Duldich in druyck // noot angst misbruik,
        In slagen daar // ghevanckenis zwaar,
        In oproer arbeid vakken // ook in waken,
        Het zij // of gij
        Om u gherucht ook rakken.
      
      
        Nimmermeer laat // aan u iet kwaad,
        Met waarheid zijn te wrekken,
        Corinthen staat // ik houde maet,
        Op dat ik andere te preken,
        Niet zelf en ben // verwerpelijk slem,
        Dus laat ook wij // waken bety,
        Dat wij zonder bezwaren // recht verklaren
        Ons baan // wilt gaan,
        Want wij zijn Gods dienaren.
      
      
        Gheen oorsaeck geeft // der onbeleeft
        In deze tijd der ghenaden,
        Dat gij niet sneeft // en hij niet heeft
        Waar mede hij u mag schaden,
        Op dat ons pont // niet ga te gront,
        Maar onze licht // kom in het gezicht,
        Op dat gelijk wij lezen // ook bij dezen,
        Gofs naam, eerzaam,
        In ons hoog werd geprezen.
      
      
        Alsulcken leer // eyscht God die Heer,
        Die beit met woorden en werken,
        Zoekt zijne eer // kloek is te weer,
        Zijn teere spruytkens te sterken,
        Deze met denkt // verwekt die jeugd,
        En maakt verblijd // alle die belijt,
        Christum en zijnen vader // drucx ontlader,
        Doet goed // met spoed,
        Gij Christen alle gader.
      
      
        Voor zulke vreugd // voor zulke denkt,
        Voor zulke grote genade,
        Die ons verheugt // al dat gij meucht,
        Lovet God vroeg en spade,
        Wie zal genoeg // in eniger voeg,
        Lof prijs en eer // doen onzen Heer,
        Van deze schone zaken // vant vermaken
        Doort zien // van dien,
        Die totter kennisse raken.
      
      
        Tot lof en dank // der lippen gheclanck,
        Laat ons geheellijk begeven,
        Vrij ende vranck // met vrolijk ghesanck,
        Altijd zo lange wij leven,
        Ook hert en zin // van dit gewin,
        Want hij is goed // die het wassen doet,
        Bidden dat hij ons soare // en beware,
        Na dit gekrijt // met hem subijt,
        Ons hem gelijk verclare.
