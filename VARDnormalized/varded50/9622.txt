
    
      
        ALle vlees is hooi, na Schrifts belijd,
        Kort is den tijd, van smenschen leven,
        Als bloem in het veld, ontploken wijd,
        Moet hier subijt, zijn glory sneven,
        Wat is met den Mense bedreven?
        Een uyr en is hem niet gegeven,
        Hoe lange hij // zal leven vrij,
        Zijn perk hebt gij // o God beschreven.
      
      
        Dat oud verbond // is stervens nood,
        End' na den dood // wij al verwachten
        Noch enen dag des oordeels groot,
        Daar werden bloot // alle gedachten,
        Als Christus komt met zijn heyrkrachten,
        Zullen huilen alle geslachten
        Der Aarden kwaad // veel drucx toestaet,
        Al die den raad // Christi verachten.
      
      
        Voor den Rechtstoel Christi ydoon,
        Op den persoon // werd gheen aanmerken,
        Die het herte kent, zonder verschoon,
        Zal geven loon // elk na zijn werken,
        Kracht en zal niet baten den sterken,
        Noch geen kloekheid den wijzen klercken,
        Recht oordeel klaar // zal openbaar
        Strijken aldaar // dat hoofd der Kerken.
      
      
        Zeer vreeslijk God hem wreken zal
        Over het getal // zijner vijanden,
        Die de Wereld beminnen al,
        Want haren val // werd groot ter schanden,
        Haarlieder vier zal eeuwig branden,
        Haarlieder worm die blijft voorhanden,
        Rouwe, gheween // werd daar gemeen,
        Angst ende een // knerssen der tanden.
      
      
        Dan zullen die rechtveerdig zijn,
        Vrij van gepijn // ofte weeklagen,
        Blinken gelijk der Zonnen schijn,
        Met vreugden fijn // niet om gewagen,
        In het Rijke Gods, na haar behagen,
        Onsterffelijcke Kronen dragen
        T'Lammeken zoet // zal haar tot spoed,
        Der tranen vloed // van het aanschijn vagen.
      
      
        Een Princelijcke Majesteit
        Heeft God bereid // zijn uitverkoren,
        Dus is des loons groot onderscheid,
        Vast toegezeid // ende gezworen,
        Tussen die Godloos God tegen sporen,
        En die hem dienen wedergeboren,
        Wat neerstigheid // men hier toe leit,
        Genen arbeid // en werd verloren.
