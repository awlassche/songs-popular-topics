
    
      
        Als Coridon zag aan de rei // om de Mei,
        Elk Harder met een Harderin // naar zijn zin,
        Gescakelt zo spelt hij een Liet // op zijn Riet,
        Daar het Meysjen op zong // en ee Knaepjen op sprong,
        Al onder de Linde-boom // schoon om de Kroon.
      
      
        Doch Erickje dat gij aan de rei // om de mei,
        Gaat spiepele zonder mij // aan u zij,
        'k Was immers zo niet gezeid // licht meid,
        Maar Erickje is wel // Ik zweer u ik zel,
        Ziet dat is nu evenveel // en ik speel:
      
      
        Mooy benje ja zonder gelijk, en ook rijk,
        Welgunstigh aan dieje bemint . maar je vint,
        Zo deugdelijk niet alsje schijnt // Ik bevind,
        Gij spelt met mijn Hart // Gij spelt met mijn smart,
        Maerr denkt mij hier bij dit u val // wezen zal.
      
      
        Als Coridon het Lietge had uit // zijn geluid,
        Begaf hem een droeve zucht // in de Lucht
        Deed stenen en wenen de Ieught // om zijn deugd,
        Doch Erickje achte // belofte noch klachte,
        Zo licht is de liefde verdreyt, met de Meid.
