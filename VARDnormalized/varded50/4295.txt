
    
      
        Minzaam beeldeken amoureus //
        Die t'heyrcracht overwint,
        Troostbaar herteken gratieus //
        Gij zijt een seedbaer kind,
        Gemind // en wtgelesen //
        In het wezen // geen als gij //
        Daeromme //
        Rein blomme //
        Ik comme // tot dij //
        Mijn plagen //
        en klagen //
        Helpt dragen // met mij.
      
      
        Aardig, edel, besneden // rijs,
        Gesproten magnifijck,
        Hester zydy in zeden // wijs,
        Waar is doch ws gelijk?
        Aardrijk // vervult dijn fame,
        Eersame // maget smal:
        O puere,
        Figuere,
        De fluere // van al,
        O schone
        Persoone,
        De krone // zijn zal.
      
      
        Reikt mij heden dijn witte // hand,
        Dus lange niet vertouft,
        Mijn teer herte deur hitte // brand,
        D'welck meer dan troost behouft:
        Bedrouft // ligh ik ter neder,
        Keert weder // zoet aanschijn:
        Oorzake?
        Dijn sprake,
        Heeft smake // zo fijn:
        Dat heden,
        Dijn reden,
        Gebeden // moet zijn.
      
      
        Is bewezen zulk vrouwen // recht,
        Aan mij, in voller macht,
        Dat ik uwen getrouwen // knecht,
        Ben aangenaam verwacht:
        Den nacht // is dan verdwenen,
        Vaart henen // vals krakeel:
        Geen perten
        En smerten,
        Twee herten // geheel,
        Met zinnen
        Wij winnen
        Der minnen // kasteel.
      
      
        Adieu, minnelijk // cordiaal,
        Jonk herte, heus, en vrij:
        Die goddinnelick // teenemael
        Mijn vreugd zijt even bij,
        Zo blij // als wij nu scheiden,
        Verbeiden // wilt mij zaan:
        
          T'onvroede
        
        In het goede
        Ten bloede // neemt aan,
        Houdt stede,
        In vrede,
        Hier mede // wij gaan.
