
    
      
        T'Is nu den tijd // subijt // zoet aangenaam,
        Die Christen zijt // verblijd // u al te zam,
        En acht gheen blaam, laster noch schande,
        Schimp en verwijt // maar lijdt // door alle faam
        Loopt tot den strijdt // in het krijt // maakt u bekwaam,
        End' u lichaam // offert te pande,
        Ontsteken met der liefden brande,
        Op den Outaer // voorwaar // Christus zeer
        Tot een levende offerande,
        Zo krijgt gij daar // eenpaar // in lijden zwaar,
        Gods Geest troostbaar // ten onderstande.
      
      
        Van 'sHeeren Geest // den keest // der liefden smaakt,
        Door geen tempeest // bevreesd // niet en versaect
        Volstandig blaeckt, zonder verflouwen,
        Hout blijde feest // nu eest // bidt ende waakt
        In dit foreest // u meest // in deuchden maakt,
        Zorgt niet, al naakt // u veel benouwen,
        Wilt Gods beloften vast betrouwen,
        Wat dat gij doet // wilt vroed // om u voorspoed,
        Christum dijn voorbeeld wel aanschouwen
        Lijdt gij om goed // hebt moet // met trooste zoet,
        Zal u den bloed // des Geests bedouwen.
      
      
        'sGeests vloed versacht // met kracht, des lijdens grief,
        Gij Gods geslacht // en acht // op geen misschief,
        Hebt altijts lief // die komst des Heren,
        Die onbedacht // in nacht // werd als een dief
        Dus in eendracht // verwacht // die u verhief
        Wt Paulus brief // daar meugt gij leren
        Hoe dat alle dit Aerdtsch verseren,
        Pijn en verdriet // al niet // is weerdig iet,
        Des Hemels vreugd niet om vermeren,
        Dus niet en vliet // als riet // Loths wijf aanziet,
        Wat haar gheschiet // doort ommekeeren,
      
      
        Een soutcolom, om t'omsien werd zij zaan
        Volgt na Iesum // Christum // zijn baan wilt gaan,
        Zonder inslaan // t'eenigher zijden,
        Maakt geen verschrom // wilt vrom // gewapend staan:
        Hebt t'eener som // ook rom // in het kruis gelaen
        Wilt onderdaan // Wettelijk strijden,
        Gij zult ontfaen het ghebenedijden,
        Een excellent // en jent // schoon kronement,
        Want die volstandig zonder mijden
        Is tot den endt // bekent // zal ongeschend,
        Christum ontrent // eeuwig verblijden.
