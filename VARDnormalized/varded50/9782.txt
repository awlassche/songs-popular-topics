
    
      
        O Bruidegom, mijn Vorst, mijn Kapitein,
        Waar ik mij kere, hier in 'swereldts plein,
        Trouwe noch troost, en vind' ik in het ghemeyn,
        Dan in uwen heiligen Naam alleyn,
        V rein // troostbaar // woord klaar
        Dat sitcht // verlicht // en doet
        Verblijden mijn gemoed.
      
      
        Te recht, o Heer, hebt gij mij wel gezeid,
        Ziet toe, op dat u niemand en verleid,
        Want die slange met haar dienaars planteit
        Gebruikt nu alzo grote listicheydt,
        Maar scheid // noch vliet // doch niet
        Van mij // o gij // of ik
        Moet vallen in haar strik.
      
      
        Gij toondet mij enen weg metter hand
        Om wandelen na het beloofde Land,
        Die wordt veracht, door menselijk verstand,
        Enen bezijden dezen aan den kant,
        Playsant // oprecht // men zegt
        Hem fijn // te zijn // en dat
        Hij ook loopt na u Stad.
      
      
        Met zulke stormwinden schalk verdraeyt
        Van alle zijden word' ik nu bewaeyt,
        O Heer, in vreden houdt mij wel ghepaeyt,
        En u vrucht der gherechticheydt mij zaeyt,
        Verfraeyt // mij zaan // om staan,
        Standvast // wat last // oft kruis
        Daar komt tegen u huis.
      
      
        Prinse, 'tbeproeven valt menighertier,
        Maar gij castijt, al die gij lief hebt hier,
        Want t'moet doch al gezouten zijn met vier,
        Maar verlost mij van al dit arts dangier,
        Dat schier // ik mag // den dag
        Ontvlien // en zien // altijd
        V aanschijn, met jolijt.
