
    
      
        O Blaeckerende Zon
        Die mij mijn vrijheid con
        Benemen door u klare glans
        Dat ik verwonne schyn altans,
        Hoe strale // en male // voor mijn gezicht
        De vonken van u kracht,
        Hoe zijne // met pijne
        V lonckjes in mijn herte zo onzacht.
      
      
        Hoe kan u schoonheid doch
        Zo vrevel syn, dat noch
        Mijn klachten of gebieden u
        Niet maken van de wreedheid schuw,
        Syn heden // de reden // gans niet geacht,
        Of wilt gij die versmaen,
        En heetent // onwetend
        Mijn Engheltje bij u mag wel gedaan.
      
      
        Ten kan mijn hertje neen,
        Neen t’ken niet zo ik meen
        Van u voor goed werden gekend
        Want mijn ghemoet dat tuycht gij bent
        Bedarich // mewarich // en goedertier,
        Weldadig en volmaakt,
        Ik mercke // u werke
        Begint gij licht doch vaardig gij die staeckt.
      
      
        V schoonheid ongemeen
        Die doen mij u alleen
        Beminne in des Werelds dal
        Ik brommen uit met groot geschal,
        Volmondig // en bondig // door rens beleid
        Dat ghy’t hier al verwint
        In gave // zo brave
        Als gij men niemand meer op aarde vint
      
      
        Veel hebbender vermaard
        Geschreven van de vaart
        Van Helena de Griekxsche Maagd
        Daar Grieckelant noch roem van dracht,
        Heur wezen // dat prezen // zij volmaakt
        ‘tWas een zo schone Vrouw,
        Doch waarder // vermaerder
        En schoonder als Helena ik u houw.
      
      
        Prinses mijn lieve lust
        Mijn aangename rust
        Mijn leven en mijn ondergang
        Mijn vrijheid en mijn staffe dwang,
        Te spade // genade // mijn niet betoond
        Blust doch mijn vlamme heet
        Die d’ooghe // uit vlooghe
        Van u Mijn Zonne laas! zoet en wreed.
