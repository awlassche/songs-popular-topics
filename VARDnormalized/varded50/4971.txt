
    
      
        In ‘het geest’lijck Bloemen-hof // staat ene Blom gheplant,
        Waardig om hebben lof // hij staat in ‘het effen Land,
        Te Saron zeer playsant // lieflijk en uytghelesen,
        Boven all’ schoonheid: want // het is Godes Soon geprezen.
      
      
        Hij staat in ‘het herte: daar // niet is enig onrein,
        Want deze hebben haar // gekeerd tot de Fontein
        Des levens, daar is plein // geen Was-bat meer te vinden,
        Om ‘het hert te wassen rein // dan des enig beminden:
      
      
        Druckt mij spreekt Christus zoet // als een Zegel op ‘het hert
        Een herte vol ootmoed // dat rouwe draagt met smert,
        Veel troost gegeven werd // een troost vol soeter minne,
        Komt al, ‘het zij na oft vert // gaat tot dit Blom-Hof inne.
      
      
        Spreekt dan met blyden // spreekt dan met des herten zin,
        O Name Iesu! goed // hoe zoet is uwe Min,
        Hoe groot is dit gewin // hoe zalig zijn de ogen,
        Die dit te recht zien in // die werden niet bedrogen.
      
      
        O Name Iesu! wien // hoog zit ter rechter hand,
        Voor u moet alle knieën // ootmoedig buigen, want
        Gij verwond triumphant // alle onze vijanden,
        O Koning zeer valiant // naar u moet ons liefd branden.
      
      
        O Iesu, Lam Gods el // op Syons Berge schoon,
        Daar gij de Scharen veel // opzetten zult de Kroon,
        Hoe heerlijk is u troon // hoe lieflijk is u wezen,
        Hoe groot is uwen loon // u Naam moet zijn geprezen.
      
      
        O Iesu, u aanschijn // is het schoonste boven al,
        Hoe lieflijk salt daar zijn // daar men ‘het aanschouwen zal,
        Komt al gij Christen tal // komt laat ons vrolijk zingen,
        Komt maakt nu blij geschal, komt wilt Gods lof voortbringen
      
      
        Komt maar uit zondig huis, kleeft niet meer aan het kwaad,
        Verlaat ‘het zondig abuis // met al des werelds staat,
        Niet meer met haar en gaat // op hare kromme wegen,
        Haar arts gierigheid haat // zijt tot wel doen genegen.
      
      
        Dan vind gij des herten wens // dan vint gij wat gij wilt,
        Kont gij verstaan o mens // zo werd u hert gestild,
        Ik denck dat gij zwaar tilt // de wereld heel te laten,
        Maar of gij die behilt // wat zout u konnen baten.
      
      
        ‘het Is doch al ijdelheid // wat de wereld hoog acht,
        Rijkdom haast onder leyd // hoe hoog dat schijnt haar macht
        Al wast Konincx geslacht // het moet al ter Aarden onder,
        Dat dit niet meer bedacht // en werd, dat is groot wonder.
      
      
        Den onderganck is zwaar // van sulcken dwazen rijk;
        Van zulk een arts dienaar // die ‘het hert van God afwijck,
        Al tot dat aardse slijk // al wilt gij ‘het nu niet weten,
        n Zal haast komen, gelijk // een val-strick neer ghesmeten.
      
      
        Dan zult gij worpen heen // u gouden Goden rond,
        Die hier u troost wel scheen // maar dan in den afgrond
        Laat dalen u terstond // o mens! wilt dit bedenken,
        Laat af u kwade vont // God wil u beter schenken.
      
      
        Wilt gij dat beter zien // dat God u schenken zel;
        Wilt gij weten in wien // gij dit zult vinden wel:
        In Christo, haast u snel // ziet zijn leven, manieren,
        Zijn woord, waerheydts bevel // daar in zal hij u stieren.
      
      
        Toeft niet, beit niet, o mens // in dit Blom-hof te gaan:
        Hier vint gij des levens wens // haast u, wilt niet stil staan:
        ‘het Moet hier doch al vergaan // maar Gods rijk zal beklijven
        Gaat voort door de enge baan // zo zult gij eeuwig blijven.
