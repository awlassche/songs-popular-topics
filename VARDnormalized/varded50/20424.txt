
    
      
        Vrolijke geesten // die liever zijt
        Op Kermis-feesten // als in den strijdt,
        Is het u behagen // rijt vaardig uit,
        Treet op de wagen // of in de schuit,
        De vrienden van het spar // die nemen u waar,
        Weest wellekom gasten // Wel benje daar.
      
      
        Ziet wat hier binnen // Den kramer doet,
        Om wat te winnen // Veylt hij zijn goed,
        Den waeffel-kraemer // Ik heb zeid hij
        De beste kamer // Gaat niet voorbij.
        Dus wordje van de een // En de ander gebeen,
        Om dat-je zout // Jouw geld besteen.
      
      
        V geld verhandeld // u tijd besteedt,
        Na buiten wandelt // En daar wat eet,
        Dan met u Liefje // daar weer in het groen,
        Steelt Snobbel-diefje // dan zoen op zoen,
        Of graest u bemint // zoekt boompjes geblint,
        Of doet daar gij // vermaak in vindt.
      
      
        Leit dan dat meisje // daar gij mee gaat,
        Na duin een reysje // of zo ze praat,
        Haar zwakke leden // niet toe vertrouwt,
        Zo veer te treden // blijft dan in het hout,
        Of geeft ze haar stem // na Romen het heeft klem,
        In Emaus, of Jerusalem.
      
      
        Ziet daar de brackjes // van Sparen-stad,
        Drinken niet zwakjes // het Haerlemsche nat,
        Bij drie vier glaesjes // van bier, of wijn,
        Nooit fraeyer baesjes // wout altijd zijn,
        Een vijf of zes vaan // kan dan daar niet schaen,
        het Is kermis // het moeder nu opstaan.
      
      
        De geest moet woelen // van buiten weer,
        Straks na de Doelen // elk zet hem neer,
        Daar weer gedronken // tot dat ze sluit,
        Smorgens beschoncken // de poort weer uit,
        Daar menig in het wout // in het veld bedout,
        Of op een bank // zijn nacht-rust houdt.
      
      
        De een zijnde wakker // geeuwt als een koe,
        De ander zijn makker // ziet wonder toe,
        Hoe duiker kom ik // dus op een bank?
        Wat drommel droom ik // het is door den drank,
        Waar kwam ik dus krom // ik wed dat ik om
        Het jaar niet weer // te kermis kom.
