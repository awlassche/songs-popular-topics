
    
      
        Aardig rein // gelaat,
        In het wezen amoreus,
        In geen plein // en staat
        Zulk bloemken pretieus,
        Schoon maget smal,
        Die wezen zal
        Geheel, en al // mijn vreugd:
        Laat mij (bij uwer deugd)
        Met u eens zijn verheugt.
      
      
        Een eerbare // bruyt
        Zijt gij, die wonder doet:
        Als in het klare // luid,
        V voysken honig-zoet:
        Mijn droufheyt groot
        (Zo zwaar als loot)
        Als van der dood // verrijst,
        T'is recht dat men u prijst,
        T'bevinden zulks bewijst.
      
      
        Laat dan henen // gaan
        Vileyne kwade becx:
        Houdt verdwenen // staan
        T'gepeyns vol van gebrecx:
        Met u alleen,
        (En anders geen)
        Salt algemeen // doch zijn,
        Cause hier van, is dijn
        Goddinnelick aanschijn.
      
      
        Kiest (O vrouwe // reen)
        Geen ander lief oprecht,
        Want in trouwe // geen
        Zo vast, als uwen knecht:
        Mijn hert geheel
        (Als een juweel)
        Zal u tot deel // gezien:
        Dus lief, wilt mij bij dien
        Genadelijk aanzien.
      
      
        Est dat heden // mijn
        Versouck, u eens behaagt:
        Geen voorleden // pijn
        En werd van mij beclaeght:
        Nooit heb ik schat,
        Zo lief gehad,
        Als u rein vat // gij zijt
        Van God (tot aller tijd)
        Meer dan gebenedijt.
      
      
        Neemt (Princesse // vrij)
        Dit lied, in u geduld:
        Mijn Meestresse // gij
        Voort altijd wezen zult:
        Des herten gront,
        Getuigt den mond,
        Waar bij gij cont // verstaan,
        Dat dit simpel vermaan
        Wt liefde is gedaan.
