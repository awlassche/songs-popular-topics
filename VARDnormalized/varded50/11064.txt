
    
      
        EEn  dankbaar liet // der Heren Heer aenbiet
        Die op ons neder ziet // om weder te ontladen
        Van t'groot verdriet, dat velen bracht te niet
        Door zwaren ziekte hiet // zeer kwalijk omversaden
        En merkt hoe zeer //
        Dat God den Heer //
        De Mensen teer // verblijden,
        Als hij zand neer //
        Ghenade weer //
        Wie zal zijn eer // in dezen nieuwe Iaar belijden.
      
      
        Wij die in noot // van hulpe scheenen bloot
        En telcker uur de dood //zeer broesselijck wacht
        Door vreze groot // veel volckr ons ontbloot
        Om sulcken zwaren stoot // t'ontgaen met haar geslachten
        Maar onverstand //
        Doet buiten t'lant
        Haar ook met sant // verzuchten,
        Want Godes hand //
        En kan niemand//
        Hoe zeer vayliant // tot gener tijd van het Iaar ontvluchten.
      
      
        De mens, gemien // wel spoedig tracht t'ontvlien
        De plaatsen daarmede zie // dat God bereid zijn plagen
        Maar t'punt van dien // waar door de straffen schien
        Wij uit ons niet en wien // dat meest is te beklagen,
        Wou elk met boet,
        Het kwaad verwoed,
        Wt zijn ghemoet // verdrijven,
        De Here goed,
        Liet metter spoed,
        Die straf ontsoet // in het nieuwe Iaar wel achter blijven
      
      
        O Mens verheft // zijn lof die eeuwig leeft
        Door dien ghi met en bleeft, in dotelick verstrangen
        Zijn hulp aencleeft // ootmoedig u begeeft
        Voor hem daart al voorbeest, dat leven heeft ontvangen
        De wijl hij staat,
        En u omvat,
        Om troost en baat // te geven
        Die zich verlaat,
        Op zijn ghenaet,
        En metter daad // in het nieuwe Iaar zijn woord beleven.
      
      
        Hierom besnijt // u hart en ziel met vlijt
        En wilt tot alder tijd // des Hemels Prins verbeyen
        U schuld belijd // en maakt u alles kwijt
        Dat tegen t'ghoede strijdt // zo suldy u bereyen
        Om met bescheid //
        De Zaligheid,
        Daar elk na beit // te horen,
        Als Christus zeid //
        Werd in geleid //
        T'rijc is bereid // voor u mijns vaders wt verkoren.
