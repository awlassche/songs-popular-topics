
    
      
        Plag mijn jonk hert
        Dus te verzuchten // zwaar,
        Gelijk het nu begint:
        Zo dat mij smert
        Volgt, met veel duchten // naar,
        Om een maagd wel bemint:
        Geen zo gezind
        Te voren // ooit,
        Geen schoonder kind // en is geboren // nooit.
      
      
        Een sulcken last
        Doet mij der minnen // god,
        Dat ik vertwijfeld sta:
        Of zij mij vast
        Zal doen gewinnen // t'lodt
        Daar ik om dolen ga,
        Ik overslae
        Mijn smerten vrij,
        T'blijft even na // geen troost en is voor mij.
      
      
        Troostbaerder was
        Zij ten beginne // dan
        Zij mij nu dunkt te zijn:
        Hoe kan zo ras
        Sulcken vriendinne // van
        Mij keren haar aanschijn?
        Meerder gepijn
        Doolyck bereid,
        En heeft nooit mijn // jonk hert neder geleid.
      
      
        Roup ik om troost
        Wie zal verhoren // mij,
        T'is al om niet gewend:
        Voor mijn propoost
        Verstopt haar oren // zij,
        Rebel, en hert versteend:
        Ziet mij vercleent
        Arm serviteur,
        Ander verleend // zij troost, en mij rigeur.
      
      
        Van Antheros,
        Is zij deurschoten // sterk,
        Wiens schickt verkoelen biedt,
        Dies Atropos.
        Met een gesloten // zerk
        Fineert al mijn verdriet:
        Misdeed' ik iet,
        Dat zij mij haat,
        Och neen ik niet // dan een rein hert toestaet.
      
      
        Scheyd' ik dan met
        Een deerlijk schromen // ach?
        Wt dezen wederspoed:
        Van zulk op zet
        Zij haar beromen // mag,
        Als een maagd straf verwoed,
        Mijn rein gemoet
        Niet falen // zal,
        Dwelck met zijn bloed // t'rantsoen betalen // zal.
