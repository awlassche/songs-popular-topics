
    
      
        Mijn hert is vol verdriet
        Om het geen dat mij geschied
        Van een vrouw // die mijn rouw // noch klagen
        Acht steeds veel min als niet,
        Ik offer haar mijn ziel
        Met smeken en gekniel
        Om haar min // daar ik in // door het behagen
        Straks plotselijk verviel.
      
      
        Kan zo een Dochters hert
        Aanzien eens Minnaers hert
        Door het gevlei // noch geschrei // bewegen
        Maar staegh noch wreeder werd,
        Hoe zeer mijn geesje steent
        En klaeght of droevig weent,
        Niet en hoort // maar verstoort // en tegen
        Verdooft, van mij ieder woord.
      
      
        Nu wens ik dat ik nooit
        Haar lichaam opgetoyt
        Haar gelaat // noch haar praat // noch leden
        Die mijn zin maakt verstroyt,
        Voor dezen had gezien,
        Maar hadse mogen vlien,
        Zo souw mij // los en vrij // nu heden
        Deze pijn niet gezien.
      
      
        Want mijn verlieft' gemoet
        Zij sulcken smert aendoet,
        Ja mijn geest // is bedeest // vol treuren
        Door al de tegenspoed
        Mijn hert innerlijk jaagt,
        En zeer weemoedig klaeght,
        Nu zij mijn // laat in pijn // wilt scheuren
        Van al de ramp die het draagt.
      
      
        Och of den hemel wou
        Tot teiken van mijn trouw
        Ik om haar // straks nu maar // mocht sterven,
        En einden zo mijn rouw,
        Zo waar mijn luk noch groot,
        Want dan souw mij de dood
        Maken kwijt // deze strijd // doen beerven
        Alle pijn ramp en nood.
