
    
      
        MEt Brood, met Bier, zeer fijn // met Wijn,
        Pastay // Lampray // met haas Konijn,
        Met Hoen // Kapoen // Wilt-braedt // verzaad,
        Elk is // gewis // op dit termijn.
      
      
        Ons hert is heel met vreugd, verheugt
        Ons moet // zeer zoet // strekt tot gheneucht,
        Wij zingen // en springen // wij klinken // en winken,
        Versterkt O Heer ons ieucht // in denkt.
      
      
        Men ziet hier staan // gesoon gebraen
        Wie spart // die taart // die is verlaen
        Marsepain // niet klein // Men begint // het Hindt,
        Die Haan // die Zwaan // Kan niemand schaen.
      
      
        Wij danken v veel meer // O Heer
        Dan wij // zeer blij // v schaepkens teer
        Met monden // verkonden // versterkt // en werkt,
        In ons nu voort // v woord // en leer.
