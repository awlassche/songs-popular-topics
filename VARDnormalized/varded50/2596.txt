
    
      
        T'vertoont zich Vrouwe // in alder ootmoed
        U Lief getrouwe // en Dienaar goed,
        Met weenend' ogen // bedenkt, beschaamt
        Verzoek meedoogen // Prinses vernaamt,
        Mijns lijder smart
        Des wetten hart
        Zijn vels te vart,
        Van een goed Maakt befaamt.
      
      
        Ik onderdanen // heb nacht, op nacht
        Veel zucht en traanen // ja klacht, op klacht,
        Ghestort, gegoten // met bitter getreur
        Maar nooit ontslooten // u wreede deur,
        Dit klaag ik God
        Want treed' ik tot
        Het staale slot,
        En grendels synder veur.
      
      
        Kom ik behendig // ontrent u gelas
        Mij ontmoet schendig // snel het gebas
        En t'gerucht der mense // dat ik zeer haat
        Mijn Vrou ik wensche // en bid ghenaad
        O quamt // ik namt
        Vant t'hart dat vlamt
        Geeft dat dit ambt,
        Maar voor mij open staat.
      
      
        Zo uit ghenaade // des jonst gheschiet,
        Ik zal die weldaade // vergeten niet,
        Maar die beloone // met danckbaarheyt
        Met Lof betoone // ben ik bereid,
        Tot alder stond
        Doort Heil verbond,
        Met hert en mond
        Werd uwen naam verbreyt.
      
      
        Voor u wel vaare // Prinses eerbaar
        Stelt hij t'syn Gare // brand u Dienaar
        In jonst dus vierich // weg arech vermoen
        Zijt gij eergierig // zo wilt u spoen
        Na dit rekwest
        Zo doet u best
        Ik sech in het lest:
        Dus doende zult wel doen.
