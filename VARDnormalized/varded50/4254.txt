
    
      
        PEerle der vrouwen // zoet //
        Versierd met aller deugden //
        In dijn aanschouwen // moet //
        Den geest vrolijk verjeugden //
        Noch even zeer //
        ben ik dijn knecht volveerdich //
        Gij zijt het mij weerdig //
        Al waart duizend maal meer.
      
      
        Inwendig lijd' ik // grief,
        Van buiten toog ik vreugde:
        Zelden verblijd' ik // lief,
        Want gij (die mij verheugde)
        In mijn dangier,
        Zijt als Daphne voor // vluchtig,
        Nochtans gij door // luchtig,
        Zult blijuen mijn laurier,
      
      
        Eertijds dijn herte // louch
        Op mij, als lief tot lieve:
        Ik (die nooit smerte // drough)
        Ben nu vervult met grieue:
        Mijn bloed verschrikt,
        Mijn ieught versmelt in pijne,
        Cruyt, noch Medecijne,
        Mijn droevig hert verkwikt.
      
      
        Tis doch verloren // werk
        Geclaeght, ik ben in handen:
        Ik zie dijn toren // sterk
        Te mij-waarts vierich branden:
        Onzalig man,
        Wat deed' ik ooit geboren,
        Zij wil mij versmoren
        Die mij meest helpen kan.
      
      
        Es mijn geen wrake // noot
        Van tgeen ik nu verhale:
        V is d'oorsake // groot,
        Vilein en teenemale:
        Gij waart gelooft
        Te spreken u belieuen,
        Wreede mensen dienen,
        Gij hebt mijn lief geroofd.
      
      
        Ras op, het scheiden // naakt,
        Adieu, lief ongestadig,
        Die in het verbeiden // haect
        Naar troost, weest hem genadig,
        Neemdy nu aan
        Te zijn eens anders vrouwe,
        Tocht hem meerder trouwe
        Dan gij mij hebt gedaan.
