
    
      
        HOe lang is het heden // schone Iuliaan,
        Doch wel geleden // dat gij hebt voldaan
        Op rode tipjes // duizendmaal gekust,
        Van zoete lipjes // onze minnelust?
      
      
        het Was toen u woorden // en geveinsde Tong,
        Mijn hart bekoorden // en tot kussen dwong.
        het Was toen de vlamme // van u geile Ziel,
        Mijn tegen kwam, en, in mijn armen viel.
      
      
        Wat hebt gij strikken // listig uitgeleit,
        Om te verstikken // mijn onnozelheit.
        Hoe kost gij vleiden // midden in die vlam,
        Die strak na het scheien // bij een ander kwam.
      
      
        Neen Iuliane // denk vrij dat mijn Min
        Haar eerste tranen // offert mijn Goddin:
        Geen Liefde, zweer ik // dat van Iuliaan,
        Zal het hart van Fredrik // in haar vlamen braan.
      
      
        Ik acht geen Eeden // die een Minnaar doet:
        Zo vaak vertreden // met een losse voet.
        Zo lang de dagen // zijn van Iuliaan,
        Zal haar u klagen // tegen het harde staan.
