
    
      
        HOe mag, o God, het volk zo razen, koen,
        En Mensen dus uws Woords vruchtbaarheid // mijden,
        Te zwak eilaas zij doch als dwazen // doen
        Die tegen dij en dyne waarheid // strijden,
        V heilig kind en kan geen swaerheyt // lijden
        Meer door vervolg, dan aan zijn leden // teer,
        Zo heden doen, die blind u klaarheid // mijden,
        Als Pilaet, ook Herodes deden // eer.
      
      
        Gelijk als daar, in zo versteenden // stad
        Ierusalem, Mensen genegen // waren
        Leed doen u kind en volk, ja meenden // dat
        Te roepen uit, met een fel tegen // varen,
        Gaat het noch toe, dan gij door zegen //baren
        Wt eeuwig zaad u noch veel kinders // laat
        Des laat gij noch om wel bewegen // naren
        Heer, uwen Geest, die in veel hinders // baat
      
      
        Doe Farao ook meend' uit te roeien // fel
        Israel wies door u vermogen // krachtig
        'sGelijcx u volk heeft men zien groeien // wel
        Daar doch al veel benynders loghenachtigh
        V maert' eilaas zeer loos bedrogen // zachtig,
        Dat zij het zweerd van u gegeven // uit,
        Om doden toog, en houdt getogen // machtig,
        Op u huysvrou u zeer verheven // Bruid.
      
      
        Maar Syons Vorst zal in de wolken // klaar
        Zijn Bruid tot troost zeer haast verschijnen // horen
        Zal men geschrei, want alle volken // daar
        Zeer zullen zijn bevreesd voor zijnen // toren,
        Vergolden werd al dobbel 'tpijnen // voren
        V volk gedaan, met brand en enen // rook,
        Die kwellig zal zonder verdwijnen // smoren,
        Ia eeuwig werd daar druk en wenen // ook.
      
      
        De Bruid zal hoog uit d'Ardtsche banen // gaan,
        Want bij zijn hoofd moet zijn t'lichame // boven
        'tVerwinnigh Lam zal haar van tranen dwaan,
        Aan slevens born' in die bekwame // hoven,
        Gerust zij zal eeuwig den Name // loven,
        Met blijdschap groot, van haren werden // Man,
        Maar vlucht, en zucht, in onveylsame // kloven
        Den rootsten, gij Tyrans op eerden // dan.
