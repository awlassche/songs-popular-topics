
    
      
        Aenhoort wat Marcus schrijft // van Jezus wandeling
        Na ‘het Doopsel hij beklijft, in Satans bekoring,
        Symon zijn Net afging // onreine geest moet vrezen,
        Die koorts’ oorlof ontving Melaatsen wordt genezen.
      
      
        Boven door ‘het dak een Gicht // afdaelt en werd gezond,
        Levi zijn voeten licht // medecijn kranken vond:
        Vasten is gheen verbond // Sabboth is heel gebleven
        Als ‘het Koren met den mond // door honger werd gewreven.
      
      
        Klein tijd hier na geneest // hij een verdorde hand,
        Raad is daarom geweest // doch hij wijkt aan een kant:
        Verkiest tvvaelef Ghesan’ // en des Belzebubs treken
        Duiden zij hem tot sant // zijn Moeder wil hem spreken.
      
      
        Door zaads gelijkenis // hij ‘het rijke Gods voorstelt,
        Driederley hinder is // van Mosterd hij vertelt,
        ‘het Licht nummermeer beweld // wie heeft, die wordt gegeven
        Tempeest het schipkan quelt // het welk hij heeft verdreven.
      
      
        En komend’ aan het Land // verdrijft hij ‘het Legioen,
        Die haast hebben vermant // de verckens in het groen:
        Den bloedt-ganck krijgt versoen // Dochter van tvvalef jaren
        Des Oversten als doen // brocht hij uit ‘s doods bezwaren.
      
      
        Flauw was zijn kracht bekent // bij zijn Lands-volk verdooft
        Apostels hij uit zendt // Johannes wordt onthooft:
        Den honger wordt berooft // van vijf duizend. Hij wandelt
        Op zee: en wie geloofd // in zieckt zijn Zoom behandelt.
      
      
        Ghevvasch van hand en mond // van binnen niet en baat,
        En door der mensen vond // men Gods gebod verlaat:
        Hondeken wordt verzaad // van de afgevallen kruymen:
        Den doven hij ontslaet // van zijn gehoor ‘het verzuimen.
      
      
        Hier na vier duizend Mann’ // verzadinge geniet:
        Die teiken zoeken an // van haren desem vlied;
        De blinde bomen ziet // Petrus komt hem belijden,
        Die hem hier schamen niet // zal hij naemaels bevrijden.
      
      
        In klaarheid hij vertoond // Elias komen zal:
        Gezeten hij verschoont // die schuyn’ en wenteld’ al,
        Zij ‘het Kindeken houdt stal // en leert ootmoedigheden,
        Niemand brenget ten val // door ergherlijcke leden.
      
      
        Kracht heeft nu geen Scheidbrief // Kinders hij benedijt:
        Iongh’lingh heeft rijkdom lief // loon vint men in des tijd:
        ‘het Hoog zitten maakt veel spijt // dus zijnt Heydensche werk
        Bartimeus met vlijt // het daglicht kan bemerken.
      
      
        Looft Syons Koning dan // die zit op d’Edelin,
        Den Vijgeboom en kan // gheen vruchten brengen in,
        Jezus en heeft gheen zin // aan Wisselaers in Tempel,
        Zij eisen machts begin // hij wijst des Doops exempel.
      
      
        Met de Wijngaerdeniers // haar ontrou hij bewijst,
        De Phariseen vol viers // vraeg’ waarom men Exyst,
        Hoe naemaels men verrijst // ook na de hoogste Lere,
        de arme Wedyvve hij prijst // Davids Soon is zijn Here.
      
      
        Neder op de olijf berg // zittende hij voorzeid,
        Van zeer veel secten erg // die menig Mens verleid,
        Ook Oorlog uytghebreyd // eer zijnen dag zal naken,
        Dus met voorzichtigheid // beveelt hij haar te vraecken,
      
      
        Op hem men Zalve giet // Brood breking en Paaslam,
        Hij met de zijn geniet // Droefheid hem over kwam,
        Gevangen werd hij tam // voor Cayphas lijdt hij schennis,
        Die hem te kwellen nam // Petrus ontkent zijn kennis.
      
      
        Pilatus zaad den haat // doen men hem ‘het kruike boot.
        Hij woud’ hem doen genaed’ // maar de ander hoop was groot
        Tussen twee Moorders snood // is hij ghekruyst gestorven,
        Hij heeft ons dood, gedood // en ons verderf verdorven.
      
      
        Kwalijk den dag aenbrack // de Vrouwen zijn verschrikt,
        Den Engel met haar sprak // de klaarheid Iesu blickt,
        Doch werden zij verkwikt // als hij met haar kwam spreken,
        Als hij ‘het al hadd’ beschikt // is ten Henel geweken.
