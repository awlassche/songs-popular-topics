
    
      
        GOd verlaat zijn dienaars niet
        Schoon hij haar laat proeven veel verdriet.
        Na het leed // is zijn heil haast gereed,
        Om hun hert' // weer te stellen buiten vrees en smert.
        Des is het gemoet // hetwelk hier doet // alle goed,
        Wel gesterkt // als het Godes aard bemerkt.
        Want hij laat // zelden het quaat, het geen hij haat,
        Zoo veel macht // of zijn troost heeft 'et eerst verzacht.
      
      
        Jezus liet in des doods gevaar,
        Zoo het scheen, zijn kindren allegaar.
        Maar zijn macht // is daar na weer volbracht,
        Als het graf // na drie dagen, hem weer levend gaf.
        Op het Pinksterfeest // bleek het noch meest // aan de geest
        Die hij zand // tot een heilig onderpand.
        Hun geween // was nu heen // en met ene
        Zagen zij // dat Hij haar bleef gestadig bij.
      
      
        O wat vrolijkheit en vreugd,
        Wat vermaak heeft hun de ziel verheugt
        Die den Heer // zoo getrouw in zijn leer,
        En zoo goed // ondervonden in haar tegenspoed!
        Al het zwaar verdriet // eerst geschiet // trof haar niet.
        Spot en smaat // noch der Joden bozen raat;
        Hoe gekweld // geen geweld // heeft ontstelt
        Hun gedacht' // deur den troost die het was aangebracht.
      
      
        Zeldzaam kwam in yders oor'
        Haar verstand zoo wonderlijk te voor.
        Galileen // was bekent in het gemeen,
        Voor geen troon // daar de wijsheit heerlijk stond ten toon.
        D'onwijste schaar' // heeft van haar // openbaar
        Wel gezeit // het is de wijn die haar verleit.
        Maar dien het oog' // niet bedroog' // ziend' om hoog,
        Was bevreesd // door het gehoor van 'er grote geest.
      
      
        Maar, o redenloos besluit!
        Steekt u oog' niet heel moedwillens uit;
        Hier is het woord // nu vervult; het welk, gehoord
        Uitten mond // des Profeten, rust' op dezen gront.
        God, die het gemoet // stromen doet // als een vloed,
        Heeft zijn macht // openbaar in het licht gebracht.
        Elk zie toe // wat, en hoe // hij nu doe:
        Want Gods Zoon // heerst nu zelf in zijn des vaders troon.
      
      
        Dus heeft ieder in zijn taal
        Aangehoord een Goddelijk verhaal,
        Dat hun ziel // in de gront wel beviel,
        En, belaan // dede vragen; ach! wat diend gedaan?
        Zij zeiden doet // rechte boet // in het gemoet;
        Vliet van het quaat // met een onvermoeiden haat:
        En dan voort // na zijn woord // zoo het behoort
        Laat gezien // dat men u het heilig Doopsel dien'.
      
      
        Menig, als dit was gezeit,
        Was terstond tot dezen dienst bereit.
        En het vermaan // de het getal groeien aan
        Op dien dag // bij drie duizend die men dopen zach.
        Dus onder een // zeer gemeen // en te vreen
        Wel gerust // met vermaak en zoete lust,
        Bleef die schaar // met malkaar // lang daar naar
        Eens gezind // het welk men nu weinig ondervint.
