
    
      
        SChoon lief mij niet versmaad // maar laat // v oude wegen,
        Dat gij dus obstinaat // mijn scaet // in liefden tegen.
        Vernieut uwes sins ghemoet // en doet // mij blijde mare
        Zo sing ik v met spoed // een groet In het nieuwe Jare.
      
      
        Wilt gij in mijn gheween // na reen // mijn drucknontladen
        Zo wordt mijn hart besneen // alleen // door v ghenaden,
        Indien gij niet aanziet // t'verdriet // van v dienare
        Zo wordt een droeuich liet // bespiet In het nieuwe Jare.
      
      
        Zo gij v niet verneert // en keert, v stramme zinnen,
        Zo wordt ik door begeert // verteerd in het vier der minnen
        Ik maak aan Venus macht // mijn klacht voor den Outare
        Dat gij mij dag en nacht // veracht, In het nieuwe Jare.
      
      
        Prinses van dit rapport // en stort v niet in dezen,
        Maar spreekt eens met akkoord // een woord, ik zal genezen
        Gunt dat ik voor mijn vrou // v hou, in het openbare,
        Zo lost gij mijne trou // van rou In het nieuwe Jare.
