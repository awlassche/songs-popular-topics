
    
      
        Ik heb bedacht // van het geslacht
        Dat dag en nacht // neerstich verwacht
        Gij nieuw kracht // van God te cryghen:
        In haar gevecht // recht ende slecht
        Gelijk Iob zegt // haar hope lecht
        Op hem wiens recht // niemand mag bedriegen
        Als met Arends vleugelen zij opvliegen;
        Zij lopen en wandelen en worden niet moede:
        Maar haar vijand // die raakt in sant
        Aan elken kant // na Schrifts verstand;
        Doch Godes hand // is haar behoede.
      
      
        Op Noachs daad // nu denken gaat
        Hij vont ghenaet // door Godes raad
        Voor zijnen saet // en wordt verheven:
        Die wereld rond // die wederstondt
        Des Heren mond // bleef in haar zond
        Gingen te grond // verloren het leven;
        Die heeft God beide door het water bedreven:
        Dat vier deed ook die vrome gheen hinder,
        Sodoma ziet // en die Lot riet
        En hoorden niet // bleef in het verdriet
        Maar hij ontvliet // met zijne kinder.
      
      
        Heel Israel // leefden rebel
        Godes bevel // hij verdeed snel
        Haar met gekwel // zes honderd duizen
        De een afgod eert // de ander hoereert,
        Die derd' temteert // vierd' murmureert,
        Die vijd' begeert // menig abuysen,
        Zo verginghen ook al haar geslachten en huizen:
        Maar Iosua en Caleb en de ander haar jongen
        Zijn wel gespaard // van God bewaard,
        Van bozen aard // zo Schrift verklaart,
        En zij vergaard // van vreugde zongen.
      
      
        Al die niet vreest // in dit foreest
        Mits dat men leest // dat altijd meest
        Kwaad zijn geweest // zijn buiten gesloten,
        Die God bemint // en recht begint
        Geestelijk ghesint // hem zelfs verwint
        Blijft hij ontrint // haar medeghenoten
        Die vrome te bewaren heeft hem nooit verdroten:
        Die boze brengt hij tot straffe ook klaarlijk
        Nu 'tlicht doorbreckt // 'tvuyl wordt ontdekt,
        Nu ruymt die sect // Gods volk begeckt.
        En maakt suspect, gants openbaarlijk.
      
      
        Neemt hier op acht // hoe dat elk wacht,
        Met list bij nacht // bij dag met kracht,
        Gods volk met smacht // te brengen in sneven
        Die Salmists reen // tnegentig een,
        Acht dit niet cleen // troostet gemeen,
        Allen den gheen // die haar begeven,
        Onder die bescherminge des almachtigen te leven
        En onder zijn schaduwe hem aldus aanroepen
        Mijn Heer mijn God // mijn burg mijn slot,
        Mijn deel mijn lot // wij Iacobs rot,
        . Na u gebod // op u wij hoepen.
      
      
        Niet een vergat // in het strik ghevaet
        Die hem verlanct // op Godes raad,
        Maar t'sijnder baat // zal men verheffen,
        Al viel in het sant // aan u zijds kant,
        Duizend ter sant // ter rechter hand,
        Tien duizend want // t'sal u niet treffen,
        Maar zult u lust zien hier beneffen,
        Ook hoed den Godlozen werd vergouden,
        V zal gheen schat // door iemands daad,
        T'eenigher maet // vroeg nochte spaat,
        Aankomen kwaad // wilt dit onthouden.
      
      
        Elk neem noch waar // gans openbaar,
        Dat bij Gods schaer // druk angst en vaar,
        Bangigheid naar // vaak is t'werven,
        Maar zulk verdriet // werd niet bediedt.
        Schriftuur aanziet // dat het gheschiet,
        Om haar tot niet // gants te bederven,
        Wij hadden besloten zeid Paulus te sterven
        Voor den druk die ons in Asien bedekte,
        T'welck ons gheschach // op dat elk mag
        Denken die dag // doen hij aensach,
        Al ons gheclach // die de doden verwekte.
      
      
        Sulcken dood bank // in haar aenvanck,
        Stijf ende stranck // al duerdse lank,
        God noch bedwanck // tot onze lere,
        Gedenkt ook meed' // een wijze reed',
        Die Iudith deed' // in haar ghebeed',
        Ook t'hooft af sneed // van den Veld-here,
        Tot die ouders des volks nam zij haar kere,
        Men mag den Heer tijd noch dagen aanreden
        Lijd' wij gekwel // vermaentse snel
        Exempel stel // Abraham wel,
        Izak Israhel // en Moyses veel leden.
      
      
        Ia allegaar // die God lief waar,
        Perikel zwaar // kwam over haar,
        En bleven daar // trou zonder crakielen,
        Maar die in noot' t'zy klein of groot,
        Door elk aanstoot // oft valse boodt,
        Ook in den dood // van God vervielen,
        Ginck hij door den verderver vernielen,
        En zal hem noch aan zulke wreken,
        Daarom zo zult // gij met geduld,
        Kennen u schuld // op dat Gods hult,
        In u vervult // nimmermeer ghebreecken.
      
      
        Al die God vreest // gedachtig weest
        Dit zwaar tempeest // minst ende meest
        Hoe het Davids geest // dus gaat belijden:
        Ik prijs u Heer // dat gij mij weer
        Verhooghet zeer // door ws-naems eer
        Mijn vijand neer // niet laat verblijden;
        Doen ik u aanriep Heer gij waart t'mijnder zijden
        En ghinckt mijn ziel voor het verderven bewaren:
        En hebbet mij // levendig gij
        Behouden blij // daar alle zij
        Ter Hellen zij // zijn neder gevaren.
      
      
        Komt Heyl'ghen teer // lof zingt den Heer
        Dankt prijst zijn eer // want zijn toorn zeer
        Die duert gheen meer // maar ons verheugde
        Den avond lank // gheduerde stranck.
        Het wenen bank // maar nam een ganck
        Smorgens vrij vranck // zo kwam die vreugde:
        Maar ik sprak doent mij wel ginck in deuchde:
        Ik zal nimmermeer ter neder vallen;
        Want Here goed, door u behoed
        Met rechter spoed // hebt gij mijn moet
        Als een berg zoet // sterk maakt met allen.
      
      
        Och doen gij dijn // heilig aanschijn
        Berchde voor mijn // och watte pijn
        Tot dien termijn // verschrikte ik seere;
        Och here God // wij Iacobs rot
        Waren tot spot // zonder u lot
        Na u gebod // wij smeken den Here
        Wat is er nut aan mijn bloed nu meere?
        Want na mijn dood zal u 'tstof ook danken:
        Here toch hoort // na u akkoord
        En heilig Woord // zo helpt ons voort
        Tot aan dien oord // daar wij na janken.
      
      
        Boven dit al // mijn plaech misval
        Veel in het getal // meer dan ik kal
        Hier in dit dal // hebt gij verandert,
        In rei verhoogd, want gij vermoocht
        En mij uit tocht // den zak verdroogd
        Begort niet vroocht // als een vast standert
        Op dat al die in deuchde wandert
        Vwen lof zingen met eerbare zeden;
        En uwen eer // toch nimmermeer
        Stilswyghen weer // zo zal ik zeer
        Danken mijn Heer // in eeuwicheden.
      
      
        Wie dit nu vaat // en meent hij staat
        In rechter maet // in denkt voortgaet
        Ziet toe met raad // dat hij niet  valle
        Niet stout van geest // nu zijt maar vreest
        En zelver leeft // hij is geweest
        Streng over meest // die natuerlijcke alle
        Die poort is eng' die weg is smalle,
        Hij zal ons niet ontzien noch sparen
        Lucas dertien // gaat wel bezien
        Van deze twieÃ«n // Galcese lien,
        En noch achttien // die tot Syloa waren.
      
      
        Zo wie gheen boet // oprecht en doet,
        Hem neerstich spoed // altijd in het goed,
        Die zelfde moet // zo gaan te gronde,
        Leeft noch voort daar // hij zeide haar,
        Doen allegaar // een byspil zwaar,
        Op dat elk klaar // die zake verstonde,
        Een had een vijgeboom geplant hij vermonde
        In zijn wijngaard daar hij vruchten of zochte,
        En van der gheen // sprak hij gemeen,
        Met kloeke reen / drie jaar aan een,
        Met kwade zeen // gheen vruchten hij brochte.
      
      
        Dus hout hem of, dat wilde stof
        Is veel te grof / te hebben t'lof,
        In s'Heeren hof // als vruchtbare bomen,
        Merkt hoe ik las // hij antwoordt ras,
        Laat hem dit pas // staan als hij was
        Ende in cas // gheen vruchten en komen,
        Na dat ik hem neerstich heb waar genomen
        Met graven en messen na terecht der vromen
        Daar na met vlijt // maakt hem u kwijt,
        Zonder respijt // op dat gheen tijd
        Elk ypocrijt // met bladeren staat roemen.
      
      
        Ei neemt nu waar // elk ghetals jaar,
        Zeven en vijftig daar // twee sessen klaar,
        Twee achten zwaar // wat gingher te gronden
        Komt nu iemand // roemrijk ter hand
        En zeide want // zij zijn te sant,
        Ook al van kant // door hare zonden,
        En wij zijn noch oprecht bevonden,
        T'blijckt wij staan in des Heren garde,
        Daar stond hij meed // verhaalt alreed,
        Drie jaar men leed // noch een ter bed,
        Vervult een steed // maar niet en baarde.
      
      
        Dat wij noch staan // den dan ontfaen,
        Is niet gedaan // om ons recht gaan,
        Maar God ziet aan // hoe wij ons bekeren,
        Die nu vertraagt // t'zy knecht oft maagd,
        Jonk oft bedacht // God niet behaagt,
        Maar van hem jacht // na Salomons leren
        Daarom laat ons die lanckmoedicheydt des Heren
        Voor onze meeste zaligheid betrachte
        Wie niet is kloek // in elken hoek,
        Die krijgt een vloek // dit zelfs na zoek,
        In het propheets boek // Ieremias veertig achte.
      
      
        Iammer waart groot // sou gene noot,
        Noch zwaar aanstoot // hoe naakt en bloot,
        Ia veler dood // ons maken verlegen,
        Boven dit al  ons eigen mis val,
        Bitter als gal // clinckt met geschal,
        En't ons niet zal // mengen bewegen,
        Och vraagt na die voorgaande wegen,
        En wandeltse trou zo suldy ontvangen,
        Der zielen rust // in dien gij blust
        Stilt ende sust // des vleysches lust
        Die waarheid kust // al met verlangen.
      
      
        Tot een besluit // laat dit geluid
        Als een goed kruid // als Christus bruyt
        Niet een minuyt // uit hert en zinnen;
        Al u gedacht // bedwingt met kracht
        V neerstich wacht // dag ende nacht
        Voor het Helsche gracht // om te verwinnen
        Als oft gij van nieuws aan wildet beginnen
        Dient God met vrezen verblijd u met beven
        Die Heer is zoet // barmhertig goed
        Ook metter spoed // toornig verwoed
        Want hij elk doet // zulk loon zulk leven.
      
      
        Memorie hout // op nieu en oud
        Christum betrout // hebt altijd fout
        Dat u niet rond // het eeuwigen tijden;
        Niet stout maar vreest // gedachtig weest
        En zelver leest // hoe noch het beest
        Noch listig geest // niet viert van strijden;
        Dus wilt gij eeuwig met God verblijden
        Omgort u lenden u licht laat branden
        En verlangt zeer // na uwen Heer
        Want hij komt neer // met groter eer
        Beveelt u teer // in zijnen handen.
      
      
        Eer God // mijn Heer // het lot // ws eer
        Dus bot // sou weer // als spot // ter neer
        Tot elks begeer // duert kwaad allensken
        Sou zijn // gewend // goud fijn, hoe jent
        Noch pijn // ellent // ten fijn // bekent
        Kwam mijn // ontrent // oft ik sou wensken
        Den dood te sterven // als alle mensken
        Om te verwerven // de ruste stilswyghen;
        Profijt // hier doet // zo gij het // zijt vroed
        Altijd // tot boet // met vlijt // u spoed
        Verblijd // om het goed // eeuwig te krijgen.
      
      
        Prins // wins // niet kwaad // suleys // eis // noch raad
        het Welk // elk // aenvaet // maar // daar // toe gaat
        het Woord // hoort // verstaat // in zijn valeure
        Tot ducht // jucht // blij // tot goed // spoed // dij
        Ook oud // cout // gij // nieman // kan // vrij
        Gods Geest // leest // hij // zel // wel // ter keure
        Bespien // deursien // ellick zijn labeure;
        Ons dagen // jagen // snellijk // na te leste
        Dus u tijd // slijt // waakt bidt strijd // mijt
        Som u smijt // lijt // gedocht spijt // nijt
        Noch verblijd // zijt // diet is die reste.
