
    
      
        WT louter kracht der minnen,
        Claegh ik al mijn verdriet:
        Een maagd doet mij ontzinnen,
        Die mijnen druk aanziet // niet,
        Hoe wel mij hert-zweer // meer
        Dan te veel geschiet.
      
      
        Ik dan aldus ellendig
        Gestoten van der hand,
        Is mij arm knecht noodwendig
        Een troostbaar onderstand // want,
        Niet en coemt hier van // dan
        Steruen in den brand.
      
      
        Laat zij mij dus (met pijnen)
        Zuchten in tegenspoed:
        Mag mij eens troost om-schijnen,
        Laeth' kosten vlees, en bloed // t'moet
        Eerst zijn ongeval // zal
        T'eynde werden zoet.
      
      
        Leun, en dood, achermen,
        Is al aan haar besteet:
        Het ooght zij mij geen ontfermen,
        (Naardien zij mijn secreet // weet)
        Ik blijf een arm schu // nu
        Zij mij dus vergeet.
      
      
        Een rouwig wederstrijden,
        Beclemt vast mijnen geest:
        Al hoop ik naar verblijden,
        T'wert al stormig tempeest // meest,
        Droufheyt, en geween // geen
        Herte zo bevreesd.
      
      
        Moet ik de over-schone
        Princesse, eigen zijn,
        Niet el wens ik te loone,
        Dan t'vriendelick aanschijn // fijn:
        Het is den weerdtsten schat // dat
        Beeldeken divijn.
