
    
      
        Schoon lief playsant,
        Dit nieuwe liet ontvangt
        in liefden triumphant,
        Zal ik v dienaar wezen
        Door Venus brand
        Ontgaet mij het verstand,
        Ontbint mijn dezen band
        Gij muecht mij genezen
        mijn druk versacht,
        Gij hebt die macht,
        Door liefden kracht
        Schoon Lief daar ik na bei,
        Slaat op mij acht
        Verhoord toch mijn gheclacht
        Daar ik na wacht,
        In deze coele Mei.
      
      
        Waart gij eens vroed,
        Wat pijn ik lijden moet
        Gij sijt diet mij doet //
        Nochtans ik v beminne //
        Mijn Lijf, mijn Goed,
        Ja ook mijn t'sherten bloed //
        Is voor v Liefken zoet //
        Hoe wel ik niet en winne
        Maar voelde gij
        eens wat ik lij,
        Zo waar ik vrij //
        Van alle druk in dezen contrei //
        Dan zouden mij,
        Die fantasy,
        heel maken blij
        In dezen toekomende Mei.
      
      
        Schoon Lief minjoot //
        Ik zal ontdekken bloot //
        Mijn lijden zwaar als loot,
        Gij muecht mij verblijden //
        Troost mij tis noot //
        Mijn smart en pijn is groot
        Leuendich ben ik dood,
        Ik sterft tallen tijden //
        Want doen wij iet,
        Het werd bespiet //
        Al eer dat het gheschiet
        Door nyders ingheuen,
        Doet toch te niet,
        Dit zwaar verdriet //
        Welk gij al aan mijn ziet //
        In dezen toekomende Mei.
      
      
        Hoort en verstaat //
        Princesse delicaat,
        Oft v ter herten ghaet,
        Dat gij mij dus laat wachten,
        Niet en versmaad //
        V dienaar vroeg ofte laedt //
        Het is uwen eigen baat,
        Gij muecht mijn druk verzachten
        Zegt niet meer neen //
        Maar ia alleen,
        Maakt van ons twen //
        Een paar in dit gheschraey //
        alle gheween
        Zal hem ontspreen,
        Van ons alzo ik meen
        In deze toekomende Mei.
