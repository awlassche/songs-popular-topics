
    
      
        Jong besneden // vat //
        Bloemken weerdig geeert,
        Och! is het reden // dat //
        Gij mij dus tormenteert,
        Bevroet het: // gij //
        Lief, doet het // mij,
        Versoet het // vrij, want geen //
        En mag al mijn geween //
        Blussen dan gij alleen.
      
      
        ACO leye // baard
        T'genadich herte schoon:
        In den Meye // waart
        Gij lief // mijns hoofts een kroon,
        Genezen // doet
        T'verwesen // bloed,
        In dezen // moet gij zijn
        (Onder der zonnen schijn)
        Mijn enig medecijn.
      
      
        Beleeft, ende // wijs,
        Gestadig rein gelaat,
        Voor ellende // prijs
        Mij geeft, aanziet den staat,
        Mijn leven // bloot,
        Daar neven // dood,
        Is even // groot // geplant
        Liefste in uwer hand:
        Dus doet mij onderstand.
      
      
        MYNder zielen // vreugd,
        In als confortatijf:
        Die vernielen // meugt,
        En weder zinken t'lijf:
        Voor bloedig // strijd,
        Die gloedig // snit,
        Zachtmoedig // zijt // oprecht,
        Die niet en wederleght
        Te zijn dijn eigen knecht.
      
      
        KENt in het laatste // nu
        Dijn Prinse, teer, en mat:
        Die op t'naetste // u
        Begeert voor goed, of schat,
        T'vertouven // lank,
        Is drouven // drank,
        T'behouven // dwanck // ter noot
        Te komen in dijn schoot,
        Dus toont u hulpe groot.
