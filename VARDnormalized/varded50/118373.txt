
    
      
        Ik moet een lied // in's Heren naam beginnen
        Op dat ik jet // mocht an u Ziel gewinnen:
        Al mocht ik dij // de gantse Wereld geven,
        het Heeft geen weerdy // bij een Godtsaligh leven:
        Der Zielen Goed // is het hoogste goed geprezen,
        Een rein gemoed // ontvangt Gods Heilig wezen.
      
      
        Wilt gij zijn een // verkoren Bruyt des Heren,
        Tot hem alleen // moet gij u liefde keren,
        Vergetet al // u Vaders huis en magen,
        Den koninh zal // in u hebben behagen,
        Maar is het dat gij // Egipten wilt gedenken,
        Zo en zal hij // zijn liefde u niet schenken:
      
      
        Ieloers is God // en met liefde ontsteken,
        Zelfs wil hij tot // ons Ziel inwendig spreken,
        Zijn Bedehuis // ter eren zijnen name,
        Moet rein en kuis // zijn, an Ziel en lichame:
        Geen sond' onrein // staat hier vrij om te dragen,
        Daar God alleyn // regeert na zijn behagen.
      
      
        Zeidet u van // de Wereltlijcke lusten,
        Roertse niet an // die u Ziele ontrusten,
        Laat zijne Geest // een woning in u maken,
        de Hemelse feest // zult gij dan lieflijk smaken
        Zijn Avondmaal // zult gij met hem genieten,
        Zo gij zijn taal // u niet en laat verdrieten.
      
      
        Zulk een beloft,  heeft ons den Heer gegeven
        Wie dit versloft // die mist het eeuwig leven,
        In duisternis // wandelt hij als de blinden,
        Zeer ongewis // waeyt hij met alle winden:
        Maar die verlaat // des werelds woeste wegen,
        En doet Gods raad // die heeft de rust verkregen
