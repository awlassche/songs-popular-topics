
    
      
        Gij Christelijcke lieve jeugd,
        Klein spruytjes, jonk van jaren,
        Laat varen // alle ijdel vreugd
        Verheugt // in denkt
        Zo veel gij meucht,
        En wilt u Ziel bewaren.
      
      
        Bewaard u Ziel en zorge draagt
        Dat gij niet wordt bevangen,
        Van slangen list die menig plaeght,
        Maar jaagt // en vraagt
        Wat God behaagt,
        Beveelt hem al u gangen.
      
      
        Let op u ganck, en zijt geen kind
        Al in ‘het verstand onvroedigh,
        Maar spoedig // zoekt en ondervint
        Hoe blind // zij mint //
        en heeft ghesint
        De wel-lust wankel-moedig.
      
      
        Wel-lust bedrieght als zij verblijd,
        ‘het Jolijt // is niet gedurig
        Doleurigh // haren dienaar krijt
        Als tijd // is kwijt //
        dan heeft hij strijd
        In zijn gemoet zeer treurig.
      
      
        En daar ‘het gemoet zo knaegt en wroet
        Onzoet // is sulcken leven,
        Bedreven // en vol angstig bloed,
        Maar doet // gij goed //
        in u gemoet
        Werd hogen troost gegeven.
      
      
        Wie dezen troost is toe ghevloeyt
        Besproeyt // van ‘s Heren zegen,
        Ter degen // hij in deughden blpoeyt
        Hij roeyt // en snoeyt //
        al wat er groeyt
        Door ‘het vleysschelijck genegen.
      
      
        Gheneghentheyt is den aenvanck
        Des sonds, dus wie het kwade
        heeft stadelijk in zijn bedwanck
        Lofsanck // en dank //
        van die ontfanck
        Bewijs hij vroeg en spade.
      
      
        Neemt aan in dank dit klein gedicht,
        Zijt den Leraar der scholen
        Bevolen // die u onder-ricjt
        Hij sticht // verlicht //
        en geeft gezicht
        Om nimmermeer te dolen.
