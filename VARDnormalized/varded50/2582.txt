
    
      
        Hou, Herder slecht // waar trekt de loop dus heen?
        (De Liefst oprecht // te Off'ren mijn gebeen
        Ja hert en zin // in will'ghe slavernij)
        Weg met de Min // t'is niet dan kasserry.
      
      
        Het is wel waar // ik ken mijn cleene macht,
        Dat niet een haar // Ik, bij heur ben geacht,
        Maar t'out gepeys // van deze schoon Goddin
        Ziet in het Fourneys // mijn Ziel door heete Min.
      
      
        Want als ik weer // aanzie des waarde Vrouw
        Vermeert 'tbegeer // en 'tverdubbelt mijn rouw,
        Haar schone glans // en lieffelijck ghedaant
        Mij nu althans // tot d'ouwe Liefd' vermaant.
      
      
        Door u gezicht // zo lijd ik Garen Brand,
        Wiens Fackels licht // verlicht mijn dom verstand
        Hoe wel ik swerf // na het gene dat mijn schaat
        Ja leevend' sterf // veel duizend doden quaat.
      
      
        Doch in ootmoed // werp ik mij sellifs neer
        Voor u te voet // verschoont mijn trots begeer
        U heuschen aard // laat blijken op het minst
        Ik bendt niet waard // al wens ik deze winst.
      
      
        Maar belght mijn Vrouw // dat een zo slechten slaaf
        Als ik mij houw // staat na de waartste gaaf:
        Maar niet dat gij // maar ik zal u ontzien
        Heerscht Liefste vrij // wat wilt gij mij gebien.
      
      
        Wat u Hert lust // gij krijgt wat u behaagt
        Van die d'Aard kust // die uwe Voeten draagt
        K'droech u op d'hand' // nam gij het Prinses in het goed:
        Komt sulcken brand' // ws dienaars te gemoet.
      
      
        Prinses te hoog // ik mijn begeer gevoel
        Voorwaar zat ooch // waart gij mijn wit mijn doel,
        Och zot syn mocht // maar zot niet mag gezien?
        Nemt u genocht // in mij dat ik u dien.
