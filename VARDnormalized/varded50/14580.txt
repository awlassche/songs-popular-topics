
    
      
        Mijn lief is mijn, en ik ben zijn,
        Die mij zo dikmaal trueren // doet:
        Als hij absent // hem van mij wendt,
        Droefheid mijn hert bezuren // moet.
        Dikmaal tot hem // heff' ik mijn stem,
        Die mij nooit te versmaden // plag,
        Met ongeluk // of zwaren druk,
        Als hij mij maar beladen // zag.
        Dus met geklag // en zwaar gewag
        Zoek ik, die mij verzaden // mag.
      
      
        Hij vrijdt mijn hert // van alle smert,
        En al mijn krankheid hij // verdraagt:
        Hij komt daar in // vernieuwt mijn zin,
        Verdrijft wat hem in mij // mishaeght,
        Dus is het met spoed // geworden zoet,
        Dat ik de wereld derven // mag:
        De zonden kwaad // ik nu zeer haat,
        Die ik zo noo te sterven // plag,
        Dat eerst mijn ziel // zo bitter viel,
        Om het selv' ik nu te erven // lag.
      
      
        Nu ben ik blij // van zorge vrij,
        Die mij eertijds benoude // zwaar:
        O werelds eer // quelt mij niet meer,
        'k Wil mij van u nu houden // klaar:
        Wellust, voorspoed, al het aardse goed
        Mag nu mijn ziel vermeyden // niet.
        Door haren raad // en valsen praat
        Ik mij wel eer verleiden // liet:
        Nu metter daad // ik vroeg en laat
        Wil God alleen verbeiden // ziet.
      
      
        Jesu mijn lief // al mijn gerief
        Gij blijven zult tot aller tijd:
        Met u alleen // ben ik te vre'en:
        Wilt mij toch helpen in den strijdt.
        Boven gheneught // en alle vreugd,
        De welke spruit van vlees en bloed,
        Is het vermaak // daar ik na haak,
        Van uwer liefden zoeten vloed.
        Om dezen smaak // ik heel versaeck
        Mijn eigen ziel, en des werelds goed.
