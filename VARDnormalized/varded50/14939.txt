
    
      
        VVAer is de lieve Maagd, gebleven
        Die mijn jonk herte zo behaagt,
        Waar is de denkt // die mij verheugt
        Ja die verslijt mijns levens vreugd:
        Die schone // persoone, wiens kracht
        En macht // bij dag en nacht
        Haar vriendelijkheid
        En minnelijkheid
        Haar zoet gezicht, mijn staech verleid.
      
      
        Eilaas! wat is de hoop, een zorge
        Een strijd ghestaech in s’weerelts loop,
        Alzo men ziet // en anders niet,
        Ja dikmaals volchter noch verdriet:
        Al zuchten // en duchten, heel zwaar
        Daar naar volgt dikwijls klaar,
        Na Dochters manier
        Zo groot getier
        D’its doch den aard, van t’minnevier.
      
      
        Hoe dikmaals zuchten ik, met tranen
        Wanneer ik viel in deze strik,
        O lijden groot // k’wens om de dood
        Zo zij mij laat in deze noot,
        Want Maghet // gij jaghet // mijn zeer
        Om veer // hoe lang hoe meer,
        Helpt mynder, doch nu,
        En weest niet schu
        K’en lief gheen ander lief dan u.
      
      
        Gheen vreugd ter Wereld schoon, verwacht ik
        Zo het niet is van u persoon,
        Een uur, een Jaar // dunkt mijn voorwaar,
        Maar sprak ik een reis tegen haar,
        Hoe luckigh // ondruckigh // mijn ziel
        Hem hiel // dat zulks gheviel,
        Zo haddet een endt
        Mijn groot torment,
        Zo was gedaan mijn zwaar ellendt.
      
      
        De vrees bevanght mij staech // o wonder
        Hoe dat mijn ziele is zo graag
        Om een Jonkvrouw // om een Kersouw
        Op wien ik stelle mijn vertrouw
        Maar lacy // ter spacy // ik quyn,
        Verdwijn // eilaas om dijn
        O Edele Maagd
        V is het beclaeght
        Gij zijt de Perel die mijn behaagt.
      
      
        Prinses Amstels-puyck, in schoonheid
        En weet ik gants niet uw’s gelijk
        Het Amstel-land // roem wydt bekant
        Mag hem roemen heel abonbant,
        Princesse,, Matresse // mijn schat,
        O dat // ghyer noch wat
        Mij kwaamt ter baat,
        Want Nimf ik laat
        De tongen boos, en alle de haat.
