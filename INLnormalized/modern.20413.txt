


ALs d'Harders haar Schaepjes bewaeckten zo scheen
De Stralen // te dalen // op d'Aerde benenen
Den Engel vertellen // de Harders in 't Velt
Godts Soon is geboren dus maken u te been


Dus loven en zingen de Enghelen reen
Treedt vaardig // 't is waardig // gij Harders gemeen
Verhoort ons stem // te Bethlehem
Daer sullen gij vinden dat Kindeken klein


De Hardertjes lopen en laten haar Vee
Met zingen // en springen // vol vreugde alree ;
Dus lopen zij al // en vinden de Stal ;
Daer brengen zij Iesus veel zoetigheid mede


Daer sage zij Iesus bij Beesten geleid
Ghewonden // en vinden // als haar was zeggen :
Doen ginghen zij weer // en loven den Heer ;
Sy hebben Godts grond genade verbreiden


Nou laten wi als deze Harders bij nacht
Ons leven // begeven // om houden goed wacht
En zoeken Godts Rijck // op dat wi ghelijck
Door Iesus Godts Kinderen worden achten
