


TObias tot sterven genegen
Riep zijn soon en spreken daar tegen
Mijn kind // beminnen
Hoort met akkoord // al mijn woorden


Als God mijn ziel hebben wechghenomen
Mijn lichaam in een graf laat komen
En wild // ook milt
V Moeder dragen // gunst al u dag


Houdt haar in en waard eend ' in eer
Denckt op perikel en 'tverseeren
Hoe zij // dragen du
Met grond smart // onder haar hart


End ' als zij zullen sterven wees :
Begraeftse nevens ik bij deze
Altijt // met vlijt
En naarstig pogen // hebben God voor ogen


Draeght in u hart Godt almachtig
Soo dat gij hem altijd ghedachtigh
Zijn meug // van jeugd
Tot 'slevens eend // t'hemwaerts u wenden


Siet dat gij niet t'eenighen staan
V en bewilligen in gheen sonde
T'ghebodt // van Godt
En wild versteken // noch gheensins breken


Van u goed helpen den armen
Aensiet zijlieden met ontfermen
By die // aanzien
Sal u ghestade // Godt met genade


Helpt naarstig die behoeftig leven
Hebt gij vedel // wild vedel gheven
En doen // ook god
Met hart reynigh // al hebben gij weinig


Gheeft trouwelijk al zondaar sparen
Soo sullen gij rechten loon vergaren
Ter node // die doden
En zenden lossen // de alemossen


De Aelmossen vol caritaat
Is in der noot een grond baat
En troost // altoost
Seer groot beseffen // voor God verheven


Mijne zoon wild u ook vermijden
Van hoerery laat die bezijden
Hebt een // Wijf reen
Wilt daar beneven // gheen meer aankleven


De hooveerdy en geven gheen stede
In u hart noch woorden mede
Van in // 'tbegin
Wracht zij 'tbederven // daar wi om sterven


End ' al die om u arbeiden
Terstont wild haren loon bereiden
En houden // niet stout
Van gheen persoon // verdiend lonen


En doen ook geen mens geschieden
Dat gij nood ' hebben van ander lieden
Ter node // u broden
Deelt t'allen tijden // die honger lijden


Wilt om den naken te bedekken
Van u kleed aantrekken
Maer meest // diegheest
Lijck zijn begraven // delen u gaaf


Met den Sondaeren vol vermeten
Soo en wild ook drinken noch eten
Maer gaan // om raad
Den Wijsen vraag // tot al dag


Danct altijd God en bidden eendrachtig
Dat hei u wil regeren krachtig
Dat fijn // al dijn
Voornemen blijken // zijn woord ghelijcke


En zorgen niet mijn Sone verheven
Al leggen wi een arm leven
Voorwaer // hier na
Wy hebben sullen // van als te vullen


Indien dat wi tot al tijd
God vrees en de zonde mijden
En god // doen vroed
Nae recht wi zullen // deze wel onthouden
