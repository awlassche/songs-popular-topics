


O Broeders reen // loven den Heere ghestade
Die hier enten plein // laat vlo zijn genade
Als een Fonteyn // voor die af staan van t ’ kwaad
En voort alleyn // met Christum gaan te raad
Die zo het graan // in goed aard ’ ontvangen
Worden van deze zaad
Kinderen vroed // van Godt den Vader god
Door zijne zoon bloed // vergeten overvloed
Light onder voet // dood en hel vertreden
Lof hem zijn moet // van zijn genade zoet
Wie boet // hier doen // zij staan noch heden


Eeuwich houden stand // s ’ Heeren goedheid prijzen
Hy hebben planten // zijn gemeend uitgelezen
Want triomfant // ten Hemel opgheresen
Ter rechter hand // zijne vader kort na deze
Sant zijn geest // met een wind tempeest
Tonghen ghedeelt enten wees
Vierich en klaar // op iegelijk van haar
T ’ ghetal aldaar // in zijn naâm te gaar
Dat en was maar omtrent twintig en honderd
Den Gheest hier na // werken wonderbaar
Heeft zwaar // de schaar // des Volcx verwonderd


Van Zuyt of Noort // uit menig lantpal
Heeft daar gehoren // een iegelijk zijn taal
Betuygen t ’ wart // en Gods werk loyaal
Vrees en discoord // hebben t ’ volk generaal
Want deze zijn // vol zoet Wijn
Ghedroncken al te maal
Spraken rebel // enig ’ uit spot en spel
Op zulk vertel // antwoord Petrus wel
Dits gheen voortstel // van dronk lieden
Maer door t ’ bevel // Gods na t ’ profeteersel
Ioel // blijken snel // hier deze gheschieden


O Prins o Heer // u macht laten gij aanschouwen
Van leden teer // een sterk huis ginght gij bouwen
Meer eend meer // laten Mannen en Vrouwen
Op u leer // haar doop met betrouwen
Maer door den nijd // der Ioden wijt
Verstroyt door zwaar benauwen
V dienaar/diener jent // van t ’ nieuw Testament
Tot s ’ wereld ent // maken u woord bekend
Heer excellent // lof prijs moet gij gewinnen
V wart omtrent // ons laat zijn ongeschend
Present // prenten // in ons zin
