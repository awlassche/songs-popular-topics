


WA ynne schuyll ' fenne heagste zit
Dy schil yn 't schaden fornachtje
Fenne ' Hoeder du fen slom naad wit
Dy g'noagsam ' God de ' Almacht'ge
Ick siz ik ernje yn mijn petear
Mijn Stinz ' dear ik op fortrouwje
Mijn toflecht ' binnen ' Y ljeave HEER !
Dear ik fest op gruwnje in bouwje


Hy sil du holpje ( ô fromme ziel ! )
Uwt ' Fuwgg'lers tijzjerns strijk
In uwt fordearsieck ' pest 't forniel
Dat 't tjerckhoaf dong't mei Lijcken
Hy stopp't Dy mei zijn fleeck ' ticht oer
Du fijn ' onder zijn wjuecke
Behâd : zijn wierheyts yvigdoer '
Kinste ' az dijn schermschild ' bruwcke


Du schit naad 't klienhertig blea
For ' Nachtgrijmgruw'le ' aeng schrillje
Du heste fen nin pijl nea
Dy deys de loft trogdrillje
For 't pestfjoer dat oer schuwll ' leagloer't
Dat flijppjend ' sluwpt ynn ' tjuester :
Nog fen 't fordear dat middeys schoert ?
Schey n't baar n't in breken toe bjuester


Dy tommelje ' oone ' liftse ' ig forby
Wol tuwzen ; in toe rjuecht 're
Tjien tuwzen ; lijck'wol blieuwste fry
Fenne ' yene ' en ' oog ' igs fjuechtre
Allinne ' iz 't du om om toe sjean
Om eernjende t'oereagjen
Ho 't godleaz ' schom schieden moat forgean
Trog ongeheard ' zwier pleagjen


Want mijn toflecht ' dat binnen ' Y HEER
Siel wezz ' naad heel mizmoedig
To de ' Alderheagste ' hestu dijn keer '
Hy de dijn fortjog hosck goedig
Dy sil nin quell ' nin quelck nin quea
Nog schea nog pleag ' weerfearje
Noge ' uwnck ynn ' gaest yn goed yn ljea
Dijn wenning'tint benearje


Hy sil zijn Ing'len ( : tinck ho goed
Ho goed iz God : ) belest'gje
Dat ja du yn sorgfad'ge njoe
Yn al ' dijn weg befest'gje
Jae zille du yn frjuenlijckheyt
Op holpmijlde ' hanne dreagje
Dat foor dijn foet nin dwerzstien leed
Mey seertme du toe pleagjen


Oer ' wrea Lieuw ' deadlijcke ' Adderslange
Oer ' Drake ' oer ' jong Lieuwe
Schitte ' onfortseage blea nog ' bang
Fry schealeaz traepje ' in klieuwe
Oermits hei ik ( kat God ) beminnen
Sil ik holpmijld him ontmette
Hy kin mijn Namme ' ( ô wolbesint ! )
Dearom sil ik him heag zet


Hy rop m ' om zo forhear ik him
Ynn ' neare ' eangst ' binnen ik bij him
Ick tjog him uwt neads kloere klim
ik Meyts him ontsjeande ' in ik fry him
ik Lit him yn deagen fol in sed
Bly libbjend ' lang foldoerje
Ja ik lit him yvig wol in bet
Mijn silligheyt begloerje
