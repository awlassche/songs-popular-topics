


STea bij behâd ' 't iz oertijd ljeave HEERE !
't Klien ' heapsen fromme ljue worden lijts in tin
De guwt klieut deugd duwckt vuil heet schieden schann ' eer
Dy trouw ' oprjuechte ' iz onder tuwzen nin


Jae sprecke ' oorz naad az smoargledd ' guwch'lerye
Elck flaey't zijn frjuene ' elk schild'ret wijt zijn zwart
Mey ' gledde tong ' schieden praat plomstrijckerye
Jae kâtje mei in duwbbelduwbbeld hert


Uwz God schil wis oefsnye ' uwt muwlle ' In wangen
d'Oergoune lipp ' 't oerhuynnigd ' gallefel :
God wol uwttjean mei gleone schroeynijptange
Dy greatop jaende tong ' njuencke him rebel


Dy oerjuwn ' quea helhuwnen bijlje ' in râlje
Uwz tong ' jout de ' oerhân uwz jild oonsjean steat
Uwz lippen binnen ' uwz eyn : wi kinnet stâlje :
Wa de maester oer uwz ? nimmen nimmen neat


Mar hertse ' het God zeggen : Om 't illindig ' lijden
Fenne ' earme ' onrjuechtferwoeste neaddriftleaz '
Wol ik opstean : Ick ik schilse ' hoedje ' in frye
Fenne har dyse ' oonblieze ' eangft ' spijt in freez '


Dy wirdden du God eernet ijn zijn sprecken
Dy binnen nolck rjuecht enkel lot 're klear
Az 't zilver dat ijnne ' uwn ' leed gleon toe blaken
Dat sânreyz ' smelt't in schomschien worden aldear


Y ljeave HEERE ' Y silleze bewearje
Fen 't fuwll ' wradschomm ' dat gunnen folck leagen leed
Het hoeder som ' uwz hoeder lijk bejearje ?
O wertse hoede ! uwz haed ijn yvigheyt


Dy boos binnen ' op 't moed in ruwn om troje
Jae bosckje gear ho ! 't rinnen roem ney bejear
Wenneerse 't schom dear snood schodden schoje
Op 't kessen fen lânz heagste steat ijn eer
