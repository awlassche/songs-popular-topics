


De Liefde dood // mijn schaamte root
Door t'fel bedwingen
Dwingt ik de noot // nu aan een twijfel groot
Merckt o Mondeken root
Met een voordacht op mijn zingen


Mijn hart voorwaar // dat zucht sieren zwaar
Met inwendig ik claaghen
O lief eerbaar // Isset u wijl maar
Ghy moet deze haast van daar
Met u leven gunst verjagen


Wilt gij ik leed // en mijn droefheid
Ghy sold met rede
Haest zijn bereid // met troostelijk bescheid
'tWelck van ik waard verbei
Met groot Lytsaemheden


Och tis mijn pijn // van u te zijn
Waerde Vriendinne
Soo waard gij mijn // die u oprecht en fijn
En niet met vals schijn
Maer met mijn hart beminne


Dat gij o Maecht // de Vlammen zacht
Die mijn vertederen
Of t'geen ik plagen // antwoord zo u behagen
En soud u ( Lief ) ik vragen
Mijn droef smart niet deren


Ick meen wel ja // Maechdelijck zaad
Bloemptje der Vrouwen
Met raad en daad // u dienaar/diener komen te baat
In deze bedroefd staan
Soo ik u deugd vertrouwen


Voorts ik begeer // met u in eer
Vreetsaem te leven
Al na de leer // van ons Godt den Heer
Dat hei ons Menschen teer
Tot stichting hebben ghegheven


Mocht ik deze scene // na wens gheschien
O Vrouw vol waard
Hier voor de leen // sou ik u Liefste been
Mijn trouw als gij die
In t'goet wilde aanvaarden
