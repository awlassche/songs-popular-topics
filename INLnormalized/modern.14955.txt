


SILVIA ik scholen en trots Goddin
Alleen den roem // en den bloem // van al scholen
Segt hoe lang sullen gij noch in u zin
My u slaaf och ! och ! met veel ellents gaan lonen ?
Sal ten geen tijden dan
Op houden deze plagen
Die ‘ k niet lang lijden en kan
Noch enten hart verdragen :
V gebaar // geest ik vaar // en zullen ik doen sterven
Wat suld gij // dan t ’ ontijd // meer van ik verwerven


Merct ô ik schoon // op vertoon // van ik min
En t ’ eeuwig besluit van ik getrouwicheden
Die t ’ al tijden achter haar Goddin
V ik lusten haar Goddelijck aanbidden
Sult gij die verlaten gaan
Oock ik u arm slaave :
Dat en kan ik niet verstaan
Want u goed gaaf
Neyghen na // het ghenaa // der barmherticheden
En u oog // tot medoogh // van liefde vrede


Al wat de Son // gestralen kennen // hier om laag
Is maar schijn // zo dunken ik van u Hemels wees ?
Ist dan gheen reen dat ik eerbiedig draagh
V naams lof boven al Sterren prijzen
Schoon Goddin ik troost ik hert
Al laat gij ik in lijden
K ’ hoop niet dat gij in ik smart
V noch sullen verblijden
Maer de vreuchds // zijn ik jeuchds // die ik hebben gegeven
In ik smart // t ’ meest hart // dat is u leven
