


HEEr als mijn Hert recht overdenck
Mijn Elendt zondig Leven
Daer in ik gans eend gaar versinck
Kan ik ik niet verheven
Ick keer mijn Ooghen totter Eerd '
( Helpt gij HEER t'Allen Daghen )
Het is met al ' mijn Doen niet waard
Moet aan ik zelf versagen


Als ik Roemd ' en denken aan mijn Macht
End Gaven daar beneven
End hebben ' op mijn goed Wercken acht
Wert mijn Ghemoedt verheven
Doch bleeff mijn Hert in Twijffel staan
Het kond ' hem t'geenen Daghen
Op du O HEER verlaten gaan
Het worden van Ancxt verslaan


Want als ik niet mijn Siel en sett
Op dijn Ghenaed ' te gronden
So worden zij met Onrust verlett
Verschrickt van straf der Sonden
So worden ' ik als een Kindt voorwaar
Dat steeds ( om Troost te krijgen )
Siet na zijn leven Moeder daar
End niemand kant doen zwijgen


Dies Raede ik dat Israel
Op Godt hoop ' Onverdrooten
Hy Rijcklijck ons zullen helpen wel
Syn Handt is niet sluiten
Syn Goetheyt eend Barmherticheyt
End zijn Ghenaed ' om prijs
Van nu aan tot in Eewicheyt
Wil hei ons al ' bewijs
