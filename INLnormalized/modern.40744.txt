


BEwerre ik ô God ik trouwje ' op Dy
Ick gruwnje ' op jong moed tinsen hert in zin
O sjongt mijn ziel Y binnen Heer oer ik ;
Dogg ' mijn goed dwaan kin ' Y nin baat trog winne
Mar d'heyl'ge Ljue op ' yerd ' gunnen ljeave uwtlezzen '
Wier bij mijn tier wijl ' in formeyts môt wezze


O ! smart ' in wee oer jerm ' waems gruwle dwaan
( : 't Spoar bjuester ' zijn môt pleage ' oer pleag ' bestiete : )

ik Schil dat Wierms oefferbloedwiet naad uwtjiette
My girzmet ho s ' opsnolckje de ' OEfgoads kreamen

Oljeave HEERE ' ! ô oermijlde ' yngoe God !
Mijn eyndeel erv ' ik kroes swietlibb'ne drippen
Binne Y ; eack onderhâder fen mijn lot
't Schiedmiett'tou koe naad ljeaflijckers oerglippe
Az mijn wirdd ' Deel O eyndeel heag fen wirdden !
In schieden erve ' ( ô lok oer lok ! ) iz mijn ' wierde


Ick lofsjong de ' HEERE ' ik lofsjongj ' yer in let
God du ik goed rei jout yn dwaan in lid :
Salm yn mijn slapen yn 't droagjen op mijn bed
Drieuwt 't ynborst ik toe sill'ge lederen ' in witten
ik Wol onophâdlijck op mijn goed God mickje
Hy laat mei eyn hân ik ; ho zij ik forwrickje


Loyts ' loyts ' mijn ziel hert muwlle ' ho blij ho blij !
