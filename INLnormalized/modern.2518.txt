


Myn zinnelijkheid // heb ik leggen // op een die ik moet derven
Die ik onthoud // 'twelck ik zo rout // dat ik vaak wens om sterven :
Niet dat ik haar // mijn Lief eerbaar // iets kwaad zij willen gunnen
Ja woud ' ik al // ( dat ik niet zullen ) ik en zij voorwaar niet kunnen


Want d'oogjens blij // die zij tot ik // eertijds zo liefelijk keren
Die hebben zij noch // waar door ik och ! haar hartelijk begeren
En haar welvaart // is lief en waar // in mijn getrouw zin
Souw 't hart van mijn // wel kunnen zijn // zondaar haar te beminnen ?


Nee n't zeker neen // het zucht en wenen // als het mijn Lief zieden lijden
En in haar vreugde // schep ik geneugte // en over groot verblijden
Want zij bezinnen // die haar beminnen // zo onberoemd mogen blijk
Geen last of noot // zullen voor de dood // ik van mijn Lief doen wiek


Hoe soud ' ik // zieden beminne niet // die zo men mogen bemerken ;
Is wijs van Raad // arm van praat // en rijk van goed werk :
Hier door 't oog viel // op d'ed'le Ziel // van mijn Lief uitgelezen
In wie ik ach ! denken dat ik zaak // het allerschoonst wees


Kleyn is mijn roem // of ik schoon noemen // haar welgeschikt zede
Want d'Ouders die // gaan zo ik zieden // haar aan Rijcker besteden
En die haar eerst // scheen wel begeerst // gaan vluchtig zij verlaten
Maer die aannemen // voor eigen vreemd // laat zek door 't geld bepraten


Het duivel Geld // hebben ook geweld // zieden ik in de mens zidel
Want die kwaad doen // kan met spoed // de Liefde licht verniel
Geen Liefd maar schijn // want zieden an mijn // of het ik Kan verkeren
Daermen Jonst kopen // hoe dat aflopen // zullen ons de Tijd wel leder
