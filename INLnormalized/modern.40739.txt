


WIerom uwz God wierom steane ' Y zo fier ?
Wierom oerschuwlle ' ynn ' tijd fen neare nead ?
Och ! 't godleas ' schom f joerhiet az wreade ' ondjier
Oermoedsch forfolgje 't earm ' from folck toe dead
d'Illindige ' achtje ' lijts ja min az neat
Litte ' har ' eyne ' oonslag har ' eyn hol ' oongrijpje
Litte ' har ' salmtins 't boos ' salmhert ' trogknijpje


Want ( ô fordoarne ' in ijn 't fordear forhirdd '
Dy great opjout fen 't fuwll ' trog him uwtrjuecht ' )
Dy godleaz ' bôlt de ' uwtwircke bijzepertt '
Hy spijllet raap in schrapen het rinnen oef fljuecht
Hy prijs't hel ' om slok yn al het er sjoecht
Hy âll't hei mâll't hei râll't smaad schamme oneer
( O Hymmelspijt ! ) hei lasteret de HEERE


Hy steckt zijn noaz om heag yn heagmoed dol
Hy ondertaest nog ondersijckt zijn dwaan
't Iz jamck trieut hei zijn blauw secken fol
Hy schil om fromheyt naad in fijtter ja
Hy tinckt : Dear iz nin God yn uwz oonslaen
Sijn weg bringt smart ' : Jôn dwaan ontfljuecht zijn miette
Hy bliest du him naad lijck't az moude ' oer striette


Hy kât him zalm in moed yn 't hert : Nae na
Forwrickje sil ik ninter tijd oef stuwn '
Fen bern toe bern sil ik ford gean buwtte kwaad
Sijn muwlle flockt in zwart yn 't vuil oerjuwn '
Hy bijllet ' list moeite ' onrjuecht az in huwn
Hy leagloer't t'schuwl onschild'ge ljoe toe deyen
Sijn eag mierck't om ' earme ' uwtte wrâd toe feyen


Allijck in Lieuwe ' ynn ' kuwl oer schuwl oonloer't
( Az hôlle honger yn zijn termen kryet )
In ljeapt op 't ninqueatinckjend ' Schiep : zo gloer't
Hy op d'illind'ge du hei 't not oerstiet
Hy duwckbuwgt az yen du wijld fuwggelt ' sjet
OEf az de Lieuw ' fôr ' zeggen mei deadlijck hapje
Komt hei 't earm ' folck mei ' lobben sterk fortrapjen


Hy zeggen yn 't hert ( ô beestig bot ! ) hei zeggen
God forjit 't sljuecht folcks ' pleag ' him om toe tjean
Hy sjoecht naad ney har ' om yn yvigheyt
Hy draaien zijn hol ' oef 't de him naad waard toe sjean
Stan op HEER ' God litt ' deze naad lang schean
Litt ' lang naad jôn hân ynn ' boes'me zit
Steek zien ' uwt HEER ' wol d'illind ' gij naad forjitte


Wierom last'ret du godleaz ' fuwltuwt God ?
Sizzende : Y sijkje naad ney het du det
Y sjea lijck'wol djiep trog zijn tjock hart slot
Y schoagje ' om wa 't hei trog fortriet forplett :
Op datme 't opjout Jo du 't wreckswird vet
Wier de ' earme ' him op forlit ynn ' nead in freze :
Yholpje ' in blieuwe ' in neadholp ' fenne Weeze


Breck ' briezje in buwts ' du godleaz ' earm omtwa
Sijkje ' al her ' dwaan ei ! grypknijp zien ' yn 't gemoed
Fijnne ' Y zien naad ? in grypjes ' ? wis y dwa
Jae stuwe ' az moal toe traap fen gunnen foet
De HEERE ' iz Kening du uwz yvig hoed't
In hoedje sil De Heyd'nen hei toe schânne
De Wrâd uwt stiet in drieutse ' uwt al zijn lânnen


O ljeave HEERE ' ! ho goed ho mijld ho suite
Habbe ' y 't seftmoedigh ' hert zijn winds forheard ?
Y sil ' him 't hert sterckje ' az mei Hymmelsck wede
Jon eer sil mijld toherckje ' op zijn bejeart '
Omm ' Weez ' bedold yn druwck dat hei kort steart
Rjuecht toe dwaan : Dat in Minscke ' ho hei moacht klieuwe
Naet ford ferre ' om meer boozdied ' toe bedrieuwen
