


HOe lang is 't heden // scholen Iuliaan
Doch wel geleden // dat gei hebben voldaan
Op rood tipje // duizentmaal kussen
Van zoete lipje // ons minnelust ?


't Was toen u woorden // en geveinsd Tong
Mijn hart bekoren // en tot kussen dwingen
't Was toen de vlam // van u geil Ziel
Mijn tegen komen en in mijn armen viel


Wat hebben gei strikken // listig uitleggen
Om te verstikken // mijn onnozelheid
Hoe kost gei vleien // midden in die vlam
Die strak na 't schei // bij een ander komen


Neen Iuliane // denken vrij dat mijn Min
Haar eerst tranen // offeren mijn Goddin :
Geen Liefde zwaar ik // dat van Iuliaan
Zal 't hart van Fredrik // in haar vlaam braden


Ik acht geen Eeden // die een Minnaar doen :
Zo vaak vertreden // met een los voet
Zo lang de dagen // zijn van Iuliaan
Zal haar u klagen // tegen 't hard staan
