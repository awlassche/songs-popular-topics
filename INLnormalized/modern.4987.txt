


Christus die roepen : mijn last mijn jok is zoet
Hy heylt en schenckt de rust van ‘ t gemoed :
Elck lustvernoegingh bieden // hei mild om niet
Hier is ’ s leven Broot // uit des Vaders schoot
Om Hel Duyvel Doodt // vernielen :
O rust o zoet rust der Zielen


Komt proeven de smaak die ’ s Hemels Manna hebben
Der Zwijnen draf gheen voedsel toch en geven :
V Heylbenijder brommen // en moedig romt
Op wees ’ leus schijn // suyvker zoet venijn ;
Met wees ’ leus schijn // de hart
Hy tronen maar lonen en kronen met smart


Of hei met du al vrij wat hoog rijst
End ’ al de macht en pracht des werelds wijst ;
Of hei dijn wrevel hoofd // al wraak beloven ;
Of hei lust en weelde // voor dijn ogen beeld
En u hart steeld // meinedig :
Dijn Ziel dijn arm Ziel blijven ledig


Daer spoken en kwellen de zwarte honghers straf
En dreigen na ‘ t eerst ook met het tweede Graf :
Hier wonen teghenlust // wroeging onrust ;
Hier is nummer vrede // maar een woeste zee ;
Hier is nummer vrede // maar plaag
Een worm die altijd blijven knagen


‘ t Gheschapen al des Scheppers eigendom
Deylt God natuers behoeftigheid om
Voor ‘ t dierelijck slachten // tot voedsel kracht :
Maer die boven maat // ’ t geen zo haast vergaan
Voor hun toeverlaat // verkiezen
Haer heil en hoog goed verlezen


Op driftig zand die heylgrond is gesteld
Daer zo ‘ t gheval kan bruik zijn geweld :
Maer wie op Christum sticht // geen winden zwicht
Of uit d ’ as al viel // ’ t hoog en lagher wiel
Iae dat lijf en Ziel // verdwenen
Zijn god zijn hoog goed blijven een
