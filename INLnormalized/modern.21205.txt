


Ach Bellinde ! leven Susje
Ach mijn woeste lusje
De zoetigheid // ach leven Meyt
Kan ik vergeten niet :
Want doen ik bij // u groen zeggen
Lestmael verkreegh mijn rusje
Ie troost ik // waar deur dat jij
Verkeerde mijn verdriet


Ick was heel moed en verslaan
Als een Hart van 't jagen
Daer toe bezwaren // en onghepaert
Als een verloren Schaep
Maer doen ik komen // bij u mijn Lam
Ghedaen was al mijn klagen
ik Was zo gerust // deur zoet lust
Viel ik van vreugde in slaap


Met ik was in slaap komen
Kreegh ik zulk Droomen :
My denken ik zien // een zoet gelach
Van Nymphjes in het groen :
Noch hoord ' ik een geluid ( zo 't scheen )
In 't ruisen van de Boomen
Iae wel ik denken // gheen ding en mogen
My meerder vreugde aan doen


Met deze Droomen ik ontwaken
Vyt mijn slaap gheraeckte
Waer deur ik schier // ach zoet Dier
Op springen van enkel vreugde :
En ook te meer // verblijd ' ik sieren
Om dat ik nochghenaeckte
V aardig Beeldt // zo op strelen
't Welck een jong Hart verheugen


Want gheen vreugde op deze Aerden
Is zo hoog in waard
Wanneer een Maeght // ghetrouw'lijck dragen
Haer Liefd ' op een Ionghman ;
Ach het is een zaak // vol zoet vermaak
Die noden de Ieught bezwaren :
Want Liefd ' altijd // doven haat en nijd
Ghelijckmen denken kan
