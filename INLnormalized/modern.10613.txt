


LOf prijs en eer zij v O Heer // en mild Vader
Die voor den morgen niet te zorg hebben bevelen
Dewijl gij alleen // En anders gheen // ons allegaar
Lijf en Siel voet // eend doen // honger en dorst vlieden
Voor de weldaad // en ghenaeden die ons gheschieden
Eyscht gij maar // Een hart dankbaar // wijlig en bereid
Dus t'allen tij // Motten wi // Oock al lieden
God hier boven // dankelijk loven // inder eeuwigheid
