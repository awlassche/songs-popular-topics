


HErtse ' Isr'els Haed ô Herder herckje
Dy Joseph laat az scheppen ôch ! merckje :
Dy twissche Cherubinen zit
( : Dy 't al ' heart sjoecht det det lit in wit : )
Brecke ' uwt ijn ljeacht ijn glanze ' ijn eer
( O Isr'els haed ! ô kreften Heer ! : )


Kâtje op gunnen kreeft az moarnstuwn'Sinne
For Ephr'ims eansichte ' in ford hen
For Benjamin Manasse ei ! komm '
Holpje ' ô forloss ' gunnen ljeave eyndomm '
God bring uwz weer ; gunnen eansicht'glanz '
Ljeachtje ' uwz dat de uwz behâdwiss ' schanss '


Ho lang ' HEER ' fil gunnen hiett ' en ' oprijckje
O kreften God ! in uwz besijkje
Och ! worden gunnen toarfjoer nea uwtdien
Jin 't ljuentjen fen gunnen droav ' folcx trien ' ?
Mey trien'brea fiede ' Y zien ' : ijn eagwiet
Lauwgje ' Yse ' uwtt ' Trieling ' fol fortriet


Y habbe ' uwz dwerzdrieuseh ' neyboer ' mecke
't Uwz haed ; uwz fijnne ' uwz scheetsch forsprecke
O great'greate ' al heirkreft ' God !
Bring uwz weer wezze ' uwz schuwlle ' uwz slot
Jon ljeacht ' ijnn ' druwcks droav ' tjuesterniss '
Laytsje ' ô dat de uwz behâd stinz ' wis


Jen Wijnstoack ' fen jong zalm ' uwt kippen en
Oerbrocht uwt 't slaefsch stoackhirdde ' Egipten
( Da d'Heyd'nen Y dreun ' uwt jearm ' Lân )
Dy poot ' Y dear mei gunnen eyn hân
Y habbe ' him pleats om wirtt'ljen dold
In dongge dat hei 't Lan breed fold '


Sijn schaden eer'n berg ' in berg ' oerteyn stoe
't Heeg ' Ceder g'lijck ' : zijn rank ' nauw eyn sjoe
Hy leatscheat fier om ' MiedSeestrann ' ?
Sijn tokken rek Euphraet rann '
Wierom stiette ' Y zijn moerre ' om fier
To ploayts ' fenn ' genger tjeave ' in Djier ?


De Wadbaergs wrot ' in 't Wijld fenn ' heide
Ontwirtt'let him in eert't zijn weide
O kreften God ! loyts ' fen om heag
Uwtte ' Hijmmel : keer ' dogg ' weer : gunnen ea
Besijckje ' ijn djiepp ' meylyenheyt '
Jon Wijnberge ' hoos ' ijn lijden leed


Dy staemme ' ô ! du gunnen rjeuchte ' han poot
In kreeft joe toe gunnen gloar ' du Lote :
Jae lizze ' om yessche ' ijn fijnnefjoer !
( : O staemme ' ! ô twijg ' eer'n waard in djoer )
Ja lizze ' omboecke ' ô Jae geane ' uwt
Trog Jon eansichts hird scheldgeluwt


Jon heen Jon rjuechter han ljeav ' HEERE !

Dy MinscheSoon du uwz to'n jeft '
Mey sterckime ' oonteyne ' iz ijn Jon kreeft
So nimme wi fen Jo naad meer
( : O foei dat wi 't ae diene : ) oefkeer
