


EEn Liedt eerbaar // van de lief klaar
Voorwaer Wil ik beginnen
Want bij haar // Soo rust wel zwaar
Eenpaer met hart en zin
Sy is sieren reen In sWeerelts plein
Men mogen gheen schoon vin
Want zij alleyn Is mijn shartsen grein
Dus moet ik haar beminnen


Haer wees fier // En weet ik hier
By niemand te ghelijcken
En haar manier // Seer goedertieren
Heeft ik doen blijk :
Maer sieren rebel // Past zij i n't spel
En ik gheen trouw doen blijk
Al lijd ' ik gekwel door deze opstel
Ick zullen haar niet ontwijken


Dat zij ik boot // Haer lipkens root
Mijn moet waar rijzen
Door haar liefde groot Ick zeggen v bloot //
Van die dood Waer ik genezen
Om dat zij wenden Van ik omtrent
Dus leven ik in vrees
Maer mijn Liefken jent // Seer excellent
Moet altijd zijn prijzen


Princesse god die mijn vlees en bloed
Jae doen ik mineren
Om dat ik moet // V wees zoet
Weest deze wel vroed // Dus lang ontberen
Nochtans ik zullen // Tot mijn verhaal
In liefde glorieren
Spijt nijder geschal // Liefd ' boven al
v lief exalteren
