


HOlp God ! ho binnen d'Heyd'nen heet oonhisse
Yn buwt yn vrek yn bloedtoarst ' gunnen erv'nisse

Jae habbe ' ( ô moarddeawuwn ' ! )
Jon tjienners lijk juwn '
Ynn ' Hauck Krie Uwllskloeren :
O ! 't fles ( dat eer'n zo mijld
Jôn ginst ' nutge ) iz om 't wijld
Op 't fjild juwn toe forschoerren


Jerm ' forzjend ' bloed gjealpt az streamm'wiet yn 't rinnen
Jeruws'lem wijd in breed om buwtte ' in binnen
O grouw'le ' ! ô huyne ' ô smaede ' ! az 't gref onwirddig
Is nimmen toe jearm'lijckbedobbjen firddig
Smaed dwerzdrieuw ' guwch huin spijt
Uwz neyboerte ' uwz tobijt
Dy om uwz wenje in swiermje
Och ! sil gunnen toarne ' ali ' stuwn '
Oonhietgje ' allijk in uwn '
Dear yn wi eynleaz kiermje ?
